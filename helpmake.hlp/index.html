     The Microsoft(R) Help File Maintenance Utility (HELPMAKE) creates
     help databases used by many Microsoft applications to display
     online help.
 
     Command-Line Syntax
 
     HELPMAKE [options] { /E[n] | /D[c] | /H | /? } { sourcefiles }

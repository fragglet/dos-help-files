─────Minimally Formatted ASCII──────────────────────────────────────────────
 
     Minimally formatted ASCII files simply define each context string
     and its topic text. These files cannot contain explicit links or
     formatting.
 
     A minimally formatted ASCII text file is a series of text
     segments, each preceded by one or more unique context definitions.
     Each context definition must be on a separate line beginning with
     a help delimiter (>>). In addition, a minimally formatted ASCII
     help file must begin with the help delimiter.

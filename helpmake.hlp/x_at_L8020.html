─────Setting Up Control Codes───────────────────────────────────────────────
 
     Control codes are codes in your source file that control the way
     the application displays topic text.
 
     Each control code (and any arguments) appears on a separate line
     of your source file.
 
     There are two methods of specifying control codes:
 
        ■ Use a special control character to identify control codes in
          your source text.
          See: ◄Control Characters►
 
        ■ Insert "dot commands" in your source text.
          See: ◄Dot Commands►

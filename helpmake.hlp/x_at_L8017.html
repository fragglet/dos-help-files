─────Specify Diagnostic Output (/V)─────────────────────────────────────────
 
     Syntax:  /V[n]
 
     This option specifies the verbosity of diagnostic output,
     depending on the value of <n>. Increasing the value adds more
     information to the output. If you specify only /V, HELPMAKE
     displays its most verbose output. If no verbosity is specified,
     HELPMAKE displays only the banner.
 
     The possible values of <n> are
 
     Option     Output
 
     /V         Maximum diagnostics
     /V0        No diagnostics and no initial banner
     /V1        Initial banner only
     /V2        Pass names
     /V3        Context strings on first pass
     /V4        Context strings on each pass
     /V5        Any intermediate steps within each pass
     /V6        Statistics on help file and compression
 
     Regardless of what option you use, warnings and error messages are
     always output.

─────Highlighting: Example──────────────────────────────────────────────────
 
          \bReturns:\p    a handle if successful, -1 if not
 
     In the example above, the text "Returns:" is in bold while the
     rest of the line is plain.
 
          \bThis is bold\p
          \u\bThis is bold and underlined\p
          \bBold, \ubold and underlined, \bunderlined\p
          This is plain.
 
     In this example, the first line is boldfaced; the second is
     boldfaced and underlined; the third is boldfaced, then boldfaced
     and underlined, then underlined only; and the fourth is plain.
     Note that the formatting flags are toggles, turning an attribute
     on if off and vice versa. Also note that attributes can overlap.
 
     The \p at the end of each line turns off the attributes.

─────What Is HELPMAKE?──────────────────────────────────────────────────────
 
     The Microsoft Help File Maintenance Utility (HELPMAKE) creates
     help databases used by many Microsoft applications to display
     online help.
 
     HELPMAKE operates in two ways:
 
        ■ Encoding: creating a help database from a source text file
 
        ■ Decoding: creating a source text file from a help database
 
          ┌────────────────────┐   HELPMAKE   ┌─────────────────────┐
          │  Help Source File  ├─── encode ──►│  Help Database File │
          │       (.SRC)       │◄── decode ───┤       (.HLP)        │
          └────────────────────┘              └───────────┬─────────┘
                                                          │

                                                Application Using Help
 
     HELPMAKE is required to create and modify help databases. These
     databases are read and displayed by common "help-engine" software
     built into Microsoft applications.
 
     You can use HELPMAKE to modify a help database supplied with a
     Microsoft application.

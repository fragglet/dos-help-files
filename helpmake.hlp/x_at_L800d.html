Specify Control Character (/A)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Specify Control Character (/A)─────────────────────────────────────────
 
     Syntax:  /Ac
 
     This option specifies <c> as an application control character for
     the help database file. The control character marks a line that
     contains control codes passed to the application displaying help.
 
     Many Microsoft applications use the colon (:) as a control
     character. Any line in a source file that begins with a colon is
     passed to the application.
 
     WARNING: If you modify a Microsoft help database, be sure to use
              /A: to specify the colon as a control character.
 
     For example, the following line in a help source file tells the
     application that the help screen should be 15 lines from top to
     bottom:
 
          :l15

─────Specify Type of Input File (/S)────────────────────────────────────────
 
     Syntax:  /Sn
 
     This option specifies the type of input source file. The value of
     <n> indicates the type of source file:
 
     Option     File Type
 
     /S1        Rich Text Format (RTF)
     /S2        QuickHelp format (default)
     /S3        Minimally formatted ASCII
 
     See: ◄Preparing a Help Source File►

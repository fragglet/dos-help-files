─────Links──────────────────────────────────────────────────────────────────
 
     Links allow help users to move quickly from one topic to another.
     You can link from a topic in one database to
 
        ■ Another topic in the same help database
 
        ■ Topics in other help databases
 
        ■ Unformatted ASCII files
 
     There are two types of links: implicit and explicit.
 
     Implicit Links
 
     An implicit link is established when you label a section of topic
     text with a context string that is a keyword or key term in your
     database. A user who requests help on the term anywhere in the
     database will be automatically linked to that section of text.
 
     Explicit Links
 
     An explicit link is established when you define a "hot spot" of
     one or more words in your text, and then associate a specific
     context string with that hot spot.
 
     When users place the cursor on the hot spot and request help, they
     see the section of the database you identified when you set up the
     link.

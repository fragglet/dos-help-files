─────Display Help (/H)──────────────────────────────────────────────────────
 
     Syntax:  /H[ELP]
 
     This option displays the help file you are reading. If HELPMAKE
     cannot locate QuickHelp or the help file, it displays a brief
     summary of HELPMAKE command-line syntax.
 
     No files are encoded or decoded. All other information on the
     command line is ignored.

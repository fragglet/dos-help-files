─────Topics─────────────────────────────────────────────────────────────────
 
     Each topic in a help database is a block of text describing a
     subject. Applications display a topic in response to a user's
     request for help on a subject. Most applications display a topic
     in a scrollable window. The actual implementation varies depending
     on the application.
 
     Context Strings Identify Topics
 
     Each topic in a help source file is preceded by one or more
     context definitions. One topic consists of all text from the end
     of one set of context definitions to the start of another set of
     context definitions.
     See: ◄Context Strings►
 
     Topic Limitations
 
        ■ Each topic can contain up to 64K of unencoded text.
 
        ■ A single help database can contain up to 32,767 topics.
 
     See: ◄Setting Up Context Strings and Topics►

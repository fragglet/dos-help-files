─────Translate Commands (/T)────────────────────────────────────────────────
 
     Syntax:  /T
 
     During encoding, this option translates dot commands to
     application-specific commands. During decoding, the /T option
     translates application commands to dot commands. The /T option
     causes the option /A: to be assumed.
 
     See: ◄Dot Commands►

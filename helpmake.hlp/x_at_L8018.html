─────Specify Width (/W)─────────────────────────────────────────────────────
 
     Syntax:  /Wn
 
     This option formats the text in all topics to a fixed width of <n>
     characters. You can enter any value for <n> in the range 11 to
     255. If the /W option is omitted, the default width is 76
     characters.
 
     When encoding an RTF source file (using the /S1 option), HELPMAKE
     wraps the text to <n> characters. When encoding a QuickHelp-format
     source file (using the /S2 option) or a minimally formatted ASCII
     source file (using the /S3 option), HELPMAKE truncates lines to
     <n> characters.
     See: ◄Specify Type of Input File (/S)►
          ◄Preparing a Help Source File►
 
     Formatting flags used for highlighting (\b, \i, \u, and \p) do not
     count toward the width. Likewise, invisible text──using the \v
     formatting flag──does not count. Lines that begin with an
     application-specific control character are truncated to 255
     characters regardless of the width specification.
     See: ◄Formatting Flags►

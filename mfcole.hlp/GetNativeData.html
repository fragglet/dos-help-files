  virtual HANDLE GetNativeData();
 
  Remarks
 
  This helper function calls the CObject::Serialize member function to
  build a block of memory that contains the server item data in serialized
  form.
 
  This is an advanced feature.
 
  Return Value
 
  A handle to Windows global memory.

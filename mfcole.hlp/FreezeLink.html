COleClientItem::FreezeLink

  virtual BOOL FreezeLink( const char FAR* lpszFrozenName );
 
  Parameter        Description
 
  <lpszFrozenName> A null-terminated string that specifies the unique name
                   of the new (embedded) OLE item to create (cannot
                   contain the slash (/) character).
 
  Remarks
 
  Changes a linked OLE item into an embedded OLE item (based on the
  current state of the linked data). You will rarely use this function
  directly. The global function AfxOleLinksDialog (accessed from the Links
  command on the Edit menu) uses FreezeLink when the user clicks the
  Freeze Link button in the resulting dialog box.
 
  Return Value
 
  TRUE if successful; otherwise FALSE.

  protected:
      virtual OLESTATUS
               OnSetColorScheme( LPLOGPALETTE <lpLogPalette> );
 
  Parameter      Description
 
  <lpLogPalette> A pointer to a Windows LOGPALETTE structure.
 
  Remarks
 
  This virtual callback function is called in response to
  COleClientItem::SetColorScheme. The default updates the m_hPalette data
  member. Override this function if you need to do more.
 
  Return Value
 
  See the ◄COleException► class for a list of return values. The value
  OLE_OK indicates that the function operated correctly.

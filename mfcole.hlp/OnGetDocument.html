  protected:
      virtual COleServerItem* OnGetDocument() = 0;
 
  Remarks
 
  You must implement this virtual callback function to return an OLE
  server item object (linked or embedded) that represents the entire
  document. If you are implementing a multiple-instance SDI server, then
  this function will only be called once for the document.

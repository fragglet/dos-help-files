  protected:
      virtual OLESTATUS OnRelease();
 
  Remarks
 
  This callback function is called when OLESVR.DLL releases this server.
  The OnRelease member function is called when it is safe for the server
  to delete all structures and quit.
 
  You will seldom need to override OnRelease.
 
  Return Value
 
  See the ◄COleException► class for a list of return values. The value
  OLE_OK indicates that the function operated correctly.

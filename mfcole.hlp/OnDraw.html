  protected:
      virtual BOOL OnDraw( CMetaFileDC* <pDC> ) = 0;
 
  Parameter   Description
 
  <pDC>       A pointer to a CDC device context object on which to draw
              the item.
 
  Remarks
 
  You must implement this function to draw the server item to the
  boundaries set in the m_rectBounds data member. The OnDraw member
  function is called in response to the client's COleClientItem::Draw
  function.

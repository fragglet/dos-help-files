  protected:
      virtual OLESTATUS OnRelease();
 
  Remarks
 
  This virtual callback is usually called in response to
  COleClientItem::Delete. Override this function to delete the server
  item.
 
  There is no default implementation.
 
  Return Value
 
  See the ◄COleException► class for a list of return values. The value
  OLE_OK indicates that the function operated correctly.

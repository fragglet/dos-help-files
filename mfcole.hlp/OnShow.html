  protected:
      virtual OLESTATUS OnShow( BOOL <bTakeFocus> ) = 0;
 
  Parameter    Description
 
  <bTakeFocus> TRUE if the server item should take the input focus;
               otherwise FALSE.
 
  Remarks
 
  You must override this pure virtual function to process a client request
  to show an item that results from a primary-verb
  COleClientItem::Activate call. Ordinarily, OnShow makes the server item
  visible.
 
  Return Value
 
  See the ◄COleException► class for a list of return values. The value
  OLE_OK indicates that the function operated correctly.

  Description
 
  COleClientDoc is the Microsoft Foundation Class Library base class for
  Object Linking and Embedding (OLE) client documents. A "client document"
  is a compound document that contains different kinds of data in various
  formats. The client document acts as a container for individual OLE
  client items and usually corresponds to an application's document
  window.

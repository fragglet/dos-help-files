  protected:
      virtual OLESTATUS
               OnSetColorScheme( LPLOGPALETTE <lpLogPalette> );
 
  Parameter      Description
 
  <lpLogPalette> A pointer to a Windows LOGPALETTE structure.
 
  Remarks
 
  This virtual callback function is called when a client sets the color
  scheme for this server document. The default implementation saves the
  color information in the m_hPalette data member.
 
  Return Value
 
  See the ◄COleException► class for a list of return values. The value
  OLE_OK indicates that the function operated correctly.

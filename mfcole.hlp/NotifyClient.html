  int NotifyClient( OLE_NOTIFICATION wNotification );
 
  Parameter       Description
 
  <wNotification> An OLE notification code. Must be one of the following:
 
                  Value             Meaning
 
                  OLE_CHANGED       Item has been changed
 
                  OLE_SAVED         Item has been saved
 
                  OLE_CLOSED        Item has been closed
 
                  OLE_RENAMED       Item has been renamed
 
                  OLE_QUERY_PAINT   Interruptible paint support
 
                  OLE_RELEASE       Item is released (asynchronous
                                    operation is completed)
 
  Remarks
 
  Notifies the client that this server item has changed. Rarely called
  directly.

  BOOL GetSize( LPPOINT lpSize );
 
  Parameter   Description
 
  <lpSize>    A pointer to a CSize object or POINT structure that is the
              destination for size information.
 
  Remarks
 
  Retrieves the size of the item. This information is used to determine
  the space required for drawing.
 
  Return Value
 
  FALSE if the client item is blank; otherwise TRUE.

  void NotifyRename( const char FAR* lpszNewName );
 
  Parameter     Description
 
  <lpszNewName> A null-terminated string that specifies the fully
                qualified name of the new server document.
 
  Remarks
 
  Notifies all clients that a server document has been renamed. Call this
  function after you rename the server document to update all active
  links.

  void NotifyRename( const char FAR* lpszNewName );
 
  Parameter     Description
 
  <lpszNewName> A null-terminated string that specifies the new name of
                the document. Must be a valid filename.
 
  Remarks
 
  You must call this function after the user renames the client document
  (with a Save As command from the File menu, for example).

EM_COPY (1.2)

────────────────────────────────────────────────────────────────────────────
 
#define INCL_WINENTRYFIELDS
 
EM_COPY
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an EM_COPY message to paste the current selection in an
entry-field control to the clipboard in CF_TEXT format. The current
selection in the control is not changed.
 
This message does not use any parameters.
 
Return Value
 
The return value is TRUE if the operation is successful or FALSE if an error
occurs.
 
See Also
 
EM_CUT, EM_PASTE

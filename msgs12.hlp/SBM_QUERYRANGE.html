#define INCL_WINSCROLLBARS
 
SBM_QUERYRANGE
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an SBM_QUERYPOS message to get the minimum and maximum
values of a scroll bar.
 
This message does not use any parameters.
 
Return Value
 
The return value is the scroll-bar range. The low word contains the minimum
value; high word contains the maximum value.
 
See Also
 
SBM_QUERYPOS, SBM_SETPOS, SBM_SETSCROLLBAR

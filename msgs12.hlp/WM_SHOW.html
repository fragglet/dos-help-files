#define INCL_WINMESSAGEMGR
 
WM_SHOW
fShow = (BOOL) SHORT1FROMMP(mp1);    /* show flag */
 
The WM_SHOW message is sent when the visible state of a window changes
(controlled by the WS_VISIBLE style bit).
 
The visible state of a window can be changed by calling the WinShowWindow
function. This state is not affected by the movement of other windows which
may change the visible region of a window.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
fShow      Low word of mp1. Specifies whether the visible state of the
           window is shown or hidden. If this parameter is nonzero, the
           window is being shown. If it is zero, the window is being
           hidden.
 
Return Value
 
An application should return zero if it processes this message.

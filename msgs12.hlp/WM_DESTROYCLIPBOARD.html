#define INCL_WINCLIPBOARD
 
WM_DESTROYCLIPBOARD
 
The WM_DESTROYCLIPBOARD message is sent when the clipboard is emptied as the
result of a call to the WinEmptyClipbrd function.
 
This message does not use any parameters.
 
Return Value
 
An application should return zero if it processes this message.

#define INCL_WINENTRYFIELDS
 
EM_PASTE
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an EM_PASTE message to replace the current selection in
an entry-field control with the contents of the clipboard if the clipboard
data is in CF_TEXT format.
 
This message does not use any parameters.
 
Return Value
 
The return value is TRUE if the operation is successful or FALSE if an error
occurs.
 
See Also
 
EM_COPY, EM_CUT

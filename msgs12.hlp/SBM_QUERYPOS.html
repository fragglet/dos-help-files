SBM_QUERYPOS (1.2)

#define INCL_WINSCROLLBARS
 
SBM_QUERYPOS
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an SBM_QUERYPOS message to get the current position of
the slider in a scroll-bar window.
 
This message does not use any parameters.
 
Return Value
 
The return value is the current position of the slider.
 
See Also
 
SBM_QUERYRANGE, SBM_SETPOS, SBM_SETSCROLLBAR

EM_CUT (1.2)

#define INCL_WINENTRYFIELDS
 
EM_CUT
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an EM_CUT message to delete the current selection in an
entry-field control and place the selection into the clipboard in CF_TEXT
format.
 
This message does not use any parameters.
 
Return Value
 
The return value is TRUE if the operation is successful or FALSE if an error
occurs.
 
See Also
 
EM_COPY, EM_PASTE

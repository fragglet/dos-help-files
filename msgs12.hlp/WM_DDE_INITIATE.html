#define INCL_WINDDE
 
WM_DDE_INITIATE
mp1 = (MPARAM) ((HWND) hwnd);    /* sender's window              */
mp2 = MPFROMP(pddei);            /* pointer to DDEINIT structure */
 
The WM_DDE_INITIATE message is sent by a client application to exchange data
with one or more server applications. This message is often sent to all
current applications by calling WinBroadcastMsg.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
hwnd       Low and high word of mp1. Identifies the sender application's
           window.
 
pddei      Low and high word of mp2. Points to a DDEINIT structure that
           contains an application name and a topic name. All applications
           with matching names that support the topic are expected to
           acknowledge by calling the WinDdeRespond function.

#define INCL_WINLISTBOXES
 
LM_SETTOPINDEX
mp1 = MPFROMSHORT((SHORT) iItem);    /* item index             */
mp2 = 0L;                            /* not used, must be zero */
 
An application sends an LM_SETTOPINDEX message to scroll an item to the top
of a list box.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
iItem      Low word of mp1. Specifies the index of the item to place at the
           top of the list box.
 
Return Value
 
The return value is TRUE if the operation is successful or FALSE if an error
occurs.

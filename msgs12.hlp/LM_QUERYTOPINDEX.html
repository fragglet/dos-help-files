#define INCL_WINLISTBOXES
 
LM_QUERYTOPINDEX
mp1 = 0L;   /* not used, must be zero */
mp2 = 0L;   /* not used, must be zero */
 
An application sends an LM_QUERYTOPINDEX to determine the index of the item
currently at the top of the list box.
 
This message does not use any parameters.
 
Return Value
 
The return value is the index of the item currently displayed at the top of
the list-box window, or LIT_NONE if the list is empty.

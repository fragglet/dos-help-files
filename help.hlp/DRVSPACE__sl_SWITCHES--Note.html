Removing /SWITCHES settings
 
To remove any /SWITCHES settings in your DRVSPACE.INI file, you must edit
the file directly. To edit the file, use a text editor such as MS-DOS Editor
(EDIT).
 
DRVSPACE /SWITCHES and the SWITCHES command
 
The DriveSpace /SWITCHES command and the SWITCHES setting in the
DRVSPACE.INI file affect only the behavior of the CTRL+F5 and CTRL+F8 keys.
To control the behavior of the F5 and F8 keys, use the <SWITCHES> command.

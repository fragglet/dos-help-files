                        DRVSPACE /MAXREMOVABLEDRIVES
 
Modifes the MAXREMOVABLEDRIVES setting in the <DRVSPACE.INI> file. The
MAXREMOVABLEDRIVES setting specifies how many additional drives DriveSpace
should allocate memory for when your computer starts. This determines how
many additional compressed drives you can create, compress, or mount without
restarting your computer.
 
Note:  For this switch to take effect, you must restart your computer.
 
Syntax
 
    DRVSPACE /MAXREMOVABLEDRIVES=n
 
Parameter
 
n
 
    The number of additional drives for which DriveSpace should allocate
    memory when your computer starts. DriveSpace allocates 96 bytes of
    memory for each additional drive.

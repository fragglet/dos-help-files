                             UNFORMAT──Examples
 
To determine whether UNFORMAT can restore a formatted disk in drive A, type
the following command:
 
    unformat a: /test
 
To restore a formatted disk in drive A, listing all files and
subdirectories, type the following command:
 
    unformat a: /l

PAUSE
◄Notes►  ◄Example►
────────────────────────────────────────────────────────────────────────────
 
                                   PAUSE
 
Suspends processing of a batch program and displays a message that prompts
the user to press any key to continue. You can use this command only within
batch programs.
 
Syntax
 
    PAUSE

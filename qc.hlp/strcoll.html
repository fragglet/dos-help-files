strcoll
 ◄Description►                               ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    int strcoll( char *string1, char *string2 );
 
  Returns:   a negative value if <string1> is less than <string2>,
             0 if <string1> is equal to <string2>, or a positive value
             if <string1> is greater than <string2>.
 
  See also:  localeconv, setlocale, strcmp, strncmp, strxfrm

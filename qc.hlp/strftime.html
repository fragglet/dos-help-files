  Include:   <time.h>
 
  Syntax:    size_t strftime( char *string, size_t maxsize,
                              char *format, struct tm *timeptr );
 
  Returns:   the number of characters stored in <string> if successful,
             or 0 if not.
 
  See also:  localeconv, setlocale, strxfrm

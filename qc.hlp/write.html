  Include:   <io.h>, <errno.h>
 
  Syntax:    int write( int handle, void *buffer, unsigned count );
 
  Returns:   the number of bytes actually written if successful,
             or -1 if not.
             errno: EBADF, ENOSPC
 
  See also:  fwrite, open, read

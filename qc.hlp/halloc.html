  Include:   <malloc.h>
 
  Syntax:    void _huge *halloc( long num, size_t size );
 
  Returns:   a pointer to the allocated space if successful, or NULL
             if not.
 
  See also:  calloc, _ffree, _fmalloc, free, malloc, _nfree, _nmalloc

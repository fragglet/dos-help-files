_dos_settime
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _dos_settime routine uses system call 0x2D to set the current
     time to the value stored in the dostime_t structure pointed to by
     <time>, as defined in DOS.H.
 
     Return Value
 
     If successful, the function returns 0. Otherwise, it returns a
     nonzero value and sets errno to EINVAL, indicating that an invalid
     time was specified.

putch
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The putch function writes the character <c> directly (without
     buffering) to the console.
 
     Return Value
 
     The function returns <c> if successful, and EOF if not.

atof, atoi, atol, _atold
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdlib.h> or <math.h>  (atof)
             <stdlib.h>              (atoi, atol)
             <math.h>                (_atold)
 
  Syntax:    double atof( char *string );
             int atoi( char *string );
             long atol( char *string );
             long double _atold( char *string );
 
  Returns:   the converted string, or 0 if the string cannot be
             converted.
 
  See also:  ecvt, fcvt, gcvt

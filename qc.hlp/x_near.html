  Keyword:   _near
 
  Syntax:    _near declarator
 
  Summary:   Specifies that a data object resides in the default data
             segment. Specifies that a function resides in the current
             code segment. Functions and data are referenced with a
             16-bit address, and pointers declared with _near are
             16-bit values. The _near modifier should appear before the
             name of the function or the data object being declared.
             For compatibility with previous versions, near is a
             synonym for _near.
 
  See also:  _based, _far, _huge

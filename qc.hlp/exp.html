────────────────────────────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    double exp( double x );
             long double expl( long double x );
 
  Returns:   the exponential value if successful, HUGE_VAL on overflow,
             or 0 on underflow.
             errno: ERANGE
 
  See also:  log

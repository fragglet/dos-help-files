────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    int strcmp( char *string1,  char *string2 );
             int _fstrcmp( char _far *string1, char _far *string2 );
 
  Returns:   a negative value if <string1> is less than <string2>,
             0 if <string1> is equal to <string2>, or a positive value
             if <string1> is greater than <string2>.
 
  See also:  memcmp, memicmp, strcmpi, strncat, strncmp, strncpy,
             strnicmp, strrchr, strspn

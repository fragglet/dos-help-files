────────────────────────────────────────────────────────────────────────────
 
  Include:   <io.h>, <sys\types.h>, <sys\stat.h>, <errno.h>
 
  Syntax:    int chmod( char *filename, int pmode );
             pmode: S_IWRITE, S_IREAD, S_IREAD | S_IWRITE
 
  Returns:   0 if successful, or -1 if not.
             errno: ENOENT
 
  See also:  access, creat, fstat, open, stat

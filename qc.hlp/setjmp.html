────────────────────────────────────────────────────────────────────────────
 
  Include:   <setjmp.h>
 
  Syntax:    int setjmp( jmp_buf env );
 
  Returns:   0, or the <value> argument of longjmp if returned by a
             longjmp call, or 1 if the longjmp <value> argument is 0.
 
  See also:  longjmp

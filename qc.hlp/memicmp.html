memicmp, _fmemicmp
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>, <memory.h>
 
  Syntax:    int memicmp( void *buf1, void *buf2, unsigned count );
 
             int _fmemicmp( void _far *buf1, void _far *buf2,
                            unsigned count );
 
  Returns:   a negative value if <buf1> is less than <buf2>, 0 if
             <buf1> is equal to <buf2>, or a positive value if <buf1>
             is greater than <buf2>.
 
  See also:  memccpy, memchr, memcmp, memcpy, memset

  Macro:     __TIMESTAMP__
 
  Summary:   The date and time of the last modification of the source
             file, expressed as a string literal in the following form:
 
                  Ddd Mmm [d]d hh:mm:ss yyyy
 
  See also:  __DATE__, __TIME__

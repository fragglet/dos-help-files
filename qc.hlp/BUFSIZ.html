  Constant:  BUFSIZ
 
  Include:   <stdio.h>
 
  Context:   Stream I/O functions.
 
  Synopsis:  Buffers used in stream I/O are of size BUFSIZ by default.
             This value is generally used to establish the size of
             system-allocated buffers. It is also the required size
             that you must pass to the setbuf routine when allocating
             a buffer.

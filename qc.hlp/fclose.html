fclose, fcloseall
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int fclose( FILE *stream );
             int fcloseall( void );
 
  Returns:   (fclose) 0 if successful, or EOF if not.
             (fcloseall) the total number of streams closed if
                         successful, or EOF if not.
 
  See also:  close, fdopen, fflush, fopen, freopen

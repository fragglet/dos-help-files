FP_OFF, FP_SEG

────────────────────────────────────────────────────────────────────────────
 
     The FP_OFF and FP_SEG macros can be used to set or get the offset
     or segment, respectively, of the far pointer at <address>.
 
     Return Value
 
     The FP_OFF macro returns an offset. The FP_SEG macro returns a
     segment address.

  Include:   <math.h>, <errno.h>
 
  Syntax:    long double coshl( long double x );
             long double sinhl( long double x );
             long double tanhl( long double x );
 
  Returns:   (coshl) the hyperbolic cosine of <x>, or HUGE_VAL if the
                     result is too large.
             (sinhl) the hyperbolic sine of <x> if successful, or
                     ±HUGE_VAL if not.
             (tanhl) the hyperbolic tangent of <x>.
             errno: ERANGE
 
  See also:  acosl, asinl, atanl, atan2l, _matherrl, sinl, tanl

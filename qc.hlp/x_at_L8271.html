     The ungetch function pushes the character <c> back to the console,
     causing <c> to be the next character read by getch or getche. The
     ungetch function fails if it is called more than once before the
     next read. The <c> argument cannot be EOF.
 
     Return Value
 
     The ungetch function returns the character <c> if it is successful.
     A return value of EOF indicates an error.

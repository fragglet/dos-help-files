  Include:   <signal.h>
 
  Syntax:    int raise( int sig );
             sig: SIGABRT, SIGILL, SIGSEGV, SIGFPE, SIGINT, SIGTERM,
                  SIGUSR1, SIGUSR2, SIGUSR3
 
  Returns:   0 if successful, or a nonzero value if not.
 
  See also:  abort, signal

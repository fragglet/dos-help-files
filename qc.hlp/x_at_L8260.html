strlen, _fstrlen
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The strlen function returns the length in bytes of <string>, not
     including the terminating null character (\0).
 
     The _fstrlen form of this function is a model-independent (large-
     model) form that uses a far pointer form of the <string> argument.
     This model-independent function can be called from any point in
     the program.
 
     Return Value
 
     The strlen and _fstrlen functions return the string length. There
     is no error return.

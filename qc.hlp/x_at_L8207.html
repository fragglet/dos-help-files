Rotate Functions
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The rotate functions rotate the specified value by <shift> bits.
     The functions, rotation direction, and argument type for <value>
     are shown below:
 
     Function     Rotation     Argument Type
 
     _lrotl       left         long
     _lrotr       right        long
     _rotl        left         unsigned
     _rotr        right        unsigned
 
     Bits rotated off the end of <value> are "wrapped" to the other
     end.
 
     Return Value
 
     These functions return the rotated value. There is no error
     return.

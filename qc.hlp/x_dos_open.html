_dos_open
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>, <fcntl.h>, <share.h>
 
  Syntax:    unsigned _dos_open( char *filename, unsigned mode,
                                 int *handle );
             mode: O_NOINHERIT, O_RDONLY, O_RDWR, O_WRONLY, SH_COMPAT,
                   SH_DENYNO, SH_DENYRD, SH_DENYRW, SH_DENYWR
                   (may be joined by |)
 
  Returns:   0 if successful, or the DOS error code if not.
             errno: EACCES, EINVAL, EMFILE, ENOENT
 
  See also:  _dos_close, _dos_read, _dos_write

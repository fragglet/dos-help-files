remove, unlink
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The remove and unlink functions delete the file specified by
     <path>. The unlink function is supported in the UNIX(R) and
     XENIX(R) operating systems. In the DOS and OS/2 environments, the
     two functions are identical.
 
     Return Value
 
     The functions return 0 if the file is successfully deleted.
     Otherwise, they return -1 and set errno to ENOENT.

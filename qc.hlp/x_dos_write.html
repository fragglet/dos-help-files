  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_write( int handle, void _far *buffer,
                                  unsigned count, unsigned *numwrt );
 
  Returns:   0 if successful, or the DOS error code if not.
             errno: EACCES, EBADF

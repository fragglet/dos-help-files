_fsopen
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdio.h>, <share.h>
 
  Syntax:    FILE *_fsopen( char *filename, char *mode, int shflag );
             mode:   r, w, a, r+, w+, a+, t, b
                     (appended to <mode> to indicate type)
             shflag: SH_COMPAT, SH_DENYNO, SH_DENYRD, SH_DENYRW,
                     SH_DENYWR
 
  Returns:   a pointer to the stream if successful, or NULL if not.
 
  See also:  fclose, fdopen, ferror, fileno, freopen, open, setmode

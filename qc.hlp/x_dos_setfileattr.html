_dos_setfileattr
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_setfileattr( char *pathname, unsigned attrib );
             attrib: _A_ARCH, _A_HIDDEN, _A_NORMAL, _A_RDONLY,
                     _A_SUBDIR, _A_SYSTEM, _A_VOLID
                     (may be joined by |)
 
  Returns:   0 if successful, or the DOS error code if not.
             errno: EACCES, ENOENT
 
  See also:  _dos_getfileattr

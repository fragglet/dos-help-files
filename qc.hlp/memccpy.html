memccpy, _fmemccpy
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>, <memory.h>
 
  Syntax:    void *memccpy( void *dest, void *src,
                            int c, unsigned count );
 
             void _far *_fmemccpy( void _far *dest, void _far *src,
                                   int c, unsigned count );
 
  Returns:   if successful, a pointer to the byte in <dest> following
             <c>; or NULL if not successful.
 
  See also:  memchr, memcmp, memcpy, memset

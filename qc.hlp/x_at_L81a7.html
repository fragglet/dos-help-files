_disable, _enable
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _disable routine disables interrupts by executing an 8086 CLI
     machine instruction. Use _disable before modifying an interrupt
     vector.
 
     The _enable routine enables interrupts by executing an 8086 STI
     machine instruction.
 
     Return Value
 
     None.

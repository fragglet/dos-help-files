putc, putchar
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The putc routine writes the single character <c> to the output
     stream at the current position. The putchar routine is identical
     to putc( <c>, stdout ). These routines are implemented as both
     macros and functions.
 
     Return Value
 
     The putc and putchar routines return the character written, or EOF
     in the case of an error. Any integer can be passed to putc, but
     only the lower eight bits are written.

Absolute Value Functions
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The abs, fabs, fabsl, and labs functions return the absolute value
     of their argument <x>. The functions and their return types are
     listed below:
 
     Function     Argument/Return Value
 
     abs          Integer value
     fabs         Floating-point value
     fabsl        Long double floating-point value
     labs         Long integer value
 
     Return Value
 
     The return values for these functions are described above. There
     is no error return.

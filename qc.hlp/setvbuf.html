────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int setvbuf( FILE *stream, char *buffer,
                          int mode, size_t size );
             mode: _IOFBF, _IOLBF, _IONBF
 
  Returns:   0 if successful, or a nonzero value if an illegal type
             or buffer size is specified.
 
  See also:  fclose, fflush, fopen, setbuf

ferror
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The ferror routine (implemented as a macro) tests for reading or
     writing errors on the file associated with <stream>. If an error
     has occurred, the error indicator for the stream remains set until
     the stream is closed or rewound, or until clearerr is called
     against it.
 
     Return Value
 
     If no error has occurred on <stream>, ferror returns 0. Otherwise,
     it returns a nonzero value.

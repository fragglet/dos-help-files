cprintf
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The cprintf function formats and prints a series of characters and
     values directly to the console, using the putch function to output
     characters.
 
     Each <argument> (if any) is converted and output according to the
     corresponding format specification in <format>. The format has the
     same form and function as the <format> argument for the printf
     function; see printf for a description of the format and
     arguments.
 
     Note that unlike the fprintf, printf, and sprintf functions,
     cprintf does not translate line-feed (LF) characters into
     carriage-return──line-feed (CR-LF) combinations on output.
 
     Return Value
 
     The cprintf function returns the number of characters printed.

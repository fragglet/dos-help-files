strncmp, _fstrncmp
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    int strncmp( char *string1, char *string2,
                          size_t count );
 
             int _fstrncmp( char _far *string1, char _far *string2,
                            size_t count );
 
  Returns:   a negative value if <string1> is less than <string2>,
             0 if <string1> is equal to <string2>, or a positive value
             if <string1> is greater than <string2>.
 
  See also:  strcat, strcpy, strncat, strncpy, strrchr, strset, strspn

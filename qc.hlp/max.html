────────────────────────────────────────────────────────────────────────────
 
  Include:  <stdlib.h>
 
  Syntax:   type max( type a, type b );
            type min( type a, type b );
 
  Returns:  (max) the larger of the two arguments.
            (min) the smaller of the two arguments.

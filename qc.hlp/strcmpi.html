stricmp, _fstricmp, strcmpi
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    int stricmp( char *string1, char *string2 );
             int _fstricmp( char _far *string1, char _far *string2 );
             int strcmpi( char *string1, char *string2 );
 
  Returns:   a negative value if <string1> is less than <string2>,
             0 if <string1> is equal to <string2>, or a positive value
             if <string1> is greater than <string2>.
 
  See also:  memcmp, memicmp, strcat, strcpy, strncat, strncmp,
             strncpy, strnicmp, strrchr, strset, strspn

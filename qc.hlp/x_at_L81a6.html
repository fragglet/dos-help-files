────────────────────────────────────────────────────────────────────────────
 
     The difftime function computes the difference between the supplied
     time values <timer0> and <timer1>.
 
     Return Value
 
     The difftime function returns the elapsed time, in seconds, from
     <timer0> to <timer1>. The value is returned as a double-precision
     number.

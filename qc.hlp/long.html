  Keyword:   long
 
  Syntax:    long declarators
 
  Summary:   Fundamental type used to store a 32-bit integer value. Can
             also precede double as a modifier specifying a fundamental
             type for storing 10-byte floating-point values.
 
  See also:  char, double, float, int, short

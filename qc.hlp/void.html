  Keyword:  void
 
  Syntax:   void declarator
 
  Summary:  If used as the return type of a function, the function does
            not return a value. If used as the parameter list of a
            function, the function takes no parameters. If used as a
            pointer type, the specified pointer can match a pointer of
            any type.

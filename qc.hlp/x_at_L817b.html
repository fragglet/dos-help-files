     The access function, when used with files, determines whether or
     not a specified file exists and can be accessed in <mode>. The
     possible <mode> values and their meanings in the access call are
     as follows:
 
     Value     Meaning
 
     00        Check for existence only
     02        Check for write permission
     04        Check for read permission
     06        Check for read and write permission
 
     With directories, the access function determines only whether the
     specified directory exists; under MS-DOS(R) and OS/2, all
     directories have read and write access.
 
     Return Value
 
     The access function returns the value 0 if the file has the given
     mode. A return value of -1 indicates that the named file does not
     exist or is not accessible in the given mode, and errno is set to
     either EACCES or ENOENT.

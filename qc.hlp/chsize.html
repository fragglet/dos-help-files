  Include:   <io.h>, <errno.h>
 
  Syntax:    int chsize( int handle, long size );
 
  Returns:   0 if the size change is successful, or -1 if not.
             errno: EACCES, EBADF, ENOSPC
 
  See also:  close, creat, open

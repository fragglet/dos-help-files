  Include:   <math.h>, <errno.h>
 
  Syntax:    double ldexp( double x, int exp );
             long double ldexpl( long double x, int exp );
 
  Returns:   an exponential value if successful, or ±HUGE_VAL on
             overflow.
             errno: ERANGE
 
  See also:  frexp, modf

scanf
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int scanf( char *format [, argument]... );
 
  Returns:   the number of fields converted and assigned if successful
             (0 if no fields were assigned), or EOF for an attempt to
             read end-of-file.
 
  See also:  fscanf, printf, sscanf, vfprintf, vprintf, vsprintf

dup, dup2
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    int dup( int handle );
             int dup2( int handle1, int handle2 );
 
  Returns:   (dup) a new handle if successful, or -1 if not.
             (dup2) 0 if successful, or -1 if not.
             errno: EBADF, EMFILE
 
  See also:  close, creat, open

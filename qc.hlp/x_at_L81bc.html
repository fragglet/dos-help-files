_dos_setftime
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _dos_setftime routine uses system call 0x57 to set the date
     and time at which the file identified by <handle> was last written
     to. Those values appear in the DOS date and time format.
 
     See: ◄DOS Time/Date Format►
 
     Return Value
 
     If successful, the function returns 0. Otherwise, it returns the
     DOS error code and sets errno to EBADF, indicating that an invalid
     file handle was passed.

fdopen
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    FILE *fdopen( int handle, char *mode );
             mode: r, w, a, r+, w+, a+, t, b
                   (appended to <mode> to indicate type)
 
  Returns:   a pointer to the open stream if successful, or NULL
             if not.
 
  See also:  dup, fclose, fopen, freopen, open

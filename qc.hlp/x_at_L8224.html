printf Flags: Leading Blanks
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Flag            Meaning
 
     blank (' ')     Precede the output value with a blank if the
                     output value is signed and positive. The blank is
                     ignored if both the blank and + flags appear. The
                     default is no blanks.

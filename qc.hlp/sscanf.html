  Include:   <stdio.h>
 
  Syntax:    int sscanf( char *buffer, char *format [, argument] ... );
 
  Returns:   the number of fields converted and assigned if successful
             (0 if no fields were assigned), or EOF for an attempt
             to read at end-of-string.
 
  See also:  fscanf, scanf, sprintf

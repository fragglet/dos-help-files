     CVPACK Error CK1001
 
     out of memory
 
     The executable file is too big for the available memory. This
     error can occur under DOS when there is little extra memory. Even
     though CVPACK uses virtual memory, which involves swapping to
     disk, some information can be stored only in real memory.
 
     One of the following may be a solution:
 
        ■ Recompile one or more of the object files without debugging
          information. If the file was compiled using the /Zi option,
          use either /Zd or no option.
 
        ■ Add more memory to your computer.

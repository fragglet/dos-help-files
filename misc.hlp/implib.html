─────IMPLIB─────────────────────────────────────────────────────────────────
 
     IMPLIB creates import libraries that can be used by LINK to link
     dynamic-link libraries (DLLs) with applications.
 
     Command-Line Syntax
 
     IMPLIB [options] implibname {dllfile... | deffile...}
 
     <options>      See: ◄IMPLIB Options►
 
     <implibname>   Import library to be created
 
     <deffile>      One or more module-definition (.DEF) files used to
                    create a DLL to be linked
 
     <dllfile>      One or more DLLs to be linked
 
     You can specify any number of either module-definition files or
     DLLs. IMPLIB assumes no default extensions for filenames. You can
     specify a path with a filename.
 
     Example
 
          IMPLIB MYLIB.LIB MYLIB.DLL
 
     This command creates the import library named MYLIB.LIB from the
     dynamic-link library MYLIB.DLL.

     CVPACK Error CK1013
 
     debugging information missing or unknown format
 
     One of the following has occurred:
 
        ■ The program did not contain debugging information. Recompile
          using /Zi or /Zd, then link using /CO.
 
        ■ The executable file was linked using an obsolete or
          unsupported linker. Use Microsoft LINK version 5.3x or later.
 
        ■ The executable file was already packed using a previous
          version of CVPACK.

─────SBRPACK────────────────────────────────────────────────────────────────
 
     Options names are not case sensitive. Only the /NOLOGO option
     applies to a packing session; the other options provide help and
     then halt SBRPACK.
 
     /HELP
          Calls the QuickHelp utility. If SBRPACK cannot find the Help
          file or QuickHelp, it displays a brief summary of SBRPACK
          command-line syntax.
 
     /NOLOGO
          Suppresses the SBRPACK copyright message.
 
     /?
          Displays a brief summary of SBRPACK command-line syntax.

     CVPACK Warning CK4003
 
     error in lexical scopes for module 'module', symbols deleted
 
     The scoping of symbols in the given object module was corrupted.
     CVPACK deleted the symbols in the module.
 
     This is probably a compiler error. Recompile and relink the object
     file.

/Es Option
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────BSCMAKE────────────────────────────────────────────────────────────────
 
     Syntax:  /Es
 
     Excludes from the database every include file specified with an
     absolute path or found in an absolute path specified in the
     INCLUDE environment variable. /Es does not exclude include files
     specified without a path or with relative paths or found in a
     relative path in INCLUDE.
 
     You can use the /Ei option along with /Es to exclude other files.
     If you want to exclude only some of the files that /Es excludes,
     use /Ei instead of /Es and list the files you want to exclude.
     See: ◄/Ei Option►

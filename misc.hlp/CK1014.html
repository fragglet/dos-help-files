     CVPACK Error CK1014
 
     module 'module' type 'number' refers to skipped type index;
     contact Microsoft Product Support Services
 
     The debugging information in the executable file is corrupt. This
     is due to an internal error in the compiler. Recompile the
     program. If the problem persists, note the circumstances of the
     error and notify Microsoft Corporation by following the
     instructions in the Microsoft Product Assistance Request form at
     the back of one of your manuals.

─────BSCMAKE────────────────────────────────────────────────────────────────
 
     Syntax:  /Iu
 
     Includes unreferenced symbols. By default, BSCMAKE does not
     record any symbols that are defined but not referenced. If an .SBR
     file has been processed by SBRPACK, this option has no effect for
     that file because SBRPACK has already removed the symbols.

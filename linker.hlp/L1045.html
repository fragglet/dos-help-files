     LINK Fatal Error L1045
 
     too many TYPDEF records
 
     An object file contained more than 255 TYPDEF records.
 
     TYPDEF records describe communal variables. (TYPDEF is a DOS term.
     It is explained in the "Microsoft MS-DOS Programmer's Reference"
     and in other reference books on DOS.)
 
     This error appears only with programs created by the Microsoft
     FORTRAN Compiler or other compilers that support communal
     variables.

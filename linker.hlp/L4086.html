     LINK Warning L4086
 
     /r not first on command line; ignored
 
     The /r option was specified, but it was not the first argument
     after LINK on the command line.
 
     The /r option prevents LINK from using extended memory under DOS
     and forces LINK to use only conventional memory. This option must
     appear first on the command line. It cannot appear in a response
     file or the LINK environment variable.

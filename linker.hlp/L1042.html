     LINK Fatal Error L1042
 
     nonresident names table overflow
 
     The size of the nonresident names table exceeded 65,535 bytes.
 
     An entry in the nonresident names table is made for each exported
     routine not designated RESIDENTNAME and consists of the name plus
     three bytes of information. The first entry is the DESCRIPTION
     statement.
 
     Reduce the number of exported routines or change some to resident
     status.

     LINK Fatal Error L1054
 
     requested segment limit too high
 
     LINK did not have enough memory to allocate tables describing the
     requested number of segments. The number of segments is the value
     specified with the /SEG option or the default of 128.
 
     One of the following may be a solution:
 
        ■ Link again using the /SEG option to set fewer segments.
 
        ■ Remove terminate-and-stay-resident programs or otherwise free
          some memory.

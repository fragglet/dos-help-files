─────LINK───────────────────────────────────────────────────────────────────
 
     Syntax:  REALMODE
 
     The REALMODE statement specifies that the application runs under
     Windows  in either real mode or protected mode. By default,
     EXETYPE WINDOWS assumes PROTMODE.
     See: ◄EXETYPE Statement►
          ◄REALMODE Statement►
 
     Example
 
          EXETYPE WINDOWS 3.0
          REALMODE
 
     This statement combination defines an application that runs under
     Windows 3.0 or higher in any mode.

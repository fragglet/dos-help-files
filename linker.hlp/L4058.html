     LINK Warning L4058
 
     ignoring alias for already defined symbol 'symbol'
 
     The specified symbol was redefined in the program. However, it is
     an identifier from a C run-time library that has an alias to a new
     name in OLDNAMES.LIB. LINK ignored the alias for the symbol.
 
     This warning appears only when the /INFO option is specified.

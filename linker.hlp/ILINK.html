     LINK version 5.30 does not support ILINK and incremental linking.
     To allow existing makefiles to remain compatible, a file called
     ILINK.EXE is provided with the Microsoft C/C++ Windows Development
     System version 7.0. This form of ILINK always invokes a full link.

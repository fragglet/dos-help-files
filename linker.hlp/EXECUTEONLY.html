     Attribute:  <executeonly>
 
     Used by:    CODE, SEGMENTS
 
     Determines whether a code segment can be read as well as executed.
 
     Keyword         Purpose
 
     EXECUTEONLY     Specifies that the segment can only be executed.
                     Alternate spelling is EXECUTE-ONLY.
 
     EXECUTEREAD     Default. Specifies that the segment is both
                     executable and readable. Required for CodeView.

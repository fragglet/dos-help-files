     EXEHDR Error U1102
 
     automatic data segment greater than 64K;
     correcting stack size
 
     There was not enough space in the automatic, or default, data
     segment (DGROUP) to accommodate the requested new stack size.
     EXEHDR adjusted the stack size to the maximum available space.
 
     This error applies only to segmented executable files.

─────LINK───────────────────────────────────────────────────────────────────
 
     Syntax:  /NOG[ROUPASSOCIATION]
 
     The /NOGROUP option tells LINK to ignore group associations
     when assigning addresses to data and code items. It is provided
     primarily for compatibility with previous versions of LINK and of
     Microsoft compilers. /NOGROUP is valid only for assembly-language
     programs that create MS-DOS programs.

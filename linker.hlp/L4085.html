     LINK Warning L4085
 
     'option' : argument invalid; assuming 'argument'
 
     The given option was specified with a numeric argument that was
     out of range for the option. LINK assumed the given argument.
 
     For example, the option specification /DYNAMIC:11000 causes the
     following error:
 
          /DYNAMIC:11000 : argument invalid; assuming 10922

     LINK Warning L4001
 
     frame-relative fixup, frame ignored near 'offset' in segment 'segment'
 
     A reference was made relative to a segment or group that is
     different from the target segment of the reference.
 
     For example, if _id1 is defined in segment _TEXT, the instruction
     call DGROUP:_id1 produces this warning. The frame DGROUP is
     ignored, so LINK treats the call as if it were call _TEXT:_id1.

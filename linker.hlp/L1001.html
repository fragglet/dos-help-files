     LINK Fatal Error L1001
 
     'option' : option name ambiguous
 
     A unique option name did not appear after the option indicator.
 
     An option is specified by a forward slash (/) and a name. The
     name can be specified by an abbreviation of the full name, but
     the abbreviation must be unambiguous.
 
     For example, since many options begin with the letter N, the
     following command causes this error:
 
          LINK /N main;
 
     This error can also occur if the wrong version of the linker is
     being used. Check the directories in the PATH environment variable
     for other versions of LINK.EXE.

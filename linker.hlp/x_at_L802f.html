─────LINK───────────────────────────────────────────────────────────────────
 
     Syntax:  /OL[DOVERLAY]
 
     The /OLDOVERLAY option links an overlaid DOS program using the
     Static Overlay Manager instead of the MOVE library. This option
     may not be supported in future versions of LINK.

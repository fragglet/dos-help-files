/BATCH Option
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────LINK───────────────────────────────────────────────────────────────────
 
     Syntax:  /B[ATCH]
 
     The /BATCH option suppresses prompting for libraries or object
     files LINK cannot find. By default, LINK prompts for a new path
     whenever it cannot find a file. When /BATCH is used, LINK
     generates an error message and leaves external references
     unresolved. /BATCH also suppresses the LINK copyright message and
     the echoed input from response files. This option is primarily
     used when LINK is called from a batch file or makefile.
 
     NOTE: This option does not suppress prompts for input fields.

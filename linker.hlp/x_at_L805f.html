     Command-Line Syntax
 
     EXEHDR [options] filenames
 
     <options>     Options used to modify output or change the header.
                   See: ◄EXEHDR Options►
 
     <filenames>   One or more applications or DLLs. If no extension is
                   specified, EXEHDR assumes .EXE. You can specify a
                   path with the filename.

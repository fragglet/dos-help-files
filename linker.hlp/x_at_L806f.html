Output for a DLL
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────EXEHDR─────────────────────────────────────────────────────────────────
 
     The first part of the EXEHDR output for a DLL appears as follows:
 
          ◄Library:►
          ◄Description:►
          ◄Data:►
          ◄Initialization:►
          ◄Initial CS:IP:►
          ◄Initial SS:SP:►
          ◄DGROUP:►
 
     The rest of the output for a DLL is similar to the output for
     other segmented executable files.
     See: ◄Output for a Segmented Executable File►

     LINK Fatal Error L1070
 
     'segment' : segment size exceeds 64K
 
     A single segment contained more than 65,536 bytes of code or data.
 
     Try changing the memory model to use far code or data as
     appropriate. If the program is in C, use CL's /NT option or the
     __based keyword (or its predecessor, the alloc_text pragma) to
     build smaller segments.

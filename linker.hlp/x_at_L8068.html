─────EXEHDR─────────────────────────────────────────────────────────────────
 
     Syntax:  /R[ESETERROR]
 
     Clears the error bit in the header of a segmented executable file.
 
     LINK sets the error bit if certain errors occur during linking.
     The operating system does not load a program if this bit is set.
     If the error bit is set, EXEHDR displays "Error in image".
 
     This option allows you to run a program that contains LINK errors
     and is useful during application development.

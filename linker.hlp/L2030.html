     LINK Error L2030
 
     starting address not code (use class 'CODE')
 
     The program starting address, as specified in the END statement of
     an .ASM file, should be in a code segment. Code segments are
     recognized if their class name ends in 'CODE'. This is an error
     in a segmented executable file.
 
     The error message can be disabled by including the REALMODE
     statement in the module-definition (.DEF) file.

─────EXEHDR─────────────────────────────────────────────────────────────────
 
     Syntax:  /MA[X]:number
 
     Sets the maximum memory allocation to <number> 16-byte paragraphs.
 
     The memory allocation fields tell DOS the minimum and maximum
     memory needed to load and run the program. The <number> must equal
     or exceed the minimum allocation.
     See: ◄/MIN Option►
 
     Specify <number> in decimal format or C-language notation.
     See: ◄Entering Numeric Arguments►
 
     This option is equivalent to LINK's /CPARM option and applies only
     to DOS programs.
     See: ◄LINK /CPARM Option►

     EXEHDR Error U1106
 
     minimum allocation greater than maximum;
     correcting maximum
 
     If the minimum allocation is greater than the maximum allocation,
     the maximum allocation value is adjusted.
 
     If a display of DOS header values is requested, the values shown
     will be the values after the packed file is expanded.
 
     This error applies only to DOS programs.

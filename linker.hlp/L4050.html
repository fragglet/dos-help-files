     LINK Warning L4050
 
     file not suitable for /EXEPACK; relink without
 
     The size of the packed load image plus packing overhead was
     larger than it would be for the unpacked load image. There is
     no advantage to packing this program.
 
     Remove /EXEPACK from the LINK command line. In PWB, turn off the
     Pack EXE File check box in the Additional Debug/Release Options
     dialog box under Link Options.

     Syntax:  PROTMODE
 
     The PROTMODE statement specifies that the application or DLL runs
     only under Windows in protected mode (either standard mode or 386
     enhanced mode). PROTMODE is assumed by EXETYPE WINDOWS. To define
     a program that runs under any Windows mode, specify REALMODE.

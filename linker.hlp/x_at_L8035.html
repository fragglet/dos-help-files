─────LINK───────────────────────────────────────────────────────────────────
 
     Syntax:  /PAU[SE]
 
     The /PAUSE option pauses the session before LINK writes the main
     output file to disk. /PAUSE is supplied for use with machines that
     have floppy drives but no hard disk.

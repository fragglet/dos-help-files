     LINK Fatal Error L1007
 
     /OVERLAYINTERRUPT : interrupt number exceeds 255
 
     An overlay interrupt number greater than 255 was specified with
     the /OV option value.
 
     Check the "DOS Technical Reference" or other DOS technical manual
     for information about interrupts.

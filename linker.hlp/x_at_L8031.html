     Syntax:  /O[VERLAYINTERRUPT]:number
 
     The /OV option sets an interrupt number for passing control to
     overlays. Specify a number in the range 0-255. The default is 63.
     See: ◄Entering Numeric Arguments►
 
     This option is valid only when linking DOS programs.
 
     Use /OV if necessary to avoid a conflict with another program
     that uses interrupt 63. Numbers that conflict with DOS interrupts
     can be used; however, their use is not advised.
 
     See: ◄Creating Overlaid DOS Programs►
 
     NOTE: The minimum unique abbreviation for this option is /OV.

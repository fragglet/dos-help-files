     LINK Reserved Words in Module-Definition Files
 
     The following words are reserved by LINK. A reserved word can be
     used as an argument in module-definition statements only if the
     word is enclosed in double quotation marks (").
 
     ALIAS           FIXED           NEWFILES           PURE
     APPLOADER       FUNCTIONS       NODATA             READONLY
     BASE            HEAPSIZE        NOEXPANDDOWN       READWRITE
     CLASS           HUGE            NOIOPL             REALMODE
     CODE            IMPORTS         NONAME             RESIDENT
     CONFORMING      IMPURE          NONCONFORMING      RESIDENTNAME
     CONSTANT        INCLUDE         NONDISCARDABLE     SECTIONS
     CONTIGUOUS      INITGLOBAL      NONE               SEGMENTS
     DATA            INITINSTANCE    NONPERMANENT       SHARED
     DESCRIPTION     INVALID         NONSHARED          SINGLE
     DEV386          IOPL            NOTWINDOWCOMPAT    STACKSIZE
     DEVICE          LIBRARY         NULL               STUB
     DISCARDABLE     LOADONCALL      OBJECTS            SUBSYSTEM
     DOS             LONGNAMES       OLD                SWAPPABLE
     DYNAMIC         MACINTOSH       OVERLAY            TERMINSTANCE
     EXECUTE-ONLY    MAXVAL          OVL                UNKNOWN
     EXECUTEONLY     MIXED1632       PERMANENT          VERSION
     EXECUTEREAD     MOVABLE         PHYSICAL           VIRTUAL
     EXETYPE         MOVEABLE        PRELOAD            WINDOWAPI
     EXPANDDOWN      MULTIPLE        PRIVATELIB         WINDOWCOMPAT
     EXPORTS         NAME            PROTMODE           WINDOWS

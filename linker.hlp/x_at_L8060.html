     Option names are not case sensitive. They can be shortened to any
     unique abbreviation; brackets mark the optional part. This help
     file uses meaningful yet legal forms of the option names. Specify
     number arguments in decimal format or C-language notation.
     See: ◄Entering Numeric Arguments►
 
     Option      Purpose
 
     /HEAP       ◄Set heap allocation►
     /HELP       ◄Display help►
     /MAX        ◄Set maximum allocation►
     /MIN        ◄Set minimum allocation►
     /NEW        ◄Use installable file system►
     /NOLOGO     ◄Suppress copyright message►
     /PM         ◄Set application type►
     /RESET      ◄Clear error bit►
     /STACK      ◄Set stack allocation►
     /V          ◄Display verbose output►
     /?          ◄Display short help►

     Syntax:  /MI[N]:number
 
     Sets the minimum memory allocation to <number> 16-byte paragraphs.
 
     The memory allocation fields tell DOS the minimum and maximum
     memory needed to load and run the program. The /MAX option sets
     the maximum memory allocation.
     See: ◄/MAX Option►
 
     Specify <number> in decimal format or C-language notation.
     See: ◄Entering Numeric Arguments►

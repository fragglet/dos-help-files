     EXEHDR Error U1107
 
     unexpected end of resident/nonresident name table
 
     While decoding run-time relocation records, EXEHDR found the end
     of either the resident names table or the nonresident names table.
     The executable file is probably corrupted.
 
     This error applies only to segmented executable files.

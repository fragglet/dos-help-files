─────EXEHDR─────────────────────────────────────────────────────────────────
 
     Describe tables in the segmented executable file. Each description
     gives the table name, its address within the file, and its length
     in hexadecimal and in decimal. The following tables are described:
 
          Segment Table:
          Resource Table:
          Resident Names Table:
          Module Reference Table:
          Imported Names Table:
          Entry Table:
          Non-resident Names Table:
 
     Displayed in verbose output for segmented executable files.

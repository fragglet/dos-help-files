     Syntax:  /HI[GH]
 
     The /HIGH option causes DOS to place the executable file as high
     in memory as possible. By default, programs are loaded as low as
     possible.
 
     /HIGH is usually used with the /DSALLOC option. These options are
     valid only for assembly-language programs that create MS-DOS .EXE
     files.

MCI_INFO_PARMS
 
        The MCI_INFO_PARMS structure contains parameters for the
        MCI_INFO message. When assigning data to the fields in this
        data structure, set the corresponding MCI flags in the dwFlags
        parameter of mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            LPSTR lpstrReturn;
            DWORD dwRetSize;
        } MCI_INFO_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        LPSTR lpstrReturn
            Specifies a long pointer to a user-supplied buffer for the
            return string.
 
        DWORD dwRetSize
            Specifies the size in bytes of the buffer for the return
            string.

MCI_VD_ESCAPE_PARMS
 
        The MCI_VD_ESCAPE_PARMS structure contains parameters for the
        MCI_ESCAPE message for videodisc devices. When assigning data
        to the fields in this data structure, set the corresponding MCI
        flags in the dwFlags parameter of mciSendCommand to validate
        the fields.
 
        typedef struct {
            DWORD dwCallback;
            LPCSTR lpstrCommand;
        } MCI_VD_ESCAPE_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        LPCSTR lpstrCommand
            Specifies a far pointer to a null-terminated buffer
            containing the command to send to the device.

MCI_VD_PLAY_PARMS
 
        The MCI_VD_PLAY_PARMS structure contains parameters for the
        MCI_PLAY message for videodiscs. When assigning data to the
        fields in this data structure, set the corresponding MCI flags
        in the dwFlags parameter of mciSendCommand to validate the
        fields. You can use the MCI_PLAY_PARMS data structure in place
        of MCI_VD_PLAY_PARMS if you are not using the extended data
        fields.
 
        typedef struct {
            DWORD dwCallback;
            DWORD dwFrom;
            DWORD dwTo;
            DWORD dwSpeed;
        } MCI_VD_PLAY_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        DWORD dwFrom
            Specifies the position to play from.
 
        DWORD dwTo
            Specifies the position to play to.
 
        DWORD dwSpeed
            Specifies the playing speed in frames per second.

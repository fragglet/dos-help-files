MODM_CLOSE
 
        This message is sent to a MIDI output device driver to
        deallocate a specified device.
 
Parameters
        DWORD dwParam1
            Unused.
 
        DWORD dwParam2
            Unused.
 
Return value
        The return value is an error code, or zero (MMSYSERR_NOERROR) if
        the operation is successful. Possible error codes are:
 
        MMSYSERR_NOTENABLED
            The driver failed to load or initialize.
 
        MIDIERR_STILLPLAYING
            There are still buffers in the queue.
 
Comments
        If there are output buffers that have been received with the
        MODM_LONGDATA message and haven't been returned to the
        application, the driver should fail the close operation by
        returning a MIDIERR_STILLPLAYING error. If the close operation is
        successful, the driver should use the DriverCallback function to
        send the client a MOM_CLOSE message.

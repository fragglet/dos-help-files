MCI_GENERIC_PARMS
 
        The MCI_GENERIC_PARMS structure contains the information for
        MCI command messages that have empty parameter lists. When
        assigning data to the fields in this data structure, set the
        corresponding MCI flags in the dwFlags parameter of
        mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
        } MCI_GENERIC_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.

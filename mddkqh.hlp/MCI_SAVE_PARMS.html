MCI_SAVE_PARMS
 
        The MCI_SAVE_PARMS structure contains the information for
        MCI_SAVE message. When assigning data to the fields in this
        data structure, set the corresponding MCI flags in the dwFlags
        parameter of mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            LPCSTR lpfilename;
        } MCI_SAVE_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        LPCSTR lpfilename
            Specifies a far pointer to the buffer containing a
            null-terminated string.

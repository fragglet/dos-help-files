MCI_ANIM_UPDATE_PARMS
 
        The MCI_ANIM_UPDATE_PARMS structure contains parameters for the
        MCI_UPDATE message for animation devices. When assigning data
        to the fields in this data structure, set the corresponding MCI
        flags in the dwFlags parameter of mciSendCommand to validate
        the fields.
 
        typedef struct {
            DWORD dwCallback;
            RECT rc;
            HDC hDC;
        } MCI_ANIM_UPDATE_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        RECT rc
            Specifies a window rectangle.
 
        HDC hDC
            Specifies a handle to the device context.

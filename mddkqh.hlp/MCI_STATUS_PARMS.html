MCI_STATUS_PARMS
 
        The MCI_STATUS_PARMS structure contains parameters for the
        MCI_STATUS message. When assigning data to the fields in this
        data structure, set the corresponding MCI flags in the dwFlags
        parameter of mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            DWORD dwReturn;
            DWORD dwItem;
            DWORD dwTrack;
        } MCI_STATUS_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        DWORD dwReturn
            Contains the return information on exit.
 
        DWORD dwItem
            Identifies the capability being queried.
 
        DWORD dwTrack
            Specifies the length or number of tracks.

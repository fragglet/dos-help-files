WIDM_GETNUMDEVS
 
        This message is sent to a waveform input device driver to
        request the number of waveform input devices available.
 
Parameters
        DWORD dwParam1
            Unused.
 
        DWORD dwParam2
            Unused.
 
Return value
        Returns the number of input devices the driver supports.

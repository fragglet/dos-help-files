mciFreeCommandResource
 
        BOOL WINAPI mciFreeCommandResource(uTable)
 
        Frees the memory used by the specified command table.
 
Parameters
        UINT uTable
            Specifies the table index returned from a previous call to
            mciLoadCommandResource.
 
Return value
        Returns FALSE if the table index is not valid, TRUE otherwise.

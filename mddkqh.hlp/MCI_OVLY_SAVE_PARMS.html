MCI_OVLY_SAVE_PARMS
 
        The MCI_OVLY_SAVE_PARMS structure contains the information
        for the MCI_SAVE message for video overlay devices. When
        assigning data to the fields in this data structure, set the
        corresponding MCI flags in the dwFlags parameter of
        mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            LPCSTR lpfilename;
            RECT rc;
        } MCI_OVLY_SAVE_PARMS;
 
Parameters
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        LPCSTR lpfilename
            Specifies a far pointer to the buffer containing a null-
            terminated string.
 
        RECT rc
            Specifies a valid display rectangle identifying the area of
            the video buffer to save.

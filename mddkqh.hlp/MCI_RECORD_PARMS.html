MCI_RECORD_PARMS
 
        The MCI_RECORD_PARMS structure contains parameters for the
        MCI_RECORD message. When assigning data to the fields in this
        data structure, set the corresponding MCI flags in the dwFlags
        parameter of mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            DWORD dwFrom;
            DWORD dwTo;
        } MCI_RECORD_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        DWORD dwFrom
            Specifies the position to play from.
 
        DWORD dwTo
            Specifies the position to play to.

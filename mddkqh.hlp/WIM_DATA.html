WIM_DATA
 
        This message is sent by a waveform input driver to a client
        application to return an input buffer to the application.
 
Parameters
        DWORD dwParam1
            Specifies a far pointer to a WAVEHDR structure identifying
            the buffer.
 
        DWORD dwParam2
            Not used.
 
Return value
        There is no return value.

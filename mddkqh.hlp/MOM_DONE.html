MOM_DONE
 
        This message is sent by a MIDI output driver to a client
        application to return a system-exclusive data block to the
        application.
 
Parameters
        DWORD dwParam1
            Specifies a far pointer to a MIDIHDR structure identifying
            the data block.
 
        DWORD dwParam2
            Not used.
 
Return value
        There is no return value.

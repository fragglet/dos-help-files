WOM_CLOSE
 
        This message is sent by a waveform output driver to a client
        application when a waveform output device is closed.
 
Parameters
        DWORD dwParam1
            Not used.
 
        DWORD dwParam2
            Not used.
 
Return value
        There is no return value.

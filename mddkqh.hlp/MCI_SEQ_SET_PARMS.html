MCI_SEQ_SET_PARMS
 
        The MCI_SEQ_SET_PARMS structure contains parameters for the
        MCI_SET message for MIDI sequencer devices. When assigning data
        to the fields in this data structure, set the corresponding MCI
        flags in the dwFlags parameter of mciSendCommand to validate
        the fields.
 
        typedef struct {
            DWORD dwCallback;
            DWORD dwTimeFormat;
            DWORD dwAudio;
            DWORD dwTempo;
            DWORD dwPort;
            DWORD dwSlave;
            DWORD dwMaster;
            DWORD dwOffset;
        } MCI_SEQ_SET_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        DWORD dwTimeFormat
            Specifies the time format of the sequencer.
 
        DWORD dwAudio
            Specifies the audio output channel.
 
        DWORD dwTempo
            Specifies the tempo.
 
        DWORD dwPort
            Specifies the output port.
 
        DWORD dwSlave
            Specifies the type of synchronization used by the sequencer
            for slave operation.
 
        DWORD dwMaster
            Specifies the type of synchronization used by the sequencer
            for master operation.
 
        DWORD dwOffset
            Specifies the data offset.

WODM_RESTART
 
        This message is sent to a waveform output device driver to
        continue playback of a waveform after playback has been paused
        with the WODM_PAUSE message.
 
Parameters
        DWORD dwParam1
            Unused.
 
        DWORD dwParam2
            Unused.
 
Return value
        The return value is an error code, or zero (MMSYSERR_NOERROR) if
        the operation is successful. Possible error codes are:
 
        MMSYSERR_NOTENABLED
            The driver failed to load or initialize.
 
Comments
        If the output is not paused, this message should have no effect,
        and the driver should return MMSYSERR_NOERROR.

MCI_SYSINFO_PARMS
 
        The MCI_SYSINFO_PARMS structure contains parameters for the
        MCI_SYSINFO message. When assigning data to the fields in this
        data structure, set the corresponding MCI flags in the dwFlags
        parameter of mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            LPSTR lpstrReturn;
            DWORD dwRetSize;
            DWORD dwNumber;
            UINT wDeviceType;
            UINT wReserved0;
        } MCI_SYSINFO_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        LPSTR lpstrReturn
            Specifies a long pointer to a user-supplied buffer for the
            return string. It is also used to return a DWORD when the
            MCI_SYSINFO_QUANTITY flag is used.
 
        DWORD dwRetSize
            Specifies the size in bytes of the buffer for the return
            string.
 
        DWORD dwNumber
            Specifies a number indicating the device position in the
            MCI device table or in the list of open devices if the
            MCI_SYSINFO_OPEN flag is set.
 
        UINT wDeviceType
            Specifies the type of device.
 
        UINT wReserved0
            Reserved.

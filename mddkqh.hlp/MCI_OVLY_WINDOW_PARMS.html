MCI_OVLY_WINDOW_PARMS
 
        The MCI_OVLY_WINDOW_PARMS structure contains parameters for the
        MCI_WINDOW message for video overlay devices. When assigning
        data to the fields in this data structure, set the
        corresponding MCI flags in the dwFlags parameter of
        mciSendCommand to validate the fields.
 
        typedef struct {
            DWORD dwCallback;
            HWND hWnd;
            UINT wReserved1;
            UINT nCmdShow;
            UINT wReserved2;
            LPCSTR lpstrText;
        } MCI_OVLY_WINDOW_PARMS;
 
Fields
        DWORD dwCallback
            The low-order word specifies a window handle used for the
            MCI_NOTIFY flag.
 
        HWND hWnd
            Specifies a handle to the display window.
 
        UINT wReserved1
            Reserved.
 
        UINT nCmdShow
            Specifies how the window is displayed.
 
        UINT wReserved2
            Reserved.
 
        LPCSTR lpstrText
            Specifies a long pointer to a null-terminated buffer
            containing the window caption.

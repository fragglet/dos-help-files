q.mh1 - Help for Microsoft QuickHelp
Print (=)  Use this command to copy all or part of a topic to LPT1. When
you choose the Print command, the Paste menu is replaced with the Print
menu, which allows you to choose which portion of the topic to print. If
you are viewing a list of topics when you choose the Print command, one
of the Print-menu commands is All Topics. If you choose the All Topics
command, every topic in the list will be copied to LPT1.

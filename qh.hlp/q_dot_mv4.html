q.mv4
Hide Window (W)  Use this command to temporarily hide the QuickHelp
window, restoring the screen to its state prior to starting or activating
QuickHelp. You can then use a mouse or the cursor to choose a topic from
this screen. If you use a mouse, click the desired topic. If you use the
cursor, use the ARROW keys to position the cursor over the topic
name and then press ENTER.

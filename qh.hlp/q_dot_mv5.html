q.mv5 - Help for Microsoft QuickHelp
View History (I)  Use this command to display a list of the last 23
topics viewed in the order you viewed them. You can return to one of
these topics by clicking it with a mouse or by using the ARROW keys
to move the highlighted line over the topic name and then pressing
ENTER.
 
Topics that QuickHelp creates, such as the Global Search topic, will
be deleted if QuickHelp runs out of memory trying to display a topic
(DOS only).

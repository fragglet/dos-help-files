.popup
 
The .popup command specifies that the topic should be displayed in a
temporary window. QuickHelp determines the size of window to use and
displays the text in that window. As soon as the user presses a key or
clicks the mouse, the window is removed.
 
The limit for a .popup topic is 21 lines of text, with each line not
exceeding 76 characters.
 
Example
 
The following is an example of a popup topic:
 
    .context ULONG
    .popup
    \bULONG\p  an unsigned 32 bit value.

                            Microsoft QuickHelp
 
For additional information, double click on any of the following topics, or
press the TAB key until the desired topic is highlighted, and press ENTER.
 
Search commands
 
    ◄ S - Searches for a topic ►
    ◄ E - Continues a search for a topic ►
    ◄ D - Searches for duplicate topics ►
    ◄ A - Searches for text in current topic ►
    ◄ G - Searches for text in all topics ►
 
Navigation commands
 
    ◄ B - Views previous topic in database ►
    ◄ N - Views next topic in database ►
    ◄ L - Views last screen ►
    ◄ M - Moves to next list topic ►
 
Special topic commands
 
    ◄ I - Views history of topics viewed ►
    ◄ T - Displays table of contents ►
    ◄ U - Capture screen underneath QuickHelp ►
    ◄ W - Hides QuickHelp window ►
    ◄ = - Prints current topic ►
 
Exit command
 
    ◄ X - Exits QuickHelp ►

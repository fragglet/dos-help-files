q.mv13
Capture Screen (U)  Use this command to capture the screen in its state
prior to running or activating QuickHelp and save it in QuickHelp as a
topic. You can then use a mouse to select a topic from this screen. You
can also paste or print this topic just as you would any other topic.
Note that the rightmost 4 characters will not be displayed within the
QuickHelp window, but are saved and will be included in any paste or
print of this topic.
 
The Capture Screen command will replace any previous captured screen.

q.mv11 - Help for Microsoft QuickHelp
Current Search (A)  Use this command to search for a string within the
current topic. You can search for the string repeatedly; as long as the
string is on the top line of the QuickHelp window, QuickHelp will find
the next occurrence of the string, wrapping around to the beginning of
the topic if necessary. You can use uppercase or lowercase letters in
the string.

.freeze n
 
The freeze command specifies the number of lines at the beginning of the
topic that are to remain in place even if the user scrolls through the
topic. You typically use this command to keep navigation buttons visible at
all times.
 
Example
 
The following example freezes the top three lines at the top of the
QuickHelp window:
 
    .freeze 3

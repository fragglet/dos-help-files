q.mv12 - Help for Microsoft QuickHelp
Global Search (G)  Use this command to search for a string within all
topics in all open databases. QuickHelp will begin searching the
database containing the currently displayed topic. You can stop the
search at any time by pressing the ESC key. When the search stops,
QuickHelp will display a list of the topics found, if any. You can
then choose which topic you want QuickHelp to display.

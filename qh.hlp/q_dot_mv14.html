q.mv14
View Next List Item (M)  Use this command to move to the next topic in
the last list that was displayed. If the list is not already being
shown, QuickHelp will redisplay the last list topic viewed, move the
highlighted line down one line, and display the topic that is
underneath the highlighted line. If the list is being shown when the M
key is pressed, QuickHelp will start by displaying the topic under the
highlight.

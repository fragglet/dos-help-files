#define INCL_WINWINDOWMGR
 
BOOL WinIsThreadActive(hab)
HAB hab;    /* handle of the anchor block */
 
The WinIsThreadActive function determines whether the active window belongs
to the calling thread.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
hab        Identifies the anchor block of the calling thread.
 
Return Value
 
The return value is TRUE if the active window belongs to the calling thread,
or it is FALSE if the active window does not belong to the thread.

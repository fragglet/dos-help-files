WinCreateGroup (1.2)

────────────────────────────────────────────────────────────────────────────
 
 
The WinCreateGroup function provides compatibility with MS OS/2 versions 1.1
and earlier. Applications intended exclusively for MS OS/2 versions 1.2 and
later should use the PrfCreateGroup function.

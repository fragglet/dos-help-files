puts
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The puts function writes <string> to the standard output stream
     stdout, replacing the string's terminating null character (\0)
     with a newline character (\n) in the output stream.
 
     Return Value
 
     The puts function returns a nonnegative value if successful. If
     the function fails, it returns EOF.

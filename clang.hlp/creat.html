─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <sys\types.h>, <sys\stat.h>, <errno.h>
 
  Syntax:    int _creat( char *filename, int pmode );
             pmode: _S_IWRITE, _S_IREAD, _S_IREAD | _S_IWRITE
 
  Returns:   a handle if successful, or -1 if not.
             errno: EACCES, EMFILE, ENOENT
 
  See also:  _close, _open, _sopen, _dup, _dup2

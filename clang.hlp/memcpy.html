─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>, <memory.h>
 
  Syntax:    void *memcpy( void *dest, void *src, size_t count );
             void __far * _fmemcpy( void __far *dest, void __far *src,
                                    size_t count );
 
  Returns:   the value of <dest>.
 
  See also:  _memccpy, memmove, _movedata, memset, strcpy

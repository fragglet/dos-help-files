─────Run-Time Library───────────────────────────────────────────────────────
 
  Constant:  EOF
 
  Context:   Returned by stream input and output functions such as
             putc, ungetc, scanf, fflush, and _fcloseall as well as non-
             stream functions such as _ungetch, _putch, _and __isascii.
 
  Summary:   This value is returned by an I/O routine when the
             end-of-file (or in some cases, an error) is encountered.

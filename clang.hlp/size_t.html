─────Run-Time Library───────────────────────────────────────────────────────
 
  Type:     size_t
 
  Include:  <direct.h> <malloc.h> <memory.h> <new.h>
            <search.h> <stddef.h> <stdio.h>
            <stdlib.h> <string.h> <time.h>
 
  Summary:  Type returned by sizeof operator. This type is also used
            as an argument or return type by numerous functions.
            Generally, size_t is used for any value that represents a
            count of bytes.

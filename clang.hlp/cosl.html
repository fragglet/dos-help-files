─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>
 
  Syntax:    long double _cosl( long double x );
             long double _sinl( long double x );
             long double _tanl( long double x );
 
  Returns:   (_cosl) the cosine of <x> if successful, or 0 if not.
             (_sinl) the sine of <x> if successful, or 0 if not.
             (_tanl) the tangent of <x> if successful, or 0 if not.
             errno: ERANGE
 
  See also:  _acosl, _coshl, cos, _matherrl

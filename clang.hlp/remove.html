─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <stdio.h>, <errno.h>
 
  Syntax:    int remove( char *path );
             int _unlink( char *path );
 
  Returns:   0 if the file is deleted successfully, or -1 if not.
             errno: EACCES, ENOENT

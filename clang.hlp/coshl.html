─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    long double _coshl( long double x );
             long double _sinhl( long double x );
             long double _tanhl( long double x );
 
  Returns:   (_coshl) the hyperbolic cosine of <x>, or _LHUGE_VAL if the
                     result is too large.
             (_sinhl) the hyperbolic sine of <x> if successful, or
                     ±_LHUGE_VAL if not.
             (_tanhl) the hyperbolic tangent of <x>.
             errno: ERANGE
 
  See also:  _cosl, acos, _acosl, _matherrl

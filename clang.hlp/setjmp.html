longjmp and setjmp
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <setjmp.h>
 
  Syntax:    void longjmp( jmp_buf env, int value );
             int setjmp( jmp_buf env );
 
  Returns:   for longjump, there is no return value.
             for setjmp, 0 or the <value> argument of longjmp if
             returned by a longjmp call, or 1 if the longjmp <value>
             argument is 0.
 
  See also:  _fpreset, jmp_buf, signal

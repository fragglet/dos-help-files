─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <process.h>, <stdlib.h>, <errno.h>
 
  Syntax:    int system( char *command );
 
  Returns:   a nonzero value if <command> is NULL and the command
             interpreter is found. If the command interpreter is not
             found, the return is 0 and errno is ENOENT. If <command> is
             not NULL, the return is 0 if the command interpreter is
             successfully started.
             -1 indicates an error.
             errno: E2BIG, ENOENT, ENOEXEC, ENOMEM
 
  See also:  _execl..., _spawn...

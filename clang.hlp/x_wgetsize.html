─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>
 
  Syntax:    int _wgetsize( int wfh, int reqtype,
                            struct _wsizeinfo *wsize );
             reqtype: _WINCURRREQ, _WINMAXREQ
 
  Returns:   0 if successful (and fills in <_wsizeinfo>), or -1 if not.
 
  See also:  _fwopen, _wabout, _wclose, _wgetexit, _wgetfocus,
             _wgetscreenbuf, _wmenuclick, _wopen, _wsetexit, _wsetfocus,
             _wsetscreenbuf, _wsetsize, _wyield

_FP_OFF, _FP_SEG
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _FP_OFF and _FP_SEG macros can be used to set or get the
     offset or segment, respectively, of the far pointer at <address>.
 
     Return Value
 
     The _FP_OFF macro returns an offset. The _FP_SEG macro returns a
     segment address.

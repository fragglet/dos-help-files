─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int ungetc( int c, FILE *stream );
 
  Returns:   the character argument <c> if successful, or EOF if not.
 
  See also:  getc, fgetc, putc

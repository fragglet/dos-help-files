─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <sys\types.h>, <sys\stat.h>, <errno.h>
 
  Syntax:    int _chmod( char *filename, int pmode );
             pmode: _S_IWRITE, _S_IREAD, _S_IREAD | _S_IWRITE
 
  Returns:   0 if successful, or -1 if not.
             errno: ENOENT
 
  See also:  _stat, _access, _utime, _open, _creat

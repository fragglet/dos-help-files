─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    char *gets( char *buffer );
 
  Returns:   a pointer to its argument if successful, or a NULL pointer
             if at end-of-file or if unsuccessful.
 
  See also:  fgets, fputs, puts, _cgets

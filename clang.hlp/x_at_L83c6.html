─────Run-Time Library───────────────────────────────────────────────────────
 
     The sqrt functions calculate the square root of <x>.
 
     The _sqrtl function uses an 80-bit long double form of arguments
     and return values. In all other respects, it is identical to the
     regular function.
 
     Return Value
 
     The sqrt functions return the square-root result. If <x> is
     negative, the function prints a _DOMAIN error message to stderr,
     sets errno to EDOM, and returns 0.
 
     Error handling can be modified by using the _matherr (or
     _matherrl) routine.

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    char *strtok( char *string1, char *string2 );
             char __far *_fstrtok( char __far *string1,
                                   char __far *string2 );
 
  Returns:   a pointer to a token in <string1>.
 
  See also:  strspn, strcspn, strpbrk

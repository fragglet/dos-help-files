─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    void *memmove( void *dest, void *src, size_t count );
             void __far *_fmemmove( void __far *dest, void __far *src,
                                    size_t count );
 
  Returns:   the value of <dest>.
 
  See also:  memcpy, _memccpy, _movedata, memset, strcpy

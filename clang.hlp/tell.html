─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    long _tell( int handle );
 
  Returns:   the current position if successful, or -1L if not.
             errno: EBADF
 
  See also:  _lseek, ftell, fgetpos

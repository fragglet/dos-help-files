─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int fprintf( FILE *stream, char *format [, argument]... );
 
  Returns:   the number of characters printed, or a negative number in
             case of an output error.
 
  See also:  printf, fscanf, _cprintf, sprintf

─────Run-Time Library───────────────────────────────────────────────────────
 
     The _dos_getdrive routine uses system call 0x19 to obtain the
     current disk drive. The current drive is returned in the word
     pointed to by <drive>: 1=drive A, 2=drive B, and so on.
 
     Return Value
 
     None.

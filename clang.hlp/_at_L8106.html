─────C/C++ Language─────────────────────────────────────────────────────────
 
  The following are the statements recognized by the C language:
 
  break        default     for      return     typedef
  case         do          goto     switch     while
  continue     else        if
 
  The C++ language adds the following statements:
 
  delete     operator     template
  new        this

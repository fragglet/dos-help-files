─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <malloc.h>, <stdlib.h>
 
  Syntax:    void *calloc( size_t num, size_t size );
 
             void __based( void ) *_bcalloc( __segment seg, size_t num,
                                             size_t size );
 
             void __far *_fcalloc( size_t num, size_t size );
 
             void __near *_ncalloc( size_t num, size_t size );
 
  Returns:   a pointer to the allocated space if successful, or NULL if
             not (_NULLOFF if _bcalloc fails).
 
  See also:  malloc, free, _halloc, realloc, __based, new

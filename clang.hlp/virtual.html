─────C/C++ Language─────────────────────────────────────────────────────────
 
  Keyword:   virtual
 
  Syntax:    virtual member-function-declarator
                 virtual [access-specifier] base-class-name
 
  Summary:   Declares a virtual function or a virtual base class.
 
             See: ◄Virtual Function►
                  ◄Virtual Base Class►
                  ◄Pure Virtual Functions and Abstract Base Classes►
 
  See also:  class, private, public, protected
 
     The elements of a virtual declaration are as follows:
 
     Element                       Description
 
     <member-function-declarator>  Declares a member function.
 
     <access-specifier>            Defines the level of access to the
                                   base class. Can appear before or
                                   the after virtual keyword.
 
     <base-class-name>             Identifies a previously declared
                                   class type.

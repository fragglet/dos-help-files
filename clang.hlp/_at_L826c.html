─────Run-Time Library───────────────────────────────────────────────────────
 
     The _setwritemode function sets the current logical write mode,
     which is used when drawing lines with the _lineto, _polygon, and
     _rectangle functions.
 
     The <action> argument defines the write mode. The possible values
     are _GAND, _GOR, _GPRESET, _GPSET, and _GXOR.
 
     Return Value
 
     The _setwritemode function returns the previous write mode, or -1
     if an error occurs.

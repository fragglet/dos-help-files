─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    size_t strcspn( char *string1, char *string2 );
             size_t _fstrcspn( char __far *string1, char __far *string2 );
 
  Returns:   the position of the first matching character in <string1>.
 
  See also:  strspn, strpbrk, strtok, strstr, strncmp, strchr

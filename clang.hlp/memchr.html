─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>, <memory.h>
 
  Syntax:    void *memchr( void *buf, int c, size_t count );
             void __far *_fmemchr( void __far *buf, int c, size_t count );
 
  Returns:   a pointer to location of <c> in <buf> if successful, or
             NULL if <c> is not within first <count> bytes of <buf>.
 
  See also:  memcmp, _memicmp, _memccpy, memset, strchr

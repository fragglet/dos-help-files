─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <malloc.h>
 
  Syntax:    void __huge *_halloc( long num, size_t size );
 
  Returns:   a pointer to the allocated space if successful, or NULL
             if not.
 
  See also:  _hfree, malloc, free, new

_fstat, _stat
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <sys\types.h>, <sys\stat.h>, <errno.h>
 
  Syntax:    int _fstat( int handle, struct _stat *buffer );
             int _stat( char *pathname, struct _stat *buffer );
 
  Returns:   0 if successful, or -1 if not.
             errno: EBADF (_fstat), ENOENT (_stat)
 
  See also:  _access, _chmod, _filelength, _utime

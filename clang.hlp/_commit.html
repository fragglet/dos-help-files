─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    int _commit( int handle );
 
  Returns:   0 if the file was successfully flushed to disk, -1 if not.
             errno: EBADF
 
  See also:  _creat, _open, _read, _write

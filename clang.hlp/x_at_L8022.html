_commit
 ◄Summary►                                   ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _commit function forces the operating system to write the file
     associated with <handle> to disk. This call ensures that the
     specified file is flushed immediately──not at the operating
     system's discretion.
 
     Return Value
 
     The _commit function returns 0 if the file was successfully
     flushed to disk. A return value of -1 indicates an error, and
     errno is set to EBADF, indicating an invalid file-handle argument.

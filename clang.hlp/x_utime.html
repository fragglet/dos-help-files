─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <sys\types.h>, <sys\utime.h>, <errno.h>
 
  Syntax:    int _utime( char *filename, struct _utimbuf *times );
 
  Returns:   0 if successful, or -1 if not.
             errno: EACCES, EINVAL, EMFILE, ENOENT
 
  See also:  _chmod, _access, _stat, time, _ftime

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:  <stdlib.h>
 
  Syntax:   type __max( type a, type b );
            type __min( type a, type b );
 
  Returns:  (__max) the larger of the two arguments.
            (__min) the smaller of the two arguments.

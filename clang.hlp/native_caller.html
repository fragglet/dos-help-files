  Pragma:    native_caller
 
  Syntax:    #pragma native_caller( [{ on | off }] )
 
  Summary:   The native_caller pragma controls the removal of
             native-code entry points from within source code. If you
             have p-code functions that are called only by other p-code
             functions, you can omit those entry points and save those
             bytes by using the /Gn compiler option or on a
             function-by-function basis with this pragma.
 
  See also:  /Gn

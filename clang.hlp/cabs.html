─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>
 
  Syntax:    double _cabs( struct _complex z );
             long double _cabsl( struct _complexl z );
 
  Returns:   the absolute value if successful, or HUGE_VAL on overflow.
             errno: ERANGE
 
  See also:  abs, fabs, labs, _hypot

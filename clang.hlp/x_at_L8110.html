─────Run-Time Library───────────────────────────────────────────────────────
 
   The following functions control display of pixels in graphics
   modes. Note that functions with trailing ellipses (...) have
   _w and/or _wxy variations:
 
  _arc...                    _getviewcoord...     _setbkcolor
  _clearscreen               _getwindowcoord      _setcliprgn
  _ellipse...                _getwritemode        _setcolor
  _floodfill...              _grstatus            _setfillmask
  _getarcinfo                _imagesize...        _setlinestyle
  _getbkcolor                _lineto...           _setpixel...
  _getcolor                  _moveto...           _setvideomode
  _getcurrentposition...     _pie...              _setvideomoderows
  _getfillmask               _polygon...          _setvieworg
  _getimage...               _putimage...         _setviewport
  _getlinestyle              _rectangle...        _setwindow
  _getphyscoord              _remapallpalette     _setwritemode
  _getpixel...               _remappalette
  _getvideoconfig            _selectpalette
 
   NOTE: These functions are not available in tiny model.

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_findfirst( char *filename, unsigned attrib,
                                      struct _find_t *fileinfo );
 
             unsigned _dos_findnext( struct _find_t *fileinfo );
 
             attrib: _A_ARCH, _A_HIDDEN, _A_NORMAL, _A_RDONLY,
                     _A_SUBDIR, _A_SYSTEM, _A_VOLID
                     (may be joined by |)
 
  Returns:   0 if a matching file is found, or the DOS error code
             if not.
             errno: ENOENT
 
  See also:  _setargv

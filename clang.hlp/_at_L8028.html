─────Run-Time Library───────────────────────────────────────────────────────
 
     The _cputs function writes the null-terminated string pointed to
     by <string> directly to the console. Note that a carriage-return─
     linefeed (CR-LF) combination is not automatically appended to the
     string.
 
     Return Value
 
     If successful, _cputs returns a 0. If the function fails, it
     returns a nonzero value.

sprintf, _snprintf
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int sprintf( char *buffer, char *format [, argument]... );
             int _snprintf( char *buffer, size_t count,
                            char *format [, argument]... );
 
  Returns:   the number of characters stored in <buffer>. For _snprintf,
             if <count> is exceeded, -1 is returned.
 
  See also:  sscanf, printf, _vsnprintf, vsprintf, fprintf, _cprintf

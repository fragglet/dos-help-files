─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    char *_strdup( char *string );
             char __far *_fstrdup( char __far *string );
             char __near *_nstrdup( char __far *string );
 
  Returns:   a pointer if successful, or NULL if not.
 
  See also:  strcpy, strcat, strncpy, _strset

_pg_vlabelchart
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _pg_vlabelchart function writes text vertically on the screen.
     The <x> and <y> arguments are pixel coordinates for the beginning
     location of text relative to the upper-left corner of the chart
     window.
 
     Return Value
 
     The _pg_vlabelchart function returns 0 if there were no errors. A
     nonzero value indicates a failure.

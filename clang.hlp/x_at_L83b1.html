_rmdir
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _rmdir function deletes the directory specified by <dirname>.
     The directory must be empty, and it must not be the current
     working directory or the root directory.
 
     Return Value
 
     The _rmdir function returns 0 if the directory is successfully
     deleted. A return value of -1 indicates an error, and errno is set
     to either EACCES or ENOENT.

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <vmemory.h>
 
  Syntax:    void __far *_vload( _vmhnd_t handle, int dirty );
             dirty: _VM_CLEAN, _VM_DIRTY
 
  Returns:   a far pointer to DOS memory if successful, or NULL if not.
 
  See also:  _vmalloc, _vlock

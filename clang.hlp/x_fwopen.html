─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    FILE* _fwopen( struct _wopeninfo *wopeninfo,
                            struct _wsizeinfo *wsizeinfo, char *mode );
             mode: "r", "w", "r+", "w+"
             ("t" or "b" appended to <mode> to indicate type)
 
  Returns:   A pointer to FILE if successful, or NULL if not.
 
  See also:  _wabout, _wclose, _wgetexit, _wgetfocus, _wgetscreenbuf,
             _wgetsize, _wmenuclick, _wopen, _wsetexit, _wsetfocus,
             _wsetscreenbuf, _wsetsize, _wyield

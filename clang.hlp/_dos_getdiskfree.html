─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_getdiskfree( unsigned drive,
                                        struct _diskfree_t *diskspace );
 
  Returns:   0 if successful, or a nonzero number if not.
             errno: EINVAL
 
  See also:  _dos_getdrive, _dos_setdrive, _stat

_expand Functions
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <malloc.h>
 
  Syntax:    void *_expand( void *memblock, size_t size );
 
             void __based( void ) *_bexpand( __segment seg,
                                             void __based( void ) *memblock,
                                             size_t size );
 
             void __far *_fexpand( void __far *memblock, size_t size );
 
             void __near *_nexpand( void __near *memblock, size_t size );
 
  Returns:   a pointer to the reallocated memory block if successful, or
             NULL if not (_NULLOFF if _bexpand fails).
 
  See also:  realloc, malloc, free, new

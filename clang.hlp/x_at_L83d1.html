_strdate, _strtime
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _strdate function copies the date to the buffer pointed to by
     <datestr>. The date is formatted mm/dd/yy, where mm is two digits
     representing the month, dd is two digits representing the day of
     the month, and yy is the last two digits of the year.
 
     The _strtime function copies the current time into the buffer
     pointed to by <timestr>. It is formatted hh:mm:ss, where hh is two
     digits representing the hour in 24-hour notation, mm is two digits
     representing the minutes past the hour, and ss is two digits
     representing seconds.
 
     The buffer must be at least nine bytes long.
 
     Return Value
 
     The _strdate function returns a pointer to the resulting text
     string <datestr>. The _strtime function returns a pointer to the
     resulting text string <timestr>.

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>
 
  Syntax:    int strncmp( char *string1, char *string2,
                          size_t count );
 
             int _fstrncmp( char __far *string1, char __far *string2,
                            size_t count );
 
  Returns:   a negative value if <string1> is less than <string2>,
             0 if <string1> is equal to <string2>, or a positive value
             if <string1> is greater than <string2>.
 
  See also:  _strnicmp, strcmp, strchr, strncpy, strncat, memcmp

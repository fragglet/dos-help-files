─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>
 
  Syntax:    char *_mktemp( char *template );
 
  Returns:   a pointer to the modified <template> argument, or NULL if
             <template> is badly formed or if no more unique names can
             be created from it.
 
  See also:  _tempnam, tmpnam, tmpfile, _creat

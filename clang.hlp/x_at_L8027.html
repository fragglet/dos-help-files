_cprintf
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _cprintf function formats and prints a series of characters
     and values directly to the console, using the _putch function to
     output characters.
 
     Each <argument> (if any) is converted and output according to the
     corresponding format specification in <format>. The format has the
     same form and function as the <format> argument for the printf
     function.
     See: ◄Format Specifiers►
          ◄Escape Sequences►
 
     Note that unlike the fprintf, printf, and sprintf functions,
     cprintf does not translate linefeed (LF) characters into
     carriage-return─linefeed (CR-LF) combinations on output.
 
     Return Value
 
     The _cprintf function returns the number of characters printed.

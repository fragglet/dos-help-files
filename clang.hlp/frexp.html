─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>
 
  Syntax:    double frexp( double x, int *expptr );
             long double _frexpl( long double x, int *expptr );
 
  Returns:   the mantissa, or 0 if <x> is 0.
 
  See also:  ldexp, sqrt, pow, modf, fmod

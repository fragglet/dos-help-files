─────Run-Time Library───────────────────────────────────────────────────────
 
     The _isatty function determines whether <handle> is associated with
     a character device (a terminal, console, printer, or serial port).
 
     Return Value
 
     The _isatty function returns a nonzero value if the device is a
     character device. Otherwise, the return value is 0.

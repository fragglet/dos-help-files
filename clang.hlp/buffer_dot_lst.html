─────Run-Time Library───────────────────────────────────────────────────────
 
   The following functions operate on buffers of bytes in memory.
   Since the buffers need not be null-terminated, a size must be
   specified for each:
 
  _fmemccpy     _fmemicmp     _memccpy     _memicmp     _movedata
  _fmemchr      _fmemmove      memchr       memmove     _swab
  _fmemcmp      _fmemset       memcmp       memset

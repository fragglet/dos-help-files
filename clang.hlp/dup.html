_dup, _dup2
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    int _dup( int handle );
             int _dup2( int handle1, int handle2 );
 
  Returns:   (_dup) a new handle if successful, or -1 if not.
             (_dup2) 0 if successful, or -1 if not.
             errno: EBADF, EMFILE
 
  See also:  _open, _creat, _close, freopen

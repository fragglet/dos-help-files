─────C/C++ Language─────────────────────────────────────────────────────────
 
  Pragma:    page
 
  Syntax:    #pragma page( [pages] )
 
  Summary:   Skips the specified number of pages in the source listing.
 
  See also:  linesize, pagesize, skip, title, subtitle
 
     The page pragma generates a formfeed (page eject) in the source
     listing (created with /Fs) at the place where the pragma appears.
 
     The optional <pages> parameter is an integer constant in the range
     1-127 that specifies the number of pages to eject. If <pages> is
     absent, the pragma uses a default value of 1, in which case the
     next line in the source file appears at the top of the next
     listing page.

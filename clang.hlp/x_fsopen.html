_fsopen
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>, <share.h>
 
  Syntax:    FILE *_fsopen( char *filename, char *mode, int shflag );
             mode:   "r", "w", "a", "r+", "w+", "a+"
                     ("t" or "b" appended to <mode> to indicate type)
             shflag: _SH_COMPAT, _SH_DENYNO, _SH_DENYRD, _SH_DENYRW,
                     _SH_DENYWR
 
  Returns:   a pointer to the stream if successful, or NULL if not.
 
  See also:  fopen, fclose, _sopen, _locking, ferror

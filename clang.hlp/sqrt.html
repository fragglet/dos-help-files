─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    double sqrt( double x );
             long double _sqrtl( long double x );
 
  Returns:   a square root if successful, or 0 if not.
             errno: EDOM
 
  See also:  pow, exp, ldexp, _ldexpl, frexp, log, _matherr

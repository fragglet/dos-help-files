─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    FILE *_fdopen( int handle, char *mode );
             mode: "r", "w", "a", "r+", "w+", "a+", "c", "n"
                   ("t" or "b" appended to <mode> to indicate type)
 
  Returns:   a pointer to the open stream if successful, or NULL
             if not.
 
  See also:  _open, fclose, fopen, _dup, freopen

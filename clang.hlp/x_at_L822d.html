_getpixel, _getpixel_w

─────Run-Time Library───────────────────────────────────────────────────────
 
     The functions in the _getpixel family return the pixel value (a
     color index) at a specified location. The _getpixel function uses
     the view coordinates (<x>, <y>). The _getpixel_w function uses the
     window coordinates (<wx>, <wy>).
 
     For both _getpixel and _getpixel_w, the ranges of possible pixel
     values and their color translations are determined by the current
     video mode and palette, respectively.
 
     The _getpixel_w routine is implemented as a macro.
 
     Return Value
 
     If successful, the function returns the color index. If the
     function fails (for example, the point lies outside the clipping
     region, or the program is in text mode), it returns -1.

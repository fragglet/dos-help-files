─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    long ftell( FILE *stream );
 
  Returns:   the current position if successful, or -1L if not.
             errno: EBADF, EINVAL
 
  See also:  fgetpos, fseek, _lseek, _tell

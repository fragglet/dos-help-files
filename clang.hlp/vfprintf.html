─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>, <stdarg.h>, <varargs.h>
 
  Syntax:    int vfprintf( FILE *stream, char *format, va_list argptr );
             int vprintf( char *format, va_list argptr );
             int vsprintf( char *buffer, char *format, va_list argptr );
             int _vsnprintf( char *buffer, size_t count, char *format,
                             va_list argptr );
 
  Returns:   the number of characters written, or a negative value if
             an error occurs.
 
  See also:  printf, fprintf, sprintf, _cprintf, va_arg

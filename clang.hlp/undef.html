─────C/C++ Language─────────────────────────────────────────────────────────
 
  Directive:  #undef
 
  Syntax:     #undef identifier
 
  Summary:    Removes the current definition of <identifier>, which
              must have been previously defined with the #define
              directive.
 
  See also:   #define, defined, #if, #ifdef, #ifndef

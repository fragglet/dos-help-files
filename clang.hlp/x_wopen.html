─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>
 
  Syntax:    int _wopen( struct _wopeninfo *wopeninfo,
                         struct _wsizeinfo *wsizeinfo,  int oflag );
             oflag: _O_BINARY, _O_RDONLY, _O_RDWR, _O_TEXT, _O_WRONLY
 
  Returns:   0 if successful, or -1 if not.
             errno: EINVAL, EMFILE
 
  See also:  _fwopen, _wabout, _wclose, _wgetfocus, _wgetscreenbuf,
             _wgetsize, _wmenuclick, _wsetfocus, _wsetscreenbuf,
             _wsetsize, _wyield

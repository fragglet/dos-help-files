─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_read( int handle, void __far *buffer,
                                 unsigned count, unsigned *numread );
 
  Returns:   0 if successful, or the DOS error code if not.
             errno: EACCES, EBADF
 
  See also:  _dos_write, _read, fread

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <graph.h>
 
  Syntax:    short _lineto( short x, short y );
             short _lineto_w( double wx, double wy );
 
  Returns:   a nonzero value if the line is drawn successfully,
             or 0 if not.
 
  See also:  _moveto, _getcurrentposition, _setlinestyle,
             _setwritemode

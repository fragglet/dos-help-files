─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <dos.h>, <errno.h>
 
  Syntax:    unsigned _dos_getftime( int handle, unsigned *date,
                                     unsigned *time );
 
  Returns:   0 if successful, or the DOS error code if not.
             errno: EBADF
 
  See also:  _dos_setftime, _utime

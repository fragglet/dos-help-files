─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <malloc.h>
 
  Syntax:    void *_alloca( size_t size );
 
  Returns:   a void pointer to the allocated space if successful, or
             NULL if not.
 
  See also:  _stackavail, malloc, calloc, new

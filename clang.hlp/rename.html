─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <stdio.h>, <errno.h>
 
  Syntax:    int rename( char *oldname, char *newname );
 
  Returns:   0 if successful, or a nonzero value if not.
             errno: EACCES, ENOENT, EXDEV
 
  See also:  _creat, fopen, _open, remove

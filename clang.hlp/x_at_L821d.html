_clearscreen
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _clearscreen function erases the target area, filling it with
     the current background color. The <area> argument can be one of
     the following manifest constants (defined in GRAPH.H):
 
     Constant          Area Cleared and Filled
 
     _GCLEARSCREEN     Entire screen
     _GVIEWPORT        Current viewport
     _GWINDOW          Current text window
 
     Return Value
 
     None.

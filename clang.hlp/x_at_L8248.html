─────Run-Time Library───────────────────────────────────────────────────────
 
     The _pg_hlabelchart function writes text horizontally on the
     screen. The <x> and <y> arguments are pixel coordinates for the
     beginning location of text relative to the upper-left corner of
     the chart window.
 
     Return Value
 
     The _pg_hlabelchart function returns 0 if there were no errors. A
     nonzero value indicates a failure.

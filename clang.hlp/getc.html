─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int getc( FILE *stream );
             int getchar( void );
 
  Returns:   the character read. EOF indicates an error or end-of-file.
 
  See also:  fgetc, _getw, putc, ungetc, _getch

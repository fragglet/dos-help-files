Hyperbolic Trigonometric Functions
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    double cosh( double x );
             double sinh( double x );
             double tanh( double x );
 
  Returns:   (cosh) the hyperbolic cosine of <x>, or HUGE_VAL if the
                    result is too large.
             (sinh) the hyperbolic sine of <x> if successful, or ±HUGE_VAL
                    if not.
             (tanh) the hyperbolic tangent of <x>.
             errno: ERANGE
 
  See also:  cos, acos, _cosl, _matherr

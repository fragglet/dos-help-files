─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    long _filelength( int handle );
 
  Returns:   the file length in bytes, or -1L if an error occurs.
             errno: EBADF
 
  See also:  _chsize, _fstat, _fileno

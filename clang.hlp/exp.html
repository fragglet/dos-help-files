─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    double exp( double x );
             long double _expl( long double x );
 
  Returns:   the exponential value if successful, HUGE_VAL (or
             _LHUGE_VAL for _expl) on overflow, or 0 on underflow.
             errno: ERANGE
 
  See also:  log, ldexp, pow, sqrt, frexp

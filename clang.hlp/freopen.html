─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    FILE *freopen( char *filename, char *mode, FILE *stream );
             mode: "r", "w", "a", "r+", "w+", "a+"
                   ("t" or "b" appended to <mode> to indicate type)
 
  Returns:   a pointer to the newly opened file if successful, or a NULL
             pointer if not.
 
  See also:  _dup, fopen, _fileno

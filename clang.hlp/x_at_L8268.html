_setvideomoderows
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The _setvideomoderows function selects a screen mode for a
     particular hardware/display combination. The _setvideomoderows
     function also specifies the number of text rows to be used in a
     text mode.
 
     The <mode> argument can be one of the manifest constants listed
     in any of the following tables:
     See: ◄Constants for Screen Mode►
          ◄VESA Constants for Screen Mode►
          ◄_MAXRESMODE and _MAXCOLORMODE Constants for Screen Mode►
 
     If the constant _MAXTEXTROWS is specified for the <rows> argument,
     the function will choose the maximum number of rows available.
     See: _MAXTEXTROWS
 
     See also: ◄Hercules Support►
 
     Return Value
 
     This function returns the number of rows set. The function returns
     0 if an error occurred (e.g., if the mode is not supported).

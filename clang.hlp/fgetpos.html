─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>, <errno.h>
 
  Syntax:    int fgetpos( FILE *stream, fpos_t *pos );
             int fsetpos( FILE *stream, fpos_t *pos );
 
  Returns:   0 if successful, or a nonzero value if not.
             errno: EBADF, EINVAL
 
  See also:  fseek, rewind, ftell

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int fclose( FILE *stream );
             int _fcloseall( void );
 
  Returns:   (fclose) 0 if successful, or EOF if not.
             (_fcloseall) the total number of streams closed if
                         successful, or EOF if not.
 
  See also:  fopen, _fsopen, fflush, _close

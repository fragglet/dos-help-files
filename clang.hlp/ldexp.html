─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>, <errno.h>
 
  Syntax:    double ldexp( double x, int exp );
             long double _ldexpl( long double x, int exp );
 
  Returns:   an exponential value if successful; ±HUGE_VAL (or
             ±_LHUGE_VAL for _ldexpl) on overflow.
             errno: ERANGE
 
  See also:  frexp, sqrt, pow, modf, fmod

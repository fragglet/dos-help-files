─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int _getw( FILE *stream );
 
  Returns:   the integer value read. EOF may indicate error or
             end-of-file.
 
  See also:  _putw, getc, fgetc

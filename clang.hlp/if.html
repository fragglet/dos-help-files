if, else
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────C/C++ Language─────────────────────────────────────────────────────────
 
  Keyword:   if
 
  Syntax:    if( expression )
                  statement1
             [else
                  statement2]
 
  Summary:   Executes <statement1> if <expression> is true (nonzero);
             if else is present and <expression> is false (zero),
             executes <statement2>. After executing <statement1> or
             <statement2>, control passes to the next statement.
 
  See also:  switch

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int putc( int c, FILE *stream );
             int putchar( int c );
 
  Returns:   the character written. EOF indicates an error.
 
  See also:  fputc, _putw, getc

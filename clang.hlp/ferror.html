─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int ferror( FILE *stream );
 
  Returns:   a nonzero value to indicate an error in <stream>, or 0 to
             indicate no error.
 
  See also:  clearerr, feof, perror

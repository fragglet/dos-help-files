__asm Keyword Statement Format
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────C/C++ Language─────────────────────────────────────────────────────────
 
     Since the __asm keyword is a statement separator, you can put
     assembly instructions on the same line. The following statements
     are equivalent:
 
          __asm                  // __asm block
          {
             mov ax, 01h
             int 10h
          }
 
          __asm mov ax, 01h      // Separate __asm lines
          __asm int 10h
 
          // Multiple __asm statements on a line
          __asm mov ax, 01h   __asm int 10h

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <sys\locking.h>, <errno.h>
 
  Syntax:    int _locking( int handle, int mode, long nbytes );
             mode: _LK_LOCK, _LK_NBLCK, _LK_NBRLCK, _LK_RLCK, _LK_UNLCK
 
  Returns:   0 if successful, or -1 if not.
             errno: EACCES, EBADF, EDEADLOCK, EINVAL
 
  See also:  _sopen, _fsopen

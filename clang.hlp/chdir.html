─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <direct.h>, <errno.h>
 
  Syntax:    int _chdir( char *dirname );
 
  Returns:   0 if successful, or a nonzero value if not.
             errno: ENOENT
 
  See also:  _mkdir, _rmdir, _getcwd, _fullpath

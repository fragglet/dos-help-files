─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int fflush( FILE *stream );
 
  Returns:   0 if successful, or if <stream> has no buffer or is open
             only for reading; otherwise, EOF.
 
  See also:  fclose, _flushall, clearerr, commode.obj

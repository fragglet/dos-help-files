atof, atoi, atol, _atold

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <math.h>  (atof, _atold)
             <stdlib.h>  (atof, _atold, atoi, atol)
 
  Syntax:    double atof( char *string );
             int atoi( char *string );
             long atol( char *string );
             long double _atold( char *string );
 
  Returns:   the converted string, 0 if the string cannot be
             converted, or undefined in case of an overflow.
 
  See also:  strtol, strtod, _itoa, _ecvt

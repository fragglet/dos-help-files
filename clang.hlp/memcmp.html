─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <string.h>, <memory.h>
 
  Syntax:    int memcmp( void *buf1, const void *buf2, size_t count );
             int _fmemcmp( void __far *buf1, const void __far *buf2,
                           size_t count );
 
  Returns:   a negative value if <buf1> is less than <buf2>, 0 if
             <buf1> is equal to <buf2>, or a positive value if <buf1>
             is greater than <buf2>.
 
  See also:  _memicmp, memchr, strcmp

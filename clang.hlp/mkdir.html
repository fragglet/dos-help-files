─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <direct.h>, <errno.h>
 
  Syntax:    int _mkdir( char *dirname );
 
  Returns:   0 if successful, or -1 if not.
             errno: EACCES, ENOENT
 
  See also:  _chdir, _rmdir, _chdrive

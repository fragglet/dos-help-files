_write

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <io.h>, <errno.h>
 
  Syntax:    int _write( int handle, void *buffer, unsigned count );
 
  Returns:   the number of bytes actually written if successful, or -1
             if not.
             errno: EBADF, ENOSPC
 
  See also:  _read, fwrite

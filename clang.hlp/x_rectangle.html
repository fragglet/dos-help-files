_rectangle Functions
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <graph.h>
 
  Syntax:    short _rectangle( short control, short x1, short y1,
                               short x2, short y2 );
 
             short _rectangle_w( short control, double wx1, double wy1,
                                 double wx2, double wy2 );
 
             short _rectangle_wxy( short control,
                                   struct _wxycoord __far *pwxy1,
                                   struct _wxycoord __far *pwxy2 );
 
             control: _GBORDER, _GFILLINTERIOR
 
  Returns:   a nonzero value if the rectangle is drawn successfully, or
             0 if not.
 
  See also:  _ellipse, _lineto, _setlinestyle, _setwritemode

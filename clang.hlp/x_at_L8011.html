─────Run-Time Library───────────────────────────────────────────────────────
 
     The _bios_memsize routine uses INT 0x12 to determine the total
     amount of main memory available.
 
     Return Value
 
     The routine returns the total amount of installed memory in 1K
     blocks. The maximum return value is 640, representing 640K of main
     memory.

─────Run-Time Library───────────────────────────────────────────────────────
 
     The clearerr function resets the error indicator and end-of-file
     indicator for <stream>. Error indicators are not automatically
     cleared. Once the error indicator for a specified stream is set,
     operations on that stream continue to return an error value until
     clearerr, fseek, fsetpos, or rewind is called.
 
     Return Value
 
     None.

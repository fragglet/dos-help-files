remove, _unlink
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
─────Run-Time Library───────────────────────────────────────────────────────
 
     The remove and _unlink functions delete the file specified by
     <path>. The _unlink function is supported in the UNIX and XENIX
     operating systems. In DOS, the two functions are identical.
 
     Return Value
 
     The functions return 0 if the file is successfully deleted.
     Otherwise, they return -1 and set errno to ENOENT or EACCES.

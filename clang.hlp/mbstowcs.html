mbstowcs, _fmbstowcs

─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdlib.h>
 
  Syntax:    size_t mbstowcs( wchar_t *wcstr, char *mbstr, size_t count );
             size_t _fmbstowcs( wchar_t __far *wcstr, char __far *mbstr,
                                size_t count );
 
  Returns:   the number of successfully converted multibyte characters;
             -1 if an invalid multibyte character is encountered.
 
  See also:  mblen, mbtowc, wchar_t, wcstombs, wctomb, MB_CUR_MAX,
             MB_LEN_MAX

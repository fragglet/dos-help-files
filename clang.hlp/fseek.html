─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <stdio.h>
 
  Syntax:    int fseek( FILE *stream, long offset, int origin );
             origin: SEEK_CUR, SEEK_END, SEEK_SET
 
  Returns:   0 if successful, or a nonzero value if not.
 
  See also:  ftell, _lseek, rewind, fgetpos, fsetpos

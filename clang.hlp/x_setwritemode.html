─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <graph.h>
 
  Syntax:    short _setwritemode( short action );
             action: _GAND, _GOR, _GPRESET, _GPSET, _GXOR
 
  Returns:   the previous write mode, or -1 if an error occurs.
 
  See also:  _getwritemode, _lineto, _rectangle, _setlinestyle

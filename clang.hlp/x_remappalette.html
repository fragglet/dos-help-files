─────Run-Time Library───────────────────────────────────────────────────────
 
  Include:   <graph.h>
 
  Syntax:    long _remappalette( short index, long color );
             color: _BLACK, _BLUE, _BRIGHTWHITE, _BROWN, _CYAN, _GRAY,
                    _GREEN, _LIGHTBLUE, _LIGHTCYAN, _LIGHTGREEN,
                    _LIGHTMAGENTA, _LIGHTRED, _YELLOW, _MAGENTA, _RED,
                    _WHITE
 
  Returns:   the previous color value of the <index> argument if
             successful, or -1 if not.
 
  See also:  _remapallpalette, _selectpalette, _setbkcolor,
             _setvideomode

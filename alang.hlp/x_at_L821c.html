     Macro           Description
 
     @GetCsr         Gets the cursor position and size
     @SetCsrPos      Sets the cursor position
     @SetCsrSize     Sets the cursor size and shape
     @GetCharAtr     Gets the character and attribute at cursor
     @PutCharAtr     Puts characters and attributes at cursor
     @PutChar        Puts characters at cursor
     @Scroll         Scrolls a specified window up or down
     @Cls            Clears the screen

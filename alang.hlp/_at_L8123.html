  Interrupt:   15h     Function:   4Fh
 
  Title:       Keyboard Intercept
 
  See also:    Keyboard I/O, Keyboard Interrupts, Key Scan Chart,
               Extended Key Codes, Keyboard Data Memory
 
  Description:                                                    [PS/2]
 
     Invoked for each keystroke by the ROM BIOS's Int 09h keyboard
     interrupt handler.
 
     Input              Output
 
     AH = 4Fh           If scan code consumed
     AL = Scan code       Carry flag: clear
 
                        If scan code not consumed
                          Carry flag: set
                          AL = Unchanged or new scan code
                                    -♦-

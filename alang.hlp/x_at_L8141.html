  Title:       Mouse Pointer Interrupts
 
  Subfunction     Name
 
  00h             Enable/Disable Pointing Device
  01h             Reset Pointing Device
  02h             Set Sample Rate
  03h             Set Resolution
  04h             Get Pointing Device Type
  05h             Initialize Pointing Device Interface
  06h             Set Scaling or Get Status

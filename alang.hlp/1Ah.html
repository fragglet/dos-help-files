  Interrupt:   1Ah
 
  Title:       Clock Interrupts
 
  Function     Name

  01h          Set Tick Count
  02h          Get Time
  03h          Set Time
  04h          Get Date
  05h          Set Date
  06h          Set Alarm
  07h          Reset Alarm
  0Ah          Get Day Count
  0Bh          Set Day Count
  80h          Set Sound Source

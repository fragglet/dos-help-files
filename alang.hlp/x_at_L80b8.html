  Interrupt:   10h     Function:   10h     Subfunction:   15h
 
  Title:       Get Color Register
 
  See also:    Set Color Register, Get Block of Color Registers
 
  Description:                                              [MCGA] [VGA]
 
     Returns the contents of a color register as its red, green, and
     blue components.
 
     Input                   Output
 
     AH = 10h                CH = Green value
     AL = 15h                CL = Blue value
     BX = Color register     DH = Red value

  Include:   DOS.INC
 
  Syntax:    @GetVer
 
  Summary:   Gets the DOS version
 
  Arguments: None
 
  Returns:   AL        Major version (0 for versions prior to 2.0)
             AH        Minor version
             BH        OEM serial number
             BL:CX     24-bit user number
 
  Modifies:  AX, BX, CX
 
  See also:  Int 21h Function 30h, SetDate, GetTime

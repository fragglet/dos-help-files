  Interrupt:   10h     Function:   00h
 
  Title:       Set Video Mode
 
  See also:    Get Video Mode, @SetMode
 
  Description:                     [MDA] [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Selects the current video display mode. Also selects the active
     video controller if more than one is present.
 
     Input                Output
 
     AH = 00h             None
     AL = Video mode

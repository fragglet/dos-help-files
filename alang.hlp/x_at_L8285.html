  Interrupt:   21h     Function:   19h
 
  Title:       Get Current Drive
 
  See also:    Get Drive Data, Get Drive Parameters, Select Disk, @GetDrv
 
  Description:
 
     Returns the drive code of the current, or default, disk drive.
 
     Input        Output
 
     AH = 19h     AL = Drive code (0 = A, 1 = B,...)

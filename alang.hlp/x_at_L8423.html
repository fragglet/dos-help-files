Address Operators
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Select an operator for more information.
 
     Operator     Description
 
     []           Treats enclosed expression as an index
     :            Performs segment override
     :/::         Delimits label
     $            Gives current value of location counter
 
     OFFSET       Gives offset address of expression (assembler
                  resolved)
     LROFFSET     Gives offset address of expression (linker resolved)
     SEG          Returns frame of expression
     THIS         Returns operand with current offset and segment
     SHORT        Defines short (distance < 128 bytes) label
 
     ADDR         Yields address of label for INVOKE directive
     See also: Data types

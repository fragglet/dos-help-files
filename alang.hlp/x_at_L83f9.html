     See also: Scaling (ss) Field, Base Register (base) Field
 
     In scaled indexing, the index register (index) field specifies the
     index register used. Scaled addressing is allowed only under the
     .386 and .486 directives.
 
     Index     Register
 
     000       EAX
     001       ECX
     010       EDX
     011       EBX
     100       No index
     101       EBP
     110       ESI
     111       EDI

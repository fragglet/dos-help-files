────────────────────────────────────────────────────────────────────────────
 
  Include:   BIOS.INC
 
  Syntax:    @GetCsr [page]
 
  Summary:   Gets the cursor position and size
 
  Argument:  <page>     8-bit page with cursor; if none given,
                        0 assumed
 
  Returns:   DL     Column
             DH     Row
             CL     Starting scan line
             CH     Ending scan line
 
  Modifies:  AX, DX, CX, BH
 
  See also:  Int 10h Function 03h, @SetCsrPos, @SetCsrSize

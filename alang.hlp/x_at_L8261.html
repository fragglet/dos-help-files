  Interrupt:   21h     Function:   03h
 
  Title:       Auxiliary Input
 
  See also:    Read Character from Communications Port, Auxiliary Output
 
  Description:
 
     Reads a character from the standard auxiliary device. The default
     is the first serial port (COM1).
 
     Input        Output
 
     AH = 03h     AL = 8-bit input data

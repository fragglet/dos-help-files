────────────────────────────────────────────────────────────────────────────
 
  Include:   DOS.INC
 
  Syntax:    @Exit [return]
 
  Summary:   Exits to DOS with return code
 
  Argument:  <return>     8-bit code to return to DOS; if none given,
                          AL is used. If given, must be a constant.
 
  Returns:   No return value
 
  Modifies:  AX
 
  See also:  .EXIT, Int 21h Function 4Ch

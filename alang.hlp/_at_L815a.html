  Interrupt:   17h     Function:   01h
 
  Title:       Initialize Printer Port
 
  Description:                                          [PC] [AT] [PS/2]
 
     Initializes the specified parallel printer interface port and
     returns its status.
 
     Input                             Output
 
     AH = 01h                          AH = Status
     DX = Printer number (0 = LPT1,         See: Int 17h Function 00h
          1 = LPT2, 2 = LPT3)

  Interrupt:   10h     Function:   11h     Subfunction:   04h/14h
 
  Title:       Load ROM 8-by-16 Font and Reprogram Controller
 
  Description:                                               [MCGA] [VGA]
 
     Loads the ROM BIOS default 8-by-16 font table into the specified
     block of character generator RAM.
 
     Input               Output
 
     AH = 11h            None
     AL = 04h or 14h
     BL = Block

     Under OPTION OFFSET:SEGMENT (part of the MASM 5.1 compatibility
     option), the SEG operator returns:
 
       ■ The label's group, if there is a valid group in effect
       ■ The label's segment, if there is no valid group but there is a
         valid segment
       ■ The value of label, if there is neither a valid group nor a
         valid segment

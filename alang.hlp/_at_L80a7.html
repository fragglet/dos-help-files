  Interrupt:   10h     Function:   0Eh
 
  Title:       Write Character in Teletype Mode
 
  See also:    Character Output, Write String in Teletype Mode
 
  Description:                     [MDA] [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Writes an ASCII character to the display at the current cursor
     position, using the specified color (if in graphics modes), and
     then increments the cursor position appropriately.
 
     Input                                      Output
 
     AH = 0Eh                                   None
     AL = Character
     BH = Page
     BL = Foreground color (graphics modes)

     Macro       Description
 
     @MkDir      Creates a directory
     @RmDir      Deletes a directory
     @ChDir      Changes the current directory
     @GetDir     Gets the current directory of a specified drive
     @GetDrv     Gets the current drive
     @SetDrv     Sets the current drive
     @ChkDrv     Gets various data about a disk

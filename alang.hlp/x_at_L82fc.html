  Interrupt:   21h     Function:   54h
 
  Title:       Get Verify Flag
 
  See also:    Set Verify Flag
 
  Description:
 
     Obtains the current value of the system verify (read-after-write)
     flag.
 
     Input         Output
 
     AH = 54h      AL = Current verify flag value
                      = 00h (if verify OFF)
                      = 01h (if verify ON)

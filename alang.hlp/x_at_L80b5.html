  Interrupt:   10h     Function:   10h     Subfunction:   12h
 
  Title:       Set Block of Color Registers
 
  See also:    Get Block of Color Registers, Set Color Register,
               Set Gray-Scale Values
 
  Description:                                              [MCGA] [VGA]
 
     Programs a group of consecutive color registers in one operation.
 
     Input                                     Output
 
     AH = 10h                                  None
     AL = 12h
     BX = First color register
     CX = Number of color registers
     ES:DX = segment:offset of color table

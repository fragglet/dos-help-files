  Include:   DOS.INC
 
  Syntax:    @GetTime
 
  Summary:   Gets the system time
 
  Arguments: None
 
  Returns:   CH     Hour (0-23)
             CL     Minute (0-59)
             DH     Second (0-59)
             DL     Hundredth (0-99)
 
  Modifies:  AX, CX, DX
 
  See also:  Int 21h Function 2Ch, @GetDate, @SetTime

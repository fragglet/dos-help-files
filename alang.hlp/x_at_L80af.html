  Interrupt:   10h     Function:   10h     Subfunction:   03h
 
  Title:       Toggle Blink/Intensity Bit
 
  Description:                                 [PCjr] [EGA] [MCGA] [VGA]
 
     Determines whether the most significant bit of a character
     attribute will select blinking or intensified display.
 
     Input                           Output
 
     AH = 10h                        None
     AL = 03h
     BL = Blink/intensity toggle
          0 = Enable intensity
          1 = Enable blinking

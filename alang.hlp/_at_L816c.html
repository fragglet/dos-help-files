  Interrupt:   1Ah     Function:   0Bh
 
  Title:       Set Day Count
 
  Description:                                                    [PS/2]
 
     Stores an arbitrary value in the system's day counter.
 
     Input                        Output
 
     AH = 0Bh                     If function successful
     CX = Count of days since       Carry flag: clear
          1-1-1980
                                  If function unsuccessful
                                    Carry flag: set
                                    -♦-

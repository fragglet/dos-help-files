  Syntax:   .LIST
 
  See also: .LISTALL, .NOLIST, .LISTIF, .NOLISTIF, .TFCOND,
            .LISTMACROALL, .LISTMACRO, .NOLISTMACRO, PUSHCONTEXT, /Fl,
            /S<x>
 
  Description:
 
     Enables listing of all source statements to the .LST file. The
     .LST file must first be enabled with a PWB MASM option or the
     /Fl command-line option. This directive is in effect by default.

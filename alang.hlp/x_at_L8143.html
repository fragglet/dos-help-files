  Interrupt:   15h     Function:   C2h     Subfunction:   01h
 
  Title:       Reset Pointing Device
 
  Description:                                                    [PS/2]
 
     Resets the system's mouse or other pointing device, setting the
     sample rate, resolution, and other characteristics to their
     default values.
 
     Input        Output
 
     AH = C2h     If function successful
     AL = 01h       Carry flag: clear
                    AH = 00h
                    BH = Device ID
 
                  If function unsuccessful
                    Carry flag: set
                    AH = Status
                         See: Int 15h Function C2h
                              Subfunction 00h

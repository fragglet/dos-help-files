  Interrupt:   10h     Function:   03h
 
  Title:       Get Cursor Information
 
  See also:    Set Cursor Position, Set Cursor Type, @GetCsr
 
  Description:                     [MDA] [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Obtains the current position of the cursor on the display, in text
     coordinates.
 
     Input         Output
 
     AH = 03h      CH = Starting line for cursor
     BH = Page     CL = Ending line for cursor
                   DH = Row (y coordinate)
                   DL = Column (x coordinate)

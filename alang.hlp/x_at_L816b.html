  Interrupt:   1Ah     Function:   0Ah
 
  Title:       Get Day Count
 
  Description:                                                    [PS/2]
 
     Returns the contents of the system's day counter.
 
     Input        Output
 
     AH = 0Ah     If function successful
                    Carry flag: clear
                    CX = Count of days since 1-1-1980
 
                  If function unsuccessful
                    Carry flag: set

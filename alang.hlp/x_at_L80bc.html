Interrupt 10h, Function 11h (Fonts)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Interrupt:   10h     Function:   11h
 
  Title:       Font Interrupts
 
     Subfunction     Name
 
     00h             Load User Font and Reprogram Controller
     10h             Load User Font and Reprogram Controller
     01h             Load ROM 8-by-14 Font and Reprogram Controller
     11h             Load ROM 8-by-14 Font and Reprogram Controller
     02h             Load ROM 8-by-8 Font and Reprogram Controller
     12h             Load ROM 8-by-8 Font and Reprogram Controller
     03h             Set Block Specifier
     04h             Load ROM 8-by-16 Font and Reprogram Controller
     14h             Load ROM 8-by-16 Font and Reprogram Controller
     20h             Set Int 1Fh Font Pointer
     21h             Set Int 43h for User's Font
     22h             Set Int 43h for ROM 8-by-14 Font
     23h             Set Int 43h for ROM 8-by-8 Font
     24h             Set Int 43h for ROM 8-by-16 Font
     30h             Get Font Information

  Interrupt:   10h     Function:   11h     Subfunction:   01h/11h
 
  Title:       Load ROM 8-by-14 Font and Reprogram Controller
 
  Description:                                               [EGA] [VGA]
 
     Loads the ROM BIOS default 8-by-14 font table into the specified
     block of character generator RAM.
 
     Input               Output
 
     AH = 11h            None
     AL = 01h or 11h
     BL = Block

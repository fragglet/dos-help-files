  Interrupt:   10h     Function:   11h     Subfunction:   02h/12h
 
  Title:       Load ROM 8-by-8 Font and Reprogram Controller
 
  Description:                                        [EGA] [MCGA] [VGA]
 
     Loads the ROM BIOS default 8-by-8 font table into the specified
     block of character generator RAM.
 
     Input               Output
 
     AH = 11h            None
     AL = 02h or 12h
     BL = Block

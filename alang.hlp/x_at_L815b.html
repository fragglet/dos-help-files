  Interrupt:   17h     Function:   02h
 
  Title:       Get Printer Status
 
  Description:                                          [PC] [AT] [PS/2]
 
     Returns the current status of the specified parallel printer
     interface port.
 
     Input                             Output
 
     AH = 02h                          AH = Status
     DX = Printer number (0 = LPT1,         See: Int 17h Function 00h
          1 = LPT2, 2 = LPT3)

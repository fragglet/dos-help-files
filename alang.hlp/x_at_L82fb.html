  Interrupt:   21h     Function:   51h
 
  Title:       Get PSP Address
 
  See also:    Program Segment Prefix, Get PSP Address
 
  Description:
 
     Returns the segment address of a current program segment prefix
     (PSP).
 
     Input        Output
 
     AH = 51h     BX = PSP segment address

  Interrupt:   21h     Function:   1Ah
 
  Title:       Set DTA Address
 
  See also:    Get Data Transfer Area (DTA) Address,
               @SetDTA
 
  Description:
 
     Specifies the address of the disk transfer area (DTA) to be used
     for subsequent FCB-related function calls.
 
     Input                          Output
 
     AH = 1Ah                       None
     DS:DX = segment:offset of
             disk transfer area

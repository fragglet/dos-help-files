  Interrupt:   10h     Function:   02h
 
  Title:       Set Cursor Position
 
  See also:    Get Cursor Information, Set Cursor Type, @SetCsrPos
 
  Description:                     [MDA] [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Positions the cursor on the display, using text coordinates.
 
     Input                          Output
 
     AH = 02h                       None
     BH = Page
     DH = Row (y coordinate)
     DL = Column (x coordinate)

  Title:       Set Border Color
 
  See also:    Get Border Color
 
  Description:                                        [PCjr] [EGA] [VGA]
 
     Controls the color of the screen border (overscan).
 
     Input                Output
 
     AH = 10h             None
     AL = 01h

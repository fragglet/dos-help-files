  Interrupt:   10h     Function:   10h     Subfunction:   07h
 
  Title:       Get Palette Register
 
  See also:    Set Palette Register
 
  Description:                                                     [VGA]
 
     Returns the color associated with the specified palette
     register.
 
     Input                     Output
 
     AH = 10h                  BH = Color
     AL = 07h
     BL = Palette register

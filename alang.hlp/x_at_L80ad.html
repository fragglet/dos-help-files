  Interrupt:   10h     Function:   10h     Subfunction:   02h
 
  Title:       Set Palette and Border
 
  See also:    Get Palette and Border
 
  Description:                                        [PCjr] [EGA] [VGA]
 
     Sets all palette registers and the border color (overscan) in one
     operation.
 
     Input                                    Output
 
     AH = 10h                                 None
     AL = 02h
     ES:DX = segment:offset of color list

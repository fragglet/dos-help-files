  Interrupt:   10h     Function:   0Fh
 
  Title:       Get Video Mode
 
  See also:    Set Video Mode, Get Configuration Information, @GetMode
 
  Description:                     [MDA] [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Obtains the current display mode of the active video controller.
     This function can be called to obtain the screen width before
     clearing the screen with Int 10h Functions 06h or 07h.
 
     Input        Output
 
     AH = 0Fh     AH = Number of character columns on screen
                  AL = Display mode
                       See: Int 10h Function 00h
                  BH = Active display page

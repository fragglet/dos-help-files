  Interrupt:   16h     Function:   04h
 
  Title:       Set Keyclick
 
  Description:                                                      [PC]
 
     Turns the keyboard click on or off. This function is supported by
     the PCjr BIOS only.
 
     Input                                Output
 
     AH = 04h                             None
     AL = Subfunction
        = 00h Turn off keyboard click
        = 01h Turn on keyboard click

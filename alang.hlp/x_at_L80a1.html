  Interrupt:   10h     Function:   0Bh
 
  Title:       Set Palette, Background, or Border
 
  See also:    Color Summary, @SetColor, @SetPalette
 
  Description:                           [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Selects a palette, background, or border color.
 
     Input                                                  Output
 
     To set the background color and border color for       None
     graphics modes or the border color for text modes
       AH = 0Bh
       BH = 00h
       BL = Color
 
     To select the palette (320-by-200 4-color
     graphics modes)
       AH = 0Bh
       BH = 01h
       BL = Palette

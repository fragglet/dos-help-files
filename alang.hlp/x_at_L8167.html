  Interrupt:   1Ah     Function:   05h
 
  Title:       Set Date
 
  See also:    DOS Set Date, Get Date
 
  Description:                                               [AT] [PS/2]
 
     Sets the date in the CMOS time/date chip.
 
     Input                                 Output
 
     AH = 05h                              None
     CH = Century (19 or 20) in binary
          coded decimal (BCD)
     CL = Year in BCD
     DH = Month in BCD
     DL = Day in BCD

     If you specify a language type with the .MODEL directive, the
     assembler makes the procedure name PUBLIC, unless you explicitly
     declare the procedure name to be PRIVATE. Without a specified
     language type, procedures are always PRIVATE.
 
     Names declared with the PASCAL, BASIC, or FORTRAN language types
     are not mapped to uppercase.

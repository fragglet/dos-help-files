  Interrupt:   10h     Function:   10h     Subfunction:   10h
 
  Title:       Set Color Register
 
  See also:    Get Color Register, Set Block of Color Registers,
               Set Gray-Scale Values
 
  Description:                                              [MCGA] [VGA]
 
     Programs an individual color register with a red-green-blue (RGB)
     combination.
 
     Input                   Output
 
     AH = 10h                None
     AL = 10h
     BX = Color register
     CH = Green value
     CL = Blue value
     DH = Red value

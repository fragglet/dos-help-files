Device and File Control
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Macro            Description
 
     @Read            Reads data from a file or device
     @Write           Writes data to a file or device
     @MakeFile        Creates a file
     @OpenFile        Opens a file for input or output
     @CloseFile       Closes an open file handle
     @DelFile         Deletes a specified file
     @MoveFile        Moves or renames a file
     @GetFirst        Gets first matching file
     @GetNext         Gets additional matching files
     @GetDTA          Gets the Disk Transfer Address (DTA)
     @SetDTA          Sets the Disk Transfer Address (DTA)
     @GetFileSize     Gets the file size
     @MovePtrAbs      Moves file pointer to absolute position
     @MovePtrRel      Moves file pointer to relative position

  Interrupt:   15h     Function:   88h
 
  Title:       Get Extended Memory Size
 
  Description:                                               [AT] [PS/2]
 
     Returns the amount of extended memory installed in the system.
 
     Input        Output
 
     AH = 88h     AX = Amount of extended memory
                       (in kilobytes)

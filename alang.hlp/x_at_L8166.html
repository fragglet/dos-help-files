  Interrupt:   1Ah     Function:   04h
 
  Title:       Get Date
 
  See also:    DOS Get Date, Set Date
 
  Description:                                               [AT] [PS/2]
 
     Reads the current date from the CMOS time/date chip.
 
     Input        Output
 
     AH = 04h     CH = Century (19 or 20) in binary
                       coded decimal (BCD)
                  CL = Year in BCD
                  DH = Month in BCD
                  DL = Day in BCD
                  And if clock running
                    Carry flag: clear
                  Or if clock stopped
                    Carry flag: set

  Interrupt:   21h     Function:   3Ah
 
  Title:       Delete Directory
 
  See also:    Create Directory, Set Current Directory,
               Get Current Directory, @RmDir
 
  Description:
 
     Removes a directory using the specified drive and path.
 
     Input                                Output
 
     AH = 3Ah                             If function successful
     DS:DX = segment:offset of ASCIIZ       Carry flag: clear
             path name
                                          If function unsuccessful
                                            Carry flag: set
                                            AX = Error code (03h, 05h,
                                                 or 10h)
                                            See: ◄Error codes►

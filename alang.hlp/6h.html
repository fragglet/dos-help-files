  Interrupt:   06h
 
  Title:       Illegal Instruction Error
 
  Description:                                                    [186+]
 
     Invoked when the 80x86 processor attempts to
 
        ■ Decode a bit pattern not recognized as a valid instruction
        ■ Execute an instruction with invalid operands
        ■ Execute a protected-mode instruction in real mode
 
     This interrupt should not be called directly. Default service is
     interrupt return (IRET). To trap invalid opcode errors, a program
     must set up a handler routine to gain control when the interrupt
     occurs.
 
     Input     Output
 
     None      None

  Interrupt:   10h     Function:   0Ch
 
  Title:       Write Graphics Pixel
 
  See also:    Read Graphics Pixel
 
  Description:                           [CGA] [PCjr] [EGA] [MCGA] [VGA]
 
     Draws a point on the display at the specified graphics
     coordinates.
 
     Input                                   Output
 
     AH = 0Ch                                None
     AL = Pixel value
     BH = Page
     CX = Column (graphics x coordinate)
     DX = Row (graphics y coordinate)

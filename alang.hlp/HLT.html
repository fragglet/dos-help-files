  Title:    Halt                                  Flags: O D I T S Z A P C
                                                         ═════════════════
  Syntax:   HLT                                               -none-
 
  Description:
 
     Stops CPU execution until an interrupt restarts execution at the
     instruction following HLT. In protected mode, this instruction
     works only in privileged mode.

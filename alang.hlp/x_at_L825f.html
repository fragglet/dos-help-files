  Interrupt:   21h     Function:   02h
 
  Title:       Character Output
 
  See also:    Write Character at Cursor, Write Character and Attribute
               at Cursor, Display String, Printer Output, @ShowChar
 
  Description:
 
     Outputs a character to the standard output device. Output can be
     redirected. However if output is redirected, there is no way to
     detect that the disk is full.
 
     Input                          Output
 
     AH = 02h                       None
     DL = 8-bit data for output

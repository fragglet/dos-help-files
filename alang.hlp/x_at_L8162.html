  Interrupt:   1Ah     Function:   01h
 
  Title:       Set Tick Count
 
  See also:    Timer Data Memory
 
  Description:                                               [AT] [PS/2]
 
     Stores a 32-bit value in the clock tick counter.
 
     Input                            Output
 
     AH = 01h                         None
     CX:DX = Tick count
             (high 16-bits in CX)

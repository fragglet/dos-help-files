     Directive     Description
 
     PROC          Starts procedure block
     ENDP          Ends procedure block
     LOCAL         Declares local procedure placeholder
     PROTO         Defines parameter prototype for procedure
     INVOKE        Invokes stack-based procedure
     LABEL         Declares label in current segment
     :/::          Delimits label

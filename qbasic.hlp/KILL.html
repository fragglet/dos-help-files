──────────────────────────────────────────────────────────────────────────────
Deletes files from disk.
 
KILL filespec$
 
    ■ filespec$    Identifies the file or files to delete. It may include
                   a path and the DOS wildcard characters ? and *.
 
Example:
    INPUT "File to delete: "; f$
    KILL f$

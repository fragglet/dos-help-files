──────────────────────────────────────────────────────────────────────────────
Transfers control from the current program to another Basic program.
 
CHAIN filespec$
 
    ■ filespec$    The name of the program to which control is passed.
 
Example:
    'Assumes the program TEST.BAS is in a \DOS directory.
    CHAIN "C:\DOS\TEST.BAS"

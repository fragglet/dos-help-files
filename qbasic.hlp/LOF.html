Returns the length of a file in bytes.
 
LOF(filenumber%)
 
    ■ filenumber%    The number of an open file.
 
Example:
    INPUT "Enter filename: "; f$
    OPEN f$ FOR BINARY AS #1
    PRINT "File length = "; LOF(1)
    CLOSE

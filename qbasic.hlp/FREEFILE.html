FREEFILE Function
  ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
Returns the next valid unused file number.
 
FREEFILE
 
Example:
    OPEN "TEST.DAT" FOR OUTPUT AS #1
    PRINT "Next file number: "; FREEFILE
    CLOSE

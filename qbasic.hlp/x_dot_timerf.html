TIMER Function
  ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
Returns the number of seconds elapsed since midnight.
 
TIMER
 
    ■ Use TIMER to time programs or parts of programs, or with the RANDOMIZE
      statement to seed the random-number generator.
 
Example:
    RANDOMIZE TIMER

_getvisualpage
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _getvisualpage function returns the number of the current
     visual page. All hardware combinations support at least one page
     (page number 0). In OS/2, only page 0 is available.
 
 
     Return Value
 
     The function returns the page number of the current visual page.

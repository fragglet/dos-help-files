_getphyscoord
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _getphyscoord function translates the view coordinates
     (<x>, <y>) to physical coordinates, returning them in an xycoord
     structure (defined in GRAPH.H.)
 
     See also: ◄Graphic Coordinate Systems►
 
     Return Value
 
     None.

     The _setvideomoderows function selects a screen mode for a
     particular hardware/display combination. The manifest constants
     for the screen mode are given in the reference screen for
     _setvideomode. The _setvideomoderows function also specifies the
     number of text rows to be used in a text mode.
 
     If the constant _MAXTEXTROWS is specified for the <rows> argument,
     the function will choose the maximum number of rows available. In
     text modes, this is 50 on the VGA, 43 on EGA, and 25 on others. In
     graphics modes that support 30 or 60 rows, _MAXTEXTROWS specifies
     60 rows.
 
     Return Value
 
     This function returns the number of rows set. The function returns
     0 if an error occurred (e.g., if the mode is not supported).

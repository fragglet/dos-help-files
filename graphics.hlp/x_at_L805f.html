_getfontinfo
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _getfontinfo function gets the current font characteristics
     and stores them in the _fontinfo structure (defined in GRAPH.H).
 
     Return Value
 
     The _getfontinfo function returns a negative number if a font has
     not been registered or loaded; otherwise, it returns a non-negative
     number.

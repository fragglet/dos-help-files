_setvieworg
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _setvieworg function moves the view-coordinate origin (0, 0)
     to the physical point (<x>, <y>). All other view-coordinate points
     move the same direction and distance.
 
     NOTE: This function replaces the _setlogorg function used in
           Microsoft C version 5.1.
 
     See also: ◄Graphic Coordinate Systems►
 
     Return Value
 
     The function returns the physical coordinates of the previous view
     origin in an xycoord structure (defined in GRAPH.H).

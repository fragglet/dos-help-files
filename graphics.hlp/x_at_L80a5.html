_setwritemode
 ◄Summary► ◄Example►                                   ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _setwritemode function sets the current logical write mode,
     which is used when drawing lines with the _lineto and _rectangle
     functions.
 
     The <action> argument defines the write mode. The possible values
     are _GAND, _GOR, _GPRESET, _GPSET, and _GXOR.
 
     Return Value
 
     None.

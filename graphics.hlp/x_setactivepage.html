_setactivepage
 ◄Description► ◄Example►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Include:   <graph.h>
 
  Syntax:    short _setactivepage( short page );
 
  Returns:   the page number of the previous active page if
             successful, or a negative value if not.
 
  See also:  _getactivepage, _getvideoconfig, _getvisualpage,
             _setvisualpage

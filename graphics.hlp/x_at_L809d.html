_settextrows
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _settextrows function specifies the number of screen rows to
     be used in text modes.
 
     If the constant _MAXTEXTROWS is specified for the <rows> argument,
     the function will choose the maximum number of rows available. In
     text modes, this is 50 rows on VGA, 43 on EGA, and 25 on others.
     In graphics modes that support 30 or 60 rows, _MAXTEXTROWS
     specifies 60 rows.
 
     Return Value
 
     This function returns the numbers of rows set. The function
     returns 0 if an error occurred.

_getcolor
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The _getcolor function returns the current graphics color index.
     The default is the highest legal value of the current palette.
 
     Return Value
 
     The _getcolor function returns the current color index.

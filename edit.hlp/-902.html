Save As Dialog
Use to save and name the current file.
┌──────────── Save As ────────────┐
│                                 │  Accept this filename or
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────── type a new name here.
│                                 │
│ C:\DOS ◄────────────────────────── File will be saved in
│                                 │  this directory.
│            Dirs/Drives          │
│            ▒▒▒▒▒▒▒▒▒▒▒          │  Use Dirs/Drives to
│            ▒▒▒▒▒▒▒▒▒▒▒ ◄────────── save to a different
│            ▒▒▒▒▒▒▒▒▒▒▒          │  directory or drive.
│                                 │
└─────────────────────────────────┘  Then choose <OK>.

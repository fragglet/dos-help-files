Print Dialog
Use to print all or part of the document you are editing.
┌──────── Print ────────┐
│ ▒ Selected Text Only ◄── Prints selected text.
│                       │
│ ▒ Complete Document ◄─── Prints the entire document.
└───────────────────────┘
 
If your cursor is in the Help window and you choose the
Print command, you have the following options for printing.
┌──────── Print ────────┐
│ ▒ Selected Text Only ◄── Prints selected help text.
│                       │
│ ▒ Current Window ◄────── Prints the entire help topic.
└───────────────────────┘
 
The MS-DOS Editor prints to the printer connected to the
LPT1 device on your computer.
 
Tip: Select text in your document or the Help window
     before you choose the Print command. Place the
     cursor at the beginning of the text you want to
     print, press Shift+direction keys until all the text
     is highlighted, then choose the Print command.

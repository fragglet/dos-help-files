Insert and Copy Keys
  ◄Getting Started►  ◄Keyboard►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Switch between insert and overstrike modes           Ins         Ctrl+V
──────────────────────────────────────────────────────────────────────────────
 Copy selected text to the Clipboard                  Ctrl+Ins
──────────────────────────────────────────────────────────────────────────────
 Delete selected text and copy it to the Clipboard    Shift+Del
 Delete current line and copy it to the Clipboard     Ctrl+Y
 Delete to end of line and copy it to the Clipboard               Ctrl+Q,Y
──────────────────────────────────────────────────────────────────────────────
 Paste the contents of the Clipboard                  Shift+Ins
 Insert a blank line below the cursor position        End+Enter
 Insert a blank line above the cursor position                    Home,Ctrl+N
──────────────────────────────────────────────────────────────────────────────
 Insert special characters                            Ctrl+P,Ctrl+key

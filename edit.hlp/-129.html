Document Too Large
The MS-DOS Editor cannot load or copy a portion of a
document that exceeds 32,700 lines in length.
 
Use another editor to divide your document into multiple,
smaller files. If this error occurs when using the Copy
command, try copying and pasting smaller portions of text.

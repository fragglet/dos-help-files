Open Dialog
Use to open an existing file for editing.
┌────────────── Open ─────────────┐
│                                 │  Type the name of the
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────── file here if it is in
│                                 │  the current directory
│ C:\DOS ◄────────────────────────── (shown here), or
│                                 │
│      Files          Dirs/Drives │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │  Select a file here,
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │  or a directory or
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │  drive if the file
│        ▲                 ▲      │  is located elsewhere.
│        └─────────────────└───────────┘
└─────────────────────────────────┘  Then choose <OK>.
 
You can also select a new drive and directory by typing
them in the File Name text box.
 
To browse the contents of other drives and directories,
type a wildcard filename (such as *.TXT) in the File
Name text box, and select a new drive or directory.

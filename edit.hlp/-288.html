New Command
Use to create a new document. A dialog box will appear if
you already have a document open. Then:
  ■ Choose <Yes> to save the existing document.
  ■ Choose <No> to close the document without saving it.
  ■ Choose <Cancel> to cancel the New command and return
    to the existing document.
 
Tip: Use the Open command to load an existing document.

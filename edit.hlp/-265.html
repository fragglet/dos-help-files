Insert Diskette Prompt
You have referred to your floppy disk drive as A: when it
is named B:, or B: when it is named A:.
 
Make sure the file you want this command to operate on is
in the floppy disk drive, then choose <OK>. The MS-DOS
Editor displays the new drive name for you.

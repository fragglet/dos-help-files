The MS-DOS Editor searches for the help file EDIT.HLP in
the directories you indicate in this dialog box. You don't
need to set the Help path if you have done so with the PATH
environment variable in DOS or if EDIT.HLP is in the
current directory.

Tip: If the MS-DOS Editor cannot find EDIT.HLP, check to
     make sure that you entered the correct directory name.

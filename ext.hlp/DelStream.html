  Syntax: void DelStream( PFILE pfFile, COL xStart, LINE yStart,
                                        COL xEnd,   LINE yEnd );
 
  See:    DelBox, DelLine, ◄Processing Text►
 
     DelStream deletes all characters in a specified stream from a file
     specified by <pfFile>. A stream is a sequence of text, including
     line breaks.
 
     These arguments define the stream to be deleted:
 
     Argument    Position
 
     <xStart>    First character
     <yStart>    First line
     <xEnd>      Position immediately after the last character
     <yEnd>      Last line

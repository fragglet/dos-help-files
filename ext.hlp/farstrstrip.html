  Syntax:  char far *farstrstrip( char far *String );
 
  Returns: A pointer to the modified string.
 
     The farstrstrip function replaces multiple consecutive white-space
     characters (spaces or tabs) in the argument string with a single
     space character. It also strips white space from the end of the
     string.
 
     The function returns a pointer to the modified string.

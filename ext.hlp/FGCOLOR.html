─────PWB Extensions─────────────────────────────────────────────────────────
 
  The following constants specify the color index for the corresponding
  PWB Color switch:
 
  Constant        Color Switch     Screen Element
 
  Background      Background       (Not visible)
  Hilite          Hilitectrl       Highlighted control item
  Greyed          Greyed           (Not visible)
  Enabled         Enabled          Available items in menus and dialogs
  Disabled        Disabled         Disabled items in menus and dialogs
  Alert           Alert            Message box
  DialogBox       DialogBox        Dialog box
  PushButton      PushButton       Button that is not pressed
  ButtonDown      ButtonDown       Button while it is down
  ListBox         ListBox          List box within a dialog box
  Scrollbar       Scrollbar        Gray area and arrows in scroll bar
  Elevator        Elevator         Scroll box
  MenuBox         MenuBox          Menu
  Menu            Menu             Menu bar
  MenuSelected    MenuSelected     Selected menu
  MenuHilite      MenuHilite       Highlighted character in menu
  MenuHiliteSel   MenuHiliteSel    Highlighted character in selected menu
  ItemHiliteSel   ItemHiliteSel    Highlighted character in selected item
  DialogAccel     DialogAccel      Dialog box accelerator
  DialogAccelBor  DialogAccelBor   Dialog box accelerator border
  Shadow          Shadow           Shadows
  FGCOLOR         Text             Text in a window
  HGCOLOR         Highlight        Highlighted text
  INFCOLOR        Info             Noise indicator on status bar
  SELCOLOR        Selection        Current selection
  WDCOLOR         Border           Window borders
  STACOLOR        Status           Indicator area of status bar
  BECOLOR         Builderr         Build message line in active window
  DTCOLOR         Desktop          Desktop
  PWBTCOLOR       Pwbwindowtext    PWB window text
  PWBBCOLOR       Pwbwindowborder  PWB window borders
  MSGCOLOR        Message          Message area of status bar
  LOCCOLOR        Location         Location indicator on status bar
 
  The following constants define the range of new colors that may be
  added by extensions:
 
  Constant          Description
 
  USERCOLORMIN      Beginning of extension colors
  USERCOLORMAX      End of extension colors

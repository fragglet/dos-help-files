farstradd
                                             ◄Upp► ◄Contents► ◄Index► ◄Back►
─────PWB Extensions─────────────────────────────────────────────────────────
 
  Syntax:  char far *farstradd( char far *Dest, char far *Source );
 
  Returns: A pointer to the modified string.
 
     The farstradd function concatenates the <Source> string onto the
     <Dest> string, inserting a space between the last character of
     <Source> and the first character of <Dest>. If <Source> is an empty
     string, farstradd has no effect.
 
     The function returns a pointer to <Dest>.

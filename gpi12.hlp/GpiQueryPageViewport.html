#define INCL_GPITRANSFORMS
 
BOOL GpiQueryPageViewport(hps, prclViewport)
HPS hps;                /* presentation-space handle         */
PRECTL prclViewport;    /* address of structure for viewport */
 
The GpiQueryPageViewport function retrieves the page viewport. The function
cannot be used if no device context is associated with the presentation
space.
 
Parameter     Description
────────────────────────────────────────────────────────────────────────────
 
hps           Identifies the presentation space.
 
prclViewport  Points to the RECTL structure that receives the page
              viewport.
 
Return Value
 
The return value is GPI_OK if the function is successful or GPI_ERROR if an
error occurred.
 
Errors
 
Use the WinGetLastError function to retrieve the error value, which may be
the following:
 
     PMERR_INV_HPS
     PMERR_PS_BUSY
 
See Also
 
GpiCreatePS, GpiSetPageViewport, RECTL

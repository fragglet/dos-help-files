LONG GpiQueryDrawingMode(hps)
HPS hps;    /* presentation-space handle */
 
The GpiQueryDrawingMode function returns the drawing mode, as set by
GpiSetDrawingMode.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
hps        Identifies the presentation space.
 
Return Value
 
The return value is the current drawing mode if the function is successful,
or DM_ERROR if an error occurred.
 
Errors
 
Use the WinGetLastError function to retrieve the error value, which may be
one of the following:
 
     PMERR_INV_HPS
     PMERR_INV_MICROPS_FUNCTION
     PMERR_PS_BUSY

LONG GpiQueryStopDraw(hps)
HPS hps;    /* presentation-space handle */
 
The GpiQueryStopDraw function returns the state of the stop-draw condition.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
hps        Identifies the presentation space.
 
Return Value
 
The return value is SDW_ON if the stop-draw condition is on or SDW_OFF if it
is not. The return value is SDW_ERROR if an error occurred.
 
Errors
 
Use the WinGetLastError function to retrieve the error value, which may be
the following:
 
     PMERR_INV_HPS
     PMERR_INV_MICROPS_FUNCTION
 
See Also
 
GpiSetStopDraw, WinGetLastError

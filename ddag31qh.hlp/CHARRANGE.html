CHARRANGE

typedef struct tagCHARRANGE {
    BYTE chFirst;
    BYTE chLast;
} CHARRANGE;
 
The CHARRANGE structure contains character codes for the first and last
characters in a range of characters.
 
Member   Description
────────────────────────────────────────────────────────────────────────────
 
chFirst  Specifies the character code of the first character.
 
chLast   Specifies the character code of the last character.

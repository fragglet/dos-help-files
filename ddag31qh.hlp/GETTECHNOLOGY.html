GETTECHNOLOGY
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
#define GETTECHNOLOGY 20 
short Control(lpDevice, GETTECHNOLOGY, NULL, lpTechnology)
LPPDEVICE lpDevice;
LPSTR lpTechnology;
 
The GETTECHNOLOGY escape retrieves the general technology type for a
printer. This allows an application to perform technology-specific actions.
 
Parameter     Description
────────────────────────────────────────────────────────────────────────────
 
lpDevice      Points to a PDEVICE structure specifying the destination
              device.
 
lpTechnology  Points to a buffer to which the driver copies a
              null-terminated string containing the printer technology type,
              such as "PostScript."
 
Return Value
 
The return value is 1 if the escape is successful. Otherwise, it is 0 if the
escape is not successful or not implemented.

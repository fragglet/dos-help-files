LPDCB getdcb(cid)
int cid;
 
The getdcb function retrieves the device-control block (DCB) for the
specified device. USER calls this function whenever an application calls the
GetCommState function (USER.202).
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
cid        Identifies the communications device.
 
Return Value
 
The return value is a pointer to a DCB structure.
 
Comments
 
The export ordinal for this function is 15.

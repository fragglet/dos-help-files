typedef struct tagPOINT {
    short   x;
    short   y;
} POINT;
 
The POINT structure contains the x- and y-coordinates of a point.
 
Member  Description
────────────────────────────────────────────────────────────────────────────
 
x       Specifies the x-coordinate of a point.
 
y       Specifies the y-coordinate of a point.

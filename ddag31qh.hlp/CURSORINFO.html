CURSORINFO

────────────────────────────────────────────────────────────────────────────
 
typedef struct tagCURSORINFO {
    short   dpXRate;
    short   dpYRate;
} CURSORINFO;
 
The CURSORINFO structure contains information about the cursor for the
display driver.
 
Member   Description
────────────────────────────────────────────────────────────────────────────
 
dpXRate  Specifies the horizontal mickey-to-pixel ratio. For the IBM PC and
         the Microsoft Mouse, this is 1.
 
dpYRate  Specifies the vertical mickey-to-pixel ratio. For the IBM PC and
         the Microsoft Mouse, this is 2.

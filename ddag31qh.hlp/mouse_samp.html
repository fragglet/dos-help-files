The source code for MOUSE.DRV included on the device driver disk is intended
as a sample of a driver for Microsoft compatible mouse products. It is not
equivalent to the retail version of the driver found on the Windows 3.1
installation disks.
 
This sample driver includes minor updates since Windows 3.0, and
demonstrates compatibility with tablet drivers for pen-based systems (see
"Windows for Pen Compatibility").

VOID Enable(VOID)
 
The Enable function enables the network driver. Windows calls Enable when it
loads the driver for the first time and when Windows resumes execution after
having been suspended, for example, after switching away from a non-Windows
application.
 
Return Value
 
This function has no return value.
 
Comments
 
The export ordinal for this function is 33.
 
If a network driver needs to hook an interrupt, it should make such a hook
during the Enable function and remove it during the Disable function. This
would prevent the driver's hook routine from being called when Windows was
swapped out to disk.

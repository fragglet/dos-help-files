OEMANSI struct
    oaLength    dw ?            ;number of bytes that follow (must be 258)
    oaCodePage  dw ?            ;code page
    oaTable     db 256 dup(?)   ;OEM/ANSI translation table
OEMANSI ends
 
The OEMANSI file format specifies information needed to translate OEM
character values, in a given code page, to their corresponding ANSI
character values.
 
Member      Description
────────────────────────────────────────────────────────────────────────────
 
 
oaLength    Specifies the size (in bytes) of the file except for the first
            two bytes. The size must be 258.
 
oaCodePage  Specifies the code page of the given translation table. It can
            be any one of the following values.
 
            Value  Description
            ────────────────────────────────────────────────────────────────
            437    Default (U.S.)
 
            860    Portugal
 
            863    French Canada
 
            865    Norway/Denmark
 
            850    International code page
 
oaTable     Specifies an array of translation values.

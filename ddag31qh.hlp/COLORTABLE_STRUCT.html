typedef struct tagCOLORTABLE_STRUCT {
    WORD Index;
    LONG rgb;
} COLORTABLE_STRUCT;
 
The COLORTABLE_STRUCT structure contains color information for an entry in
the color table.
 
Member  Description
────────────────────────────────────────────────────────────────────────────
 
Index   Specifies the color-table index. Color table entries start at zero
        for the first entry.
 
rgb     Specifies an RGB color value.

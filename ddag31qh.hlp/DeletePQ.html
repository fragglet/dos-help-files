int DeletePQ(hPQ)
HPQ hPQ;
 
The DeletePQ function deletes a priority queue.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
 
hPQ        Identifies a priority queue.
 
Return Value
 
The return value is TRUE if the queue is deleted. Otherwise, it is -1.

     DOS Extender Run-Time Error R6901
 
     This is a protected-mode application that requires DPMI (DOS
     Protected Mode Interface) services. Examples of hosts that
     provide these services include:
        - a DOS session under Windows 3.x in enhanced mode
        - Qualitas' 386MAX or BlueMAX version 6.x

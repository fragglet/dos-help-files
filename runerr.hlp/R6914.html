     DOS Extender Run-Time Error R6914
 
     cannot initialize device : 'filename'
 
     The DOS Extender could not find or initialize the given device
     driver.
 
     The Windows initialization file SYSTEM.INI must contain commands
     to load the driver. The commands must be in a section marked with
     the label [386Enh]. The required commands are as follows:
 
        ■ For Windows 3.0, use:
 
               device=[path\]vmcpd.386
               device=[path\]vpfd.386
 
          If the command device=*vmcpd appears, either delete it or
          change it to a comment by preceding it with a semicolon (;).
 
        ■ For later versions of Windows, use:
 
               device=*vmcpd
               device=[path\]vpfd.386

     DOS Extender Run-Time Error R6950
 
     VMCPD.386 has been superceded by a later version of Windows.
     Change the [386Enh] section in SYSTEM.INI by replacing the line
 
          device=[path\]vmcpd.386
 
     with the line
 
          device=*vmcpd
 
     The device driver is loaded by commands in the [386Enh] section in
     the SYSTEM.INI initialization file. For Windows versions later
     than 3.0, the following commands are required:
 
          device=*vmcpd
          device=[path\]vpfd.386
 
     For Windows 3.0, the following commands are required:
 
          device=[path\]vmcpd.386
          device=[path\]vpfd.386

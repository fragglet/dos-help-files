     C/FORTRAN Run-Time Error R6006
 
     invalid format on exec
 
     The file to be executed by one of the exec functions was not in
     the correct format for an executable file.
 
     This error occurs when a child process that was spawned by one of
     the exec library routines fails and the operating system cannot
     return control to the parent process.

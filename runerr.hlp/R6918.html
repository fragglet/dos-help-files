     DOS Extender Run-Time Error R6918
 
     DPMI, VCPI, or XMS host required
 
     The program is DOS-extended and requires extended or expanded
     memory, which must be provided by one of the following:
 
        ■ A DPMI server, such as the server provided by a DOS session
          within Windows in 386 enhanced mode. This server provides
          extended or expanded memory.
 
        ■ A VCPI server, such as Microsoft's EMM386.EXE. This server
          provides expanded memory.
 
        ■ An XMS server, such as Microsoft's HIMEM.SYS. This server
          provides extended memory.

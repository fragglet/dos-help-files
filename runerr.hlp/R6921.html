     DOS Extender Run-Time Error R6921
 
     no expanded memory under VCPI host
 
     The VCPI driver was loaded but no expanded memory was allocated.
 
     Possibly the CONFIG.SYS file contained a line such as:
 
          DEVICE=C:\OS\MM386.EXE NOEMS
 
     In this case, replace NOEMS with another argument. Specify a
     decimal number in kilobytes. You can specify RAM (or leave the
     argument blank) to allocate 256K (the default). If error R6907
     subsequently occurs, increase the specified value.
 
     It is recommended that you run this program under a DPMI host such
     as a DOS session within Windows in 386 enhanced mode.

     DOS Extender Run-Time Error R6908
 
     cannot find file : 'filename'
 
     The DOS Extender couldn't find the given file.
 
     If the given filename is MS32KRNL.DLL or MS32EM87.DLL, set the
     SYSTEM environment variable to the directory that contains this
     file.

     DOS Extender Run-Time Error R6920
 
     invalid XMS host
 
     The XMS server cannot provide any extended memory. This error
     occurs in the following environments:
 
        ■ Windows 3.0 standard mode
 
        ■ MS-DOS Shell in MS-DOS version 5.0

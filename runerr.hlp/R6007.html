     C/FORTRAN Run-Time Error R6007
 
     invalid environment on exec
 
     During a call to an exec function, the operating system found that
     the child process was given an invalid environment block.
 
     This error occurs when a child process that was spawned by one of
     the exec library routines fails and the operating system cannot
     return control to the parent process.

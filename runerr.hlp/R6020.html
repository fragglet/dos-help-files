     C/FORTRAN Run-Time Error R6020
 
     unexpected QuickWin error
 
     The program encountered an unexpected QuickWin error.
 
     One of the following may have occurred:
 
        ■ The program tried to access QuickWin, but the program was
          built without QuickWin libraries.
 
        ■ A QuickWin operation had an unrecoverable error.

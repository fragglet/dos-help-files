     Math error M6102
 
     denormal
 
     A very small floating-point number was generated, which may no
     longer be valid because of a loss of significance. Denormal
     floating-point exceptions are usually masked, causing them to be
     trapped and operated upon.
 
     This error terminates the program with exit code 130.

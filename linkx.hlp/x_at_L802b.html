─────/NOGROUP Option────────────────────────────────────────────────────────
 
     Syntax:  /NOG[ROUPASSOCIATION]
 
     The /NOGROUP option tells LINK to ignore group associations
     when assigning addresses to data and code items. It is provided
     primarily for compatibility with previous versions of the linker
     (2.02 and earlier) and early versions of Microsoft compilers.
     /NOGROUP is valid only for assembly-language programs that create
     DOS .EXE files.

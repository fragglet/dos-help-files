─────IMPLIB Options─────────────────────────────────────────────────────────
 
     Options names are not case sensitive and can be abbreviated to the
     shortest unique name.
 
     Syntax:  /H[ELP]
 
     Calls the QuickHelp utility. If IMPLIB cannot find the help file
     or QuickHelp, it displays a brief summary of IMPLIB command-line
     syntax.
 
 
     Syntax:  /NOI[GNORECASE]
 
     Turns on case sensitivity for entry names inside DLLs.
 
 
     Syntax:  /NOL[OGO]
 
     Suppresses the IMPLIB copyright message displayed when IMPLIB
     starts.
 
 
     Syntax:  /?
 
     Displays a brief summary of IMPLIB command-line syntax.

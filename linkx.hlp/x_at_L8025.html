/INFO Option
 ◄Example►                                   ◄Up► ◄Contents► ◄Index► ◄Back►
─────/INFO Option───────────────────────────────────────────────────────────
 
     Syntax:  /INF[ORMATION]
 
     The /INFO option displays information about the linking process,
     including the phase of linking and the names of the object files
     being linked. Use /INFO to determine the locations of the object
     files being linked, the number of segments, and the order in which
     the segments are linked.
 
     The output from this option is sent to standard output.

─────<movable> Attribute────────────────────────────────────────────────────
 
     Used by:  CODE, DATA, SEGMENTS
 
     Determines whether a segment can be moved in memory. For Windows
     only.
 
     Keyword     Purpose
 
     MOVABLE     Segment can be moved in memory. Alternate spelling is
                 MOVEABLE.
 
     FIXED       Default. Segment cannot be moved in memory.

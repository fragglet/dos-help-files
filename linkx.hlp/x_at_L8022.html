─────/HELP Option───────────────────────────────────────────────────────────
 
     Syntax:  /HE[LP]
 
     The /HELP option calls the QuickHelp utility. If LINK cannot find
     the help file or QuickHelp, it displays a brief summary of LINK
     command-line syntax and options.

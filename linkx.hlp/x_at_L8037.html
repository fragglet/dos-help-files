─────/Q Option──────────────────────────────────────────────────────────────
 
     Syntax:  /Q[UICKLIB]
 
     The /Q option tells LINK to produce a Quick library for use with
     Microsoft QuickBasic or early versions of Microsoft QuickC(R). The
     default extension is .QLB.
 
     For further information on the use of these libraries, consult the
     documentation for the appropriate Quick product.

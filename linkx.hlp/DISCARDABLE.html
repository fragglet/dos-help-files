<discard> Attribute
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────<discard> Attribute────────────────────────────────────────────────────
 
     Used by:  CODE, SEGMENTS
 
     Determines whether a code segment can be discarded from memory to
     fill a different memory request. If the discarded segment is
     accessed later, it is reloaded from disk. For Windows only.
 
     Keyword            Purpose
 
     DISCARDABLE        Segment can be discarded.
 
     NONDISCARDABLE     Default. Segment cannot be discarded.

─────/NOLOGO Option─────────────────────────────────────────────────────────
 
     Syntax:  /NOL[OGO]
 
     The /NOLOGO option suppresses the LINK copyright message displayed
     when LINK starts. This option has no effect if it is not the first
     option specified on the command line or in the LINK environment
     variable.

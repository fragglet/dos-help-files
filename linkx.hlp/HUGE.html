     LINK Reserved Words in Module-Definition Files
 
     The following words are reserved by LINK. These names cannot be
     used as arguments in module-definition statements.
 
     APPLOADER       HUGE *          NEWFILES           READONLY
     CLASS           IMPORTS         NODATA             READWRITE
     CODE            IMPURE          NOIOPL             REALMODE
     CONFORMING      INITGLOBAL      NONCONFORMING      RESIDENT
     DATA            INITINSTANCE    NONDISCARDABLE     RESIDENTNAME
     DESCRIPTION     IOPL            NONE               SEGMENTS
     DISCARDABLE     LIBRARY         NONSHARED          SHARED
     DOS4 *          LOADONCALL      NOTWINDOWCOMPAT    SINGLE
     EXECUTE-ONLY    LONGNAMES       OLD                STACKSIZE
     EXECUTEONLY     MAXVAL          OS2                STUB
     EXECUTEREAD     MOVABLE         PRELOAD            UNKNOWN
     EXETYPE         MOVEABLE        PRIVATELIB         WINDOWAPI
     EXPORTS         MULTIPLE        PROTMODE           WINDOWCOMPAT
     FIXED           NAME            PURE               WINDOWS
     HEAPSIZE
 
     * DOS4 and HUGE are obsolete but are still reserved by the linker.
 
     Additional Words Reserved by Linkers
 
     The following words are reserved for use by future or other
     versions of the linker and should be avoided.
 
     ALIAS           INVALID         PERMANENT
     CONTIGUOUS      MIXED1632       PHYSICAL
     DEV386          NONAME          RESIDENT
     DEVICE          NONPERMANENT    SWAPPABLE
     DYNAMIC         OBJECTS         TERMINSTANCE
     INCLUDE         ORDER           VIRTUAL

─────/NOPACKC Option────────────────────────────────────────────────────────
 
     Syntax:  /NOP[ACKCODE]
 
     The /NOPACKC option turns off code-segment packing. Code-segment
     packing is off by default. However, if an environment variable
     (such as LINK or FL) includes the /PACKC option to turn on
     code-segment packing, you can use /NOPACKC to override /PACKC.
     /NOPACKC overrides /PACKC if /PACKC is specified first.
     See: ◄/PACKC Option►

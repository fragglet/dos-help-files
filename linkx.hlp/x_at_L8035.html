─────/PAUSE Option──────────────────────────────────────────────────────────
 
     Syntax:  /PAU[SE]
 
     The /PAUSE option pauses the session before LINK writes the
     executable file or DLL to disk. /PAUSE is supplied for
     compatibility with machines that have floppy drives but no hard
     disk, to let you swap floppy disks before LINK writes the
     executable file.
 
     When you use /PAUSE, LINK displays the following message:
 
          About to generate .EXE file
          Change diskette in drive <letter> and press <ENTER>
 
     The <letter> is the current drive. LINK resumes processing when
     you press ENTER.
 
     Do not remove a disk containing the map file or the temporary
     file. If the disk you need to remove contains either of these
     files, terminate the LINK session. Then rearrange your files so
     that the files are on a disk that does not need to be removed.
     See: ◄LINK Temporary File►

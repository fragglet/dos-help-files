─────REALMODE Statement─────────────────────────────────────────────────────
 
     Syntax:  REALMODE
 
     The REALMODE statement specifies that the application runs only
     in real mode. This statement is supported for compatibility with
     existing module-definition files. Use EXETYPE instead.

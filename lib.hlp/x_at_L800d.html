/IGN Option

     Syntax:  /I[GNORECASE]
 
     Tells LIB to ignore case when comparing symbols. LIB does this by
     default. Use /IGN when combining a case-sensitive library with
     others that are not case sensitive to create a new library that is
     not case sensitive.

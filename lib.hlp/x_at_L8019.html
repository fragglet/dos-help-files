Replace Command: Example
 ◄Replace Command►                           ◄Up► ◄Contents► ◄Index► ◄Back►
─────LIB────────────────────────────────────────────────────────────────────
 
          LIB MAINLIB -+FLASH;
 
     This command replaces the module FLASH in the MAINLIB.LIB library
     with the contents of FLASH.OBJ from the current directory. The
     file FLASH.OBJ is not altered.

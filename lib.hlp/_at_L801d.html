─────LIB────────────────────────────────────────────────────────────────────
 
          LIB MAINLIB -*FLASH;
 
     This command moves the module FLASH from the MAINLIB.LIB library
     to a FLASH.OBJ file in the current directory. Upon completion,
     MAINLIB.LIB no longer contains the module FLASH.

     LIB Warning U4153
 
     'option' : 'value' : page size invalid; ignored
 
     The argument specified with the /PAGE option was not valid for
     that option. The value must be an integer power of 2 between 16
     and 32,768. LIB assumed an existing page size from a library that
     is being combined.

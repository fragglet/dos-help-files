          LIB MAINLIB +FLASH;
 
     This command adds the file FLASH.OBJ to the library MAINLIB.LIB.
 
          LIB MATH +TRIG.LIB;
 
     This command adds the contents of the library TRIG.LIB to the
     library MATH.LIB. The library TRIG.LIB is unchanged.

─────LIB────────────────────────────────────────────────────────────────────
 
     The Microsoft Library Manager (LIB) creates and maintains standard
     libraries. Standard libraries are collections of compiled or
     assembled object modules that provide useful routines and data
     to a program at link time.
 
     LIB works with several kinds of files:
 
        ■ Object files in Microsoft Relocatable Object-Module Format
          (OMF), based on the Intel 8086 OMF
 
        ■ Standard libraries in Microsoft library format
 
        ■ Import libraries created by IMPLIB
          See: ◄IMPLIB► (in Miscellaneous Utilities Help)
 
        ■ 80286 XENIX archives and Intel-style libraries
 
     Library files usually have a .LIB filename extension.

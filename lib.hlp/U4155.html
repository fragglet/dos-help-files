     LIB Warning U4155
 
     'modulename' : module not in library
 
     The given module specified with a command operator does not exist
     in the library.
 
     If the replacement command (-+) was specified, LIB added the file
     anyway. If the delete (-), copy (*), or move (-*) command was
     specified, LIB ignored the command.

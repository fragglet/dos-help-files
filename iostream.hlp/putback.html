  istream& putback( char ch );
 
  Parameter   Description
 
  <ch>        The character to put back; must be the character previously
              extracted.
 
  Remarks
 
  Puts a character back into the input stream. The putback function may
  fail and set the error state. If <ch> does not match the character that
  was previously extracted, then the result is undefined.

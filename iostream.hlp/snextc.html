  int snextc();
 
  Remarks
 
  First tests the get pointer, then returns EOF if it is already at the
  end of the get area. Otherwise, it moves the get pointer forward one
  character and returns the character that follows the new position. It
  returns EOF if the pointer has been moved to the end of the get area.

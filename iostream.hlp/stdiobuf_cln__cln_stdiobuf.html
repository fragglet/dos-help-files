  stdiobuf( FILE* fp );
 
  Parameter   Description
 
  <fp>        A standard I/O file pointer (can be obtained through an
              fopen or _fsopen call).
 
  Remarks
 
  Objects of class stdiobuf are constructed from open standard I/O files,
  including stdin, stdout, and stderr. The object is unbuffered by
  default.

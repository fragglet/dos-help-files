  Protected: void init( streambuf* psb );
 
  Parameter   Description
 
  <psb>       A pointer to an existing streambuf object.
 
  Remarks
 
  Associates an object of a streambuf-derived class with this stream and,
  if necessary, deletes a dynamically created stream buffer object that
  was previously associated. The init function is useful in derived
  classes in conjunction with the protected default istream, ostream, and
  iostream constructors. An ios-derived class constructor can thus
  construct and attach its own pretermined stream buffer object.

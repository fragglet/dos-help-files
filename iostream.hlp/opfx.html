  int opfx();
 
  Remarks
 
  This output prefix function is called before every insertion operation.
  If another ostream object is tied to this stream then the opfx function
  flushes that stream.
 
  Return Value
 
  If the ostream object's error state is not 0, opfx returns 0
  immediately; otherwise it returns a nonzero value.

  ios( streambuf* psb );
 
  Parameter   Description
 
  <psb>       A pointer to an existing streambuf object.
 
  Remarks
 
  Constructor for ios. You will seldom need to invoke this constructor
  except in derived classes. Generally, you will be deriving classes not
  from ios but from istream, ostream, and iostream.

  SMANIP( int ) setfill( int nFill );
 
  #include <iomanip.h>
 
  Parameter   Description
 
  <nFill>     The new fill character to be used as padding between
              fields.
 
  Remarks
 
  This parameterized manipulator sets the stream's fill character. The
  default is a space. This setting remains in effect until the next
  change.

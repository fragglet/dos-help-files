  long unsetf( long lFlags );
 
  Parameter   Description
 
  <lFlags>    Format flag bit values. See the ◄flags► member function for
              a list of format flags.
 
  Remarks
 
  Clears the format flags specified by 1s in <lFlags>. It returns a long
  that contains the previous value of all the flags.

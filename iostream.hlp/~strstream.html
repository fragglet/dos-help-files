  ~strstream();
 
  Remarks
 
  Destroys a strstream object and its associated strstreambuf object, thus
  releasing all internally allocated memory. If you used the void-argument
  constructor, then the internally allocated character buffer is released;
  otherwise, you must release it yourself.
 
  An internally allocated character buffer will not be released if it was
  previously frozen by calling rdbuf->freeze( 0 ).

  ~fstream();
 
  Remarks
 
  Flushes the buffer, then destroys an fstream object, along with its
  associated filebuf object. The file is closed only if it was opened by
  the constructor or by the open member function.
 
  The filebuf destructor releases the reserve buffer only if it was
  internally allocated.

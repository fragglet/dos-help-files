'This example uses the LCASE$ function to convert uppercase characters in a
'string to lowercase characters.
 
 
CLS                'Clear screen.
READ Word$
PRINT LCASE$(Word$);
DATA "THIS IS THE STRING in lower case."
 
'Sample Output
'
'this is the string in lower case.

'This example uses the ERASE statement with the $DYNAMIC and $STATIC
'metacommands:
 
REM $DYNAMIC
DIM A(100, 100)
'Deallocates array A.
ERASE A
'Redimension array A.
REDIM A(5, 5)
 
REM $STATIC
DIM B(50, 50)
'This sets all elements of B equal to zero.
'B still has the dimensions assigned by DIM.
ERASE B

'This example uses the SPACE$ function to create a string of spaces
'(ASCII 32). The program inserts the spaces on a printed line before
'the number of spaces that the string contains.
 
CLS            'Clear the screen.
FOR I=1 TO 5
   X$=SPACE$(I)
   PRINT X$;I
NEXT I
 
'Sample Output
'
' 1
'  2
'   3
'    4
'     5

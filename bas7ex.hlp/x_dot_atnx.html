'This example first finds the tangent of PI/4, then uses the ATN function
'to find the arctangent of the value. The result is PI/4.
 
CONST PI=3.141592653
PRINT ATN(TAN(PI/4.0)), PI/4.0
 
'Sample Output
'
' .78539816325  .78539816325

  Description:
 
     The routine starts at the left-hand end of source and target and
     IMPLIES the bits together, storing the result in target. The truth
     table for this is:
 
                source target result
                   0     0      1
                   0     1      1
                   1     0      0
                   1     1      1
 
     See also: ◄Introduction to logic routines►
 
  Parameters on Entry:
 
     source                  One of the data items to IMPLIES.
 
     target                  The other data item to IMPLIES.
 
     length                  The number of bytes of source and target
                             to IMPLIES. Positions in target beyond
                             this are unchanged.
 
  Parameters on Exit:
 
     target                  The result.

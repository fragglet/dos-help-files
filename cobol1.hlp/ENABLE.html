  Verb:         ENABLE
 
  Function:     Notifies MCS to allow data transfer between specified
                output queues and destinations for output, or between
                specified sources and input queues for input.
 
  Syntax:
 
    ►►─ENABLE─┬─INPUT─┬────────┬────┬─cd-name─┬────┬─KEY─┬─identifer-1─┬─►◄
              │       └TERMINAL┘    │         └WITH┘     └─literal-1───┘
              ├─I-O TERMINAL────────┤
              └─OUTPUT──────────────┘

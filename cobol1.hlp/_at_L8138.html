  Parameters on Entry:
 
     old-file-name           The file to rename. The name can contain
                             a path-name, and is terminated by a
                             space. If no path is given, the current
                             directory is assumed.
 
     new-file-name           The new name, terminated by a space. If
                             old-file-name contains a path-name, this
                             must contain the same path-name.
 
  Parameters on Exit:
 
     None

     The ASCII codes are listed, and beside each is shown the character
     it represents and the code that represents the same character in
     EBCDIC. The ASCII and EBCDIC codes are given in both decimal and
     hexadecimal. For convenience the table is divided into four parts,
     each with four columns; the "Other ranges" button takes you to
     other parts of the table.

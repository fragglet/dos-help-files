  Verb:         CHAIN
 
  Function:     Transfers control to another program in the run-unit,
                with no subsequent return of control.
 
  Syntax:
 
     ►►──CHAIN──┬─identifier-1─┬────►
                └─literal-1────┘
 
              ┌──────────────────────────────────────────────────┐
              │                    ┌───────────┐                 │
              ▼                    ▼           │                 │
     ►──USING─┬─┬──────────────┬─┬─identifier-2────────────────┬─┬─►
              │ └─BY REFERENCE─┘ │                ┌──────────┐ │ │
              │                  │                ▼          │ │ │
              │                  └─┬────────────┬─record-name──┘ │
              │                    └─ADDRESS OF─┘                │
              │             ┌────────────────────────┐           │
              │             ▼                        │           │
              └─BY CONTENT──┬─literal-2──────────────┬───────────┘
                            └─LENGTH OF identifier-3─┘
 
 
     ►─┬───────────┬──►◄
       └─END-CHAIN─┘

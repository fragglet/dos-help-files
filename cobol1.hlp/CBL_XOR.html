  Routine:      CBL_XOR
 
  Function:     Does a logical XOR between the bits of two
                data items.
 
  Syntax:       CALL "CBL_XOR"         USING     source
                                                 target
                                       BY VALUE  length
                                       RETURNING status-code
 
  Parameters:   source                 Any data item.
                target                 Any data item.
                length                 Numeric literal or PIC X(4) COMP-5.

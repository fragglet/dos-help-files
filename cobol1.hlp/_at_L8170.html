  Parameters on Entry:
 
     printer-no              Number of the printer to check; must be
                             0, 1, or 2. OS/2 ignores this, but
                             it still must contain a legal value.
 
  Parameters on Exit:
 
     printer-status          A combination of the following values:
 
                               128 (x"80") = not busy
                                64 (x"40") = acknowledge
                                32 (x"20") = out of paper
                                16 (x"10") = selected
                                 8 (x"08") = I/O error
                                 4 (x"04") = spare
                                 2 (x"02") = spare
                                 1 (x"01") = timeout

  Construct:    Reference Modification
 
  Function:     References part of a data item. The first item in the
                parentheses is the leftmost character position; the second
                item is the length.
 
  Syntax:
 
     ►►─┬─data-name-1──────────────────────────┬──►
        └─FUNCTION function-name-1─┬───────────┤
                                   │ ┌───────┐ │
                                   │ ▼       │ │
                                   └─argument──┘
 
 
     ►──(─┬─integer-1─────────────────────┬─►
          └─data-name-2──┬──────────────┬─┘
                         └─ ± integer-2─┘
 
                            ►──:─┬─integer-3─────────────────────┬─)───►◄
                                 └─data-name-3──┬──────────────┬─┘
                                                └─ ± integer-4─┘

  Routine:      CBL_OR
 
  Function:     Does a logical OR between the bits of two data items.
 
  Syntax:       CALL "CBL_OR"          USING     source
                                                 target
                                       BY VALUE  length
                                       RETURNING status-code
 
  Parameters:   source                 Any data item.
                target                 Any data item.
                length                 Numeric literal or PIC X(4) COMP-5.

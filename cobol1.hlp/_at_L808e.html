  Verb:         WRITE
 
  Function:     Sends a record to a file. For a sequential file this
                statement can also position lines vertically on the page.
 
  Syntax:       Which format?  ◄ADVANCING►  ◄POSITIONING►
                                INVALID KEY
 
     ►►───WRITE record-name──┬─────────────────────┬──►
                             └─FROM─┬─identifier-1─┤
                                    └─literal──────┘
 
     ►─┬──────────────────────────────────────┬───────►
       └─INVALID─┬───┬─imperative-statement-1─┘
                 └KEY┘
 
     ►─┬──────────────────────────────────────────┬───►
       └─NOT INVALID─┬───┬─imperative-statement-2─┘
                     └KEY┘
 
     ►─┬───────────┬──►◄
       └─END-WRITE─┘

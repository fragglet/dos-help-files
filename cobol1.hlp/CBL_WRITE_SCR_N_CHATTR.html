  Routine:      CBL_WRITE_SCR_N_CHATTR
 
  Function:     Writes a specified character and attribute to a
                string of positions on the screen.
 
  Syntax:       CALL "CBL_WRITE_SCR_N_CHATTR" USING     screen-position
                                                        character
                                                        attribute
                                                        fill-length
                                              RETURNING status-code

                  column-number          PIC X COMP-X.

                attribute              PIC X.

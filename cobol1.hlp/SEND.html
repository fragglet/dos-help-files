  Verb:         SEND
 
  Function:     Causes a message to be released to one or more output
                queues maintained by MCS.
 
  Syntax:
 
     ►►──SEND cd-name─┬───────────────────┬─┬────┬─┬─identifier-2─┬───►
                      └─FROM identifier-1─┘ └WITH┘ ├─ESI──────────┤
                                                   ├─EMI──────────┤

     ►─┬─────────────────────────────────────────────────────┬────────►
       └─┬─BEFORE─┬─ADVANCING──┬─┬─identifier-3──┬─┬─────┬─┬─┘
         └─AFTER──┘            │ └─integer───────┘ ├LINE─┤ │
                               │                   └LINES┘ │
                               └─┬─mnemonic-name─┬─────────┘
                                 └─PAGE──────────┘
 
     ►─┬──────────────────┬───────────────────────────────────────────►◄
       └─REPLACING─┬────┬─┘

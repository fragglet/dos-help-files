PC_FIND_DRIVES Details (↑ Library of Routines)
 ◄Key► ◄Summary►                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Description:
 
     This routine should only be used on DOS and OS/2.
 
     When called, the 26 least-signigicant bits in drive-info correspond
     to drives A: through Z:, with A: represented by bit 0 and Z:
     represented by bit 25. The corresponding bit is set for each drive
     that is present.
 
  Parameters on Entry:
 
     None
 
  Parameters on Exit:
 
     drive-info              All drives that are present.

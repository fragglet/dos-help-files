  Parameters on Entry:
 
     file-name1              The file to copy. The name can contain a
                             path-name and is terminated by a space. If
                             no path is given, the current directory
                             is assumed.
 
     file-name2              The name of the new file. The name can
                             contain a path-name and is terminated by a
                             space. If no path is given, the current
                             directory is assumed.
 
 
  Parameters on Exit:
 
     None.

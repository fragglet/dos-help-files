────────────────────────────────────────────────────────────────────────────
 
  Parameters on Entry:
 
     file-name               The file to look for. The name can contain
                             a path-name, and is terminated by a space. If
                             no path is given, the current directory is
                             assumed.
 
  Parameters on Exit:
 
     file-size               The size of the file in bytes.
 
     file-date               The date the file was created.
 
     file-time               The time the file was created.

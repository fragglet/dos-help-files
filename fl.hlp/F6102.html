     FORTRAN run-time error F6102
 
     REAL indefinite (uninitialized or previous error)
 
     An illegal argument was specified for an intrinsic function (for
     example, SQRT(-1)> or ASIN(2)). This error message does not always
     appear where the mistake was originally made. It may appear if the
     invalid value is used later in the program.

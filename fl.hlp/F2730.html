     FORTRAN compilation error F2730
 
     <name> : cannot modify active DO variable
 
     A DO variable cannot be modified within its range. For example,
     the following program fragments cause this error:
 
      DO 100 I = 1,10
      OPEN (33, IOSTAT = I)
  100 CONTINUE
 
      READ (*,*) (I, I = 1,10)

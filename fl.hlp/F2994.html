     FORTRAN compilation error F2994
 
     \ or $ : nonstandard edit descriptor in format
 
     The \ and $ edit descriptors are not part of standard
     FORTRAN 77 but are extensions to the language. This error occurs
     only if the /4Ys compiler option is used (or if the $STRICT
     metacommand is in effect).

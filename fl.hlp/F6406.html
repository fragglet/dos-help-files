     FORTRAN run-time error F6406
 
     truncation error : file closed
 
     This is a transient error. While the file was being truncated, it
     was temporarily closed.
 
     After a few minutes, the file should be run again. If this error
     message reappears, the file should be checked for characteristics,
     such as locking or permissions, that would prevent it from being
     accessed.

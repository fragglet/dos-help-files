     FORTRAN run-time error F6424
 
     invalid argument
 
     The system could not begin a thread of execution because an
     argument to the BEGINTHREAD routine is incorrect. This usually
     occurs when the stack argument does not start on a word boundary
     (even address), or the stack size argument is odd or zero, or the
     stack crosses a segment boundary.

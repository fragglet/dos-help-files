─────FORTRAN Language Option────────────────────────────────────────────────
 
     Use the FORTRAN Language option to enable and disable extensions
     to the strict FORTRAN 77 definition.
 
     Option            Effect
 
     FORTRAN 77        Disables all extensions
     MS Extensions     Enables all Microsoft FORTRAN extensions
                       (default)
     VMS               Enables only VAX VMS extensions
     SAA               Enables only SAA extensions
     IBM VS            Enables only IBM VS extensions

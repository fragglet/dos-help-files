     FORTRAN compilation error F2508
 
     <array>: array bound used array reference
 
     An expression having an array was used when declaring an
     adjustable-size array.
 
     Only simple variables in common blocks on the current subprogram's
     formal argument list are allowed as variables in the bound
     expression.

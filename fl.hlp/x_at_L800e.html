─────Windows DLL Options────────────────────────────────────────────────────
 
     The /Gw option is used for compiling Windows-compatible object
     code.
 
     Syntax: /Gw
 
     All routines that are exported from a Windows dynamic-link
     library must be compiled with the /Gw option.

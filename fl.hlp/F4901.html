     FORTRAN warning error F4901
 
     -4Y and -4N : both options used; -4Y assumed
 
     The $DEBUG, $DECLARE, $LIST, $STRICT, or $TRUNCATE metacommand was
     specified with both the /4Y and /4N compiler options. For example,
     $DEBUG was specified using both a /4Yb option and a /4Nb option.

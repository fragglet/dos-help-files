     FORTRAN fatal error F1908
 
     ASSIGN : too many format labels
 
     The program overflowed the assigned format-label table.
 
     This error probably occurred because an INTEGER*1 variable, which
     has a limit of 127 labels, was specified. To avoid this error,
     use an INTEGER*2 variable instead.

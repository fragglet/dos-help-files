─────Controlling Source-File Syntax─────────────────────────────────────────
 
     The following options disable and enable specific extensions to
     the FORTRAN language.
 
     Option         Effect
 
     /4{Y | N}f     Enables (Y) or disables (N) free-form format
 
     /4{Y | N}t     Enables (Y) or disables (N) truncation of
                    variable names
 
     /4{Y | N}6     Enables (Y) or disables (N) FORTRAN 66
                    style DO statements
 
     Example
 
          FL /4Yf /4Nt *.FOR
 
     The command line above enables free-form format and disables
     truncation of variable names when compiling and linking all
     source files in the current directory.

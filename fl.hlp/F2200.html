     FORTRAN compilation error F2200
 
     <subprogram> : formal argument <name> : CHARACTER*(*) cannot pass
                    by value
 
     Arguments that are passed by value must have a length that can be
     determined at run time. CHARACTER*(*) lengths are determined at
     run time.

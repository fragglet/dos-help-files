     FORTRAN run-time error F6414
 
     access not allowed
 
     This error is caused by one of the following occurrences:
 
          ■ The file name specified in an OPEN statement was a
            directory.
 
          ■ An OPEN statement tried to open a read-only file for
            writing.
 
          ■ The file's sharing mode does not allow the specified
            operations (DOS Versions 3.0 and later only).

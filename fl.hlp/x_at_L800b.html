─────MASM Cross-Reference File──────────────────────────────────────────────
 
     The /Fx option specifies a name for a Macro Assembler (MASM)
     cross-reference file.
 
          /Fx[filename]
 
     If <filename> is unspecified, the default output file name is
     sourcefilename.CRF.

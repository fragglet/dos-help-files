     FORTRAN compilation error F2320
 
     <name> : EQUIVALENCE : extends common block <commonblock> forward
 
     In an EQUIVALENCE statement, it is illegal to increase the size
     of a named common block by adding memory elements before the
     beginning common block declared in the COMMON statement.

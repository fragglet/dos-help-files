     FORTRAN compilation error F2321
 
     <name1>, <name2> : EQUIVALENCE : conflicting offsets
 
     The processing of an EQUIVALENCE statement detected two items
     that should have had the same offsets but did not. Inconsistent use
     of EQUIVALENCE statements caused this problem.

     FORTRAN compilation error F2339
 
     <array> : adjustable-size array not in subprogram
 
     The specified adjustable-size array was declared in a subprogram
     declared with a PROGRAM or BLOCK DATA statement.
 
     An adjustable-size array is legal only in an ENTRY, FUNCTION, or
     SUBROUTINE statement in a subprogram.

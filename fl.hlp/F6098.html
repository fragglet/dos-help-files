     FORTRAN run-time error F6098
 
     label not found in assigned GOTO list
 
     The label assigned to the integer-variable name was not specified
     in the label list of the assigned GOTO statement. This message
     appears only if the /4Yb option is used in compiling (or the
     $DEBUG metacommand is in effect).

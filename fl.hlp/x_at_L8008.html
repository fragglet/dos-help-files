─────Getting Help with FL Options (/HELP)───────────────────────────────────
 
     Syntax: /HELP
             /help
 
     Causes FL to invoke the QuickHelp program (QH.EXE). QuickHelp
     displays the contents of the FL help file (FL.HLP).
 
     If QH.EXE or FL.HLP is not available, then FL displays the
     contents of the file FL.MSG.
 
     QH.EXE and FL.HLP must be in the current directory or in a
     directory named in the PATH environment variable.

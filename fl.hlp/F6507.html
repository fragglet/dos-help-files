     FORTRAN run-time error F6507
 
     T or F expected in LOGICAL read
 
     The wrong format was used for the input field for logical data.
 
     The input field for logical data consists of optional blanks,
     followed by an optional decimal point, followed by a T for true
     or F for false. The T or F may be followed by additional
     characters in the field, so that .TRUE. and .FALSE. are
     acceptable input forms.

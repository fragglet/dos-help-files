     FORTRAN run-time error F6099
 
     INTEGER overflow
 
     This error occurs whenever integer arithmetic results in overflow,
     or when assignment to an integer is out of range. This message
     appears only if the /4Yb option is used in compiling (or the
     $DEBUG metacommand is in effect).

─────FORTRAN Compiler Options───────────────────────────────────────────────
 
     Use the FORTRAN Compiler Options dialog box to set compiler
     options for the project.
 
     Compiler Option                  Description
 
     ◄Memory Model►                   Set memory model
     ◄Warn Level►                     Set warning threshold
     ◄FORTRAN Libraries►              Select run-time libraries
     ◄FORTRAN Language►               Language extensions
     ◄Defines►                        Define symbols
     ◄Additional Options►             Set additional FL options
     ◄Set Debug/Release Options►      Set specific build options
     ◄Show Debug/Release Options►     Show FL command line

     FORTRAN run-time error F6096
 
     array subscript expression out of range
 
     An expression used to index an array was smaller than the lower
     dimension bound or larger than the upper dimension bound. This
     message appears only if the /4Yb option is used in compiling (or
     the $DEBUG metacommand is in effect).

     FORTRAN compilation error F2805
 
     label <number> : redefined in program unit
 
     The specified label appeared earlier in the subprogram.
 
     Labels may not be defined more than once within a single subprogram
     unit. This error may also occur if a DO loop references a
     previously defined label.

     FORTRAN compilation error F2804
 
     <name> : ASSIGN : too many INTEGER*1 variables
 
     Only the first 127 ASSIGN statements may use variables of type
     INTEGER*1 in a subprogram. This is caused by the storage
     limitations of INTEGER*1 items.

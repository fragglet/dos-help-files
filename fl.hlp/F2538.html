     FORTRAN compilation error F2538
 
     <array> : subscript <number> : not integer
 
     When the /4Ys compiler option is used (or the $STRICT metacommand
     is in effect), a subscripting expression used in the specified
     array must be of type INTEGER. Otherwise, it must be an arithmetic
     type that is truncated to INTEGER by an implicit use of the INT
     intrinsic function.

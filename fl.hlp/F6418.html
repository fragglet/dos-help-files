     FORTRAN run-time error F6418
 
     too many units connected
 
     The program exceeded the limit on the number of open files per
     program.
 
     Close any unnecessary files. See the FILES= command in the
     Microsoft MS-DOS User's Guide and User's Reference for more
     information.

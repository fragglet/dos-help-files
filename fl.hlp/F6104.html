     FORTRAN run-time error F6104
 
     REAL math overflow
 
     A real value was too large. Floating-point overflows in either
     direct or emulated mode generate NAN (Not-A-Number) exceptions,
     which appear in the output field as asterisks (*) or the letters
     NAN.

─────Source Browser Options─────────────────────────────────────────────────
 
     These options create files that contain symbolic information for
     the Source Browser facility.
 
     Option   Default Name         File type
 
     /Fr      sourcefilename.SBR   PWB Source Browser database
     /FR      sourcefilename.SBR   Extended PWB Source Browser database
 
     See: ◄PWB Source Browser Menu Commands► (in PWB help)
 
     In general, the space savings realized by using the /Fr option
     are minimal. Therefore, in most circumstances you will want to
     generate extended Source Browser information using the /FR
     option.
 
     Extended information means that you can view the fully expanded
     version of macros in the Source Browser as well as get
     information about parameters and local variables.

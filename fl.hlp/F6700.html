     FORTRAN run-time error F6700
 
     heap space limit exceeded
 
     The program tried to open too many files at once. A file control
     block (FCB) must be allocated from the heap for each file opened,
     but no more heap space was available.
 
     Try compiling the program with the /Gt option to free up additional
     near heap space.

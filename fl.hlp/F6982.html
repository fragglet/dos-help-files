     FORTRAN run-time error F6982
 
     positive integer expected in format
 
     A zero or negative integer value was used in a format.
 
     Negative integer values can appear only with the P edit
     descriptor. Integer values of 0 can appear only in the <d> and <m>
     fields of numeric edit descriptors.

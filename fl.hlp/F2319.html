     FORTRAN compilation error F2319
 
     <name> : EQUIVALENCE : extends blank common block forward
 
     In an EQUIVALENCE statement, it is illegal to increase the size
     of a blank common block by adding memory elements before the
     beginning common block declared in the COMMON statement.

     FORTRAN compilation error F2404
 
     <name> : can only initialize common block in BLOCK DATA subprogram
 
     An attempt was made to initialize the specified item named in a
     common block in a DATA statement.
 
     Items in named common blocks can be initialized only in BLOCK
     DATA subprograms.

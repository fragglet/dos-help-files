     FORTRAN run-time error F6310
 
     illegal LOCKMODE value
 
     An illegal value was used with the LOCKMODE= option.
 
     LOCKMODE= accepts the values 'LOCK', 'NBLCK', 'NBRLCK', 'RLCK',
     and 'UNLCK'.

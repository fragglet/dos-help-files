     FORTRAN compilation error F2995
 
     Z : nonstandard edit descriptor in format
 
     The Z edit descriptor is not part of standard FORTRAN 77 but is
     an extension to the language. This error occurs only if the /4Ys
     compiler option is used (or if the $STRICT metacommand is in
     effect).

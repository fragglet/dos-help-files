     FORTRAN run-time error F6992
 
     format not set by ASSIGN
 
     The format specifier in a READ, WRITE, or PRINT statement was an
     integer variable, but an ASSIGN statement did not properly assign
     it the statement label of a FORMAT statement in the same program
     unit.

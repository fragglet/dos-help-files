     FORTRAN run-time error F6309
 
     illegal SHARE value
 
     An illegal value was used with the SHARE= option.
 
     SHARE= accepts the values 'COMPAT', 'DENYRW', 'DENYWR', 'DENYRD',
     and 'DENYNONE'.

─────FORTRAN Compiler Optimization Options──────────────────────────────────
 
     Use the Optimization options to control the way the compiler
     optimizes your program.
 
                                                       FL
     Option               Effect                       Equivalent
 
     Optimize for Time    Execution is faster.         ◄/Os►
 
     Optimize for Space   Code is smaller.             ◄/Ot►
 
     Loop Code            Loops execute faster.        ◄/Ol►
     Optimization
 
     Improve Float        Changes compiler's default   ◄/Op►
     Consistency          handling of floating-point
                          values. Intermediate
                          results are placed in
                          variables instead of
                          machine registers.

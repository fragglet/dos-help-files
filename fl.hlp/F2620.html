     FORTRAN compilation error F2620
 
     <subprogram> : formal argument <name> : subprogram mismatch
 
     The type of the formal argument to the subprogram was not the same
     as the actual argument.
 
     Both the formal and the actual argument must be subroutines or
     functions.

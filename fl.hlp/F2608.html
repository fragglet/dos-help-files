     FORTRAN compilation error F2608
 
     <subprogram> : formal argument <name> : Hollerith illegal with
                    CHARACTER
 
     Hollerith constants can only be used with items of type INTEGER,
     LOGICAL, and REAL in DATA statements and subprogram references.

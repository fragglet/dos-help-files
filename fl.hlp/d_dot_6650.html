─────FORTRAN Compiler Show Debug/Release Options────────────────────────────
 
     Select Show Debug Options or Show Release Options to see the
     currently selected options as they appear on the FL command line.
 
     NOTE: The displayed options do not include any options specified
     in the FL environment variable.

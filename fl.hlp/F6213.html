     FORTRAN run-time error F6213
 
     TITLE illegal with non-window unit.
 
     A title can only be specified for units opened as windows (opened
     with FILE = 'USER' and linked with the QuickWin library).
 
     Either reopen the specified unit as a window, or remove the TITLE
     parameter from the OPEN statement.

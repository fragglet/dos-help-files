     FORTRAN run-time error F6407
 
     terminal buffer overflow
 
     More than 131 characters were input to a record of a unit connected
     to the terminal (keyboard). Note that the operating system may
     impose additional limits on the number of characters that can be
     input to the terminal in a single record.

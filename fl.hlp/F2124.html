     FORTRAN compilation error F2124
 
     CODE GENERATION ERROR Contact Microsoft Technical Support
 
     The compiler could not generate code for an expression. Usually
     this error occurs with a complicated expression.
 
     Try rearranging the expression. Please report this error to
     Microsoft Corporation using the Product Assistance Request
     form at the back of one of your manuals.

     FORTRAN compilation error F2731
 
     <BI>iooption> : noncharacter array nonstandard
 
     If the /4Ys compiler option is used in compiling (or the
     $STRICT metacommand is in effect), standard forms of the
     language must be used. In these cases, only character variables,
     arrays, array elements, and substrings are legal as I/O specifiers.

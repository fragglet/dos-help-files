Simple Types

  Description
 
    Simple types are the basic QuickPascal types. They are indivisible
    units. Simple types can be used to declare variables directly or as
    the building blocks of structured types. There are three categories
    of simple types:
 
      ═══════
      String
      Ordinal
      Real
 
    Choose any type for help.

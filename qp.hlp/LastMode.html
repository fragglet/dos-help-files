LastMode
  ◄Crt Unit► ◄Example► ◄Contents► ◄Index►                         ◄Back►
 
  VAR
      LastMode : Word;    { current text mode }
 
  Unit:     Crt
 
  Context:  TextMode
 
  Description
 
    LastMode holds the current text mode. At program start-up, LastMode
    is set to the video mode in effect at that time. Thereafter, each
    time TextMode is called, the setting passed to TextMode is stored
    in LastMode.

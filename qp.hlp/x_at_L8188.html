Round Function
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Argument
 
    number     Real expression
 
  Returns
 
    LongInt    Whole number nearest to the argument value
 
  Description
 
    The Round function takes a real value and returns a value rounded
    to the nearest whole number to the given real number.
 
    Fractional parts of exactly .5 or -.5 are rounded to the whole
    number with the next larger absolute value.
 
    If the return value exceeds the range of a LongInt, a run-time
    error occurs.

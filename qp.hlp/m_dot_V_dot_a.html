Use the Cascade command to arrange all displayed windows
  in a cascading style. All open windows are moved and
  sized so that they overlap and cascade from the top left
  to the bottom right of the screen.
 
  Cascading Windows:
 
      ┌───────────────────────┐
      │ ┌──1────────────────┐ │
      │ │┌──2───────────────┤ │
      │ ││┌──3──────────────┤ │
      │ │││┌──4─────────────┤ │
      │ ││││                │ │
      │ └┴┴┴────────────────┘ │
      └───────────────────────┘

Disk full
  There isn't enough room on your disk to save the file
  you were editing.
 
  Try one of the following:
 
    ■ Save the file to a different disk drive.
 
    ■ Open the File menu, choose the DOS Shell command,
      and delete files you don't need from the disk. Then
      type "exit" to return to QuickPascal, and choose the
      Save command again.

ClrEol Procedure
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Description
 
    The ClrEol procedure clears a line from the current cursor location
    to the end of the line. The cursor remains in its original position
    and rests on a blank character.
 
    ClrEol clears the line by setting all characters to blanks from the
    cursor position to the right-hand border of the current window. The
    attributes of the blanks depend on the current setting for text
    background.
 
    The current window may be the default window or one defined with
    the Window procedure. The TextBackground procedure sets the
    background attribute.

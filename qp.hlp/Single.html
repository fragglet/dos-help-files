Single Type
  ◄Real Types► ◄Contents► ◄Index►                                 ◄Back►
 
  Description
 
    A Single is a floating-point type that occupies four bytes. Use to
    store a floating-point number with 7 to 8 significant digits.
 
  Range:  1.5E-45 through 3.4E+48

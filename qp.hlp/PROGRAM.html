PROGRAM Keyword
  ◄Keywords► ◄Contents► ◄Index►                                   ◄Back►
 
  [ PROGRAM <identifier>[ ( <program parameters> ) ] ]
 
  Description
 
    The keyword PROGRAM indicates the start of a program. PROGRAM,
    along with its associated identifier and optional parameters, is
    called a program header. The use of a program header is optional.

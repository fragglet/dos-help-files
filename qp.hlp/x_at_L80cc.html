ArcTan Function
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Argument
 
    tangent        Real expression representing the tangent of an angle
 
  Returns
 
    <real type>    Angle in radians
 
  Description
 
    The ArcTan function takes a real argument representing a tangent
    and returns the arctangent of the argument.
 
    An arctangent is the inverse function to a tangent. If y is the
    tangent of the angle Θ, then Θ is the arctangent of tangent y.

STRING Type
  ◄Keywords► ◄Contents► ◄Index►                                   ◄Back►
 
  STRING[ [ length ] ]
 
  Description
 
    STRING is a simple type that declares a string constant, type, or
    variable. The first position in a string, [0], contains its
    declared length. The maximum size of a string is 255 characters. If
    the optional square brackets and size are omitted from its
    declaration, the default size of a string is 255 characters.
 
  See also:  CSTRING, PACKED

This File does not exist, Create?
  The entered file name is not the name of an existing
  file. Do You want to create a new file with that name?
 
  Choose <Yes> to open a new window and create a file
  with the given name.
 
  Choose <No> to cancel the command. You can then choose
  the command again and enter a different file name.

Help Menu
  Use the Help menu to obtain overview information about
  help and as an entrance into the help system. In
  addition, if the cursor is on a keyword in a file, the
  Topic command will bring up help on that keyword.
 
  ┌────────────────────────┐
  │ Index                  │ ■ Alphabetic list of topics
  │ Contents               │ ■ Topical table of contents
  │ Topic:              F1 │ ■ Help on specific topic
  │ Help On Help  Shift+F1 │ ■ How to use help
  └────────────────────────┘
 
  TIP: While a help file is displayed, use Find and Repeat
  Last Find (on the Search menu) to find particular
  information in the file.

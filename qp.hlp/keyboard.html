  QuickPascal Keys
 
    The QuickPascal editor uses common keystroke combinations to
    manipulate text. These keystrokes are similar to those found in
    other Microsoft products. Keystrokes that correspond to WordStar(R)
    key commands are also available.
 
    ═════════════════════     ════════════════════
    Cursor movement keys      Insert and copy keys
    Debugging keys            Search keys
    Delete keys               Text scrolling keys
    Exit-and-save keys        Text selection keys
    Function keys             View-changing keys
    Help keys
 
    Choose any item for help.

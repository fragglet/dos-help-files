Use the commands in the Debug menu to debug your
  programs.
 
  ┌──────────────────────┐
  │ Calls...             │ ■ Lists chain of calls *
  │ Set Breakpoint... F9 │ ■ Sets breakpoints
  │ Edit Breakpoints...  │ ■ Edits existing breakpoints *
  ├──────────────────────┤
  │ Watch Value...       │ ■ Monitors value of a variable
  │ Modify Value...      │ ■ Changes value of a variable *
  └──────────────────────┘
 
  *Items marked by an asterisk appear only with full
   menus.

VAR Keyword
  ◄Keywords► ◄Contents► ◄Index►                                   ◄Back►
 
  VAR <identifier> : <type>;
                 .
                 .
        <identifier> : <type>;
 
  Description
 
    VAR declares the section of a function, procedure, or program that
    defines variables. VAR is also used to indicate a variable
    parameter in the parameter list in a function or procedure. The
    type can be either predefined or user-defined types.
 
  See also:  CONST, LABEL, TYPE

InsLine Procedure
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Description
 
    The InsLine procedure inserts a blank line at the row location of
    the cursor in the window. The entire line on which the cursor rests
    is pushed down. All lines below the new line scroll down. Any text
    on the bottom line scrolls off and is lost. The cursor remains in
    its original position.
 
    Insline sets all characters on the new line to blanks with the
    current background color.
 
    The new line extends from border to border within the current
    window. Insline does not affect text outside of the current window.
 
    The current window may be the default window or one defined with
    the Window procedure. The TextBackground procedure sets the
    background attribute.

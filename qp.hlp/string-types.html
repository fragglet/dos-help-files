String Types
  ◄Simple Types► ◄Contents► ◄Index►                               ◄Back►
 
  Description
 
    Strings are sequences of characters. Because strings are stored
    in packed form, they are an effective way to represent any data
    that will not require numeric calculations. For example, names,
    addresses, zip codes, and lines of text are efficiently stored as
    strings. QuickPascal provides the following two basic string types:
 
      ═══════
      STRING
      CSTRING
 
    Choose either type for help.

Debug Window Command
 
  Use the Debug command to open the window that displays
  watch expressions. In this window you may
 
    ■ Add a watch expression either by using the Watch
      Value command in the Debug menu or by typing the
      expression on a new line
 
    ■ Disable a watch expression making the line a
      comment line
 
  Regular editing commands can be used in the Debug
  window.

Stack-Checking Directive
  ◄Compiler Directives► ◄Contents► ◄Index►                        ◄Back►
 
  {$S<+│->}
 
  Description
 
    Switches on or off the generation of code that checks for stack
    overflow. The default is {$S+}.
 
    {$S+}  Turns on stack-overflow checking for each subsequent
           procedure and function call. If there is not enough stack
           space for a call, the program is terminated with a
           run-time error message.
 
    {$S-}  Turns off {$S+}.

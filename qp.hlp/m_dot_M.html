Make Menu
 
  Use the Make menu to compile QuickPascal programs and to
  set the main file.
 
    ┌────────────────────────┐
    │ Compile File       XXX │ ■ Compiles file named XXX *
    │ Build Main File    XXX │ ■ Compiles changed files
    │ Rebuild Main File  XXX │ ■ Compiles all files *
    ├────────────────────────┤
    │ Set Main File...       │ ■ Sets the main file *
    │ Clear Main File        │ ■ Clears the main file *
    └────────────────────────┘
 
    * Items marked by an asterisk appear only with
      full menus.

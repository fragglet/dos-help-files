Cannot execute compiler
  QuickPascal was unable to invoke the QPL.COM compiler.
  Check the following conditions:
 
    ■ Make sure the compiler is present and in the QP
      directory specified in the Environment dialog box
      under the Options menu.
 
    ■ Make sure that you are not running QuickPascal in a
      shell──from the DOS Shell command in the File menu.
      If you are, there might not be enough memory for the
      compiler.

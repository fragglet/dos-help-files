Var-String Checking Directive
  ◄Compiler Directives► ◄Contents► ◄Index►                        ◄Back►
 
  {$V<+│->}
 
  Description
 
    Switches between two settings that control how strictly VAR
    parameters of type STRING are type checked. The default is {$V+}.
 
    {$V+}  Checks that actual and formal parameters are of the
           identical STRING type.
 
    {$V-}  Checks that actual and formal parameters are strings.
           However, their declared lengths do not have to match.

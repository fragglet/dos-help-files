Use the Tile command to arrange all displayed windows
  in a tiled fashion. All windows are moved and sized so
  that the entire screen is used without any windows
  overlapping.
 
      ┌─────────────────────────┐
      │ ┌─────────┐ ┌─────────┐ │
      │ │ 1       │ │ 2       │ │
      │ │         │ │         │ │
      │ └─────────┘ └─────────┘ │
      │ ┌─────────┐ ┌─────────┐ │
      │ │ 3       │ │ 4       │ │
      │ │         │ │         │ │
      │ └─────────┘ └─────────┘ │
      └─────────────────────────┘

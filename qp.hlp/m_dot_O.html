Options Menu
 
  The Options menu gives you control over:
 
  ┌───────────────┐
  │ Display...    │ ■ Color, line, and tab options
  │ Mouse...      │ ■ Right mouse button options *
  │ Compiler...   │ ■ Compiler and memory options *
  │ Run/Debug...  │ ■ Run and debug options *
  │ Environment...│ ■ QuickPascal file paths *
  │ Short Menus   │ ■ Use of full or short menus
  └───────────────┘
 
  * Items marked by an asterisk appear only with full
    menus.

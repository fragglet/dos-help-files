Use the View menu to select and change windows.
  ┌────────────────────┐
  │ Close      Ctrl+F4 │ ■ Closes a window
  │ Move       Ctrl+F7 │ ■ Moves a window to a new spot
  │ Size       Ctrl+F8 │ ■ Changes a window's size
  │ Duplicate          │ ■ Duplicates a file window *
  ├────────────────────┤
  │ Cascade            │ ■ Layers all windows *
  │ Tile               │ ■ Tiles all windows *
  ├────────────────────┤
  │ Output Screen   F4 │ ■ Displays the program output
  │ Debug Window...    │ ■ Displays the Debug window *
  │ Help Window        │ ■ Displays the Help window *
  └────────────────────┘
  * Items with an asterisk appear only with full menus.

Copy Command (2 of 2)
 
  Use the Copy command to copy selected (highlighted)
  text from the active window to the Clipboard. The
  shortcut key is CTRL+INS. Use the SHIFT+ARROW keys
  or the mouse to select text in the active window.
 
  Once a block of text is in the Clipboard, you can choose
  Paste or press SHIFT+INS to insert the copied text into
  an active window.
 
  TIP: The copied text is still in both the active
  window and the Clipboard.
  To use the Copy and Paste commands together
 
1. Select text     2. Choose Copy     3. Move cursor to
   with mouse or      or press           insert point;
   SHIFT+ARROW        CTRL+INS           choose Paste

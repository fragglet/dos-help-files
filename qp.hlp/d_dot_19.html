This File has Changed
 
  This dialog box occurs once for each open window that
  displays a file that has changed since the last time
  the file was saved.
 
  ■ Choose <Yes> to save the file in the current window.
 
  ■ Choose <No> to lose the changed information.
 
  ■ Choose <Cancel> to cancel the command.

FileMode
  ◄System Unit► ◄Contents► ◄Index►                                ◄Back►
 
  CONST
      FileMode : Byte = 2;    { file access mode }
 
  Unit:  System (used automatically)
 
  Description
 
    FileMode holds the access code passed to DOS when the Reset
    procedure is called for a typed or untyped file.
 
    The possible modes depend on the DOS version. The basic modes are
 
      Mode    Access
      ════    ══════════
 
      0       Read only
      1       Write only
      2       Read/Write
 
    The default is 2.
 
    Assign a new value to FileMode to cause subsequent calls to Reset
    to use the new mode.

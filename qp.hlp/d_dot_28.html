Compiler Options Modified
 
  The compiler options have changed since the last
  debugging session.
 
  ■ Choose <Yes> to recompile, using the new compiler
    options, and continue with the debugging session.
    Note that unless you use the <Save> button in the
    Compiler Option's dialog box, the new options are
    not saved when you exit QuickPascal.
 
  ■ Choose <No> to continue the debugging session without
    the new options or a recompile.
 
  ■ Choose <Cancel> to cancel the command and stop the
    debugging session.

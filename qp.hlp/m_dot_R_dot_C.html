Continue To Cursor Command
 
  Use the Continue To Cursor command to execute your
  program as far as the start of the line that currently
  has the cursor. The shortcut key is F7.
 
  TIP: If you use this command often, go to the Mouse
  command in the Options menu and assign Continue To
  Cursor to the Right mouse button. The help key will
  change to SHIFT+Right mouse button.

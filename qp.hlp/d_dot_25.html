Not Enough Memory to Debug, Execute Anyway?
 
  There isn't enough available memory to load debugging
  information.
 
  ■ Choose <Yes> to either execute the program or continue
    debugging. However, you cannot examine variables.
 
  ■ Choose <No> to cancel the command.

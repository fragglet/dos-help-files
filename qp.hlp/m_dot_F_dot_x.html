Exit Command
 
  Use the Exit command to quit when you have finished
  using QuickPascal.
 
  A dialog box appears if you have not saved a new or
  changed file. Your options are to
 
    ■ Choose <Yes> to save your changes
    ■ Choose <No> to discard your changes
    ■ Choose <Cancel> or press Esc to return to
      QuickPascal where you can look at the program
      in memory before deciding what to do

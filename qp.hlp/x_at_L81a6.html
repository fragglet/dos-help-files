Sqr Function
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Argument
 
    number         Integer or real expression
 
  Returns
 
    <same type>    Square of the number
 
  Description
 
    The Sqr function takes an integer or real expression and returns
    the square of the number. The value returned is equal to number
    multiplied by number. The value has the same type as the number.

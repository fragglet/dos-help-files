Boolean Evaluation Directive
  ◄Compiler Directives► ◄Contents► ◄Index►                        ◄Back►
 
  {$B<+│->}
 
  Description
 
    Switches between two settings that control evaluation of
    expressions containing Boolean AND and OR operators. The default
    is {$B-}.
 
    {$B+}  Guarantees the evaluation of every operand in a Boolean
           expression
 
    {$B-}  Forces short-circuit Boolean evaluation--expression
           evaluation is stopped as soon as the result is known

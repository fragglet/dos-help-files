ON KEY(n%) GOSUB {linenumber | linelabel}
 
    ■ n%              An integer expression; the number of a function
                      key, direction key, or user-defined key.
    ■ linenumber or   The first line of the event-handling routine to
      linelabel       branch to when a function key, direction key,
                      or user-defined key is pressed.

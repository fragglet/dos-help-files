WRITE [expressionlist]
 
    ■ expressionlist    Specifies one or more values to be written.
 
    ■ The values are written on the screen separated by commas, with
      double quotation marks around strings and no spaces around numbers.
    ■ If expressionlist is omitted, a blank line is written.

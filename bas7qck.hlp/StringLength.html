length = StringLength(string-descriptor%);
 
    ■ The preceding syntax is for the C language. For more information,
      see Details.
 
    ■ string-descriptor%   The near address of a string descriptor within
                           a non-BASIC routine.
 
Returns
    The number of characters in the variable-length string defined by
    string-descriptor%.

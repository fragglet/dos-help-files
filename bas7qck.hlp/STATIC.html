STATIC variable[()] [AS type] [,variable[()] [AS type]]...
 
    ■ variable    The variable the procedure will use. It is either an
                  array name followed by (), or a variable name.
    ■ AS type     Declares the variable's type (see Details for more
                  information about the AS type clause).

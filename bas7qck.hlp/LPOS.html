LPOS(n%)
 
    ■ n%   An integer expression with a value between 0 and 3 that
           indicates one of the printers:
               0 = LPT1, 1 = LPT1, 2 = LPT2, 3 = LPT3
 
Returns
    Number of characters sent to the printer since the last carriage
    return was sent.

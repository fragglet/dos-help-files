EVENT ON
EVENT OFF
 
    ■ EVENT ON    Enables trapping of events until BASIC encounters the
                  next EVENT OFF statement.
    ■ EVENT OFF   Disables trapping of events until BASIC encounters the
                  next EVENT ON statement.

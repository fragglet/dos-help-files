OUT port, data%
 
  ■ port    A numeric expression with an integer value between 0 and
            65,535, inclusive, that identifies the destination hardware
            I/O port.
  ■ data%   A numeric expression with an integer value between 0 and
            255, inclusive, that is the data to be sent to the port.
 
  ■ The OUT statement is not available in OS/2 protected mode.

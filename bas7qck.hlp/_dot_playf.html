PLAY(n)
 
    ■ n    A dummy argument that may be any numeric value. It is
           not used by the PLAY function.
 
    ■ The PLAY function is not available for OS/2 protected mode.
 
Returns
    The number of notes currently in the background-music queue,
    or 0 when music is running in the foreground.

STRIG(n%)
 
    ■n%  A numeric expression with an integer value between 0 and 7
         that indicates what type of information you want returned by
         the function. See the Details screen for the meaning of the
         values.
 
    ■ The STRIG function is not available for OS/2 protected mode.

COM(n%) ON
COM(n%) OFF
COM(n%) STOP
 
    ■ n%             The number of the COM (serial) port:  1 = COM port 1;
                                                           2 = COM port 2.
    ■ COM(n%) ON     Enables trapping of a communications event by an
                     ON COM statement.
    ■ COM(n%) OFF    Disables the event trap. Even if an event takes
                     place, it is not remembered.
    ■ COM(n%) STOP   Suspends the event trap. Any events that occur are
                     remembered and are trapped as soon as a COM ON
                     statement is executed.

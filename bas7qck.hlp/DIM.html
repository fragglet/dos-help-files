DIM [SHARED] variable[(subscripts)] [AS type]
            [,variable[(subscripts)] [AS type]]...
 
    ■ SHARED        Allows all procedures in a module to share arrays and
                    simple variables.
    ■ variable      A BASIC variable name.
    ■ subscripts    Dimensions of the array (see Details).
    ■ AS type       Declares the datatype (see Details).

SSEG(stringvariable$)
 
    ■ stringvariable$    The string variable for which you want to find
                         the segment address. It can be a simple string
                         variable or a single element of a string array.
                         You cannot use fixed-length string arguments.
 
Returns
    The segment address of the argument (or 0 if the argument is a null
    string).

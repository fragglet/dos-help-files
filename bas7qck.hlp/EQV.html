EQV Operator
                                                 ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
result = numeric-expression1 EQV numeric-expression2
 
    ■ The logical-equivalence operator compares corresponding bits in
      numeric-expression1 and numeric-expression2, then sets the corresponding
      bit in the result according to the following table:
 
      ═══════════════════════   ════════════════════════   ═════════════
      Bit in first expression   Bit in second expression   Bit in result
      ═══════════════════════   ════════════════════════   ═════════════
               1                         1                       1
               1                         0                       0
               0                         1                       0
               0                         0                       1
      ═══════════════════════   ════════════════════════   ═════════════

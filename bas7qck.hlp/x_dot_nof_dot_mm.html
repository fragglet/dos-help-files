Using the /Nofrills Option
When you invoke QBX with the /Nofrills option, QBX uses about 19K less
memory, but does not support any of the following features:
 
          Utility menu and commands
          Options menu and commands
          Help menu and commands
 
To load QBX with the /Nofrills option, type:
 
          QBX /nof
 
The three menus listed above are greyed out and cannot be opened when
/Nofrills is used. If you attempt to access any of the features discarded
by the /Nofrills option, the "Feature Removed" error dialog is generated.

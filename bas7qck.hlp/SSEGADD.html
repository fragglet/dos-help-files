SSEGADD(stringvariable$)
 
    ■ stringvariable$    The string variable for which you want to find
                         an address. It can be a simple string variable or a
                         single element of a string array. You cannot
                         use fixed-length string arguments.
 
Returns
    The far address of stringvariable& (both segment and offset).

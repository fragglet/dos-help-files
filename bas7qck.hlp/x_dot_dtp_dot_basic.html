                                                     When compiling with

   #       DOUBLE precision        8 bytes           -1.79769313486231D308
           floating point                            to -4.94065D-324 for
                                                     negative values;

                                                     for positive values
 
                                                     When compiling with
                                                     /Fpa:
                                                     -1.79769313486231D308

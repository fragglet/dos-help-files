──────────────────────────────────────────────────────────────────────────────
FIELD [#]filenumber%, fieldwidth% AS stringvariable$
                    [,fieldwidth% AS stringvariable$] ...
 
    ■ filenumber%        The number of an open file.
    ■ fieldwidth%        The number of characters in a field.
    ■ stringvariable$    A variable that contains the data read from a
                         record, or assignment data.

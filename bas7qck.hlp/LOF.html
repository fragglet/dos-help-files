LOF(filenumber%)
 
    ■ filenumber%    The number of an open file or ISAM table.
 
Returns
    For an open file:                The size of the file in bytes.
    For an ISAM table:               The number of records in the table.
    For a device opened as a file:   The number of bytes free in the
                                     output buffer.

SHELL(commandstring$)
 
    ■ commandstring$   The name of the OS/2 process to be executed, along
                       with any command-line arguments that the executed
                       process requires.
 
    ■ The SHELL function is used only in OS/2 protected mode.
 
Returns
    An integer value that is the OS/2 process ID for the shelled process.

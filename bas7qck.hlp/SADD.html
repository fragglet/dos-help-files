SADD (stringvariable$)
 
    ■ stringvariable$   The string for which you want to find the
                        offset. It can be a simple string variable or a
                        single element of a string array. You cannot use
                        fixed-length string arguments.
 
Returns
    The 2-byte offset address of the specified string variable. The
    returned address is an integer.

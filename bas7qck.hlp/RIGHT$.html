RIGHT$(stringexpression$,n%)
 
    ■ stringexpression$   Any string variable, string constant, or string
                          expression.
    ■ n%                  A numeric expression whose value is between 0
                          and 32,767, inclusive, indicating how many
                          characters are to be returned.
 
Returns
    A string that consists of the rightmost n% characters of a string.

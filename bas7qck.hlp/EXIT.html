EXIT {DEF | DO | FOR | FUNCTION | SUB}
 
    ■ EXIT DEF         Immediately exits from the executing DEF FN
                       function.
    ■ EXIT DO          Exits from a DO...LOOP.
    ■ EXIT FOR         Provides another way to exit from a FOR...NEXT loop.
    ■ EXIT FUNCTION    Immediately exits from a FUNCTION procedure.
    ■ EXIT SUB         Immediately exits from a SUB procedure.

INP(port)
 
    ■ port    A numeric expression with an integer value between
              0 and 65,535, inclusive, that identifies the hardware
              I/O port from which to read the byte.
 
    ■ The INP function is not available in OS/2 protected mode.
 
Returns
    The byte read from a hardware port.

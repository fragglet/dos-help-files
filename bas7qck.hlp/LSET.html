──────────────────────────────────────────────────────────────────────────────
LSET stringvariable=stringexpression
LSET recordvariable1=recordvariable2
 
    ■ stringvariable      A random-access file field defined in a FIELD
                          statement, or any string variable.
    ■ stringexpression    The value that is assigned to stringvariable
                          and is left-justified.
    ■ recordvariable1     A record variable of any data type.
    ■ recordvariable2     A record variable of any data type.

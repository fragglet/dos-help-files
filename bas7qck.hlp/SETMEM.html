SETMEM(numeric-expression&)
 
    ■ numeric-expression&   Specifies the number of bytes by which the
                            far heap will increase or decrease. (Use
                            positive numbers to increase the far heap;
                            negative numbers to decrease the far heap.)
 
Returns
 
    If numeric-expression& is   SETMEM returns
    ═════════════════════════   ═════════════════════════════════════
    Positive or negative        Total number of bytes in the far heap.
    0                           Current size of far heap.

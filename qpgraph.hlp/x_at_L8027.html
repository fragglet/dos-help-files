_GetTextCursor Function
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Returns
 
    Word    The current cursor attribute (shape) or -1 if error
 
  Description
 
    The _GetTextCursor function gets the current cursor attribute
    (the shape), as defined by the BIOS. This function works only in
    text video modes.
 
  _GrStatus Returns
 
    _GrNotInProperMode

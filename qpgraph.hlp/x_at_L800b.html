  Returns
 
    Integer    Active video page number
 
  Description
 
    The _GetActivePage function returns the active page number. All
    mode/hardware combinations support at least one page (page 0).
 
    The active page is the area of video memory where text or graphics
    are written. The visual page is the area of video memory currently
    being displayed. See the _SetActivePage help screen for animation
    techniques using multiple pages.

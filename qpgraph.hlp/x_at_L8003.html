_ClearScreen Procedure
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Argument
 
    area    _GClearScreen, _GViewPort, or _GWindow
 
  Description
 
    The _ClearScreen procedure erases the target area, filling it with
    the current background color. This procedure works in all video
    modes, although _GViewPort can only be used in graphics modes.
 
  _GrStatus Returns
 
    _GrInvalidParameter, _GrNotInProperMode

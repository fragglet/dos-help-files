_GetWindowCoord Procedure
  ◄Summary► ◄Details► ◄Example►                                   ◄Back►
 
  Arguments
 
    x, y    View coordinates of point to translate
 
    wxy     Window coordinates of point to translate
 
  Description
 
    The _GetWindowCoord procedure translates the view coordinates (x,y)
    to window coordinates and stores them in a _WXYCoord record.
 
  _GrStatus Returns
 
    _GrNotInProperMode

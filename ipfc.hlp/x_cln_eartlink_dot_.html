 :eartlink.
 
The :eartlink tag ends a link-definition file. A link-definition file
contains :link tags that establish hypergraphic links. A hypergraphic is a
bitmap that is linked to a section, footnote, spot, another program, or an
application message. By choosing the hypergraphic, the user can cause the
referenced section, footnote, or spot to appear on the display, start the
referenced program, or send a message to the application. The user chooses a
hypergraphic by clicking it with the mouse or by selecting it with the TAB
key and pressing the ENTER key.
 
This tag does not use parameters.

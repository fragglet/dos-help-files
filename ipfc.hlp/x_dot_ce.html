 .ce
 
The .ce tag centers the subsequent line of text.
 
This tag does not use parameters.
 
Comments
 
The .ce tag must be placed in column one. The text to be centered must be on
the same line as the .ce tag, and one or more spaces must separate the tag
from the text. Any formatting tags included on the same line as .ce are not
processed but displayed as is.
 
Example
 
This example centers the line "MS OS/2 Help Manager":
 
    \.ce
    MS OS/2 Help Manager

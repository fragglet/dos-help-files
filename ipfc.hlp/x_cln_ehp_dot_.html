 :ehpn.
 
The :ehp tag ends a font or text color, restoring the previous (or default)
font and color. The tag must be preceded by a matching :hp tag.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
n          Specifies the font or color to end. This value can be any integer
           in the range 1 to 9.

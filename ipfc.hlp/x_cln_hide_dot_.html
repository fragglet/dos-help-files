 :hide key='name'.
 
The :hide tag starts conditional viewing of text. To view the subsequent
text, the user must set the keys environment variable to match the value
specified in name. The :ehide tag ends conditional viewing of text. You use
the :hide tag to restrict access to information in a file so that only those
who know the key can view the restricted information.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
name       Specifies the name of a key for conditional viewing.
 
Comments
 
One hide tag must end before another can begin.
 
Example
 
This example uses a hide tag to restrict access to proprietary information:
 
    :p.
    You can choose one of the following options:
    :ul compact.
    :li.Delete
    :li.Copy
    :li.Save
    :eul.
    :hide key='sys+app'.
    :p.FOR INTERNAL USE ONLY.
    :p.The following options are still in development.
    They will be included in the next release:
    :ul compact.
    :li.Revise
    :li.Restore
    :eul.
    :ehide.

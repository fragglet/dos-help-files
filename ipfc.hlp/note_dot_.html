 <note [text='text']>
 
The note tag starts a new line and inserts the word "Note:" (or the text
specified by the text option) immediately preceding subsequent text. Notes
are typically used to alert the user to additional information.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
text       Specifies the text used to start the note. The text value can be
           any string of characters but must be enclosed in single quotation
           marks ('). If this parameter is not given, the default text is
           "Note:".
 
Example
 
This example uses the note tag to create a note:
 
    <note>Some adapters may not provide all colors.
    Actual colors depend on the monitor.
 
When displayed, the note looks like this:
 
    Note: Some adapters may not provide all colors.
    Actual colors depend on the monitor.

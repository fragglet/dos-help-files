 :i2 refid=label.
 
The :i2 tag creates a second-level index entry. The entry consists of the
text immediately following the tag and is placed under the first-level index
entry identified by the refid parameter. The second-level entry is
associated with the current panel such that the panel is displayed when the
user chooses the entry.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
refid      Identifies the first-level index entry to associate this
           second-level entry with. The label value must be the index label
           value that uniquely identifies the first-level entry and must be
           given in the id parameter of a :i1 tag.
 
Comments
 
You cannot have index entries in a footnote.

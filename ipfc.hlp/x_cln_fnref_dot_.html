 :fnref refid=label.
 
The :fnref tag creates a footnote reference. A footnote reference links a
footnote to a hypertext word or phrase. When the user clicks on the
hypertext, the system displays the corresponding footnote. A hypertext word
or phrase can be created by using the :hpt and :ehpt tags.
 
In the help source file, a footnote reference can be used anywhere in the
file, including before the definition of the footnote.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
refid      Identifies the footnote to display. The label value must be the
           name or integer that uniquely identifies a footnote and must be
           given in the id parameter of a :fn tag.
 
Example
 
This example creates a footnote reference for the hypertext phrase "key
labels":
 
    Different keyboards may have different
    :hpt.key labels:ehpt.:fnref refid=keylabel.

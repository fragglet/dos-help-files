 :ainform num=number [x=n y=n cx=n cy=n].
 
The :ainform tag creates a reference to an information message. The
reference is linked to the art specified by the preceding :hpart tag. The
system sends a WM_INFORM message to the application whenever the user clicks
on the art (or clicks on the area of the art specified by the x, y, cx, and
cy parameters).
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
num        Identifies the number to be passed as the first parameter in the
           WM_INFORM message. The number value must be an integer.
 
x          Specifies the x-coordinate of the lower-left corner of the
           portion of the art linked to the footnote reference.
 
y          Specifies the y-coordinate of the lower-left corner of the
           portion of the art linked to the footnote reference.
 
cx         Specifies the width of the portion of the art linked to the
           footnote reference.
 
cy         Specifies the height of the portion of the art linked to the
           footnote reference.
 
Example
 
This example uses the :ainform tag to create a reference to an information
message. The reference is linked to the art file copy.bmp.
 
    :hpart name='copy.bmp'.
    :ainform num=1.
    :ehpart.

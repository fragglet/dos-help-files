 :isyn root='word'.synonyms
 
The :isyn tag specifies index synonyms. An index synonym is a word that is
linked to a first-level index entry. When the user enters an index synonym
during an index search, the corresponding index entry is displayed.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
root       Specifies the word that serves as the link between the synonyms
           and a first-level index entry. The word value must be a character
           string enclosed in single quotation marks ('). To establish the
           link, you must specify the same word value in the roots parameter
           of a :i1 tag.
 
synonyms   Specifies a list of variations and synonyms for the root word.
           The synonyms must be separated by one space.
 
Comments
 
The :isyn tag can be placed in any help panel that contains a related  :i1
tag.
 
Example
 
This example uses the :isyn tag to create synonyms for "copy" and "folder":
 
    :h1 id=copy03.Help for Copying
    :isyn root=copy.copy copying duplicate duplicating
    :isyn root=folder.folder folders document documents
    :i1 roots='copy folder'.Copying a document
    :p.When copying a file from the current directory...

 <rp [help=name] [res=id]>
 
The rp tag creates a reference phrase. A reference phrase is a word or
phrase that is linked to a specified help panel. If the user clicks on or
chooses the reference phrase, the system displays the corresponding help
panel.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
help       Identifies the help panel to display. The name value must be the
           help name that uniquely identifies the help panel and must be the
           name given in the name parameter of a help tag.
 
res        Identifies the help panel to display. The number value must be
           the resource number that uniquely identifies the panel and must
           be the number given in the res parameter of a help tag. It must
           be an integer value integer in the range 1 to 32,767.
 
Example
 
This example uses the rp tag to define the reference phrase "key labels":
 
    Different keyboards may have different
    <rp help=keylabel>key labels</rp>

 <pd>
 
The pd tag specifies the parameter data for a matching parameter term. The
parameter data is one or more lines of text displayed in the right column of
a parameter list. The tag must be followed immediately by one or more lines
of text.
 
The pd tag may be used only in a parameter list created using the parml tag.
The pd tag must be preceded by a matching pt tag.
 
This tag does not use parameters.

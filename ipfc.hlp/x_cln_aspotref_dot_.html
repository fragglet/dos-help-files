 :aspotref refid=label [x=n y=n cx=n cy=n].
 
The :aspotref tag creates a reference to a spot. The reference is linked to
the art specified by the preceding :hpart tag. The system displays the spot
identified by the label value whenever the user clicks on the art (or clicks
on the area of the art specified by the x, y, cx, and cy parameters).
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
refid      Identifies the spot to display. The label value must be the label
           that uniquely identifies the spot and must be given in the id
           parameter of a :spot tag.
 
x          Specifies the x-coordinate of the lower-left corner of the
           portion of the art linked to the spot reference.
 
y          Specifies the y-coordinate of the lower-left corner of the
           portion of the art linked to the spot reference.
 
cx         Specifies the width of the portion of the art linked to the spot
           reference.
 
cy         Specifies the height of the portion of the art linked to the spot
           reference.
 
Example
 
This example uses the :aspotref tag to create a reference to the spot
labeled "syntax". The reference is linked to the art file copy.bmp.
 
    :hpart name='copy.bmp'.
    :aspotref refid=syntax.
    :ehpart.

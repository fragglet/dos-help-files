 :lm [margin='margin'].
 
The :lm tag sets the left margin for the subsequent text. The left margin is
reset to column one when any text-formatting tag other than :p is
encountered.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
margin     Specifies the size, in spaces, of the left margin. The margin
           value must be an integer representing the number of spaces from
           the left side of the help panel. A "space" is the average width
           of the characters in the system display font. If no margin value
           is specified, the default is one space.
 
Comments
 
You can use any number of :lm tags in a help file. The :lm tag has no effect
on formatting tags other than :p.
 
Example
 
This example uses the :lm tag to set the left margin at the tenth space from
the left side of the help panel:
 
    :lm margin=10.
    Use the Copy command to copy files or
    to combine two or more files.

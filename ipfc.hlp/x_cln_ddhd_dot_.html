 :ddhd.
 
The :ddhd tag specifies the heading for the right (definition-data) column
and starts a new row in a definition list. The definition-data heading is
one or more words displayed in the right column of a definition list. The
tag must be immediately followed by one or more words.
 
The :ddhd tag may be used only in a definition list created using the  :dl
tag. The :ddhd tag must be preceded by a matching :dthd tag.
 
This tag does not use parameters.

 :artwork name='filename' [runin] [linkfile='filename']
    [align={left | center | right}].
 
The :artwork tag places a bitmap or other art in a help panel.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
name       Specifies the name of the file containing the bitmap or other
           art. The bitmap must have the MS OS/2 bitmap format; filename
           must be a valid MS OS/2 filename and must be enclosed in single
           quotation marks (').
 
runin      Inserts the bitmap in the current text line. If this option is
           not given, the current line ends and the bitmap starts on a new
           line. Also any subsequent text starts on a new line after the
           bitmap.
 
linkfile   Specifies the name of a link definition file. The filename value
           must be a valid MS OS/2 filename specifying the directory,
           filename, and file extension of the link definition file. The
           filename must be enclosed in single quotation marks (').
 
align      Places the bitmap at the left edge, right edge, or middle of the
           help panel, depending on the option chosen. If runin is not
           specified, the bitmap is aligned on the left edge by default. The
           align option is ignored if runin is given.
 
Example
 
This example uses the :artwork tag to center a bitmap (picture.bmp) in a
help panel:
 
    :artwork name='picture.bmp' align=center.

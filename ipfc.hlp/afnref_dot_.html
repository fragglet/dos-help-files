 :afnref refid=label [x=n y=n cx=n cy=n].
 
The :afnref tag creates a reference to a footnote. The reference is linked
to the art specified by the preceding :hpart tag. The system displays the
footnote identified by the label value whenever the user clicks on the art
(or clicks on the area of the art specified by the x, y, cx, and cy
parameters).
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
refid      Identifies the footnote to display. The label value must be the
           label that uniquely identifies the footnote and must be given in
           the id parameter of a :fn tag.
 
x          Specifies the x-coordinate of the lower-left corner of the
           portion of the art linked to the footnote reference.
 
y          Specifies the y-coordinate of the lower-left corner of the
           portion of the art linked to the footnote reference.
 
cx         Specifies the width of the portion of the art linked to the
           footnote reference.
 
cy         Specifies the height of the portion of the art linked to the
           footnote reference.
 
Example
 
This example uses the :afnref tag to create a reference to the footnote
labeled "syntax". The reference is linked to the art file copy.bmp.
 
    :hpart name='copy.bmp'.
    :afnref refid=syntax.
    :ehpart.

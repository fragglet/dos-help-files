 :ecgraphic.
 
The :ecgraphic tag ends an area that contains character graphics. A
character-graphics area is any table, chart, or other art that uses graphics
characters. The character-graphics area starts with the  :cgraphic tag and
must end with the :ecgraphic tag.
 
This tag does not use parameters.

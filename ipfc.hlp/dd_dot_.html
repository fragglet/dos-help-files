 <dd>
 
The dd tag specifies the definition data for a matching definition term. The
definition data is one or more lines of text that are displayed in the right
column of a definition list. The tag must be followed immediately by one or
more lines of text.
 
The dd tag may be used only in a definition list created using the dl tag.
The dd tag must be preceded by a matching dt tag.
 
This tag does not use parameters.

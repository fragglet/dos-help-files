 :title.
 
The :title tag specifies the title for the help document. This title appears
in the title bar of the help window. The tag must be followed immediately by
the title text and the complete text must be given on one line. The text
must not contain more than 72 characters.
 
This tag does not use parameters.
 
Example
 
This example uses the :title tag to set the title to "Command Reference":
 
    :title.Command Reference

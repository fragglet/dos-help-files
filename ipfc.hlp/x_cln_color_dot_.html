 :color [fc=color] [bc=color].
 
The :color tag sets the foreground and background colors of the subsequent
text. The specified colors apply until another :color tag changes the
colors.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
fc         Specifies the foreground color (that is, the color of the text).
 
bc         Specifies the background color.
 
color      Specifies the color. It can be one of the following values:
 
           default blue cyan green neutral red yellow
 
           To return to the foreground or background color originally set
           for your system, choose default.
 
Example
 
This example uses the :color tag to change the text to red and the
background to neutral:
 
    :p.
    The
    :color fc=red bc=neutral.
    Delete
    :color fc=default bc=default.
    command deletes all of your files.

 .im filename
 
The .im tag inserts the content of the specified file into the help file.
The file must contain valid IPF tags.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
filename   Specifies the name of the file containing IPF tags. The filename
           must be a valid MS OS/2 filename. Wildcard charcaters are not
           allowed.
 
Comments
 
You must specify the filename and extension exactly. If the file is not in
the current directory, a full pathname must be specified.
 
Example
 
This example uses the .im tag to insert the contents of the file
artwork.ipf in the help file:
 
    .im artwork.ipf

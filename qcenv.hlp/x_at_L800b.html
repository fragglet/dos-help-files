Menu and Dialog Box Help
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
    The online help system contains information on every menu
    and menu command. You can also get help on every dialog box
    by choosing the dialog's < Help > command button. Both menu
    help and dialog box help appear in a dialog box.
 
      ■ To get help on a menu, press ALT, move the cursor to the
        item of interest in the menu bar and then press F1.
 
      ■ To get help on a menu command, select the menu command and
        then press F1.
 
      ■ To get help on a dialog box, press F1 or select its
        <Help> button.
 
    The ESC key closes menu help and dialog boxes.

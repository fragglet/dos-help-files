     Warning: A4016
     Reserved word used as symbol
 
     An assembler keyword was used as a symbol. This is a warning, not
     an error and can be ignored if you wish. However, the keyword is
     no longer available for its original purpose. For example, if you
     name a macro "add", it replaces the ADD instruction.

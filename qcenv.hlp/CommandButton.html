  Using the Mouse
 
  Position the mouse cursor between the angle brackets and click the
  Left mouse button.
 
  Using the Keyboard
 
  Use the TAB key to move the cursor between the angle brackets
  and press ENTER or the SPACEBAR.
 
  See also:  ◄Command Button Actions► for details on the meaning
  of the command buttons.

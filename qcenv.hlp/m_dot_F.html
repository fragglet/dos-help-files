File Menu
┌────────────────────┐
│ New                │ ■ Creates a new file
│ Open...            │ ■ Opens an existing file
│ Open Last File  F2 │ ■ Reopens the previous file *
│ Merge...           │ ■ Merges two files *
│ Save               │ ■ Saves current version of file
│ Save As...         │ ■ Saves this file with a new name
│ Save All           │ ■ Saves all files in this project *
├────────────────────┤
│ Print...           │ ■ Prints any file in project
│ DOS Shell          │ ■ Lets you issue DOS commands *
├────────────────────┤
│ Exit        Alt+F4 │ ■ Exits from QuickC
└────────────────────┘
* Items marked by an asterisk appear only with full menus.

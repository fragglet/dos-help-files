     Error: A2064
     Near JMP/CALL to different CS
 
     A near jump or call instruction attempted to access an address in
     another code segment. (The current code segment is determined by
     an ASSUME CS directive.) To correct the error, use a far call or
     jump, or use an ASSUME statement to change the code segment
     currently referenced by CS.

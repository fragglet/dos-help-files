Help Menu
Use the Help menu for help on the C language, C run-time
library, error messages, or the QuickC environment.
 
┌────────────────────────┐
│ Index                  │ ■ Alphabetic index of topics
│ Contents               │ ■ Table of contents
│ Topic:              F1 │ ■ Help on specific topic
│ Help On Help  Shift+F1 │ ■ Help on using Quick Advisor
└────────────────────────┘
 
While a help file is displayed, use Find and Repeat Last
Find (on the Search menu) to find text in the file.
 
Select Search Multiple Help Topics from the Options:Display
command to search all topics in the current help file.

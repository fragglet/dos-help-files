n.63
The expression you specified is not a legal watchpoint
expression.
 
Note that QuickC considers expressions that contain local
variables to be invalid if it has not yet entered the
function that defines the variables. If your expression
contains a local variable, select <Yes> in response to the
<Use it anyway?> prompt. QuickC will enter the expression
in its list of watchpoints.

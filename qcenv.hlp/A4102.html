     Warning: A4102
     Segment near (or at) 64K limit
 
     A bug in the 80286 processor causes jump errors when a code
     segment approaches within a few bytes of the 64K limit in
     privileged mode. This error warns about code that may fail
     because of the bug. The error can only be generated when the .286
     directive is given.

Controlling Check Boxes
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
  Check boxes control compiler directives and search options that
  are usually not mutually exclusive.
 
    [X]  An X indicates that the option is enabled.
    [ ]  No X indicates that the option is disabled.
 
  Using the Mouse
 
  Position the mouse cursor on the check box and click Left.
 
  Using the Keyboard
 
  Use the TAB key to move the cursor into the check box, then
  press the SPACEBAR.

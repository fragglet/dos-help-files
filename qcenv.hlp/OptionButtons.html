Using Option Buttons
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
  Option buttons control mutually exclusive options. You can use
  them to select one option from a list of options.
 
    (•)  A bullet indicates that an option is enabled.
    ( )  No bullet indicates that an option is disabled.
 
  Using the Mouse
 
  Position the mouse cursor between the parentheses and click Left.
 
  Using the Keyboard
 
  Move the bullet with the UP and DOWN arrow keys.

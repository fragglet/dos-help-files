     Warning: A4125
     Start address on END ignored with .STARTUP
 
     A start address was specified on the END directive although
     .STARTUP was used. With .STARTUP, no start address should be
     given on the END directive since the program will begin execution
     at the .STARTUP directive. This message is not generated if the
     address of the start label specified is the same as the address
     of the .STARTUP directive.

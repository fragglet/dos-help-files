View Menu
Use the View menu to select which part or parts of your
program you want to see.
┌───────────────────────┐
│ Source...             │ ■ Displays source file *
│ Include...            │ ■ Displays include (.H) file *
│ Show Listing..Ctrl+F2 │ ■ Displays listing (.LST) file *
├───────────────────────┤
│ Output Screen   F4    │ ■ Displays the program output
│ Maximize  Ctrl+F10    │ ■ Sets all windows to full size *
│ Windows...            │ ■ Controls window visibility *
└───────────────────────┘
 
* Items marked by an asterisk appear only with full menus.

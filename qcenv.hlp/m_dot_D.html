The Debug Menu
┌─────────────────────┐
│ Calls...            │ ■ Lists chain of function calls *
│ Breakpoint...     F9│ ■ Sets and removes breakpoints
│ Watchpoint...       │ ■ Sets and removes watchpoints *
├─────────────────────┤
│ Quickwatch..Shift+F9│ ■ Monitors and changes variables *
│ Watch Value...      │ ■ Monitors the value of a variable
│ Modify Value...     │ ■ Changes the value of a variable *
├─────────────────────┤
│ History On          │ ■ Records debug session in a file *
│ Undo                │ ■ Moves back through history file *
│ Replay              │ ■ Replays recorded debug session *
│ Truncate User Input │ ■ Truncates recorded user input *
└─────────────────────┘
* Items marked by an asterisk appear only with full menus.

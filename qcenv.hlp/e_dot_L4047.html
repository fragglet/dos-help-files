     Warning: L4047
     Multiple code segments in module of overlaid program
     incompatible with /CODEVIEW
 
     You defined multiple code segments within the same module. Only
     the first code segment in the module can be debugged. To enable
     full use of debugging commands, combine all the code segments in
     the module into one segment, or move some of the code segments to
     other modules.

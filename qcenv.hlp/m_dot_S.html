Search Menu
Use the Search menu to find text, functions, and errors,
beginning at the cursor position.
┌──────────────────────────┐
│ Find...                  │ ■ Searches for text
│ Selected Text    Ctrl+F3 │ ■ Searches for selected text *
│ Repeat Last Find      F3 │ ■ Searches again for same text
│ Change...                │ ■ Replaces text
│ Go to...                 │ ■ Go to specified line *
├──────────────────────────┤
│ Function...              │ ■ Finds function declarations*
├──────────────────────────┤
│ Next Error      Shift+F3 │ ■ Finds line with next error
│ Previous Error  Shift+F4 │ ■ Finds line with last error
└──────────────────────────┘
* Items marked by an asterisk appear only with full menus.

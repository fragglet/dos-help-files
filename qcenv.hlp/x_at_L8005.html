Using the Mouse

────────────────────────────────────────────────────────────────────────────
    To get help by using the mouse, click the Right mouse button.
 
    QuickC keeps track of the cursor position, which always
    determines the help topic. For example,
 
      ■ If you click Right when the cursor is on a C keyword,
        manifest constant, or function, help on that topic appears.
 
      ■ If you need help on a menu item or a dialog box, you must
        press F1 for help.
 
    NOTE: You must enable the Context-Sensitive Help option
          before the Right mouse button will select help topics.
          Look for it in the Options menu under Display. This is
          normally set as the default option.

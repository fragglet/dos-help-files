     Error: A2035
     Operand must have size
 
     An operand was expected to have a specified size, but no size was
     supplied. For example, the following statement is illegal:
 
                 inc     [bx]
 
     Often this error can be remedied by using the PTR operator to
     specify a size type, as shown below:
 
                 inc     BYTE PTR [bx]

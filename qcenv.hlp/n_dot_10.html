n.10
The program you were running exceeded the DOS limit on
open files.
 
If you still want to run this program, do the following:
 
  1. End your program with the Restart (SHIFT+F5) command
 
  2. Exit QuickC
 
  3. Set FILES to a larger number in your CONFIG.SYS file
 
  4. Reboot your machine
 
  5. Start QuickC again and try to run your program

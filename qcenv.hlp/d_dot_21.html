Customizing the Menu
To add an item to the menu:
  1. Choose <Add>, and fill in the dialog box that appears.
 
To change parameters for an item already on the menu:
  1. Use the arrow keys or mouse to select the item.
  2. Choose <Edit>, then make changes in the dialog box.
 
To delete an item from the menu:
  1. Use the arrow keys or mouse to select the item.
  2. Choose <Delete>.
 
To rearrange items on the menu:
  1. Use the arrow keys or mouse to select an item.
  2. Choose <To Begin> to make the item first on the menu,
     or choose <To End> to make the item last on the menu.

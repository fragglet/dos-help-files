The Index command displays an alphabetic index of topics.
 
  1. Type one of the letters shown and press ENTER to see a
     list of entries starting with that letter.
 
  2. Use the scroll bars and arrow keys to move through
     the list.
 
  3. Press F1 to get help on the topic under the cursor.
 
You can use Find and Repeat Last Find (on the Search menu)
to find text in the current help file. Select Search
Multiple Help Topics from the Options:Display command to
search all help files.

────────────────────────────────────────────────────────────────────────────
  [*.C   ]    Use text boxes either to select the listed item or to
              enter another item. All of the keyboard QuickC editing
              functions work in a text box──the Edit menu isn't
              available from within a dialog box.
 
  Using the Mouse
 
  Use the mouse to select one or more characters within the string.
 
  Using the Keyboard
 
  Use any of the keyboard (not the menu) QuickC editing functions.

Source Command
Use the View Source command to move quickly between the
different source files in a multimodule program. The
files in a program list may be in any directory.
 
To view a source file:
┌─────────────────────────────────┐
│ xxxxxxxx.MAK  ◄─────────────────┼─ Program list name
│                                 │
│ View: C:XX ◄────────────────────┼─ Source file to open
│                                 │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒◄┼─■ Select file name here
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │ ■ Choose <OK> to view
└─────────────────────────────────┘   that source file

     Error: A2031
     Operand types must match
 
     You used registers of different sizes in the same instruction, as
     in the following example:
 
                 mov     ax,bl
 
     The assembler cannot resolve such expressions, since the size of
     a register is absolutely fixed. Rewrite the instruction so that
     the registers have the same size.

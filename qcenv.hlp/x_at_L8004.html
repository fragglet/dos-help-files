Using the Keyboard
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
    To get help using the keyboard, press F1.
 
    QuickC keeps track of the cursor position, which always determines
    the help topic. For example,
 
      ■ If you press F1 when the cursor is on a C keyword,
        manifest constant, or function, help on that topic appears.
 
      ■ If you press F1 while in a dialog box or on a menu item,
        help on the dialog box or menu item appears.

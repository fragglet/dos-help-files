n.302
You tried to set a breakpoint on a symbol that is not
recognized as a function name. This message occurs when
you try to set a breakpoint on a label.
 
Select <yes> if you are setting a breakpoint on a label.
Note that you should not set breakpoints on labels of
data items in the data segment.
 
Select <no> or <cancel> to cancel setting the breakpoint.

Utility Menu
Use the Utility menu to run DOS commands and frequently
used applications without exiting the QuickC environment.
┌────────────────────┐
│ Run DOS Command... │ ■ Runs any DOS command
│ Customize Menu...  │ ■ Adds items to menu
├────────────────────┤
│ Learn QuickC       │ ■ Invokes QuickC tutorial
│ Custom Editor      │ ■ Invokes user-defined editor
│ ░░░░░░░░░░░░░░░    │┐
│ ░░░░░░░░░░░░░░░    ││■ Invokes other custom menu items
└────────────────────┘┘

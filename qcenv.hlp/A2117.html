     Error: A2117
     Segment address reference illegal with tiny
 
     A reference to a segment address was found.  The following are
     examples of references which would draw this error if TINY model
     is specified:
 
            mov    ax, SegmentName
            mov    ax, SEG labelName
            jmp    FAR PTR labelName

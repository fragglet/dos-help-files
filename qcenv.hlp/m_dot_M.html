Make Menu
 
A program list (also called a .MAK file) helps you manage
programs that have multiple source files.
┌──────────────────────┐
│ Compile File   XXXX  │ ■ Compiles file named XXXX  *
│ Build Program  XXXX  │ ■ Compiles/links changed files
│ Rebuild All    XXXX  │ ■ Compiles/links whole project *
├──────────────────────┤
│ Set Program List...  │ ■ Sets .MAK file for project *
│ Edit Program List... │ ■ Changes contents of .MAK file *
│ Clear Program List   │ ■ Removes .MAK file from project *
└──────────────────────┘
* Items marked by an asterisk appear only with full menus.

This error occurs if you try to browse forward (SHIFT+F10)
from the end of the debug history file.
 
It also occurs after the following sequence of events:
 
  1. You created a debug history
  2. You rebuilt the program
  3. You turned debug history off and then on again
  4. You answered <Yes> to the "Use existing program.HIS"
     prompt

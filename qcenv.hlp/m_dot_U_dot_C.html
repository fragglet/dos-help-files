Customize Menu Command
The Customize Menu command adds up to eight commands to the
Utility menu. You can add any DOS program to the menu.
 
┌─────────────────┐
│ Learn QuickC    ↑
│ Custom  Editor  ░ ■ Use arrow keys to select an item.
│ ░░░░░░░░░░░░░░  ░
│ ░░░░░░░░░░░░░░  ↓
└─────────────────┘
 
  ■ Choose <Add> to add a program to the menu.
  ■ Choose <Edit> to change parameters for a program.
  ■ Choose <Delete> to remove the item from the menu.
  ■ Choose <To Begin> to make the item first on the menu.
  ■ Choose <To End> to make the item last on the menu.

Color Command
Use the Color command to set the display colors. Four
default sets of colors are supplied:
 
  ■ Use Color 1 for 16 color displays
  ■ Use Color 2 for 8 color displays
  ■ Use Color 3 for monochrome displays
  ■ Use LCD for liquid crystal displays.
 
To change any item's color, select it from the Item
list box, then select appropriate colors from the
Foreground and Background lists.
 
  ■ Choose <Redraw> to test the new colors.
  ■ Choose <Default> to restore the default color sets.
  ■ Choose <OK> to make your color changes permanent.

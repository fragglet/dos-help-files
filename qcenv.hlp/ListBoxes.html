  Using the Keyboard
 
  Use the TAB key to move to the list box. Use the arrow keys
  as well as the PGUP and PGDN keys to move between list items.
  Alternatively, you can select any list item by pressing its
  initial letter. (If more than one item begins with the same
  letter, press the letter again to move to the next item.) Press
  ENTER to choose the item.

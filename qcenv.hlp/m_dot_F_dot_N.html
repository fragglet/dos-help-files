New Command
Use the New command to create a new file in memory. The
New command clears memory and the Source window.
 
All new files are given the name UNTITLED.C. Assign a name
to the file when you save it with Save or Save As.
 
A dialog box appears if a program is already loaded into
memory.
 
  ■ Choose <Yes> to save the program
  ■ Choose <No> to clear the program without saving it
  ■ Choose <Cancel> or press ESC to look at the program in
    memory before making a decision

     Error: A2135
     feature requires language specifier on .MODEL
 
     You used one of the high-level features of the PROC directive
     (USES or an argument list), or used the LOCAL directive to create
     local variables, but did not specify a language type with the
     .MODEL directive. You must specify a language with .MODEL, even
     if you also specify a language in the PROC directive.

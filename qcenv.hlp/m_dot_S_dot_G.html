Go To
Use the Go To command to move the cursor to a specified
line number.
 
  1. Type the line number.
  2. Press ENTER or select <OK> to go to that line.
     Select <Cancel> to return to the QuickC editor.
 
┌────────────────────────────────┐
│Line:      [▒▒▒▒▒▒▒▒▒▒▒▒▒] ◄────│─ ■ Type line number
└────────────────────────────────┘    between 1 and 65535
 
TIPS: * To go to the end or beginning of the file use the
        CTRL+END or CTRL+HOME shortcut key.
      * The current line and column numbers are displayed
        in the lower right corner of the status bar.

     Error: A2006
     Phase error between passes
 
     An ambiguous instruction or directive caused the relative address
     of a label to be changed between Pass 1 and Pass 2. You can use
     the /D option to produce a Pass 1 listing to aid in resolving
     phase errors between passes.

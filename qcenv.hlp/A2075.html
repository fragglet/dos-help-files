     Error: A2075
     Illegal use of undefined operand (?)
 
     The undefined operand (?) was used incorrectly. For example, the
     following statements are illegal:
 
     stuff     DB     5 DUP (?+5) ; Can't use in expression
               mov    ax,?        ; Can't use in code

Command Button Actions
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
  You can use command buttons to choose any of the following actions:
 
  < Help >      Provides information that describes the command or
                the error message within the dialog box.
 
  < Yes >       Acknowledges the message in the dialog box and
                initiates the described action.
 
  <  No  >      Acknowledges the message in the dialog box and
                initiates an alternative action. In some cases,
                this button may be equivalent to the Cancel button.
 
  <  OK  >      Acknowledges the message or action in the dialog
                box. If you've used a dialog box to choose an
                action, such as selecting a file from a list of
                files, the OK button initiates the action.
 
  < Cancel >    Either acknowledges the message or cancels the
                action in the dialog box.

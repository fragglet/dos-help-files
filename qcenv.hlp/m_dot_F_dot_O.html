Open Command (3 of 3)
Use the Open command to clear memory and load an existing
program file from disk. The dialog box below appears:
 
┌─────────────────────────────────┐ To load a file from
│                                 │ the current directory:
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────│─────┐
│                                 │     │
│ C:XX ◄──current directory       │ ■ Type the file name
│                                 │   here,
│ File List:          Drives/Dirs │   OR
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │ ■ Select the file name
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │   here
│        ▲                        │    │
│        └────────────────────────│────┘
└─────────────────────────────────┘ ■ Choose <OK>
To open a program in another directory, either
┌─────────────────────────────────┐
│                                 │ ■ Type the directory
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────│── name and file name
│                                 │   here,
│ C:MYPROGS ◄──current directory  │   OR
│                                 │ ■ Select the directory
│ File List:         Drives/Dirs  │   name here,
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒ ◄│── then
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │ ■ Select the file name
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │   here
│        ▲                        │     │
│        └────────────────────────│─────┘
└─────────────────────────────────┘ ■ Choose <OK>
TIP: ".." refers to the directory one level up.
To look for files in other drives or directories:
┌─────────────────────────────────┐
│                                 │ ■ Type the wildcard
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────│── form of the file
│                                 │   name (such as *.H or
│ C:MYPROGS ◄──current directory  │   *.C) here,
│                                 │ ■ Select the drive and
│ File List:         Drives/Dirs  │   directory name
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒ ◄│── here, and
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │ ■ Scan the file names
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │   here
│        ▲                        │
│        └────────────────────────│─────┘
└─────────────────────────────────┘ ■ Choose <Cancel>

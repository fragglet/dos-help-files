     Error: A2129
     model must be TINY with /AT option
 
     You used the QCL /AT option or selected Generate COM File from
     the Assembler Flags dialog box, but used the .MODEL directive
     without specifying TINY model. To generate a COM file, you must
     specify TINY with the .MODEL directive, or else use full segment
     definitions.

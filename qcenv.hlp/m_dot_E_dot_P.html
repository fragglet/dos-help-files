Shortcut key: SHIFT+INS
 
Use the Paste command, or press SHIFT+INS, to copy text
from the Clipboard to the active window.
 
QuickC inserts the text in one of two places:
  ■ If any text in the active window is selected
    (highlighted), QuickC deletes it and replaces it with
    the text from the Clipboard.
 
  ■ If no text is selected, the text from the Clipboard is
    inserted to the right of the cursor.
 
WARNING: If the pasted text replaces highlighted text,
         QuickC does not save the highlighted text.

Copying Text from the Help System
                                          ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
    Use the Copy command, or press CTRL+INS, to copy selected
    (highlighted) text from the help window to the Clipboard.
 
    Once a block of text is in the Clipboard, choose Paste,
    or press SHIFT+INS, to insert it into an active window.
 
    NOTE: The Help window is a read-only window. This means
          that you cannot cut or paste in the Help window.
 
    See ◄Selecting Text► for information on how to select
    text in a window.

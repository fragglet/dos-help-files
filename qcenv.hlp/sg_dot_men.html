Using Menus and Commands

────────────────────────────────────────────────────────────────────────────
  Menu Action           Mouse Control             Keyboard Control
 
  Pull down a menu      Move mouse cursor to      Press and release
                        menu name, then press     ALT to highlight
                        and release ("click")     menu letters, then
                        the Left mouse button.    press letter key
                                                  for desired menu.
 
  Choose a command      Move mouse cursor to      Press letter key
                        command name and          that matches the
                        click.                    highlighted letter
                                                  on the command.
 
  Cancel a menu         Move mouse cursor out     Press ESC.
                        of menu and click.
 
  TIP: The reference bar at the bottom of the screen always contains
       help about the most useful commands in each QuickC context. If
       the command name is within angle brackets, it can be executed
       from the reference bar by choosing it with the mouse.

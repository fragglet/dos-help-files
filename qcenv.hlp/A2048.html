     Error: A2048
     Must be index or base register
 
     An indirect memory operand requires a base or index register, but
     some other register was specified. For example, the following
     line is illegal:
 
            mov    ax,[bx+ax]
 
     Only BP, BX, DI, and SI may be used in indirect operands.

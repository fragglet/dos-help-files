Print Command
Use the Print command for a hard copy of an entire file
or of selected text from any window.
 
  ► The Print command sends the text to the printer
    connected to the DOS port named LPT1. Make sure a
    printer is attached to this port, is powered on, and
    is on line.
 
┌────── Print ──────┐
│ ( ) Source File   │◄────── Prints entire file
│ ( ) Selected Text │◄────── Prints selected text
└───────────────────┘
 
TIP: Use the Print command to print parts of the QuickC
     Advisor that are of special interest to you.

Exit Command
Choose the Exit command when you have finished using
QuickC.
 
A dialog box appears if you have changed the text in
memory since you last saved it (or if you have not yet
saved it).
  ■ Choose <Yes> to save your changes.
  ■ Choose <No> to discard your changes.
  ■ Choose <Cancel> or press Esc to look at the program in
    memory before deciding what to do.
 
Upon exit, QuickC saves the following in the file QC.INI:
  ■ Color and tab settings
  ■ Menu setting (full or short)
  ■ Customized menu information

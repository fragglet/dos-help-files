Save As Command (3 of 3)
Use the Save As command to save the current file under a
new name. This retains the previous version of the file
under its original name. After saving, you can continue
to work on the file -- it is still in memory.
 
 ► The current file is the one displayed in the active
   Source or Notepad window. The file's name is shown at
   the top of the window.
Use the Save As dialog box to name a disk file.

QuickC assumes you are working on C source code, so it
uses the extension .C if you do not type one. To name the
file with another extension, or with no extension,
explicitly specify the file name as shown below.
 
If you type:     QuickC names the file:
 
MYFILE           MYFILE.C
MYFILE.C         MYFILE.C
MYFILE.H         MYFILE.H
MYFILE.          MYFILE.

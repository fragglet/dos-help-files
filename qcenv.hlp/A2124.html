     Error: A2124
     stack must be near with TINY model
 
     You used the .MODEL directive to specify both TINY model and the
     farStack attribute. TINY model does not support farStack, because
     all segment registers must point to the same address. Either
     remove the farStack attribute or choose another memory model.

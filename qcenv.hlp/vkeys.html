  Window Switching Keys
 
  Switch Windows                                       Key
 
  Toggle display of the output screen and
  the View Window                                      F4
  Close the active window                              CTRL+F4
  ─────────────────────────────────────────────────    ────────────
  Switch to the next window                            F6
  Switch to the previous window                        SHIFT+F6
  Zoom current window                                  CTRL+F10
  Toggle between shared windows                        CTRL+F6
  Size current window                                  CTRL+F8 UP
                                                    or CTRL+F8 DOWN

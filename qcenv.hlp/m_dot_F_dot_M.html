Merge Command (2 of 2)
The Merge command inserts the contents of another file at
the start of the line on which the cursor rests.
 
┌─────────────────────────────────┐ To merge a file from
│                                 │ the current directory:
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────│─────┐
│                                 │     │
│ C:MYPROGS ◄──current directory  │ ■ Type the file name
│                                 │   here,
│ File List:          Drives/Dirs │   OR
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │ ■ Select the file name
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │   here
│        ▲                        │    │
│        └────────────────────────│────┘
└─────────────────────────────────┘ ■ Choose <OK>
To merge a file from another directory:
┌─────────────────────────────────┐
│                                 │ ■ Type the directory
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────│── name and file name
│                                 │   here,
│ C:MYPROGS ◄──current directory  │   OR
│                                 │ ■ Select the directory
│ File List:         Drives/Dirs  │   name here,
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒ ◄│── and then
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │ ■ Select the file name
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │   here
│        ▲                        │    │
│        └────────────────────────│────┘
└─────────────────────────────────┘ ■ Choose <OK>
TIP: ".." refers to the directory one level up.

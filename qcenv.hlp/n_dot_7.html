n.7
You told QuickC to close a window before you saved
changes to the file in the window. This error also
occurs if you told QuickC to load a new file before
saving the changes to the old file.
 
Choose <Yes> to save the changes to the file.
 
Choose <No> to close the window without saving the
changes. Also choose <No> to load the new file without
saving changes.
 
Choose <Cancel> to cancel the command and resume editing
the current file.

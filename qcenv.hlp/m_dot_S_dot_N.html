Next Error
Shortcut key: SHIFT+F3
 
Use the Next Error command to quickly find and correct
syntax errors. Next Error moves the cursor to the source
code line that contains the next error. At the same time,
the Error window opens to display the corresponding error
message.
 
Another way to find the next error is to move the cursor
through the Error window's list of errors.

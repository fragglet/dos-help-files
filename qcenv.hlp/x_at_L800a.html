    The online help system contains a C language reference
    and information on every QuickC function and constant.
 
    The online help system also contains a complete assembler
    language reference and information on every BIOS and DOS
    system call.
 
    The help appears at the top of your screen in a scrollable,
    read-only window.
 
    The ESC key closes language and library help.

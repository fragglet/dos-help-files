─────Changing Segment Definitions───────────────────────────────────────────
 
     An incremental violation occurs
 
        ■ If you add, delete, or change the order of segment
          definitions
 
        ■ If you alter any DGROUP directives programmed in assembler
 
        ■ If you add or delete any modules programmed in a high-level
          language

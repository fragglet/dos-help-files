─────Adding CodeView Debugger Information───────────────────────────────────
 
     ILINK supports linking modules with Microsoft CodeView(R) debugger
     information if the CodeView information was provided when you last
     made a full link. ILINK maintains symbolic information for current
     symbols, and it adds information for any new symbols.
 
     However, if you add CodeView debugger information to a module that
     previously did not have this information, an incremental violation
     results.

─────Deleting Code Symbols──────────────────────────────────────────────────
 
     You can move or add code symbols, but an incremental violation
     occurs if you delete any code symbols from a module. Code symbols
     can be moved within a module but cannot be moved between modules.
 
     NOTE: A code symbol is the address of a function, subroutine, or
           procedure.

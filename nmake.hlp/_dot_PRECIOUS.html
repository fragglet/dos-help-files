     Syntax:  .PRECIOUS : targets
 
     Tells NMAKE not to delete <targets> if the commands that build
     them are interrupted. This directive has no effect if a command
     is interrupted and the command handles the interrupt by deleting
     the file. Separate the target names with one or more spaces or
     tabs. By default, NMAKE deletes the target if building was
     interrupted by CTRL+C or CTRL+BREAK. Multiple specifications are
     cumulative; each use of .PRECIOUS applies to the entire makefile.

─────NMAKE──────────────────────────────────────────────────────────────────
 
     Syntax:  /T
 
     Changes time stamps of command-line targets (or first target in
     the makefile if no command-line targets are specified) to the
     current time and executes preprocessing commands but does not run
     the build. Contents of target files are not modified.

     NMAKE Fatal Error U1003
 
     syntax error : '=' missing in macro substitution
 
     A macro invocation contained a colon (:), which begins a
     substitution, but it did not contain an equal sign (=).
 
     The correct form is:
 
          $(macroname:oldstring=newstring)

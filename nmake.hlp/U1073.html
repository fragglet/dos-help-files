     NMAKE Fatal Error U1073
 
     don't know how to make 'targetname'
 
     The specified target does not exist, and there is no command to
     execute or inference rule to apply.
 
     One of the following may be a solution:
 
        ■ Check the spelling of the target name.
 
        ■ If <targetname> is a pseudotarget, specify it as a target
          in another description block.
 
        ■ If <targetname> is a macro invocation, be sure it does not
          expand to a null string.

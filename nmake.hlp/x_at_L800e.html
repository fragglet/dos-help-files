─────NMAKE──────────────────────────────────────────────────────────────────
 
     Syntax:  /N
 
     Displays but does not execute the commands that would be executed
     by the build. Preprocessing commands are executed. This option is
     useful for debugging makefiles and checking out-of-date targets.
 
     To set or clear /N for part of a makefile, use !CMDSWITCHES.
     See: ◄!CMDSWITCHES Directive►

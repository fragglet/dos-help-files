.SILENT Directive
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────NMAKE──────────────────────────────────────────────────────────────────
 
     Syntax:  .SILENT :
 
     Suppresses display of command lines. By default, NMAKE echos each
     command before it executes it. .SILENT is in effect from the place
     it is specified to the end of the makefile.
     See: ◄Commands►
 
     To turn .SILENT off, use the !CMDSWITCHES preprocessing directive.
     See: ◄!CMDSWITCHES Directive►
 
     To suppress display of a single command, use a command modifier.
     See: ◄At-Sign (@) Modifier►
 
     To suppress the command display for an entire session, run NMAKE
     with the /S option.
     See: ◄/S Option►

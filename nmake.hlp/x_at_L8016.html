     Syntax:  /X filename
 
     Sends all error output (from both NMAKE and the executed commands)
     to <filename>, which can be a file or a device. Zero or more
     spaces or tabs precede <filename>. If you supply a dash (-)
     instead of a filename, error output is sent to standard output. By
     default, NMAKE sends errors to standard error.

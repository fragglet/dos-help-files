     Syntax:  /Q
 
     Checks time stamps of targets that would be updated by the build
     but does not run the build. NMAKE returns a zero exit code if the
     targets are up-to-date and a nonzero exit code if any target is
     out-of-date. Only preprocessing commands are executed. This option
     is useful when running NMAKE from a batch file.
     See: ◄NMAKE Exit Codes►

     NMAKE Fatal Error U1094
 
     syntax error : only (NO)KEEP allowed here
 
     Something other than KEEP or NOKEEP appeared after the closing
     set of angle brackets (<<) specifying an inline file. Only KEEP,
     NOKEEP, or a newline character may follow the angle brackets. No
     spaces, tabs, or other characters may appear.
 
     KEEP preserves the inline file on disk. NOKEEP deletes the file
     after the NMAKE session. The default is NOKEEP.

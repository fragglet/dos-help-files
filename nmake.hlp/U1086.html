     NMAKE Fatal Error U1086
 
     inference rule cannot have dependents
 
     The colon (:) in an inference rule must be followed by one of the
     following:
 
        ■ A newline character
 
        ■ A semicolon(;), which can be followed by a command
 
        ■ A number sign (#), which can be followed by a comment

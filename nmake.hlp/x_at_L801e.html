Long Filenames
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────NMAKE──────────────────────────────────────────────────────────────────
 
     You can use long filenames if they are supported by your file
     system. However, you must enclose the name in double quotation
     marks, as in the following dependency line:
 
          all : "VeryLongFileName.exe"

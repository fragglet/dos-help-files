     NMAKE Fatal Error U1001
 
     syntax error : illegal character 'character' in macro
 
     The given character appeared in a macro but was not a letter,
     number, or underscore (_).
 
     If the colon (:) is omitted in a macro expansion, the following
     error occurs:
 
          syntax error : illegal character '=' in macro

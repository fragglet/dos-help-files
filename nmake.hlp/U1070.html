     NMAKE Fatal Error U1070
 
     cycle in macro definition 'macroname'
 
     The given macro definition contained a macro whose definition
     contained the given macro. Circular macro definitions are invalid.
 
     For example, the following macro definitions
 
          ONE=$(TWO)
          TWO=$(ONE)
 
     cause the following error:
 
          cycle in macro definition 'TWO'

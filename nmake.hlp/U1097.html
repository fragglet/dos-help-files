     NMAKE Fatal Error U1097
 
     filename-parts syntax requires dependent
 
     The current dependency does not have either an explicit dependent
     or an implicit dependent. Filename-parts syntax, which uses the
     percent (%) specifier, represents components of the first
     dependent of the current target.

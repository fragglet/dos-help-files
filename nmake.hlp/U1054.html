     NMAKE Fatal Error U1054
 
     cannot create inline file 'filename'
 
     NMAKE failed to create the given inline file.
 
     One of the following may be a cause:
 
        ■ A file by that name exists with a read-only attribute.
 
        ■ The disk is full.

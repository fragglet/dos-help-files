     Syntax:  /M
 
     Swaps NMAKE to disk to conserve extended or expanded memory.
 
     By default, NMAKE leaves room for commands to be executed in low
     memory by swapping itself to extended memory (if enough space
     exists there) or to expanded memory (if there is not sufficient
     extended memory but there is enough expanded memory) or to disk.
     The /M option tells NMAKE to ignore any extended or expanded
     memory and to swap only to disk.

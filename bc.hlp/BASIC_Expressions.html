BASIC Expressions
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     BASIC expressions can contain the following elements when used in
     CodeView Command or Watch windows:
 
     ◄Operators►
     ◄Constants►
     ◄Symbols►
     ◄Strings►
     ◄Arrays►
     ◄Records (User-Defined Types)►
     ◄Intrinsic Functions►
 
     NOTE:    When a BASIC expression is used as an argument with a
              CodeView command that takes multiple arguments, the expression
              cannot contain spaces.
 
     In addition to the format specifiers that can be used with the Display
     Expression (?) command, BASIC expressions can use the letter m as the
     format specifier for expressions of the CURRENCY data type. See the
     ◄Display Expression Command► for information on the
     other format specifiers.

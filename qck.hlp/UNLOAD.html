Unload Event
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Occurs when a form is about to be unloaded from memory. The Unload event is
 triggered either by user action - for example, closing the form using the
 Control-menu box - or by the UNLOAD Statement.
 
    SUB Form_Unload (Cancel AS INTEGER)
 
    ■ Cancel  Any nonzero value; setting Cancel to any nonzero value
              prevents the form from being unloaded
 
 Applies To
    ◄Form►

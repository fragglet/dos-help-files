WRITE Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sends data to the screen; not available when forms are showing. The values
 written on the screen are separated by commas, with double quotation marks
 around strings and no spaces around numbers.
 
    WRITE [expressionlist]
 
    ■ expressionlist    Specifies one or more values to write

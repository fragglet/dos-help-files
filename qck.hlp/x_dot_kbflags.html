 Keyboard Flag Table
 
 ■ Forms and controls trap key events using the KeyUp and KeyDown events.
   See: ◄KeyDown Event►  ◄KeyUp Event►
 
 ■ The table below defines Visual Basic keyboard flag codes used in
   traditional Basic key event-trapping routines. See: ◄ON KEY Statement►
 
         Value          Key
         ══════════     ═══════════════════════════════════════════════════
         0              No keyboard flag
         1, 2, or 3     Either Shift key
         4              Ctrl
         8              Alt
         32             Num Lock
         64             Caps Lock
         128            Keys on the 101-key keyboard that are duplicates of
                        keys on smaller keyboards (such as Alt and Del)
         ───────────────────────────────────────────────────────────────────

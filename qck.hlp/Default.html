 Determines which command button is the default button on a form.
 
    [form.]commandbutton.Default[ = boolean%]
 
    ■ boolean%     Valid settings are:
 
                   Value        Description
                   ═════════    ════════════════════════════════════════════
                   True (-1)    Specifies the control as the default command
                                button; only one button on a form can be the
                                default button
                   False (0)    (Default) Disables this functionality

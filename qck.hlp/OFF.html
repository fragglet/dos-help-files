 OFF Keyword (Different Uses)
 
 ■ The OFF keyword turns off the display of function-key assignments when
   used with the KEY assignment statement, or disables event trapping when
   used with event OFF statements, such as:
 
   • COM OFF       • KEY OFF      • PEN OFF      • PLAY OFF
   • STRIG OFF     • TIMER OFF    • UEVENT OFF
 
   With the exception of KEY OFF, these event statements can be used with
   object-specific events.  See: ◄Object Events Summary►

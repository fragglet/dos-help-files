SSEGADD Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the far address of a string variable (both segment and offset); used
 in mixed-language programming to obtain far addresses before passing far
 strings to procedures written in other languages.
 
    SSEGADD(stringvariable$)
 
    ■ stringvariable$   String variable for which you want to find an
                        address; can be either a simple string variable or
                        a single element of a string array; cannot be a
                        fixed-length string argument

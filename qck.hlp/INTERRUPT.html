Interrupt[X] Routine
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Enables Visual Basic programs to perform MS-DOS system calls. Requires
 VBDOS.QLB (VBDOS.LIB when compiling); include VBDOS.BI.
 
    CALL Interrupt[X] (interruptnum%, inregs, outregs)
 
    ■ interruptnum%     MS-DOS interrupt number between 0 and 255, inclusive
 
    ■ inregs            Register values before the interrupt is performed;
                        declare inregs as type RegType (for Interrupt) or
                        type RegTypeX (for InterruptX)
 
    ■ outregs           Register values after the interrupt is performed;
                        declare outregs as type RegType (for Interrupt) or
                        type RegTypeX (for InterruptX)

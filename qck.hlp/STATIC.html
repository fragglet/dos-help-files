STATIC Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Makes simple variables or arrays local to a SUB or FUNCTION procedure or a
 DEF FN statement and preserves values between calls.
 
    STATIC variable[()] [AS type] [,variable[()] [AS type]]...
 
    ■ variable    Any valid array or variable name
 
    ■ ()          Indicates array declaration
 
    ■ AS type     (Default - SINGLE) Declares the data type of an array or
                  variable; use a separate AS type clause for each variable
                  defined

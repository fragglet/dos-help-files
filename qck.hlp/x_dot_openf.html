 Adding an Existing Form
 
 ■ To add an existing form to the current project:
 
   1. Choose Add File (programming environment) or Add Form (Form Designer)
      from the File menu
 
   2. Select the form to add to the current project
 
   3. Press Enter or choose Edit in Active to modify the form

ListCount Property
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the number of items in the list portion of a control; not available
 at design time and read-only at run time. When ListCount = 0, there are no
 items in the list.
 
    [form.]ctlname.ListCount
 
 Applies To

    ◄Drive List Box Control►           ◄File List Box Control►
    ◄List Box Control►

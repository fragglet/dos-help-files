 Determines if a check mark is displayed next to a menu command.
 
    [form.]menuitem.Checked[ = boolean%]
 
    ■ boolean%   Valid settings are:
 
                 Value       Description
                 ═════════   ═══════════════════════════════════════════════
                 True (-1)   Places a check mark next to the menu command,
                             indicating that the item is selected
                 False (0)   (Default) Removes a check mark from a menu
                             command

 Toolbox
 
 ■ The Toolbox (or Tools window) displays the controls that you can place on
   a form at design time. The Toolbox is visible only when you're working in
   the Form Designer. By default, it is displayed on the left side of the
   screen.
 
 ■ The Tools menu on the Form Designer menu bar contains the same set of
   tools as the Toolbox. This menu can be used with or in place of the
   Toolbox. It provides convenient keyboard access to tools and allows you to
   conserve screen space by closing the Toolbox window.

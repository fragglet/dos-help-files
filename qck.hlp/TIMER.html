Timer Event
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Occurs when a preset interval for a timer control has elapsed. The
 interval's frequency is stored in the control's Interval property, which
 specifies the length of time in milliseconds.
 
    SUB ctlname_Timer ([Index AS INTEGER])
 
    ■ Index    Uniquely identifies a control in a control array
 
 Applies To

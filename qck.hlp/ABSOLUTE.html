Absolute Routine
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Transfers control to a machine-language procedure. Requires VBDOS.QLB
 (VBDOS.LIB when compiling); include VBDOS.BI.
 
    CALL Absolute ([argumentlist,] offset%)
 
    ■ argumentlist      Arguments passed to a machine-language procedure as
                        offsets from the current data segment
 
    ■ offset%           Offset from the current code segment, set by DEF SEG,
                        to the starting location of the procedure

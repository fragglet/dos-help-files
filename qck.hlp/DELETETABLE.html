 Removes a table and any indexes based on the table from an ISAM database.
 Requires PROISAM.EXE or PROISAMD.EXE in the programming environment
 (VBDOS.EXE).
 
    DELETETABLE database$, tablename$
 
    ■ database$      Optional device, followed by a file name or path
                     conforming to MS-DOS naming conventions; file name must
                     be the name of an ISAM file
 
    ■ tablename$     Name of existing ISAM table (same as name used in
                     OPEN statement)

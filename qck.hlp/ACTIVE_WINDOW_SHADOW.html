ACTIVE_WINDOW_SHADOW Constant
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Determines if the active form is displayed with a shadow.
 
    SCREEN.ControlPanel({ACTIVE_WINDOW_SHADOW | subscript%})[ = boolean%]
 
    ■ subscript%   ACTIVE_WINDOW_SHADOW has a value of 3
 
    ■ boolean%     Valid settings are:
 
                   Value        Description
                   ═════════    ════════════════════════════════════════════
                   True (-1)    (Default) Shadow is displayed
                   False (0)    No shadow is displayed
 
 Applies To
    ◄ControlPanel Property►            ◄SCREEN Object►

ALIAS Clause
                                                 ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 The ALIAS clause in a DECLARE statement specifies that the procedure being
 called has another name in the library file or .OBJ file. For example, you
 might use ALIAS where the name of a procedure in a Quick library is
 different from the name you use in your program to invoke that procedure:
 
    name ALIAS "name-in-Quick-Library"

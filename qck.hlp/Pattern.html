 Determines which file names are displayed in a file list box at run time;
 may include wildcards (? or *).
 
    [form.]filelistbox.Pattern[ = filespec$]
 
    ■ filespec$    Any valid file specification; default is *.*
 
 Applies To
    ◄File List Box Control►

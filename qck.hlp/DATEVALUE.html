DATEVALUE Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns a serial number that represents the date of the argument.
 
    DATEVALUE(date$)
 
    ■ date$    Any valid date string between January 1, 1753 and December 31,
               2078, inclusive

 Creates an index consisting of one or more columns of an ISAM table.
 Requires PROISAM.EXE or PROISAMD.EXE in the programming environment
 (VBDOS.EXE).
 
    CREATEINDEX [#]filenumber%,indexname$,unique%,columnname$[,columnname$]
 
    ■ filenumber%    Number used in the OPEN statement to open the table;
                     integer expression with a value between 1 and 255,
                     inclusive
 
    ■ indexname$     Valid table index
 
    ■ unique%        Nonzero value indicates a unique index
 
    ■ columnname$    Valid table column name

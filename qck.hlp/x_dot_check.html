Check Box Control
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►

 Displays an option that can be toggled (on/off, true/false,
 checked/unchecked); can also be grayed. An X in a check box
 indicates that the option is selected.

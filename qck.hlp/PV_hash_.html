 Returns the present value, or lump sum amount that a series of payments to
 be paid in the future is worth now. Requires FINANCE.QLB (FINANCE.LIB when
 compiling); include FINANCE.BI.
 
    PV# (rate#, nper#, pmt#, fv#, type%, status%)
 
    ■ rate#      Interest rate per period (for example, .12)
 
    ■ nper#      Number of payment periods in the annuity
 
    ■ pmt#       Payment to be made each period
 
    ■ fv#        Future value
 
    ■ type%      Integer expression:
                 0 = end-of-period payment
                 1 = beginning-of-period payment
 
    ■ status%    Variable that returns a value indicating calculation
                 failure or success:
                 0 = success
                 1 = failure

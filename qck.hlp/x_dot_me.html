 Editing Keys
 
   To:                                            Press:
   ═══════════════════════════════════════════    ══════════════════════════
   Beep                                           Ctrl+B
   Beginning of file                              Ctrl+Home or Ctrl+Q,R
   Beginning of line                              Home or Ctrl+Q,S
   Cancel                                         Esc
   Change text                                    Alt+S,C or Ctrl+Q,A
   Character Left                                 Ctrl+S or Left Arrow
   Character Right                                Ctrl+D or  Right Arrow
   Copy selected text onto the CLIPBOARD          Ctrl+C or Ctrl+Ins
   Cut selected text onto the CLIPBOARD           Ctrl+X or Shift+Del
   Cut current line to CLIPBOARD                  Ctrl+Q,L or Ctrl+Y
   Cut from cursor to end of line to CLIPBOARD    Ctrl+Q,Y
   Delete character to left of cursor             Ctrl+H or Backspace
   Delete character to the right of cursor        Del or Ctrl+G
   Delete selected text                           Del
   Delete word                                    Ctrl+T
   Edit Undo                                      Alt+BackSpace
   End of line                                    End or Ctrl+Q,D
   End of program                                 Ctrl+End or Ctrl+Q,C
   End of screen                                  Ctrl+Q,X
   Erase (to EOL)                                 Ctrl+Q,Y
   Find selected or entered text                  Ctrl+\ or Ctrl+Q,F
   Go to book mark 0                              Ctrl+Q,0
   Go to book mark 1                              Ctrl+Q,1
   Go to book mark 2                              Ctrl+Q,2
   Go to book mark 3                              Ctrl+Q,3
   Home Screen                                    Ctrl+Q,E
   Insert Mode (toggle)                           Ins
   Line Down                                      Down Arrow
   Line Up                                        Ctrl+E or Up Arrow
   Menu                                           Alt or F11
   Move the text one tab stop to the left         Shift+Tab
   Move the text one tab stop to the right        Tab
   New Line                                       Ctrl+M
   Next Line                                      Ctrl+J or Ctrl+Enter
   Page Down                                      PgDn
   Page Left                                      Ctrl+PgUp
   Page Right                                     Ctrl+PgDn
   Page Up                                        PgUp or Ctrl+R
   Paste CLIPBOARD contents at cursor location    Ctrl+V or Shift+Ins
   Scroll Down                                    Ctrl+Z or Ctrl+Down Arrow
   Scroll Up                                      Ctrl+W or Ctrl+Up Arrow
   Set book mark 0                                Ctrl+K,0
   Set book mark 1                                Ctrl+K,1
   Set book mark 2                                Ctrl+K,2
   Set book mark 3                                Ctrl+K,3
   SplitLine                                      Ctrl+N
   Word Left                                      Ctrl+A or Ctrl+Left Arrow
   Word Right                                     Ctrl+F or Ctrl+Right Arrow
   ─────────────────────────────────────────────────────────────────────────

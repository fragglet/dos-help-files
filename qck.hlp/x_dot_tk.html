 Toolbox Keys
 
 ■ The following key combinations work in the Toolbox:
 
   To:                                 Press:
   ═══════════════════════════════     ═════════════════════════════════════
   Cycle backward through tools        Up Arrow or Left Arrow
   Cycle forward through tools         Down Arrow or Right Arrow
   Place highlight control on form     Enter, or DblClick

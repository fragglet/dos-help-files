FD Command-Line Options
                                                 ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 FD.EXE starts the Form Designer utility; used to create and manipulate
 objects.
 
    FD [options] [formfile]
 
    ■ formfile      Form to load at startup; can include drive and path
 
    ■ options       Command-line options available with FD.EXE:
 
      Option        Description
      ═══════════   ════════════════════════════════════════════════════════
      /?            Displays available command-line options
      /B            Allows use of monochrome monitor with color graphics card
      /H            Displays maximum number of lines available with hardware
      /NOHI         Allows use of monitor without high-intensity support

──────────────────────────────────────────────────────────────────────────────
 Gives a SUB or FUNCTION procedure access to variables declared at the module
 level without passing them as parameters.
 
    SHARED variable[()] [AS type] [,variable[()] [AS type]]...
 
    ■ variable      Any valid array or variable name
 
    ■ ()            Indicates array declaration
 
    ■ AS type       (Default - SINGLE) Declares the data type of an array or
                    variable; use a separate AS type clause for each variable
                    defined

PrintTarget Property
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Specifies the target device or file name for PRINT method or PRINTFORM
 method output.
 
    PRINTER.PrintTarget = target$
 
    ■ target$    Valid settings are:
 
                 Value                 Description
                 ═══════════════════   ═════════════════════════════════════
                 LPT1, LPT2, or LPT3   Valid printer port
                 filename              Valid MS-DOS file name or device

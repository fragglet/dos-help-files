UNLOAD Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Removes a form or control array element from memory.
 
    UNLOAD {form | ctlname[index%]}
 
    ■ form         Name of form to unload
 
    ■ ctlname      Control array from which an element will be removed
 
    ■ index%       Indicates the position in the control array

 Adds an item to a list box or combo box at run time.
 
    [form.]{listbox | combobox}.ADDITEM item$ [, index%]
 
    ■ item$        String expression to add to the list
 
    ■ index%       Integer specifying the position at which item$ is
                   inserted in the list; an index% of 0 represents the first
                   position
 
 Applies To
    ◄Combo Box Control►                ◄List Box Control►

 Returns a specified number of leftmost characters (LEFT$) or rightmost
 characters (RIGHT$) in a string.
 
    LEFT$(stringexpression$, n%)
    RIGHT$(stringexpression$, n%)
 
    ■ stringexpression$    Any string expression
 
    ■ n%                   Number of characters to return, beginning with
                           the leftmost or rightmost string character

 Exits a DO or FOR loop, a FUNCTION or SUB procedure, or a DEF FN statement.
 
    EXIT {DEF | DO | FOR | FUNCTION | SUB}
 
    ■ DEF         Exits a DEF FN statement
 
    ■ DO          Exits a DO...LOOP statement
 
    ■ FOR         Exits a FOR...NEXT loop
 
    ■ FUNCTION    Exits a FUNCTION procedure
 
    ■ SUB         Exits a SUB procedure

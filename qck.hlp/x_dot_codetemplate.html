 Code Template
 
 ■ A code template in Visual Basic consists the first and last lines of a
   procedure. For example:
 
         SUB
         .
         .
         .
         (statementblock)
         .
         .
         .
         END SUB
 
 ■ Note: No code is executed until you add at least one statement.

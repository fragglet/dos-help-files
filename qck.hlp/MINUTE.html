MINUTE Function
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Evaluates a date/time serial number and returns the minute, a value between
 0 and 59, inclusive.
 
    MINUTE(serial)
 
    ■ serial    Serial number in the range -53,688 to 65,380 that represents
                a date and/or time

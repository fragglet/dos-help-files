CurrentX, CurrentY Properties
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sets or returns the current horizontal position or column number (CurrentX),
 or vertical position or row number (CurrentY).
 
    [form.][picturebox.]CurrentX[ = coordinate%]
    [form.][picturebox.]CurrentY[ = coordinate%]
 
    ■ coordinate%       Valid settings are:
 
                        Value     Description
                        ═══════   ══════════════════════════════════════════
                        0 - 254   Valid X or Y coordinate where 0 is the
                                  default

    ◄Form►                             ◄Picture Box Control►

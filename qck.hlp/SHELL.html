SHELL Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Suspends execution of a Visual Basic program, runs a .COM, .EXE, or .BAT
 program, or an MS-DOS command, and resumes execution of the Visual Basic
 program at the statement following the SHELL statement.
 
    SHELL [commandstring$]
 
    ■ commandstring$    Name of a program or executable file

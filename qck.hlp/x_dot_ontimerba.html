ON TIMER Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Branches to the specified routine when n& seconds have passed; can be used
 with object-specific events. The Timer control and the Timer event are
 easier to use with forms. See: ◄Timer Control►  ◄Timer Event►
 
    ON TIMER(n&) GOSUB {linenumber | linelabel}
 
    ■ n&               Numeric expression with a value between 1 and 86,400,
                       inclusive (24 hours); the number of seconds in the
                       time interval
 
    ■ linenumber       First line of an event-handling routine to execute
      linelabel        when n% seconds have passed

 Determines the type of combo box and the behavior of the list box portion
 of the control; read-only at run time.
 
    [form.]combobox.Style
 
    ■ Valid settings are:
 
      Value   Type       Includes         User Action
      ═════   ════════   ══════════════   ══════════════════════════════════
      0       Dropdown   (Default) Edit   Display list by clicking the down
              Combo      area and a       scroll arrows. The user can select
                         list             from the list or type in the edit
                                          area. The list closes when the
                                          user makes a selection.
 
      1       Simple     Edit area and    Select from a list or type in the
              Combo      a list always    edit area.
                         displayed
 
      2       Dropdown   Drop-down list   Click scroll arrow to display the
              List       with no edit     list. This style permits selection
                         capabilities     of a list item only. The list
                                          closes when the user makes a
                                          selection.

Sorted Property
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Determines if the elements of a combo box or list box are automatically
 sorted; read-only at run time.
 
    [form.]{combobox. | listbox.}Sorted
 
    ■ Valid settings are:
 
      Value        Description
      ═════════    ═════════════════════════════════════════════════════════
      True (-1)    List items are sorted alphabetically (case-insensitive)
      False (0)    (Default) Specifies no sorting of list items
 
 Applies To

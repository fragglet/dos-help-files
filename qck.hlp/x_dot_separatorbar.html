 Separator Bar
 
 ■ A separator bar divides commands into logical groups on a long menu. For
   example:
 
         ░░File░░░░░░░...
         ┌───────────┐
         │ New...    │
         │ Open...   │
         │ Close     │
         │ Save      │
         ├───────────┤◄──── Separator bar
         │ Exit      │
         └───────────┘
 
 ■ At design time, a separator bar is specified in the Menu Design window by
   selecting the Separator check box.

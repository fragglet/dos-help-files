TIMER Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Enables, disables, or suspends timer event trapping; can be used with
 object-specific events. The Timer control and the Timer event are easier
 to use with forms. See: ◄Timer Control►  ◄Timer Event►
 
    TIMER {ON | OFF | STOP}
 
    ■ ON    Enables event trapping
 
    ■ OFF   Disables event trapping; if an event takes place, it is ignored
 
    ■ STOP  Suspends event trapping; any events that occur are remembered
            and processed as soon as a TIMER ON statement is executed

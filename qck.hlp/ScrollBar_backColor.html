SCROLLBAR_BACKCOLOR, SCROLLBAR_FORECOLOR Constants
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sets or returns the background and foreground colors for all non-attached
 scroll bars in a program.
 
    SCREEN.ControlPanel({SCROLLBAR_BACKCOLOR | subscript%})[ = color%]
    SCREEN.ControlPanel({SCROLLBAR_FORECOLOR | subscript%})[ = color%]
 
    ■ subscript%  SCROLLBAR_BACKCOLOR has a value of 13;
                  SCROLLBAR_FORECOLOR has a value of 14
 
    ■ color%      Valid color settings are:
 
                  Value      Description
                  ══════     ═══════════════════════════════════════════════
                  0 - 15     Valid color attribute
                  7          (Default - SCROLLBAR_BACKCOLOR) White
                  0          (Default - SCROLLBAR_FORECOLOR) Black

GETINDEX$ Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
                 ISAM available in Professional Edition only
──────────────────────────────────────────────────────────────────────────────
 Takes the number of an open ISAM table and returns the name of the current
 index. Requires PROISAM.EXE or PROISAMD.EXE in the programming environment
 (VBDOS.EXE).
 
    GETINDEX$ (filenumber%)
 
    ■ filenumber%    Number used in the OPEN statement to open the table;
                     integer expression with a value between 1 and 255,
                     inclusive

IF...END IF Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Allows conditional execution based upon the evaluation of a boolean
 expression.
 
 Syntax 1: Block Form
    IF condition-1 THEN
         [statementblock-1]
    [ELSEIF condition-2 THEN
         [statementblock-2]]...
    [ELSE
         [statementblock-n]]
    END IF
 
 Syntax 2: Single-Line Form
    IF condition THEN statements [ELSE statements]
 
    ■ condition         Any expression that can be evaluated as True
                        (nonzero) or False (zero)
 
    ■ statementblock-1  Any number of statements on any number of lines,
      ...               separated by colons (:)
      statementblock-n

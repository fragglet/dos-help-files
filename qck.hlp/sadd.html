SADD Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the address of a string as an offset (2-byte integer) from the
 current data segment; used in mixed-language programming to obtain far
 addresses before passing far strings to procedures written in other
 languages.
 
    SADD(stringvariable$)
 
    ■ stringvariable$   String for which you want to find the offset; can be
                        either a simple string variable or a single element
                        of a string array; cannot be a fixed-length string
                        argument

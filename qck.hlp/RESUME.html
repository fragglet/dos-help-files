RESUME Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Resumes program execution after an error-handling routine is finished.
 
    RESUME [{line | NEXT}]
 
    ■ line    Label or line number where execution resumes; if 0 or
              omitted, execution resumes with the statement that caused
              the error
 
    ■ NEXT    Resumes execution at the statement following the statement
              that caused the error

MENU_SELECTED_BACKCOLOR, MENU_SELECTED_FORECOLOR Constants
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sets or returns the background or foreground color of the selected menu
 item.
 
    SCREEN.ControlPanel({MENU_SELECTED_BACKCOLOR | subscript%})[ = color%]
    SCREEN.ControlPanel({MENU_SELECTED_FORECOLOR | subscript%})[ = color%]
 
    ■ subscript%   MENU_SELECTED_BACKCOLOR has a value of 11;
                   MENU_SELECTED_FORECOLOR has a value of 12
 
    ■ color%       Valid color settings are:
 
                   Value      Description
                   ══════     ══════════════════════════════════════════════
                   0 - 15     Valid color attribute
                   0          (Default - MENU_SELECTED_BACKCOLOR) Black
                   7          (Default - MENU_SELECTED_FORECOLOR) White

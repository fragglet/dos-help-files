ACTIVE_BORDER_BACKCOLOR, ACTIVE_BORDER_FORECOLOR Constants
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sets or returns the background or foreground border color of the active
 window.
 
    SCREEN.ControlPanel({ACTIVE_BORDER_BACKCOLOR | subscript%})[ = color%]
    SCREEN.ControlPanel({ACTIVE_BORDER_FORECOLOR | subscript%})[ = color%]
 
    ■ subscript%   ACTIVE_BORDER_BACKCOLOR has a value of 1;
                   ACTIVE_BORDER_FORECOLOR has a value of 2
 
    ■ color%       Valid color settings are:
 
                   Value      Description
                   ══════     ══════════════════════════════════════════════
                   0 - 15     Valid color attribute
                   7          (Default - ACTIVE_BORDER_BACKCOLOR) White
                   0          (Default - ACTIVE_BORDER_FORECOLOR) Black
 
 Applies To
    ◄ControlPanel Property►                  ◄SCREEN Object►

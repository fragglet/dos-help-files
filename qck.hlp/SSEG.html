SSEG Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the segment address for a string or 0 if the argument is a null
 string; used in mixed-language programming to obtain far addresses before
 passing far strings to procedures written in other languages.
 
    SSEG(stringvariable$)
 
    ■ stringvariable$    String variable for which you want to find the
                         segment address; can be either a simple string
                         variable or a single element of a string array;
                         cannot be a fixed-length string argument

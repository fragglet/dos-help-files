SPC Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Skips a specified number of spaces in a PRINT, LPRINT, or PRINT # statement,
 or in a PRINT method.
 
    SPC(n%)
 
    ■ n%  Number of spaces to skip; a value in the range 0 to 32,767,
          inclusive

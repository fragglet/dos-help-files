 Run-Time Error Codes
 
 ■ Run-time errors can occur while an application is running, either within
   the Visual Basic environment or as a stand-alone executable file. Some of
   these errors can also occur during design time or compile time.
 
 ■ You can test and respond to these errors using the ON ERROR statement and
   the ERR function. See: ◄ERR Function►  ◄ON ERROR Statement►
 
 ■ While developing in Visual Basic, you can get help on any error message by
   pressing the F1 key.

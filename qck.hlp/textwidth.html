 Returns the height in lines (TEXTHEIGHT) or length in characters (TEXTWIDTH)
 of a text string.
 
    [form. | picturebox.]TEXTHEIGHT(stringexpression$)
    [form. | picturebox.]TEXTWIDTH(stringexpression$)
 
    ■ stringexpression$      Text string for which the height or length is
                             determined

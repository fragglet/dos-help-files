IOCTL$ Function, IOCTL Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns current status information from a device driver (IOCTL$); transmits
 a control data string to a device driver (IOCTL).
 
    IOCTL$([#]filenumber%)
    IOCTL [#]filenumber%, string$
 
 
    ■ filenumber%    Number of an open device
 
    ■ string$        Control string sent to the device

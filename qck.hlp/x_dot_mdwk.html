 Menu Design Window Keys
 
 ■ General Menu design:
 
   To:                                                         Press:
   ═════════════════════════════════════════════════════════   ═════════════
   Accept input menu items and close Menu Design window        Alt+D
   Cycle backward through the boxes and buttons in the window  Shift+Tab
   Cycle forward through items in menu item list box           Enter
   Cycle forward through the boxes and buttons in the window   Tab
   Delete menu item                                            Alt+T
   Help on Menu Design window items                            Alt+H
   Insert menu item                                            Alt+I
   Move menu item one line down                                Alt+B
   Move menu item one line up                                  Alt+U
   Move menu item to a higher level in a hierarchical menu     Alt+L
   Move menu item to a lower level in a hierarchical menu      Alt+R
   Move focus to Caption box                                   Alt+P
   Move focus to CtlName box                                   Alt+M
   Move focus to Index box                                     Alt+X
   Move focus to Shortcut Key list box                         Alt+K
   Move focus to Tag box                                       Alt+G
   Select next item in menu item list box                      Alt+N
   Select/deselect Checked property                            Alt+C
   Select/deselect Enabled property                            Alt+E
   Select/deselect Separator property                          Alt+S
   Select/deselect Visible property                            Alt+V
   ─────────────────────────────────────────────────────────────────────────
 
 ■ When the Shortcut Key list box has the focus:
 
   To:                                              Press:
   ════════════════════════════════════════════     ════════════════════════
   Cycle backward through the items in the list     Up Arrow or Left Arrow
   Cycle forward through the items in the list      Down Arrow or Right Arrow
   Move to the first item displayed in the list     Page Up
   Move to the first item in the list               Home
   Move to the last item displayed in the list      Page Down
   Move to the last item in the list                End
   Open or close the list                           F4 or Alt+Down Arrow or
                                                    Alt+Up Arrow
   ─────────────────────────────────────────────────────────────────────────

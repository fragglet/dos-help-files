 Output Window
 
 ■ The Output window displays the actual output screen within a display
   window.
 
 ■ While you cannot change items displayed in the Output window, it
   does enable you to:
   • View the output screen from within the programming environment
   • Copy items from the output screen to the programming environment
   • Print output displayed within the window
 
 ■ The output window (while useful) can be somewhat limiting. To display
   entire output screen, choose Output Screen from the View menu
   or press F4. When your application is running the full screen is used.

 Returns the straight-line depreciation of an asset for a single period.
 Requires FINANCE.QLB (FINANCE.LIB when compiling); include FINANCE.BI.
 
    SLN# (cost#, salvage#, life#, status%)
 
    ■ cost#       Initial cost of the asset
 
    ■ salvage#    Value at the end of the useful life of the asset
 
    ■ life#       Useful life of the asset
 
    ■ status%     Variable that returns a value indicating calculation
                  failure or success:
                  0 = success
                  1 = failure

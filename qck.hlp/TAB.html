TAB Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Moves the text cursor to a specified print position when used in the PRINT,
 LPRINT, and PRINT # statements, or with the PRINT method.
 
    TAB(column%)
 
    ■ column%      Column number of the new print position

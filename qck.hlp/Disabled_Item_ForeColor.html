DISABLED_ITEM_FORECOLOR Constant
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Determines the foreground color of disabled items.
 
    SCREEN.ControlPanel({DISABLED_ITEM_FORECOLOR | subscript%})[ = color%]
 
    ■ subscript%   DISABLED_ITEM_FORECOLOR has a value of 8
 
    ■ color%       Valid color settings are:
 
                   Value      Description
                   ══════     ══════════════════════════════════════════════
                   0 - 15     Valid color attribute
                   8          (Default) Gray
 
 Applies To
    ◄ControlPanel Property►                   ◄SCREEN Object►

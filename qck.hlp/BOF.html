 Tests whether the current position is at the beginning of an ISAM table;
 returns True (nonzero) if the current position is before the first record
 according to the current index. Requires PROISAM.EXE or PROISAMD.EXE in the
 programming environment (VBDOS.EXE).
 
    BOF(filenumber%)
 
    ■ filenumber%   Number used in the OPEN statement to open the table;
                    integer expression with a value between 1 and 255,
                    inclusive

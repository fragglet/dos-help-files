 Opening an Existing Project
 
 ■ To open an existing project:
 
   1. From the File menu, choose Open Project
 
   2. Type or select the name of the project you want to open
 
   3. Press Enter or choose the OK button
 
 ■ Since you can have only one project open at a time, Visual Basic prompts
   you to save and close the current project before opening a new one.

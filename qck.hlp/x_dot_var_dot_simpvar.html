 Declaring Simple Variables
 
 ■ Simple variables define a single number, string, or record. You can
   specify simple variable types by using either:
   • A type-declaration suffix (MyNum%)
   • An AS declaration statement (SUB MyProc AS INTEGER)
   • A DEFtype statement (DEFINT A-Z; SUB MyProc)
 
 ■ A group of simple variables can be defined by declaring an array.

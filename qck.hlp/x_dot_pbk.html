 Properties Bar Keys
 
   To:                                              Press:
   ══════════════════════════════════════════════   ════════════════════════
   Move the cursor into the Value box               F2
   Move cursor to the property combo box            Shift + F12
   Open or close the list                           F4 or Alt+Down Arrow or
                                                    Alt+Up Arrow
   Cycle backward through the items in the list     Up Arrow or Left Arrow
   Cycle focus between Value box and Property box   Tab
   Cycle forward through the items in the list      Down Arrow or Right Arrow
   Move the focus to the Property box               Tab
   Move to the first item displayed in the list     Page Up
   Move to the first item in the list               Home
   Move to the last item displayed in the list      Page Down
   Move to the last item in the list                End
   Properties Bar/Menu Bar display (toggle)         F10
   Select highlighted item from list                Enter
   Select item from list                            Item # if enumerated or
                                                    1st letter of item
   Undo the last editing action                     Esc

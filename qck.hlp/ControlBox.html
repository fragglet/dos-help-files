 Determines if a Control-menu box appears on a form at run time; read-only
 at run time.
 
    [form.]ControlBox
 
    ■ Valid settings are:
 
      Value        Description
      ═════════    ═════════════════════════════════════════════════════════
      True (-1)    (Default) Displays Control-menu box
      False (0)    Removes Control-menu box

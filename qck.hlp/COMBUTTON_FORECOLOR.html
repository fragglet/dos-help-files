COMBUTTON_FORECOLOR Constant
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Sets or returns the foreground color of a command button.
 
    SCREEN.ControlPanel({COMBUTTON_FORECOLOR | subscript%})[ = color%]
 
    ■ subscript%   COMBUTTON_FORECOLOR has a value of 4
 
    ■ color%       Valid color settings are:
 
                   Value      Description
                   ══════     ══════════════════════════════════════════════
                   0 - 15     Valid color attribute
                   0          (Default) Black

SGN Function
  ◄Summary►             ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns a value indicating the sign of a numeric expression.
 
    SGN(numeric-expression)
 
    ■ numeric-expression     Any valid numeric expression:
 
                             Numeric Expression     Return Value
                             ══════════════════     ════════════════════════
                             Equal to zero          0
                             Positive               1
                             Negative               -1

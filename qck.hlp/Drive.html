 Sets or returns the selected drive at run time; not available at design
 time.
 
    [form.]drivelistbox.Drive[ = drivespec$]
 
    ■ drivespec$   Valid range for drives, including all drives present in
                   or connected to the system when the control is refreshed
                   at run time; default is the current drive

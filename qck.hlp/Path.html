Path Property
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►

 Sets or returns the current path, including the drive name; not available
 at design time.
 
    [form.]{dirlistbox. | filelistbox.}Path[ = pathstring$]
 
    ■ pathstring$       String value representing the complete path,
                        including the drive specification; default is the
                        current path when the control is loaded
 
 Applies To

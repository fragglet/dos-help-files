 Returns a byte read from a hardware I/O port (INP); sends a byte to a
 hardware I/O port (OUT).
 
    INP(port%)
    OUT port%, data%
 
    ■ port%   Integer expression in the range 0 to 65,535, inclusive, that
              identifies the port address
 
    ■ data%   Integer expression in the range 0 to 255, inclusive, to send
              to the port

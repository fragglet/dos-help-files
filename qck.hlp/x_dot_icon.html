 Icon
 
 ■ An icon is a graphical representation of an object or concept used to
   represent a minimized application.
 
 ■ In Visual Basic, all windows can be minimized to an icon 2 lines long and
   12 columns wide. The top line of the icon displays up to the first 10
   letters of the window title. For example, when you minimize the Help
   window, you'll see:

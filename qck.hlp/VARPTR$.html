VARPTR$ Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns a string representation of a variable's address for use in DRAW and
 PLAY statements.
 
    VARPTR$(variablename)
 
    ■ variablename    String variable containing DRAW or PLAY commands; if
                      an array element, the dimensions for the array must be
                      specified before VARPTR$ is used and the array must
                      consist of variable-length strings

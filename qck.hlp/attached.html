 Determines if a scroll bar is attached to the form; read-only at run time.
 
    [form.]scrollbar.Attached
 
    ■ Valid settings are:
 
      Value          Description
      ═════════      ═══════════════════════════════════════════════════════
      True (-1)      Scroll bar attached to bottom or right border of the
                     form (horizontal or vertical scroll bar)
      False (0)      (Default) Scroll bar not attached to the form

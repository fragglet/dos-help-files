STOP Statement
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Terminates the active program or suspends trapping of several ON event
 statements.
 
    STOP [n%]
 
    ■ n%    Sets the MS-DOS error level variable used in batch files
            or non-Visual Basic programs (untrapped errors return a value of
            -1); if omitted, STOP returns a value of 0 when successful

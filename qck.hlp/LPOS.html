LPOS Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the number of characters sent to a printer since the last carriage
 return was sent.
 
    LPOS(n%)
 
    ■ n%      Number specifying a printer port:
 
              Value   Printer Port
              ═════   ══════════════════════════════════════════════════════
              0       LPT1
              1       LPT1
              2       LPT2
              3       LPT3

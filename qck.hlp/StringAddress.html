  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►

 Used by a non-Basic routine to return the far address of a Visual Basic
 variable-length string.
 
    far-address = StringAddress(string-descriptor%);
 
    ■ string-descriptor%     Near address of a string descriptor within a
                             non-Basic routine

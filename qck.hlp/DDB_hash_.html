 Returns the depreciation of an asset for a specific period using the
 double-declining-balance method. Requires FINANCE.QLB (FINANCE.LIB when
 compiling); include FINANCE.BI.
 
    DDB# (cost#, salvage#, life#, period#, status%)
 
    ■ cost#       Initial cost of the asset
 
    ■ salvage#    Value at the end of the useful life of the asset
 
    ■ life#       Useful life of the asset
 
    ■ period#     Period for which asset depreciation is desired
 
    ■ status%     Variable that returns a value indicating calculation
                  failure or success:
                  0 = success
                  1 = failure

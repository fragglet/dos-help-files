 Returns an MS-DOS environment string (ENVIRON); changes or adds an
 environment string in the environment-string table (ENVIRON$).
 
    ENVIRON$ (env-variable$)
    ENVIRON$ (n%)
    ENVIRON stringexpression$
 
    ■ env-variable$        Name of an MS-DOS environment variable; must be
                           all uppercase
 
    ■ n%                   Causes ENVIRON$ to return the nth string from the
                           environment-string table
 
    ■ stringexpression$    Name and setting of an MS-DOS environment variable
                           (such as PATH or PROMPT) in one of the following
                           forms:
 
                                env-variable$ = env-string$
                                env-variable$ env-string$

 Window Manipulation Keys
 
   To:                                                   Press:
   ═══════════════════════════════════════════════════   ═══════════════════
   Arrange all windows                                   Alt+W,A
   Close MDI container or Exit                           Alt+F4
   Close active window                                   Ctrl+F4
   Display of the output screen/Output window (toggle)   F4
   Maximize window                                       Ctrl+F10
   Minimize window                                       Ctrl+F9
   Move window                                           Ctrl+F7
   Next window (when available)                          F6 or Ctrl+F6
   New window                                            Alt+W,N
   Open active window Control Menu                       Alt+Minus
   Open Window menu                                      Alt+W
   Open/Activate Calls window                            Alt+W,C
   Open/Activate code window                             Alt+W,#
   Open/Activate Debug window                            Alt+W,D
   Open/Activate Immediate window                        Alt+W,I
   Open/Activate Output window                           Alt+W,O
   Open/Activate Project window                          Alt+W,P
   Previous window (when available)                      Shift+F6 or
                                                         Ctrl+Shift+F6
   Restore window size                                   Ctrl+F5
   Size window                                           Ctrl+F8
   ─────────────────────────────────────────────────────────────────────────

ASC, CHR$ Functions
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns the ASCII code for the first character in a string expression (ASC),
 or returns the character corresponding to a specified ASCII code (CHR$).
 
    ASC(stringexpression$)
    CHR$(code%)
 
    ■ stringexpression$    Any string expression
 
    ■ code%                Valid ASCII code (0 - 255)

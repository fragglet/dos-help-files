INPUT$ Function
  ◄Summary►  ◄Details►  ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 Returns a string of characters read from a specified file. Use the INPUTBOX$
 function with forms. See: ◄INPUTBOX$ Function►
 
    INPUT$(n[,[#]filenumber%])
 
    ■ n                 Number of characters (bytes) to read
 
    ■ filenumber%       Number of an open file; if omitted, INPUT$ reads from
                        the keyboard

                 ISAM available in Professional Edition only
──────────────────────────────────────────────────────────────────────────────
 Makes the specified ISAM table index the current index. Requires PROISAM.EXE
 or PROISAMD.EXE in the programming environment (VBDOS.EXE).
 
    SETINDEX [#]filenumber% [,indexname$]
 
    ■ filenumber%    Number of open ISAM table (same number used
                     in OPEN statement)
 
    ■ indexname$     Name of existing ISAM index (same name used in
                     CREATEINDEX statement)

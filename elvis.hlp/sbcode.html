KEY STATUS
 
The key status codes remind you that you have one
or more of these options turned on:
 
Code  Description            Code  Description
CS    Column selection       CL    Caps lock
EX    Extend selection       OT    Overtype mode
NL    Number lock            MR    Macro record
SL    Direction keys scroll  RM    Revision marks
LD    Line draw mode         MX    Maximize window
ST    Macro step mode        LY    Layout view

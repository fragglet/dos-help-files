sbpgcol
PAGE/SECTION/LINE/COLUMN NUMBER
 
This region shows the position of the cursor in
the document. The page and column numbers are
always displayed, the section number only if you
have more than one section. You can display line
numbers in the status bar by turning on Show Line
Numbers in the View Preferences dialog box.
You can display the Go To dialog box by double-
clicking the status bar.

HELP REMINDER
Click the Help reminder in the status bar to get
the Help index or information about dialog boxes.
 
Help contains:
■ Information about all of Word's commands and
  many procedures.
■ Definitions, and tables describing keyboard and
  mouse functionality.
■ References to the demos, lessons, and manuals.
  Choose the Using Help topic on the Help menu
  for more information on how to use Help.

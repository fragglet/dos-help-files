Jump if CX is Zero                               Flags:  O D I T S Z A P C
                                                         ═════════════════
Syntax:  JCXZ label
 
  Transfers program execution to the specified label if CX is 0.  If the
  count register is not 0, execution continues at the next instruction.
  The label given as the operand must be short (between -128 and 127
  bytes from the instruction following the jump).

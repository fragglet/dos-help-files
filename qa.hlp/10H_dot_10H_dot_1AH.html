  Description:  Get Color Page State      [VGA]
 
  Returns the color register paging mode and current color page.
 
  Input             Output
  ══════            ══════════════════════════════════
  AH=10H            BH=color page
  AL=1AH            BL=paging mode
                       00H if 4 pages of 64 registers
                       01H if 16 pages of 16 registers

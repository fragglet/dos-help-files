  Description:  Set DTA Address
 
  Specifies the address of the disk transfer area (DTA) to be used for
  subsequent FCB-related function calls.
 
  Input                                   Output
  ════════════════════════════            ═══════
  AH=1AH                                  Nothing
  DS:DX=segment:offset of disk
        transfer area
                                    -♦-

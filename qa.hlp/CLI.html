Clear Interrupt Flag                             Flags:  O D I T S Z A P C
Syntax:  CLI                                             ═════════════════
                                                             0
 
  Clears the interrupt flag. When the interrupt flag is cleared, maskable
  interrupts are not recognized until the flag is set again with the STI
  instruction.

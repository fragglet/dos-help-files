  Description:  Get Color Register      [MCGA] [VGA]
 
  Returns the contents of a color register as its red, green, and blue
  components.
 
  Input                        Output
  ═════════════════            ══════════════
  AH=10H                       CH=green value
  AL=15H                       CL=blue value
  BX=color register            DH=red value

.FARDATA

  Syntax:  .FARDATA [name]
 
  Starts a far-data segment for initialized data (segment name FAR_DATA
  or name) ending previous segment, if any. The .MODEL directive must
  have previously been used. Data in segments defined with .FARDATA is
  not placed in a group or combined with .FARDATA segments for other
  source modules.
 
  name   A unique segment name. This optional parameter allows you to
         create multiple far-data segments within a source module.
 
  See Also:  .DATA, .MODEL

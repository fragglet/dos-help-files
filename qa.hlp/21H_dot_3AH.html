  Description:  Delete Directory
 
  Removes a directory using the specified drive and path.
 
  Input                                     Output
  ══════════════════════════════            ══════════════════════════════
  AH=3AH                                    If function successful
  DS:DX=segment:offset of ASCIIZ            Carry flag=clear
        pathname                            If function unsuccessful
                                            Carry flag= set
                                            AX=error code
                                    -♦-

▀ MS-DOS versions 1.x and 2.x do not reliably detect a disk change, and
  an error can occur if the user changes disks while a file is still open
  on that drive. In the worst case, the directory and file allocation
  table of the newly inserted disk may be damaged or destroyed.
 
▀ Int 21H Function 3EH should be used in preference to this function.

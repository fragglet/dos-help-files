Int 21H Function 2CH
   ◄Summary►  ◄Notes►  ◄Example►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
▀ This function's register format is the same as that required for Int
  21H Function 2DH (Set Time).
 
▀ On most IBM PC-compatible systems, the real-time clock does not have a
  resolution of single hundredths of seconds. On such machines, the
  values returned by this function in register DL are discontinuous.

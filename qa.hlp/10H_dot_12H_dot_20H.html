  Description:  Select Alternate PrintScreen      [EGA] [VGA]
 
  Selects an alternate print-screen routine for the EGA and VGA that works
  properly if the screen length is not 25 lines. The ROM BIOS default print-
  screen routine always prints 25 lines.
 
  Input             Output
  ══════            ═══════
  AH=12H            Nothing
  BL=20H

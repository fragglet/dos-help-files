IF
   ◄Directive Summary►  ◄Summary►  ◄IF Block Syntax►  ◄Example►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  IF expression
           IFE expression
 
  IF causes assembly of a block of statements (see IF Block Syntax) if
  expression is true (nonzero). IFE causes assembly of a block of
  statements if expression is false (zero).
 
  ELSEIF and ELSEIFE correspond to IF and IFE, respectively.
 
  See Also: IF1, IFB, IFDEF, IFDIF

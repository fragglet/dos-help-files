Wait                                             Flags:  O D I T S Z A P C
                                                         ═════════════════
Syntax:  WAIT
 
  Suspends CPU execution until a signal is received that a coprocessor has
  finished a simultaneous operation. It should be used to prevent a
  coprocessor instruction from modifying a memory location that is being
  modified at the same time by a processor instruction. WAIT is the
  same as the coprocessor FWAIT instruction.

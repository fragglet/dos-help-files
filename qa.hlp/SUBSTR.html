SUBSTR
   ◄Directive Summary►  ◄Summary►  ◄Example►
──────────────────────────────────────────────────────────────────────────────
  Syntax:  textlabel SUBSTR string, start [,length]
 
  Extracts specified characters from a string and assigns the resulting
  substring to textlabel.
 
  textlabel   A unique symbolic name.
 
  string      The string from which the substring is extracted.
 
  start       Starting position of substring, in which the first character
              is indexed as 1.
 
  length      Number of characters to extract.
 
   See Also:  CATSTR, SIZESTR, INSTR

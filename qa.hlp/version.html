@Version
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  @Version
 
  Prefined macro returning a three-digit string giving the name of the
  current version of the assembler. For QuickAssembler Version 2.01,
  @Version returns the string "520". For Version 5.1 of the Microsoft
  Macro Assembler, @Version returns the string "510".

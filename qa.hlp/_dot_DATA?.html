  Syntax:  .DATA?
 
  Starts the uninitialized data segment (_BSS) and ends previous segment,
  if any. The .MODEL directive must have previously been used. This
  segment should contain all global data that have no initial values.
 
  Although uninitialized data can be placed in the .DATA segment, use of
  .DATA? is recommended for maximum compatability with high-level
  languages.
 
  The .MODEL directive generates a GROUP statement that places _DATA
  in DGROUP.
 
  See Also:  .MODEL

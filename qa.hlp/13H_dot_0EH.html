  Description:  Read Sector Buffer      [PC]
 
  Transfers the contents of the fixed disk adapter's internal sector buffer
  to system memory. No data is read from the physical disk drive.
 
  Input                                     Output
  ══════════════════════════════            ══════════════════════════
  AH=0EH                                    If function successful
  ES:BX=segment:offset of buffer            Carry flag=clear
                                            If function unsuccessful
                                            Carry flag=set
                                            AH=status
                                            (see Int 13H Function 01H)
                                    -♦-

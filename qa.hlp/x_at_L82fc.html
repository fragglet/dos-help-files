Int 21H Function 5FH Subfunction 02H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
▀ The parameter returned in CX is a value that was previously passed to
  MS-DOS in register CX with Int 21H Function 5FH Subfunction 03H (Redirect
  Device). It represents data that is private to the applications which
  store and retrieve it, and has no meaning to MS-DOS.

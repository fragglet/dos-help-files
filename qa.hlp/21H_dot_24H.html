──────────────────────────────────────────────────────────────────────────────
 
  Description:  Set Relative-Record Number
 
  Sets the relative-record-number field of a file control block (FCB) to
  correspond to the current file position as recorded in the opened FCB.
 
  Input                               Output
  ════════════════════════            ════════════════════════════════════
  AH=24H                              AL is destroyed (other registers not
  DS:DX=segment:offset of              affected)
        previously opened             FCB relative-record field (offset
        file control block             21H) updated

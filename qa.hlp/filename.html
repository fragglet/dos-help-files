@FileName
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  @FileName
 
  Predefined equate returning the base name of the current source file.
  This directive is useful for any situation in which you want a name to
  change whenever the name of the file changes.

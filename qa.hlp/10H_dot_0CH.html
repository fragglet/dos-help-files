  Description:  Write Graphics Pixel      [CGA] [PCjr] [EGA]
                                                [MCGA] [VGA]
 
  Draws a point on the display at the specified graphics coordinates.
 
  Input                                        Output
  ═════════════════════════════════            ═══════
  AH=0CH                                       Nothing
  AL=pixel value
  BH=page
  CX=column (graphics x coordinate)
  DX=row (graphics y coordinate)

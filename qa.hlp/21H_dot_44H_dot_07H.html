──────────────────────────────────────────────────────────────────────────────
 
  Description:  IOCTL: Check Output Status
 
  Returns a code indicating whether the device associated with a handle is
  ready for output.
 
  Input                Output
  ═════════            ══════════════════════════
  AH=44H               If function successful
  AL=07H               Carry flag=clear
  BX=handle            and, for a device:
                       AL=00H if device not ready
                          FFH if device ready
                       or, for a file:
                       AL=FFH
                       If function unsuccessful
                       Carry flag= set
                       AX=error code
                                    -♦-

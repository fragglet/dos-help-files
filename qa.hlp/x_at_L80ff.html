Int 15H Function 0FH
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    This function call can be captured by a program so that it will be
    notified as each cylinder is formatted or analyzed. The program can
    count interrupts for each phase to determine the current cylinder number.
 
    The ROM BIOS default handler for this function returns with the carry
    flag set.

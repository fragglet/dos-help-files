  Description:  Select Disk
 
  Selects the specified drive to be the current, or default, disk drive
  and returns the total number of logical drives in the system.
 
  Input                                     Output
  ══════════════════════════════            ══════════════════════════════
  AH=0EH                                    AL=number of logical drives in
  DL=drive code (0=A, 1=B, etc.)               system

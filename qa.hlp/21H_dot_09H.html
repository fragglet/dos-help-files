Int 21H Function 09H
   ◄Summary►  ◄Notes►  ◄Example►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Display String
 
  Sends a string of characters to the standard output device. Output may
  be redirected (if output has been redirected, there is no way to detect
  disk full).
 
  Input                                     Output
  ══════════════════════════════            ═══════
  AH=09H                                    Nothing
  DS:DX=segment:offset of string

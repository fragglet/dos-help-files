  Description:  Set Cursor Position      [MDA] [CGA] [PCjr]
                                         [EGA] [MCGA] [VGA]
 
  Positions the cursor on the display, using text coordinates.
 
  Input                              Output
  ═══════════════════════            ═══════
  AH=02H                             Nothing
  BH=page
  DH=row (y coordinate)
  DL=column (x coordinate)

   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Disk Reset
 
  Flushes all file buffers. All data that has been logically written by
  user programs but has been temporarily buffered within MS-DOS is phys-
  ically written to the disk.
 
  Input             Output
  ══════            ═══════
  AH=0DH            Nothing

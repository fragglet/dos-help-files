  Syntax:  expression1 NE expression2
 
  Returns true (-1) if expression1 does not equal expression2, or
  returns false (0) if it does. Typically used with conditional
  directives.
 
  expression1   Must evaluate to a constant value.
 
  expression2   Must evaluate to a constant value.
 
   See Also:  EQ, LT, LE, GT, GE

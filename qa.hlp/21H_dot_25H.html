Int 21H Function 25H
   ◄Summary►  ◄Notes►  ◄Example►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Set Interrupt Vector
 
  Initializes a CPU interrupt vector to point to an interrupt handling
  routine.
 
  Input                                        Output
  ═════════════════════════════════            ═══════
  AH=25H                                       Nothing
  AL=interrupt number
  DS:DX=segment:offset of interrupt
        handling routine

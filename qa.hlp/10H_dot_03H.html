  Description:  Get Cursor Position      [MDA] [CGA] [PCjr]
                                         [EGA] [MCGA] [VGA]
 
  Obtains the current position of the cursor on the display, in text
  coordinates.
 
  Input              Output
  ═══════            ══════════════════════════════
  AH=03H             CH=starting line for cursor
  BH=page            CL=ending line for cursor
                     DH=row ( y coordinate)
                     DL=column (x coordinate)
                                    -♦-

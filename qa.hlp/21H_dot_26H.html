Int 21H Function 26H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Create New PSP
 
  Copies the program segment prefix (PSP) of the currently executing pro-
  gram to a specified segment address in free memory and then updates the
  new PSP to make it usable by another program.
 
  Input                                               Output
  ════════════════════════════════════════            ═══════
  AH=26H                                              Nothing
  DX=segment of new program segment prefix

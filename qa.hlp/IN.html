Input from Port                                  Flags:  O D I T S Z A P C
Syntax:  IN accumulator,port                             ═════════════════
 
 
  Transfers a byte or word from a port to the accumulator register.  The
  port address is specified by the source operand, which can be DX or an
  8-bit constant. Constants can only be used for port numbers less than
  255; use DX for higher port numbers.

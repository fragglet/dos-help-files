  Description:  Load ROM 8-by-16 Font and
                Reprogram Controller        [MCGA] [VGA]
 
  Loads the ROM BIOS default 8-by-16 font table into the specified block of
  character generator RAM.
 
  Input                                Output
  ═════════════════════════            ═══════
  AH=11H                               Nothing
  AL=04H or 14H (see Notes)
  BL=block

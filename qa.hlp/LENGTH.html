LENGTH
   ◄Operator Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  LENGTH variable
 
  Returns the number of data objects of the declared size in an array or
  other variable defined with the DUP operator. If the variable was not
  declared with the DUP operator, the value returned is 1.
 
  variable   If declared with nested DUP operators, only the value
             given for the outer DUP operator is returned.
 
   See Also:  DUP

Int 15H Function 85H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  SysReq Key      [AT] [PS/2]
 
  Invoked by the ROM BIOS keyboard driver when the SysReq key is detected.
 
  Input                                      Output
  ═══════════════════════════════            ══════════════════════════════
  AH=85H                                     If function successful
  AL=key status                              Carry flag=clear
     00H if key make (depression)            AH=00H
     01H if key break (release)              If function unsuccessful
                                             Carry flag=set
                                             AH=status

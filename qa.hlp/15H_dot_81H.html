  Description:  Device Close      [AT] [PS/2]
 
  Releases ownership of a logical device for a process.
 
  Input                    Output
  ═════════════            ════════════════════════
  AH=81H                   If function successful
  BX=device ID             Carry flag=clear
  CX=process ID            AH=00H
                           If function unsuccessful
                           Carry flag=set
                           AH=status
                                    -♦-

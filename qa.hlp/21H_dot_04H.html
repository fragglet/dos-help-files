  Description:  Auxiliary Output
 
  Outputs a character to the standard auxiliary device. The default is
  the first serial port (COM1).:
 
  Input                               Output
  ════════════════════════            ═══════
  AH=04H                              Nothing
  DL=8-bit data for output

  Description:  Create Directory
 
  Creates a directory using the specified drive and path.
 
  Input                                     Output
  ══════════════════════════════            ══════════════════════════════
  AH=39H                                    If function successful
  DS:DX=segment:offset of ASCIIZ            Cary flag=clear
        pathname                            If function unsuccessful
                                            Carry flag=set
                                            AX=error code

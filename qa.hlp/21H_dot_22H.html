  Description:  Random Write
 
  Writes data from memory into a selected record in a file.
 
  Input                           Output
  ════════════════════            ═══════════════════════════════════════
  AH=22H                          AL=00H  if write successful
  DS:DX=segment:offset               01H  if disk full
        of previously                02H  if segment wrap, write canceled
        opened file
        control block

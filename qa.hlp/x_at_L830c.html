Int 21H Function 68H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
▀ The effect of this function is equivalent to closing and reopening a
  file, or to duplicating a handle for the file with Int 21H Function 45H
  and then closing the duplicate. However, this function has the advantage
  that it will not fail due to lack of handles, and the application does
  not risk losing control of the file in multitasking or network
  environments.
 
▀ If this function is requested for a handle associated with a character
  device, a success flag is returned but there is no other effect.

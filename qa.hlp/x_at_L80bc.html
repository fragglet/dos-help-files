Int 13H Function 01H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    On fixed disks, error code 11H (ECC data error) indicates that a
    recoverable error was detected during a preceding Read Sector (Int 13H
    Function 02H) function.

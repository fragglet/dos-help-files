SUBTTL
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  SUBTTL text
 
  Specifies the subtitle used on each page of assembly listings. Specified
  text is printed flush left on the third line of the listing pages. This
  directive is useful only with printer-oriented listing files.
 
  Any number of subtitles can be given in a program. Each new SUBTTL
  replaces the current subtitle. Default subtitle is blank.
 
  text   Any combination of up to 60 characters.
 
   See Also:  PAGE, PAGE +, TITLE

  Description:  Turn Off Cassette Motor      [PC]
 
  Turns off the motor of the cassette tape drive.
 
  Input             Output
  ══════            ══════════════════════════════
  AH=01H            If function successful
                    Carry flag=clear
                    If function unsuccessful
                    Carry flag=set
                    AH=status
                       86H if cassette not present

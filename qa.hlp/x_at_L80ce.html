Int 13H Function 0AH
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    This function is supported on fixed disks only.
 
    The upper 2 bits of the 10-bit cylinder number are placed in
    the upper 2 bits of register CL.
 
    Unlike the normal Read Sector function (Int 13H Function 02H), ECC errors
    are not automatically corrected. Multisector transfers are terminated
    after any sector with a read error.

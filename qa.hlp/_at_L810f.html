    An error condition is returned if DX does not contain a valid subfunction
    number.
 
    If no game adapter is installed, AL is returned as 00H for Subfunction
    00H (i.e., all switches open); AX, BX, CX, and DX are returned containing
    00H for Subfunction 01H.
 
    Using a 250 KOhm joystick, the potentiometer values usually lie within
    the range 0-416 (0000-01A0H).

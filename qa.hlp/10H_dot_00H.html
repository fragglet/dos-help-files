  Description:  Set Video Mode      [MDA] [CGA] [PCjr]
                                    [EGA] [MCGA] [VGA]
 
  Selects the current video display mode. Also selects the active video
  controller, if more than one is present.
 
  Input                                Output
  ═════════════════════════            ═══════
  AH=00H                               Nothing
  AL=video mode (see Notes)

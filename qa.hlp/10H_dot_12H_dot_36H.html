  Description:  Enable/Disable Screen Refresh      [VGA]
 
  Enables or disables the video refresh for the currently active display.
 
  Input                               Output
  ════════════════════════            ═════════════════════
  AH=12H                              If function supported
  AL=00H to enable refresh            AL=12H
     01H to disable refresh
  BL=36H

  Description:  Character Input with Echo
 
  Reads a character from the standard input device and echoes it to the
  standard output device. If no character is ready, waits until one is
  available. Input can be redirected (if input has been redirected,
  there is no way to detect EOF).
 
  Input             Output
  ══════            ═══════════════════
  AH=01H            AL=8-bit input data

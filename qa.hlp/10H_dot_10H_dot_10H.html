  Description:  Set Color Register      [MCGA] [VGA]
 
  Programs an individual color register with a red-green-blue (RGB)
  combination.
 
  Input                        Output
  ═════════════════            ═══════
  AH=10H                       Nothing
  AL=10H
  BX=color register
  CH=green value
  CL=blue value
  DH=red value

Int 21H Function 10H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Close File
 
  Closes a file, flushes all MS-DOS internal disk buffers associated
  with the file to disk, and updates the disk directory if the file has
  been modified or extended.
 
  Input                                   Output
  ════════════════════════════            ═════════════════════════════════
  AH=10H                                  If function successful (directory
  DS:DX=segment:offset of file             update successful)
        control block                     AL=00H
                                          If function unsuccessful (file
                                           not found in directory)
                                          AL=FFH

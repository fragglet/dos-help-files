Interrupt on Overflow                            Flags:  O D I T S Z A P C
Syntax:  INTO                                            ═════════════════
                                                             ± ±
 
 
  Generates interrupt 4 if the overflow flag is set. The default DOS
  behavior for interrupt 4 is to return without taking any action. You
  must define an interrupt procedure for interrupt 4 in order for INTO
  to have any effect.

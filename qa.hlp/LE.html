LE
   ◄Operator Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  expression1 LE expression2
 
  Returns true (-1) if expression1 is less than or equal to expression2,
  or false (0) if it is not. Typically used with conditional directives.
 
  expression1   Must evaluate to a constant value.
 
  expression2   Must evaluate to a constant value.
 
   See Also:  EQ, NE, LT, GT, GE

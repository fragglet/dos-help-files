  Description:  Set Palette Register      [PCjr] [EGA]
                                          [MCGA] [VGA]
  Sets the correspondence of a palette register to a displayable color.
 
  Input                                     Output
  ══════════════════════════════            ═══════
  For PCjr, EGA, or VGA                     Nothing
  AH=10H
  AL=00H
  BH=color value
  BL=palette register (00-0FH)
  For MCGA
  AH=10H
  AL=00H
  BX=0712H

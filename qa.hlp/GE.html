GE
   ◄Operator Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  expression1 GE expression2
 
  Returns true (-1) if expression1 is greater than or equal to
  expression2, or returns false (0) if it is not. Typically used with
  conditional directives.
 
  expression1   Must evaluate to a constant value.
 
  expression2   Must evaluate to a constant value.
 
   See Also:  EQ, NE, LT, LE, GT

  Description:  Rename File
 
  Alters the name of all matching files in the current directory on the
  disk in the specified drive.
 
  Input                         Output
  ═══════════════════════       ═════════════════════════════════════
  AH=17H                        If function successful (one or more
  DS:DX=segment:offset of        files renamed)
        "special" file          AL=00H
        control block           If function unsuccessful (no matching
                                 files, or new filename matched an
                                 existing file)
                                AL=FFH
                                    -♦-

Int 1AH Function 04H
   ◄Summary►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Get Date      [AT] [PS/2]
 
  Reads the current date from the CMOS time/date chip.
 
  Input             Output
  ══════            ═══════════════════════════════════════════════════
  AH=04H            CH=century (19 or 20) in binary coded decimal (BCD)
                    CL=year in BCD
                    DH=month in BCD
                    DL=day in BCD
                    and, if clock running
                    Carry flag=clear
                    or, if clock stopped
                    Carry flag=set
                                    -♦-

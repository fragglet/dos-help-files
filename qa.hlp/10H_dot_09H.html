Int 10H Function 09H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Write Character and      [MDA] [CGA] [PCjr]
                Attribute at Cursor      [EGA] [MCGA] [VGA]
 
  Writes an ASCII character and its attribute to the display at the
  current cursor position.
 
  Input                                                           Output
  ════════════════════════════════════════════════════            ═══════
  AH=09H                                                          Nothing
  AL=character
  BH=page
  BL=attribute (text modes) or color (graphics modes)
  CX=count of characters to write (replication factor)

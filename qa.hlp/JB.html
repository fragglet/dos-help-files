Jump Conditionally                               Flags:  O D I T S Z A P C
                                                         ═════════════════
 
 
  Transfers execution to the specified label if the flags condition is
  true. The condition is tested by checking the flags shown in the table
  on the following page. If the condition is false, then no jump is taken
  and program execution continues at the next instruction. On the 8088-80286
  processors, the label given as the operand must be short (between -128 and
  127 bytes from the instruction following the jump).

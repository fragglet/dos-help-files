▀ When supported by the driver, this subfunction can be used to obtain
  hardware-dependent status and availability information that is not pro-
  vided by other MS-DOS function calls.
 
▀ Block device drivers are not required to support IOCTL Subfunction 04H.
  If this subfunction is requested and the driver does not have the abil-
  ity to process control data, control returns to the program with the
  carry flag set and error code 0001H (invalid function) in register AX.

   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Set Pointing Device Handler Address      [PS/2]
 
  Notifies the ROM BIOS pointing device driver of the address for a routine
  to be called each time pointing device data is available.
 
  Input                                           Output
  ════════════════════════════════════            ═══════════════════════════
  AH=C2H                                          If function successful
  AL=07H                                          Carry flag=clear
  ES:BX=segment:offset of user routine            If function unsuccessful
                                                  Carry flag=set
                                                  AH=status (see Int 15H
                                                             Function C2H
                                                             Subfunction 00H)

  Description:  Terminate Process with Return Code
 
  Terminates the current process, passing a return code to the parent pro-
  cess. This is one of several methods that a program can use to perform a
  final exit.
 
  If you use the .STARTUP directive, you should also use the .EXIT directive,
  which automatically generates code to call this function.
 
  Input                      Output
  ═══════════════            ═══════
  AH=4CH                     Nothing
  AL=return code

  Description:  Write Character in Teletype Mode      [MDA] [CGA] [PCjr]
                                                      [EGA] [MCGA] [VGA]
 
  Writes an ASCII character to the display at the current cursor position,
  using the specified color (if in graphics modes), and then increments
  the cursor position appropriately.
 
  Input                                           Output
  ════════════════════════════════════            ═══════
  AH=0EH                                          Nothing
  AL=character
  BH=page
  BL=foreground color (graphics modes)

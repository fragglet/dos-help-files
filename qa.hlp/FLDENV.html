Load Environment State
 
Syntax:  FLDENV mem
 
Loads the 14-byte coprocessor environment state from a specified memory
location. The environment includes the control word, status word, tag
word, instruction pointer, and operand pointer. On the 80387 in 32-bit
mode, the environment state is made up of 28 bytes.

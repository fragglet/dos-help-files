+ (Plus, Binary)
   ◄Operator Summary►  ◄Summary►  ◄Unary Plus►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  expression1 + expression2
 
  Plus (binary). Returns value of expression1 plus expression2.
 
  expression   An integer constant or memory operand. The result can be
               used as a memory operand.
 
   See Also:  minus (-)

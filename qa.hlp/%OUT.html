%OUT Directive
   ◄Directive Summary►  ◄Summary►  ◄Example►  ◄OUT Instruction►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  %OUT text
 
  Displays text to the standard output device. Though this device is
  normally the screen, at the DOS command level you can redirect it to a
  file or another device. This directive can help show which statements
  were assembled and can be useful for debugging.
 
  text   Any line of ASCII characters. For multiple lines, use a separate
         %OUT directive for each.
 
   See Also:  .ERR

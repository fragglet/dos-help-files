Int 17H Function 01H
   ◄Summary►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Initialize Printer Port      [PC] [AT] [PS/2]
 
  Initializes the specified parallel printer interface port and returns
  its status.
 
  Input                                  Output
  ═══════════════════════════            ═════════════════════════════
  AH=01H                                 AH=status
  DX=printer number                         (see Int 17H Function 00H)
     (0=LPT1, 1=LPT2, 2=LPT3)
                                    -♦-

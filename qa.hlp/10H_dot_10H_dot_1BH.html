  Description:  Set Gray-Scale Values      [MCGA] [VGA]
 
  Transforms the red, green, and blue values of one or more color regis-
  ters into the gray-scale equivalents.
 
  Input                                   Output
  ════════════════════════════            ═══════
  AH=10H                                  Nothing
  AL=1BH
  BX=first color register
  CX=number of color registers
                                    -♦-

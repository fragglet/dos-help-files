  Description:  Turn On Cassette Motor      [PC]
 
  Turns on the motor of the cassette tape drive.
 
  Input             Output
  ══════            ══════════════════════════════
  AH=00H            If function successful
                    Carry flag=clear
                    If function unsuccessful
                    Carry flag=set
                    AH=status
                       86H if cassette not present

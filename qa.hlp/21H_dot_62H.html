Int 21H Function 62H
   ◄Summary►  ◄Notes►  ◄Example►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Get PSP Address     DOS 3 Only
 
  Obtains the segment (paragraph) address of the program segment prefix
  (PSP) for the currently executing program.
 
  Input             Output
  ══════            ════════════════════════════════════════════
  AH=62H            BX=segment address of program segment prefix
                                    -♦-

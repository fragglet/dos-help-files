Int 10H Function 13H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    This function is not available on the original IBM PC or PC/XT unless an
    EGA video adapter (which contains its own ROM BIOS) is installed.
 
    This function can be thought of as an extension to Int 10H Function 0EH.
    The control characters bell (07H), backspace (08H), line feed (0AH), and
    carriage return (0DH) are recognized and handled appropriately.

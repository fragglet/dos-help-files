  Description:  Set Tick Count      [AT] [PS/2]
 
  Stores a 32-bit value in the clock tick counter.
 
  Input                                            Output
  ═════════════════════════════════════            ═══════
  AH=01H                                           Nothing
  CX:DX=tick count (high 16-bits in CX)
                                    -♦-

Interrupt 27H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Terminate and Stay Resident
 
  Terminates execution of the currently executing program, but reserves
  part or all of its memory so that it will not be overlaid by the next
  transient program to be loaded.
 
  Input                                            Output
  ═════════════════════════════════════            ═══════
  DX=offset of the last byte plus one              Nothing
     (relative to the program segment
     prefix) of program to be protected
  CS=segment of program segment prefix

   ◄Summary►  ◄Example►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Read Character from Keyboard      [PC] [AT] [PS/2]
 
  Reads a character from the keyboard, also returning the keyboard scan code.
 
  Input             Output
  ══════            ═════════════════════
  AH=00H            AH=keyboard scan code
                    AL=ASCII character

  Description:  Read Character from Enhanced Keyboard      [AT] [PS/2]
 
  Reads a character and scan code from the keyboard type-ahead buffer.
 
  Input             Output
  ══════            ═════════════════════
  AH=10H            AH=keyboard scan code
                    AL=ASCII character

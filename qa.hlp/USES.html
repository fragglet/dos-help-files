  Syntax:   USES reglist
 
  An optional keyword used with PROC. Generates code to push the value
  of registers that should be preserved (and will be altered by your
  procedure) on the stack and pop them when the procedure returns. The
  .MODEL directive must have previously been used with a langtype
  argument.
 
  reglist   List of one or more registers. Multiple registers are separated
            by spaces.
 
   See Also:  PROC, PTR

SHORT
   ◄Operator Summary►  ◄Summary►  ◄Example►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  SHORT label
 
  Sets label type to SHORT. Short labels can be used in JMP instructions
  when the distance is less than 128 bytes from the start of the next
  instruction. Instructions using short labels are a byte smaller than
  identical instructions using the default near labels.
 
   See Also:  JMP

  Description:  Set Time      [AT] [PS/2]
 
  Sets the time in the CMOS time/date chip.
 
  Input                                             Output
  ══════════════════════════════════════            ═══════
  AHM=03H                                           Nothing
  CH=hours in binary coded decimal (BCD)
  CL=minutes in BCD
  DH=seconds in BCD
  DL=daylight-saving-time code
     00H if standard time
     01H if daylight saving time

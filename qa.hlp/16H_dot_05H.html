   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Push Character and Scan Code      [AT] [PS/2]
 
  Places a character and scan code in the keyboard type-ahead buffer.
 
  Input                   Output
  ════════════            ═════════════════════════════
  AH=05H                  If function successful
  CH=scan code            Carry flag=clear
  CL=character            AL=00H
                          If function unsuccessful
                            (type-ahead buffer is full)
                          Carry flag=set
                          AL=01H
                                    -♦-

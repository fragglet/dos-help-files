  Description:  Get Printer Status      [PC] [AT] [PS/2]
 
  Returns the current status of the specified parallel printer
  interface port.
 
  Input                                  Output
  ═══════════════════════════            ═════════════════════════════
  AH=02H                                 AH=status
  DX=printer number                         (see Int 17H Function 00H)
     (0=LPT1, 1=LPT2, 2=LPT3)

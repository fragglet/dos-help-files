Int 13H Function 19H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Park Heads      [PS/2]
 
  Moves the read/write arm to a track that is not used for data storage, so
  that data will not be damaged when the drive is turned off.
 
  Input                            Output
  ═════════════════════            ════════════════════════════════════
  AH=19H                           If function successful
  DL=drive                         Carry flag =clear
     80H-FFH fixed disk            If function unsuccessful
                                   Carry flag=set
                                   AH=status (see Int 13H Function 01H)
                                    -♦-

.SALL
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  .SALL
 
  Suppresses listing of macro expansions in a listing file. Macro calls
  are listed, but not the source code generated by the call.
 
   See Also:  .LALL, .XALL

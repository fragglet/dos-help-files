  Description:  Set Sound Source      [PC]
 
  Sets up the source for tones that will appear on the PCjr's "Audio Out"
  or RF modulator.
 
  Input                                                   Output
  ════════════════════════════════════════════            ═══════
  AH=80H                                                  Nothing
  AL=sound source
     00H if 8253 programmable timer, channel 2
     01H if cassette input
     02H if "Audio In" line on I/O channel
     03H if sound generator chip

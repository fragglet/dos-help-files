Int 21H Function 46H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Redirect Handle
 
  Given two handles, makes the second handle refer to the same device or
  file at the same location as the first handle. The second handle is then
  said to be redirected.
 
  Input                                   Output
  ════════════════════════════            ══════════════════════════════
  AH=46H                                  If function successful
  BX=handle for file or device            Carry flag=clear
  CX=handle to be redirected              If function unsuccessful
                                          Carry flag=set
                                          AX=error code
                                    -♦-

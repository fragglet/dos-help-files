  Description:  Get Enhanced Keyboard Status      [AT] [PS/2]
 
  Determines whether a character is ready for input, returning a flag and
  also the character itself, if one is waiting.
 
  Input             Output
  ══════            ══════════════════════════
  AH=11H            If key waiting to be input
                    Zero flag=clear
                    AH=keyboard scan code
                    AL=character
                    If no key waiting
                    Zero flag=set
                                    -♦-

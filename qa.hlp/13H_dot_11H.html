   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Recalibrate Drive      [PC] [AT] [PS/2]
 
  Causes the fixed disk adapter to recalibrate itself for the specified
  drive, positioning the read/arm to cylinder 0, and returns the drive's
  status.
 
  Input                            Output
  ═════════════════════            ════════════════════════════════════
  AH=11H                           If function successful
  DL=drive                         Carry flag=clear
     80H-FFH fixed disk            AH=00H
                                   If function unsuccessful
                                   Carry flag=set
                                   AH=status (see Int 13H Function 01H)
                                    -♦-

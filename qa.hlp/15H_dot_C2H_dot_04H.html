Int 15H Function C2H Subfunction 04H
   ◄Summary►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Get Pointing Device Type      [PS/2]
 
  Returns the identification code for the system's mouse or other
  pointing device.
 
  Input             Output
  ══════            ════════════════════════════════════════════════════
  AH=C2H            If function successful
  AL=04H            Carry flag=clear
                    AH=00H
                    BH=device ID
                    If function unsuccessful
                    Carry flag=set
                    AH=status (see Int 15H Function C2H Subfunction 00H)

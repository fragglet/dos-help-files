  Description:  Get Block of Color Registers      [MCGA] [VGA]
 
  Allows the red, green, and blue components associated with each of a
  set of color registers to be read in one operation.
 
  Input                                     Output
  ══════════════════════════════            ══════════════════════════════
  AH=10H                                    ES:DX=segment:offset of buffer
  AL=17H                                    and buffer contains color list
  BX=first color register
  CX=number of color registers
  ES:DX=segment:offset of buffer
   to receive color list

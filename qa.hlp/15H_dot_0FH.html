Int 15H Function 0FH
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Format ESDI Drive Periodic Interrupt      [PS/2]
 
  Invoked by the ROM BIOS on the ESDI Fixed Disk Drive Adapter/A during a
  format or surface analysis operation after each cylinder is completed.
 
  Input                            Output
  ═════════════════════            ═════════════════════════════════════════
  AH=0FH                           If formatting or analysis should continue
  AL=phase code                    Carry flag=clear
     0=reserved                    If formatting or analysis should
     1=surface analysis             be terminated
     2=formatting                  Carry flag=set

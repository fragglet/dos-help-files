  Description:  Enable/Disable Gray-Scale Summing      [MCGA] [VGA]
 
  Enables or disables gray-scale summing for the currently active display.
 
  Input                                            Output
  ════════════════════════════════════             ═════════════════════
  AH=12H                                           If function supported
  AL=00H to enable gray-scale summing              AL=12H
     01H to disable gray-scale summing
  BL=33H

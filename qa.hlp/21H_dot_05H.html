Int 21H Function 05H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Printer Output
 
  Sends a character to the standard list device. The default is the
  printer on the first parallel port (LPT1), unless explicitly redirected
  by the user with the MODE command.
 
  Input                                Output
  ═════════════════════════            ═══════
  AH=05H                               Nothing
  DL=8-bit data for output

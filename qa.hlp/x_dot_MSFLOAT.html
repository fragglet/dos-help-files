.MSFLOAT
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:   .MSFLOAT
 
  Specifies Microsoft Binary Format for storage of real numbers. The
  default behavior is to store real numbers in IEEE format. This directive
  is included for compatibility with some versions of Microsoft BASIC and
  the Microsoft Macro Assembler.

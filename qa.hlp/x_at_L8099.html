Int 10H Function 11H Subfunction 21H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    This function provides font selection in graphics (all-points-
    addressable) display modes. For font selection in text (alphanumeric)
    modes, see Int 10H Function 11H Subfunctions 00H-14H.
 
    If this subfunction is called at any time other than immediately after a
    mode set, the results are unpredictable.

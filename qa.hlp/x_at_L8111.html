Int 15H Function 85H
   ◄Summary►  ◄Notes►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
    The ROM BIOS handler for this function call is a dummy routine that
    always returns a success status unless called with an invalid subfunction
    number in AL. A multitasking program manager would be expected to capture
    Int 15H so that it can be notified when the user strikes the SysReq key.

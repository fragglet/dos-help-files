  Description:  Character Output
 
  Outputs a character to the standard output device. Output can be
  redirected (if output is redirected, there is no way to detect disk
  full).
 
  Input                               Output
  ════════════════════════            ═══════
  AH=02H                              Nothing
  DL=8-bit data for output

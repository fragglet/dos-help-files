Logical AND                                      Flags:  O D I T S Z A P C
                                                         ═════════════════
Syntax:  AND dest,src                                    0       ± ± ? ± 0
 
  Performs a bitwise logical AND on the source and destination operands
  and stores the result in the destination operand. For each bit position
  in the operands, if both bits are set, then the corresponding bit of the
  result is set. Otherwise, the corresponding bit of the result is cleared.

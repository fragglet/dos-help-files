  Description:  Process Termination      [AT] [PS/2]
 
  Releases ownership of all logical devices for a process that is
  about to terminate.
 
  Input                    Output
  ═════════════            ════════════════════════
  AH=82H                   If function successful
  BX=process ID            Carry flag=clear
                           AH=00H
                           If function unsuccessful
                           Carry flag=set
                           AH=status
                                    -♦-

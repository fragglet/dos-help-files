Int 14H Function 03H
   ◄Summary►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Get Communications Port Status      [PC] [AT] [PS/2]
 
  Returns the status of the specified serial communications port.
 
  Input                             Output
  ═════════════════════════════     ══════════════════════════════════════════
  AH=03H                            AH=port status (see Int 14H Function 00H)
  DX=communications port number     AL=modem status (see Int 14H Function 00H)
     (0=COM1, 1=COM2,   etc.)

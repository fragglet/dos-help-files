.ALPHA
   ◄Directive Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax:  .ALPHA
 
  Outputs segments to the object file by alphabetical order (using the
  names of the segments). The default order is sequential, which outputs
  segments in the order they appear in source code.
 
  See Also:  .SEQ and DOSSEG

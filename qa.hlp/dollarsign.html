$ (Current Line Indicator)
   ◄Operator Summary►  ◄Summary►
──────────────────────────────────────────────────────────────────────────────
 
  Syntax: $
 
  Equates to the current value of the location counter (current offset
  address). This operator is frequently used to determine the length of
  strings, as in the following example:
 
  strn   DB  'This is a string'
  strnl  EQU $-strn                 ; strnl is length of strn--current

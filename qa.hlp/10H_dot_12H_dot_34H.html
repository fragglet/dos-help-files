   ◄Summary►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
 
  Description:  Enable/Disable Cursor Emulation      [VGA]
 
  Enables or disables cursor emulation for the currently active display.
  When cursor emulation is enabled, the ROM BIOS automatically remaps
  Int 10H Function 01H cursor starting and ending lines for the current
  character cell dimensions.
 
  Input                                         Output
  ══════════════════════════════════            ═════════════════════
  AH=12H                                        If function supported
  AL=00H to enable cursor emulation             AL=12H
     01H to disable cursor emulation
  BL=34H

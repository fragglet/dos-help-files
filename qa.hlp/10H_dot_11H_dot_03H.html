  Description:  Set Block Specifier      [EGA] [MCGA] [VGA]
 
  Determines the character blocks selected by bit 3 of character attribute
  bytes in alphanumeric (text) display modes.
 
  Input                                   Output
  ════════════════════════════            ═══════
  AH=11H                                  Nothing
  AL=03H
  BL=character generator block
   select code (see Notes)

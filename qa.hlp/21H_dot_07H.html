  Description:  Unfiltered Character Input Without Echo
 
  Reads a character from the standard input device without echoing it to
  the standard output device. If no character is ready, waits until one
  is available. Input may be redirected (if input has been redirected,
  there is no way to detect EOF).
 
  Input             Output
  ══════            ═══════════════════
  AH=07H            AL=8-bit input data

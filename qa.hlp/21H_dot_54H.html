  Description:  Get Verify Flag
 
  Obtains the current value of the system verify (read-after-write) flag.
 
  Input             Output
  ══════            ════════════════════════════
  AH=54H            AL=current verify flag value
                    00H  if verify off
                    01H  if verify on

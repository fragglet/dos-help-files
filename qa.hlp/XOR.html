Exclusive OR                                     Flags:  O D I T S Z A P C
                                                         ═════════════════
Syntax:  XOR dest,src                                    0       ± ± ? ± 0
 
  Performs a bitwise exclusive OR on the source and destination operands
  and stores the result to the destination. For each bit position in the
  operands, if both bits are set or if both bits are cleared, the
  corresponding bit of the result is cleared. Otherwise, the
  corresponding bit of the result is set.

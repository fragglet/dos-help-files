Open Program Dialog
Use to clear memory and load an existing program.
┌───────── Open Program ──────────┐ To load a program from
│                                 │ the current directory:
│ File Name: ▒▒▒▒▒▒▒▒ ◄─────────────────┐
│                                 │     │
│ C:\MYPROGS ◄── current directory│   Type a filename
│                                 │   here, or
│      Files          Dirs/Drives │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │   Select a
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │   filename here.
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒    ▒▒▒▒▒▒▒▒▒▒▒ │     │
│        ▲                        │     │
│        └──────────────────────────────┘
└─────────────────────────────────┘   Then choose <OK>.
 
The Dirs/Drives box provides access to other directories
and drives.
 
Note that the current directory for the Open Program
command can vary from the current DOS directory used
by other QBX environment commands.
 
To open a program in another directory:
┌───────── Open Program ──────────┐
│                                 │  Type a directory
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────── and filename here,
│                                 │  or
│ C:\XXXX ◄── current directory   │
│                                 │
│      Files         Dirs/Drives  │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒ ◄── Select a directory
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │  name here, then
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │  select a filename
│        ▲                        │  here.
│        └─────────────────────────────┘
└─────────────────────────────────┘  Then choose <OK>.
 
To look at the contents of other directories:
┌───────── Open Program ──────────┐
│                                 │  Type a wildcard
│ File Name: ▒▒▒▒▒▒▒▒ ◄───────────── filename (such as
│                                 │  *.BAS or *.BI) here.
│ C:\XXXX ◄── current directory   │
│                                 │
│      Files         Dirs/Drives  │     Select a directory
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒ ◄───── name here.
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒  │     Look at the
│        ▲                        │     filenames here.
│        └───────────────────────────────┘
└─────────────────────────────────┘ Then choose <Cancel>.

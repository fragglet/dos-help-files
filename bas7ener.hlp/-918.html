Customize Menu Dialog
Adds, deletes, or edits a custom Utility menu command.
┌────────── Customize Menu ──────────┐
│  Applications List:                │
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  <Add>    ◄─── Adds a custom menu
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒              │ entry to the list.
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  <Edit>   ◄─── Edits an existing
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒              │ menu entry.
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  <Delete> ◄─── Deletes a menu
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒              │ entry.
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  <To Begin> ◄─ Moves current entry
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒              │ to beginning of list.
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  <To End> ◄─── Moves current entry
│                                    │ to end of list.
└────────────────────────────────────┘

Syntax error
Several conditions can cause this error:
  ■ At compile time, the most common cause is an
    incorrectly typed BASIC keyword or argument.
  ■ At run time, this error can be caused by an
    improperly formatted DATA statement.
  ■ In an ISAM program, this error can occur if the
    number of keyvalues in a SEEKoperand operation is
    greater than the number of values that makes up
    the current index.
 
ERR code: 2

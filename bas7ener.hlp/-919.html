Customize Menu
Use to add or edit a custom menu entry. If you are editing
an existing command, the current settings for the command
are displayed.

To assign an accelerator key to a command:
    ■ Select the Alt+F box from Accelerator Key: field. A
      black dot will appear between the parentheses.
    ■ Type a value between 2 and 10 after Alt+F (this
      corresponds to the F2 through F10 function keys).
Note: When using an Alt+key combination defined on this
      menu, you must keep the Alt key depressed to avoid
      unexpected behavior in the program.

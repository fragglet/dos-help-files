Instant Watch Dialog
Use to look at the current value of a variable or the
condition of an expression, and to add the variable or
expression to the Watch window.
┌──────────────────────────────┐
│   ┌────── Expression ─────┐  │  The text you selected
│   │                       │◄─── in the active window
│   └───────────────────────┘  │  appears here.
│   ┌──────── Value ────────┐  │
│   │                       │◄─── Current value or
│   └───────────────────────┘  │  condition appears here.
│                              │
│<Add Watch>  ◄────────────────── Adds the expression
└──────────────────────────────┘  to the Watch window.

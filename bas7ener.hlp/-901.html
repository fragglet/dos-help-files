Create File Dialog
Before you can create a new file, the QBX environment needs
to know its filename and type.
 
┌────────── Create File ──────────┐  Type the filename
│ Name: ▒▒▒▒▒▒▒▒▒▒  ◄─────────────── here.
│                                 │
│ ▒ Module  ▒ Include  ▒ Document◄── Select a file type
└─────────────────────────────────┘  here.
 
Selecting the Document file type turns off the Smart Editor
functions.

The QBX environment searches for specific files in the
directories you indicate in this box. You don't need to set
search paths if you have done so with DOS environment
variables or if all files needed by your program are in the
current directory.
┌────────────── Set Paths ─────────────┐ For example:
│ Search Path For:                     │
│ Executable Files: ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒◄── c:\qbx\bin
│                                      │
│ Include Files:    ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒◄── c:\qbx\qbbi
│                                      │
│ Library Files:    ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒◄── c:\qbx\qblib
│                                      │
│ Help Files:       ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒◄── c:\qbx\qbhlp
└──────────────────────────────────────┘
Tip: When you enter a path, the QBX environment does not
     check what you typed for syntax or meaning. The QBX
     environment uses the same syntax as DOS environment
     variables.

Modified INCLUDE file
One or more include files are loaded into memory that
have not been saved to disk since they were modified.
QBX needs to know which version of the include files
you want to run. Either:
  ■ Choose <Yes> to save the modified include files and
    run.
  ■ Choose <No> to run the program with the versions that
    are currently on disk (the unmodified versions).
  ■ Choose <Cancel> if you don't want to run the program.

File not found
If you encounter this error when running your program,
a FILES, KILL, NAME, OPEN, PRINT, or RUN statement refers
to a file that does not exist.
 
If you encounter this error when starting QBX or when
opening a file in QBX, you have a PWB .MAK file with
the same base filename as your source file. Rename
the PWB .MAK file.

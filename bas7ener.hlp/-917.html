Run DOS Command Dialog
Run a DOS command without leaving the QBX environment.
┌─────────── Run DOS Command ───────────┐
│                                       │  Name of DOS
│ DOS Command: ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ ◄──── command to run.
│                                       │
│ [X] Prompt before returning   ◄───────── If selected, QBX
│                                       │  prompts you
└───────────────────────────────────────┘  before returning.
To run a DOS command without leaving the QBX environment:
    ■ Type the program name and arguments in the text box.
      The program must be in one of the directories
      specified by the current PATH environment variable,
      unless the full path of the command is given.
    ■ Select the Prompt before Returning option to see
      the program output before returning to QBX.

ISAM - Invalid column
When you open a file for ISAM and specify a data type,
all element names must be column names.
 
When you create an ISAM index, you cannot name a column
that:
   ■ Does not exist.
   ■ Has a data type that cannot be indexed (an array or
     user-defined type).
 
ERR code: 84

ISAM - At index boundary
Either of these conditions can generate this error:
   ■ A unique index is current for an ISAM table, and the
     user has attempted to enter a value for a column that
     is already in the column.
   ■ A CREATEINDEX statement has been executed with a
     non-zero unique argument value, but the existing
     column(s) already contained values that aren't unique.
 
ERR code: 86

Feature unavailable
You may be attempting to do one of the following:
  ■ Use a feature of another version of BASIC that
    is not available with Microsoft BASIC.
  ■ Perform an ISAM operation without the correct
    ISAM TSR loaded (PROISAM or PROISAMD).
  ■ Use a feature of Microsoft BASIC that is unavailable
    under the operating system you are running. Such as:
      ■ Using the LOCK or UNLOCK statement while running
        DOS 2.1.
      ■ Using the SHELL function while running DOS
        instead of OS/2.
      ■ Using the PALETTE or PCOPY statement while
        running OS/2.

     Fatal error C1068
 
     cannot open file 'filename'
 
     The compiler cannot open the given file.
 
     There are several possible causes for this error:
 
        ■ The file does not exist.
 
        ■ The file is marked read-only.
 
        ■ Not enough file handles exist.

     Compiler warning (level 1) C4020
 
     'function' : too many actual parameters
 
     The number of arguments specified in a function call was greater
     than the number of parameters specified in the function prototype
     or function definition.
 
     The extra parameters were passed according to the calling
     convention used on the function.

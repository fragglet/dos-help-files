     Fatal LINK error L1123
 
     'segment' : segment defined for both 16- and 32-bit
 
     This error can occur when a segment is defined as 32-bit an an
     assembly-language module and as a 16-bit segment in a C module
     (16 bits is the default in QuickC).
 
     Define the segment as either 16-bit or 32-bit.

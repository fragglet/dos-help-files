     LINK warning L4013
 
     'option' : option ignored for segmented-executable file
 
     The use of overlays and options /CPARMAXALLOC, /DSALLOCATE, /HIGH,
     /NOGROUPASSOCIATION, and /TINY is not allowed with either OS/2
     protected-mode or Windows executable files.

     Fatal error C1059
 
     compiler is out of near heap space
 
     The compiler ran out of storage for items that it stores in the
     near (default data segment) heap.
 
     One of the following may be a solution:
 
        ■ Eliminate unnecessary include files, especially unneeded
          function prototypes.
 
        ■ Split the function existing at the given line number into two
          or more functions.
 
        ■ Split the current file into two or more files and compile
          them separately.

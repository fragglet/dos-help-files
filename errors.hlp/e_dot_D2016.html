     Command-line error D2016
 
     'option1' and 'option2' are incompatible
 
     Two incompatible command-line options were specified.
 
     The following is an example of a command line that causes this
     error:
 
          qcl /Gw /NDxx program.c
 
     In this example, the /Gw and /NDxx options are incompatible
     because each has a different special-entry sequence.

     Compiler warning (level 1) C4112
 
     macro name 'name' is reserved, 'command' ignored
 
     The given command attempted to define or undefine the predefined
     macro <name> or the preprocessor operator 'defined'. The given
     command is displayed as either #define or #undef, even if the
     attempt was made using command-line options.

     LINK warning L4067
 
     Possible .COM/.EXE inconsistency
 
     The linker detected a possible inconsistency in compile
     and link options. This warning message has four possible
     causes:
 
        ■ You selected Generate .COM File in the Link Options dialog
          box and did not specify Tiny model in the Compile Options
          dialog box.
 
        ■ You selected Generate .COM File while using full segment
          definitions. As long as you use the required statements
          to successfully generate a COM file, you can safely ignore
          this warning.

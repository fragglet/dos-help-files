     Compiler warning (level 2) C4056
 
     floating-point overflow
 
     The compiler generated a floating-point exception while doing
     constant arithmetic on floating-point items at compile time.
 
     For example:
 
          float fp_val = 1.0e100;
 
     In this example, the floating-point constant 1.0e100 exceeds the
     maximum allowable value for a double-precision data item.

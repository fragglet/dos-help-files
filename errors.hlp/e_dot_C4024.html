     Compiler warning (level 1) C4024
 
     'function' : different types : parameter 'number'
 
     The type of the given parameter in a function call did not agree
     with the type given in the argument-type list or function
     definition.
 
     The parameter will be passed without change. The function will
     interpret the parameter's type as the type expected by the
     function.

     Compiler warning (level 1) C4021
 
     'function' : too few actual parameters
 
     The number of arguments specified in a function call was less
     than the number of parameters specified in the function prototype
     or function definition.
 
     Only the provided actual parameters are passed. If the called
     function references a variable that was not passed, the results
     are undefined and may be unexpected.

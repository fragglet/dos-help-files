     Fatal error C1026
 
     parser stack overflow, please simplify your program
 
     The program cannot be processed because the space required to
     parse the program causes a stack overflow in the compiler.
 
     Simplify the program by decreasing the complexity of expressions.
     Decrease the level of nesting in for and switch statements by
     putting some of the more deeply nested statements in separate
     functions. Break up very long expressions involving ',' operators
     or function calls.

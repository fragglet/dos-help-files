     Compiler error C2429
 
     'label' : illegal far label reference
 
     FAR PTR may not be used on jumps or calls to labels. Far
     references to functions are allowed as long as the function has
     been declared.
                   C4001, C4140 remove reference to /qc
                   global changes for QC 2.50 compatibility

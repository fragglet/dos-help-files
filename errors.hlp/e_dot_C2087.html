     Compiler error C2087
 
     'identifier' : missing subscript
 
     The definition of an array with multiple subscripts was missing
     a subscript value for a dimension other than the first dimension.
 
     The following is an example of an illegal definition:
 
          int func(a)
              char a[10][];
              { }
 
     The following is an example of a legal definition:
 
          int func(a)
              char a[][5];
              { }

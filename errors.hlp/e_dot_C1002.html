     Fatal error C1002
 
     compiler is out of heap space in Pass 2
 
     The compiler ran out of dynamic memory space. Usually this means
     the program has too many symbols and/or complex expressions.
 
     One of the following may be a solution:
 
        ■ Divide the file into several smaller source files.
 
        ■ Break expressions into smaller subexpressions.
 
        ■ Remove other programs or drivers running in the system
          which could be consuming significant amounts of memory.
 
        ■ Compile without using NMAKE.

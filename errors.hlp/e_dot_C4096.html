     Compiler warning (level 2) C4096
 
     'attribute1' must be used with 'attribute2'
 
     The use of attribute2 requires the use of attribute1.
 
     For example, using a variable number of arguments (...) requires
     that _cdecl be used. Also, _interrupt functions must be _far and
     _cdecl.
 
     The compiler assumed <attribute1> for the function.

     Command-line error D2027
 
     could not execute 'component'
 
     The QCL driver was unable to execute one of the compiler components
     or the linker, possibly for one of the following reasons:
 
        ■ There is not enough memory to load the component.
 
        ■ The given component is for the wrong operating system.
 
        ■ The given component is corrupted.

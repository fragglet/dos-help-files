     Compiler warning (level 1) C4029
 
     declared parameter list different from definition
 
     The types in the argument list given in a function declaration did
     not agree with the types of the formal parameters given in the
     function definition.
 
     The parameter list of the definition is used instead of the
     parameter list of the declaration.

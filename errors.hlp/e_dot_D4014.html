     Command-line warning D4014
 
     invalid value 'number1' for 'option'; default 'number2' is used
 
     An invalid value was given for an option that only accepts certain
     specific values. The value actually used by the compiler is <number2>.
 
     The following is an example that can cause this warning:
 
          qcl /Zp3 program.c
 
     In this example, 3 is an invalid value. In the /Zpn option, the
     value of <n> may be 1, 2, or 4 only.

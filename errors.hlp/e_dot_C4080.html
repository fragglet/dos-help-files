     Compiler warning (level 1) C4080
 
     expected identifier for segment name, found 'token'
 
     The first argument in the argument list for the alloc_text pragma
     is missing a segment name. This happens if the first token in the
     argument list is not an identifier.
 
     The pragma was ignored.

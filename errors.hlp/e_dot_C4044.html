     Compiler warning (level 1) C4044
 
     _huge on 'identifier' ignored, must be an array
 
     The compiler ignored the _huge memory attribute on the given
     identifier. Only arrays may be declared with the _huge memory
     attribute. On pointers, _huge must be used as a modifier, not as
     a memory attribute.

     Compiler warning (level 1) C4078
 
     case constant 'value' too big for the type of switch expression
 
     A value appearing in a case statement was larger than the size of
     the type in the switch expression. The compiler converted the case
     value to the type of the switch expression.
 
     A problem can occur when two case constants have different values
     before being cast but the same value afterward.

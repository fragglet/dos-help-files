     Compiler warning (level 4) C4127
 
     conditional expression is constant
 
     The controlling expression of an if statement or while loop
     evaluates to a constant.
 
     As a result, the code in the body of the if statement or while
     loop is either always executed or never executed.
 
     This warning is informational.

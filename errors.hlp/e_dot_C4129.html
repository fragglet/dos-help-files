     Compiler warning (level 4) C4129
 
     'character' : unrecognized character escape sequence
 
     The <character> following a backslash in a character or string
     constant was not recognized as a valid escape sequence.
 
     As a result, the backslash is ignored and not printed, and the
     character following the backslash is printed.
 
     To print a single backslash (\), specify a double backslash
     (\\).

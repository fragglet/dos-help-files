     Compiler warning (level 1) C4090
 
     different const/volatile qualifiers
 
     A pointer to an item declared as const was assigned to a pointer
     that was not declared as const. As a result, the const item
     pointed to could be modified without being detected.
 
     The expression was compiled without modification.
 
     The following example causes this warning:
 
          const char *p = "abcde";
          int str(char *s);
          str(p);

     LINK warning L4034
 
     more than 239 overlay segments; extra put in root
 
     The link command line or response file designated too many
     segments to go into overlays.
 
     The limit on the number of segments that can go into overlays is
     239. Segments starting with the 240th segment are assigned to the
     permanently resident portion of the program (the root).

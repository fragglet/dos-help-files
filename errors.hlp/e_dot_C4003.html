     Compiler warning (level 1) C4003
 
     not enough actual parameters for macro 'identifier'
 
     The number of actual arguments specified with the given identifier
     was less than the number of formal parameters given in the macro
     definition of the identifier.
 
     When a formal parameter is referenced in the definition and the
     corresponding actual parameter has not been provided, empty text
     is substituted in the macro expansion.

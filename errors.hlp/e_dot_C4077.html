     Compiler warning (level 1) C4077
 
     unknown check_stack option
 
     An unknown option was given with the old form of the check_stack
     pragma. With the old form, the argument to the pragma must be
     empty, +, or -.
 
     For example, the following causes this warning:
 
          #pragma check_stack yes
 
     The following code corrects this situation:
 
          #pragma check_stack +    /* Old form */
     or
          #pragma check_stack (on) /* New form */

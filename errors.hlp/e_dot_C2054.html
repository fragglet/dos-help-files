     Compiler error C2054
 
     expected '(' to follow 'identifier'
 
     The context requires parentheses after the function identifier.
 
     One cause of this error is forgetting an equal sign (=) on a
     complex initialization, as in
 
          int array1[]    /* Missing = */
          {
            1,2,3
          };

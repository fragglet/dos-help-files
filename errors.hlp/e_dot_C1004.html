     Fatal error C1004
 
     unexpected end-of-file found
 
     The default disk drive did not contain sufficient space for the
     compiler to create temporary files. The space required is
     approximately two times the size of the source file.
 
     This message also appears when the #if directive occurs without a
     corresponding closing #endif directive while the #if test directs
     the compiler to skip the section.

     Compiler warning (level 4) C4128
 
     storage-class specifier after type
 
     A storage-class specifier (auto, extern, register, static) appears
     after a type in a declaration. The compiler assumed the storage
     class specifier occurred before the type.
 
     New-style code places the storage-class specifier first.

     Compiler error C2416
 
     'identifier' : illegal opcode for processor
 
     The instruction is legal on a later processor but not on the
     current processor.
 
     Check an assembly reference manual to determine which processors
     support this opcode.
 
     The problem can be solved by compiling with the /G1 or /G2 option,
     but then only machines with the newer processor will be able to
     execute the extended instruction.

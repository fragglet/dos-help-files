     Fatal ILINK error L1218
 
     .EXE file too big, change alignment
 
     The segment-sector alignment value in the .EXE file is too small
     to express the size of one of the segments.
 
     Perform a full link and increase the alignment value with the
     /ALIGNMENT option to LINK.

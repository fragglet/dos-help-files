     Compiler error C2173
 
     'function' : actual is not a pointer : parameter 'number1',
     parameter list 'number2'
 
     An attempt was made to pass a nonpointer argument to a function
     that expected a pointer.
 
     This error occurs in calls that return a pointer to a function.
     The first number indicates which argument was in error; the second
     number indicates which argument list contained the invalid
     argument.

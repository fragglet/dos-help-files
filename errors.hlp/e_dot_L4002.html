     LINK warning L4002
 
     frame-relative absolute fix-up near 'offset' in segment 'segment'
 
     A reference was made relative to a segment that was different from
     the target segment of the reference, and both segments are
     absolute (defined with AT).
 
     The linker treats the executable file as if the file were to run
     in real mode only.

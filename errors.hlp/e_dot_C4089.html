     Compiler warning (level 1) C4089
 
     'function' : different types : parameter 'number', parameter list
     'number'
 
     The argument passed to the given function did not have the same
     type as the given parameter in the function definition.
 
     The parameter will be passed without change. The function will
     interpret the parameter's type as the type expected by the
     function.

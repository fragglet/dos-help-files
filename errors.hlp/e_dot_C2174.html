     Compiler error C2174
 
     'function' : actual has type void : parameter 'number1',
     parameter list 'number2'
 
     An attempt was made to pass a void argument to a function. Formal
     parameters and arguments to functions cannot have type void. They
     can, however, have type void * (pointer to void).
 
     This error occurs in calls that return a pointer to a function.
     The first number indicates which argument was in error; the second
     number indicates which argument list contained the invalid
     argument.

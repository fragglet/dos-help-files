     Compiler warning (level 1) C4049
 
     'operator' : indirection to different types
 
     The pointer expressions used with the given operator had different
     base types.
 
     The expressions were used without conversion.
 
     For example, the following code causes this warning:
 
          struct ts1 *s1;
          struct ts2 *s2;
          s2 = s1;   /* Warning */

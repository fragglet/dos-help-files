     Fatal error C1071
 
     unexpected end-of-file found in comment
 
     The compiler found the end of a file while scanning a comment.
 
     Probably a comment was not terminated. Begin at the end of the
     file and search backward for the beginning of a comment. A comment
     begins with /* and ends with */ as in:
 
          /* This is a
          comment */
 
     A comment may not be split across files.

     Compiler error C2128
 
     'identifier' : huge array cannot be aligned to segment boundary
 
     The given huge array was large enough to cross two segment
     boundaries, but could not be aligned to both boundaries to prevent
     an individual array element from crossing a boundary.
 
     If the size of a huge array causes it to cross two boundaries, the
     size of each array element must be a power of two, so that a whole
     number of elements will fit between two segment boundaries.

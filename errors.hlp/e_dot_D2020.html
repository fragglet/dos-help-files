     Command-line error D2020
 
     Either the /Gr or /Gc option was specified on the command line in
     combination with the /Za option, which disables extended keywords.
 
 
     The /Gr option requires the extended keyword _fastcall to be enabled,
     and the /Gc option requires the extended keywords _pascal and _fortran.
     Extended keywords are enabled by default, or if the /Ze option is
     specified.

     Compiler warning (level 1) C4022
 
     'function' : pointer mismatch : parameter 'number'
 
     The pointer type of the given parameter was different from the
     pointer type specified in the argument-type list or function
     definition.
 
     The parameter will be passed without change. Its value will be
     interpreted as a pointer within the called function.

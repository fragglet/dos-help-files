     Compiler error C2152
 
     'identifier' : pointers to functions with different attributes
 
     An attempt was made to assign a pointer to a function declared
     with one calling convention (_cdecl, _fortran, _pascal, or
     _fastcall) to a pointer to a function declared with a different
     calling convention.

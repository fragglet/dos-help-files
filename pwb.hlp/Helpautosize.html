     Helpautosize
 
     The Helpautosize switch determines if PWB displays the Help window
     according to the size of the current topic or displays Help with
     its previous size and position.
 
     This switch must appear in the [PWB-PWBHELP] tagged section of
     TOOLS.INI.
 
     Syntax
 
     Helpautosize:{ yes │ no }
 
     yes    When displaying a new topic, automatically resize the Help
            window to the size of the topic and move the cursor to the
            first Help button.
 
     no     Do not automatically resize the Help window. The Help
            window is displayed with its previous size and position.
 
     Default
 
     Helpautosize:no
 
     Update
 
     In PWB 1.x, the Help window is always automatically resized. In
     PWB 2.00, the Help window is not resized.

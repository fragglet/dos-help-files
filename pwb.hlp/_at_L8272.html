     _pwbhelp_searchres:=cancel pwbhelp.pwbhelpsearch <
 
     Cancel            Establishes a uniform "ground state" by
                       cancelling any selection or argument.
 
     Pwbhelp.          Specifies that the function is the PWBHELP
                       extension function.
 
     Pwbhelpsearch     Opens the Help window and displays the results
                       of the last global Help search.
 
     <                 Restores the function's prompt (if any). By
                       default, function prompts are suppressed while a
                       macro is running.

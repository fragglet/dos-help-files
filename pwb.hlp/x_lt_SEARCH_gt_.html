     "<SEARCH>Search Results"
 
     The <SEARCH> pseudofile contains the results of logged searches.
 
     You can open this file by opening the Window menu and choosing
     Search Results from the PWB Windows cascaded menu. You can also
     choose Search Results in the All Files dialog box.

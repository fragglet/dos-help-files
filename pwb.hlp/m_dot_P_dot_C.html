     Project menu, Compile File command («_pwbcompile»)
 
     Macro  ◄_pwbcompile►
 
     Choose Compile File to compile the file in the active window. This
     command is available when the active project template supplies the
     commands to compile the type of file in the active window.
     See: ◄Options menu, Project Templates Cascaded Menu►
 
     PWB compiles the file without running NMAKE. PWB runs NMAKE for the
     Build, Rebuild All, and Build Target commands.

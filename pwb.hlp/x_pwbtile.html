     _pwbtile («_pwbtile»)
 
     Menu  ◄Window menu, Tile►
 
     The _pwbtile macro tiles all unminimized windows on the desktop so
     that no window overlaps another and the desktop is completely
     covered. Up to 16 unminimized windows can be tiled.
 
     If a minimized window has the focus when you execute _pwbtile, that
     window is restored.

     file history
 
     The list of files that have been opened into a window.
 
     Unlike other applications, PWB can open more than one file in a
     single window. PWB does this when the Newwindow switch is set to
     no, when you turn off the New Window option in the Open File dialog
     box, or when you are browsing the results of a multifile search or
     the results of a build.
 
     When a window has a file history, the File menu lists the most
     recently accessed files at the bottom of the menu. Choose a file
     from the menu to switch to that file.
 
     See
 
     Nreplace, Newfile, Newwindow, Nextmsg, Nextsearch, Openfile,

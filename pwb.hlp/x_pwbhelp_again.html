     _pwbhelp_again («_pwbhelp_again»)
 
     Menu  ◄Help menu, Next►
 
     The _pwbhelp_again macro displays the next occurrence of the last
     topic for which you requested Help. If no other occurrences of
     that topic are defined in the open Help files, PWB redisplays the
     current topic.
 
     The topic that PWB looks up when you use this command is displayed
     after the Next command on the Help menu, as follows:
 
          Next: xxxxxxxxxxx  «_pwbhelp_again»
                     ▲
                     └─── Topic to look up

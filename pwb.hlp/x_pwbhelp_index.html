     _pwbhelp_index («_pwbhelp_index»)
 
     Menu  ◄Help menu, Index►
 
     The _pwbhelp_index macro opens the Help window and displays the
     top-level indexes in the Microsoft Advisor Help system.
 
     Within the Microsoft Advisor, most Help topics have an Index button
     at the top of the window. This button also takes you to the top-
     level indexes.

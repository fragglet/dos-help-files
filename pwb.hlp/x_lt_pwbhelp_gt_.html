     "<pwbhelp>Help"
 
     The <pwbhelp> pseudofile contains the current Help topic.
 
     When a Help topic is shown in the Help window, the full pseudofile
     name is
 
          "<pwbhelp>Help: topictitle"
 
     where <topictitle> is the title of the Help topic in the window.
 
     You can open this file by requesting Help. You can also open this
     file by opening the Window menu and choosing Help from the PWB
     Windows cascaded menu. You can also choose Help in the All Files
     dialog box.
 
     See

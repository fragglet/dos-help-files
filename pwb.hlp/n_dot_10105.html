     Cannot open [<file>]: Not a help file
 
     The PWBHELP extension attempted to load the given Help file, but it
     was not a valid Microsoft Advisor Help file.
 
     The file may be a Windows or Presentation Manager Help file. These
     Help files are not compatible with the Microsoft Advisor.
 
     Remove the specified file from the directory or list of files given
     in the HELPFILES environment variable or the Helpfiles switch.

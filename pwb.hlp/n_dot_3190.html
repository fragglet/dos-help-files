PWB Error - Programmer's WorkBench
 
     defwinstyle: invalid value <number>
 
     An attempt was made to set the Defwinstyle switch to a value out
     of range.
 
     The possible values for Defwinstyle are
 
     Value   Style
 
     1       No scroll bars
     3       Vertical scroll bar
     5       Horizontal scroll bar
     7       Both scroll bars
 
     See: Defwinstyle

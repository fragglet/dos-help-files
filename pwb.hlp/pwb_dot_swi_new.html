     The following switches in PWB 2.00 do not exist in PWB 1.x:
 
     Switch            Description
 
     ◄Autoload►         Enables and disables extension autoload
     ◄Beep►             Selects audio or visual alert
     ◄Cursormode►       Sets cursor type based on insert mode
     ◄Enterlogmode►     Enters PWB with search logging turned on
     ◄Envcursave►       Saves the environment in <proj>.STS
     ◄Envprojsave►      Saves the environment in CURRENT.STS
     ◄Helpautosize►     Automatically sizes Help window to topic
     ◄Infodialog►       Enables and disables selected message boxes
     ◄Keepmem►          XMS/EMS memory kept during shell and compile
     ◄Lastproject►      Restores last project on startup
     ◄Mousemode►        Turns mouse off or swaps mouse buttons
     ◄Newwindow►        Opens a new window for new files
     ◄Restorelayout►    Restores project windows and file history
     ◄Savescreen►       Restores the operating-system screen
     ◄Searchdialog►     Displays logged Search Results dialog box
     ◄Searchflush►      Flushes previous logged search results
     ◄Tilemode►         Selects window tiling style
     ◄Timersave►        Auto-saves files at timed intervals
     ◄Word►             Defines a word

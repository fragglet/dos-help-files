     "<browse>Browser Output"
 
     The <browse> pseudofile contains the result of the latest browser
     query.
 
     You can open this file by perfoming a browser query. You can also
     open this file by opening the Window menu and choosing Browser
     Output from the PWB Windows cascaded menu. You can also choose
     Browser Output in the All Files dialog box.

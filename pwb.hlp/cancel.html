     Cancel («cancel»)
 
     Cancels the current selection, argument, or operation. If a message
     appears on the status bar, the Cancel function restores the
     original contents of the status bar.
 
     If a dialog box or menu is open, Cancel closes the dialog box or
     menu and takes no further action. If Help on a dialog box, menu, or
     message box is being displayed, Cancel closes the Help dialog box.
 
     Returns
 
     Cancel always returns true.

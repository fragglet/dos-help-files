     The View Relationship, Function Hierarchy, and Class Hierarchy
     dialog boxes provide the following command buttons:
 
     Command        Description
 
     History        Select a previous query to display.
                    See: ◄Source Browser History►
 
     Back           Display the previous query.
 
     Goto           Close the dialog box and go to the definition of the
                    name shown in the Name box.
 
     Keep Info      Close the dialog box and keep the information in the
                    Browser Output window.
 
     OK             Perform the selected query. Note that OK does not
                    close the dialog box.
 
     Cancel         Close the dialog box.
 
     Help           Display this Help.

     Window menu, Move command («_pwbmove»)
 
     Macro  ◄_pwbmove►
 
     Choose Move to start window-moving mode for the active window. In
     window-moving mode, you can do only the following:
 
     Action                           Key
 
     Move window up one line          UP
     Move window down one line        DOWN
     Move window left one column      LEFT
     Move window right one column     RIGHT
     Accept the new position          ENTER
     Cancel the move                  ESC
 
     To move a window with the mouse:
 
       ■  Drag the top border of the window.

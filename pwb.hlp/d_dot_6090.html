Save Key Assignments

─────Programmer's WorkBench─────────────────────────────────────────────────
 
     Choose Save Key Assignments to save new key assignments
     permanently.
 
     The Modified Key Assignments List box shows all the unsaved key
     assignments made by using the Key Assignments dialog box during
     the PWB session.
 
     To save all new assignments:
 
       ■  Choose OK.
 
     To save only some of the new assignments:
 
       1. Select the assignment you do not want to save.
 
       2. Choose Delete.
 
          Repeat steps 1 and 2 above for each assignment you do not
          want to save.
 
       3. Choose OK.
 
     When you choose OK, PWB modifies the [PWB] section of your
     TOOLS.INI file to reflect the new key assignments. When you exit
     PWB, you are prompted to save TOOLS.INI. Your assignments are not
     permanent until you actually save TOOLS.INI to disk.
 
     NOTE: If you started PWB with the /D or /DT (disable TOOLS.INI)
           command-line options, you cannot save new settings.

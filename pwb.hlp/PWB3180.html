PWB Error - Programmer's WorkBench
 
     PWB3180: Cannot start: not enough conventional memory below 640K
 
     PWB ran out of memory while starting up.
 
     Make more memory available and restart PWB.
 
     You may be able to start PWB in the current configuration by not
     loading some or all PWB extensions. To start PWB with no
     extensions, specify the /DA command-line option.

PWB Message
 
     File is read-only. Invoke: "<command>"?
 
     A request was made to save a file that is marked read-only on
     disk. The Readonly switch is set to <command>, and PWB is
     prompting for confirmation to run the command.
 
     Response    Action
 
     Y (yes)     Run <command>.
     N (no)      Do not run <command> and prompt for a new filename.
     «cancel    »Do not run <command> or prompt for a new name. This
                 cancels the save and returns you to normal editing.

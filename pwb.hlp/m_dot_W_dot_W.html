     Window menu, All Windows command
 
     Choose All Windows to display the All Windows dialog box, which
     lists all open windows in PWB.
 
     To switch to an open window:
 
       1. Select the window you want to make the active window.
 
       2. Choose OK.
 
          PWB closes the dialog box and makes the selected window the
          active window.
 
     To leave the dialog box without switching to a window:
 
       ■  Choose Cancel.

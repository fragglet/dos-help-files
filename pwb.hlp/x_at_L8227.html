  In the View Relationship dialog box, you can select from the
  following queries. The query specifies the relationship to display
  for the object specified in the Name box.
 
  Queries Option    Results
 
  Program Files     All source files in the project
  Program Symbols   All names in the database
  Contains          Names defined in the selected file
  Defined In        Files that define the selected name
  Calls             Functions/macros that the selected function calls
  Called By         Functions/macros that call the selected function
  Uses              Names used by the selected function
  Used By           Objects that refer to the selected name
  Used In           Files that refer to the selected name
  Base Classes      Base classes of the selected class
  Derived Classes   Classes derived from the selected class
  Members           Members of the selected class
  Member Of         Classes of which the selected name is a member
  Vir. Callers      Classes that may call the selected function virtually
  Inh. Members      Inherited members of the selected class
  Imp. Members      Implemented members of the selected class
  Friends In        Classes that declare the selected class as a friend
  Friends Out       Classes declared as friends by the selected class

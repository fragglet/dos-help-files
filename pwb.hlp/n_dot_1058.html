PWB Message
 
     Save all changed files?
 
     A request was made to leave a PWB session, and more than one
     modified file has not been saved.
 
     Response    Action
 
     Y (yes)     Save all modified files and leave PWB.
 
     N (no)      Do not save all modified files. Prompt to save
                 each modified file individually.
 
     «cancel    »Do not save files or leave the session. PWB returns to
                 normal editing.
 
     NOTE: The TOOLS.INI file and a mark file (if you have set one) can
           be modified indirectly and may be saved by this command.
 
           The TOOLS.INI file is modified when you save changes in
           the Key Assignments dialog box, Editor Settings dialog
           box, Customize Run menu dialog box, or Record window. A
           mark (.MRK) file is modified when you set or delete a mark
           and a mark file is set.

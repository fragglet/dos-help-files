     Emacsnewl («emacsnewl»)
 
     In insert mode, starts a new line. In overtype mode, moves the
     cursor to the beginning of the next line.
 
     PWB automatically positions the cursor on the new line, depending
     on the setting of the Softcr switch.
     See: Softcr
 
     Returns
 
     Emacsnewl always returns True.
 
     Update
 
     In PWB 1.x, PWB performs special automatic indentation for C files.
     In PWB 2.00, language-specific automatic indentation is handled by
     language extensions if the feature is enabled. Otherwise, PWB uses
     its default indentation rules.
     See: Softcr
 
     See
 
     Newline, Softcr, C_Softcr

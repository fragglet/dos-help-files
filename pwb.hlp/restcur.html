     Restcur («restcur»)
 
     Moves the cursor to the last position saved with the Savecur
     function («savecur»).
     See: Savecur
 
     Restcur always clears the saved position.
 
     Returns
 
     True:  Position restored.
     False: No saved position to restore.
 
     See
 
     Selcur

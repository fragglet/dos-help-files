     Search menu, Goto Match command («_pwbgotomatch»)
 
     Macro  ◄_pwbgotomatch►
 
     Choose Goto Match to jump to a given match listed in the Search
     Results window. Goto Match is available only when the cursor is on
     a match in the Search Results window.
 
     To jump to a match listed in Search Results:
 
       1. Open the Window menu and choose Search Results from the PWB
          Windows cascaded menu.
       2. Move the cursor to the match you want to jump to.
       3. Choose Goto Match or execute the _pwbgotomatch macro
          («_pwbgotomatch»).
 
     If the file that contains the match is open, PWB makes the window
     with the file the active window and moves the cursor to the
     indicated location. If the file containing the match is not open,
     PWB opens a new window for the file or opens the file in the
     active window according to how the Newwindow switch is set.
     See: Newwindow

PWB Message
 
     Out of disk space for VM swap file - recovering
 
     PWB ran out of disk space for its Virtual Memory (VM) file. This
     is not a fatal error, and PWB attempts to recover.
 
     To avoid this condition, allow more space on the disk that the TMP
     environment variable points to.

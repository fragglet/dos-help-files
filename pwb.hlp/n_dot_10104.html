     Cannot open [<file>]: Invalid appended help file
 
     The PWBHELP extension attempted to load a Help file that contains
     more than one Help database, but one or more of the concatenated
     files was not a valid Help database.
 
     Try using HELPMAKE with the /DS option to split the Help file into
     its component parts. By loading the files individually, you may be
     able to discover the invalid file. You can then concatenate the
     Help file, omitting the invalid files.

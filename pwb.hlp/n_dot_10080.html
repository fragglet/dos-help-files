     No Open Help Files
 
     A request for Help was made, but there are no open Help files.
 
     The HELPFILES environment variable may not be correct.
 
     Within a PWB session, you can use the following methods to open
     Help files:
 
       ■  Choose Environment Variables from the Options menu to set the
          HELPFILES environment variable. Type the path to the Help
          files.
 
       ■  To load a specific Help file, use the Sethelp function
          («Sethelp»).

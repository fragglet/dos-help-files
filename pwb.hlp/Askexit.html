     Askexit (Boolean)
 
     The Askexit switch determines if PWB prompts for confirmation
     before returning to the operating system.
 
     Syntax
 
     Askexit:{ yes | no }
 
     yes    Prompt for confirmation before leaving PWB.
 
     no     Do not prompt before leaving PWB.
 
     Default
 
     Askexit:no
 
     See
 
     Exit

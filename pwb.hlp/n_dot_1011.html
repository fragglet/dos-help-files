PWB Message
 
     File has been deleted from disk
     <file>
     Discard from memory?
 
     PWB is synchronizing the file in memory with the file on disk and
     has detected that the file on disk has been deleted. PWB is
     prompting for confirmation to discard the file from memory. You
     can choose one of the responses below:
 
     Response   Action
 
     Yes        PWB discards the file. The contents of the file are
                lost and cannot be recovered.
 
     No         PWB retains the file in memory.
 
                To write the file back to disk, choose the Save command
                from the File menu.
 
                To write the file to disk with a different name, choose
                the Save As command from the File menu.

─────Programmer's WorkBench─────────────────────────────────────────────────
 
     The Options menu provides the following commands to display and
     set environment variables, customize PWB, and change options for
     the project:
 
     Command                  Description
 
     ◄Environment Variables►  Set environment variables.
     ◄Key Assignments►        Assign functions and macros to keys.
     ◄Editor Settings►        Change PWB and extension switches.
     ◄Colors►                 Customize PWB screen colors .
 
     ◄Build Options►          Set build type and directory.
     ◄Project Templates►      Manipulate project templates.
 
     ◄Language Options►       Set compiler options (cascaded list of
                              installed languages).
 
     ◄Browse Options►         Set browser options.
 
     ◄LINK Options►           Set linker options.
     ◄NMAKE Options►          Set NMAKE options.
     ◄CodeView Options►       Set CodeView(R) debugger options.
 
     Some utilities, such as RC, WX, and MPC, do not have an Options
     menu command. To set options for one of these utilities, you must
     use the Customize Project Templates command to create a <util>
     FLAGS macro for that utility.
     See: ◄Build:macro►

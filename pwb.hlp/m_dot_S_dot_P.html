     Search menu, Previous Match command («_pwbpreviousmatch» when logging off)
                                         («_pwbpreviouslogmatch» when logging on)
 
     Macros  ◄_pwbpreviousmatch► (when logging off)
             ◄_pwbpreviouslogmatch► (when logging on)
 
     Choose Previous Match to perform one of two actions, depending on
     the search-logging state:
 
     Logging    Next Match Action
 
     Off        Repeats the last find, searching backward from the cursor
                by executing _pwbpreviousmatch.
 
     On         Moves the cursor to the previous match in the Search
                Results pseudofile by executing _pwbpreviouslogmatch.
 
     Previous Match is unavailable when no search has been performed and
     logging is off or when there are no matches in the Search Results
     window and logging is on.
 
     See
 
     Clearsearch, Enterlogmode, Logsearch, Nextsearch, _pwbnextlogmatch,
     _pwbgotomatch, _pwbpreviouslogmatch

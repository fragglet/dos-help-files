     Lastproject (Boolean)
 
     The Lastproject switch determines if PWB automatically opens the
     last project on startup.
 
     The /PN, /PP, /PL, and /PF command-line options override the
     setting of the Lastproject switch.
     See: ◄PWB Command Line►
 
     Syntax
 
     Lastproject:{ yes | no }
 
     yes    On startup, open the last project that was open.
 
     no     Do not open the last project on startup.
 
     Default
 
     Lastproject:no

     Clearmsg («clearmsg»)
          Clear the contents of the Build Results window.
 
     Arg Clearmsg («arg» «clearmsg»)
          Clear the current set of messages in the Build Results
          window.
 
     Returns
 
     True:  Cleared a message set or the contents of Build Results.
     False: The Build Results window is empty.
 
     See
 
     Nextmsg, _pwbnextmsg, _pwbprevmsg, _pwbsetmsg

     The Usercmd function executes a custom command added to the Run
     menu by using Run Customize or setting the User switch.
     See: User, ◄Run menu, Customize Run Menu►
 
     Arg <number> Usercmd («arg» <number> «usercmd»)
 
     Executes the given custom Run menu command. The <number> can be in
     the range 1-9.
 
     Returns
 
     True:  Command exists.
     False: Command does not exist, or invalid argument.
 
     See

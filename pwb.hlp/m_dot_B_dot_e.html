     Browse menu, Module Outline command («pwbrowseoutline»)
 
     Function  Pwbrowseoutline
 
     Choose Module Outline to display the Outline dialog box, where you
     can select a file or set of files to outline.
 
     To outline a set of files, type a wildcard in the File Name box. To
     outline a single file, type the name of that file in the File Name
     box or select it in the File List box.
 
     You can select the types of objects that are listed in the outline
     by changing the options under Show only. You can display functions,
     variables, types, macros, and classes.
 
     Choose OK to display an outline for the specified files in the
     Browser Output window.

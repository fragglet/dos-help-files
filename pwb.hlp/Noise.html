     Noise (Numeric)
 
     The Noise switch specifies the number of lines counted at a time
     as PWB traverses a file while reading, writing, or searching. PWB
     displays the line counter on the right-hand side of the status
     bar, in the area which usually shows the current line.
 
     Set Noise to 0 to turn off the display of scanned lines.
 
     Default
 
     Noise:50

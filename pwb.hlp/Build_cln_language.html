     The Build:language rule holds data that is defined and maintained
     by PWB. It should not appear in a custom template.
 
     Syntax
 
     Build:language "<language>"
 
     <language>    Specifies the Runtime Support group of templates
                   that this template belongs to.
 
     Example
 
     When a custom template contains the following rule:
 
          Build:language "None"
 
     the template is listed along with the other templates that are
     available when None is chosen as the run-time support.

  The PWB Help menu usually contains the following commands:
 
  Command          Description                          Key
 
  ◄About►          Show PWB version and copyright       ───
  ◄Index►          Show Help indexes                    «_pwbhelp_index»
  ◄Contents►       Show Help contents                   «_pwbhelp_contents»
  ◄Topic►          Show Help on current topic           «_pwbhelp_context»
  ◄Help on Help►   Show Help on the Microsoft Advisor   «_pwbhelp_general»
  ◄Next►           Show next Help on current topic      «_pwbhelp_again»
  ◄Global Search►  Search all Help for text or pattern  ───
  ◄Search Results► Show global search results           «_pwbhelp_searchres»
 
  When the PWBHELP extension is not loaded, the Help menu contains only
  the About command.
 
  When you request Help on a menu command or dialog box, PWB displays
  the Help in a pop-up window.

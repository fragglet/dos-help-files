     To switch to any open window by window number:
 
       1. Execute Arg («arg»).
       2. Type the window number.
       3. Execute Selwindow («selwindow»).
 
     To go to the next window:
 
       ■  Execute Selwindow («selwindow»).
 
     To go back to the previous window:
 
       ■  Execute the _pwbprevwindow macro («_pwbprevwindow»).
        OR
       ■  Execute the function sequence Meta Selwindow («meta» «selwindow»).

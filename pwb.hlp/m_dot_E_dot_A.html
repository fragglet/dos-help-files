     Edit menu, Set Anchor command («savecur»)
 
     Function  ◄Savecur►
 
     Choose Set Anchor to save the cursor position for later use with
     the Select To Anchor command or the Selcur («selcur») and Restcur
     («restcur») functions.
     See: Restcur, Selcur, ◄Edit menu, Select To Anchor►
 
     To select text with the anchor commands:
 
       1. Move the cursor to the start of the text you want to select.
 
       2. Choose Set Anchor, or execute the Savecur function («savecur»).
 
       3. Move the cursor to the end of the text you want to select.
 
       4. Choose Select To Anchor, or execute the Selcur function
          («selcur»).
 
     The selection is a stream, box, or line selection according to the
     current selection mode. To change the type of selection, use the
     Stream Mode, Box Mode, or Line Mode commands on the Edit menu, or
     the Selmode function («selmode»).
     See: Selmode, ◄Edit menu, Stream Mode► ◄Edit menu, Box Mode►
          ◄Edit menu, Line Mode► ◄Selection Modes (Defined)►

     "<ASSIGN>Current Assignments and Switch Settings"
 
     Update
 
     The <ASSIGN> pseudofile in PWB 1.x is obsolete and does not exist
     in PWB 2.00.
 
     In PWB 1.x, this pseudofile contains the current assignments and
     switch settings. It provides the means of changing these settings.
     You make these changes in PWB 2.00 by using the dialog boxes in
     the Options menu.

     No previously viewed Help
 
     A request was made to view the last Help topic when no Help has
     yet been viewed.
 
     The request was made by executing the _pwbhelp_back macro or by
     pressing a Back Help button. This error also occurs when a request
     is made to view the next Help topic in the viewing sequence by
     executing the _pwbhelp_again macro.
 
     The current key assignment for _pwbhelp_back is «_pwbhelp_back».
     The current key assignment for _pwbhelp_again is «_pwbhelp_again».

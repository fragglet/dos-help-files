     Options menu, Key Assignments command
 
     Choose Key Assignments to view the current key assignments and
     assign functions and macros to keys. The Key Assignments dialog
     box provides the following options:
 
     Option                Description
 
     Macro/Function Name   Specifies the name of the command to assign
                           to a key.
 
     New Key               Specifies the key for the command.
                           See: ◄Using the Key Box►
 
     Assigned To           Shows the command assigned to the key.
 
     Macro/Function List   Lists the available commands.
 
     Current Keys          Lists the keys that are assigned the command
                           shown in the Macro/Function Name box.
 
     Unassigned Keys       Lists the keys that are assigned the
                           Unassigned function.
 
     The following commands operate on the key shown in the New Key box
     and the macro or function named in the Macro/Function Name box:
 
     Command           Description
 
     Assign           Assigns the macro or function to the key.
 
     Unassign         Assigns the Unassigned function to the key and
                      moves the key to the Unassigned Keys list.
 
     Save             Displays the Save Key Assignments dialog box
                      where you can select the assignments that you
                      want to save for the next session.
                      See: ◄Save Key Assignments►
 
     Function Help    Displays help on the text in the Macro/Function
                      Name box.
 
     To accept the new assignments, choose OK. To cancel all
     assignments you make while the dialog box is open, choose Cancel.
 
     NOTE: Some keys listed as available keys have a reserved meaning
           in certain conditions or operating environments.
           See: ◄Available Keys Note►

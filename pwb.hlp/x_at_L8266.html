     _pwbhelp_back:=cancel meta pwbhelp.pwbhelpnext <
 
     Cancel         Establishes a uniform "ground state" by canceling
                    any selection or argument.
 
     Meta           Sets the meta prefix for the function.
 
     Pwbhelp.       Specifies that the function is the PWBHELP
                    extension function.
 
     Pwbhelpnext    Displays the previously viewed Help topic.
 
     <              Restores the function's prompt (if any). By
                    default, function prompts are suppressed while a
                    macro is running.

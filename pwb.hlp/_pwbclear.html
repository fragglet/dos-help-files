     _pwbclear («_pwbclear»)
 
     Menu  ◄Edit menu, Delete►
 
     The _pwbclear macro removes the selected text from the file. If
     there is no selection, PWB removes the current line.
 
     The selection or line is not copied to the clipboard. It can be
     recovered only by using the Undo command on the Edit menu or the
     Undo function («undo»).
     See: Undo, ◄Edit menu, Undo►

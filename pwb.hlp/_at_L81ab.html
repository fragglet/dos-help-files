     To define a mark at the cursor:
 
       1. Execute Arg Arg («arg» «arg»).
       2. Type the name of the mark.
       3. Execute Mark («mark»).
 
     A mark defined this way is saved in the mark file.
 
     To delete a mark:
 
       1. Execute Arg Arg Meta («arg» «arg» «meta»).
       2. Type the name of the mark.
       3. Execute Mark («mark»).

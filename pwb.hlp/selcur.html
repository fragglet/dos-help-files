     Selcur («selcur»)
 
     Selects text from the cursor to the position saved using the Set
     Anchor command from the Edit menu or the Savecur function
     («savecur»).
     See: Savecur, ◄Edit menu, Set Anchor►
 
     If no position has been saved, Selcur selects text from the cursor
     to the beginning of the file.
 
     Returns
 
     Selcur always returns true.

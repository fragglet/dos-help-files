     File menu, All Files command
 
     Choose All Files to display the All Files dialog box, which lists
     all files that are open in PWB.
 
     To close an open file:
 
       1. Select the file you want to close.
 
       2. Choose Close.
 
          The Close button is unavailable for files that PWB keeps
          open, such as Build Results, Clipboard, CURRENT.STS, or
          TOOLS.INI.
 
     To switch to an open file:
 
       1. Select the file you want to switch to.
 
       2. Choose OK.
 
     To leave the All Files dialog box without closing or switching to
     a file:
 
       ■  Choose Cancel.

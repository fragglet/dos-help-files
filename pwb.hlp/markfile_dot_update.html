     With PWB 1.x, when you assign a new value to the Markfile switch
     (by using Set Mark File or the Assign function) and no mark
     file is currently open, all marks are lost.
 
     With PWB 2.00, if no mark file is open and you have set marks, all
     marks are saved in the new mark file.

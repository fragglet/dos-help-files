PWB Error
 
     Must specify a valid environment variable setting
 
     A request was made to set an environment variable, but no
     environment variable was specified. Enter the name and value for
     an environment variable in the Variable text box, or cancel the
     Environment Variables dialog box.
 
     The correct syntax for an environment variable setting is
 
          <variable>=<value>

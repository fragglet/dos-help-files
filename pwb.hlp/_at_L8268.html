     _pwbhelp_errhelp:=cancel pwbhelp.pwbhelperr <
 
     Cancel        Establishes a uniform "ground state" by canceling
                   any selection or argument.
 
     Pwbhelp.      Specifies that the function is the PWBHELP extension
                   function.
 
     Pwbhelperr    Displays Help for the message displayed at the bottom
                   of the window.
 
     <             Restores the function's prompt (if any). By default,
                   function prompts are suppressed while a macro is
                   running.

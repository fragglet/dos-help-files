     Options menu, Language Options cascaded menu
 
     The Language Options cascaded menu lists the compiler options
     commands for all loaded language extensions. Use these commands to
     set the build options for each language's compiler.
 
     The commands that are displayed in this menu depend on which
     language extensions are loaded. The commands may include:
 
          ◄BASIC Compiler Options►
          ◄C Compiler Options►
          ◄C++ Compiler Options►
          ◄COBOL Compiler Options►
          ◄FORTRAN Compiler Options►
          ◄MASM Options►
 
     For help on additional options not listed here, select that
     command and press F1.

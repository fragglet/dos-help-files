     Choose Open Custom to open a custom browser database, save a custom
     database selection in TOOLS.INI, or use the default database. The
     Open Custom command displays the Custom Database Management dialog
     box, which has the following options:
 
     Option                  Description
 
     Use Custom Database     Selects a database other than the database
                             for the current project.
                             See: ◄Select Browser Database►
 
     Use Default Database    Selects the database for the current
                             project. If no project is open, the current
                             database is closed.
 
     Save Current Database   Saves the name of the database in the
                             [PWB-PWBROWSE] section of TOOLS.INI. This
                             changes the setting of the Browdbase
                             switch.
 
     To close an open database, assign an empty value to the Browdbase
     switch by using the Editor Settings command on the Options menu or
     the Assign function («assign»).
     See: Assign, ◄Options menu, Editor Settings►

     _pwbsavefile («_pwbsavefile»)
 
     Menu  ◄File menu, Save►
 
     The _pwbsavefile macro saves the current file to disk.
 
     If the current file is a pseudofile (an 'Untitled' file), PWB
     displays the Save As dialog box so you can give the file a more
     meaningful name.

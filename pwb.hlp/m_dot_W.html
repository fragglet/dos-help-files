     The Window menu provides the following commands to open, close,
     resize, move, rearrange, and select windows in PWB:
 
     Command        Description                          Key
 
     ◄New►          Duplicate the active window          «_pwbnewwindow»
     ◄Close►        Close the window                     «_pwbclose»
     ◄Close All►    Close all windows and files          «_pwbcloseall»
     ◄Move►         Move the window                      «_pwbmove»
     ◄Size►         Resize the window                    «_pwbresize»
 
     ◄Restore►      Restore the window's size            «_pwbrestore»
     ◄Minimize►     Shrink the window to an icon         «_pwbminimize»
     ◄Maximize►     Enlarge the window to maximum size   «_pwbmaximize»
 
     ◄Cascade►      Cascade all unminimized windows      «_pwbcascade»
     ◄Tile►         Tile all unminimized windows         «_pwbtile»
     ◄Arrange►      Arrange all unminimized windows      «_pwbarrange»
 
     ◄PWB Windows►  Show a cascaded menu of PWB windows  ───
 
     ◄1 file1►      Switch to window number 1            «_pwbwindow1»
     . . .
     ◄9 file9►      Switch to window number 9            «_pwbwindow9»
     ◄All Windows►  Show list of all open windows        ───

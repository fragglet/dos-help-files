     Edit menu, Line Mode command («_pwblinemode»)
 
     Macro  ◄_pwblinemode►
 
     Choose Line Mode to change the selection mode to line mode.
 
     A line selection is a range of one or more complete lines, as
     follows:
 
          abcdefghijklmnopqrstuvwxyz     ◄Stream►
          ██████████████████████████     ◄Box►
          ██████████████████████████     ◄Line►
          abcdefghijklmnopqrstuvwxyz
 
     TIP: To switch among the three selection modes while selecting
          text with the mouse, click the right button while you hold
          down the left button.
 
     A bullet appears next to the Edit menu command for the current
     selection mode.
 
     See
 
     Selmode, _pwbstreammode, _pwbboxmode, _pwblinemode

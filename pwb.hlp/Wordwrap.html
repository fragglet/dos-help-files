     Wordwrap (Boolean)
 
     The Wordwrap switch determines if PWB performs automatic word wrap
     as you enter text.
 
     When word wrapping is turned on and you type a nonspace character
     past the column specified by Rmargin, PWB brings the current word
     down to a new line. A word is defined by the Word switch.
     See: Rmargin, Word
 
     Syntax
 
     Wordwrap:{ yes | no }
 
     yes    Wrap words as you enter text.
 
     no     Do not wrap.
 
     Default
 
     Wordwrap:no

     Options menu, Colors command
 
     The Colors dialog box provides the following options to customize
     the colors of the PWB screen:
 
     Option         Description
 
     Color          Select the name of the item to change.
                    See: ◄Color Names►
 
     Foreground     Select the foreground color.
 
     Bright Fore    Turn on for a bright foreground color.
 
     Background     Select the background color.
 
     Bright Back    Turn on for a bright background color.
 
     The sample text displays the currently selected color combination.
 
     The following command buttons operate upon the currently selected
     color:
 
     Command        Description
 
     Set Color      Sets the selected color name to the selected
                    foreground and background colors. PWB updates the
                    entire screen to show the new color.
 
     Save           Brings up the Save Colors dialog box where you can
                    select the changed colors that you want to save for
                    the next session.
                    See: ◄Save Colors►

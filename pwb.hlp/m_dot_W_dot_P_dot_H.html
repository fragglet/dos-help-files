     Window menu, PWB Windows, Help
 
     Choose Help to display the Help window.
 
     PWB opens the Help window and displays the last Help that you
     viewed during the current session. If you have not used Help
     during the current session, the window is empty.
 
     To close the Help window, press ESC when Help is active or choose
     Close from the Window menu («_pwbclose»).

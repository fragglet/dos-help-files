PWB Error
 
     <directory> does not exist
 
     A request was made to set a build directory using the Build
     Options command from the Options menu, but PWB could not find
     the specified directory.
 
     One of the following conditions may hold:
 
        ■ The specified directory does not exist.
        ■ The specified directory name contains a spelling error.

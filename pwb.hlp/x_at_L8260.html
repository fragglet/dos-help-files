     While performing a global Help search, PWB displays the Searching
     dialog box, which shows the following information:
 
       ■  Text or pattern you are searching for.
       ■  Current Help database being searched.
       ■  Total number of topics with a match.
       ■  Number of topics not yet searched in the current database.
       ■  Number of databases remaining to be searched.
 
     To stop the search at any time to examine the partial results of
     the search, choose the Stop Search button.

     Curfilenam («curfilenam»)
 
     The Curfilenam macro types the base name of the current file. This
     macro is redefined each time you switch to a new file.
 
     Definition
 
     curfilenam := "<basename>"
 
     Example
 
     The following macro copies the base name of the current file to
     the clipboard for later use:
 
     Name2clip := Arg Curfilenam Copy
 
     See
 
     Arg, Copy, Curdate, Curday, Curfile, Curfilenam, Curfileext,
     Curtime

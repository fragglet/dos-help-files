     The Build:template rule holds data that is defined and maintained
     by PWB. It should not appear in a custom template. You can,
     however, change the template name by editing this rule in the
     Customize Template dialog box.
 
     Syntax
 
     Build:template "<template>" [modified]
 
     <template>    Specifies the name of the template.
 
     modified      If present, indicates that some macro values or
                   rules in the template have been modified since the
                   template was set.

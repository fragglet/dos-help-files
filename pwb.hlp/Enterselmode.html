     Enterselmode (Text)
 
     The Enterselmode switch determines the selection mode when PWB
     starts up.
     See: ◄Selection Modes (Defined)► ◄Selection Modes (Setting)►
 
     Syntax
 
     Enterselmode:{ stream | box | line }
 
     stream    Starts PWB in stream selection mode.
 
     box       Starts PWB in box selection mode.
 
     line      Starts PWB in line selection mode.
 
     Default
 
     Enterselmode:stream
 
     See

     Do you want to Build/Rebuild current target?
 
     A request was made to execute the current program, and PWB has
     detected that a target in the project is out-of-date. That is, a
     source file in the project has been modified.
 
     You can choose from the following commands:
 
     Command         Action
 
     Build Target    Builds the current target.
 
     Rebuild All     Rebuilds all targets in the project.
 
     Run Program     Runs the project's program.
 
     Cancel          Closes this dialog box.

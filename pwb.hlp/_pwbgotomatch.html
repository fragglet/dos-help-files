     _pwbgotomatch («_pwbgotomatch»)
 
     Menu  ◄Search menu, Goto Match►
 
     The _pwbgotomatch macro sets the match listed at the current
     location in the Search Results pseudofile as the current match
     and moves the cursor to the location specified by that match.

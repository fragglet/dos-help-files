     Update: Project Templates
 
     With PWB 1.x, the default templates are named Default Options.
 
     With PWB 2.00, these templates are called Generic Options to more
     accurately describe the type of project that is produced when the
     template is used for a project.
 
     The Generic Options templates generally specify only the default
     command-line options for all tools and make no assumptions about
     the nature of the target. For most applications, you must
     customize the starting options provided by the Generic Options
     template.

  Cursor Movement                    Command          Keystroke
 
  Up one line                        Up               «up»
  Down one line                      Down             «down»
  Left one column                    Left             «left»
  Right one column                   Right            «right»
 
  Upper-left corner of window        Home             «home»
  Top of window                      Meta Up          «meta» «up»
  Bottom of window                   Meta Down        «meta» «down»
  Leftmost column in window          Meta Left        «meta» «left»
  Rightmost column in window         Meta Right       «meta» «right»
 
  Up one window                      Mpage            «mpage»
  Down one window                    Ppage            «ppage»
  Column one                         Meta Begline     «meta» «begline»
  One column past window width       Meta Endline     «meta» «endline»
 
  Back one word                      Mword            «mword»
  Forward one word                   Pword            «pword»
  Beginning of line                  Begline          «begline»
  End of line                        Endline          «endline»
 
  Next paragraph                     Ppara            «ppara»
  Previous paragraph                 Mpara            «mpara»
  End of paragraph                   Meta Ppara       «meta» «ppara»
  End of previous paragraph          Meta Mpara       «meta» «mpara»
 
  Beginning of file                  Begfile          «begfile»
  End of file                        Endfile          «endfile»
  To specific line number            Arg "line" Mark  «arg» "line" «mark»
 
  Position before last scroll        Arg Mark         «arg» «mark»

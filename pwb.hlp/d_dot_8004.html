     The Source Browser History dialog boxes provide the following
     options to work with previous queries performed in the View
     Relationship, Function Hierarchy, and Class Hierarchy dialog boxes.
 
     Field                      Description
 
     Recently Visited History   Lists the recent queries. The object
                                name is followed by the query that was
                                performed.
 
     Operation was              If displayed, shows the query that was
                                performed for the selected name.
 
     Select a previous query, then choose one of the following commands:
 
     Command    Action
 
     Clear      Clears the history.
 
     OK         Closes the dialog box and restores the specified query
                in the previous dialog box.
 
     Cancel     Closes the History dialog box.
 
     Help       Displays this Help.

     Curdate («curdate»)
 
     Types the current date at the cursor in the format day-month-year,
     for example: 17-Apr-1999.
 
     Returns
 
     True:  Date typed.
     False: Typing the date would make the line too long.
 
     See
 
     Curdate, Curday, Curfile, Curfilenam, Curfileext, Curtime

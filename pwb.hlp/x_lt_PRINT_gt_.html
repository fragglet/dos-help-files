     "<PRINT>Print Results"
 
     The <PRINT> pseudofile contains redirected output from the current
     session's print operations.
 
     You can open this file by opening the Window menu and choosing
     Print Results from the PWB Windows cascaded menu. You can also
     choose Print Results in the All Files dialog box.

     File menu, Merge command
 
     Choose Merge to insert a file or a series of files into the
     current file before the line at the cursor.
 
     To insert a single file:
 
       ■  Type the name of the file in the File Name box and choose OK.
 
     To insert a series of files that match a single wildcard:
 
       ■  Type the wildcard in the File Name box and choose OK.
 
          PWB inserts all files that match the specified wildcard.
 
     To insert a series of files:
 
       ■  Choose the Files button.
 
          PWB brings up the File Merge dialog box, where you can build
          a list of files and wildcards to merge into the current file.

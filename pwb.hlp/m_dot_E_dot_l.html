     Edit menu, Select To Anchor command («selcur»)
 
     Function  ◄Selcur►
 
     Choose Select To Anchor to create a selection that begins at the
     anchor you previously set and ends at the cursor.
 
     The selection is a stream, box, or line selection according to the
     current selection mode. To change the type of selection, use the
     Stream Mode, Box Mode, or Line Mode commands on the Edit menu, or
     the Selmode function («selmode»).
     See: Selmode, ◄Edit menu, Stream Mode► ◄Edit menu, Box Mode►
          ◄Edit menu, Line Mode► ◄Edit menu, Set Anchor►
          ◄Selection Modes (Defined)►
 
     Select To Anchor is unavailable when no anchor is set.

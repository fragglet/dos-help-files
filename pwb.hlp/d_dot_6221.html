     The Additional Global LINK Options dialog box lets you set other
     options that are in effect for all builds. Choices in the dialog
     box are equivalent to the LINK utility options.
     See: ◄LINK Contents► (in LINK Help)
 
     No Ignore Case
          Preserves case sensitivity for identifiers.
          See: ◄LINK /NOI Option►
 
     No Extended Dictionary in Library
          Prevents LINK from using extended dictionaries in libraries.
          See: ◄LINK /NOE Option►
 
     Additional Global Options
          Specifies other options for all builds.
          See: ◄LINK Options►

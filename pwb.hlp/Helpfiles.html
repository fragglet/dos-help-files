     Helpfiles (Text)
 
     The Helpfiles switch lists Help files or directories containing
     Help files that PWB should open in addition to the Help files
     listed in the HELPFILES environment variable.
 
     This switch must appear in the [PWB-PWBHELP] tagged section of
     TOOLS.INI.
 
     Syntax
 
     Helpfiles: [<file>][;<file>]...
 
     <file>    The filename of a Help file to open or the name of a
               directory.
 
     If a directory name is used, all Help files in the directory are
     opened. Each <file> can contain wildcards or environment-variable
     specifiers.
     See: ◄Environment-Variable Specifiers►
 
     Default
 
     Helpfiles:
 
     By default, PWB uses only the Help files in the current directory
     and those listed in the HELPFILES environment variable.
 
     TIP: You can open and close Help files using Sethelp («sethelp»).
          See: Sethelp

     File menu, Close command («_pwbclosefile»)
 
     Macro  ◄_pwbclosefile►
 
     Choose Close to close the file in the active window. If no more
     files remain in the window's file history, the window is also
     closed. The file is removed from all other windows that have the
     file in their file history.
 
     PWB always keeps certain files in memory. These include TOOLS.INI,
     CURRENT.STS, and PWB pseudofiles such as Clipboard, Build Results,
     and Search Results. These files always appear in the All Files
     dialog box. When you choose the Close command for one of these
     files, PWB closes only the views of the file. When you reopen a
     window for the file, the contents are the same.

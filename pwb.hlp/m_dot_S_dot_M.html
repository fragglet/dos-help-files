     Search menu, Set Mark File command
 
     Choose Set Mark File to open or create a mark file.
 
     After you open a mark file, marks that you define are added to
     that file. When you clear a mark that is listed in the file, PWB
     deletes the mark's definition from the file.
 
     When you open a mark file and no mark file is currently open, all
     marks in memory are saved in the new mark file. Otherwise, the
     current marks are saved in the old mark file.

     Remove Custom Project Templates
 
     The Remove Custom Project Template dialog box provides the
     following options to delete custom project templates:
 
     Runtime Support
          Lists the installed languages and 'None'.
 
          Select the run-time support associated with the template you
          want to remove.
 
     Custom Project Templates with Runtime Support for: <language>
          Lists the custom templates for the selected <language>
          run-time support. If you have not defined a custom template
          for the selected language, the list is empty.
 
          Select the name of the template you want to delete.
 
     Choose Remove Project Template to delete the selected template.
     When you choose OK, PWB modifies your TOOLS.INI file by removing
     the tagged section that define the template. The deletions are not
     permanent until you save TOOLS.INI.

     Options menu, Build Options command
 
     Choose Build Options to determine the set of options (debug or
     release) that are to be used to build the project and to set the
     directory where the project is to be built.
 
     Option                 Description
 
     Use Debug Options      Specifies that the debug options are to be
                            used when the project is built.
 
     Use Release Options    Specifies that the release options are to
                            be used when the project is built.
 
     Build Directory        Specifies the target directory where the
                            project is to be built. This directory
                            contains the object files and executables
                            produced when you build the project.
                            Default: project directory
 
     To set specific debug and release options for each language and
     utility, use the language and utility commands in the Options
     menu. The active set of options is determined by the debug/release
     setting you make in the Build Options dialog box.

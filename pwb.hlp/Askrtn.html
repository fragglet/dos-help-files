     Askrtn (Boolean)
 
     The Askrtn switch determines if PWB prompts before returning to
     PWB after running a shell or external command.
 
     Syntax
 
     Askrtn:{ yes | no }
 
     yes    Prompt for confirmation before returning to PWB.
 
            This setting allows you to review the contents of the
            operating-system screen before returning to the editor.
 
     no     Do not prompt before returning to PWB.
 
     Default
 
     Askrtn:yes

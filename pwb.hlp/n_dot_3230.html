PWB Error
 
     Print failed on file/pattern
 
     PWB could not print the specified file or wildcard.
 
     One of the following may have occurred:
 
       ■  The file could not be found.
       ■  No files matching the wildcard were found.
       ■  PWB could not create a required temporary file.
       ■  A printing error occurred.
       ■  The printer was offline or not connected.
 
     Choose OK to continue printing the rest of the list of files and
     wildcards (if any).
 
     Choose Cancel to stop printing the list.

Viewing Global Search Results

─────Programmer's WorkBench─────────────────────────────────────────────────
 
     Choose the Stop Search button in the Searching dialog box to view
     the results of a global Help search.
 
     The result of the search is a Help list of all matches.
 
     To view a Help topic in the list using the mouse:
 
       ■  Double-click the line that lists the match.
 
     To view a Help topic in the list using the keyboard:
 
       1. Move the cursor to the line that lists the topic you want.
       2. Press ENTER or «_pwbhelp_context».
 
     To return to the list of matches:
 
       ■  Choose the Search Results command from the Help menu.
       ■  Execute the _pwbhelp_searchres macro («_pwbhelp_searchres»).
       ■  Choose the Back button on the Help screen.
       ■  Repeatedly press «_pwbhelp_back».
 
     To resume an interrupted search:
 
       1. Choose the last topic in the list.
       2. Start the search again.

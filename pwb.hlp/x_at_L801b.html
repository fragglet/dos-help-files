     Microsoft extensions include:
 
     Extension    Description
 
     PWBBASIC     Microsoft Basic
     PWBC         Microsoft C and C++
     PWBCOBOL     Microsoft COBOL
     PWBFORT      Microsoft FORTRAN
     PWBHELP      Microsoft Advisor Help system
     PWBMASM      Microsoft Macro Assembler
     PWBPROF      Microsoft Source Profiler
     PWBROWSE     Microsoft Source Browser
     PWBUTILS     Microsoft Utilities: LINK, NMAKE, CodeView

     Update: PWB Command Line
 
     With PWB 1.x, the PWB command line is case sensitive. With PWB
     2.00, all command-line options are case insensitive.
 
     The following PWB 2.00 option is not present in PWB 1.x:
 
     Option    Description
 
     /PN       Set no project. Overrides 'Lastproject:yes'.

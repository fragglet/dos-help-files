     Search menu, Define Mark command
 
     Choose Define Mark to set a mark. A mark is a saved location (also
     called "bookmark") that you can refer to by name. Mark names must
     be unique.
 
     Option              Description
 
     Mark Name           Specifies the name of the mark to define.
 
     File Name           Specifies the file to set the mark in.
                         Default: current file
 
     Row                 Specifies the row (line number) for the mark.
                         Default: cursor row
 
     Column              Specifies the column number for the mark.
                         Default: cursor column
 
     Add To Mark File    When turned on, adds the mark to the current
                         mark file.
 
                         When turned off, defines a temporary mark that
                         is known only for the current PWB session.
 
     See

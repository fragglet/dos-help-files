─────Programmer's WorkBench─────────────────────────────────────────────────
 
     The Search menu provides the following commands to find text or
     regular expressions in one or more files, browse logged search
     results, and define or jump to marks:
 
     Command           Description                          Key
 
     ◄Find►            Find text or regular expression      ───
     ◄Replace►         Replace text or regular expression   ───
 
     ◄Log►             Toggle search-result logging         «_pwblogsearch»
     ◄Next Match►      Go to next match                     «_pwbnextmatch»
                       Go to next match in log              «_pwbnextlogmatch»
     ◄Previous Match►  Go to previous match                 «_pwbpreviousmatch»
                       Go to previous match in log          «_pwbpreviouslogmatch»
     ◄Goto Match►      Go to match at cursor                «_pwbgotomatch»
 
     ◄Cancel Search►   Stop background search               «_pwbcancelsearch»
 
     ◄Goto Mark►       Go to mark or line                   ───
     ◄Define Mark►     Set a mark at the cursor             ───
     ◄Set Mark File►   Open a mark file                     ───

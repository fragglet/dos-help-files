     Search menu, Log command («_pwblogsearch»)
 
     Macro  ◄_pwblogsearch►
 
     Choose Log to toggle search logging on and off. When PWB starts
     up, the logging mode is determined by how the Enterlogmode switch
     is set.
     See: Enterlogmode
 
     When search logging is on:
 
       ■  The results of the Find command are logged in the Search Results
          window.
          See: ◄Search menu, Find► ◄Search Results►
 
       ■  PWB displays a bullet next to the Log command as shown:
 
          ┌──────────────────────────────────┐
          │•Log                              │
          │ Next Match    «_pwbnextlogmatch »│
          . . .
 
       ■  Choosing Next Match executes the _pwbnextlogmatch macro
          («_pwbnextlogmatch»).
 
       ■  Choosing Previous Match executes the _pwbpreviouslogmatch
          macro («_pwbpreviouslogmatch»).
 
     When search logging is off:
 
       ■  No bullet appears next to the Log command.
 
       ■  Choosing Next Match executes the _pwbnextmatch macro
          («_pwbnextmatch»).
 
       ■  Choosing Previous Match executes the _pwbpreviousmatch macro
          («_pwbpreviousmatch»).
 
     See
 
     Logsearch, _pwbnextmatch, _pwbnextlogmatch, _pwbpreviousmatch,
     _pwbpreviouslogmatch

     Edit menu, Copy command («Copy»)
 
     Function  ◄Copy►
 
     Choose Copy to copy selected text from the active window to the
     clipboard. When no text is selected, this command copies the
     current line to the clipboard.
 
     To insert the copied text, use the Paste command or the Paste
     function («paste»).
     See: Paste, ◄Edit menu, Paste►
 
     TIP: To view the contents of the clipboard, choose Clipboard from
          the PWB Windows cascaded menu on the Window menu.

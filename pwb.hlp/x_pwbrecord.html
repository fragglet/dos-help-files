     _pwbrecord («_pwbrecord»)
 
     Menu  ◄Edit menu, Record On►
 
     The _pwbrecord macro toggles macro recording on and off. If you
     have not set the recorded macro name and key, PWB displays the
     Set Record dialog box so you can set them. Execute _pwbrecord
     again to start recording.

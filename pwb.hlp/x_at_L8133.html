     _pwblinemode := :>more selmode ->more selmode selmode
 
     :>more      Defines the label 'more'.
 
     Selmode     Advances to the next selection mode.
 
     ->more      Branches to the label 'more' if Selmode returns false.
 
     The Selmode function advances the selection mode from box, to
     stream, to line. Selmode returns true when the mode is stream
     mode. The macro executes Selmode until it returns true (sets
     stream mode), then advances the selection mode twice to set line
     selection mode.

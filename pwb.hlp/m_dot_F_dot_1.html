     File menu, <n> <file> command
 
     Choose a file named in the File menu to switch to that file in the
     active window. The File menu shows up to the first nine files in
     the window's file history.
 
     To see a list of all files open in PWB (not just the active
     window's files), choose All Files from the File menu.
     See: ◄File menu, All Files►
 
     The nine files are opened by the following predefined macros:
 
     Macro        Key                 Macro        Key
     _pwbfile1    «_pwbfile1        » _pwbfile6    «_pwbfile6»
     _pwbfile2    «_pwbfile2        » _pwbfile7    «_pwbfile7»
     _pwbfile3    «_pwbfile3        » _pwbfile8    «_pwbfile8»
     _pwbfile4    «_pwbfile4        » _pwbfile9    «_pwbfile9»
     _pwbfile5    «_pwbfile5»
 
     See: ◄Predefined PWB Macros _pwbfile<n>►
 
     TIP: When you move the selection cursor to a filename in the File
          menu, PWB displays the full path on the status bar.

     Browse menu, Match Case command
 
     Choose Match Case to toggle the case sensitivity of browser
     queries. A bullet appears next to the Match Case command when the
     browser matches the exact letter case of names in the browser
     database.
 
     The default state of the browser when you open a browser database
     is determined by the Browcase switch in TOOLS.INI.
     See: Browcase

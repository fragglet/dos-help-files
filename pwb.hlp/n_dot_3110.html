PWB Error
 
     Line <arg> too long; replacement skipped. Continue?
 
     A replacement on the given line would make the line exceed 250
     characters. PWB is asking for confirmation to continue the find-
     and-replace operation.
 
     Response    Action
 
     Y (yes)     Continue the search-and-replace operation.
 
     N (no)      Stop the operation at the given line.
 
                 To reverse the replacements made up to the point of
                 the error and restore the state of the file to its
                 state before you started the replacement, use the
                 Undo command on the Edit menu or the Undo function
                 («undo»).

     The following PWB 2.00 functions do not exist in PWB 1.x:
 
     Function           Description
 
     ◄Arrangewindow►    Arrange windows or icons
     ◄Cancelsearch►     Cancel background search
     ◄Clearmsg►         Clear Build Results or set of messages
     ◄Clearsearch►      Clear Search Results or set of messages
     ◄Closefile►        Close current file
     ◄Logsearch►        Toggle search logging
     ◄Movewindow►       Move window
     ◄Newfile►          Create a new pseudofile
     ◄Nextsearch►       Go to search match location
     ◄Openfile►         Open a new file
     ◄Project►          Set or clear project
     ◄Pwbwindow►        Open a PWB window
     ◄Selwindow►        Move to window

PWB Message
 
     File exists. Rewrite?
 
     A request was made to save a file that already exists on disk.
 
     PWB is asking for confirmation to overwrite the file on disk with
     the contents of the file in PWB.
 
     Response    Action
 
     Y (yes)     Overwrite the existing file.
     N (no)      Do not overwrite the existing file. PWB gives you the
                 opportunity to save the file with a different name.
     «cancel    »Do not save the file, and return to normal operation.

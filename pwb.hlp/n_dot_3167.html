PWB Error
 
     Can't Set Project: Unable to find project makefile
 
     PWB cannot load a project specified on the command line.
 
     If you specify a project on the command line, PWB gives this error
     when it cannot find the project makefile, or the filename is not
     valid.
 
     If you specify "/PL" on the command line or the Lastproject
     switch is set to Yes, PWB gives this error when there is no
     previous project.

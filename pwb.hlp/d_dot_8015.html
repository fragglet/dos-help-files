     Use the Select Browser Database dialog box to open a custom browser
     database. A custom database is especially useful when you build the
     browser database separately from your project or when you build
     special-purpose browser databases.
 
     To specify a custom browser database, type the name of the database
     in the File Name box or select the database using the File List and
     Drives/Dirs list boxes, then choose OK.
 
     To save the name of the database in TOOLS.INI for later sessions,
     choose the Save Current Database button in the Custom Database
     Management dialog box.
     See: ◄Custom Database Management►

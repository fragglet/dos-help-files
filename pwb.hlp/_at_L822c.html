─────Programmer's WorkBench─────────────────────────────────────────────────
 
     For each type of object in the output from the List References
     command, the browser creates a two-column list. The left column
     names an object. The right column lists the references to that
     object.
 
     References can have the following annotations:
 
     Annotation    Meaning
 
     (<parent>)    The defining object is listed in parentheses.
     [<number>]    The object is referenced <number> times.

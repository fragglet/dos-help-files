     _pwbsetmsg («_pwbsetmsg»)
 
     Menu  ◄Project menu, Goto Error►
 
     The _pwbsetmsg macro sets the message listed at the current
     location in the Build Results pseudofile as the current message
     and moves the cursor to the location specified by that message.

     _pwbnextmatch («_pwbnextmatch»)
 
     Menu  ◄Search menu, Next Match►
 
     The _pwbnextmatch macro searches forward in the file using the
     last search pattern and options. The search options are Case
     Sensitive, Wrap Around, and Regular Expression.
 
     The Next Match command executes this macro when search logging is
     off. When search logging is on, Next Match executes the
     _pwbnextlogmatch macro.
     See: _pwbnextlogmatch

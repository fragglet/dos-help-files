     Lasttext («lasttext»)
          Displays the last text argument in the Text Argument dialog
          box. You can edit the text and then execute any PWB function
          that accepts a text argument, or you can cancel the dialog box.
          See: Textarg
 
          If you edit the text and then cancel the dialog box, PWB
          retains the modified text. Thus, when you execute Lasttext
          again, the new text appears in the dialog box.
 
     Arg [Arg]... [Meta] Lasttext
     («arg» [«arg»]... [«meta»] «lasttext»)
          Displays the last text argument in the Text Argument dialog
          box with the specified Arg count and Meta state.
 
     Arg [Arg]... <linearg> │ <boxarg> │ <streamarg> [Meta] Lasttext
     («arg» [«arg»]... <linearg> │ <boxarg> │ <streamarg> [«meta»] «lasttext»)
          Displays the first line of the selection in the Text Argument
          dialog box with the specified Arg count and Meta state.
 
     Returns
 
     The return value of Lasttext cannot be tested.
 
     See
 
     Arg, Lastselect, Meta, Prompt

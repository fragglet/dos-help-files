     Nextmsg («nextmsg»)
          Advances to next message in the Build Results window.
 
     Arg <number> Nextmsg («arg» <number> «nextmsg»)
          Moves to the nth message in the current set of messages,
          where n is specified by <number>.
 
          To move relative to the current message, use a signed number.
          For example, when <number> is +1, PWB moves to the next
          message, and when it is -1, PWB moves to the previous
          message.
 
     Arg Nextmsg («arg» «nextmsg»)
          Moves to the next message in the current set of messages that
          does not refer to the current file.
 
     Meta Nextmsg («meta» «nextmsg»)
          Advances to the next set of messages.
 
     Arg Arg Nextmsg («arg» «arg» «nextmsg»)
          Sets the message at the cursor as the current message. This
          works only when the cursor is on a message in the Build
          Results window.
 
     Returns
 
     True:  Message found.
     False: No more messages found.

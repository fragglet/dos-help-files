     Window menu, Size command («_pwbresize»)
 
     Macro  ◄_pwbresize►
 
     Choose Size to start window-sizing mode for the active window. In
     window-sizing mode, you can do only the following:
 
     Action                         Key
 
     Shrink window one row          UP
     Expand window one row          DOWN
     Shrink window one column       LEFT
     Expand window one column       RIGHT
     Accept the new size            ENTER
     Cancel the sizing operation    ESC
 
     To size a window with the mouse:
 
       ■  Drag the lower-right corner of the window.

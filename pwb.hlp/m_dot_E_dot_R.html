     Edit menu, Repeat command («_pwbrepeat»)
 
     Macro  ◄_pwbrepeat►
 
     Choose Repeat to repeat the last edit or macro.
 
     To repeat the edit a specific number of times, pass a numeric
     argument to the Repeat function («repeat»).

PWB Message
 
     Open Project
 
     PWB found a .MAK file (project makefile) with the same base name
     as the source file that was just opened, and the Deflang switch is
     set to the language corresponding to the source file's extension.
 
     Response    Action
 
     Y (yes)     Open the project. Sets the makefile as the current
                 project makefile.
     N (no)      Do not open the project.

     File menu, DOS Shell command («_pwbshell»)
 
     Macro  ◄_pwbshell►
 
     Choose DOS Shell to leave PWB temporarily so that you can use
     operating-system commands.
 
     To use the operating-system command processor:
 
       1. Choose DOS Shell.
 
       2. Perform any operating-system commands.
 
          WARNING: Do not start TSR programs in a shell because
                   unpredictable results occur.
 
     To return to PWB:
 
       ■  Type 'Exit' at the operating-system prompt.
 
     To execute an operating-system command and return to PWB in one
     step, choose Run DOS Command from the Run menu.
 
     TIP: You can add frequently used operating-system commands and
          programs to the Run menu.
          See: ◄Run menu, Customize Run Menu►

Pop-Up Help Window

─────Programmer's WorkBench─────────────────────────────────────────────────
 
     PWB displays Help in a pop-up window whenever you request Help on a
     menu or dialog box. You can use all of the usual Help naviagation
     methods to view related Help information.
 
     When you want to go back to the menu or dialog box for which you
     requested Help, choose the Cancel button or execute the Cancel
     function («cancel»).
 
     To move the topic displayed in the pop-up Help window to the
     regular Help window, use the following steps:
 
       1. Cancel the pop-up Help window.
 
       2. Cancel all the nested dialog boxes below it until you return
          to the editor.
 
       3. Execute the _pwbhelp_back macro («_pwbhelp_back»).
 
     PWB opens the Help window and displays the Help that you were
     viewing in the pop-up Help window. This has the disadvantage of
     canceling the operation you were performing with the menu or
     dialog box.

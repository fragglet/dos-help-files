     Browdbase (Text)
 
     The Browdbase switch specifies the browser database to use. When
     this switch is not set or the setting is empty, the browser uses
     the database for the open project (if any).
 
     Set this switch by using the Save Current Database command in the
     Open Custom dialog box.
     See: ◄Browse menu, Open Custom►
 
     Syntax
 
     Browdbase: <database>
 
     <database>    The full path of a browser database (.BSC file).
 
     When <database> is not specified, the browser uses the database
     for the open project.
 
     NOTE: This switch must appear in the [PWB-PWBROWSE] tagged section
           of TOOLS.INI.

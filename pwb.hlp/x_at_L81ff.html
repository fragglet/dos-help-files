─────Programmer's WorkBench─────────────────────────────────────────────────
 
     Turn on the Execute Commands option to specify Command-window
     commands that CodeView is to execute on startup.
 
     Separate multiple commands with a semicolon. If any commands
     contain redirection symbols (<>) or spaces, you must enclose the
     entire list of options in double quotation marks.
 
     See: ◄Execute Commands (/C) Option►

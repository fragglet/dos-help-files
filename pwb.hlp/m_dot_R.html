     The Run menu provides commands to execute, debug, and set
     command-line arguments for your project's program. You can also
     run operating system commands and external programs and add
     programs to the Run menu.
 
     Command               Description                     Key
 
     ◄Execute►             Run project program             ───
     ◄Program Arguments►   Set program's command line      ───
     ◄Debug►               Debug project program           ───
 
     ◄Run DOS Command►     Run DOS command or program      ───
     ◄Customize Run Menu►  Add commands to the Run menu    ───
 
     ◄<custom_command_1>►  Run custom command 1            «_pwbuser1»
     . . .
     ◄<custom_command_9>►  Run custom command 9            «_pwbuser9»

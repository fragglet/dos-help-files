PWB Error
 
     Word switch text must be quoted
 
     The value of the Word switch must be enclosed in double quotation
     marks.
 
     Note that because the value is a PWB macro string, all backslashes
     in the regular expression must be doubled.
 
     See: Word, ◄Macro String Syntax►

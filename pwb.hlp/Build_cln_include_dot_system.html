Build:include system Rule
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Programmer's WorkBench─────────────────────────────────────────────────
 
     The Build:include system rule is ignored when specified in a
     template. PWB uses it to hold the current value of the Ignore
     System Include Files check box in the Edit Project dialog box.
 
     Syntax
 
     Build:include [no] system
 
     no        Do not include system include files when generating
               include-file dependencies.
 
               If No is not present, PWB generates dependencies for
               include files that match the system-tagged pattern for
               include files.
 
     system    Tag to indicate that this rule applies to the processing
               of system include file dependencies.

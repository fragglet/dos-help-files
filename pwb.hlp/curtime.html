     Curtime («curtime»)
 
     Types the current time in the format hours:minutes:seconds, for
     example, 17:08:32.
 
     Returns
 
     True:  Time typed.
     False: Typing the time would make the line too long.
 
     See
 
     Curdate, Curday, Curfile, Curfilenam, Curfileext, Curtime

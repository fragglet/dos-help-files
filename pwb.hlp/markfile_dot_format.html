─────Programmer's WorkBench─────────────────────────────────────────────────
 
     A mark file is a text file containing mark definitions of the
     form:
 
     markname filename line column
 
     The mark <markname> is defined as the location given by <line> and
     <column> in the file <filename>. The <markname> cannot contain
     spaces and cannot be a number.

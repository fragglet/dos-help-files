     Use the Browse Exclude Files List dialog box to remove selected
     files from the browser database. You can exclude individual files
     and files that match a pattern. This option is equivalent to the
     BSCMAKE /Ei option.
 
     To add a file to the Exclusion List:
 
       1. Type the name in the File Name box
       OR
          Select the name in the File List box
       2. Choose Add/Delete
 
     To filter the File List:
 
       1. Type a pattern in the File Name box
       2. Press ENTER
       OR
          Choose Add/Delete
 
     To add a file pattern to the Exclusion List:
 
       1. Type the pattern in the File Name box
       2. Choose Add Pattern
 
     To remove a pattern from the Exclusion List:
 
       1. Select the pattern
       2. Choose Add/Delete
 
     When you choose OK, PWB passes the files and patterns in the
     Exclusion List box to the browser and BSCMAKE. These files are
     excluded from the browser database the next time you build it.
     See: BSCMAKE

     To use the View Relationship dialog box:
 
       1. Specify the name on which to operate:
 
            ■  Type the name in the Name box.
            OR
            ■  Select the name from the List box.
 
       2. Select options under Queries.
          See: ◄View Relationship Queries►
 
       3. To filter the results:
 
            ■  Type a regular expression in the Pattern box.
            OR
            ■  Turn off the appropriate Show options.
 
          Only the matching results are displayed.
 
       4. To perform the query, choose OK or double-click the Queries
          option.
 
          OK does not close the dialog box as in other PWB dialog boxes.
 
       5. Repeat steps 1 to 4 to select another query.
 
       6. Choose Keep Info to close the dialog box and show the results
          in the Browser Output window.
       OR
          Choose Cancel to close the dialog box.

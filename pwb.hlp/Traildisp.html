     Traildisp (Numeric)
 
     The Traildisp switch specifies the decimal ASCII code for the
     character used to display trailing spaces on a line. If you
     specify 0 or 255, PWB uses the space (ASCII 32) character.
     See: ◄ASCII Table►
 
     Default
 
     Traildisp:0

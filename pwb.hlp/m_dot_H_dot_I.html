     Help menu, Index command («_pwbhelp_index»)
 
     Macro  ◄_pwbhelp_index►
 
     The Index command opens the Help window and displays the top-level
     indexes in the Microsoft Advisor Help system.
 
     Within the Microsoft Advisor, most Help topics have an Index button
     at the top of the window. This button also takes you to the
     top-level indexes.

     Cannot open [<file>]: Invalid help file version
 
     The PWBHELP extension attempted to load the given Help file, but
     it is not compatible with this version of the PWBHELP extension.
 
     Remove the specified file from the directory or list of files given
     in the HELPFILES environment variable or the Helpfiles switch.

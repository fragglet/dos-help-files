     The following macro is useful when writing new macros (the '!' is
     the macro name):
 
          ! := Meta Message
 
     With this definition you can place an exclamation point in your
     macros wherever you want a screen update. If you also want to
     display a status-bar message at the time of the update, use the
     phrase:
 
          ... arg "<text of message>" ! ...

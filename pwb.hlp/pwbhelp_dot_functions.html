     The PWBHELP extension defines the following functions for opening
     and closing Help files, listing open Help files, looking up Help
     topics, navigating Help relative to the current topic, and
     searching for arbitrary text in the Help system.
 
     Function           Description
 
     ◄Pwbhelp►          Performs Help topic lookup
     ◄Pwbhelperr►       Shows Help on the current message
     ◄Pwbhelpnext►      Performs relative Help topic lookup
     ◄Pwbhelpsearch►    Performs global full-text Help searches
     ◄Sethelp►          Opens, closes, and lists Help files

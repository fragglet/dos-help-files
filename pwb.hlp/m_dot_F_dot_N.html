     File menu, New command («_pwbnewfile»)
 
     Macro  ◄_pwbnewfile►
 
     Choose New to create a new, untitled file.
 
     If the Newwindow switch is set to yes, PWB opens a new window for
     the file. Otherwise, PWB opens the file in the active window and
     places the current file in the window's file history.
     See: Newwindow

     Color (Text)
 
     The PWBHELP Color switch specifies the colors for various parts of
     the Help window.
 
     The settings for Help colors must appear in the [PWB-PWBHELP]
     tagged section of TOOLS.INI.
 
     Syntax
 
     Color: name value
 
     <name>    Identifies the part of the Help window affected by the
               color value. It can be one of the names listed below.
 
     <value>   Two hexadecimal digits specifying the color of the
               indicated item.
               See: ◄Color Values►
 
     Default
 
     Name             Default    Description
 
     Helpnorm         87         Plain Help text
     Helpbold         8f         Bold Help text
     Helpitalic       8a         Italic Help text and the ◄► characters
     Helpunderline    8c         Emphasized Help text
     Helpwarning      70         Current hyperlink

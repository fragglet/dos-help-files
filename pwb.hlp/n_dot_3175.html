PWB Error - Programmer's WorkBench
 
     Arg type invalid for '<function>' with minimized window
 
     A request was made to execute the indicated function, but the
     function cannot operate with the specified argument when the
     active window is minimized (that is, when the window is an icon).
 
     The requested form of the command has no meaning when the active
     window is minimized.

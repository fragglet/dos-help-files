PWB Message
 
     Empty replacement string, confirm:
 
     A request was made to replace a search pattern with an empty
     replacement string (nothing). PWB is prompting for confirmation.
 
     Response    Action
 
     Y (yes)     Perform the replacement, deleting occurrences of the
                 search pattern.
     N (no)      Do not perform the replacement.

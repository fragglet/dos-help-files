     Backtab («backtab»)
 
     Moves the cursor to the previous tab stop on the line.
 
     Returns
 
     True:  Cursor moved.
     False: Cursor is at left margin.
 
     Update
 
     PWB 2.0 supports variable tab stops. PWB 1.x supports only
     fixed-width tab stops.
 
     See
 
     Tab, Tabstops

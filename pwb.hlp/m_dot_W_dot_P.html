     Window menu, PWB Windows cascaded menu
 
     The PWB Windows cascaded menu displays a list of available PWB
     windows.
 
     With all default Microsoft extensions loaded, you can choose from
     the following windows:
 
     Window              Contents
 
     ◄Build Results►     Results of build operations
     ◄Search Results►    Results of logged searches
     ◄Print Results►     Results of print operations
     ◄Record►            Recorded macro
     ◄Clipboard►         PWB clipboard
     ◄Help►              Help
     ◄Browser Output►    Results of Source Browser queries

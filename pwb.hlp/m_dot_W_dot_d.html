     Window menu, Cascade command («_pwbcascade»)
 
     Macro  ◄_pwbcascade►
 
     Choose Cascade to arrange all unminimized windows so that they
     overlap and each title is visible. Up to 16 windows can be
     cascaded.
 
     Cascaded windows appear much like this:
 
          ┌─[  1]── C:\work\sample.txt ───────┐
          │┌─[  2]─ C:\work\sample.txt ───────┐
          ││┌─[  3]─ C:\work\sample.txt ──────┐
          │││┌─[  4]═ C:\work\sample.txt ══│↓│↑
          ││││                                │
          ││││                                │
          ││││                                │
          └└└└────────────────────────────────┘
 
     If a minimized window has the focus when you execute _pwbcascade,
     that window is restored.

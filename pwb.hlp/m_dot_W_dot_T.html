     Window menu, Tile command («_pwbtile»)
 
     Macro  ◄_pwbtile►
 
     Choose Tile to arrange all unminimized windows so that no windows
     overlap. Up to 16 windows can be tiled.
 
     When two or three windows are open, the top two are displayed side
     by side or all are stacked vertically according to how the
     Tilemode switch is set.
     See: Tilemode
 
     If a minimized window has the focus when you execute _pwbtile, that
     window is restored.

     Browser out of conventional memory
     Some functions may not be available
 
     The source browser could not obtain conventional memory below 640K
     to carry out a requested operation. Not all browser functions may
     be available, and some browser output may not be complete. However,
     you can continue using PWB and the source browser.
 
     To make more memory available within PWB, you can close some
     unneeded files or windows, close the open project, refresh some
     files, or quit and restart PWB.

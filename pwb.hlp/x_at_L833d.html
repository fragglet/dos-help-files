     The following list summarizes the new and changed PWB menus.
 
     ◄File►
 
       ◄Open►              'Pseudofile' and 'New Window' options
       ◄Find►              Was 'Search For File'
       ◄Cancel Print►      Cancels a background print
 
     ◄Edit►
 
       ◄Delete►            Was Edit Clear
       ◄Stream Mode►       ┐
       ◄Box Mode►          ├─ Were one dynamic Edit menu command
       ◄Line Mode►         ┘
 
     View                Obsolete: now the Window menu
 
     ◄Search►
 
       ◄Find►              Logged searches
       ◄Log►               Toggles logged searches
       ◄Next Match►        ┐
       ◄Previous Match►    ├─ New: browse logged Search Results
       ◄Goto Match►        ┘
       ◄Cancel Search►     Cancels a background search
 
     Make                Obsolete: now the Project menu
 
     ◄Project►           Was the Make menu
 
       ◄New Project►       Creates a new project
       ◄Open Project►      Was Set Program List, improved
       ◄Edit Project►      Was Edit Program List, improved
       ◄Close Project►     Was Clear Program List
       ◄Cancel Build►      Cancels a background build
 
     ◄Options►
 
       ◄Environment►       Controls the entire environment table
       ◄Key Assignments►   Dialog box for assigning keys
       ◄Editor Settings►   Dialog box for changing settings
       ◄Colors►            Dialog box for changing PWB colors
 
     ◄Window►            Was the View menu
 
     ◄Browse►
 
       ◄Function Hierarchy►   ┐
       ◄Module Outline►       │
       ◄Which Reference?►     ├─ New C++ and OOP support
       ◄Class Tree (Fwd/Rev)► │
       ◄Class Hierarchy►      ┘
 
     ◄Help►
 
       ◄About►             Shows PWB version and copyright
       ◄Global Search►     Full-text Help search with regular expressions
       ◄Search Results►    View results of full-text Help search

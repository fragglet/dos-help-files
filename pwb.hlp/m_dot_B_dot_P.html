     Browse menu, Previous command («pwbrowseprev»)
 
     Function  Pwbrowseprev
 
     Choose Previous to jump to the previous definition or reference in
     the list of definitions or references for the previously accessed
     name.
 
     To initiate a browse sequence, choose a browser Goto command. You
     can then choose the Next command or the Previous command from the
     Browse menu to view each definition or reference in turn.
 
     See
 
     Pwbrowse1stdef, Pwbrowse1stref, Pwbrowsegotodef, Pwbrowsegotoref,
     Pwbrowsenext

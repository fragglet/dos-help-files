     _pwbuser<n> := cancel arg "n" usercmd <
 
     Cancel             Establishes a uniform "ground state" by canceling
                        any selection or argument.
 
     Arg "n" Usercmd    Executes custom run menu item number <n>.
 
     <                  Restores the function's prompt (if any). By
                        default, function prompts are suppressed while
                        a macro is running.
 
     Example Definition
 
     _pwbuser1 := cancel arg "1" usercmd <
 
     This macro executes custom Run menu command number 1.

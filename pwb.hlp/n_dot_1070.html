PWB Message - Programmer's WorkBench
 
     <key> is unassigned
 
     The specified key that was pressed is not assigned to a function
     or macro.
 
     To assign a key to a function or macro, or view the current key
     assignments, use the Key Assignments command on the Options menu
     or the Assign function («assign»).
     See: Assign, ◄Options menu, Key Assignments►

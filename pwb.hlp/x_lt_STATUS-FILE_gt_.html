     "<STATUS-FILE>PWB Status File"
 
     Update
 
     The PWB 1.x <STATUS-FILE> pseudofile is obsolete and does not
     exist in PWB 2.00. In PWB 1.x, this pseudofile contains a copy of
     the last status file (CURRENT.STS or <project>.STS) that was read
     or written. In PWB 2.00, the CURRENT.STS file is kept in memory.

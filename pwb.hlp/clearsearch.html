     Clearsearch («clearsearch»)
          Clears the contents of the Search Results window.
 
     Arg Clearsearch («arg» «clearsearch»)
          Clears the current set of matches in the Search Results
          window.
 
     Returns
 
     True:  Cleared a match set or the contents of Search Results.
     False: The Search Results window is empty.
 
     See
 
     Clearmsg, Logsearch, _pwbnextlogmatch, _pwbpreviouslogmatch,
     _pwbgotomatch

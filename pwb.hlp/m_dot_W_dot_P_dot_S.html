     Window menu, PWB Windows, Search Results
 
     Choose Search Results to display the Search Results window. This
     window contains the results of logged searches.
 
     To jump to any match shown in the Search Results window:
 
       1. Move the cursor to the match.
 
       2. Use one of the following commands:
 
            ■  Choose Goto Match from the Search menu.
            ■  Execute the _pwbgotomatch macro («_pwbgotomatch»).
            ■  Execute the function sequence Arg Arg Nextsearch
               («arg» «arg» «nextsearch»).
     See
 
     Nextsearch, Logsearch, _pwbgotomatch

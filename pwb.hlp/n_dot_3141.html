Build macro expansion: syntax error
 
     A build command or build macro for the current project contained a
     syntax error.
 
     If you have specified additional compiler options, check the entry
     for correct syntax. If the entry contains a dollar sign ($), you
     must precede the dollar sign with a caret (^).
 
     This error also happens when you specify a dollar sign in the Run
     Command Line dialog box. To specify a literal dollar sign for the
     command line, precede it with a caret.

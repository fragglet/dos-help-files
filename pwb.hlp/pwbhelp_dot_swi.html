     Use the following the PWBHELP extension switches to customize the
     behavior of the Help extension and set the colors of the Help
     window.
 
     Boolean Switch           Description
 
     ◄Helpautosize►           Automatically size Help window to topic
     ◄Helpwindow►             Obsolete
     ◄Helplist►               Display list for multiple topics
 
     Text Switch              Description
 
     ◄Helpfiles►              List of additional Help files
     ◄Color: Helpnorm►        Plain Help text
     ◄Color: Helpbold►        Bold Help text
     ◄Color: Helpitalic►      Italic Help text and the ◄► characters
     ◄Color: Helpunderline►   Emphasized Help text
     ◄Color: Helpwarning►     Current hyperlink
 
     The settings for the Help switches must appear in the
     [PWB-PWBHELP] tagged section of TOOLS.INI.

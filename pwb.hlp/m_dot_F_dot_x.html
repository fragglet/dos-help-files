     File menu, Exit command («_pwbquit»)
 
     Macro  ◄_pwbquit►
 
     Choose Exit to leave PWB and return to the operating system or to
     the program that started PWB.
 
     PWB saves information about the session in the CURRENT.STS file so
     that you can return to PWB and resume working where you left off.
     See: ◄CURRENT.STS► ◄PWB Command Line►

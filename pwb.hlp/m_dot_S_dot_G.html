     Search menu, Goto Mark command
 
     Choose Goto Mark to jump to a line in the current file by line
     number or to jump to a named mark in any file where you have set a
     mark.
     See: Mark, ◄Search menu, Define Mark►
 
     The Go To Mark dialog box contains the following items:
 
     Item          Description
 
     Mark Name     Specifies the line number or the name of the mark.
 
     File          Displays the name of the current mark file if a mark
                   file is open. Otherwise, this field is empty. Choose
                   Set Mark File to open a mark file.
                   See: Markfile, ◄Search menu, Set Mark File►
 
     Mark List     Choose the mark to go to.
 
     TIP: To jump back to the previous location after going to a mark
          in the current file, use the function sequence Arg Mark
          («arg» «mark»).

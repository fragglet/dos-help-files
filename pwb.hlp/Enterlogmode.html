     Enterlogmode (Boolean)
 
     The Enterlogmode switch determines if search logging is on or off
     when PWB starts up.
 
     The current search-logging mode can be changed at any time using
     Log or the Logsearch function («logsearch»).
     See: Logsearch, ◄Search menu, Log►
 
     Syntax
 
     Enterlogmode:{ yes | no }
 
     yes    Start PWB with search logging on.
 
     no     Start PWB with search logging off.
 
     Default
 
     Enterlogmode:no

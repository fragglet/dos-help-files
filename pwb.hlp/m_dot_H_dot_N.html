     Help menu, Next command («_pwbhelp_again»)
 
     Macro  ◄_pwbhelp_again►
 
     The Next command displays the next occurrence of the last topic
     you requested Help on. If no other occurrences of the topic are
     defined in the open Help files, PWB redisplays the current topic.
 
     The topic that PWB looks up when you use this command is displayed
     in the Help menu as follows:
 
          Next: xxxxxxxxxxx  «_pwbhelp_again»
                     ▲
                     └─── Topic to look up.

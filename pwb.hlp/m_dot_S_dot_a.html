     Search menu, Cancel Search command («_pwbcancelsearch»)
 
     Macro  ◄_pwbcancelsearch►
 
     Choose Cancel Search to stop a background search.
 
     When you choose Cancel Search, the search stops and any partial
     results remain in the Search Results window. To browse the results,
     choose the Next Match, Previous Match, and Goto Match commands from
     the Search menu or execute the Nextsearch function.
     See: Nextsearch
          ◄Search menu, Next Match►
          ◄Search menu, Previous Match►
          ◄Search menu, Goto Match►
 
     This command is available only when running with a multithreaded
     environment and a background (logged) search is in progress.

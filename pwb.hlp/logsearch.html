     Logsearch («logsearch»)
 
     Toggles the search-logging state.
 
     The default search-logging mode when PWB starts up is determined
     by the Enterlogmode switch.
     See: Enterlogmode
 
     Returns
 
     True:  Search logging turned on.
     False: Search logging turned off.

     Beep (Boolean)
 
     The Beep switch determines PWB's alerting method. When set to
     Yes, PWB issues an audible sound. When No, PWB flashes the
     menu bar──a visual "beep".
 
     Syntax
 
     Beep:{ yes | no }
 
     yes    Generate an audible beep.
 
     no     Flash the menu bar.
 
     Default
 
     Beep:yes

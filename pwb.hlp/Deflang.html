     Deflang (Text)
 
     The Deflang switch determines the default file extension for file
     lists in PWB dialog boxes.
 
     Syntax:
 
     Deflang:<language>
 
     <language>    One of the following settings:
 
                   Setting    Extension
 
                   NONE       .*
                   Asm        .asm
                   Basic      .bas
                   C          .c
                   C++        .cpp
                   CXX        .cxx
                   COBOL      .cbl
                   FORTRAN    .for
                   LISP       .lsp
                   Pascal     .pas
 
     Default
 
     Deflang:NONE

     Wherever you use a filename or path in PWB, you can use the
     following syntax as part of the name.
 
     Syntax
 
     $ENV: │ $(ENV)
 
     <ENV>    The uppercase name of any environment variable
 
     Example
 
     To change PWB's current directory to the directory specified by
     the INCLUDE environment variable:
 
       1. Execute Arg («arg»).
 
       2. Type the environment variable specifier:
 
               $INCLUDE:
 
       3. Execute Setfile («setfile»).

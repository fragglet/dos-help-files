     _pwbpreviouslogmatch («_pwbpreviouslogmatch»)
 
     Menu  ◄Search menu, Previous Match►
 
     The _pwbpreviouslogmatch macro moves the cursor to the previous
     match listed in the Search Results pseudofile.
 
     The Previous Match command executes this macro when search logging
     is on. When search logging is off, Search Previous Match executes
     the _pwbpreviousmatch macro.
     See: _pwbpreviousmatch

     The Source Browser displays the File Not Found dialog box when it
     cannot find a source file in the directory specified for that file
     in the browser database.
 
     Type the new directory for the source file.
 
     The browser looks in the directory you specify for subsequent
     browser queries before displaying this dialog box. This means that
     you must specify a new directory only once for each changed
     directory in the project.

     The Pwbrowse1stref function is similar to the Pwbrowsegotoref
     function but does not bring up a dialog box. If you specify an
     ambiguous name, the browser displays the Ambiguous Name dialog
     box.
     See: Pwbrowsegotoref, ◄Ambiguous Name►
 
     Pwbrowse1stref («pwbrowse1stref»)
          Goes to the first place where the name at the cursor is
          referenced.
 
     Arg textarg │ streamarg Pwbrowse1stref
     («arg» textarg │ streamarg «pwbrowse1stref»)
          Goes to the first place where the specified name is
          referenced.
 
     See
 
     Pwbrowsenext, Pwbrowsepop, Pwbrowseprev

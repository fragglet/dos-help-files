PWB Error
 
     Warning: Cannot update TOOLS.INI if /D[T] specified
 
     A request was made to modify TOOLS.INI, and the PWB /D or /DT
     command-line option was specified. Changes to TOOLS.INI will
     not be saved.
 
     TOOLS.INI is modified when you save changes in the Key Assignments
     dialog box, the Editor Settings dialog box, the Run Customize
     dialog box, or the Record window.

     Out of conventional memory below 640K for topic in
            <file>
     Skip topic and continue?
 
     During a global search of Help, the PWBHELP extension could not
     allocate enough memory for a large topic in the specified Help
     file.
 
     Choose Yes to skip the failed topic and continue searching Help.
 
     Choose No to stop searching Help and examine the partial list of
     topics that contain the search string.
 
     If the list you have found does not contain the Help you need, try
     a different string or pattern or a more specific one.
 
     To continue searching Help for the same pattern, view Help on the
     last match listed in the global search results then restart the
     global search.
 
     You can also close the Help files you have seen (using the Sethelp
     function) and repeat the search.

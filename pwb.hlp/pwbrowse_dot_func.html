     The PWBROWSE extension provides the following functions for
     browsing your program. Most functions are tied to a Browse menu
     command.
 
     Function             Browse Menu Command    Key
 
     ◄Pwbrowsenext►       Next                   «Pwbrowsenext»
     ◄Pwbrowseprev►       Previous               «Pwbrowseprev»
     ◄Pwbrowsegotodef►    Goto Definition        «Pwbrowsegotodef»
     ◄Pwbrowsegotoref►    Goto Reference         «Pwbrowsegotoref»
     ◄Pwbrowse1stdef►     ───                    «Pwbrowse1stdef»
     ◄Pwbrowse1stref►     ───                    «Pwbrowse1stref»
     ◄Pwbrowselistref►    List References        «Pwbrowselistref»
     ◄Pwbrowseoutline►    Module Outline         «Pwbrowseoutline»
     ◄Pwbrowsecalltree►   Call Tree (Fwd/Rev)    «Pwbrowsecalltree»
     ◄Pwbrowseviewrel►    View Relationship      «Pwbrowseviewrel»
     ◄Pwbrowsepop►        ───                    «Pwbrowsepop»
     ◄Pwbrowseclhier►     Class Hierarchy        «Pwbrowseclhier»
     ◄Pwbrowsefuhier►     Function Hierarchy     «Pwbrowsefuhier»
     ◄Pwbrowsecltree►     Class Tree (Fwd/Rev)   «Pwbrowsecltree»
     ◄Pwbrowsewhref►      Which Reference        «Pwbrowsewhref»

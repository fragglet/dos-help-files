     Begline («begline»)
          Places the cursor on the first nonblank character in the
          line.
 
     Meta Begline («meta» «begline»)
          Places the cursor in the first character position of the line
          (column one).
 
     Returns
 
     True:  Cursor moved.
     False: Cursor not moved; the cursor is already at the destination.

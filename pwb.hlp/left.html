     Left («left»)
          Moves the cursor one character to the left. If this movement
          results in the cursor moving out of the window, the window is
          adjusted to the left as specified by the Hscroll switch.
          See: Hscroll
 
     Meta Left («meta» «left»)
          Moves the cursor to the first column in the window.
 
     Returns
 
     True:  Cursor moved.
     False: Cursor not moved; the cursor is in column one.
 
     See
 
     Begline, Down, Endline, Home, Right, Up

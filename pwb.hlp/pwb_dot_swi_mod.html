     The following switches have changed from PWB 1.x to PWB 2.00.
 
     Switch            Change
 
     ◄Autosave►        Default changed from Yes to No
     ◄Build►           New types of build rules
     ◄Color►           Colors for new interface elements
     ◄Defwinstyle►     No-border window state obsolete
     ◄Factor►          Default value changed to Factor:%50 10
     ◄Friction►        Default value changed to Friction:40
     ◄Rmargin►         Changed from trigger to hard margin
     ◄Tabstops►        Was Numeric. Allows variable tab stops
     ◄Wordwrap►        Simplified wordwrap behavior

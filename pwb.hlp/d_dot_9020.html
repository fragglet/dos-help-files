     The Text Argument dialog box is used for many tasks. The way the
     dialog box looks and the way you execute the task vary, depending
     on the circumstances under which it is opened.
 
     To open the dialog box:
 
       ■  Execute the Arg function («arg») to enter information
          for a command about to be executed.
      OR
       ■  Enter information (such as a search or replace string) in
          response to a prompt by PWB.
 
     To delete text in the Text Argument dialog box:
 
       1. Press SHIFT + the LEFT ARROW or RIGHT ARROW key to highlight
          the text to be deleted.
 
       2. Type the new text, which replaces the highlighted text.
 
     To close the Text Argument dialog box:
 
        ■ Choose Cancel or press ESC. This also terminates any function
          being executed.
 
       OR
 
        ■ After you type information for a command to execute, press
          the function key assigned to that command. The dialog box
          closes, and the command is executed.
 
       OR
 
        ■ After you type information in response to a prompt by PWB,
          press ENTER. The dialog box closes, and the task is executed.

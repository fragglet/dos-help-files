     _pwbnextlogmatch («_pwbnextlogmatch»)
 
     Menu  ◄Search menu, Next Match►
 
     The _pwbnextlogmatch macro advances the cursor to the next match
     listed in the Search Results pseudofile.
 
     The Next Match command executes this macro when search logging is
     on. When search logging is off, Next Match executes the
     _pwbnextmatch macro.
     See: _pwbnextmatch

     Cannot open [<file>]: Not found
 
     The PWBHELP extension tried to load the given Help file, but the
     file could not be found.
 
     Possible causes of this error include:
 
       ■  The Help file was listed in the HELPFILES environment variable
          or the Helpfiles switch, but the file does not exist in the
          specified directory.
 
       ■  The filename or directory is misspelled.

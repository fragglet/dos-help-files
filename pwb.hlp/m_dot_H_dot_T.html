     Help menu, Topic command («_pwbhelp_context»)
 
     Macro  ◄_pwbhelp_context►
 
     Choose the Topic command to get Help on a specific topic. Place
     the cursor anywhere on the topic you want to look up, or select
     the text you want to look up, then choose the Topic command.
 
     PWB displays the topic in the Help menu:
 
     Topic: xxxxxxxxxxx  «_pwbhelp_context»
                ▲
                └───── The topic to look up
 
     The Topic command is unavailable when no Help topic is selected.
 
     If the topic you want to get Help on is not displayed in a window,
     use the following sequence of commands:
 
       1. Type Arg («arg»).
 
          PWB highlights the character at the cursor.
 
       2. Type the topic you want Help on.
 
          When you type the first letter of the topic, PWB displays the
          Text Argument dialog box. Type the topic you want Help on.
 
       3. Execute _pwbhelp_context («_pwbhelp_context»).
 
     If you frequently want to use a dialog box to request Help, put the
     example macro for the Prompt function into your TOOLS.INI file.
     See: ◄Prompt Macro Example►

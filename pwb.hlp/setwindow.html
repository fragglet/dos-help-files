     Setwindow («setwindow»)
          Redisplays the contents of the active window.
 
     Meta Setwindow («meta» «setwindow»)
          Redisplays the current line.
 
     Arg Setwindow («arg» «setwindow»)
          Adjusts the window so that the cursor position becomes the
          home position (upper-left corner).
 
     Returns
 
     Setwindow always returns true.

     Msgflush (Boolean)
 
     The Msgflush switch determines if previous build results are
     retained in the Build Results window or flushed when a new build
     is started.
 
     Syntax
 
     Msgflush:{ yes | no }
 
     yes    Flush previous build results when a new build is started.
 
     no     Save previous build results.
 
     Default
 
     Msgflush:yes
 
     See
 
     Nextmsg, Searchflush

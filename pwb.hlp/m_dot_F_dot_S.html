     File menu, Save command («_pwbsavefile»)
 
     Macro  ◄_pwbsavefile►
 
     Choose Save to save the current file to disk.
 
     If the current file is a pseudofile (such as an untitled file),
     PWB brings up the Save As dialog box where you can type a disk
     filename.
     See: Pseudofile, ◄File menu, Save As►
 
     If the current file is the Record pseudofile, PWB writes the
     recorded macro and its key assignment to TOOLS.INI.
     See: Record, ◄Edit menu, Record On► ◄Edit menu, Set Record►

PWB Message
 
     Unable to restore screen image
 
     PWB is exiting or returning from an external command (build,
     shell, or compile) and the Savescreen switch is set to Yes. PWB
     is unable to restore the operating-system screen image.
 
     This is an informative message only.
 
     If this message is a common occurrence with your environment, set
     the Savescreen switch to No.

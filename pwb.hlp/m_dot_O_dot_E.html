     Options menu, Environment Variables command
 
     Choose Environment Variables to display the Environment Variables
     dialog box, where you can view, define, and change environment
     variables.
 
     Environment variables are saved for PWB sessions when the
     Envcursave switch is set to Yes and saved for individual projects
     when the Envprojsave switch is set to Yes. If both switches are
     set to No, changes that you make in this dialog box affect only
     the current PWB session. Project settings override session
     settings.

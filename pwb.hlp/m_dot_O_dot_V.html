     Choose CodeView Options to display the CodeView Options dialog box,
     which provides the following options to control how CodeView runs,
     displays information, and handles the display of output:
 
     Option                    Description
 
     Screen Size               Selects the size of the CodeView display.
                               See: ◄CodeView /25, /43, /50 Options►
 
     ◄Screen Swap►             Selects the screen-exchange method.
 
     2-Monitor Debugging       The CodeView display appears on the
                               second monitor.
                               See: ◄CodeView /2 Option►
 
     Black-and-White Mode      Selects monochrome CodeView display.
                               See: ◄CodeView /B Option►
 
     Disable the Mouse         Disables the mouse in CodeView.
                               See: ◄CodeView /M Option►
 
     ◄386 Debug Registers►     Enables the use of special debugging
                               registers.
 
     ◄Execute Commands►        Specifies Command-window commands to
                               execute at startup.
 
     Additional Options        Specifies additional CodeView command-
                               line options.
                               See: ◄CodeView Command-Line Options►
 
     Options                   Displays the options that are passed to
                               CodeView.
 
     To set command-line arguments for your program, choose Program
     Arguments from the Run menu.
     See: ◄Run menu, Program Arguments►

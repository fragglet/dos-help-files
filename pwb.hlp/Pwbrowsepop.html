     Pwbrowsepop («pwbrowsepop»)
 
     The Pwbrowsepop function goes to the location last visited by a
     PWBROWSE 'goto' command. Up to 16 locations are recorded.
 
     The following are the commands for which locations are recorded:
 
     Function             Browse Menu Command    Key
 
     ◄Pwbrowsegotodef►    Goto Definition        «Pwbrowsegotodef»
     ◄Pwbrowsegotoref►    Goto Reference         «Pwbrowsegotoref»
     ◄Pwbrowse1stdef►     ───                    «Pwbrowse1stdef»
     ◄Pwbrowse1stref►     ───                    «Pwbrowse1stref»
 
     The locations visited by the Next and Previous commands are not
     recorded for use with Pwbrowsepop. Only the first location in a
     browse sequence is recorded.
     See: Pwbrowsenext, Pwbrowseprev

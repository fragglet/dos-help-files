How to Discard a File
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Programmer's WorkBench─────────────────────────────────────────────────
 
     To remove a file from the active window and discard any changes
     that you have made, type Arg Refresh («arg» «refresh»).
 
     Note that if the file is in more than one window, it is discarded
     only from the active window.
 
     If the Autosave switch is set to No (the default) you can discard
     all instances of the file:
 
       1. Do one of the following:
 
            ■  Choose Close from the File menu.
            OR
            ■  Execute the Closefile function («closefile»).
 
          PWB prompts for confirmation to save changes.
 
       2. Choose No.

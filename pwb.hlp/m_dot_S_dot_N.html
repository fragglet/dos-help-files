     Search menu, Next Match command («_pwbnextmatch» when logging off)
                                     («_pwbnextlogmatch» when logging on)
 
     Macro  ◄_pwbnextmatch► (when logging off)
            ◄_pwbnextlogmatch► (when logging on)
 
     Choose Next Match to perform one of two actions, depending on the
     search-logging state:
 
     Logging     Next Match Action
 
     Off         Repeats the last find, searching forward from the cursor
                 by executing _pwbnextmatch.
 
     On          Moves the cursor to the next match in the Search Results
                 pseudofile by executing _pwbnextlogmatch.
 
     Next Match is unavailable when no search has been performed and
     logging is off or when there are no matches in the Search Results
     window and logging is on.

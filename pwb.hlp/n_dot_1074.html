PWB Message
 
     Current directory is not the project directory
              Change to project directory?
 
     A request was made to perform a build operation for the project,
     but the current directory is not the project's build directory.
     PWB is asking for confirmation to change the current directory
     to the project's build directory.
 
     Response    Action
 
     OK          Change the current directory to the project directory,
                 and continue the operation.
     Cancel      Do not change directories, and cancel the operation.

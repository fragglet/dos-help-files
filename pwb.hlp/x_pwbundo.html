     _pwbundo («_pwbundo»)
 
     Menu  ◄Edit menu, Undo►
 
     The _pwbundo macro reverses the last modification made to the
     current file. The maximum number of modifications that can be
     undone for each file is determined by the Undocount switch.
     See: Undo, Undocount

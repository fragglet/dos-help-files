     Browse menu, List References command («pwbrowselistref»)
 
     Function  Pwbrowselistref
 
     Choose List References to display the List References dialog box,
     where you can choose a type of object for which to list all
     references in the project.
 
     You can choose from the following options: Functions, Variables,
     Types, Macros, and Classes.
     See: ◄Source Browser Object Equivalents►
 
     Turn on an option to list references for that type of object.
     Turn off an option to omit that type of object from the list.
 
     Choose OK to list all references to the specified types of objects.
     The results are displayed in the Browser Output window.

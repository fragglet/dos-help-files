PWB Error
 
     user switch: Cannot prompt on return from a redirected command
   OR
     Cannot prompt on return from a redirected command
 
     An attempt was made to add a program to the Run menu, but the
     program was set to prompt before returning and to redirect output.
     PWB cannot support both prompting and redirected output for the
     same command.

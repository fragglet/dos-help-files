     To build a target using the keyboard:
 
       1. Execute Arg («arg» «arg»).
       2. Type the name of the target or targets.
       3. Execute Compile («compile»).
 
     To rebuild a target and its dependents using the keyboard:
 
       1. Execute Arg Meta («arg» «meta»)
       2. Type the name of the target or targets.
       3. Execute Compile («compile»).
 
     TIP: You can also include NMAKE command-line options and macro
          definitions in the argument you pass to the Compile function.
          See: ◄NMAKE menu, Options►

     Traillinesdisp (Numeric)
 
     The Traillinesdisp switch specifies the decimal ASCII code for the
     character displayed in the first column of blank lines at the end
     of the file. If you specify 0 or 255, PWB uses the space (ASCII
     32) character.
     See: ◄ASCII Table►
 
     Default
 
     Traillinesdisp:32
 
     See
 
     Traillines, Traildisp, Trailspace

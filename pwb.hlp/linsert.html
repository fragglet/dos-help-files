     Linsert («linsert»)
          Inserts one blank line above the current line.
 
     Arg Linsert («arg» «linsert»)
          Inserts or deletes blanks at the beginning of a line to move
          the first nonblank character to the cursor.
 
     Arg <boxarg> │ <linearg> Linsert («arg» <boxarg> │ <linearg> «linsert»)
          Inserts blanks within the specified area.
 
          The argument is a linearg or boxarg regardless of the current
          selection mode. The argument is a linearg if the starting and
          ending points are in the same column.
 
     Arg mark Linsert («arg» mark «linsert»)
          Like <boxarg> │ <linearg> except the specified area is given by
          the cursor position and the position of the specified mark.
          The <mark> argument must be a named mark: it cannot be a line
          number.
          See: Mark
 
     Returns
 
     Linsert always returns true.

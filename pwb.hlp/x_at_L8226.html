     These are the major options in the View Relationship dialog box:
 
     Label       Description
 
     Name        Specifies the name of a symbol or file to be queried.
 
     List        Displays the results of the specified query. Select a
                 name from the list to place it in the Name box. You can
                 then query the new name.
 
     Show        Specifies the types of objects to view. You can choose
                 any combination of Funcs (functions), Classes, Types,
                 Vars (variables), and Macros.
                 See: ◄Source Browser Object Equivalents►
 
     Pattern     Specifies a regular expression to filter the results of
                 the query.
 
     Queries     Specifies the query to perform. To see the results of a
                 query, double-click an option under Queries or select
                 an option and choose OK.
                 See: ◄View Relationship Queries►
 
     The View Relationship dialog box also provides command buttons
     and options to specify queries.
     See: ◄Source Browser Command Buttons►

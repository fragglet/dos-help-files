     "<COMPILE>Build Results"
 
     The <COMPILE> pseudofile contains the result of the last build or
     compile operation.
 
     This file is displayed in the Build Results window, which you open
     by opening the Window menu and choosing Build Results from the PWB
     Windows cascaded menu.

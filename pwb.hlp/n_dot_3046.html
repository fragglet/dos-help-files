PWB Error
 
     File name missing
 
     A filename must be specified when a mark is defined.
 
     The correct syntax for a mark definition in a mark file is
 
          markname filename line column
 
     The mark <markname> is defined as the location given by <line> and
     <column> in the file <filename>.

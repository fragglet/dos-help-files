     _pwbbuild := cancel arg "all" compile <
 
     Cancel               Establishes a uniform "ground state" by
                          canceling any selection or argument.
 
     Arg "all" Compile    Builds 'all' in the current project.
 
     <                    Restores the function's prompt (if any). By
                          default, function prompts are suppressed while
                          a macro is running.

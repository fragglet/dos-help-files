     Browse menu, View Relationship command («pwbrowseviewrel»)
 
     Function  Pwbrowseviewrel
 
     Choose View Relationship to display the View Relationship dialog
     box, which allows you to view many kinds of interrelationships
     among objects in your project. You can go to the definition or
     references of a name in your source files. You can save the
     results of the queries for study or to document your code.
 
     The View Relationship dialog box initially displays the name at the
     cursor in the Name box. You can use this name, type a new name, or
     select a name from the list. You can then query the browser
     database to display relationships between the specified name and
     other objects in the project.

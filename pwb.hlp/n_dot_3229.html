PWB Error - Programmer's WorkBench
 
     '<file>': cannot access extension
 
     PWB attempted to load the specified PWB extension, but PWB could
     not access the file.
 
     The file may not exist, the directory or file may be access-
     protected, or another process may have the file open.
 
     Check that the extension filename and directory is specified
     correctly.

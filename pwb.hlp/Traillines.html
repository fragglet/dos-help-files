     Traillines (Boolean)
 
     The Traillines switch determines if PWB preserves or removes empty
     trailing lines in a file when the file is written to disk.
 
     You can make trailing lines visible by setting the Traillinesdisp
     switch to a value other than 0 or 32.
     See: Traillinesdisp
 
     Syntax
 
     Traillines:{ yes | no }
 
     yes    Preserve trailing blank lines in the file.
 
     no     Remove trailing blank lines from the file.
 
     Default
 
     Traillines:no
 
     See
 
     Traildisp, Trailspace

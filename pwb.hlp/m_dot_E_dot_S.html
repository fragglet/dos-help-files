     Edit menu, Set Record command
 
     Choose Set Record to specify the name and key association for the
     macro you plan to record. In addition, you can choose to clear
     or append to the recorded macro.
 
     The Set Macro Record dialog box provides the following options to
     control macro recording:
 
     Option            Description
 
     Name              Type the name for the recorded macro.
                       Default: playback
 
     Key Assignment    Type the key to associate with the macro.
                       See: ◄Using the Key Box►
 
     Clear First       Turn on to clear the current macro definition
                       before recording.
 
                       Turn off to append the recording to the current
                       definition.
 
     Once you have set the options for macro recording, choose the
     Record On command to toggle macro recording on and off.
     See: ◄Edit menu, Record On►
 
     More information on how to record and save macros is available.
     See: ◄Macro Recording►

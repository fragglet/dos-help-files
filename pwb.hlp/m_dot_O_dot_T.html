     The Project Templates cascaded submenu provides the following
     commands to set a project template, save the current settings as a
     custom project template, or remove a custom project template:
 
     Command                            Description
 
     ◄Set Project Template►             Set a project template
     ◄Customize Project Template►       Customize the current template
     ◄Save Custom Project Template►     Save the current template
     ◄Remove Custom Project Template►   Delete a custom template

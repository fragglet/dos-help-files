About Global Help Search

     Global searching is unlike the keyword or key-phrase indexing
     performed by the other Help commands. The Global Search command on
     the Help menu and the Pwbhelpsearch function search the full text
     of all Help databases for the specified text or regular expression.
 
     When you search for a string that includes spaces, PWB matches the
     string even if the words are wrapped and indented across lines in
     the Help text. When you specify a regular expression, the pattern
     is not matched across line boundaries.
 
     The search starts with the current Help database, then continues
     through all open Help databases. While searching, PWB displays the
     Searching dialog box.
     See: ◄The Searching Dialog Box►
 
     TIP: Use the Contents or Index commands to open the most likely
          Help to start searching.
 
     Choose the Stop Search button in the Searching dialog box to view
     the results of a global Help search.
     See: ◄Viewing Global Search Results►

     _pwbfile<n> := cancel arg "!n" setfile <
 
     Cancel              Establishes a uniform "ground state" by
                         canceling any selection or argument.
 
     Arg "!n" Setfile    Opens file number <n> in the file history.
 
     <                   Restores the function's prompt (if any). By
                         default, function prompts are suppressed while
                         a macro is running.
 
     Example Definition
 
     _pwbwindow1 := cancel arg "!1" setfile <
 
     This macro switches to file number 1 in the active window's file
     history.

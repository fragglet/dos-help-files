     Searchflush (Boolean)
 
     The Searchflush switch determines if previous logged search
     results are flushed or retained when you start a new logged
     search.
     See: Logsearch, ◄Search menu, Find► ◄Search menu, Log►
          ◄Window menu, PWB windows, Search Results►
 
     This switch has an effect only when PWB performs a logged search.
 
     Syntax
 
     Searchflush:{ yes | no }
 
     yes    Flush the previous search results from the Search Results
            window when a new search is begun.
 
     no     Preserve previous search results in the Search Results
            window.
 
     Default
 
     Searchflush:yes

─────Programmer's WorkBench─────────────────────────────────────────────────
 
     PWB displays the Log Search Complete dialog box when it finishes a
     logged search. Choose one of the following actions:
 
     Option          Action
 
     View Results    Opens the Search Results window.
 
                     PWB executes the _pwbviewsearchresults macro when
                     you choose this button. You can redefine the macro
                     to change the action of the View Results button.
 
     Cancel          Closes this dialog box. The results of the search
                     remain in the Search Results window for browsing.
 
     Help            Displays this help.
 
     See
 
     Clearsearch, Logsearch, Nextsearch, Enterlogmode, _pwbnextlogmatch,
     _pwbgotomatch, _pwbpreviouslogmatch, _pwbviewsearchresults

     Select (assigned to many keys)
 
     Causes a shifted key to take on the cursor-movement function
     associated with the unshifted key and begins or extends a
     selection.
 
     To see the key combinations currently assigned to this function,
     use Options Key Assignments.
     See: ◄Options menu, Key Assignments►
 
     Default:
 
     select:Shift+Pgup
     select:Shift+Ctrl+Pgup
     select:Shift+Pgdn
     select:Shift+Ctrl+Pgdn
     select:Shift+End
     select:Shift+Ctrl+End
     select:Shift+Home
     select:Shift+Ctrl+Home
     select:Shift+Left
     select:Shift+Ctrl+Left
     select:Shift+Up
     select:Shift+Right
     select:Shift+Ctrl+Right
     select:Shift+Down

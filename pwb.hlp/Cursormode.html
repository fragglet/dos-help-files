     Cursormode (Numeric)
 
     The Cursormode switch determines the shape of the cursor when PWB
     is in insert and overtype mode, according to the following table:
 
     Cursormode Value   Insert Mode Cursor   Overtype Mode Cursor
     0                  Underscore           Underscore
     1                  Block                Block
     2                  Block                Underscore
     3                  Underscore           Block
 
     Default
 
     Cursormode:2

PWB Error - Programmer's WorkBench
 
     PWB3231: Not enough memory below 640K for requested video mode
 
     PWB was unable to allocate enough memory to support the requested
     video mode. The request may be implicit by starting PWB in that
     mode.
 
     Change to a lower-resolution video mode and restart PWB. Note that
     there still may not be enough memory to load PWB and its extensions
     or to operate PWB with a reasonable capacity.
 
     Make more memory available and restart PWB.

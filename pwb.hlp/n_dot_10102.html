     Cannot open [<file>]: Read error
 
     The PWBHELP extension attempted to load the specified file, but an
     operating-system error was encountered while reading from the file.
 
     The file system may be corrupt. Run CHKDSK to examine the status of
     the disk where the file is located.
 
     The Help file may be corrupt. Delete the file and reinstall it from
     the distribution disk.

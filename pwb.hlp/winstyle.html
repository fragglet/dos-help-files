     Winstyle («winstyle»)
 
     Advances through the following series of window styles, starting
     from the current style:
 
     Horizontal Scroll Bar    Vertical Scroll Bar
     No                       No
     No                       Yes
     Yes                      No
     Yes                      Yes
 
     When the horizontal scroll bar is not shown, a maximized window
     does not show its bottom border. Similarly, when the vertical
     scroll bar is not shown, a maximized window does not show its left
     and right borders. PWB always displays the title bar.
 
     To get the "clean-screen" look, maximize the window and advance
     the window style until the borders disappear.
     See: Maximize
 
     Default
 
     Set the default window style with the Defwinstyle switch.
     See: Defwinstyle
 
     Returns
 
     True:  Changed window style.
     False: No windows open.
 
     Update
 
     The no-border state in PWB 1.x is not available in PWB 2.00.
     In PWB 2.00, when a window is maximized and no scroll bars are
     present, PWB displays the window without borders.
     See: Maximize

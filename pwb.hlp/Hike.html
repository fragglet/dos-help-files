     Hike (Numeric)
 
     The Hike switch determines the number of lines from the cursor to
     the top of the window after you move the cursor out of the window
     by more than the number of lines specified by the Vscroll switch.
     See: Vscroll
 
     The minimum value is 1. When the window occupies less than the
     full screen, the value is reduced in proportion to the window
     size.
 
     Default
 
     Hike:4

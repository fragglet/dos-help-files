     Help menu, Contents command («_pwbhelp_contents»)
 
     Macro  ◄_pwbhelp_contents►
 
     The Contents command opens the Help window and displays the
     top-level contents of the Microsoft Advisor Help system.
 
     Within the Microsoft Advisor, most Help topics have a Contents
     button at the top of the window. This button also takes you to the
     top-level contents.

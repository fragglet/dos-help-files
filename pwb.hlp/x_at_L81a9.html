     To go to a named mark:
 
       1. Execute Arg («arg»).
       2. Type the name of the mark.
       3. Execute Mark («mark»).
 
     To go to a specific line in the current file:
 
       1. Execute Arg («arg»).
       2. Type the line number.
       3. Execute Mark («mark»).

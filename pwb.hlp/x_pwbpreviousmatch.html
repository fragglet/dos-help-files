     _pwbpreviousmatch («_pwbpreviousmatch»)
 
     Menu  ◄Search menu, Previous Match►
 
     The _pwbpreviousmatch macro searches backward in the file, using
     the last search pattern and options. The search options are Case
     Sensitive, Wrap Around, and Regular Expression.
 
     The Previous Match command executes this macro when search logging
     is off. When search logging is on, Search Previous Match executes
     the _pwbpreviouslogmatch macro.
     See: _pwbpreviouslogmatch

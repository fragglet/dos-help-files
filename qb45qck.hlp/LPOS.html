LPOS - a device I/O function that returns the current position of
       the line printer's print head within the printer buffer
 
Syntax
  LPOS(n)
    ■ n, a numeric expression that has an integer value between 0 and 3,
      is the index of the printer port being tested. 1 means LPT1:, 2 means
      LPT2:, etc.

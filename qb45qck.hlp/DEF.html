DEF FN - a procedure statement that defines and names a function
 
Single-Line Syntax
  DEF FNname[(parameterlist)] = expression
 
Block Syntax
  DEF FNname[(parameterlist)]
     [statements]
  FNname = expression
     [statements]
  END DEF
 
  ■ The FUNCTION...END FUNCTION statement offers a more straight-forward
    way to define and name a new function. If you must use DEF FN, see
    ◄Details► for the meaning of the various arguments.

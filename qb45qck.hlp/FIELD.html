FIELD - a file I/O statement that allocates space for variables in a
        random-access file buffer
 
Syntax
  FIELD [#]filenumber, fieldwidth AS stringvariable
                     [,fieldwidth AS stringvariable] ...
    ■ filenumber is the number of an open file
    ■ fieldwidth is the number of characters in the field
    ■ stringvariable is the location in the program where the field data
      is read to or written from

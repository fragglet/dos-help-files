LEN - a string processing function that returns the number of
      characters in a string or the number of bytes required by a variable
 
Syntax
  LEN(stringexpression)
  or
  LEN(variable)
    ■ stringexpression is a string constant or string expression
    ■ variable identifies the object for which you want the number of
      required bytes

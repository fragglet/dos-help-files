WHILE...WEND - a control flow statement that executes a series of
               statements in a loop, as long as a given condition is true
Syntax
  WHILE condition
    [statements]
  WEND
    ■ condition is an expression that will return non-zero (true) or zero
      (false)
    ■ statements are one or more BASIC statements to be executed as long as
      condition is true
 
See also  ◄DO...LOOP► - to see a more flexible way to build a loop
          ◄FOR...NEXT►

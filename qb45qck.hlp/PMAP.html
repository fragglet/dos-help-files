PMAP - a graphics function that maps view-coordinate expressions to
       physical locations or maps physical expressions to a view-coordinate
       location
 
Syntax
  PMAP (expression, function)
    ■ expression indicates the coordinate to be mapped
    ■ function, a numeric expression which has an integer value
      between 0 and 3, specifies one of four different mapping functions
      to be done (see ◄Details►).

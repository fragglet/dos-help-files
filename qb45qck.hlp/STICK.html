STICK - a device I/O function that returns the x and y coordinates
        of the two joysticks
 
Syntax
  STICK(n)
    ■ n, a numeric expression that has an unsigned integer value between
      0 and 3, tells the STICK function which one of four possible
      joystick coordinate values to return (see ◄Details►)

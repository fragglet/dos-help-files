UNLOCK - a file I/O statement that releases locks applied to parts
         of a file
  Tip: LOCK and UNLOCK are only useful in networked environments where
       several processes my need access to the same file. They only
       function if you are using DOS version 3.1 or later.
 
Syntax
  UNLOCK [#] filenumber [,{record| [start] TO end}]
    ■ filenumber is the number of an open file
    ■ record, start, and end indicate the range of records or bytes to
      be locked

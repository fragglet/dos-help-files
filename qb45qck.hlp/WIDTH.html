WIDTH - a file I/O statement that assigns an output-line width to
        a file or device or changes the number of columns and lines
        displayed on the screen
 
Syntax
  WIDTH [columns][,lines]
  WIDTH {#filenumber | device},width
  WIDTH LPRINT width
 
    ■ See ◄Details► for the different ways the WIDTH statement arguments
      can be used

IOCTL$ - a file I/O function that receives a control data string
         from a device driver
 
Syntax
  IOCTL$([#]filenumber)
    ■ filenumber is a the number of an open device
 
    Refer to the device driver documentation to find out what status
    information the device driver is capable of sending.

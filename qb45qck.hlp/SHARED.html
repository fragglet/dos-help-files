SHARED - a statement that gives a SUB or FUNCTION procedure access to
         variables declared at the module level without passing them as
         parameters
Syntax
  SHARED variable [AS type] [,variable [AS type]]...
    ■ variable identifies the module-level variable the procedure will use.
      It is either an array name followed by (), or a variable name.
    ■ The optional AS type clause is a ◄Detail►.

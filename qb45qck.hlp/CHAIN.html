CHAIN - a program control statement that transfers control to another
        program
Syntax
  CHAIN filespec
    ■ filespec is a string expression that identifies the program to which
      control is passed. The filespec may include a path specification.
 
See Also  ◄RUN► - another way to transfer control to a program
          ◄COMMON► - which enables sharing of variables between chained
                     programs

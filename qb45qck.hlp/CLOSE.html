CLOSE - a file I/O statement that concludes I/O to a file or device
 
Syntax
  CLOSE [[#]filenumber[,[#] filenumber]...]
    ■ If all arguments are omitted, all open files and devices are closed
    ■ filenumber is the number of an open file

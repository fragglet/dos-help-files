IOCTL - a file I/O statement that transmits a control data string to
        a device driver
 
Syntax
  IOCTL [#]filenumber, string
    ■ filenumber is the number of an open device
    ■ string, a string-expression, is the command sent to the device.
      Refer to the device driver documentation to find out what the valid
      IOCTL command strings are.

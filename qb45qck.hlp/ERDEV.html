ERDEV and ERDEV$ - error-trapping functions that provide
                   device-specific status information after an error
Syntax
  ERDEV
  ERDEV$
 
  ■ ERDEV is an integer function that returns an error code from the
    last device to return an error
  ■ ERDEV$ is a string function that returns the name of the device
    returning the error

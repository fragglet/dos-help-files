SETMEM - a memory management function that changes the amount of memory
         used by the far heap - the area where far objects and internal
         tables are stored
 
Syntax
  SETMEM(numeric-expression)
    ■ numeric-expression has a value that is less than zero, zero, or
      greater than zero. See ◄Details► about how the SETMEM function
      interprets each of these values.

Error Message
 
Error 34:  Type identifier expected
 
QuickPascal expects a data type at the cursor location.
 
Add the appropriate data type (for example, Integer,
Char, or Double) as required by the statement. For
instance, array declarations require the data type of
the array elements, and function declarations require
both the data types of the returned value and the types
of parameters passed to the function.

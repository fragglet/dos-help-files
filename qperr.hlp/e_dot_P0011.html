Error Message
 
Error 11:  Error in integer constant
 
The integer constant at the cursor was not in the
range -2147483647 to 2147483648.
 
Check that the number is an integer within range.
 
This error occurs when specifying the lowest value in the
LongInt range, -2147483648. Use First( LongInt ) to
represent this value.

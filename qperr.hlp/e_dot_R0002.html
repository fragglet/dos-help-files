Run-Time Error Message
 
Run-Time Error 2:  File does not exist
 
No file exists by the name assigned to the file variable.
 
When a file variable that has been assigned the name of a
nonexistent file is passed to the Append, Erase, Rename,
or Reset procedures, this error occurs.

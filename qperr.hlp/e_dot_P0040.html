Error Message
 
Error 40:  Incompatible subrange types
 
The statement contains two different subrange types.
 
A subrange type is composed of a specified range of any
standard or user-defined ordinal types. Subranges consist
of elements of the same type. The lower and upper bounds
of a subrange must be of the same data type.
 
Change one of the boundary values so that both values
share the same data type.

Error Message
 
Error 39:  Subrange bounds must be scalar
 
The limiting values of a subrange must be scalar.
 
"Scalar" refers to values that are either integers or
or enumerated data types, such as A..Z. The boundary
values cannot be real.
 
Replace the subrange bounds with the appropriate scalar
values.

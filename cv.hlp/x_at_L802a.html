Use Single Monitor (/V) (CVW only)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The CVW /V option enables the use of a single monitor for Windows
     debugging on systems with a display other than a VGA. With a VGA,
     this option is not necessary.
 
     When using the /V option, the Output command from the View menu is
     disabled and the CVW screen displays 25 lines.
 
     This option requires installation of the VCV.386 driver and a
     device entry in the [386enh] section of your Windows SYSTEM.INI
     file. If this driver is in the SYSTEM subdirectory of your Windows
     installation, the entry would be
 
          device=system\vcv.386

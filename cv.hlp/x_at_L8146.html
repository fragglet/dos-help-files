     To replay part of a debug session:
 
       1. Browse forward to a history point with SHIFT+F10, or browse
          backward with SHIFT+F8.
 
       2. Choose the Replay command from the Run menu. CodeView reruns
          the session from the start of the program to the point where
          you stopped.
 
     NOTE: Use the Replay command from the Options menu to tell
           CodeView what you want to replay.

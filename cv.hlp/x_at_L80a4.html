────────────────────────────────────────────────────────────────────────────
 
     The Quit (Q) command terminates CodeView and returns control to
     the environment from which CodeView was invoked, Programmer's
     WorkBench (PWB), Windows, or the operating system.
 
     CodeView always saves state information on exit.

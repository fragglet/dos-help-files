Search Menu: Selected Text Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: CTRL+\
 
     Use the Selected Text command to search for highlighted (selected)
     text. The entire string must be on one line.
 
       1. Select the text to search for.
 
       2. Choose Selected Text from the Search menu (or use CTRL+\).

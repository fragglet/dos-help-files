│ Mouse:                                                                   │
│    ■ To maximize the current window, click the "maximize" icon (↑) in    │
│      the upper right window border.                                      │
│    ■ To return the current window to normal size, click the "restore"    │
│      icon (↕) in the upper right window border.                          │
│ Keyboard:                                                                │
│    ■ To maximize the current window, choose Maximize from the View       │
│      menu or press CTRL+F10.                                             │
│    ■ To return the current window to normal size, choose Restore from    │
│      the View menu or press CTRL+F10 again.                              │

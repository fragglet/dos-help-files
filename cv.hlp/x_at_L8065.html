     The Breakpoint Enable (BE) command enables breakpoints that have
     been temporarily disabled with the Breakpoint Disable (BD)
     command.
     See: ◄Breakpoint Disable (BD) Command-Window Command►
 
     If you specify a breakpoint list with the BE command, the
     command enables the breakpoints named in the list. The
     Breakpoint List (BL) command shows the number for each existing
     breakpoint. If you specify an asterisk (*) as the argument, the
     command enables all breakpoints.
     See: ◄Breakpoint List (BL) Command-Window Command►

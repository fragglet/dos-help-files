     The View (V) command displays the lines of a text file, usually a
     source module or include file. You can use this command to page
     through the source file or to load a new source file.
 
     If you enter the View command with no arguments, the display
     scrolls up so that the source line at the bottom of the window
     becomes the line at the top.
 
     If you specify a line number or view address with the command,
     CodeView redraws the Source window so that the source line
     corresponding to the given address appears at the top of the
     Source window. If a line number includes a filename, CodeView
     loads that file before it redraws the Source window.

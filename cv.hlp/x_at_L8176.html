Debug Multiple Processes (CVP Only)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     OS/2, Versions 1.10 and later, supports multiprocess programs.
     Currently, programs written with Microsoft C, FORTRAN, Pascal,
     and Macro Assembler (MASM) can have more than one process.
 
     CVP.EXE, the protected-mode version of CodeView, can debug
     multiprocess programs.
 
     To debug a multiprocess program, invoke CodeView with the
     ◄/O (Debug Multiple Processes)► option on the command line.
 
     To view the status of a child process you are debugging:
 
        ■ Choose the ◄Process► command from the Run menu
       OR
        ■ Enter the ◄Process (|)► command in the ◄command window►

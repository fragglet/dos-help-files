     Command: Help (H)
 
     Syntax:  H [command-window command]
 
     Summary: Displays help on command-window commands.
 
     Argument:
 
     <command-window command>   Displays help on this command. If
                                omitted, the command displays the
                                contents screen for CodeView help.

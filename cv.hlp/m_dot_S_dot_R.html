Search Menu: Repeat Last Find Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: ALT+/ (CTRL+R in CVW)
 
     Use the Repeat Last Find command to repeat a search without
     retyping the text you are searching for.
 
     Repeat Last Find uses the same search string and options as the
     most recently executed Find command. If CodeView searches to the
     end of the file without finding another match for the search
     string, it "wraps around" and searches from the beginning.

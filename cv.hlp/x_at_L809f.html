          >O
          Flip/Swap On
          Bytes Coded On
          Case Sense Off
          386 Off
          Show Symbol Address Off
          Symbols Off
          >O3
          386 Off
          >O3+
          386 On
          >OF
          Flip/Swap On
          >OF-
          Flip/Swap Off
 
     In the example above, the O, O3, and OF commands allow you to
     view the current state of an option. Each of the O3+ and OF-
     commands modifies an option and then reports the results of the
     modification.
 
          CV /c"O3+;OB-" test
 
     In the example above, the DOS-level command line brings up
     CodeView with the 386 option on and the Bytes Coded option turned
     off.

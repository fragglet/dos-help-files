     Command: Radix (N)
 
     Syntax:  N [radixnumber]
 
     Summary: Changes the current radix for entering arguments and
              displaying expressions.
 
     Argument:
 
     <radixnumber>   New radix: 8 (octal), 10 (decimal), or 16
                     (hexadecimal). If omitted, the command displays
                     the current radix.

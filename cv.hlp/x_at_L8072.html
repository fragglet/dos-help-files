BP Command: Break When Expression Is True
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     To set a breakpoint when an expression is true, enter a Breakpoint
     Set (BP) command of the form
 
          BP ?expression
 
     where <expression> is a conditional expression.

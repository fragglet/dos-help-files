Search for Text
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     To search for text in the current Source window, choose commands
     from the ◄Search menu►. CodeView offers these search commands:
 
     Command                Action
 
     ◄Find►                 Searches for any ◄regular expression►.
 
                            NOTE: You can also enter a ◄Search (/)►
                            command in the ◄command window►.
 
     ◄Selected Text►        Searches for ◄selected text►.
 
     ◄Label/Function►       Searches for an assembly-language label or
                            a function name.
 
     ◄Repeat Last Find►     Repeats the most recently entered search
                            command.

     Command: View Memory (VM)
 
     Syntax:  VM[type] [address]
 
     Summary: Sets the display format for the Memory window.
 
     Arguments:
 
     <type>      A one-character abbreviation giving the display format
                 for the Memory window.
 
     <address>   Starting address for the Memory window.

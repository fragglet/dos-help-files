     To search for files in other drives or directories:
 
                                         ■ Source window where file
     ┌────── Open Source File ─────────┐ │ will be loaded
     │     into Source Window #◄───────┼─┘
     │                                 │ ■ Type the wildcard form
     │ File Name: [········] ◄─────────┼─┘ of the filename (such
     │                                 │   as *.H or *.C)
     │ C:\MYPROGS ◄──current directory │
     │                                 │ ■ Select the drive and
     │ File List:        Drives/Dirs:  │ │ directory name
     │ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒▒ ◄┼─┘
     │ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒   ▒▒▒▒▒▒▒▒▒▒▒▒  │
     │        ▲                        │ ■ Scan the filenames
     │        └────────────────────────┼─┘
     └─────────────────────────────────┘ ■ Choose <OK>

     To load a file from the current directory:
 
        ■ Press TAB to move the cursor to the File List box
 
        ■ Use the arrow keys to highlight the file you want to open
 
        ■ Press ENTER
       OR
        ■ Type the name of the file you want to open
 
        ■ Press TAB to move the cursor to <OK> if you want to load the
          file, or to <Cancel> if you want to exit without loading
 
        ■ Press ENTER

     Use the Options (O) command to set options when you first start up
     CodeView. You can set these options in the following ways:
 
        ■ Give one or more O commands with the /C option on the Codeview
          command line, or include such a command line in a batch file
          for convenient execution.
          See: ◄Execute Commands (/C) Option►
 
        ■ Give one or more O commands as the "autostart" entry in the
          TOOLS.INI file.
          See: ◄Configuring CodeView in TOOLS.INI►

     Command: Compare Memory (C)
 
     Syntax:  C range address
 
     Summary: Compares two blocks of memory and displays differences
              between the two.
 
     Arguments:
 
     <range>     Compares bytes in <range> with the corresponding
     <address>   bytes in <address>.

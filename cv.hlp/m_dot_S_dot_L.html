Search Menu: Label/Function Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The Label/Function command searches the current Source window for
     an assembly-language label or a function name. If CodeView finds
     the label or name, it moves the cursor to the line defining that
     label or name.
 
     If you start the search in Source mode, the debugger switches to
     Assembly mode to show a label in a library routine or an assembly-
     language module.

Run Menu: Undo Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     When debug history is on, you can rescind the last debugger
     command by using the Undo command. The Undo command traverses back
     through the debug session one command at a time, until it reaches
     the beginning.
 
     Undo has the same effect as pressing SHIFT+F8 and then choosing
     the Replay command from the Run menu.
 
     Undo has no effect when debug history is off or when you are not
     recording CodeView commands.
 
     See: ◄Run Menu: History On Command►
          ◄Run Menu: Replay Command►
          ◄Debug History►
 
     NOTE: The Run Menu: Undo command is not available in CVW.

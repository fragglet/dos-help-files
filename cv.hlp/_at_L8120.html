     NOTE: When using the mouse to click an object, always use the
           left mouse button.
 
       1. Click the arrows on the side of the Drives/Dirs box until the
          directory or drive you want is visible.
 
       2. Double-click in the directory or drive you want. Click the
          double dots (..) to move up one directory level. The File
          List box shows the files in the selected directory that match
          the expression in the File Name box.
 
       3. Click the arrows on the bottom of the File List box to view
          all the files in this directory.
 
       4. Double-click the filename you want to open.
 
     The file automatically opens in the active Source window.

CodeView Error
     Invalid flag (CV1022)
 
     An attempt was made to examine or change a flag, but the flag name
     was not valid.
 
     Any flags preceding the invalid name were changed to the values
     specified. Any flags after the invalid name were not changed.
 
     Use the flag mnemonics displayed after entering the R FL command.

Options Menu: 386 Instructions Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The 386 Instructions command displays registers in 80386 format.
     To use this option, your computer should have an 80386/486
     processor running in 80386 mode.
 
     When this option is on, the Register window displays registers in
     the wider 386 format. You can also assemble and execute
     instructions that reference 32-bit registers and memory. If the
     386 option is off, any data stored in the high-order word of a
     32-bit register will be lost.
 
     This option is not available in CVP.EXE.

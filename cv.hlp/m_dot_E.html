     ┌─────────────────────┐
     │ Undo       Alt+Bksp │ ■ Undoes most recent edit
     │ Copy       Ctrl+Ins │ ■ Copies selected text to clipboard
     │ Paste     Shift+Ins │ ■ Inserts text from clipboard
     └─────────────────────┘
 
     Use the Edit menu to manipulate text in CodeView windows. You can
     copy a single line of text from the active window to an invisible,
     temporary storage area called a "clipboard."
 
     You can paste text from the clipboard into any window that allows
     write access: the Memory, Local, Watch, Register, 8087, or Command
     window.

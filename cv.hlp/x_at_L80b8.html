Unassemble (Description)
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The Unassemble (U) command displays assembly-language code,
     beginning at a given address, in the active Source window. If the
     active Source window is in Source or Mixed mode, the command
     automatically changes the Source-window mode to Assembly.
 
     If the current Source window is in Assembly mode, the Unassemble
     command scrolls the Source window down a screen.

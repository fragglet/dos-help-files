Set CodeView Options
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     CodeView has two sets of options:
 
       1. ◄Command-line options►. You set these options on the
          command line. These options control such CodeView features as
 
             ■ Video display mode
 
             ■ Video-page swapping
 
             ■ Use of extended memory, expanded memory, or disk
               overlays
 
             ■ 80386/486 debug registers
 
             ■ Dynamic-link library debugging
 
       2. Options that you set during a debugging session using
          commands on the ◄Options menu► or with the ◄Options (O)►
          Command-window command. These options control features
          such as
 
             ■ Source and Memory window characteristics
 
             ■ ◄Screen swapping and flipping►
 
             ■ ◄Debug history► options
 
             ■ Case Sensitivity
 
             ■ Language used for expression evaluation in CodeView
               commands, or the Watch window
 
             ■ 32-bit register display and 32-bit instruction assembly

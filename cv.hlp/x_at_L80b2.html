Make Window Active (Description)
 ◄Summary►                                   ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     CodeView can display up to two Memory windows and up to two Source
     windows at once. Choose the S1 command to make Source window 1
     active or the S2 command to make Source window 2 active.
 
     Similarly, choose the M1 command to make Memory window 1 active
     or the M2 command to make Memory window 2 active. If the window
     you are activating is not open, the command opens the window
     before activating it.

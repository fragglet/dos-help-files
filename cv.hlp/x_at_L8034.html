     The Current Location (.) command displays the source line or
     assembly-language instruction corresponding to the current
     program location. It puts the current program location (marked
     with reverse video or a contrasting color) in the center of the
     active Source window.
 
     Use this command if you have scrolled the current source line or
     assembly instruction out of the active Source window.

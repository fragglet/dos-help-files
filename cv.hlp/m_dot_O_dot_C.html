     The Case Sensitivity command toggles case sensitivity on or off.
     When Case Sensitivity is turned on, CodeView treats symbol names
     as case sensitive (each lowercase letter is different from the
     corresponding uppercase letter). If Case Sensitivity is turned
     off, symbol names are not case sensitive.
 
     NOTE: This command does not control case sensitivity for commands
           on the Search menu.

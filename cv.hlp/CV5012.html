CodeView Error
     Packed file (CV5012)
 
     (DOS only)
 
     CodeView cannot debug files in DOS that are linked with the
     /EXEPACK option. Relink without this option to debug the file and
     then switch back to linking with /EXEPACK for the release version
     of your program.

Command Window
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The command window is where you enter commands. These are the
     commands with optional arguments that you can enter at the
     CodeView prompt (>).
 
     You can scroll up or down in this window to view previous
     commands and command output. You can also edit and reenter
     previously entered commands.

Options Menu: Replay Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The Replay command tells CodeView whether to replay debug history,
     user input, or both.
 
     ┌────── Replay Options ──────┐ ┌ ■ Select to replay both debug
     │                            │ │   commands and user input
     │  ( ) Replay Both ◄─────────┼─┘
     │  ( ) Replay User Input ◄───┼── ■ Select to replay user input
     │  ( ) Replay Debug Input ◄──┼─┐
     │                            │ └ ■ Select to replay debug commands
     │                            │
     └────────────────────────────┘   ■ Choose <OK>
 
     See: ◄Run Menu: Replay Command►

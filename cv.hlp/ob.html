     Command: Options (O)
 
     Syntax:  O[option[+|-]]
 
     Summary: Displays the value of CodeView options or turns those
              options on or off.
 
     Arguments:
 
     <option>   Character indicating the option to be turned on or off.
 
                Character     Option
 
                F             Flip/Swap
                B             Bytes Coded
                C             Case Sense
                S             Symbols
                L             Show Symbol Address
                3             386
 
     +          Turns option(s) on.
     -          Turns option(s) off.

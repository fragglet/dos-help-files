     If the program you are debugging accepts command-line arguments,
     you can set those arguments by
 
        ■ Giving the options on the ◄command line► when you first start
          up CodeView.
       OR
        ■ Choosing the ◄Set Runtime Arguments► command from the Run
          menu during a debugging session. This command automatically
          restarts program execution.

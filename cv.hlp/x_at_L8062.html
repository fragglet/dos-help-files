     The Breakpoint Disable (BD) command temporarily disables one or
     more existing breakpoints. The breakpoints are not deleted; they
     can be restored at any time using the Breakpoint Enable (BE)
     command.
     See: ◄Breakpoint Enable (BE) Command-Window Command►
 
     If you specify a breakpoint list with this command, the command
     disables the breakpoints named in the list. The Breakpoint List
     (BL) command shows the number for each existing breakpoint. If you
     specify an asterisk (*) as the argument, the command disables all
     breakpoints.
     See: ◄Breakpoint List (BL) Command-Window Command►
 
     The command-window commands for setting and clearing breakpoints
     can also enable or clear disabled breakpoints.
 
     When a breakpoint is disabled, it appears the same as other
     Source-window text. When a breakpoint is enabled, it appears in
     high-intensity text.

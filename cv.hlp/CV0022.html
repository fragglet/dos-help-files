CodeView Error
     Invalid argument (CV0022)
 
     An invalid value was given as an argument in the most recent
     command.
 
     One of the following may have occurred:
 
        ■ An invalid argument was passed to the Go (G) command.
 
        ■ An invalid argument was passed to the Delete Watch Expression
          (Y) command.

     The Screen Exchange (\) command allows you to switch temporarily
     from the debugging screen to the output screen. The CodeView
     debugger uses either screen flipping or screen swapping to store
     the output and debugging screens. In the protected-mode version
     of CodeView (CVP), you can specify the number of seconds that
     CodeView waits before redisplaying the current screen.
 
     When the output screen appears, press any key to return to the
     debugging screen.

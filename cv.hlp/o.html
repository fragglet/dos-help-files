     Command: Port Output (O)
 
     Syntax:  O port byte
 
     Summary: Sends a specific byte value to a hardware port.
 
     Arguments:
 
     <port>   A 16-bit value representing a port address.
 
     <byte>   Byte value sent to <port>.

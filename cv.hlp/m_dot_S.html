Search Menu
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Use the Search menu to find text, function definitions, or label
     definitions.
 
     Search operations begin at the cursor, extend to the end of the
     file, and "wrap around" to the starting point.
 
     ┌──────────────────────────┐
     │ Find...                  │ ■ Searches for expression
     │ Selected Text     Ctrl+\ │ ■ Searches for selected text
    *│ Repeat Last Find   Alt+/ │ ■ Repeats last Find command
     │ Label/Function...        │ ■ Searches for label or function
     └──────────────────────────┘
 
    *In CVW, the shortcut key is CTRL+R.

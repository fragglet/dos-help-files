     Command: Screen Exchange (\)
 
     Syntax:  \[time]
 
     Summary: Toggles between the CodeView screen and the output
              screen.
 
     Argument:
 
     <time>   For protected-mode versions of CodeView (CVP.EXE) only;
              sets the number of seconds that CodeView waits before
              the debugger returns to the current screen.

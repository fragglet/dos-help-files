Run Menu
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Use the Run menu to load and execute a program.
 
     ┌───────────────────────────┐
     │ Restart                   │ ■ Reloads program into memory
     │ Animate                   │ ■ Traces code, highlighting lines
     │ Set Runtime Arguments...  │ ■ Sets command-line arguments
     ├───────────────────────────┤
     │ History On                │ ■ Records debug session in a file
     │ Undo                      │ ■ Moves back through history file
     │ Replay                    │ ■ Replays a recorded debug session
     ├───────────────────────────┤
     │ Thread...                 │ ■ Accesses threads (CVP only)
     │ Process...                │ ■ Accesses processes (CVP only)
     └───────────────────────────┘
 
     NOTE: The Animate command is the only Run menu command that
           appears in CVW.

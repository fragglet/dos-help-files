Options Menu: Trace Speed Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The Trace Speed command sets tracing and execution speed for
     programs being debugged.
 
     ┌───── Trace Speed ────┐ ■ Select trace speed:
     │                      │
     │    ( ) Slow     ◄────┼── 1/2 second between steps
     │    ( ) Medium   ◄────┼── 1/4 second between steps
     │    ( ) Fast     ◄────┼── No wait between steps
     │                      │
     │                      │
     └──────────────────────┘ ■ Choose <OK>
 
     See: ◄Trace Speed (TS) Command-Window Command►

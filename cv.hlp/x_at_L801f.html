Use Expanded Memory (/E) (CV only)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The /E option enables the use of expanded memory. "Expanded
     memory" refers to memory made accessible according to the
     Lotus/Intel/Microsoft (LIM) Expanded Memory Specification (EMS).
     If your machine has expanded memory, and an Expanded Memory
     Manager (EMM) driver is installed, CodeView uses it to store the
     symbolic information of the program and overlays.
 
     NOTE: You do not need this option in order to use expanded memory,
           since CodeView uses expanded memory automatically if an EMM
           driver is installed. However, if you use this option, CV
           displays an error message if EMM is not installed.

     Command: Source Display Mode (S+, S-, S&)
 
     Syntax:  S[+|-|&]
 
     Summary: Sets the mode in which code is displayed in the active
              Source window.
 
     Arguments:
 
     +   Source mode
 
     -   Assembly mode
 
     &   Mixed mode

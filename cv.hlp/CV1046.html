CodeView Error - CodeView
     Invalid executable file - please relink (CV1046)
 
     The executable file did not have a valid format.
 
     One of the following may have occurred:
 
        ■ The executable file was not created with the linker released
          with this version of CodeView. Relink the object code using
          the current version of LINK.EXE.
 
        ■ The .EXE file may have been corrupted. Recompile and relink
          the program.

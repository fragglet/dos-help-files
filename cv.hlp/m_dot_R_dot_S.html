Run Menu: Set Runtime Arguments Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Use the Set Runtime Arguments command to specify a command line
     for the program being debugged.
 
     ┌─── Set Runtime Arguments ────┐
     │ Command Line: [···········] ◄┼ ■ Type command line and arguments
     │                              │   to invoke the program
     │                              │
     └──────────────────────────────┘ ■ Choose <OK>
 
     This command automatically resets execution to the logical
     beginning of the program. It has the same effect as the Restart
     command on the Run menu.
 
     See: ◄Run Menu: Restart Command►
 
     NOTE: The Set Runtime Arguments command is not available in CVW.

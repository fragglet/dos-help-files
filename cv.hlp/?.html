     Command: Display Expression (?)
 
     Syntax:  ? expression[,format]
              ? variable=expression
 
     Summary: Evaluates and displays local variables, symbol values,
              or expressions.
 
     Arguments:
 
     <expression>              The expression you are displaying. This
                               can be any valid CodeView expression.
 
     ,<format>                 A CodeView format specifier that indicates
                               the format in which <expression> is
                               displayed.
 
     <variable>=<expression>   Assigns the value of <expression>
                               to the given program variable.

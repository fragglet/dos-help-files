Run Menu: Process Command (CVP Only)

────────────────────────────────────────────────────────────────────────────
 
     The Process command displays the status of child processes or
     switches to the debug session for a particular child process. To
     use this command, you must start up CodeView in protected mode
     (CVP) with the /O (Debug Multiple Processes) command-line option.
 
     NOTE: The /O option allows you to spawn a new CodeView process for
           each child process that your program spawns.

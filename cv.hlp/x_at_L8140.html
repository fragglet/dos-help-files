History On (Create New History)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     To create a new debug-history file, choose History On from the
     Run menu. If <PROGRAM>.CVH or <PROGRAM>.CVI exists, CodeView asks
     whether to use it. Choose <No> to clear the file and create a new
     history.
 
     While recording a debugging session, you can use SHIFT+F8 at any
     time to display the previous history point and SHIFT+F10 to
     display the next history point.

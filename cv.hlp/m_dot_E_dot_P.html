Edit Menu: Paste Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: SHIFT+INS
 
     Use the Paste command to copy text from the clipboard to the
     active window.
 
     To replace text:
 
        ■ Highlight text in the active window and select the Paste
          command.
 
        ■ CodeView deletes the text and replaces it with text from the
          clipboard.
 
     To insert text:
 
        ■ Move the cursor to the position where text is to be inserted
          and select the Paste command.
 
        ■ CodeView inserts text to the right of the cursor.
 
 
     WARNING: If the pasted text replaces highlighted text, the
              highlighted text is not saved to the clipboard.

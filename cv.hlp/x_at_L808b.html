Port Input (Example)
 ◄Summary► ◄Description►                     ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
          >I 2F8 ;* hexadecimal radix assumed
          E8
 
     The example above reads input port, number 2F8, and displays
     the result, E8. You can enter the port address using any radix
     but the result is always displayed in current radix.

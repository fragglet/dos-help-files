     The Language command specifies the expression evaluator used for
     the program being debugged.
 
     ┌───── Language ───────┐
     │                      │
     │    ( ) Auto      ◄───┼ ■ Chooses language automatically when
     │    ( ) BASIC         │   new source file is loaded based on its
     │    ( ) C             │   file-name extension
     │    ( ) FORTRAN       │
     │                      │
     └──────────────────────┘

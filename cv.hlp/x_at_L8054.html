Comment (Example)

     > T>OUTPUT.TXT
     > * Dump first 20 bytes of screen buffer
     > D #B800:0 L 20
     B800:0000 54 17 6F 17 20 17 72 17 65 17 74 17 75 17 72 17
          T.o. .r.e.t.u.r.n. .
     B800:0010 6E 17 20 17
     > CON
 
     In the example above, the user is sending a copy of a CodeView
     session to the file OUTPUT.TXT. Comments are added to explain the
     purpose of the command. The text file will contain commands,
     comments, and command output.
 
     > * Dump first 20 bytes of screen buffer
     > D #B800:0 L 20
     ∙ ∙ ∙
     > CON
 
     The example above illustrates another way to use the Comment (*)
     command. You can put comments into a text file of commands that
     are executed automatically when you redirect the file to the
     CodeView debugger.

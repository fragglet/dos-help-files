     Command: Search (/)
 
     Syntax:  / [regularexpression]
 
     Summary: Searches for a regular expression.
 
     Argument:
 
     <regularexpression>   Searches for the first line containing this
                           expression. If omitted, the command searches
                           for the next occurrence of the last
                           regular expression given.

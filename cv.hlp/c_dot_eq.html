     Command: Redirect CodeView Input and Output
 
     Syntax:  = device
 
     Argument:
 
     <device>   Device to redirect input and output to. Specify con for
                the CodeView Command window.
 
     Description:
 
     The Redirect Input and Output (=) command causes the CodeView
     debugger to read all subsequent command input from the device and
     write all subsequent output to the device. You cannot redirect
     both input and output to a file.
 
     To reset the input and output for CodeView after you have entered
     one of the other redirection commands, use the command
 
          = con

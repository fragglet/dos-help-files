     Command: Examine Symbols (X?)
 
     Syntax:  X[L|*|?[module!][function.][symbol][*]]
 
     Summary: Displays names and addresses of variables and functions
              within a program.
 
     Arguments:
 
     (none)   If you give no arguments, searches for all symbolic names
              in the program.
 
     L        Displays the names, addresses, types, and values of all
              local variables for the current function.
 
     *        Displays all module names in the program.
 
     ?        Displays the names, addresses, and types of symbols in
              given modules (<module>!) and functions (<function>.).

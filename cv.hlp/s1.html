     Command: Make Window Active (S, M)
 
     Syntax:  [S|M] [1|2]
 
     Summary: Makes a Source or Memory window active.
 
     Arguments:
 
     S     Makes a Source window active.
     M     Makes a Memory window active.
     1 2   Makes Source (S) window or Memory (M) window 1 or 2 the
           active window.

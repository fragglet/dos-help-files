     Command: Windows Display Modules (WDM)
 
     Syntax:  WDM
 
     Summary: Displays a list of all DLL and application modules that
              Windows has loaded. The display lists the module handle,
              the type of module (DLL or TASK), the name of the module,
              and the full path of the module.

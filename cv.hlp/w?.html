     Command: Add Watch Expression (W?)
 
     Syntax:  W? expression [,format]
 
     Summary: Displays the value of an expression in the watch window.
 
     Arguments:
 
     <expression>   Adds an expression to the watch window.
 
     <format>       A CodeView format specifier that indicates the
                    format in which <expression> is displayed.

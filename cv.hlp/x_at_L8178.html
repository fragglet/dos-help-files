     You can view memory contents in a Memory window or in the
     ◄command window►.
 
     To view memory contents in a Memory window:
 
        ■ Choose the ◄Memory Window► command from the View menu
        OR
        ■ Enter a ◄View Memory (VM)► command in the ◄command window►
 
     CodeView updates the memory-window display automatically as you
     execute your program.
 
     To view memory contents in the command window, enter one of the
     ◄Dump (D)► commands in the ◄command window►. Dump commands show a
     "snapshot" of memory; CodeView does not update the display as you
     execute the program.

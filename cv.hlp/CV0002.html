CodeView Error - CodeView
     No such file or directory (CV0002)
 
     The specified file does not exist or a pathname does not specify
     an existing directory.
 
     Check the file or directory name in the most recent command.
 
     One of the following may have occurred:
 
        ■ The View (V) command or the Open Source command from the File
          menu was used to view a nonexistent file.
 
        ■ An attempt was made to print to a nonexistent file or
          directory.

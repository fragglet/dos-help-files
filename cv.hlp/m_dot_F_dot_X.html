File Menu: Exit Command

────────────────────────────────────────────────────────────────────────────
 
     Use the Exit command to end a debugging session and return to the
     environment from which you invoked CodeView (PWB, DOS, OS/2, or
     Windows). CodeView always saves its state in the CURRENT.STS file
     on exit.
 
     In Windows under normal termination conditions, CVW exits to
     Windows. Abnormal termination in CVW may also terminate the
     Windows session.

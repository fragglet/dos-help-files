Edit Menu: Undo Command
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: ALT+BACKSPACE
 
     Use the Undo command to retract your most recent edit. Undo
     restores the current line to its previous condition.
 
     NOTE: The Undo command works only while the cursor is still on the
           edited line. Once you move the cursor off that line, Undo no
           longer has any effect.

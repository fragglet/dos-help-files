────────────────────────────────────────────────────────────────────────────
 
     The Enter Commands (E) command enters one or more byte values into
     memory at a given address.
 
     The command may include a list of expressions separated by spaces.
     The expressions are evaluated and entered in the current radix.
     If an invalid value appears in the list, values at and following
     the error are not entered. If no list is given, the CodeView
     debugger prompts for new values.

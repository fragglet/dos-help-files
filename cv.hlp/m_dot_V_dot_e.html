     Shortcut key: CTRL+F4
 
     Use the Close command to close the window where the cursor is
     located. If only one window is open, you cannot close that window.
 
     The Close command only closes one window at a time.
 
     TIP: If you are using a mouse, close the window by clicking in
          the Close Window icon │♦│ on the top left corner of the
          window border.

Change Directories: Keyboard How-To

────────────────────────────────────────────────────────────────────────────
 
       1. Press TAB to move the cursor to the Drives/Dirs box.
 
       2. Use the arrow keys to highlight the directory you want.
          Highlight the double dots (..) to move up one directory.
 
       3. Press ENTER. The File List box shows the files in the
          selected directory that match the expression in the File Name
          box.
 
       4. Press TAB to move the cursor to the File List box.
 
       5. Use the arrow keys to highlight the file you want to load.
 
       6. Press ENTER.
 
     The file automatically opens in the active Source window.

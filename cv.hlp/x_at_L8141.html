History On (Use Existing History)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     To use an existing debug-history file:
 
       1. Choose History On from the Run menu. A dot appears beside it.
 
       2. Choose <Yes> when the dialog box appears to load the existing
          <PROGRAM>.CVH and <PROGRAM>.CVI files.
 
       3. Choose the Replay command from the Run menu to replay the
          debug session.
 
       4. Use SHIFT+F8 to browse backward through the history or
          SHIFT+F10 to browse forward.

────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: CTRL+F8
 
     Use the Size command to expand or contract the active window. You
     must have at least two windows open to use this command.
 
     You can size windows without choosing the Size command.
 
     Mouse:
 
       1. Position the cursor on the window border.
 
       2. Press and hold the left mouse button.
 
       3. Drag the window to the size you want.
 
       4. Release the left mouse button.
 
     Keyboard:
 
       1. Press F6 to move the cursor to the window you want to size.
 
       2. Choose the Size command.
 
       3. Use the arrow keys to size the window.
 
       4. Press ENTER when you have finished sizing the window, or
          press ESC to cancel the sizing operation.

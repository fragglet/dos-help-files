     To replay an entire debug session:
 
       1. Choose History On from the Run menu.
 
       2. Choose the Replay command from the Run menu. The Replay
          command restarts the program and executes debugging commands
          from <PROGRAM>.CVH, user input from <PROGRAM>.CVI, or both.
 
     NOTE: Use the Replay command from the Options menu to tell
           CodeView what you want to replay.

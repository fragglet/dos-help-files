     CodeView provides several windows for use during a debugging
     session.
 
     Different CodeView windows are used to
 
        ■ Display program code
 
        ■ Show the values of local and global variables
 
        ■ Show memory and register contents
 
        ■ Allow the user to enter CodeView commands

     Command: Search Memory (S)
 
     Syntax:  S range list
 
     Summary: Searches a given range of memory for one or more byte
              values.
 
     Arguments:
 
     <range>   The range of memory to be searched.
 
     <list>    A list of byte values separated by spaces or commas.

View Menu: Maximize/Restore Commands
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     Shortcut key: CTRL+F10
 
     Use the Maximize command to toggle between "maximized" and
     "normal" window modes.
 
     The Maximize command enlarges the current window. The Restore
     command returns it to its prior size.
 
     NOTE: The Restore command appears on the menu when the current
           window is in maximized window mode.
 
     If you have a mouse, you can also use the box in the upper right
     corner of the window. Click the maximize (↑) symbol to maximize
     the window, and click the resize (↕) symbol to return the window
     to its normal size.
 
     TIP: Use the F6 key and the SHIFT+F6 key sequence to move from one
          window to another. F6 moves from left to right and top to
          bottom; SHIFT+F6 moves in the reverse direction.

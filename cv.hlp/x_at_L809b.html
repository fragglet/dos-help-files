          O 2F8 4F
          >
 
     In the example above, the byte value 4F hexadecimal is sent to
     output port 2F8.
 
     The example assumes that the system radix is hexadecimal. However,
     as with all other CodeView commands, you can enter the Port Output
     (O) command using any radix. Both of the port byte arguments assume
     system radix unless you specify a radix override.

     The help window displays help information about CodeView
     features and language keywords.
 
     To open the help window:
 
        ■ Choose any command from the Help menu
 
        ■ Press SHIFT+F1 to see the Contents menu
       OR
        ■ Press F1 to get help on the current context
 
          NOTE: The cursor defines the current context. For example,
                if the cursor is on a C keyword or function in a Source
                window, the F1 key displays help on that keyword or
                function. If you have just opened a dialog box or
                typed a command in the command window, the F1 key
                displays help on that command.

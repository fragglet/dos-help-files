     To load a file from the current directory:
 
        ■ Double-click the filename in the Files box
       OR
        ■ Type the name of the file to open in the File Name box and
          click <OK>
       OR
        ■ Click the filename in the Files box and click <OK>
 
     Click the arrows on the bottom of the Files box to view the entire
     list of files.
 
     Click <Cancel> to exit without loading the file.

     Compiler error C2081
 
     'identifier' : name in formal parameter list illegal
 
     The specified identifier caused a syntax error.
 
     This error can be caused by using the old style for the formal
     parameter list. The type of formal parameters must be specified
     in the formal parameter list.
 
     The following is an example of this error:
 
          void func( int i, j )  // error, no type specified for j
          {
          }

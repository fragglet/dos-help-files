     Compiler error C2672
 
     'function' : new/delete member functions do not have 'this'
     pointers
 
     The specified new or delete operator did not have a this pointer
     to access.
 
     The following is an example of this error:
 
          class C
          {
             void* operator new( unsigned int )
             {
                this = 10;      // error
                return 0;
             }
          };

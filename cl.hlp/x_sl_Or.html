Common Function-Exit Sequence (/Or)
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────C/C++ Compiler─────────────────────────────────────────────────────────
 
     Syntax:  /Or
 
     This option causes all returns within a function to use a common
     exit sequence at the end of the function. This allows you to set a
     CodeView breakpoint on the closing brace (}) of the function
     rather than on every return statement within the function. This
     can be used, for example, to quickly get to the end of a function
     you wish to bypass.
 
     NOTE: The /Or option is useful only when using CodeView; do not
           use it when generating code for a finished product.

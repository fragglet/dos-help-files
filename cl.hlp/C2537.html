     Compiler error C2537
 
     'identifier' : illegal linkage specification
 
     The indicated linkage specifier was not legal.
 
     This error can be caused by using a linkage specifier that is
     not supported. Only the "C" linkage specifier is supported.
 
     This error can also be caused by overloading more than one
     function with "C" linkage. Only one of a set of overloaded
     functions can be declared with "C" linkage.

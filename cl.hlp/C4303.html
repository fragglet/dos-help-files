     Compiler warning (level 4) C4303
 
     truncation during conversion of pointer to function to pointer
     to object
 
     A function pointer was converted to a smaller object or data
     pointer type. Information was lost in the casting.
 
     This condition is an error with the C compiler.
 
     This warning is generated by the following C++ example:
 
          void __far myfunc( void ) {};
 
          void * ptr;
          ptr = ( void * ) myfunc;

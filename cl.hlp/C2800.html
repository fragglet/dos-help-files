     Compiler error C2800
 
     operator 'operator' cannot be overloaded
 
     The specified operator was overloaded.
 
     The following operators cannot be overloaded: class member access
     operator (.), pointer to member operator (.*), scope resolution
     operator (::), conditional expression operator (?:), and sizeof
     operator.
 
     The following is an example of this error:
 
           class C
           {
              operator:: ();  // error :: is overloaded
           };

     Compiler error C2812
 
     second formal parameter required for based form of
     'operator delete'
 
     There were too few formal parameters for the operator delete for
     based pointers.
 
     The following are the prototypes for the based forms of the
     operator delete:
 
          void operator delete( __segment, void __based( void ) * );
          void operator delete( __segment, void __based( void ) *, \
                                 unsigned int );

     CL options are available to
 
        ■ Control the types of warnings generated at compile time
 
        ■ Help with syntax checking
 
        ■ Verify compatibility between the actual arguments and formal
          parameters of a function
 
        ■ Add information to the object file for use with the Microsoft
          CodeView debugger

     Compiler warning (level 1) C4014
 
     concatenating mismatched wide strings
 
     A wide string literal was concatenated with a standard string
     literal.
 
     Make sure that all wide string literals are prefixed with an
     "L" character, as in:
 
     char   hello[] = L"Hello, " "world"   // Warning
 
     char   hello[] = L"Hello, " L"world"  // OK

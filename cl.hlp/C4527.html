     Compiler warning (level 3) C4527
 
     instances of type 'class' can never be destroyed - user-defined
     destructor required
 
     The given derived class did not include a destructor, and the
     compiler could not generate one. The class cannot be destroyed.
 
     This warning can be avoided by including a user-defined destructor
     for the given class.

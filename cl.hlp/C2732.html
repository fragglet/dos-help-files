     Compiler error C2732
 
     linkage specification contradicts earlier specification for
     'function'
 
     The specified function was already declared with a different
     linkage specifier.
 
     This error can be caused by different linkage specifiers found
     in include files.
 
     Change the extern statements so that the linkages agree.
 
     The following is an example of this error:
 
          extern void func( void );      // implicit C++ linkage
          extern "C" void func( void );  // error

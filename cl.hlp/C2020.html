     Compiler error C2020
 
     'member' : 'class' member redefinition
 
     The specified member of the specified base class or structure was
     redefined.
 
     A function inherited from a base class or structure cannot be
     redefined.
 
     If the function is redefined in the derived class, it should be
     declared as virtual in the base class.

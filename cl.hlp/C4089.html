     Compiler warning (level 1) C4089
 
     'function' : different types in actual parameter 'number', formal
     parameter 'number'
 
     The actual parameter passed to the given function had a different
     type than the corresponding formal parameter.
 
     The actual parameter will be passed without change. The function
     will cast the actual parameter to the type specified in the
     function definition.

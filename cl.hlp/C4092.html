     Compiler warning (level 4) C4092
 
     sizeof returns 'unsigned long'
 
     The operand of the sizeof operator was very large, so the sizeof
     operator returned an unsigned long.
 
     This warning occurs only under the default /Ze Microsoft
     extensions. Under the /Za ANSI-compatibility option, this
     condition truncates the result of the sizeof operator.

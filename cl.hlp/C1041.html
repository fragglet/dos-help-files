     Fatal error C1041
 
     cannot open compiler intermediate file - no more files
 
     The compiler could not create intermediate files for use in the
     compilation process because no more file handles were available.
 
     To increase the number of file handles available under DOS, change
     the FILES setting in CONFIG.SYS. FILES=50 is the recommended
     setting.

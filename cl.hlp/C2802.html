     Compiler error C2802
 
     static member operator 'symbol' has no formal parameters
 
     The specified static member operator had a void formal parameter
     list.
 
     An operator declared by a static member function must take at
     least one parameter.
 
     The following is an example of this error:
 
          class A
          {
             static operator+ ();     // error, void parameter list
             static operator* ( A& ); // OK
          };

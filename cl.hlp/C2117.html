     Compiler error C2117
 
     'identifier' : array bounds overflow
 
     Too many initializers were present for the given array.
 
     Make sure that the array elements and initializers match in size
     and quantity.
 
     This error can be caused by lot leaving space for the null
     terminator in a string.
 
     The following is an example of this error:
 
          char abc[4] = "abcd"; // error, array contains 5 members

     Compiler error C2075
 
     'identifier' : array initialization needs curly braces
 
     There were no curly braces ({}) around the specified array
     initializer.
 
     The following is an example of this error:
 
          int i[] = 1, 2, 3 };     // error
          int i[] = { 1, 2, 3 };   // OK

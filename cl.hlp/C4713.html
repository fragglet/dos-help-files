     Compiler warning (level 1) C4713
 
     'function': INTERNAL COMPILER ERROR; restarting (compiler
     file 'file', line 'line'). Contact Microsoft Product Support
     Services
 
     The specified function could not be successfully compiled with
     global optimizations. The compiler has restarted the compilation
     without any global optimizations. The code generated is correct,
     but uses only local optimizations.
 
     Note the circumstances that generated this warning and inform
     Microsoft Product Support Services.

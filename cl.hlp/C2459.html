     Compiler error C2459
 
     'identifier' : is being defined; cannot add as an anonymous member
 
     The specified class, structure, or union was redefined in its own
     scope by a member of an anonymous union.
 
     The following is an example of this error:
 
          class C
          {
              union { int C; };   // error
          };

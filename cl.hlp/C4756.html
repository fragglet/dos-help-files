     Compiler warning (level 2) C4756
 
     overflow in constant arithmetic
 
     The compiler generated an exception while doing constant
     arithmetic during compilation.
 
     For example:
 
          float fp_val = 1.0e100;
 
     In this example, the floating-point constant 1.0e100 exceeds the
     maximum allowable value for a double-precision data item.

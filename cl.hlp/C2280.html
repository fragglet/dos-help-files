     Compiler error C2280
 
     missing '{' before identifier 'identifier'?
 
     The expression caused a syntax error that was probably due to
     a missing opening bracket ({).
 
     The following is an example of this error:
 
         int int_var;
 
         void func( int a )
                              // missing opening bracket
             int_var = 1;     // error is called on this line
         }

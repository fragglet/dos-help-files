     Compiler warning (level 1) C4054
 
     function pointer cast to a data pointer
 
     A function pointer was cast to a data pointer.
 
     Make sure that the correct cast is being used.
 
     This condition generates an error with the C compiler with the /Za
     ANSI-compatibility command-line option or under the medium memory
     model.

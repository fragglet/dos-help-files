     Compiler error C2566
 
     overloaded function in a conditional expression
 
     The overloaded function in a conditional expression could not be
     evaluated.
 
     The following is an example of this error:
 
          int f( int );
          int f( double );
 
          void main()
          {
             if( f );      // error, which f?
             if( f(1) );   // OK
          }

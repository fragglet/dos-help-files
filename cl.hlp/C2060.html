     Compiler error C2060
 
     syntax error : end-of-file found
 
     At least one more token was expected.
 
     Causes of this error include omitting a semicolon (;), as in:
 
          int *p    // error
 
     or omitting a closing brace (}) from the last function, as in:
 
          main()
          {         // error

     Failed to add Options menu item(s)
 
     While the PWBC extension loaded, the extension was unable to add
     one or more of its commands to the Options menu.
 
     Possible causes for this problem include:
 
       ■  The capacity of the Options menu was been exceeded.
 
          Load fewer extensions.
 
       ■  There was not enough available memory to add the menu
          commands.
 
          Load fewer extensions, or make more memory available to PWB.

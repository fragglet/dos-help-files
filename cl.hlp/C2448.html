     Compiler error C2448
 
     'identifier' : function-style initializer appears to be a function
     definition
 
     The specified function definition was incorrect.
 
     This error can be caused by using the old-style C-language formal
     list.
 
     The following is an example of this error:
 
          void func(c)
             int c;
          {}             // error

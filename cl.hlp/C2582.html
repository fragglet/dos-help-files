     Compiler error C2582
 
     'identifier' : 'operator=' function is unavailable
 
     No assignment operator, operator=(), was defined for the
     specified class.
 
     If any assignment operator has been defined that takes the class
     as a parameter, then a default assignment operator will not be
     generated by the compiler. The assignment operator must be
     explicitly defined for each class and is not inherited by derived
     classes.

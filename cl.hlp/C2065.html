     Compiler error C2065
 
     'identifier' : undeclared identifier
 
     The specified identifier was not declared.
 
     A variable's type must be specified in a declaration before it
     can be used. The parameters that a function uses must be
     specified in a declaration before the function can be used.
 
     This error can be caused if an include file containing the
     required declaration was omitted.

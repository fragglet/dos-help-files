     Compiler error C2572
 
     'identifier1::identifier2' : redefinition of default parameter :
     parameter 'identifier3'
 
     The specified default parameter in a member function was redefined.
 
     A default parameter cannot be redefined. If another value for the
     parameter is required, then the default parameter should be left
     undefined.

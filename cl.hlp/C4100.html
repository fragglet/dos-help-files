     Compiler warning (level 3 and 4) C4100
 
     'identifier' : unreferenced formal parameter
 
     The given formal parameter was never referenced in the body of the
     function for which it was declared.
 
     With a C++ program, this warning is level 3. Under the /Za ANSI-
     compatibility command-line option, this is a level 4 warning.
 
     This warning is informational.

     Command-line warning D4022
 
     option 'option' invalid for 'number'-bit target
 
     The given command-line option was not available when generating
     code for this size target. The option was ignored.
 
     This warning can be caused by using the /G3 command-line option
     from the 16-bit targeted compiler.

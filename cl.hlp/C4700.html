     Compiler warning (level 1) C4700
 
     local variable 'identifier' used without having been initialized
 
     A reference was made to a local variable that had not been
     assigned a value. As a result, the value of the variable is
     undefined.
 
     This warning is given only when compiling with the /Oe global
     register allocation command-line option.

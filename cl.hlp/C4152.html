     Compiler warning (level 4) C4152
 
     non standard extension, function/data ptr conversion in expression
 
     A pointer to a function was converted to a pointer to data, or
     visa versa.
 
     This conversion is allowed when the Microsoft extensions (/Ze)
     are enabled. However, ANSI C does not allow this conversion.

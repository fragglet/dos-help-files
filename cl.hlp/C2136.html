     Compiler error C2136
 
     'function' : prototype must have parameter types
 
     A function prototype declaration had formal parameter names, but no
     types were provided for the parameters.
 
     A formal parameter in a function prototype must either have a type
     or be represented by an ellipsis (...) to indicate a variable
     number of parameters and no type checking.
 
     One cause of this error is a misspelling of a type name in a
     prototype that does not provide the names of the formal
     parameters.

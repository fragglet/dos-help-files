     Compiler error C2273
 
     'type' : illegal as right side of '->' operator
 
     The given type was specified on the right hand side of the class
     member access operator (->).
 
     To access a user-defined type conversion, use the operator
     keyword between the -> operator and the type.
 
     The following is an example of this error:
 
          i = ClassPtr->int( a );           // error
          i = ClassPtr->operator int( a );  // OK

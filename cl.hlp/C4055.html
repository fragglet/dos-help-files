     Compiler warning (level 1) C4055
 
     data pointer cast to a function pointer
 
     A data pointer was cast to a function pointer.
 
     Make sure that the correct cast is being used.
 
     This condition generates an error with the C compiler with the /Za
     ANSI-compatibility command-line option or under the compact memory
     model.

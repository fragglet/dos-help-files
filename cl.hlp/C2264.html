     Compiler error C2264
 
     'function' : error in function definition or declaration; function
     not called
 
     Since specified function was incorrectly defined or declared, it
     could not be called.
 
     The following is an example of this error:
 
          class C
          {
          public:
             operator int( int = 10 );    // incorrect declaration here
          };
 
          void main()
          {
             int i;
             C c;
 
             i = c;                       // error
          }

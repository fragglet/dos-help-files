     Compiler error C2173
 
     'function' : actual parameter is not a pointer : parameter 'number1',
     parameter list 'number2'
 
     An attempt was made to pass a nonpointer parameter to a function
     that expected a pointer.
 
     This error occurs in calls that return a pointer to a function.
     The first number indicates which parameter was in error; the
     second number indicates which parameter list contained the invalid
     parameter.

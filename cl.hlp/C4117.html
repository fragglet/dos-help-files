     Compiler warning (level 1) C4117
 
     macro name 'name' is reserved; 'command' ignored
 
     The given command attempted to define or undefine the predefined
     macro <name> or the preprocessor operator defined.
 
     The given <command> is displayed in the warning message as either
     #define or #undef, even if the attempt was made using command-line
     options.
 
     The command was ignored.

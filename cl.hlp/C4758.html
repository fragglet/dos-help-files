     Compiler warning (level 1) C4758
 
     address of automatic (local) variable taken; DS != SS
 
     The program was compiled with the default data segment (DS) not
     equal to the stack segment (SS), and the program tried to point to
     an automatic (local) variable with a near pointer.
 
     Dereferencing a pointer to that address is illegal when DS != SS
     and will give an unpredictable result.

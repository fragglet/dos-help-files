     Compiler error C2613
 
     trailing ',' illegal in base class list
 
     A comma (,) was placed after the last base in a base class list.
 
     The following is an example of this error:
 
          class A {};
          class B : public A, {};  // error, extra comma
          class C : public A {};   // OK

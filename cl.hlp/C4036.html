     Compiler warning (level 1) C4036
 
     unnamed 'type' as actual parameter
 
     The type of the structure, union, enumerated type, or class being
     passed as an actual parameter was not given.
 
     The formal parameter in the generated function prototype will be
     commented out.
 
     This warning occurs only when using the /Zg option for generating
     function prototypes and can be avoided by providing the appropriate
     type explicitly.

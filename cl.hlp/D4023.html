     Command-line warning D4023
 
     option 'option' forces use of optimizing compiler
 
     The specified compiler option is incompatible with the fast
     compile (/f) option.
 
     The following compiler options cause the optimizing compiler to
     be used:
 
          /FPa, /FPc, /Fa, /Fs, /St, /Ss, /Sl, /Sp,
 
     This warning can be suppressed by explicitly requesting the
     optimizing compiler, using the /f- option.

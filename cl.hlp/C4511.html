     Compiler warning (level 3) C4511
 
     'class' : copy constructor could not be generated
 
     The compiler was unable to generate a copy constructor for
     the given class. No constructor was created.
 
     This warning can be caused by having a copy constructor for
     the base class that is not accessible by the derived class. An
     inherited copy constructor from a base class with a different
     ambient memory model can also cause this warning.
 
     This warning can be avoided by specifying a user-defined
     copy constructor for the class.

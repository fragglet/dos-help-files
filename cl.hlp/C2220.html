     Compiler error C2220
 
     warning treated as error - no object file generated
 
     When the compiler option /WX is used, the first warning generated
     by the compiler causes this error message to be displayed.
 
     Either correct the condition that caused the warning or compile
     at a lower warning level or without /WX.

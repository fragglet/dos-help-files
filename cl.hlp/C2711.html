     Compiler error C2711
 
     cannot delete a pointer to a function
 
     A pointer to a function was illegally deleted using the delete
     operator.
 
     The following is an example of this error:
 
          void (*pf)();
          void func();
 
          void main()
          {
             pf = func;
             delete pf;   // error, pf points to a function
          }

     Compiler error C2615
 
     memory attribute must appear before class name in pointer to
     member
 
     The memory model attribute for a pointer to a member of a class
     was specified after the name of the class in a declaration.
 
     The following is an example of this error:
 
          class C
          {
             int mem;
          };
 
          int (C ::__far*pcmem);    // error

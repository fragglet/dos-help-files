     Compiler error C2277
 
     'identifier' : cannot take the address of a constructor
 
     It is not possible to take the address of a constructor.
 
     The following is an example of this error:
 
          class A
          {
          public:
             A();
          };
 
          (*pctor)() = &A::A;   // error

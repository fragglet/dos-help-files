     Compiler warning (level 1) C4021
 
     'function' : too few actual parameters
 
     The number of actual parameters specified in a function call was
     less than the number of formal parameters specified in the
     function prototype or function definition.
 
     Only the provided actual parameters were passed. If the called
     function references a variable that was not passed, the results
     are undefined.

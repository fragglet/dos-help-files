     Compiler error C2222
 
     '->' : left operand has class/struct/union type, use '.'
 
     The left operand of the class member access operator (->) must be
     a pointer to a class (or structure or union) type but not a class
     type.
 
     The member-of operator (.) can be used with a class, structure,
     or union type.

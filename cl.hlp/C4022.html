     Compiler warning (level 1) C4022
 
     'function' : pointer mismatch for actual parameter 'number'
 
     The pointer type of the given actual parameter was different from
     the pointer type specified in the formal parameter list or function
     definition.
 
     The actual parameter was passed without change. Its value will be
     interpreted as a pointer within the called function.

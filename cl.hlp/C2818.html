     Compiler error C2818
 
     incorrect return type for 'operator ->'
 
     A redefinition of the class member access operator (->) contained
     a return statement that did not return an appropriate type.
 
     The class member access operator must return one of the following:
 
       ■ A pointer to a class, structure, or union
 
       ■ A reference to a class, structure, or union where the ->
         operator is defined
 
       ■ An instance of a class, structure, or union where the ->
         operator is defined

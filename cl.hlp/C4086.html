     Compiler warning (level 1) C4086
 
     expected '1', '2', '4', or '8'
 
     The pragma expected a parameter of either 1, 2, 4, or 8, but the
     specifed parameter was not recognized or was missing.
 
     The pack pragma will not accept an 8 when generating 16-bit code.
 
     The following example generates this warning:
 
          #pragma pack( 3 )

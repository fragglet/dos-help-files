     Compiler warning (level 1) C4039
 
     ambient class modifier on reference ignored
 
     The ambient class modifier on a reference to a class, structure,
     or union was ignored.
 
     An ambient class modifier can be used only in the declaration and
     definition of a class, structure, or union.

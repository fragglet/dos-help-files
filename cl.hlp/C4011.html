     Compiler warning (level 1) C4011
 
     'identifier' : identifier was truncated to 'number' characters
 
     The name of the specified identifier was too long.
 
     The maximum length of a linkable external identifier is 247
     characters, after name decoration.
 
     The default maximum length of an undecorated name or user-entered
     fragment of a decorated name is 32 characters. The /H compiler
     switch can be used to set a different maximum length.

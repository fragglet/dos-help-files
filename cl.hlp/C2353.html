     Compiler error C2353
 
     'constructor' : improper use of constructor initializers
 
     The constructor initializer syntax was incorrect.
 
     This error can be caused by omitting the constructor definition
     from a constructor initializer.
 
     The following is an example of this error:
 
          class C
          {
          public:
             C() : i( 10 );   // error, missing definition
             C() : i( 10 ){}; // OK
          private:
             int i;
          };

     Compiler warning (level 1) C4095
 
     expected ')'; found 'token'
 
     More than one argument was given for a pragma that can take only
     one argument.
 
     The compiler assumed the expected right parenthesis and ignored
     the remainder of the line.
 
     The following example generates this warning:
 
          #pragma skip( 10, 18, 67 )

     Command-line warning D4011
 
     preprocessing overrides source listing
 
     A source listing was not generated because a preprocessor listing
     option was specified.
 
     To generate both a source listing and a preprocessor listing, run
     CL twice: once with the /Fs (source listing) option and once with
     the /E, /P, or /EP (preprocessed listing) option.
 
     Check the CL or FL environment variable for option specifications.

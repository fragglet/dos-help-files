     Compiler error C2171
 
     'operator' : illegal operand
 
     The given unary operator was used with an illegal operand type, as
     in the following example:
 
          int (*fp)();
          double d, d1;
          ∙ ∙ ∙
          fp++;           // error
          d = ~d1;        // error

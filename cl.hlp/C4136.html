     Compiler warning (level 3) C4136
 
     conversion between different floating-point types
 
     Information was lost or truncated in the conversion between
     two floating-point types.
 
     For example, the following code generates this warning:
 
          double doublevar;
          float floatvar;
          floatvar = doublevar;
 
     Note that unsuffixed floating-point constants have type double, so
     the following code generates this warning:
 
          floatvar = 1.0;
 
     If the floating-point constant should be treated as type float,
     use the F (or f) suffix on the constant to prevent the warning:
 
          floatvar = 1.0F;

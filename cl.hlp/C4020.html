     Compiler warning (level 1) C4020
 
     'function' : too many actual parameters
 
     The number of actual parameters specified in a function call was
     greater than the number of formal parameters specified in the
     function prototype or function definition.
 
     The extra actual parameters were passed according to the calling
     convention used on the function.

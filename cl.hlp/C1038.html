     Fatal error C1038
 
     compiler limit : 'function' : control flow state too complex;
     simplify function
 
     The number of possible control flow states in the specified
     function exceeded the limit imposed by the compiler. The flow
     control of the function should be simplified or the function
     should be split into smaller functions.

     Compiler error C2283
 
     'identifier' : pure specifier not allowed on un-named class-key
 
     The specified member function of an unnamed class or structure
     was illegally declared with a pure specifier.
 
     The following is an example of this error:
 
          struct
          {
             virtual void func() = 0;   // error
          };
 
          struct S
          {
             virtual void func() = 0;   // OK
          };

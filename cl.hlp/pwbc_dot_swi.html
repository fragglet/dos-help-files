     The PWBC (C and C++ Compiler Options) extension provides the
     following switches to set the library naming convention:
 
     Switch          Libraries
 
     Doslibs         DOS (real mode) libraries
     Winlibs         Windows libraries
     Windlllibs      Windows DLL libraries
     Os2libs         OS/2 (protected mode) libraries
     Dlllibs         OS/2 DLL libraries
     Mtlibs          OS/2 multithread libraries
 
     See: ◄PWBC Library Switches►
 
     The following switches control C auto-indentation:
 
     Switch        Description
 
     ◄C_softcr►      Turns C auto-indentation on or off.
     ◄C_suffixes►    Specifies the file extensions for C indentation.
 
     PWBC switches must be placed in the [PWB-PWBC] tagged section of
     TOOLS.INI. To set any of the PWBC switches, choose PWBC from the
     Switch Owner box in the Options Editor Settings dialog box.
     See: ◄Options Editor Settings►

     Compiler error C2272
 
     'function' : modifiers not allowed on static member functions
 
     The specified static member function was declared with a memory-
     model specifier.
 
     The following is an example of this error:
 
          class C
          {
          public:
            static void func1() __far;  // error, func1 is static
            void func2() __far;         // OK
          };

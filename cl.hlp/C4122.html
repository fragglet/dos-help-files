     Compiler warning (level 1) C4122
 
     'function' : alloc_text applicable only to functions with C linkage
 
     The alloc_text pragma can be applied only to functions declared
     with extern c. This pragma cannot be used with external C++
     functions.
 
     The alloc_text pragma was ignored.

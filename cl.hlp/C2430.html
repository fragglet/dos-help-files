     Compiler error C2430
 
     more than one index register in 'identifier'
 
     More than one of the specified registers were scaled.
 
     The 32-bit targeted compiler supports scaled indexing, but you
     can only scale one register.
 
     The following is an example of this error:
 
          _asm mov eax, [ebx*2+ecx*4]

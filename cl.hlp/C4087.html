     Compiler warning (level 1) C4087
 
     'function' : declared with 'void' parameter list
 
     The given function was declared with no formal parameters, but a
     call to the function specified actual parameters.
 
     The extra parameters were passed according to the calling
     convention used on the function.
 
     The following example generates this warning:
 
          int f1( void );
          f1( 10 );

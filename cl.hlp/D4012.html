     Command-line warning D4012
 
     function declarations override source listing
 
     A source listing was not generated because a function prototype
     listing was requested.
 
     To generate both a source listing and a function prototype
     declaration listing, run CL twice: once with the /Fs (source
     listing) option and once with the /Zg (function prototype
     listing) option.
 
     Check the CL or FL environment variable for option specifications.

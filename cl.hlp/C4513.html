     Compiler warning (level 3) C4513
 
     'class' : destructor could not be generated
 
     The compiler was unable to generate a default constructor for
     the given class. No constructor was created.
 
     This warning can be caused by having an assignment operator for
     the base class that is not accessible by the derived class. An
     inherited assignment operator from a base class with a different
     ambient memory model can also cause this warning.
 
     This warning can be avoided by specifying a user-defined
     destructor for the class.

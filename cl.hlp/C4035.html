     Compiler warning (level 3) C4035
 
     'function' : no return value
 
     A function did not include a return statement.
 
     This warning is generated for all functions (including main) that
     do not have return statements. Functions with a return type of
     void will not generate this warning.
 
     An undefined value will be returned when this function is called.

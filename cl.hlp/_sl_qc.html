─────C/C++ Compiler─────────────────────────────────────────────────────────
 
     The Quick Compile (/qc) option is supported for compatibility with
     Microsoft C version 6.0. The Fast Compile (/f) option supersedes
     the /qc option in Microsoft C/C++. CL defines the _QC constant for
     the /qc option.

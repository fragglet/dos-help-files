     Compiler error C2174
 
     'function' : actual parameter has type 'void' : parameter 'number1',
     parameter list 'number2'
 
     An attempt was made to pass a void parameter to a function. Formal
     parameters and parameters to functions cannot have type void. They
     can, however, have type void* (pointer to void).
 
     This error occurs in calls that return a pointer to a function.
     The first number indicates which parameter was in error; the
     second number indicates which argument list contained the invalid
     parameter.

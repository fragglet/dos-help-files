     Compiler warning (level 4) C4132
 
     'object' : const object should be initialized
 
     The constant was not initialized.
 
     Since the value of a symbol with the const attribute cannot be
     changed after its definition, it should be given an initialization
     value.
 
     It will not be possible to assign a value to <object>.

     Compiler warning (level 1) C4151
 
     'operator' : operator should be explicitly 'model'
 
     The new operator was overloaded, but there was not an appropriate
     function for either __based or __huge.
 
     To avoid this warning, write versions of the new operator to cover
     this case.

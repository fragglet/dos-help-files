     Compiler error C2039
 
     'identifier' : is not a member of 'class-key'
 
     A nonmember of a structure or union was incorrectly used.
 
     The following is an example of this error:
 
          struct S
          {
             int mem0;
          } *pS;
 
          void main()
          {
             pS->mem1 = 0;       // error, mem1 is not a member
             pS->mem0 = 0;       // OK
          }

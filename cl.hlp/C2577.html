     Compiler error C2577
 
     'identifier' : destructor cannot return a value
 
     The specified destructor returned a value.
 
     A destructor cannot return a value of any type. This error is
     caused by defining a destructor that returns a value of any type,
     including a void return type.
 
     This error can be eliminated by removing the return statement
     from the destructor definition.

     Compiler warning (level 1) C4401
 
     'identifier' : member is bit field
 
     An attempt was made to access the given bit-field member within
     inline assembler code.
 
     The identifier is a bit field. A bit-field member cannot be accessed
     within inline assembler code. The last packing boundary before
     the bit-field member was used.
 
     To avoid this warning, cast the bit field to an appropriate type
     before making the reference in inline assembler code.

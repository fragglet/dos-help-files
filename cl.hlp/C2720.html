     Compiler error C2720
 
     'identifier' : 'specifier' storage class specifier illegal on
     members
 
     A storage class was illegally specified for the given identifier.
 
     The following is an example of this error:
 
          struct S
          {
             static int i;
          };
          static S::i;   // error

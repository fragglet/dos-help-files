     Compiler error C2221
 
     '.' : left operand points to class/struct/union. use '->'
 
     The left operand of the member-of operator (.) must be a class,
     (or structure or union) but not a pointer to a class type.
 
     The class member access operator (->) can be used with a pointer
     to a class, structure, or union.

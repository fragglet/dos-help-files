     C_softcr (Boolean)
 
     The C_softcr switch determines if PWBC handles automatic
     indentation for files with the extensions listed in the
     C_suffixes switch. This switch is used only when the PWB
     Softcr switch is set to yes.
     See: ◄C_suffixes► ◄Softcr►
 
     The setting for this switch appears in the [PWB-PWBC] tagged
     section in TOOLS.INI. To set any PWBC switch, choose PWBC from the
     Switch Owner box in the Options Editor Settings dialog box.
     See: ◄Options Editor Settings►
 
     Syntax
 
     C_softcr: { yes │ no }
 
     yes    Turns on indentation performed by the PWBC extension.
 
     no     Turns off indentation performed by the PWBC extension.
 
     Default
 
     C_softcr: yes

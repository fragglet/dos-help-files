     Compiler error C2612
 
     trailing ',' illegal in base/member initializer list
 
     A comma (,) was placed after the last base or member in an
     initializer list.
 
     The following is an example of this error:
 
          class A
          {
          public:
             int i;
             A( int ia ) : i( ia ), {}; // error, extra comma
          };
 
          class B
          {
          public:
             int i;
             B( int ib ) : i( ib ) {};  // OK
          };

     Compiler error C2147
 
     'identifier' : const array must be fully initialized
 
     The specified array was declared as const but one or more
     elememts were not initialized.
 
     All elements of a const array must be initialized when it is
     defined.
 
     The following is an example of this error:
 
          const int arr[10] = { 1, 2}; // error

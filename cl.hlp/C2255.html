     Compiler error C2255
 
     'function' : a friend function can only be declared in a class
 
     The specified function was declared with the friend specifier
     outside of a class, structure, or union.
 
     The following is an example of this error:
 
          class A
          {
          private:
             void func1();
             friend void func2();
          };
 
          friend void func1() {};    // error
          void func2() {};           // OK

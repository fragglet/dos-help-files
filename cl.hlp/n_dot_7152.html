     Project files required for precompiled header
 
     A request was made to use precompiled headers, but the project file
     list is empty.
 
     Because the precompiled header is derived from the first C or C++
     source file in the project, you must add at least one source file
     to the project file list before you can turn on the use of
     precompiled headers.
 
     See: ◄Creating a Project with Precompiled Headers►

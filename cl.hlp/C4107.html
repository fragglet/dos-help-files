     Compiler warning (level 1) C4107
 
     pragma requires an integer between 15 and 255
 
     An integer constant between 15 and 255 must be specified with the
     pagesize pragma.
 
     The compiler assumed a value of 63.
 
     The following example generates this warning:
 
          #pragma pagesize( 10 )

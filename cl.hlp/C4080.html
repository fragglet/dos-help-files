     Compiler warning (level 1) C4080
 
     expected identifier for segment name; found 'token'
 
     The first actual parameter given to the alloc_text pragma was
     missing a segment name. This happens if the first token in the
     argument list was not an identifier.
 
     The pragma was ignored.

     Compiler error C2282
 
     'identifier1' is followed by 'identifier2'. (missing ','?)
 
 
     The declaration of the specified identifiers used incorrect
     syntax, probably due to a missing comma.
 
     The following is an example of this error:
 
          int *f g;          // error

     Compiler error C2635
 
     cannot convert an 'identifier1*' to an 'identifier2*'; conversion
     from a virtual base class is implied
 
     The specified conversion required a cast from a virtual base class
     to a derived class.
 
     Casting from a virtual base class to a derived class is not
     allowed.

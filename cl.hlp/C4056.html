     Compiler warning (level 2) C4056
 
     overflow in floating point constant arithmetic
 
     The compiler generated an exception while doing floating point
     constant arithmetic during compilation.
 
     For example:
 
          float fp_val = 1.0e400;
 
     In this example, the floating-point constant 1.0e400 exceeds the
     maximum allowable value for a double-precision data item.

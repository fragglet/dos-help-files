     Compiler error C2099
 
     nonconstant initializer
 
     The initializer was not a constant.
 
     The following is an example of this error when compiled as C:
 
          int i, j;
          int *p;
          j = i();    // error, i() is not constant
          j = *p;     // error, *p is not a constant

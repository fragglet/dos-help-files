     /Fo not allowed: use Set Build Directory
 
     The /Fo option to override the default object file name is not
     allowed for PWB projects.
 
     To specify a different directory for object files and the target
     executable file, use the Build Options command on the Options menu
     and specify a directory in the Build Directory text box.
     See: ◄Options Menu, Build Options►

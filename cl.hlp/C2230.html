     Compiler error C2230
 
     'operator' : indirection to different types
 
     The pointer expressions used with the given operator had different
     base types.
 
     The pointer expressions were used without conversion. This causes
     a warning in ANSI C but is an error in C++.
 
     The following is an example of this error:
 
          struct ts1 *s1;
          struct ts2 *s2;
          s2 = s1;   // error

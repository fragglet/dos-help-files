     Compiler warning (level 1) C4023
 
     'function' : based pointer passed to unprototyped function :
     parameter 'number'
 
     There was an attempt to pass a based pointer to an unprototyped
     function.
 
     When using a memory model with near data, only the offset portion
     of a based pointer is passed to an unprototyped function. If the
     function expects a far pointer, the resulting code will be wrong.
 
     In all memory models, if the function is defined to take a based
     pointer with a different base, the resulting code may be
     unpredictable.
 
     This warning can be avoided by using a function prototype
     containing a reference to the proper base.

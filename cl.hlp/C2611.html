     Compiler error C2611
 
     'token' : illegal following '~' (expected identifier)
 
     The specified token was not an identifier.
 
     The following is an example of this error:
 
          class C
          {
             C::~operator int();    // error
             ~C();                  // OK, destructor declaration
          };

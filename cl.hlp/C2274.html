     Compiler error C2274
 
     'type' : illegal as right side of '.' operator
 
     The given type was specified on the right side of the class
     member access operator (.).
 
     To access a user-defined type conversion, use the operator keyword
     between the dot operator (.) and the type.
 
     The following is an example of this error:
 
          i = ClassName.int( a );           // error
          i = ClassName.operator int( a );  // OK

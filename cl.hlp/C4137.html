     Compiler warning (level 1) C4137
 
     'function' : no return value from floating-point function
 
     The given function had no return statement.
 
     A long double function returns its value on the floating-point
     stack or the emulation stack. If the function does not return a
     value, a run-time floating-point stack underflow error may occur.
 
     Make sure that all floating-point functions return a floating-point
     value.

     Compiler error C2602
 
     'class::identifier' is not a member of a base of 'class'
 
     An access declaration was made for the specified identifier, but
     it is not a member of a base class.
 
     A member of a class or structure cannot be accessed in another
     class that is not derived from the member's class.
 
     The following is an example of this error:
 
          struct X
          {
             int x;
          };
 
          struct A
          {
             int a;
          };
 
          struct B : public A
          {
             X::x;       // error, B is not derived from X
             A::a;       // OK
          };

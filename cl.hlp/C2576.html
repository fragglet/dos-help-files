     Compiler error C2576
 
     'identifier' : virtual used for static member function
 
     A static member function was declared as virtual.
 
     The virtual function mechanism relies on the specific object
     that calls the function to determine which virtual function is
     used. Since this is not possible for static functions, they
     cannot be declared as virtual.

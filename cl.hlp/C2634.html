     Compiler error C2634
 
     '&class::member' : pointer to reference member is illegal
 
     A pointer to a reference member was declared.
 
     The following is an example of this error:
 
          int mem;
 
          struct S
          {
             S() : rf(mem) { }
             int &rf;
          };
 
          int (S::*pdm) = &S::rf;  // error

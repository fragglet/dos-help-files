     Compiler error C2530
 
     'identifier' : references must be initialized
 
     A reference was not initialized when it was declared.
 
     The following cases are the only times a reference can be
     declared without initialization:
 
        ■ It is declared with the keyword extern.
 
        ■ It is a member of a class, structure, or union and is
          initialized in the class's constructor function.
 
        ■ It is declared as a parameter in a function declaration
          or definition.
 
        ■ It is declared as the return type of a function.

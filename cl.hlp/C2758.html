     Compiler error C2758
 
     'const' : must be initialized in constructor base/member
     initializer list
 
     The given constant was not initialized with an initializer list
     in the object constructor. The compiler left the constant
     undefined.
 
     If a const or reference member variable is not given a value when
     it is initialized, it must be given a value in the object
     constructor.

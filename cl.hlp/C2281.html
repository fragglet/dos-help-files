     Compiler error C2281
 
     'identifier1' : is not a function, but contains <function
     returning>. 'identifier2' is unexpected.
 
 
     The declaration of the specified identifier used incorrect
     syntax.
 
     The following is an example of this error:
 
          int (*f)() a;      // error

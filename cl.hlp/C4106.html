     Compiler warning (level 1) C4106
 
     pragma requires an integer between 1 and 127
 
     An integer constant between 1 and 127 must be specified with the
     page and skip pragmas.
 
     The compiler assumed a value of 1.
 
     The following example generates this warning:
 
          #pragma skip( 0 )

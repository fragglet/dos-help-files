     Compiler error C2854
 
     syntax error in #pragma hdrstop
 
     The #pragma hdrstop statement gave an invalid filename.
 
     The hdrstop pragma is followed by an optional filename
     enclosed in parentheses and quotation marks, as in:
 
          #pragma hdrstop( "source\pchfiles\myheader.pch" )
 
     The precompiled header filename cannot be enclosed in angle
     brackets (<>).

     Fatal error C1048
 
     unknown option 'character' in 'option'
 
     The given character was not a valid letter for the option.
 
     For example, the following line:
 
          #pragma optimize( "*", on )
 
     causes the following error:
 
          unknown option '*' in '#pragma optimize'

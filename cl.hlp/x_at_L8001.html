─────C/C++ Compiler─────────────────────────────────────────────────────────
 
     The CL command accepts C and C++ source files as input. It
     executes the component parts of the compiler, then links the
     resulting object code to produce an executable file. By specifying
     options to CL, you can control the behavior of the compiler and
     the linker.

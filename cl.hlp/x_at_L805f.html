─────C/C++ Compiler─────────────────────────────────────────────────────────
 
     Option   Function
 
     ◄/Zi►    Adds symbolic information to the object file for use with
              the Microsoft CodeView debugger.
 
     ◄/Zd►    Creates an object file containing only public symbols
              (global/external) and line numbers. This is useful
              if you want to save object/memory space for CodeView.
 
     ◄/Od►    Disables code optimization to help with debugging.
 
     ◄/Or►    Disables inline returns. This option can be helpful when
              using CodeView to set a breakpoint on the closing curly
              brace (}) of a function.

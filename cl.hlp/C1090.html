     Fatal error C1090
 
     data allocation exceeds 64K
 
     The size of a segment exceeded 64K.
 
     In a 16-bit program, segments cannot exceed 64K. This error occurs
     with __based allocation.
 
     Choose a segment that has more space available, or use the huge
     memory model.

     Compiler warning (level 3) C4013
 
     'function' undefined; assuming extern returning int
 
     An undefined function was called. The compiler assumed an external
     function that returned an int type.
 
     Make sure that the function name is spelled correctly. All external
     functions should be prototyped as extern.

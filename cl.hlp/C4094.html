     Compiler warning (level 2) C4094
 
     untagged 'token' declared no symbols
 
     The compiler detected an empty declaration using an untagged
     structure, union, or class.
 
     The declaration was ignored.
 
     The following example generates this warning:
 
          struct  { };
 
     This condition generates an error with the /Za ANSI-compatibility
     command-line option.

     Compiler error C2663
 
     'function' : 'number' overloads have no legal conversion for
     'this' pointer
 
     The required conversion was not defined for the this pointer to
     the specified overloaded functions.
 
     A user-defined conversion operator may be required.
 
     The following is an example of this error:
 
          class __far C
          {
          public:
             void func() __near;
             void func( int ) __near;
          } c;
 
          void main()
          {
             c.func();  // error
          }

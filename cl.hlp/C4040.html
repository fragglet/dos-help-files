     Compiler warning (level 1) C4040
 
     memory attribute on 'identifier' ignored
 
     The __near, __far, __huge, or __based keyword has no effect in the
     declaration of the given identifier and was ignored.
 
     One cause of this warning is a huge array that is not declared
     globally. Declare huge arrays outside of the main function.

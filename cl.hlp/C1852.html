     Fatal error C1852
 
     'filename' is not a valid precompiled header file
 
     The given filename is not a precompiled header file.
 
     Make sure that all /Yu command-line option and #pragma hdrstop
     statements specify valid precompiled header files. The compiler
     assumes the .PCH extension if none is provided.
 
     This error is caused by giving the filename of a file that
     is not a precompiled header, such as an .HPP file.

     Compiler warning (level 1) C4305
 
     truncation during conversion of integral type to pointer
 
     An integral type was converted to a smaller pointer type.
     Information was lost in the casting.
 
     This warning is generated by the following example:
 
          long   j;
          long * k;
 
          k = ( long * ) j;

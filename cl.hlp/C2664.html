     Compiler error C2664
 
     'function' : cannot convert parameter 'number' from 'type1' to
     'type2'
 
     The specified parameter of the specified function could not be
     converted to the required type.
 
     An explicit conversion may need to be supplied.
 
     The following is an example of this error:
 
          class A {} a;
 
          func( int, A );
 
          void main()
          {
             func( 1, 1 );  // error, no conversion from A to int
          }

     Compiler warning (level 1) C4652
 
     command-line option 'option' inconsistent with precompiled header;
     PCH option ignored
 
     The given command-line option differed from that given when the
     precompiled header (.PCH) was created. The option specified in the
     current command line was used.
 
     This warning can be avoided by regenerating the precompiled header
     with the given command-line option.

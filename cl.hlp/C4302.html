     Compiler warning (level 4) C4302
 
     truncation during conversion of pointer to function
 
     A function pointer was converted to a smaller function pointer
     type. Information was lost in the casting.
 
     This warning is generated by the following example:
 
          typedef void * ( *TYPE1 )( void );
          typedef void * ( __far *TYPE2 )( void );
 
          TYPE1   ptr1;
          TYPE2   ptr2;
 
          ptr1 = ( TYPE1 ) ptr2;

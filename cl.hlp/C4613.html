     Compiler warning (level 1) C4613
 
     'segment' : class of segment cannot be changed
 
     There was an attempt to create a segment with the same class name as
     a segment used by the compiler.
 
     No new segment class was created.
 
     This warning is generated only when compiling 32-bit programs.

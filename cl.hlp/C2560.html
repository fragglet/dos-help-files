     Compiler error C2560
 
     'function' : overloaded function differs only by calling
     convention/memory model
 
     The specified overloaded function declarations did not specify
     different formal parameter types. Overloaded functions cannot
     differ only by calling convention or memory model.
 
     The following is an example of this error:
 
          void _cdecl  func();
          void _pascal func();       // error
          void _pascal func( int );  // OK

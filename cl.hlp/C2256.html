     Compiler error C2256
 
     illegal use of friend specifier on 'destructor'
 
     The specified destructor was specified as a friend.
 
     A destructor cannot be specifed as a friend.
 
     The following is an example of this error:
 
          class C
          {
          public:
             friend ~C(); // error
             ~C();        // OK
          };

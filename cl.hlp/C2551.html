     Compiler error C2551
 
     'void *' type needs explicit cast to non-'void' pointer type
 
     A void pointer was assigned to a nonvoid pointer by implicit
     conversion.
 
     An explicit cast is necessary to convert a void pointer to a
     nonvoid pointer.

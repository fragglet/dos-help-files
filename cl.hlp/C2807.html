     Compiler error C2807
 
     second formal parameter to postfix 'operator symbol' must be 'int'
 
     The second parameter to the specified postfix operator must be
     declared to be of type int.
 
     The following is an example of this error:
 
          class X
          {
          public:
             X operator++ ( X );    // error, nonvoid parameter
             X operator++ ( int );  // OK, int parameter
          };

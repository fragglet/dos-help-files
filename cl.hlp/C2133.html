     Compiler error C2133
 
     'identifier' : unknown size
 
     An unsized array was declared as a member of a class, structure,
     union, or enumeration.
 
     Unsized member arrays are not allowed when the /Za (ANSI) switch
     has been chosen.
 
     The following is an example of this error:
 
          struct X
          {
             int a[0]; // error, unsized array
          };

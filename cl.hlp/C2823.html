     Compiler error C2823
 
     return type for based form of 'operator new' must be
     'void __based(void)*'
 
     A redefinition of the based form of the operator new contained
     a return statement that returned a type that was not a based
     pointer. The return value for the operator new must be
     void __based( void ) *.

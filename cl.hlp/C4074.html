     Compiler warning (level 1) C4074
 
     initializers put in compiler reserved initialization area
 
     The #pragma init_seg( compiler ) statement was used. The compiler
     initialization area is reserved by Microsoft.
 
     Code in this area may be executed before C run-time library
     initialization.

     Compiler error C2659
 
     'operator' : overloaded function as left operand
 
     An overloaded function was on the left side of the specified
     operator.
 
     The following is an example of this error:
 
          int func( int );
          int func( double );
 
          void main()
          {
             func = 10;  // error
          }

     Compiler warning (level 4) C4304
 
     truncation during pointer conversion
 
     A function pointer was converted to a smaller object or data
     pointer type. Information was lost in the casting.
 
     This warning is generated by the following example:
 
          int __far * j;
          int * k;
 
          k = ( int * ) j;

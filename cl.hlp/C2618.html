     Compiler error C2618
 
     'function' : inconsistent return statement
 
     The specified function did not have a return type declared, and
     a previous return statement had a different type.
 
     The function's return type should be declared.
 
     The following is an example of this error:
 
          int i;
          func()                   // no return type prototype
          {
             if( i ) return( 1 );  // int return value
             else return;          // error detected on this line
          }

     Compiler warning (level 1) C4610
 
     object 'class' can never be instantiated - user-defined constructor
     required
 
     The given class had neither user-defined nor default constructors;
     it cannot be instantiated. No instantiation was performed.
 
     Make sure that all classes have valid user-defined or default
     constructors.

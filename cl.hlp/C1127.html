     Fatal error C1127
 
     'segment': segment redefinition
 
     A segment was overwritten by another segment with the same name.
 
     For example, compiling in large model with:
 
          #pragma alloc_text( _TEXT, func1 )
 
     creates two segments, the default segment "module_TEXT" and the
     specified segment "_TEXT". However, in small model, the default
     segment is "_TEXT", and the specified segment "_TEXT" will
     overwrite the default segment.

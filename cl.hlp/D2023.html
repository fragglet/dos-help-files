     Command-line error D2023
 
     option 'option' forces use of the optimizing compiler
 
     The specified compiler option is incompatible with the fast
     compile (/f) option.
 
     The following compiler options cause the optimizing compiler
     to be used:
 
     /FPa, /FPc, /FPc87, /Fs, /St, /Ss, /Sl, /Sp
 
     This warning can be suppressed by explicitly requesting the
     optimizing compiler, using the /f- option.

     Compiler error C2803
 
     operator 'symbol' must have at least one formal parameter of
     class type
 
     The specified overloaded operator was declared without at least
     one class-type parameter.
 
     The following is an example of this error:
 
          int operator+( int, int );  // error
                                    -♦-

     Fatal error C1503
 
     default parameter definition missing ',' or ')'
 
     After one or more default parameters were defined, neither a comma
     to indicate continuation nor a closing parenthesis to indicate
     termination of the function declaration was found.
 
     Make sure that there are matching parentheses around the set of
     formal parameters.

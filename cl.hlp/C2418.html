     Compiler error C2418
 
     'identifier' : not in a register
 
     An inline assembler instruction referenced a variable with
     register storage class that was not actually allocated in a
     register.
 
     To correct this, remove the register keyword from the variable
     definition and make sure that this instruction is legal with a
     memory operand.

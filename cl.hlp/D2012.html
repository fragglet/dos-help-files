     Command-line error D2012
 
     too many linker arguments
 
     More than 128 options and object files were passed to the linker with
     the /link command-line option.
 
     This error can be caused by using filename wildcards (* and ?) to
     specify files for the linker. If you need to link a large number
     of .OBJ files, use the LIB utility to combine multiple object files.
 
     Check the CL or FL environment variable for option specifications.

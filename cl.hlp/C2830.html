     Compiler error C2830
 
     only placement parameters to 'operator new' can have default
     values
 
     The standard formal parameters for the operator new cannot be
     given default values. Only user-defined placement parameters can
     specify defaults.
 
     Since the compiler automatically passes values to the standard
     formal parameters, a default value is meaningless.

     Fatal error C1504
 
     type ambiguous after 'number' tokens
 
     The compiler could not resolve the type after looking ahead in the
     code.
 
     Simplify the code to make the statement clearer. This error can
     often be eliminated by using an explicit type cast to an ambiguous
     expression.

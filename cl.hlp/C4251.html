     Compiler warning (level 1) C4251
 
     'identifier' : base type 'type' should be exported to export
     'class-key' 'identifier'
 
     The specified base class was not declared with the __ export
     keyword.
 
     A base class or structure must be declated with the __export
     keyword if a function in a derived class is to be exported.

     Compiler error C2588
 
     '::~identifier' : illegal global destructor
 
     The specified destructor was not defined for a class, structure,
     or union.
 
     A destructor can be defined only for a class, structure, or
     union.
 
     This error can be caused by omitting the name of a class,
     structure, or union on the left side of the scope resolution
     operator (::).
 
     The following is an example of this error:
 
          ~F();  // error

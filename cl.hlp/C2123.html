     Compiler error C2123
 
     'function1' : cannot call __fastcall function 'function2' from
     p-code
 
     There was an attempt to call a fastcall function from within a
     p-code function.
 
     Rebuild <function2> with a different calling convention, or
     turn off p-code generation for <function1> by using #pragma
     optimize( "q", off ).

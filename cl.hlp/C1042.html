     Fatal error C1042
 
     cannot open compiler intermediate file - no such file or directory
 
     The compiler could not create intermediate files for use in the
     compilation process because the TMP environment variable was set
     to an invalid directory or path.
 
     Use the SET command to change the TMP environment variable so that
     it points to a valid directory.

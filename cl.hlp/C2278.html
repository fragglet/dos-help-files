     Compiler error C2278
 
     'function' : no function with C linkage found
 
     The specified function was not found.
 
     This error is caused by omitting the file containing the function
     definition from the project list or makefile, by not defining the
     function within the file scope, or by omitting the keyword extern
     in the function prototype.

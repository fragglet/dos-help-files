     Compiler error C2461
 
     'class' : constructor syntax missing formal parameters
 
     The constructor for the given class did not specify any
     formal parameters. The declaration of a constructor must
     specify a formal parameter list in parentheses. This list
     can be null.
 
     Add a pair of parentheses after the <class>::<class>
     identifier.
 
     The following is an example of this error:
 
          class C
          {
              C::C;        // error
              C::C();      // OK
          };

     Compiler error C2542
 
     'identifier' : class object has no constructor for initialization
 
     There was no constructor to initialize the specified object.
 
     A constructor with the same parameter list used in the
     initialization must be supplied.
 
     This error can also be caused by initializing an object with
     incorrect parameters.

     Compiler warning (level 4) C4710
 
     'function' : function not expanded
 
     The given function was selected for inline expansion, but the
     compiler did not perform the inlining.
 
     Inlining is performed at the compiler's discretion. The inline
     keyword, like the register keyword, is used as a hint for the
     compiler.

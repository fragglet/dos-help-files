     Fatal error C1040
 
     unexpected end-of-file condition in source file 'filename'
 
     The compiler detected an unexpected end-of-file condition while
     creating a source listing or mixed source and object listing.
 
     This error occurs if the source file is deleted or overwritten
     while it is being read.

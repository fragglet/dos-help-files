     Compiler error C2262
 
     'identifier' : cannot be destroyed
 
     The specified identifier was not instantiated because an
     appropriate destructor was not available.
 
     The following is an example of this error:
 
          struct S
          {
             ~S() __near;
          };
 
          S __far fs;  // error, wrong memory model for destructor
 
          class B
          {
             ~B();
          };
 
          class D : public B {};
          D d;         // error, B's destructor is private

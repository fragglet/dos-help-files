Property can't be modified when form minimized or maximized (384)
■ The Left, Top, Height, and Width properties cannot be
  changed on a minimized or maximized form.
 
■ Possible solutions:
 
  • Set or return the state of a form using the WindowState
    property.
 
  • Prevent the user from maximizing or minimizing a form by
    setting the form's MaxButton or MinButton properties to
    False (0).
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

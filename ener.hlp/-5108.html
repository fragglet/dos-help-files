Log-On Screen
■ Welcome to the Form Designer. This greetings screen
  displays information about your copy of the Form
  Designer, including:
    • Version number         • Copyright notice
    • Licensing information  • Serial number
 
■ To get started, press Shift+F1 from the programming
  environment to open the Help window. Visual Basic provides
  help on the following language elements:
    • Commands       • Dialog boxes         • Events
    • Functions      • Keywords             • Menus
    • Methods        • Objects              • Operators
    • Properties     • Reference topics     • Statements
 
■ The easiest way to get help is to:
    • Place the cursor on the item
    • Press F1 (or click the right mouse button)
    Note: This type of help access is referred to as
          "context sensitive."
 
■ You may want to start by reviewing the topics in the
  "Creating Applications" section of the Contents screen.
 
■ To get to the Help Contents, either:
    • From the Help menu, choose Contents
    • Press Alt+H,C

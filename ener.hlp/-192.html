Close (Ctrl+F4)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the Control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Close command closes the active window.
  ┌─────────────────────┐
  │ Close       Ctrl+F4 │
  └─────────────────────┘
 
■ Use Alt+F4 rather than Ctrl+F4 to close an MDI window.
  ┌─────────────────────┐
  │ Close        Alt+F4 │
  └─────────────────────┘
 
■ To close the active window using a mouse:
    • Double-click the Control-menu box (│■│) located in
      the upper-left corner of the window
 
■ To close the active window using the keyboard:
    • Press Ctrl+F4 (Alt+F4 if MDI type)

Device timeout
■ The program did not receive information from an I/O device
  within a predetermined amount of time. For example, you
  have chosen the PRINT command from the File menu and Visual
  Basic attempted to send text to a printer attached to the
  LPT1 port without receiving an acknowledging signal.
 
■ Possible solution:
 
  • Make sure a printer is attached to the LPT1 port.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

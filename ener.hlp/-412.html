Modified INCLUDE files must be saved before running. Save them now?
■ One or more include files that have not been saved since
  last modified are loaded into memory. Visual Basic needs
  to know which version of the include files you want to run.
 
■ Possible solutions:
 
  • Choose:
    ╔════════╗
    ║  Yes   ║ To save the modified include files and run
    ╚════════╝ the program.
    ┌────────┐
    │   No   │ To run the program with the versions that are
    └────────┘ currently on disk (the unmodified versions).
    ┌────────┐
    │ Cancel │ If you don't want to run the program.
    └────────┘
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

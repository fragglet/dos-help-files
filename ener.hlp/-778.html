'name' is a control name
■ You tried to specify a CtlName value that is already used
  to identify a different type of control.
 
■ Only controls of the same type can have the same CtlName
  property value. When two controls of the same type possess
  the same CtlName property, a control array is automatically
  created.
 
■ Possible solution:
 
  • Rename the control with either:
    ♦ A unique name.
    ♦ The same name as a similar type of control.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

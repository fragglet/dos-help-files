Can't load or unload this object (ERR: 361)
■ More than one condition can cause this error:
 
  • You tried to load or unload one of the special objects
    SCREEN, PRINTER, or CLIPBOARD.
 
  • You tried to load or unload a control that is not an
    element of an existing control array.
 
■ Possible solution:
 
  • Compare the LOAD or UNLOAD statement syntax you are using
    with the Help syntax and reconcile any differences.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Move (Ctrl+F7)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Move command prepares the active window to be moved.
  ┌─────────────────────┐
  │ Move        Ctrl+F7 │
  └─────────────────────┘
 
■ To move the active window using a mouse:
    • Hold down the left mouse button while the cursor is
      positioned over the title bar
    • Drag the window to the desired location
 
■ To move the active window using the keyboard:
    • Press Ctrl+F7
    • Press any of the arrow keys to move the window to the
      desired location

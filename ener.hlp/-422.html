Property not found (ERR: 422)
■ You tried to reference a property that is not available for
  use with the specified control.
 
■ Possible solutions:
 
  • Check to make sure you have correctly spelled the property
    name.
 
  • Check the control's topic in Help for a list of applicable
    properties; change reference as required.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

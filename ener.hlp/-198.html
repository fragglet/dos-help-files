Next Window (Ctrl+F6)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the Control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Next Window command moves the focus to the next
  window.
  ┌─────────────────────┐
  │ Next Window Ctrl+F6 │
  └─────────────────────┘
 
■ To move the focus to the next window using a mouse:
    • Click anywhere in the window you want to make
      active
 
■ To move the focus to the next window using the keyboard:
    • Press Ctrl+F6

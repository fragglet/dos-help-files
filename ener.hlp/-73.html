Feature unavailable (ERR: 73)
■ More than one condition can cause this error:
 
  • You tried to perform:
    ♦ An ISAM operation without the correct ISAM TSR loaded
      (PROISAM or PROISAMD).
    ♦ FRE(-3) when no expanded memory is available.
 
  • You tried to use a feature of:
    ♦ Another version of Basic that is not available with
      this version of Microsoft Visual Basic for MS-DOS.
    ♦ Microsoft Basic that is unavailable under the
      operating system you are running (for example, using
      the LOCK or UNLOCK statement while running MS-DOS
      version 2.1).
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

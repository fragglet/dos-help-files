Edit (Alt+W,number)
■ Activates the selected code window.
 
■ A bullet (•) appears next to the Window menu entry for the
  active window in the programming environment.
  ┌─────────────────┐
  │•1 MAIN1.FRM     │
  │ 2 MOD1.BAS      │
  │ 3 FORM1.FRM     │
  └─────────────────┘
 
■ To activate an code window, you can either:
    • Select the code window using the Shift+arrow keys,
      or click and drag the left mouse button, then press
      Enter
    • Double-click the code window you want to make active

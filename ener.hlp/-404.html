File already exists. Overwrite?
■ You are about to overwrite a file that already exists on
  disk.
 
■ Possible solutions:
 
  • Choose:
    ┌────────┐
    │  Yes   │ To overwrite the file contents on disk with
    └────────┘ the file contents in memory. Note: Be sure
               that overwriting the contents of the file on
               disk won't cause you to lose valuable
               information.
    ╔════════╗
    ║   No   ║ To redisplay the dialog box and type a unique
    ╚════════╝ disk-file name for saving the file in memory.
    ┌────────┐
    │ Cancel │ To cancel the command, leaving everything just
    └────────┘ as it was before you chose the command.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

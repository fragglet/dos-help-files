Output (Alt+W,O)
■ The Output window displays the actual output screen within
  a display window.
 
■ While you cannot change items displayed in the Output
  window, you can still:
    • View the output screen from within the environment
    • Copy items from the output screen to the environment
    • Print output displayed within the window
 
■ The output window can be somewhat limiting. To display the
  entire output screen, choose Output Screen from the View
  menu or press F4. When your application is running, the
  full screen is used.
 
■ A bullet (•) appears next to the command name on the Window
  menu when the Output window is the active window.

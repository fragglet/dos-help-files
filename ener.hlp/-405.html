Delete procedure from module?
■ Before completing execution, the SUBs command needs to
  confirm that you want to delete this procedure from the
  current module.
 
■ Possible solutions:
 
  • Choose:
    ╔════════╗
    ║   OK   ║ To carry out the command and delete the
    ╚════════╝ procedure.
    ┌────────┐
    │ Cancel │ To cancel the command, leaving everything as
    └────────┘ it was before you chose the command.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Color Palette (Alt+W,C)
■ Toggles display of the Color palette.
 
■ Tip: Use the Color palette to change the colors of a form
       or set up a custom color scheme.
 
■ A bullet (•) appears next to the command name on the Window
  menu when the Color palette is the active window.
  ┌─────────────────┐
  │ •Color Palette  │
  └─────────────────┘
 
■ To change the color of a form:
    • Select either the ForeColor or BackColor option
    • Drag the color box to the form or click the desired
      color

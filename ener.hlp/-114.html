Open Project (Alt+F,O)
■ Clears memory and loads an existing project from disk.
 
■ To load a project:
    1. Type a file name in the File Name text box,
                 ┌────────────────────────┐
       File Name:│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │
                 └────────────────────────┘
       or select a file name from the Files list box.
 
        Files:
       ┌──────────────┐
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ↑
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ░
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ↓
       └──────────────┘
    2. Press Enter or choose OK to load the selected project,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

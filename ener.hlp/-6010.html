MS-DOS Shell (Alt+F,H)
■ Temporarily leaves the programming environment so you can
  use MS-DOS commands. The Form Designer remains in memory.
 
■ To return to the Form Designer, type "EXIT" at the command
  prompt and press Enter.
 
■ Tip: While at the command prompt, be careful not to delete
       any files that the Form Designer might need.

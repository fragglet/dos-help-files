Tried to load file with duplicate form name
■ The file you tried to load contains a form name that has
  already been used in a currently loaded file.
 
■ Possible solution:
 
  • Remove or rename the form and then try adding the file.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

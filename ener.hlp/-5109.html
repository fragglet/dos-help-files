Set Help Path (Alt+O,P)
■ Changes the default MS-DOS paths that the Form Designer
  will use to find Help (.HLP) files.
 
■ The Form Designer environment searches for Help files in
  the directories you indicate. You don't need to set
  search paths if either:
    • You have already set paths with MS-DOS environment
      variables
    • All files needed by your project are in the current
      directory
 
■ To set the Help path:
    1. Using the same syntax as MS-DOS environment
       variables, enter the search path for Help files:
                   ┌──────────────────────────────┐
       Help Files: │C:\VBDOS\HLP                  │
                   └──────────────────────────────┘
       Note: When you enter a path, Visual Basic does not
             check what you type for syntax or meaning.
 
    2. Press Enter or choose OK to set the Help path,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

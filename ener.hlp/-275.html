Illegal outside TYPE block
■ The "element AS type" clause is permitted only within a
  TYPE...END TYPE block.
 
■ Possible solution:
 
  • Insert a TYPE statement at the beginning and an END TYPE
    statement at the end of the TYPE...END TYPE structure.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

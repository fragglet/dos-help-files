Instant Watch (Shift+F9)
■ Displays the current value of a variable or an expression
  when project execution is suspended.
 
■ Tip: Use Instant Watch to view the current value of a
       variable when you do not want to add that variable
       to the Debug window.
 
■ To display the value of a variable or expression:
    1. Select the item by placing the cursor on the variable
       or highlighting the entire expression
 
    2. Invoke the Instant Watch dialog by either:
       • Choosing Instant Watch from the Debug menu
       • Pressing Shift+F9
 
    3. The Instant watch dialog box displays:
       • The selected variable or expression.
 
       ┌Expression──────────────────────────┐
       │NumVar                              │
       └────────────────────────────────────┘
       • The current value of the selected item.
 
       ┌Value───────────────────────────────┐
       │<Not available>                     │
       └────────────────────────────────────┘
       • The expression is True if Instant Watch displays
         a nonzero value, and False if it displays 0.
       • The message <Not available> appears in the Value
         box if Visual Basic cannot access the value of the
         variable or expression.
 
    4. Press Enter or choose Add Watch to add the variable
       or expression to the Debug window,

Included File (Alt+V,I)
■ Loads an include file (.BI) into the active window for
  editing. This command is active only when the cursor is
  positioned on an $INCLUDE line in a loaded project.
 
■ To use the Included File command:
    • Place the cursor on the $INCLUDE metacommand that
      names the file you want to edit
    • Choose the Included File command
    • Edit the file in the active window
    • Choose Save File As from the File menu, or debug your
      project with the include file incorporated and save
      the file later, as part of the project
 
■ To load the file into a new window:
    • Choose Code from the View menu
    • Select the include file you want to edit
    • Choose Edit in New

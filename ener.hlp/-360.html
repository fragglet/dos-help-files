Object already loaded (ERR: 360)
■ You tried to load a control array element that has already
  been loaded.
 
■ Possible solutions:
 
  • If the control array element is hidden, you can show it
    by setting its Visible property to True (-1) at design
    time.
 
  • If the object was loaded at run time, you can remove it
    using the UNLOAD statement.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Invalid outside SUB/FUNCTION
■ Visual Basic has encountered a statement that is not
  allowed in module-level code or DEF FN functions.
 
■ Possible solution:
 
  • Move highlighted statement to a SUB or FUNCTION procedure.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

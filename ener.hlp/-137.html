Next Statement (Alt+V,N)
■ Places the cursor on the next executable statement
  following the point where a project has been suspended.
 
■ You can resume execution by either:
    • Pressing F5 or choosing Continue from the Run menu
    • Pressing F8 to execute only the next statement
    • Choosing the Set Next Statement command from the Debug
      menu to change the point where execution will resume

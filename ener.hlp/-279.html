SUB, FUNCTION, or variable of the same name
■ You have specified a SUB, FUNCTION, or variable name that
  already exists.
 
■ Possible solution:
 
  • Rename the existing item before using the name you
    specified or use a different SUB, FUNCTION, or variable
    name.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Form (Shift+F12)
■ Displays all forms in the active project and allows you
  to select a form to modify.
 
■ To modify a form:
    1. Choose a form name from the list displayed in the
       file list box.
 
        Choose form to modify:
       ┌──────────────────────────────────────────┐

       │ FORM2.FRM                                ↓
       └──────────────────────────────────────────┘
    2. Press Enter or choose OK to unload the current form
       (prompting to save changes if necessary) and load the
       selected form,

       ╚════════╝
       or choose one of the following:

Can't unload controls created at design time (ERR: 362)
■ Only control array elements loaded at run time can be
  unloaded. If a control is created at design time, it cannot
  later be unloaded, even if it is part of a control array.
 
■ Possible solution:
 
  • Hide the control by setting its Visible property to
    False (0).
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

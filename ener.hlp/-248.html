Breakpoints not allowed on CASE clauses or END SELECT
■ There are some restrictions on using breakpoints to debug
  SELECT...END SELECT statements. You cannot place a
  breakpoint on the first statement in the CASE block or
  the END SELECT statement.
 
■ Possible solution:
 
  • Set the breakpoint on the SELECT CASE line in your
    program and single step through the rest of the statement
    structure.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Variable not declared
■ OPTION EXPLICIT is active and you have referenced a
  variable name that has not been declared.
 
■ Possible solutions:
 
  • Declare the variable before referencing it.
 
  • Remove all OPTION EXPLICIT statements in your project.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

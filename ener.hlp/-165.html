Right Mouse (Alt+O,M)
■ Toggles the effect of a right mouse click between:
    • Displaying help about the item the cursor is on
    • Executing the active project from the beginning to
      the line the cursor is on
 
■ To assign a function to the right mouse button:
    1. Select either:
       • Context-sensitive Help: Available regardless of
         right mouse button functionality by positioning the
         cursor on a keyword and pressing F1.
 
         (•) Context-sensitive Help
 
       • Execute up to this line (useful for debugging):
         Available regardless of right mouse functionality by
         pressing F7.
 
         ( ) Execute up to this line
 
    2. Press Enter or choose OK to assign to the selected
       function,

       ║   OK   ║
       ╚════════╝
       or choose one of the following:

       └────────┘
       ┌────────┐
       │  Help  │  To get help

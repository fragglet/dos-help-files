Remove File (Alt+F,R)
■ Unloads a file from memory and removes it from the
  project.
 
■ To remove a file from a project:
    1. Choose a file name from the list displayed in the
       file list box.
 
       Choose file to unload:
       ┌──────────────────────────────────────────┐
       │ SAMPLE.BAS                               ↑
       │ Form1.FRM                                ↓
       └──────────────────────────────────────────┘
    2. Press Enter or choose OK to remove the file,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘
       Note: If the selected file has been modified since
             last saved, Visual Basic prompts you to save
             the file before it is removed.

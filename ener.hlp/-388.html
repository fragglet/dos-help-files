Search Menu (Alt+S)
■ Searches active window, current module, or all modules for
  a text string with option to replace:
 
┌────────────────────────┐ • Finds specified text
│ Find...                │ • Finds selected text
│ Selected Text   Ctrl+\ │ • Finds next occurrence of
│ Repeat Last Find    F3 │   specified text
│ Change...              │ • Finds and replaces text

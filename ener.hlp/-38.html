Array not defined (ERR: 38)
■ More than one condition can cause this error:
 
  • Your program is attempting to use an array that is not
    currently defined.
 
  • You have an undeclared function that appears to be an
    array. For example:
 
          X% = F(20)
 
■ Possible solution:
 
  • Make sure the array is defined with a DIM or REDIM
    statement.
 
  • Verify that all functions have been declared using
    the appropriate DECLARE statement.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

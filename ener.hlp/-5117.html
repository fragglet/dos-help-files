Print (Alt+F,P)
■ Prints the current form to a printer or output file.
 
■ To print the current form:
    1. Select item to print; you can print only the form
       or the form and its properties.
 
       ┌Send─────────────────────┐
       │ (•) Form and Properties │
       │ ( ) Form Only           │
       └─────────────────────────┘
    2. Select the Page Break check box if you want to print
       the property list for the form and each control on a
       separate page.
 
       [ ] Page Break Between Property Lists
 
    3. Specify print target by selecting a printer port,
       or specify file name to print to a file:
       ┌To──────────────────────────────────────┐
       │ (•) LPT1          To                   │
       │ ( ) LPT2                               │
       │ ( ) LPT3  ┌──────────────────────────┐ │
       │ ( ) File: │                          │ │
       │           └──────────────────────────┘ │
       │ If File Exists: (•) Append ( ) Replace │
       └────────────────────────────────────────┘
       • Use Append option to copy to end of file
       • Use Replace option to delete previous file contents
 
    4. Press Enter or choose OK to print,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

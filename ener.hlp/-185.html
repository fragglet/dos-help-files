ener.hlp (Topic list)
Index (Alt+H,I)
■ Provides an alphabetical list of Help topics, including
  all Visual Basic language elements. Each term in the Help
  index is linked to related information.
 
■ To get help from the Index:
    • Choose the Index command from the Help menu
    • Press the key corresponding to the item's first letter
    • Move the cursor to the item you want help on
    • Press F1

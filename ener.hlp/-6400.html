Options Menu (Alt+O)
■ Controls system color and path settings and allows you
  to define environment save options:
 
┌──────────────────┐ • Sets user-interface foreground and
│ Display...       │   background colors, desktop fill
│ Set Help Path... │   characters, and default tab stops
│ Save...          │ • Defines location of help (.HLP) files
└──────────────────┘ • Selects environment save options
 
■ Options settings are saved and remain in effect the next
  time you start the Form Designer.
 
■ Tip: Use the Options Display dialog to see what colors set
       with the ControlPanel property will look like at run
       time.

Can't assign shortcut key to menu name
■ You tried to assign a key combination to a top-level
  menu title that is not a command. Only menu items
  can have shortcut keys assigned to them.
 
■ Possible solutions:
 
  • Use the keyboard or mouse to open the Shortcut Key list
    and select "(none)" or type "(" to select the first item
    in the list.
                          ┌───┐
  • Press Alt+R or choose │ ► │ to make the menu title a
    submenu.              └───┘
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

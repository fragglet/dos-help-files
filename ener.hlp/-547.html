Make file must have .MAK extension
■ You tried to assign an invalid name to a project or make
  (.MAK) file.
 
■ Possible solutions:
 
  • Rename the project file using a .MAK extension.
 
  • Verify that the file name specified follows standard
    MS-DOS naming conventions. For example, the file name
    "GEORGEWAS.MAK" is invalid, while "GEORGWAS.MAK" is
    valid.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

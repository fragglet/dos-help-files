Paste (Shift+Ins)
■ Copies text from the Clipboard to the active window.
  Text is placed at the insertion point in the active
  window.
 
■ The insertion point is defined by:
    • Highlighted text, which is deleted; the Clipboard
      contents replaces the highlighted text.
    • The cursor position if no text is highlighted; less
      than one line of text is copied to the left of the
      cursor; more than one line is copied above the
      cursor.

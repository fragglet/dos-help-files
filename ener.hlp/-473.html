Cannot close last code window
 ■ The last code window in the programming environment
   cannot be closed.
 
■ Possible solutions:
 
  • To exit, choose Exit from the File menu.
 
  • To start the Form Designer, choose New Form from the File
    menu.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

New Sub (Alt+E,S)
■ Defines a SUB procedure and creates a code template.
 
■ Tip: Type "SUB subname" in the active code window and press
       Enter to define a SUB procedure and create a code
       template.
 
■ Use descriptive procedure names to identify the procedure's
  use and to avoid duplicate procedure or variable names.
 
■ To define a new SUB procedure:
    1. Type a procedure name (up to 40 characters) in the
       Name text box.

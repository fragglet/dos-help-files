Options Menu (Alt+O)
■ Controls system color, path, mouse, and syntax settings,
  and defines environment save options:
 
┌─────────────────┐ • Sets user-interface foreground and
│ Display...      │   background colors, desktop fill
│ Set Paths...    │   character, and default tab stops
│ Right Mouse...  │ • Defines location of executable (.EXE),
│ Save...         │   include (.BI), library (.LIB), and
│•Syntax Checking │   help (.HLP) files
└─────────────────┘ • Selects function of right mouse button
                    • Selects environment save options
                    • Toggles syntax checking in
                      programming environment
 
■ Options settings are saved and remain in effect the next
  time you start Visual Basic.
 
■ Tip: Use the Options Display dialog to see what colors
       set with the ControlPanel property will look like
       at run time.

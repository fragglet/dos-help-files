Turn off Include Lines toggle command
■ Include-file text is displayed in the code window along
  with program text.
 
■ Possible solution:
 
  • Choose:
    ╔════════╗
    ║   OK   ║ To remove the read-only include file from the
    ╚════════╝ code window so you can edit program text. The
               Include Lines command will also be turned off.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

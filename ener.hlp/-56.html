FIELD statement active (ERR: 56)
■ A GET or PUT statement specified a record variable on a file
  for which FIELD statements have already been executed.
 
■ GET or PUT with a record-variable argument can be used only
  on files where no FIELD statements have been executed.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Restore (Ctrl+F5)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the Control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Restore command restores a minimized window to its
  previous state.
  ┌─────────────────────┐
  │ Restore     Ctrl+F5 │
  └─────────────────────┘
 
■ To restore a minimized window using a mouse:
    • Double-click the window's icon
 
■ To restore a minimized window without using a mouse:
    • Press Ctrl+F5

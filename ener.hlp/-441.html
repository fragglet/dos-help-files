Match not found
■ Visual Basic could not find the string specified.
 
■ The cursor remains at the location where you started the
  search.
 
■ Possible solutions:
 
  • Verify that you typed the string correctly.
 
  • Search for a different string.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

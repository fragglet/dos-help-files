Incorrect number of event procedure arguments
■ You tried to pass an incorrect number of arguments to an
  event procedure. For example:
 
  ┌─Wrong────────────────────────────────────────────────┐
  │ SUB Form_Click (Index AS INTEGER)                    │
  └──────────────────────────────────────────────────────┘
 
  ┌─Right────────────────────────────────────────────────┐
  │ SUB Form_Click ()        ' No arguments are passed   │
  └──────────────────────────────────────────────────────┘
 
■ Possible solution:
 
  • Check the event procedure's Help topic for a list of
    valid arguments and change the reference as required.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Save Text (Alt+F,T)
■ Prompts you for a file name and saves the current file in
  text format. Only code is saved; this command doesn't save
  the binary description of a form as text.
 
■ Tip: Use Save Text to export binary form code to text
       format.
 
■ Note: A copy of the current file is saved as a text (.TXT)
        file. This command does not rename the file in the
        project like the Save As command does.
 
■ To name and save the current file:
    1. Type a file name in the File Name text box.
                  ┌────────────────────────┐
       File Name: │ UNTITLED.TXT           │
                  └────────────────────────┘
    2. Press Enter or choose OK to save the file,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘
       Note: If a disk file has the same name as the file
             being saved, Visual Basic prompts you to
             confirm before overwriting the file.

  and directories:

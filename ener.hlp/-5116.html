New Form (Alt+F,F)
■ Creates a new form and adds it to the active project.
 
■ To create a new form:
    1. Press Enter or choose OK to accept the default
       file name,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or type a file name in the File Name text box.
                  ┌────────────────────────┐
       File Name: │ FORM1.FRM              │
                  └────────────────────────┘
    2. Press Enter or choose OK to create the form,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘
 
■ The Directories list box provides access to other drives
  and directories:
 
   Directories:
  ┌──────────────┐
  │ ..           ↑ • Include a drive/path designation with
  │ [-A-]        ░   the file name
  │ [-C-]        ░
  │              ↓ • Choose the appropriate drive/directory
  └──────────────┘

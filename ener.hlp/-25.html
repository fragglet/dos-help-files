Device fault (ERR: 25)
■ A device has returned a hardware error indicating one of the
  following conditions:
 
  • You attempted to print a file when the printer is not
    attached to the parallel port LPT1.
 
  • You transmitted data to a communications file and the
    signals being tested with the OPEN COM statement were not
    found in the specified period of time.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

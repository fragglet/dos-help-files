Form (Shift+F12)
■ Allows you to modify a form by:
    • Leaving the programming environment
    • Starting the Form Designer
    • Loading the selected form
 
■ Tip: To automatically save files when switching between
       the Form Designer and the programming environment,
       select the appropriate check box on the Options Save
       dialog.
 
■ To modify a form:
    1. Choose a form name (.FRM) from the list displayed in
       the file list box.
 
       Choose form to modify:
       ┌──────────────────────────────────────────┐
       │ FORM1.FRM                                ↑
       │ FORM2.FRM                                ↓
       └──────────────────────────────────────────┘
    2. Press Enter or choose OK to exit the programming
       environment, start the Form Designer, and load the
       selected form,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

A control by that name already exists
■ You tried to give the same name to two different types of
  controls. Controls of the same type can be given the same
  name to create a control array.
 
■ Possible solution:
 
  • Specify a control name that has not already been used.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

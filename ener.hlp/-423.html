Property or control not found (ERR: 423)
■ You tried to reference a control or property that is not
  defined for the specified form.
 
■ Possible solutions:
 
  • Check to make sure you have correctly spelled the property
    name.
 
  • Check the form topic in Help for a list of applicable
    properties and change the reference as required.
 
  • Choose Event Procedures from the Edit menu (Alt+E,V) to
    display a list of the controls on this form and change the
    control name reference.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

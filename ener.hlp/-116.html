New Form (Alt+F,F)
■ Creates a new form and adds it to the active project.
 
■ Once a file name has been accepted, you leave the
  programming environment and shell to the Form Designer.
  Because the Form Designer is a separate utility, Visual
  Basic prompts you to save any modified files before
  leaving the programming environment.
 
■ To create a new form:
    1. Press Enter or choose OK to accept the default
       file name,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or type a file name in the File Name text box.
                  ┌────────────────────────┐
       File Name: │ FORM1.FRM              │
                  └────────────────────────┘
    2. Press Enter or choose OK to create the form,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

  directories:

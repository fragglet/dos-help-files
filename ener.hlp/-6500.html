Window Menu (Alt+W)
■ Enables and controls simultaneous window display:
 
┌────────────────────┐ • Enables foreground and background
│ Color Palette      │   color changes of design-time items
│ Menu Design Window │ • Defines menu items such as menu
│ Toolbox            │   controls and access keys
│ Help               │ • Displays available controls
│•Form: Form1        │ • Displays Help window
└────────────────────┘ • Selects previously loaded forms
 
■ Note: A bullet (•) appears next to the current form.
 
■ Tip: Use the Color Palette to see what colors set with the
       the ControlPanel property will look like at run time.

Bad file name (ERR: 64)
■ More than one condition can cause this error:
 
  • A file name does not follow the appropriate naming
    convention for BLOAD, BSAVE, KILL, or OPEN (for
    example, the file name has too many characters).
 
  • You are trying to load an invalid project file (Visual
    Basic for Windows, NMAKE/PWB) into Visual Basic for
    MS-DOS.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Duplicate module base names loaded. Can't compile.
■ Visual Basic cannot compile two loaded modules with the
  same base name. For example, FORM1.FRM and FORM1.BAS in
  the same project will not compile.
 
■ Possible solution:
 
  • Rename one of the modules and try again.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

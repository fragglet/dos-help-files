Change (Alt+S,C)
■ Searches for one text string and replaces it with
  another. You can search:
    • The active window
    • The current module
    • All loaded modules
 
■ You can either:
    • Review each replacement before it is made
    • Use a case-sensitive or whole-word search
 
■ Tip: To find a string without the replacement option,
       use the Find command.
 
■ To search and replace text:
    1. Type the text string to find in the Find What text
       box.
                  ┌───────────────────────────────────────┐
       Find What: │                                       │
                  └───────────────────────────────────────┘
    2. Type the replacement string in the Change To text
       box.
                  ┌───────────────────────────────────────┐
       Change To: │                                       │
                  └───────────────────────────────────────┘
    3. Select options to match uppercase and lowercase or
       whole words only:
 
       [ ] Match Upper/Lowercase
       [ ] Whole Word
 
       • Use Match Upper/Lowercase option for case-
         sensitive match
       • Use Whole Word option to exclude matches within
         words or strings; recognizes a match only when
         target text is surrounded by blanks, punctuation,
         or other special characters
 
    4. Select search range.
       ┌───────────────────────┐
       │ ( ) 1. Active Window  │
       │ (•) 2. Current Module │
       │ ( ) 3. All Modules    │
       └───────────────────────┘
    5. Press Enter or choose Find and Verify to begin the
       search and review each replacement before it is made,
       ╔═══════════════╗
       ║Find and Verify║
       ╚═══════════════╝
       choose Change All to find and replace all occurrences
       without verification,
       ┌───────────────┐
       │   Change All  │
       └───────────────┘
       or choose one of the following:
       ┌───────────────┐
       │     Cancel    │ To cancel the operation
       └───────────────┘
       ┌───────────────┐
       │      Help     │ To get help
       └───────────────┘

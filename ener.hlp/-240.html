END SUB or END FUNCTION must be last line in window
■ You are attempting to add code after the last line in a
  procedure.
 
■ Possible solutions:
 
  • Delete the code, or move it above the END statement.
 
  • Return to the start-up module or open another module,
    SUB, or FUNCTION procedure.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

ISAM - Insufficient ISAM buffers (ERR: 89)
■ There are not enough buffers to handle ISAM file processing.
 
■ Possible solution:
 
  • When you invoke BC.EXE or VBDOS.EXE, use the /Ib command-
    line option to increase the number of ISAM buffers.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

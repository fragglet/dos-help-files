Cannot change tab stops while file is loaded
■ An ASCII file that contains tabs is currently loaded in
  memory, so you can't use the Tab Stops option in the
  Display dialog to change the value of a tab stop.
 
■ Possible solution:
 
  • Unload the ASCII file and then change the Tab Stops
    option.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

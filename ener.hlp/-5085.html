Destination
■ Which module do you want the selected procedure to be
  moved to?
    1. Select the desination module from the from the
       modules listed.
 
       Choose destination module:
       ┌──────────────────────────────────────────┐
       │ THIS1.BAS                                ↑
       │ FORM1.FRM       (Form1)                  ░
       │ RUND.BAS                                 ↓
       └──────────────────────────────────────────┘
       Note: Text within the parentheses represents
             the name of the form (FormName property
             value).
 
    2. Press Enter or choose OK to move the procedure to the
       selected module,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

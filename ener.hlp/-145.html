Start (Shift+F5)
■ Clears memory and starts project execution at the first
  executable statement in the start-up file.
 
■ To designate a start-up file, choose Set Start-up File
  from the Run menu.
 
■ You can use the Start command after stopping a running
  project with Ctrl+Break to run the project again from the
  beginning. Use Start as an alternative to Restart or
  Continue.
 
■ Tip: Breakpoints and watchpoints are alternatives to
       pressing Ctrl+Break.

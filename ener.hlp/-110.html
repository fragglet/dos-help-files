Invalid ASCII form description
■ You tried to load an ASCII description of a form that
  contains invalid or missing information. If possible:
  • Visual Basic loads the form anyway. In some cases, only
    a blank form with no controls will be loaded.
  • Visual Basic creates a log (.LOG) file that contains
    specific form description errors and their location.
 
■ Possible solution:
 
  • View the log (.LOG) file, correct errors, then reload
    the form.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Minimize (Ctrl+F9)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Minimize command minimizes the active window.
  ┌─────────────────────┐
  │ Minimize    Ctrl+F9 │
  └─────────────────────┘
 
■ To minimize a window using a mouse:
    • Click the Minimize button (│▼│) located in the upper-
      right corner of the window
 
■ To minimize a window using the keyboard:
    • Press Ctrl+F9

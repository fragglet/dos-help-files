File has been modified since last saved. Save it now?
■ You are about to execute a command that will cause unsaved
  changes to a loaded file since to be lost.
 
■ Possible solutions:
 
  • Choose:
    ╔════════╗
    ║  Yes   ║ To save the file, preserve your changes,
    ╚════════╝ and finish executing this command.
    ┌────────┐
    │   No   │ To finish executing this command without saving
    └────────┘ the file. Use this choice to delete any work
               you do not want to save.
    ┌────────┐
    │ Cancel │ To cancel the command, leaving everything as
    └────────┘ it was before you chose the command.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

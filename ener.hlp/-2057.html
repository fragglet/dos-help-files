Device I/O error
■ You used the Print command from the File menu and the
  printer malfunctioned.
 
■ Possible solutions:
 
  • Verify the printer cable is securely connected to both the
    printer and the computer.
 
  • Verify the printer power is on.
 
  • Verify the printer online indicator light is on.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

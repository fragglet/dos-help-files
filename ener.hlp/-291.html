Object is not an array
■ You have incorrectly referenced an object as part of an
  array. For example:
 
         Command1(1).Left = 1
 
  where Command1 is a single control that is not part of
  a control array.
 
■ Possible solution:
 
  • Remove the parentheses and array element from the object
    reference.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

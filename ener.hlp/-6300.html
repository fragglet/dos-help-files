Tools Menu (Alt+T)
■ Enables keyboard access to the same controls found in the
  Tools window:
 
  Access Key  Control             Access Key  Control
  ══════════  ══════════════      ══════════  ═════════════
  Alt+T,K     Check Box           Alt+T,L     Label
  Alt+T,M     Combo box           Alt+T,S     List Box
  Alt+T,C     Command Button      Alt+T,O     Option Button
  Alt+T,D     Dir List            Alt+T,P     Picture Box
  Alt+T,R     Drive List          Alt+T,X     Text Box
  Alt+T,F     File List           Alt+T,T     Timer
  Alt+T,A     Frame               Alt+T,V     VScrollBar
  Alt+T,H     HScrollBar

Remove Form (Alt+F,R)
■ Unloads a form from memory and removes it from your
  project.
 
■ To remove a form from your project:
    1. Choose a file name from the list displayed in the
       file list box.
 
        Choose file to unload:
       ┌──────────────────────────────────────────┐
       │ FORM1.FRM                                ↑
       │ FORM2.FRM                                ↓
       └──────────────────────────────────────────┘
    2. Press Enter or choose OK to remove the form,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘
       Note: If the selected form has been modified since
             last saved, Visual Basic prompts you to save
             the file before it is removed.

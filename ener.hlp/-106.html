File previously loaded
■ You are attempting to load a file that is already in memory.
 
■ Possible solutions:
 
  • Work with the file already loaded in memory.
 
  • Unload the file in memory so you can load this one.
 
  • Look at the list of files available for loading. You may
    have tried to load the wrong one.
 
  • If you are trying to edit a form that has already been
    loaded, choose Form from the View menu (Shift+F12).
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

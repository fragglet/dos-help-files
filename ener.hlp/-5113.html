Add Form (Alt+F,D)
■ Loads an existing form into memory and adds it to your
  project. Forms already open remain in memory. Use New
  Form rather than Add Form to create and add a new module
  to your project.
 
■ To load an existing form:
    1. Type a file name in the File Name text box,
                  ┌────────────────────────┐
       File Name: │ *.FRM                  │
                  └────────────────────────┘
       or choose a file name from the Files list box.
 
        Files:
       ┌──────────────┐
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ↑
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ░
       │ ▒▒▒▒▒▒▒▒▒▒▒▒ ↓
       └──────────────┘
    2. Press Enter or choose OK to add the form,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘
 
■ The Directories list box provides access to other drives
  and directories:
 
   Directories:
  ┌──────────────┐
  │ ..           ↑ • Include a drive/path designation with
  │ [-A-]        ░   the file name
  │ [-C-]        ░
  │              ↓ • Choose the appropriate drive/directory
  └──────────────┘

Block IF without END IF
■ There is no corresponding END IF in a block IF construct.
 
■ Possible solutions:
 
  • Insert an END IF statement at the end of the IF...END IF
    structure.
 
  • Check other control structures within the IF...END IF
    block to verify that they are correctly matched.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Size (Ctrl+F8)
■ The Control menu provides access to standard window
  commands through a menu or by using shortcut keys.
  This menu is activated by clicking the Control-menu
  box or by pressing Alt+minus (Alt+-) on your keyboard.
 
■ The Size command prepares the active window to be sized.
  ┌─────────────────────┐
  │ Size        Ctrl+F8 │
  └─────────────────────┘
 
■ To size a window using a mouse:
    • Hold down the left mouse button while the cursor is
      positioned over a corner of the window
    • Drag the corner of the window to size
 
■ To size a window using the keyboard:
    • Press Ctrl+F8
    • Press any of the arrow keys to size the window

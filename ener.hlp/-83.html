Index not found (ERR: 83)
■ There is no index with the specified name for the ISAM
  table referenced by a file number argument. For example, in
  a DELETEINDEX or SETINDEX statement.
 
■ Possible solutions:
 
  • Verify that you have not misspelled the index name.
 
  • Create an ISAM table index with the name referenced in
    the statement.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

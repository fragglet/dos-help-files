Disk full (ERR: 61)
■ More than one condition can cause this error:
 
  • There isn't enough room on the disk to complete a PRINT #,
    WRITE #, or CLOSE operation.
 
  • There isn't enough room for Visual Basic to write out an
    object or executable file.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

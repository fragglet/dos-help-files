Operation requires selection of a SUB or FUNCTION
■ You selected the wrong type of item from the View Code list
  box when choosing Move. Only SUB and FUNCTION procedures
  can be moved.
 
■ In the View Code dialog box, the type of the highlighted
  item is displayed in the lower left-hand corner, just
  below the list box's horizontal scroll bar.
 
■ Possible solution:
 
  • Highlight a different item and choose Move.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

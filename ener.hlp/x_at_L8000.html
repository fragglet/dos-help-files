File Menu (Alt+F)
■ Controls projects and files:
 
┌─────────────────┐ • Creates new project, loads existing
│ New Project     │   project, or saves active project
│ Open Project... │ • Creates new form (switches to Form
│ Save Project    │   Designer) or new module and adds to
├─────────────────┤   project
│ New Form        │ • Adds existing form or module to
│ New Module...   │   project
│ Add File...     │ • Removes active form or module from
│ Remove File...  │   project
│ Save File       │ • Saves module to disk in binary
│ Save File As... │   (default) format or text format
├─────────────────┤ • Loads text files from other editors
│ Load Text...    │   or saves Visual Basic files as text
│ Save Text...    │ • Prints files or selected text to
├─────────────────┤   specified printer
│ Print...        │ • Allows use of MS-DOS commands
│ MS-DOS Shell    │ • Closes active project and exits
├─────────────────┤   programming environment
│ Exit     Alt+F4 │
└─────────────────┘

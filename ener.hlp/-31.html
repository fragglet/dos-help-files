Edit INCLUDE file
■ Include-file text is displayed in the code window along with
  the program text, but the include-file text is read-only.
 
■ Possible solution:
 
  • Choose:
    ╔════════╗
    ║   OK   ║ To move the include-file text into a new View
    ╚════════╝ window so you can edit it.

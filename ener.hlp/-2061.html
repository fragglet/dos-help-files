Disk full condition
■ There isn't enough room on the specified disk to save a
  specified file.
 
■ Possible solutions:
 
  • Save the file to another disk.
 
  • Cancel the command and use the Shell command from the File
    menu to delete some files from the disk and try again.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

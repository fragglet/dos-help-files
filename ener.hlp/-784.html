Menu control array element must have an index
■ An item has been added to a menu control array without
  assigning an index to the item. You must assign the
  index value.
 
■ Possible solution:
 
  • Press Alt+X or tab to the Index text box and enter an
    index value. Remember that control array index values
    must be in ascending order.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

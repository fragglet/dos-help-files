Property not available at run time (ERR: 386)
■ The CtlName and FormName properties are not available at
  run time.
 
■ Possible solutions:
 
  • Delete the CtlName or FormName property reference.
 
  • Set the CtlName or FormName property at design time.
 
  • Use the Tag property to pass an identification string
    at run time.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

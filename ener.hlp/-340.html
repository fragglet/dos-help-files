Control array element does not exist (ERR: 340)
■ You used an index value that does not correspond to an
  existing element in the specified control array.
 
■ Possible solution:
 
  • Adjust the value or load a control into the array with an
    index equal to the value specified.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Debug (Alt+W,D)
■ The Debug window displays watch variables and expressions
  and allows you to examine the value of these items during
  program execution.
 
■ You cannot directly add watches by typing in this window
  since it is read-only. You can, however, copy information
  displayed within the Debug window.
 
■ A bullet (•) appears next to the command name on the
  Window menu when the Debug window is the active window.

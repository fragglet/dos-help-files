Save (Alt+O,V)
■ Sets the programming environment save options. Available
  settings include:
    • Save before every run (with or without prompting)
    • Save without prompting before leaving the programming
      environment and starting the Form Designer
    • Create backup (.BAK) files when project files are
      modified
 
■ To select or cancel save options:
    1. Use the Spacebar or click between the square brackets
       to toggle placement of an X in front of the save
       options.
 
       [ ] Save Before Run
           [ ] Save Without Prompting
 
       [ ] Save Without Prompting
            Before Form Design
 
       [X] Create Backup File (.BAK)
 
    2. Press Enter or choose OK to accept the selected save
       options,
       ╔════════╗
       ║   OK   ║
       ╚════════╝
       or choose one of the following:
       ┌────────┐
       │ Cancel │  To cancel the operation
       └────────┘
       ┌────────┐
       │  Help  │  To get help
       └────────┘

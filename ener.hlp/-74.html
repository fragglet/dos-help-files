Rename across disks (ERR: 74)
■ You attempted to rename a file with a new drive designation.
  You cannot rename a file with a new drive designation in
  Visual Basic.
 
■ Possible solution:
 
  • Rename the file on the current disk and then copy it
    to the new drive location.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

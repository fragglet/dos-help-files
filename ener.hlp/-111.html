Invalid .MAK file
■ You have attempted to load a Make (.MAK) file which is not
  valid in Visual Basic for MS-DOS format.
 
■ Possible solution:
 
  • Check the file name and verify that it is correct.
 
  • Individually load the forms and modules contained in the
    project.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Invalid outside SUB, FUNCTION, or DEF FN
■ Visual Basic has encountered a statement that is not
  allowed in module-level code. For example:
 
  • EXIT SUB, EXIT FUNCTION, or EXIT DEF statements are not
    allowed in module-level code.
 
  • Executable statements are not allowed at the module level
    of a form module.
 
■ Possible solution:
 
  • Move highlighted statement to a SUB, FUNCTION, or DEF FN
    function.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

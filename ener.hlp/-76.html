Path not found
■ During an OPEN, MKDIR, CHDIR, or RMDIR operation, the
  operating system was unable to find the path specified.
 
■ Possible solutions:
 
  • Verify the path specified exists. For example, a path
    specified in the Options.Path dialog may not be valid.
 
  • If you are creating the path, make sure you don't
    reference the path until after it is created.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

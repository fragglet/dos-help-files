Invalid object array index (ERR: 341)
■ You used a negative index value or one that falls outside
  the valid range. The index for an object array element
  cannot be less than 0 or greater than 32,767.
 
■ Possible solution:
 
  • Change the array index to a value within the valid range.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Module with that name is already loaded
■ This error can occur when you add a file to the current
  project that has already been loaded. Generally, you get
  this message if any other files loaded into memory have
  the file name specified.
 
■ Possible solutions:
 
  • Choose the MS-DOS Shell command to leave the programming
    environment and then rename and reload the file.
 
  • Use another file name in the Add File dialog box.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

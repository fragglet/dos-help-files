END SELECT without SELECT
■ The end of a SELECT CASE statement appears without a
  beginning SELECT CASE. The beginning of the SELECT CASE
  statement may be missing or misspelled.
 
■ Possible solutions:
 
  • Insert the SELECT CASE keywords at the beginning of the
    SELECT...END SELECT structure or correct misspelling.
 
  • Check other control structures within the SELECT statement
    block to verify that they are correctly matched.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

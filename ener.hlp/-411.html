Project or source files have changed. Save them now?
■ You are about to clear the contents of memory, and one or
  more files in memory have not been saved since the last time
  they were changed.
 
■ Possible solutions:
 
  • Choose:
    ╔════════╗
    ║  Yes   ║ To save the file or files and then execute
    ╚════════╝ the command. If a file has note been named,
               you will be prompted for a file name.
    ┌────────┐
    │   No   │ To clear memory without saving the files and
    └────────┘ then execute the command.
    ┌────────┐
    │ Cancel │ To cancel the command, leaving the contents of
    └────────┘ memory the same as before you chose the
               command.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

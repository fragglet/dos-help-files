CASE without SELECT
■ The first part of a SELECT CASE statement is missing or
  misspelled.
 
■ Possible solutions:
 
  • Insert the SELECT keyword at the beginning of the
    SELECT...END SELECT structure or correct misspelling as
    necessary.
 
  • Check other control structures within the SELECT statement
    block to verify that they are correctly matched.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

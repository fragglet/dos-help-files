At least one submenu item must be visible
■ You tried to create a menu where the Visible property of
  all submenu items is set to False. This error also occurs
  when loading an ASCII project where all submenu items
  are invisible.
 
■ Possible solution:
 
  • Set the Visible property of one or more submenu items
    to True.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

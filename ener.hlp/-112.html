Multiple MDI forms not supported
■ You tried to add an MDI type form module to your project
  when a MDI type form already exists.
 
■ Possible solution:
 
  • Change the FormType property of the MDI form in your
    project retry the File Add operation.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

Support for custom control type not available
■ The appropriate custom control quick library for the type
  you are referencing is not loaded.
 
■ Possible solution:
 
  • Restart Visual Basic with using the /L command-line
    option and the appropriate quick library.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

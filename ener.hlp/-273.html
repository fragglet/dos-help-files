Statements/labels illegal between SELECT CASE and CASE
■ Statements and line labels are not permitted between SELECT
  CASE and the first CASE statement.
 
■ Possible solution:
 
  • Insert a REM statement to comment the line. Comments and
    statement separators are permitted between SELECT CASE
    and the first CASE statement
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

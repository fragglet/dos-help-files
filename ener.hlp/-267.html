Typed variable not allowed in expression
■ Variables that are user-defined types are not permitted
  in expressions such as this, where X is a user-defined type:
 
              CALL Alpha((X))
 
■ Possible solution:
 
  • Pass elements of user-defined types as arguments. For
    example:
 
              CALL Alpha((X.FirstEl))
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

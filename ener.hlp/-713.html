Array already has control at index
■ You tried to change the name of a control that is part of
  a control array to the same name as another control, and
  the two controls have the same index value.
 
■ Possible solutions:
 
  • Change the name of the control so that it does not
    conflict with the existing control array.
 
  • Change the index value of the control so that it does
    not conflict with the index values in the existing
    control array.
 
■ If you need additional help, move the cursor to a keyword
  and press F1 to open the Help window.

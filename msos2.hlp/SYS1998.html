SYS1998
 
Explanation
 
Files with the extension .BAK are considered to be backup files, with more
up-to-date versions of the files assumed to exist. Therefore,
.BAK files will not be edited.
 
Action
 
If it is necessary to edit the .BAK file, rename the file and give it an
extension other than
.BAK. Or, copy the file and give the copy a different
filename extension.

SYS1963
 
Explanation
 
This error usually occurs when:
 
1. There is no more room on the fixed disk or diskette.
 
2. The diskette is write protected.
 
3. The filename is a duplicate name of an existing file.
 
Action
 
Do one of the following:
 
1. Remove some unwanted files to make room on the fixed disk or diskette.
 
2. Enable the diskette for writing.
 
3. Rename the duplicate file.

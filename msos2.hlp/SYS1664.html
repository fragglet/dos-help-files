SYS1664
SYS1664
 
Explanation
 
The /l parameter was specified but there is not enough room to create the
log file.
 
Action
 
Press Ctrl+Break to end the BACKUP log file procedure or press Enter to
continue without adding more entries to the log file. Delete any unwanted
files from the disk and retry the command.

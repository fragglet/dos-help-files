#define INCL_VIO
 
typedef struct _VIOSIZECOUNT {    /* viosz */
    LONG MaxCount;
    LONG Count;
} VIOSIZECOUNT;
 
The VIOSIZECOUNT structure contains the size of the VIOFONTCELLSIZE
structure.
 
Field     Description
────────────────────────────────────────────────────────────────────────────
 
MaxCount  Specifies the maximum number of occurrences of the
          VIOFONTCELLSIZE structure.
 
Count     Specifies the actual number of occurrences of the VIOFONTCELLSIZE
          structure.
 
See Also
 
DevEscape, VIOFONTCELLSIZE

MAKEP(sel, off)
 
The MAKEP macro combines a segment selector and an address offset to create
a far (32-bit) pointer to a memory address.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
sel        Specifies a segment selector. It must be a valid segment
           selector──for example, if it were created by using the
           DosAllocSeg function.
 
off        Specifies an offset from the beginning of the given segment to
           the desired byte. The offset must specify an address within the
           segment.
 
See Also
 
DosAllocSeg, OFFSETOF, SELECTOROF

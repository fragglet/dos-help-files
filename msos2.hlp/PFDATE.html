typedef struct _FDATE {    /* fdate */
    unsigned day     : 5;
    unsigned month   : 4;
    unsigned year    : 7;
} FDATE;
 
The FDATE structure is used in various other structures to specify the day,
month, and year.
 
Field  Description
────────────────────────────────────────────────────────────────────────────
 
day    Specifies the day.
 
month  Specifies the month.
 
year   Specifies the year.
 
See Also
 
FILEFINDBUF, FILESTATUS, FSINFO

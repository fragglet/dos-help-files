CS_SAVEBITS  Saves the image underneath the window as a bitmap. When the
user moves or hides the window, the system restores the image by copying the
bits; there is no need to add the area to the uncovered window's update
region. Because this operation can consume a great deal of memory, it is
recommended only for transient windows such as menus and dialog boxes, not
for main application windows.

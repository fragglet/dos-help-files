SYS1616
SYS1616
 
Explanation
 
The MODE command only accepts the following baud rates: 110, 150, 300, 600,
1200, 2400, 4800, 9600, or 19200.
 
Action
 
Retry the MODE command using one of the acceptable baud rates.

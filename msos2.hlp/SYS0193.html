SYS0193
SYS0193
 
Explanation
 
The specified file or program is either a DOS mode application program or
not compatible with OS/2.
 
Action
 
If the specified application is a DOS mode program, switch to DOS mode and
retry the command. Otherwise reinstall the application and retry the
command. If the error occurs again, contact the supplier of the application
program.

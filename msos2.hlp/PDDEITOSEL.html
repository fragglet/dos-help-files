#define INCL_WINDDE
 
 
PDDEITOSEL(pddei)
 
The PDDEITOSEL macro creates a selector from a pointer to a DDEINIT
structure. This is necessary in order to use the DosFreeSeg function to free
the memory that the DDEINIT structure uses.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
pddei      Points to a DDEINIT structure.

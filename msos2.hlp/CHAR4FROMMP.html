CHAR4FROMMP(mp)
 
The CHAR4FROMMP macro retrieves the character in the high byte of the high
word of a message parameter.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
mp         Specifies the message parameter.

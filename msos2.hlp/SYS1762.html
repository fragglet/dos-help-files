SYS1762
SYS1762
 
Explanation
 
The country information file was not present during system startup.
 
Action
sure that it exists. If the CONFIG.SYS file does not specify a country
information file, make sure that the COUNTRY.SYS file exists in the root
directory of the startup drive. Correct the problem and restart the system.

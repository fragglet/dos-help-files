LONGFROMMR(mr)
 
The LONGFROMMR macro casts a message result to a LONG type.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
mr         Specifies the message result. The following functions return this
           parameter: WinDdeRespond, WinDefAVioWindowProc, WinDefWindowProc,
           WinSendMsg, and WinSendDlgItemMsg.
 
See Also
 
MRFROMLONG, WinDdeRespond, WinDefAVioWindowProc, WinDefWindowProc,
WinSendMsg, WinSendDlgItemMsg

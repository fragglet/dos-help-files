HIUCHAR(w)
 
The HIUCHAR macro retrieves the high-order unsigned byte from the 16-bit
value specified by the w parameter.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
w          Specifies a 16-bit value.

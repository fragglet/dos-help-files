SYS1234
 
Explanation
 
The diskette and drive type are incompatible with each other or the diskette
is unusable.
 
Action
 
Make sure the diskette drives are the same type or that the correct type of
diskette is in the drive. For example, high- capacity diskettes (in a
high-capacity drive) can only be copied to high-capacity diskettes (in a
high-capacity drive).

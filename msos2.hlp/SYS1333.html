SYS1333
 
Explanation
 
The drive specified is not correct, the diskette is not in the MS OS/2
format, or the diskette is unusable.
 
Action
 
Retry the CHKDSK command using a correct drive specification. If this error
occurs again, format a diskette with MS OS/2, copy the files from the old
diskette to the new one, and retry the command.

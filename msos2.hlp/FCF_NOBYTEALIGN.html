FCF_NOBYTEALIGN

────────────────────────────────────────────────────────────────────────────
 
FCF_NOBYTEALIGN  Allows the user to move the frame window to any position on
the display. If this flag is not given, a frame window always adjusts its
position so that the x coordinate of its left edge is a multiple of 8. Using
this flag affects how quickly the system can draw the frame window.

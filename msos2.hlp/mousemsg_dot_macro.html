MOUSEMSG(pmsg)
 
The MOUSEMSG macro is used to access the WM_MOUSEMOVE message parameters.
This macro defines a MOUSEMSG structure pointer, which has the following
form:
 
struct _MOUSEMSG {
    USHORT codeHitTest;     /* mp2 */
    USHORT unused;
    SHORT  x;               /* mp1 */
    SHORT  y;
};
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
pmsg       Points to the message received by the application's window
           procedure.
 
See Also
 
WM_MOUSEMOVE, MOUSEMSG

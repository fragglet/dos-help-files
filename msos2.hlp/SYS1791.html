SYS1791
 
Explanation
 
More code page specifications have been specified in the ROM= portion of the
DEVINFO= statement in the CONFIG.SYS file than the specified printer can
handle. Initialization will continue, but only for the number of code page
specifications the printer can support.
 
Action
Reduce the number of code page specifications in the DEVINFO= statement in
the CONFIG.SYS file. Then restart the system.

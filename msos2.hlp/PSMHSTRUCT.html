SMHSTRUCT (1.2)
                                                      ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
#define INCL_WINHOOKS
 
typedef struct _SMHSTRUCT {    /* smhs */
    MPARAM mp2;
    MPARAM mp1;
    USHORT msg;
    HWND   hwnd;
} SMHSTRUCT;
 
The SMHSTRUCT structure contains information about a message that is used in
a send-message hook.
 
Field  Description
────────────────────────────────────────────────────────────────────────────
 
mp2    Specifies the second message parameter.
 
mp1    Specifies the first message parameter.
 
msg    Specifies the message.
 
hwnd   Identifies the window sending the message.

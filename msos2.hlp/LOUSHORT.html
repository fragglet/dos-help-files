LOUSHORT(l)
 
The LOUSHORT macro retrieves the low-order unsigned 16-bit word from the
32-bit value specified by the l parameter.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
l          Specifies a 32-bit value.

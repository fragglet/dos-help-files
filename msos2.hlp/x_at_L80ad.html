MS OS/2 version 1.2 changes
 
The PT_FULLSCREEN, PT_REALMODE, PT_WINDOWABLEVIO, PT_PM, and PT_DETACHED
constants replace the numeric values previously defined for the typeProcess
field. The constant PS_EXITLIST is a valid value for the rfProcStatus
field.
 
Corrections
 
The rfProcStatus specifies the process status, not the subscreen group.

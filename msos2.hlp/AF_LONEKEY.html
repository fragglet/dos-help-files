AF_LONEKEY

────────────────────────────────────────────────────────────────────────────
 
AF_LONEKEY  Specifies that the user need not press another key while the
accelerator key is down. This style is typically used with the ALT key to
specify that simply pressing and releasing the ALT key triggers the
accelerator.

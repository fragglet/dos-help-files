MRFROM2SHORT(s1, s2)
 
The MRFROM2SHORT macro creates a message result from two SHORT values.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
s1         Specifies the first SHORT value.
 
s2         Specifies the second SHORT value.

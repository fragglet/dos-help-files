MPFROM2SHORT(s1, s2)
 
The MPFROM2SHORT macro creates a message parameter from two SHORT values.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
s1         Specifies the first SHORT value (low word).
 
s2         Specifies the second SHORT value (high word).

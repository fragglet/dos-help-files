SYS1187
SYS1187
 
Explanation
 
One of the following conditions exists:
 
1. The disk is full.
2. The file or drive is in use or locked
   by another process.
3. Too many files are already open.
4. The target file is a read-only file.
5. A disk error occurred.
 
Action
 
Correct the problem or wait. Then retry the command.

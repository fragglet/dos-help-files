SYS2056
SYS2056
 
Explanation
 
This command expected more information than the input file contained. There
is not enough data to complete the process.
 
Action
 
Check the information for the command that failed. Make sure that all the
information necessary for this command to complete normally is entered in
the input file. Retry the command.

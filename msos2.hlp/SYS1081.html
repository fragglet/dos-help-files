SYS1081
SYS1081
 
Explanation
 
A permanent copy of CMD.EXE was running, and one of the following situations
occurred: 1. Another program tried to cancel this copy of
CMD.EXE. 2. CMD.EXE was started using a redirected input file and the end of
the file was reached.
 
Action
 
No Action can be taken.

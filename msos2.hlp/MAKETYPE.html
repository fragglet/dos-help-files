MAKETYPE(v, type)
 
The MAKETYPE macro casts the variable specified by the v parameter as a
variable having the type specified by the type parameter. This macro permits
the contents of the variable to be accessed as if the variable had the
specified type.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
v          Specifies the name of the variable to be cast.
 
type       Specifies the name of the data type for the cast.

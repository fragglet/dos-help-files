WS_GROUP
◄Full List►                                           ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
WS_GROUP  Specifies the first control of a group of controls in which the
user can move from one control to the next by using the ARROW keys. All
controls defined after the control with the WS_GROUP style belong to the
same group. The next control with the WS_GROUP style ends the first group
and starts a new group.

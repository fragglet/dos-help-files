WC_MLE  Creates a multiple-line entry field (MLE). An MLE is a rectangular
child window in which the user can enter multiple lines of text from the
keyboard. The user selects the MLE and gives it the input focus either by
using the mouse or the TAB key. The user can enter text when the MLE
displays an insertion point. By using the mouse or the ARROW keys, the user
can select the character or characters to delete, or select the place at
which to insert new characters. The user can also press the BKSP key to
delete characters. See also, ◄Multiple-line entry-field styles►

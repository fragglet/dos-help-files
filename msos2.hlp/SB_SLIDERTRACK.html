SB_SLIDERTRACK  User is dragging the slider. Applications that draw data
quickly can set the slider to the position given in the message and scroll
the data by the same number of units the slider has moved. Applications that
cannot draw data quickly should wait for the SB_SLIDERPOSITION code before
moving the slider and scrolling the data.

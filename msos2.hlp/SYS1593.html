SYS1593
 
Explanation
 
A correct hexadecimal string consists of up to 32 hexadecimal digits with
optional imbedded hyphens. There must be an even number of hexadecimal
digits.
 
Action
 
Check the patch instruction file for errors or contact the supplier of the
patch instruction file.

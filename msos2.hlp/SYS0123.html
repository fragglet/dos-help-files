SYS0123
SYS0123
 
Explanation
 
The system does not accept characters displayed using the keyboard
combination ALT plus 0-32 or the following characters
 
  /  [ ] : |  <  >  +  ; =  .  ?  *  "
 
Action
 
Correct the filename or volume label and retry the command.

LOUCHAR(w)
 
The LOUCHAR macro retrieves the low-order unsigned byte from the 16-bit
value specified by the w parameter.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
w          Specifies a 16-bit value.

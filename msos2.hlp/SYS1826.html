SYS1826
SYS1826
 
Explanation
 
There is not enough storage available.
 
Action
 
Make more storage available. For example, decrease the value specified for
the RMSIZE=, BUFFERS=, or TRACEBUF= statement in the CONFIG.SYS file. Then
restart the system.

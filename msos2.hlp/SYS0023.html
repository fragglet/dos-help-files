SYS0023
 
Explanation
 
MS OS/2 cannot read or write the data correctly.
 
Action
 
If the error occurred on a fixed disk, retry the command. If the error
occurs again, the fixed disk may have to be reformatted. If the error
occurred on a diskette, insert a formatted diskette or the backup diskette,
and retry the command.

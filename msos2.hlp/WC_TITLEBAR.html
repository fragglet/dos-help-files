WC_TITLEBAR  Creates a title bar. A title bar, part of a standard frame
window, performs four functions. It displays the title of the window across
the top of the frame window. It changes its highlight appearance to show
whether the frame window is active. It responds to the user──for example,
when the user drags the frame window to a new location on the display. And
it flashes, notifying the user of an event. A title bar sends messages to
its owner window when the title bar receives user input.

MIA_HILITED

────────────────────────────────────────────────────────────────────────────
 
MIA_HILITED  Highlights the menu item by inverting the colors of the item
name and background when the menu item is currently selected. The
application rarely sets this attribute directly, relying instead on the
default processing of user input to set the highlighted state of an item.

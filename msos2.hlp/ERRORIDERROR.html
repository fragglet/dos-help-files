ERRORIDERROR(errid)
 
The ERRORIDERROR macro retrieves the error number from an ERRORID value. An
ERRORID value is returned by the WinGetLastError function.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
errid      Specifies the error identifier.

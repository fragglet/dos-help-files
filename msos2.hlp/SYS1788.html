SYS1788
SYS1788
 
Explanation
 
The amount of storage needed for code page switching is not available.
 
Action
 
To activate code page switching, try one of the following:
 
1. Reduce the number of programs running on your system.
2. Edit the CONFIG.SYS file to reduce the value specified
   in the BUFFERS=, TRACEBUF=, DISKCACHE=, THREADS=, or
   RMSIZE= statement, or reduce the size allocated for a
   virtual disk in a DEVICE=VDISK.SYS statement.
3. Install another memory option in your system unit.
 
Then restart the system and retry the command.

SYS1779
 
Explanation
 
The Presentation Manager detected an error while trying to write to this
device.
 
Action
 
Make sure the device is:
 
1. Installed and connected
2. Switched on
3. Not being used by another process
4. In the proper receive mode
 
Retry the command.

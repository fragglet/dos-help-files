DDES_PABDATA(pdde)
 
The DDES_PABDATA macro retrieves a far (32-bit) pointer to the data area
that follows a DDESTRUCT structure. The following messages pass a DDESTRUCT
structure as the second parameter:
 
    WM_DDE_REQUEST
    WM_DDE_ACK
    WM_DDE_DATA
    WM_DDE_ADVISE
    WM_DDE_UNADVISE
    WM_DDE_POKE
    WM_DDE_EXECUTE
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
pdde       Points to the DDESTRUCT structure that precedes the data area.
 
See Also
 
DDES_PSZITEMNAME, DDESTRUCT

typedef struct _SIZEL {    /* sizl */
    LONG cx;
    LONG cy;
} SIZEL;
 
The SIZEL structure specifies the width and the height of a rectangle.
 
Field  Description
────────────────────────────────────────────────────────────────────────────
 
cx     Specifies the rectangle width.
 
cy     Specifies the rectangle height.

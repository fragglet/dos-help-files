SS_SYSICON

────────────────────────────────────────────────────────────────────────────
 
SS_SYSICON  Displays a system pointer icon. The ID of the icon is extracted
from the control text, as in the SS_ICON style. To display this icon, the
system calls the WinQuerySysPointer function with the specified ID.

SYS0008
SYS0008
 
Explanation
 
All available storage is in use. If segment swapping is active, the swap
file may be full, or an input/output (I/O) error may have occurred on the
auxiliary storage device that contains the swap file.
 
Action
 
Do one of the following:
 
1. Reduce the number of running programs and retry the command.
 
2. Reduce the value of BUFFERS=, TRACEBUF=, DISKCACHE=, THREADS=, RMSIZE=,
   or DEVICE=VDISK.SYS in the CONFIG.SYS file. Then restart the system and
   retry the command.
 
3. Remove unwanted files from the fixed disk or diskette for the swap file,
   then restart the system.
 
4. Install additional memory on your system.
 
5. If an input/output (I/O) error occurred on the swap file check the
   auxiliary storage device that contains the swap file.

SYS1352
 
Explanation
 
The path listed can end with one or two dots. One dot indicates the current
directory is in error. Two dots indicate the parent directory is in
error. If you did not
 
enter the /F option, no corrective Action is taken. Action
 
Enter CHKDSK /F. CHKDSK will attempt to correct the error.

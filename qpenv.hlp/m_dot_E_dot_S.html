Pascal Syntax Command
 
  Use the Pascal Syntax command to color-code Pascal
  keywords, identifiers, comments, and strings. Specific
  colors may be associated with each syntax identifier
  by using the Display command in the Options menu.
 
  The Pascal Syntax command is a toggle that turns the
  option on and off. Whenever the Syntax option is on,
  Pascal keywords appear in color and a bullet precedes
  the command on the menu:

    │• Pascal Syntax      │

Search Menu
  Use the Search menu to find text and mark lines with
  tags. Searches begin at the cursor, extend to the end of
  the file, and "wrap around" to the starting point.
  ┌──────────────────────────┐
  │ Find...                  │ ■ Search for text
  │ Selected Text     Ctrl+\ │ ■ Search for selected text*
  │ Repeat Last Find      F3 │ ■ Search for same text
  │ Change...                │ ■ Replace text
  ├──────────────────────────┤
  │ Toggle Tag      Shift+F2 │ ■ Set and turn off tag*
  │ Next Tag        Shift+F3 │ ■ Move to next tag
  │ Prev Tag        Shift+F4 │ ■ Move to previous tag
  │ Clear All Tags           │ ■ Clear all tags
  └──────────────────────────┘
* Items marked by an asterisk appear only with full menus.

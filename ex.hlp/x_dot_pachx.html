' This example demonstrates how to update a label control to reflect the
' current path for a file list box control. Double-clicking a directory name
' displays a list of that directory's files in the file list box; it also
' displays the directory's complete path in the label control.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Label
'    • Directory list box
'    • File list box
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Dir1_Change ()
     File1.Path = Dir1.Path
 END SUB
 
 SUB File1_PathChange ()
     Label1.Caption = "Path: " + Dir1.Path
 END SUB

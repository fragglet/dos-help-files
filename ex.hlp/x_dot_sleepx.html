SLEEP Statement Example
                        ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
' This example uses the SLEEP statement to suspend execution for 10 seconds.
' There is no ON event statement, so the only way to interrupt suspension
' before 10 seconds have passed is to press a key.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the example code below to the code window
' 3. Press F5 to run the example
 
 CLS                                 ' Clear the screen
 PRINT "Taking a 10 second nap..."
 SLEEP 10
 PRINT "Awake!"
 END

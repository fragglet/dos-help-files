' This example uses the ScaleHeight and ScaleWidth properties to display the
' interior height and width of a form when it is selected.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Form_Click ()
     PRINT "Interior height of form = " + STR$(ScaleHeight) _
 + CHR$(13) + "Interior width of form = " + STR$(ScaleWidth)
 END SUB

HEX$ and OCT$ Functions Example
                        ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
' This example uses the HEX$ and OCT$ functions to display the hexadecimal
' and octal values of a number you supply.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 CLS
 DO                                ' Clear the screen
     LINE INPUT "Enter Decimal Number: ", Num$
     Decimal& = VAL(Num$)
     PRINT
     PRINT "Decimal", " ="; Decimal&
     PRINT "Hex", " = "; HEX$(Decimal&)
     PRINT "Oct", " = "; OCT$(Decimal&)
     PRINT
 LOOP UNTIL Num$ = ""
 END

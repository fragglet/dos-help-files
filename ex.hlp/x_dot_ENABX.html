' This example alternates between enabling and disabling a command button
' when you type text into a text box.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Text box
'    • Command button
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Text1_Change ()
     IF Command1.Enabled = -1 THEN
         Command1.Enabled = 0
         Command1.Caption = "Disabled"
     ELSE
         Command1.Enabled = -1
         Command1.Caption = "Enabled"
     END IF
 END SUB

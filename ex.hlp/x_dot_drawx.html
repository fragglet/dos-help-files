' This example uses the DRAW statement to draw a triangle's outline in
' magenta and to paint the interior cyan.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 CLS                                   ' Clear the screen
 SCREEN 1
 DRAW "C2"                             ' Set color to magenta
 DRAW "F60 L120 E60"                   ' Draw a triangle
 DRAW "BD30"                           ' Move down into the triangle
 DRAW "P1,2"                           ' Paint interior
 SLEEP 3
 WIDTH 80
 SCREEN 0

' This example shows the effect of using the TabStop property. When you run
' this example, press the Tab key several times. Note that the Command1
' button does not receive the focus since its TabStop property has been
' set to False.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with three command
'    button controls
' 3. Set the TabStop property for the first command button (Command1) to
'    False
' 4. Press Alt+F4 to return to the programming environment
' 5. Press F5 to run the example

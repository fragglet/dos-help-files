' This example shows the effect of using the Style property settings with
' combo box controls.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with three combo
'    box controls
' 3. Set the Style properties of the second and third combo box:
'    • For Combo2, Style = 1 (Simple Combo)
'    • For Combo3, Style = 2 (Dropdown List)
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Form_Load ()
         Combo1.ADDITEM "Luscious"              ' Add the entries
         Combo1.ADDITEM "Lucy"
         Combo1.ADDITEM "Edwards"
         Combo2.ADDITEM "Edmark"
         Combo2.ADDITEM "Doo"
         Combo2.ADDITEM "Drew"
         Combo3.ADDITEM "Zoo"
         Combo3.ADDITEM "Zebra"
         Combo3.ADDITEM "Xylox"
 END SUB

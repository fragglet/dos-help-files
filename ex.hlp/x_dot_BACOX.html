' This example resets background colors randomly twice each second for a form
' and a picture control.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Picture box
'    • Timer
' 3. Set the timer control's Interval property to 500
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Timer1_Timer ()
     BackColor = (INT(RND * 15))
     Picture1.BackColor = (INT(RND * 15))
 END SUB

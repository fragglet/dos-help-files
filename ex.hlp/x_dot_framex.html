' This example uses the frame control to group two option buttons. When you
' run the example and click the form, the selected option button will change.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Frame
'    • Two option buttons
'    Note: Make sure you create the frame first. When you create the option
'    buttons, place them on the frame control to visually group them
'    together.
' 3. Set the frame control's Caption property to "Option Group."
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Form_Click ()
     IF Option1.Value = -1 THEN
          Option2.Value = -1
          Option1.Value = 0
     ELSE
          Option1.Value = -1
          Option2.Value = 0
     END IF
 END SUB

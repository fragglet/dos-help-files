' This example illustrates the use of the Load event by specifying the
' contents of a combo box on a form when it is loaded.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with a combo box
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Form_Load ()
     Combo1.ADDITEM "Medieval to Renaissance"   ' Add items to list
     Combo1.ADDITEM "Baroque to Classical"
     Combo1.ADDITEM "Rock 'n Roll"
     Combo1.ADDITEM "Ethnic/Folk"
     Combo1.ListIndex = 2                       ' Set default selection
 END SUB

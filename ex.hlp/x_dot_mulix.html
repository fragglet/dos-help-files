' This example illustrates the MultiLine property.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with two large text
'    boxes
' 3. Set the MultiLine property of the first text box (Text1) to True
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Form_Load ()
     Text1.Text = "This multiline text box is great because no matter" _
     + "what the width of the box, text automatically wraps!"
     Text2.Text = "This text box truncates text rather than wrapping it!"
 END SUB

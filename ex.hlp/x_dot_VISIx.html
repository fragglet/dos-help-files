' In this example, a control disapears and reappears when the command button
' is clicked.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Picture box
'    • Command button
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Command1_Click ()
    IF Picture1.Visible = 0 THEN
          Picture1.Visible = -1
          Command1.Caption = "Visible"
    ELSE
          Picture1.Visible = 0
          Command1.Caption = "Invisible"
    END IF
 END SUB

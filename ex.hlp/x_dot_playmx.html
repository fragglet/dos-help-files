PLAY Statement (Music), and VARPTR$ Function Example

──────────────────────────────────────────────────────────────────────────────
' This example uses the PLAY statement and VARPTR$ function to play music.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 scale$ = "CDEFGAB"                 ' Play scale in seven different octaves
 PLAY "L16"
 FOR i% = 0 TO 6
    PLAY "O" + STR$(i%)
    PLAY "X" + VARPTR$(scale$)
 NEXT i%

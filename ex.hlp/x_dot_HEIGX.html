' This example sets the size of a form to 75% of screen size, and centers the
' form when it is clicked.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Form_Click ()
     Width = (SCREEN.Width * .75)          ' Set width of form
     Height = (SCREEN.Height * .75)        ' Set height of form
     Left = ((SCREEN.Width - Width) / 2)   ' Center form horizontally
     Top = ((SCREEN.Height - Height) / 2)  ' Center form vertically
 END SUB

' This example uses the LEFT$ function to print the leftmost five characters
' of a string and the RIGHT$ function to print the rightmost five characters
' of the same string.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the example code below to the code window
' 3. Press F5 to run the example
 
 CLS                             ' Clear the screen
 a$ = "Visual Basic for MS-DOS"
 PRINT LEFT$(a$, 12)             ' Output is:  Visual Basic
 PRINT RIGHT$(a$, 6)             ' Output is:  MS-DOS

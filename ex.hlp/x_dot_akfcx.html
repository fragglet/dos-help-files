' This example illustrates the effect of changing the value of the
' ACCESSKEY_FORECOLOR constant.
 
' To try this example:
' 1. Choose New Form from the File menu to create a form with a command
'    button
' 2. Set the command button Caption property to "&DoIt"
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 ' $INCLUDE: 'CONSTANT.BI'          ' Include ControlPanel definitions
 SUB Command1_Click ()
     SCREEN.ControlPanel(ACCESSKEY_FORECOLOR) = 13
 END SUB

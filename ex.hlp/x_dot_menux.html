' This example uses the Menu Design window to creat a partial menu. When you
' run the example, press Alt+E to display the Edit menu commands.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form
' 3. Choose Menu Design Window from the Window menu. Enter the following
'    settings:
 
'    Caption:    Ctlname:    Index:    Key:
'    ════════    ════════    ══════    ═════════════════════════════════════
'    File        Main        1         Press Enter
'    &Edit       Main        2         Press Enter, Alt+R
'    Cut         Edit        1         Press Enter
'    Copy        Edit        2         Press Enter
'    Paste       Edit        3         Press Enter
 
' 4. Choose Done.
' 5. Press Alt+F4 to return to the programming environment
' 6. Press F5 to run the example

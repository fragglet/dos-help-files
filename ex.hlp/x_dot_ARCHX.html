' This example switches the Archive property on and off. To see the effects
' of the setting, you must have already set the archive bit of at least one
' file to zero (0).
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'   • File list box
'   • Check box
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Check1_Click ()
     File1.Archive = Check1.Value
 END SUB

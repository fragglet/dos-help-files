HIDE and SHOW Methods Example
                        ◄Example►                ◄Contents►  ◄Index►  ◄Back►
──────────────────────────────────────────────────────────────────────────────
' The example uses the HIDE method to hide a form and the SHOW method to show
' the form.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 DECLARE SUB HideShowDemo ()
 
 SUB Form_Click ()
         CALL HideShowDemo
 END SUB
 
 SUB HideShowDemo ()
        HIDE                  ' Hide form
        Msg$ = "Choose OK to make the form reappear."
        MSGBOX Msg$           ' Display message
        SHOW                  ' Show form again
 END SUB

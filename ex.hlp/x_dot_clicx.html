' In this example, each time a picture box is clicked, it moves diagonally
' across a form.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with a picture
'    box positioned in the lower-left corner of the form
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Picture1_Click ()
     Picture1.Move Picture1.Left + 5, Picture1.Top - 2
 END SUB

' This example uses the FORMAT$ function to format date/time information.
' The current date and time are returned as a serial mumber by the NOW
' NOW function.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 CLS                               ' Clear the screen
 PRINT "Today's date is "; FORMAT$(NOW, "dd-mmm-yy"); "."
 PRINT "The correct time is "; FORMAT$(NOW, "hh:mm:ss AM/PM"); "."

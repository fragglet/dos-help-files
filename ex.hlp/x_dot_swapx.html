' This example uses the SWAP statement to exchange the values of two
' variables.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 CLS                           ' Clear the screen
 a% = 1: b% = 2
 PRINT "Before: "; a%, b%
 SWAP a%, b%
 PRINT "After:  "; a%, b%

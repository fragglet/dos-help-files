' This example shows how to use the Default property. When the Enter key
' is pressed, the Click event for the default command button is executed.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • A wide picture box
'    • Two command buttons
' 3. Set the second command button's Default property to True
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Command2_Click ()
     Msg$ = "The Default property is True for command button 2"
     Picture1.PRINT Msg$
 END SUB

' This example displays the Value, Max, and Min property settings of a
' horizontal scroll bar in a text box.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with:
'    • Wide text box
'    • Horizontal scroll bar
' 3. Set scroll bar properties:
'    • Max = 1000
'    • LargeChange = 10
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB HScroll1_Change ()
     Text1.Text = "Value=" + STR$(HScroll1.Value) + " Min=" + _
     STR$(HScroll1.Min) + "Max=" + STR$(Hscroll1.Max)
 END SUB

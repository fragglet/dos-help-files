' This example uses the ERASE statement with the $DYNAMIC and $STATIC
' metacommands:
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Copy the code example below to the code window
' 3. Press F5 to run the example
 
 CLS                       ' Clear the screen
 REM $DYNAMIC
 DIM a(100, 100)
 ERASE a                   ' Deallocates array A
 REDIM a(5, 5)             ' Redimension array A
 REM $STATIC
 DIM B(50, 50)
 PRINT B
 ERASE B                   ' This sets all elements of B equal to zero
                           ' B still has the dimensions assigned by DIM

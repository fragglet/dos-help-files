' This example changes the caption on the command button each time the user
' clicks the button.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with a command
'    button control
' 3. Press Alt+F4 to return to the programming environment
' 4. Copy the code example below to the form module
' 5. Press F5 to run the example
 
 SUB Command1_Click ()
     ' Check caption, then change it.
     IF Command1.Caption = "Clicked" THEN
          Command1.Caption = "OK"
     ELSE
          Command1.Caption = "Clicked"
     END IF
 END SUB

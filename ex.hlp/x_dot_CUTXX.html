CurrentX and CurrentY Property Example

──────────────────────────────────────────────────────────────────────────────
' This example displays the CurrentX and CurrentY property values of a
' picture box control when you click the form.
 
' To try this example:
' 1. Choose New Project from the File menu
' 2. Choose New Form from the File menu to create a form with a
'    Picture box control about 3 inches wide
' 3. Set the picture box's AutoRedraw property to True
' 4. Press Alt+F4 to return to the programming environment
' 5. Copy the code example below to the form module
' 6. Press F5 to run the example
 
 SUB Form_Load ()
      ' Display the current X and Y location.
     MsgA$ = "The CurrentX and CurrentY positions within the picture box are:"
     MsgB$ = STR$(Picture1.CurrentX) + "," + STR$(Picture1.CurrentY)
     MsgC$ = "Choose OK, then click the form to print to the picture box."
     MSGBOX MsgA$ + MsgB$ + CHR$(10) + CHR$(13) + MsgC$
 END SUB
 
 SUB Form_Click ()
     Picture1.PRINT "This is just a test..." ' Print to the form.
      ' Display the current the new X and Y location.
     MsgA$ = "CurrentX and CurrentY values have changed to:"
     MsgB$ = STR$(Picture1.CurrentX) + "," + STR$(Picture1.CurrentY)
     MSGBOX MsgA$ + MsgB$
 END SUB

ASC returns a numeric value that is the ASCII code for the first
character in the argument stringexpression$.
 
ASC(stringexpression$)
 
Usage Notes
    ■ If the argument stringexpression$ is null, BASIC generates the
      run-time error message, "Illegal function call."

RTRIM$ removes trailing (right) spaces from a string.
 
RTRIM$(stringexpression$)
 
Usage Notes
    ■ The argument stringexpression$ can be any string expression.
    ■ The RTRIM$ function works with both fixed- and variable-length string
      variables.

HEX$ returns a string that represents the hexadecimal value of the argument.
 
HEX$(numeric-expression&)
    ■ The numeric-expression& is rounded to an integer or, if the expression
      is outside the integer range, a long integer, before the HEX$ function
      evaluates it.

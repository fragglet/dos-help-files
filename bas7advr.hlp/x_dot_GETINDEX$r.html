GETINDEX$ takes the number of an open ISAM table and returns the name of
the current index.
 
GETINDEX$ (filenumber%)
    ■ The argument filenumber% is the number used in the OPEN statement
      to open the table.
 
Usage Notes
    ■ If the current index is the NULL index, the value returned is a null
      string. The NULL index represents the order in which records were
      added to the file.

BOF tests whether the current position is at the beginning of an ISAM table.
 
BOF(filenumber%)
    ■ The argument filenumber% is the number used in the OPEN statement
      to open the table.
 
Usage Notes
    ■ BOF returns true (nonzero) if the current position is at the beginning
      of a table. The beginning of an ISAM table is the position before the
      first record according to the current index.

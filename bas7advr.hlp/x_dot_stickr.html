STICK returns joystick coordinates.
 
STICK(n%)
 
Usage Notes
    ■ You must use STICK(0) before you use STICK(1), STICK(2), or STICK(3)
      because STICK(0) not only returns the x coordinate of joystick A,
      but also records the other joystick coordinates. These recorded
      coordinates are returned by calling STICK(1), STICK(2), or
      STICK(3).

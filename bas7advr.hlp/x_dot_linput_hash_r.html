LINE INPUT # reads an entire line from a sequential file to a string variable.
 
LINE INPUT #filenumber%, stringvariable$
 
Usage Notes
    ■ The LINE INPUT # statement reads all characters in the sequential
      file up to a carriage return. It then skips over the carriage-return
      and line-feed sequence.
 
Programming Tips
    ■ LINE INPUT # is especially useful if a text file is being read a line
      at a time.

FIX returns the truncated integer part of x#.
 
FIX(x#)
 
Usage Notes
    ■ FIX(x#) is equivalent to SGN(x#)*INT(ABS(x#)). The difference between
      FIX and INT is that for negative x#, FIX returns the first negative
      integer greater than x#, while INT returns the first negative integer
      less than x#.

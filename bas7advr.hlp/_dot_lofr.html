LOF returns the size of a file, the number of records in an ISAM table, or,
when used with OPEN COM, the number of bytes free in the output buffer.
 
LOF(filenumber%)
    ■ The argument filenumber% is the number used in the OPEN statement
      to open the table.
 
Important
    ■ LOF can be used only on disk files, ISAM tables, or COMn: devices.
      It cannot be used with the BASIC devices SCRN, KYBD, CONS,
      LPTn, or PIPE.

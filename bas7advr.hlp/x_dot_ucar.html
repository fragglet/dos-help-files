UCASE$ returns a string expression with all letters in uppercase.
 
UCASE$(string-expression$)
 
Usage Notes
    ■ The UCASE$ function works with both variable- and fixed-length
      strings.
    ■ The UCASE$ and LCASE$ statements are helpful in string-comparison
      operations that are not case sensitive.

VIEW PRINT sets the boundaries of the screen text viewport.
 
VIEW PRINT [topline% TO bottomline%]
 
Usage Notes
    ■ Without the arguments topline% and bottomline%, the VIEW PRINT statement
      initializes the whole screen area as the text viewport. The number of
      text lines in the screen depends on the screen mode and whether or not
      the /H option was used when BASIC was started. For more information,
      see the WIDTH statement details.

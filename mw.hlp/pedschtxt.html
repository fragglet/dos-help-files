SEARCHING FOR TEXT
 
Overview: <Edit Search command>
 
To search for specific text in your document:
1. Choose Edit Search (Alt,E,S).
2. Type text you want to search for.
3. Turn on Whole Word if the text you are
   searching for is a whole word. Otherwise, Word
   searches for all occurrences of the letter
   combination in the search text.
4. Turn on Match Upper/Lowercase to search for
   text that matches the capitalization of the
   search text.
5. Select either Up or Down for the direction
   you want to search the document.
6. Choose OK.
 
To repeat the search, press Shift+F4.
 
Note: For information about searching for special
      characters, see:
      <Search and replace special characters>.

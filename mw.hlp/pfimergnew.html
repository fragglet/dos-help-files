MERGING TO A NEW DOCUMENT
 
Overview: <File Print Merge command>
 
Merging to a new document allows you to preview
and edit the merged main and data documents in a
new document before sending merged information to
a printer.
 
To merge to a new document:
1. Open the main document.
2. Choose File Print Merge (Alt,F,M).
3. Choose the New Document button.
4. In the Document Name box, type a name for the
   new, merged document.
5. Choose OK.
6. To view the new document, choose File Open
   and specify the new document name.

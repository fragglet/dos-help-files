UTILITIES LINE DRAW
 
Procedures: <Using line draw>
 
Use the Utilities Line Draw command to draw lines
in your document with the direction keys.
 
■ Lines or portions of lines created in line draw
  mode can be deleted, copied, or moved like
  normal text.
 
■ Lines follow the cursor as soon as you turn on
  Line Draw. If you reverse the direction of the
  cursor, it moves over the previously drawn line
  without erasing it.
 
■ The default line draw character is a single
  line. Use the <Utilities Customize command> to
  change the default character.
 
■ Press Esc or click the left mouse button to
  exit line draw mode.
 
More Help: <Utilities menu commands>

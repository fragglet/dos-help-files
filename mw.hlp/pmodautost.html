MODIFYING AN AUTOMATIC STYLE
 
Overview: <Format Define Styles command>
 
To modify an <automatic style>:
1. Choose Format Define Styles (Alt,T,D) to
   open the style sheet window.
2. Choose Insert New Style (Alt,I,N).
3. Select the Style Type you want.
4. In the Style I.D. box, type the style I.D. for
   the automatic style, or select a style I.D.
   from the list.
5. Choose OK.
6. Modify the style's format using the commands
   on the Format menu.
 
More Help: <Working with styles and style sheets>
 
Using Manual: Ch. 28, "Style Sheets"

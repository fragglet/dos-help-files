CHANGING DEFAULT SETTINGS FOR MARGINS
 
Overview: <Format Margins command>
 
To change <default> settings for margins:
1. Choose Format Margins (Alt,T,M).
2. Type the default measurements you want to use.
3. Turn on the Use As Default check box.
4. Choose OK.
 
More Help: <Formatting margins>
 
Using Manual: Ch. 7, "Page Formatting"

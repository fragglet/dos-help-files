INSERT FILE
 
Procedures: <Importing and linking files>
 
Use the Insert File command to insert or link
documents and spreadsheets to the active
document.
 
■ You can import all or part of a Microsoft
  Multiplan, Microsoft Excel, or Lotus(R) 1-2-3
  spreadsheet into a Word document. An imported
  spreadsheet can be no larger than 32K.
 
■ New imported spreadsheet data or text that is
  linked should not be inserted within other
  imported areas.
 
■ Word inserts ".D." codes before and after
  linked text and ".L." codes before and after
  linked spreadsheet data as hidden text to mark
  imported areas in the document.
 
■ To update files, Word imports linked text again
  and replaces existing text.

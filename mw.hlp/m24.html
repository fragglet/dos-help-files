WINDOW MOVE
 
Procedures: <Moving a window>
 
Use the Window Move command to reposition the
active window using the Direction keys.
 
■ When you choose Window Move, the window
  border is highlighted and the word MOVE is
  displayed at the right end of the <status bar>.
  To set the new window position and return to
  normal view, press Enter.
 
■ You can position windows so that two or more
  documents or views of the same document display
  at once.
 
■ Certain controls, such as scroll bars and the
  ribbon and ruler controls, can be hidden if the
  window is moved partially off the screen. To
  use these controls, move the window so they are
  visible.

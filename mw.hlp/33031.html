COLORS DIALOG BOX                 (Screen 1 of 1)
 
Things To Color
  You can scroll through the list of things to
  color. Word displays a sample of what the
  selected item looks like in the box below.
  To change the color of a selected item, select
  the color you want from the color table.
 
Color Set
  You can reset window colors to the default
  settings by choosing from the Color Set list.

CHANGING FONT NAME, POINT SIZE, OR FONT COLOR
 
Overview: <Format Character command>
 
To change font name, point size, or font color:
1. Select the text you want to change.
   To change the font for the whole document,
   press Ctrl+5 (on the numeric keypad) or
   Shift+F10 to select the entire document.
2. Choose Format Character (Alt,T,C).
3. In the Font box, select a font name.
   You must select a font before you can
   select a point size or color.
4. In the Point Size box, select a size.
5. In the Color box, select a color.
6. Choose OK.

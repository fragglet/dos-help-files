dScrap
Scrap
 
A temporary storage area that holds the text you
last removed from a document with the Edit Cut
command. The scrap also holds text you copy with
the Edit Copy command. The contents of the scrap
are displayed between braces ( { } ) in the
status bar. To place the contents of the scrap
into your document, place the cursor where you
want the text, and then choose the Edit Paste
command or press Shift+Insert. (Note: If the
Insert function has been changed with the
Utilities Customize command, Shift+Insert will
not work.)

STOPPING PRINTING
 
Overview: <File Print Queue command>
 
To stop printing a file:
1. Choose File Print Queue (Alt,F,Q).
2. Choose the Stop Queue button.
   The rest of the file that is currently
   printing is canceled.
 
More Help: <Queueing printing>
 
Using Manual: Ch. 10, "Printing"

THE PRINT QUEUE IS STILL ACTIVE.
DO YOU WANT TO QUIT ANYWAY?
 
One or more documents are in the print queue
waiting to print. If you exit Word, printing will
be canceled.
 
■ To cancel the remaining printing, choose OK.
■ To finish the queued printing, choose Cancel.
  When printing is complete, choose File Exit
  Word again to exit Word.

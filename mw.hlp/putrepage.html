REPAGINATING A DOCUMENT
 
Overview: <Utilities Repaginate Now command>
 
To set page breaks:
1. Choose Utilities Repaginate Now (Alt,U,P).
2. Turn on Confirm Page Breaks if you want to
   confirm the placement of each page break.
3. Choose OK to start repagination.
 
To modify page breaks:
1. Choose Utilities Repaginate Now (Alt,U,P).
2. Turn on Confirm Page Breaks.
3. Choose OK.
4. Choose Enter to confirm each <soft> page
   break position or use the direction keys to
   reposition the page break.
   If the page break shown is a <hard> page
   break, you can only confirm its position or
   delete it.
 
Using Manual: Ch. 10, "Printing"

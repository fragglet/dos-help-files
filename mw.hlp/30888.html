THE SEARCH FORMAT IS NOT FOUND IN THE SELECTION.
 
1. Word searches only within the text you have
   selected. If you want to search the entire
   document or all text after the cursor, cancel
   the selection.
2. Choose Edit Search and Search For Formatting
   Only again. Then select the format you want to
   search for.
3. Choose OK.

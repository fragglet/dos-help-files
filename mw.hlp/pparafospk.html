FORMATTING A PARAGRAPH USING SPEED KEYS
 
Overview: <Format Paragraph command>
 
To format paragraphs using speed formatting keys:
1. Select the paragraph(s) you want to format.
2. Press Ctrl+the key for the format you want:
 
   For this format:              Press Ctrl+
   Centered alignment            C
   Left alignment                L
   Right alignment               R
   Justified alignment           J
   Increase indent               N
   Decrease indent               M
   Hanging indent                T
   Single space                  1
   Double space                  2
   Blank line before paragraph   O (the letter)
   Normal paragraph              X
   Indent from left and right    Q
 
Note: If you have assigned <key codes> to
      <macros>, press Ctrl+A and then
      press the format key. This prevents you
      from accidentally running a macro.

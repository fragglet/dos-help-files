SPELLING DIALOG BOX               (Screen 1 of 4)
 
Spell Indicator Message
  This line displays the word and what the spell
  checker found wrong with it. For example, "Not
  found" indicates the displayed word was not
  found in the current dictionary.
 
Replace With
  Displays the suggested spelling change if any.
  You can also type the correction you want here.
 
Suggestions
  You can select one of the suggested spellings.
  The selection will appear in the Replace With
  box.
SPELLING DIALOG BOX               (Screen 2 of 4)
 
Add To Dictionary
  Select the dictionary you want to add the
  selected word to.
 
Change
  Changes all occurrences of the misspelled word
  in the document with the Replace With word
  spelling.
 
Suggest
  Suggests alternative spelling possibilities.
SPELLING DIALOG BOX               (Screen 3 of 4)
 
Ignore
  Ignores all occurrences of the word during the
  current check session.
 
Add
  Adds the highlighted word to the selected
  dictionary.
 
Options
  Displays the Spelling Options dialog box.
 
Undo
  Returns you to, and undoes, your last spelling
  correction.
SPELLING DIALOG BOX               (Screen 4 of 4)
 
Remember Correction
  Replaces the misspelled word with the Replace
  With word and makes the correction for all
  subsequent occurrences of the misspelled word
  during the current and future Word sessions.

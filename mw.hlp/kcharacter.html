FORMATTING
 
Character Formatting
 
To apply:                      Press:
Bold                           Ctrl+B
Double underline               Ctrl+D
Hidden text format             Ctrl+H
Italic                         Ctrl+I
Remove character formatting    Ctrl+Spacebar
Remove character formatting    Ctrl+Z
 without changing to default
 font or font size
Small caps                     Ctrl+K
Subscript                      Ctrl+Equal sign
Superscript                    Ctrl+Plus sign
Underline                      Ctrl+U
 
Paragraph Formatting
 
To apply:                      Press:
Blank line before paragraph    Ctrl+O
Centered alignment             Ctrl+C
Double space                   Ctrl+2
Hanging indent                 Ctrl+T
Indent from left and right     Ctrl+Q
Justified alignment            Ctrl+J
Left alignment                 Ctrl+L
Nested format                  Ctrl+N
Remove paragraph formatting    Ctrl+X
Right alignment                Ctrl+R
Single space                   Ctrl+1
Style                          Ctrl+S
Unnested format                Ctrl+M
 
Note: If you use a macro key code that begins
      with the same letter as a speed formatting
      key, press Ctrl+A and then press the
      formatting key.

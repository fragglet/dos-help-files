MOVING INDENTS USING THE RULER
 
Overview: <View Ruler command>
 
You must use a mouse to move indents with the
ruler. Use the Format Tabs command to move indents
with the keyboard.
 
To move <indents> using the ruler:
1. Display the ruler by clicking the ruler/ribbon
   icon.
2. Move the cursor to the paragraph in which you
   want to move the indents or select a group of
   paragraphs.
3. Use the right mouse button to <drag> the
   left ([) or right (]) indent symbol to the
   position on the ruler you want.
 
To move the first-line indent:
1. Move the left indent ([) to expose the first
   line indent (|).
2. Move the first line indent as you would the
   right and left indents, using the right
   mouse button.
3. Reposition the left indent at the location
   you want.

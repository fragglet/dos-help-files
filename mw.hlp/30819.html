AN ENDWHILE INSTRUCTION MUST FOLLOW A WHILE
INSTRUCTION.
 
Word cannot run (a) a macro that has an ENDWHILE
instruction that is not preceded by a WHILE
instruction, or (b) a WHILE instruction that is
not followed by an ENDWHILE instruction.
 
â–  Edit the macro so that it includes a WHILE
  instruction for each ENDWHILE instruction.

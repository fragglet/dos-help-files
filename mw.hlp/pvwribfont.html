CHANGING FONTS USING THE RIBBON
 
Overview: <View Ribbon command>
 
To change the font using the ribbon:
 
Using the keyboard:
1. Select the text you want to change, or
   position the cursor where you want to begin
   typing text with the new font.
2. Press Ctrl+F.
3. Press the Up Arrow and Down Arrow keys to
   scroll through the list of available fonts.
4. With the new font in the Font box, press
   Enter.
 
Using the mouse:
1. Select the text you want to change, or
   position the cursor where you want to begin
   typing text with the new font.
2. Click the arrow to the right of the Font box.
   A complete list of available fonts is
   displayed.
3. Click the font you want to use.
 
More Help: <Format Character command>
 
Using Manual: Ch. 6, "Text Formatting: Characters
                      and Paragraphs"

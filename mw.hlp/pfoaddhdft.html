ADDING OR REMOVING A HEADER OR FOOTER
 
Overview: <Format Header/Footer command>
 
To add a header or footer:
1. Select the paragraph(s) you want as a header
   or footer; this paragraph should be at the top
   of the first page where you want it to appear.
2. Choose Format Header/Footer (Alt,T,H).
   If the Format Header/Footer command is
   dimmed, choose View Layout to turn off layout
   view and activate the command.
3. To format the paragraph as a header, select
   the Header option; to format the paragraph as
   a footer, select the Footer option.
4. To align the header or footer with the left
   margin, select the Left Margin option; to
   align the header or footer with the edge of
   the paper, select the Edge Of Paper option.
5. Choose OK.
 
To remove headers or footers:
1. Select the entire header or footer paragraph.
2. Choose Format Header/Footer (Alt,T,H).
3. In the Format As box, select the None option.
4. Choose OK.
 
More Help: <Formatting headers and footers>

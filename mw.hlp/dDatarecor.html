dDatarecor
Data record
 
A paragraph in a data document that stores the
varying text to be included in a single copy of a
form document, such as a form letter. A data
document can contain multiple data records, one
for each copy of the form document. The fields,
or pieces of text, in a data record are separated
by commas or tab characters and listed in the
same order as the field names in the header
record of the data document. See also: Form
document; Header record.

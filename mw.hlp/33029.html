33029
SAVE GLOSSARY DIALOG BOX          (Screen 1 of 1)
 
File Name
  Type the name of the glossary file you want to
  save, or select the glossary name from the
  Files box.
 
Files
  Displays all available glossary files in the
  current directory. Select the glossary file you
  want to save.
 
Directories
  Select the directory you want to save the
  glossary file in. If nothing is selected, the
  glossary file is saved in the current directory.

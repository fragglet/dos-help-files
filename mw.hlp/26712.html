THERE ARE NOT ENOUGH FILE HANDLES.
SET FILES=20 IN THE CONFIG.SYS FILE.
 
Word creates and uses temporary files to record
changes as you work on a document. Your system
needs to be set up so that at least 20 system
files can be open simultaneously.

  directory includes the line FILES=20.

UPDATING IMPORTED TEXT
 
Overview: <Insert File command>
 
To update imported text:
1. Select the area in the document that you want
   to update, or select the entire document if
   there is more than one area to update.
   Remember to include the hidden codes.
2. Choose Insert File (Alt,I,F) or press F9.
3. Choose Update Link.
   Word pauses at each imported area.
4. To instruct Word to import the new text and
   overwrite the previous, choose Y.
   To instruct Word not to update the text,
   choose N.
   Word then scrolls to the next imported area.
5. Choose Cancel to stop updating.
 
More Help: <Importing and linking files>
 
Using Manual: Ch. 27, "Spreadsheet or Document
              Linking"

PRINTING LINE NUMBERS
 
Overview: <Format Section command>
 
To set line numbers for printing:
1. Choose Format Section (Alt,T,S).
2. Turn on the Add Line Numbers check box.
3. Choose OK.
4. To print the document with line numbers,
   choose File Print (Alt,F,P).
 
To specify where line numbers are printed:
1. Choose Format Section (Alt,T,S).
2. Turn on the Add Line Numbers check box.
3. In the From Text box, type a measurement to
   indicate the distance between the text and the
   line numbers. This number must be smaller than
   the left margin measurement.
4. Choose OK.
 
Note: By <default>, Word prints line numbers
      0.4 inch from the left margin.

dBlock
Block
 
A part of a document. Blocks can range in size
from a single character, to many paragraphs, to
an entire document. In Word, a block of text can
be the selection. Once you have selected a block
of text, you can move it using the mouse (position
cursor, press Ctrl and click right mouse button),
the Edit Cut and Edit Paste commands, or the
Delete and Insert keys. (Note: Delete and Insert
function this way only if you turn off the Use INS
For Overtype Key option in the Utilities Customize
dialog box.) See also: Select; Selection.

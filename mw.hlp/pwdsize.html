CHANGING WINDOW SIZE
 
Overview: <Window Size command>
 
To change the size of the active window:
 
Using the mouse:
1. Point to the size icon (┘) in the lower-right
   corner of the window border.
2. <Drag> the size icon to the window size you
   want.
 
Using the keyboard:
1. Activate the window you want to resize.
2. Choose Window Size (Alt,W,S) or press Ctrl+F8.
3. Use the Direction keys to move the right and
   bottom window borders.
4. Press Enter to set the window size.

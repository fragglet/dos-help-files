UTILITIES CALCULATE
 
Procedures: <Calculating an expression>
 
Use the Utilities Calculate command to perform
mathematical calculations for numeric selections
containing the following symbols:
 
   +  addition
   -  subtraction
   *  multiplication
   /  division
   %  percentage
 
More Help: <Utilities menu>
 
Using Manual: Ch. 23, "Math"

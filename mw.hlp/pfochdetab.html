CHANGING DEFAULT TAB STOPS
 
Overview: <Utilities Customize command>
 
To change default tab stops:
1. Choose Utilities Customize (Alt,U,U).
2. Type a new measurement in the Default Tab box.
3. Choose OK.
 
To return all tab stops to default positions:
1. Position the cursor in the paragraph
   containing tab stops you want to delete, or
   select a group of paragraphs.
   Note: If you select a group, only the tab
         stops common to all paragraphs will be
         shown on the ruler or in the dialog box.
2. Choose Format Tabs (Alt,T,T).
3. Choose the Clear All button.
4. Choose OK.

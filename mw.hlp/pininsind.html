INSERTING AND FORMATTING INDEX CODES QUICKLY
 
Overview: <Insert Index command>
 
To quickly insert and format index codes:
1. To see hidden text as you type it, turn on
   Hidden Text in the View Preferences dialog
   box.
2. Press Ctrl+H to format as Hidden Text.
3. Type .i.
4. Select the index code and choose the Edit
   Glossary command.
5. Type a unique glossary name and choose
   Define.
6. To quickly insert the index code, move the
   cursor to the first word of the entry, type
   the glossary name, and press F3.

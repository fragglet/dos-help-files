MERGING AND PRINTING DOCUMENTS
 
Overview: <File Print Merge command>
 
To merge and print files:
1. Open the main document in the active window.
2. Choose File Print Merge (Alt,F,M).
3. Choose the Print button.
4. Choose OK.
   Word locates the data document you specified
   in the DATA instruction and inserts data from
   it into the main document as each copy is
   printed.

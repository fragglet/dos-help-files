EDIT COPY
 
Procedures: <Copying text or graphics>
 
Use the Edit Copy command to move a copy of the
selected text or graphic to the <scrap> without
deleting it from its original location. This
command is unavailable if nothing is selected.
 
■ Copying to the scrap is a quick way to move
  a copy of text or graphics within a document,
  or between documents.
 
■ Copying to the scrap replaces any text or
  graphics previously stored there.
 
■ Choose Edit Undo (Alt+Backspace) immediately if
  you accidentally copy text or graphics.
 
■ Use the <glossary> for long-term storage of
  frequently used text or graphics.
 
■ If you have chosen to use Word 5.0 function
  keys or to use Insert as the overtype key, see
  the Word 5.0 To 5.5 Help topics for
  the correct key functionality.
 
■ Press Ctrl+Ins to copy selected text or
  graphics.

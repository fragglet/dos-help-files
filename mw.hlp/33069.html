APPLY STYLE DIALOG BOX            (Screen 1 of 2)
 
Key Code
  Type the key code for the style you want to
  apply. If the style has no key code, indicate
  the style to be applied in the Style I.D. box.
 
Style Type
  Select the type of style you want to apply:
  Character   Applies character styles such
              as font, superscript and subscript,
              bold, or italics.
  Paragraph   Applies the style to the selected
              paragraph(s).
  Section     Applies the style to a document or
              a section of a document.
APPLY STYLE DIALOG BOX            (Screen 2 of 2)
 
Style I.D.
  Type the style I.D. for the style you want to
  apply, or select a style I.D. from the list.
 
Style to Apply
  Lists the style information for the style
  currently in the Style I.D. box. To change the
  style information, choose the Define Styles
  button.
 
Define Styles
  Opens the style sheet window, where you create
  and revise styles.

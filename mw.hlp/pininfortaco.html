INSERTING AND FORMATTING TABLE CODES QUICKLY
 
Overview: <Insert Table of Contents command>
 
To quickly insert and format table codes:
1. Insert a table code as hidden text.
2. Select the table code and choose the Edit
   Glossary command.
3. Type a short glossary name for this code, and
   choose Define.
4. To quickly insert the table code, move to the
   beginning of the first word of a table of
   contents entry, type the assigned glossary
   name, and press F3.
 
More Help: <Creating a table of contents>

ATTACHING OR DETACHING A STYLE SHEET
 
Overview: <Format Attach Style Sheet command>
 
To attach a style sheet:
1. Choose Format Attach Style Sheet (Alt,T,A).
2. Select a style sheet from the Files list, or
   type the name of the style sheet in the Style
   Sheet Name box.
3. Choose OK.
 
To detach the current style sheet:
1. Choose Format Attach Style Sheet (Alt,T,A).
2. In the Style Sheet box, select the Detach
   option.
3. Choose OK.
4. Word asks if you want to convert to direct
   formatting:
   ■ Choose Yes if you want direct formatting.
   ■ Choose No to lose all formatting.
 
Note: You do not have to detach one style sheet
      before attaching another.

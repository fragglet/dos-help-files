WINDOW NEW WINDOW
 
Procedures: <Opening a new window>
 
Use the New Window command to open another window
containing the same document as the active window.
 
■ You can open new windows to display different
  parts or views of a document simultaneously.
 
■ If you change the contents in one window, all
  other windows containing the same document
  reflect those changes.
 
■ When you open a new window, it becomes the
  active window and is displayed on top of all
  other open windows.

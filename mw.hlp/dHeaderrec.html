dHeaderrec
Header record
 
A list of field names (or categories of
information) stored in the first paragraph of a
data document. The field names of a header
record match the field names you place in a
main document, which indicate variable text to
appear in a form document. A header record should
not be confused with a header, which is text that
prints at the top of a page. See also: Data
document; Form document; Main document.

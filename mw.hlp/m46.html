UTILITIES CUSTOMIZE
 
Procedures: <Customizing Word>
 
Use the Utilities Customize command to:
 
■ Turn on or off:
  * Background pagination.
  * The prompt for <summary information>.
  * <Widow>/<Orphan> control.
  * Word 5.0 key assignments.
  * The use of Ins for the <overtype> key.
  * Display of key combinations on menus.
  * The alert beep.
 
■ Select:
  * A different line draw character.
  * A different speller.
 
■ Set:
  * Frequency of <automatic saving>.
  * Defaults for unit of measure, date, time,
    and decimal point.
  * Default tab spacing.

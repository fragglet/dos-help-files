DUPLICATE STYLES
 
A style you are pasting or merging has the same
style I.D. or key code as the style displayed in
the message box.
 
■ To insert the style, rename it:
  1. Choose the Rename Style button.
  2. Accept the Style I.D. Word suggests in the
     Style I.D. box or select one from the list.
  3. If the key code is a duplicate, change it.
  4. Choose OK.
■ To skip the duplicate style and continue
  pasting or merging styles, choose the Skip
  Style button.

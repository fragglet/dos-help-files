NAMING AND SAVING A DOCUMENT
 
Overview: <File Save As command>
 
To name and save a document:
1. Choose File Save As (Alt,F,A).
2. In the File Name box, type a filename.
3. Choose OK.
 
More Help: <Lesson: Formatting & Saving Documents>

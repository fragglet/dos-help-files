PREVIEWING PAGES
 
Overview: <File Print Preview command>
 
To preview pages:
1. Move the cursor to the page you want to begin
   previewing.
2. Choose File Print Preview (Alt,F,V) or press
   Ctrl+F9.
   The document displays beginning where the
   cursor was placed.
 
Using Manual: Ch. 10, "Printing"

THE FILE DOES NOT EXIST.
DO YOU WANT TO CREATE A NEW STYLE SHEET?
 
Word cannot find the style sheet name you typed
in the current directory.
 
■ Check the filename and path.
■ To create the style sheet with the name you
  typed, choose OK.
■ To return to your work without creating the new
  style sheet, choose Cancel.

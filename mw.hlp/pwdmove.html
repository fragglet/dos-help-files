MOVING A WINDOW
 
Overview: <Window Move command>
 
To move a window:
 
Using the mouse:
1. Point to the title bar or left border of the
   window you want to move.
2. <Drag> the window to the position you want.
 
Using the keyboard:
1. Activate the window you want to move.
2. Choose Window Move (Alt,W,M) or press Ctrl+F7.
3. Use the Direction keys to move the window.
4. Press Enter to set the window position.

INSERT NEW STYLE
 
Procedures: <Inserting a new style>
 
Use the Insert New Style command to add a <style>
to a <style sheet>. This command is available
on the Insert menu only when the style sheet
window is <active>.
 
■ You can build a style sheet by adding available
  <style I.D.s> and using the Format menu commands
  to set formatting for the styles.
 
■ You can insert <automatic styles> in the
  NORMAL.STY style sheet and change them in order
  to change the <default> formatting for your
  documents. Automatic styles do not appear in
  a style sheet until you insert them.

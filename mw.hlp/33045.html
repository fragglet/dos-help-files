SECTION MARGINS DIALOG BOX        (Screen 1 of 2)
 
Page Size
  If the paper you will print on is not 8.5-by-11
  inches, type the dimensions in the Width and
  Height boxes.
 
Margins
  Type the measurements you want for Top, Bottom,
  Left, and Right margins.
  Gutter          The inner margin of facing
                  pages. Type a number for
                  standard gutter measurement.
  Mirror Margins  Turn on to make the left and
                  right margins of odd and even
                  pages equal.
SECTION MARGINS DIALOG BOX        (Screen 2 of 2)
 
Use As Default
  Turn on only if you want the settings in the
  Section Margins dialog box to serve as the
  default settings for all new documents.
  Note: Default formats for section margins are
        determined by an automatic style. If you
        change this automatic style, it will
        override the default settings made in the
        Section Margins dialog box.
 
Format Section
  Displays the Format Section dialog box to set
  formatting options for a section.

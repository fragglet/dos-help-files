PICTURE DIALOG BOX                (Screen 1 of 4)
 
Picture File Name
  Type the name of the file you want to insert.
 
Files
  Select the file you want to insert.
 
Directories
  If the file you want is not in the Files list,
  select the appropriate directory.
PICTURE DIALOG BOX                (Screen 2 of 4)
 
Format
  If the format of the graphic is recognizable,
  Word supplies its format name. If this box is
  blank, choose from the formats listed.
 
Align In Frame
  Select how you want the graphic to be aligned
  within the frame.
   Left       Aligned with the left border of its
              frame.
   Centered   Centered in its frame.
PICTURE DIALOG BOX                (Screen 3 of 4)
 
Align In Frame (continued)
   Right      Aligned with the right border of
              its frame.
 
Graphics Size
  Accept the preset width and height, or select
  dimensions in the boxes. You may also type
  other dimensions, but Word suggests dimensions
  that keep the aspect ratio of the original
  graphic intact.
PICTURE DIALOG BOX                (Screen 4 of 4)
 
Space
  Type measurements to add white space before and
  after the graphic.
 
Preview
  Choose Preview to go to print preview and see
  how the graphic will be positioned on the page
  when it prints.

FILE PRINT PREVIEW
 
Procedures: <Previewing pages>
 
Use the File Print Preview command to open
the Print Preview window, where you can see how a
document will look when it is printed.
 
A new set of menus display in print preview.
Press F1 while in the Print Preview window to get
Help.
 
■ To preview documents, you must have a video
  adapter with graphics capabilities installed.
 
■ You cannot edit while in print preview.
 
■ You can view a document by single page, double
  pages, or facing pages (pages that face each
  other in book format).
 
■ Print preview displays different font sizes,
  graphics, fixed-position objects, footnotes,
  and margin areas, including any headers or
  footers.
 
■ Press Ctrl+F9 to toggle between print preview
  and your document.

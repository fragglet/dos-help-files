FUNCTION KEYS FOR WORD 5.5
 
Press:              To:
F1                  Get Help
Alt+F1              Go to next field or
                     instruction
Ctrl+F1             File Open
Shift+Alt+F1        Go to previous field or
                     instruction
 
F2                  Calculate
Alt+F2              File Save As
Alt+Ctrl+F2         File Open
Alt+Shift+Ctrl+F2   File Print
Ctrl+F2             Format Character
Shift+Alt+F2        File Save
Shift+F2            Toggle View Outline
 
F3                  Expand glossary
Ctrl+F3             Toggle Macro Record
Shift+F3            Toggle uppercase/initial
                     caps/lowercase
 
F4                  Repeat last action
Alt+F4              File Exit Word
Ctrl+F4             Window Close
Shift+F4            Repeat last search
 
F5                  Edit Go To
Alt+F5              Toggle overtype on and off
Ctrl+F5             Window Restore
Ctrl+Shift+F5       Insert Bookmark
Shift+F5            Toggle outline edit and
                     organize, when View Outline
                     is chosen
 
F6                  Move to next pane in
                     active window
Alt+F6              Select current, then
                     following word
Ctrl+F6             Move to next window
Shift+F6            Move to previous pane in
                     active window
Shift+Ctrl+F6       Move to previous window
 
F7                  Utilities Spelling
Alt+F7              Toggle Show Line Breaks
Ctrl+F7             Window Move
Shift+F7            Utilities Thesaurus
 
F8                  Extend selection
Alt+F8              Select current, then
                     following sentences
Ctrl+F8             Window Size
Ctrl+Shift+F8       Toggle column selection
                     on/off
Shift+F8            Shrink selection
 
F9                  Insert File, Update
                     Link option
Alt+F9              Toggle last two selected
                     display modes
Ctrl+F9             File Print Preview
Shift+F9            File Print
 
F10                 Activate menu bar
Alt+F10             Select current, then
                     following paragraphs
Ctrl+F10            Window Maximize
Ctrl+Shift+F10      Turn ruler on and move cursor
                     to ruler
Shift+F10           Select document
 
F11                 Move to next field or
                     instruction
Shift+F11           Move to previous field or
                     instruction
 
F12                 File Save As
Ctrl+F12            File Open
Ctrl+Shift+F12      File Print
Shift+F12           File Save
 
Note: If you've chosen Word 5.0 function keys in
      the Customize dialog box, choose Help Word
      5.0 to 5.5.

USING THE RIBBON AND RULER
 
To:                Point to:
Turn the ruler     Ruler/ribbon icon (┴), and
 on/off             click the left mouse button
Turn the ribbon    Ruler/ribbon icon (┴), and
 on/off             click the right mouse button
Choose Format      "Font" or "Pts" on ribbon, and
 Character          double-click the left mouse
                    button
Choose Format      "Style" on ribbon, and
 Apply Style        double-click the left mouse
                    button
Create hanging     Indent mark on ruler, and drag
 indent             to new position, using the
                    right mouse button
Set tab stops      Position on ruler where you
                    want to place tab stop, and
                    click the left mouse button
Move tab stops     Tab marker on ruler, and drag
                    to new position, using the
                    left mouse button
Remove a tab       Tab marker on ruler, and drag
 stop               off ruler, using the left
                    mouse button
Change alignment   Alignment character at left
                    end of ruler, and click the
                    left mouse button until
                    alignment choice appears
Select leader      Leader character at left end
 character          of ruler (to the immediate
                    left of the alignment
                    character), and click the
                    left mouse button until
                    leader choice appears
Add leader         Tab marker on ruler you want
 characters         to add leader characters to,
                    and click the left mouse
                    button

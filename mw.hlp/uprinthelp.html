PRINTING A HELP TOPIC
 
You can print any Help topic that appears in the
Help window. Word prints the entire Help topic.
(Note: You cannot select and print parts of a
Help topic).
 
To print a Help topic:
1. Place the cursor anywhere in the Help window.
2. Choose File Print (Alt,F,P).
 
Note: You cannot print Help about dialog boxes
      and messages.

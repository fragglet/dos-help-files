CHANGING THE DISPLAY MODE
 
Overview: <View Preferences command>
 
To change the display mode of the Word screen:
1. Choose View Preferences Display Mode
   (Alt,V,E,D).
2. Press Alt+Down Arrow to show the available
   display modes.
3. Select the display mode you want from the
   Display Mode list.
4. Choose OK.
 
More Help: <Choosing preferences>
 
Using Manual: Ch. 3, "The Word Screen"

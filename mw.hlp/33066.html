TABS DIALOG BOX                   (Screen 1 of 3)
 
Tab Position
  Type or select from the list the tab you want
  to set options for.
 
Alignment
  Select the tab alignment you want:
   Left       Left end of text aligns with tab
   Center     Text is centered on the tab
   Right      Right end of text aligns with tab
   Decimal    Decimal point aligns with tab
   Vertical   Vertical line is drawn at tab
TABS DIALOG BOX                   (Screen 2 of 3)
 
Leader
  Select the leader character you want to
  display at each tab stop.
 
  Note: Leader characters are displayed only
        when tabs are not displayed. (Turn Tabs
        on or off with the View Preferences
        command.)
 
To Be Cleared
  Lists the tab(s) you specify to be cleared
  when you choose OK.
TABS DIALOG BOX                   (Screen 3 of 3)
 
Set
  Sets the specified tab without closing the
  dialog box.
 
Clear
  Places the selected tab in the To Be Cleared
  list. Tabs in the list will be cleared when
  you close the dialog box by choosing OK.
 
Clear All
  Deletes all set tabs without closing the dialog
  box.

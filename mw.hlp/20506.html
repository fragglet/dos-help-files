NO PRINTER FILE SELECTED WITH THE FILE PRINTER
SETUP COMMAND. DO YOU STILL WANT TO PRINT?
 
If you print without having selected a printer
file, Word will print your document without
character or paragraph formatting, as ASCII text.
 
■ To print the document without a printer file
  selected, choose OK.
■ To close the message box and return to the
  document, choose Cancel.
■ To select a printer file, choose File Printer
  Setup again, type or select a printer filename,
  and make sure the name of your printer appears
  in the Printer Name box. Then choose File Print
  again.

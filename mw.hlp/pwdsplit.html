SPLITTING A WINDOW
 
Overview: <Window Split command>
 
To split a window:
 
Using the mouse:
1. Point to the split icon (═) in the upper-right
   corner of the window border.
2. <Drag> the split icon to where you want the
   split to occur.
 
Using the keyboard:
1. Activate the window you want to split.
2. Choose Window Split (Alt,W,T).
3. Move the split window lines using the Up Arrow
   and Down Arrow keys.
4. Press Enter to set the window split.
 
To remove the window split:
■ Double-click on the split icon with the left
  mouse button, or use the Direction keys to move
  the split window lines to the top or bottom of
  the window.

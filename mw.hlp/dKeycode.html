dKeycode
Key code
 
A sequence of keys you can press to apply a
style, run a macro, or insert a glossary entry.
For example, you create a style for quotations
and assign it the key code QP. To apply the style
quickly, you select the text and press
Shift+Ctrl+Q,P. To run a macro or insert a
glossary entry, position the cursor in the
document and press Ctrl+the key or keys you have
assigned as key codes. See also: Glossary; Macro;
Styles; Style bar.

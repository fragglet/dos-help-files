REPLACE SELECTION DIALOG BOX      (Screen 1 of 2)
 
Replace selection?
 
Choose Yes to replace the highlighted search
text with the replacement text.
 
Choose No to leave the search text as is.
 
Word searches to the end of the document and
asks if you want to continue the search from the
beginning if you did not begin the search there.
 
Choose Cancel to quit the replacement command.
REPLACE SELECTION DIALOG BOX      (Screen 2 of 2)
 
Confirm Changes
  If you do not want to be prompted to confirm
  each replacement, turn off Confirm Changes in
  the Replace dialog box. Word will replace all
  occurrences in the document or selection if
  Confirm Changes is turned off.

INSERTING PAGE NUMBERS
 
Overview: <Insert Page Numbers command>
 
To insert page numbers:
1. Position the cursor in the section where you
   want page numbers to begin printing.
2. Choose Insert Page Numbers (Alt,I,U).
3. Turn on a page number position: From Top or
   From Bottom.
4. Select a page number format.
5. Indicate what number you want the page
   numbering to start with in the Start At box.
6. Turn on a page number alignment.
7. Choose OK.
   Page numbers are displayed on printed copies
   and in Print Preview.
 
More Help: <Adding page numbers to headers/footers>

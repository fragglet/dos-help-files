ORGANIZING OUTLINE HEADINGS
 
Overview: <View Outline command>
 
You must be in <outline organize> mode to
rearrange headings. When you move a heading in
outline organize mode, you also move all
subheadings and body text.
 
To switch to outline organize mode:
1. Choose View Outline (Alt,V,O) if you are not
   already in outline view.
2. Press Shift+F5 to toggle between outline edit
   and outline organize modes. Word displays the
   word ORGANIZE in the status bar when you are in
   outline organize mode.
 
When you move or copy a heading in outline
organize mode, Word automatically moves or copies
any subheadings and body text below that heading.
 
To move and copy headings:
1. In outline organize mode, select the heading(s)
   you  want to move or copy.
   ■ To move a heading, choose Edit Cut (Alt,E,T)
     to cut to the scrap.
   ■ To copy a heading, choose Edit Copy (Alt,E,C)
     to copy to the scrap.
2. Move the cursor to the heading that will follow
   the moved or copied heading(s)
3. Choose Edit Paste (Alt,E,P).
 
To delete a heading:
1. In outline organize mode, select the heading
   you want to delete.
2. Press Del.
   Word deletes the heading and any subheadings
   or body text below the selected heading.
 
To rearrange headings by sorting:
1. If necessary, remove any numbers from in front
   of the headings you want to rearrange. See
   <Numbering an outline>.
2. At the beginning of each heading you want to
   sort, type the number representing its new
   position in the sequence of headings.
3. Choose View Outline (Alt,V,O).
4. Press Shift+F5 to switch to outline organize
   mode.
5. Select the portion of the outline you want to
   sort. If you don't select anything, Word sorts
   only the highest heading level.
6. Choose Utilities Sort (Alt,U,O).
7. In the Key Type box, select the Numeric option.
8. Choose OK.
 
Note: Word sorts only the highest level headings
      in a selection.

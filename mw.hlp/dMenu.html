dMenu
Menu
 
A list of commands that drops down from the menu
bar; for example, the File menu. You can select a
menu item by moving through the list with the
direction keys or the mouse (a highlight moves
along the menu indicating the current selection).
Choose a command from a menu by selecting the
command and pressing Enter, or by typing the
highlighted letter in the command name. Using the
mouse, press the left mouse button and select the
command, then release the botton. See also:
Direction keys; Click; Highlight; Select.

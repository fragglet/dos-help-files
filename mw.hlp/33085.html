FILE PRINT                        (Screen 1 of 1)
 
Use the File Print command to print and control
printing of documents.
 
To print a document:
1. Make sure the document you want to print is in
   the preview window.
2. Choose File Print (Alt,F,P) or Shift+F9.
3. Select any options you want to specify.
4. Choose OK.
 
For more information about the File Print
command, go to the document window, and choose
Help Index Commands.

THE SPREADSHEET FILE FORMAT IS NOT RECOGNIZED
BY WORD.
 
Word can import spreadsheets created with recent
versions of Microsoft Excel, Multiplan,
Lotus 1-2-3, Microsoft Works, and other
compatible programs. The filename you typed or
selected was not created with one of these
programs.
 
■ Verify that the spreadsheet you specified was
  created using one of the programs listed above.
■ Correct the filename you typed.
■ Select a spreadsheet from the list of files
  that was created in one of the valid formats.

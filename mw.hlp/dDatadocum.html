dDatadocum
Data document
 
A document that contains pieces of text to be
merged into a main document to create form
documents and other merge documents. The data
document includes a header record and data
records. The header record is a paragraph that
lists the categories of text stored in the data
document. The data records store the actual
text. See also: Data record; Form document;
Header record; Main document.

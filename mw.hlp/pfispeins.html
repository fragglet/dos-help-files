SPECIAL INSTRUCTIONS IN A MAIN DOCUMENT
 
Word has special instructions you can include
in a main document to give you more control over
varying text in form letters.
 
To:            Type:
Enter new      «SET fieldname="text to print"»
text for
all records
 
Ignore a       «IF number<50»«SKIP»«ENDIF»
record
 
Insert         «INCLUDE filename»
another
document
 
Insert a       «IF condition»result«ENDIF»
specified
result if one
condition
exists
 
Merge data     «DATA data document name»
document       «DATA header file,document name»
 
Print all      «NEXT»
records in
one document
 
Prompt for     «ASK field_name=?»
field entries  «ASK field_name=? specify message»
before
printing
 
Prompt for     «SET field_name=?»
text before
printing
 
Note: Maximum length of text string in IF
      statement is 256 characters.
 
      Maximum length of prompt in SET or ASK is
      80 characters.

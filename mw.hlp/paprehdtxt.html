WORKING WITH HIDDEN TEXT
 
Overview: <Format Character command>
 
To apply or remove hidden text format:
 
Using speed keys:
1. Select the text you want to format or remove
   formatting from.
2. Press Ctrl+H.
 
Using Format Character command:
1. Select the text you want to format or remove
   formatting from.
2. Choose Format Character (Alt,T,C).
3. In the Character dialog box, turn the Hidden
   check box on or off to apply or remove
   formatting.
4. Choose OK.
 
To show or hide hidden text:
1. Choose View Preferences (Alt,V,E).
2. Turn the Hidden Text check box on or off to
   show or hide text in your document.
 
To print hidden text:
1. Choose File Print (Alt,F,P).
2. Choose the Options button.
3. Turn on the Hidden Text check box.
   If you turn on the Hidden Text check box,
   hidden text prints whether or not it is
   showing on the screen.

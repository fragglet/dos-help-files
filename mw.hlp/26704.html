WORD CANNOT DELETE THIS FILE.
 
Word cannot delete a file if:
 
■ Word cannot find the filename on your disk.
  Check the disk directory and retype the
  filename, making sure to specify the correct
  path.
■ The disk is write-protected with a tab or the
  file is read-only. Remove the tab from the disk
  or use operating system commands to remove the
  file's read-only status.

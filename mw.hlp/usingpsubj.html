USING THE PROCEDURES INDEX
 
To go to a procedure or a list of procedures:
Using the keyboard
1. Select any term on the list by pressing the
   Tab key or the first letter of the term.
2. Press Enter.
Using the mouse
■  Point to the term and click the left mouse
   button.
 
To return to the Procedures Index from a topic:
■  Choose Back or press Backspace.

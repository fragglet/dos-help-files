INSERT PICTURE
 
Procedures: <Inserting a graphic>
 
Use the Insert Picture command to insert
a graphic file (pictures, charts, stylized text,
screen images) created with another program into
a Word document.
 
■ The graphic file is displayed or printed when
  you choose the File Print or File Print Preview
  command.
 
■ Once a graphic is inserted, you determine:
   * width and height of the graphic
   * amount of space above and below the graphic
   * alignment of the graphic within its frame
 
■ Use the Format Position command to specify the
  location of a graphic or the size of its frame.
 
■ Use the Format Border command to add a border
  around the graphic.

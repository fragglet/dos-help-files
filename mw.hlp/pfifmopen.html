OPENING A DOCUMENT FROM ANOTHER DIRECTORY
 
Overview: <File File Management command>
 
To open a document from another directory:
1. Choose File File Management (Alt,F,F).
2. Choose the Search button.
3. Using the Directories box, select the
   directory you want to add to the search path.
4. Type a comma to add the directory. It will
   display in the Search Paths box.
5. Choose OK to close the Search dialog box.
6. Select the file you want to open in the Files
   box.
7. Choose the Open button.
 
More Help: <Managing files>
 
Using Manual: Ch. 14, "File Management"

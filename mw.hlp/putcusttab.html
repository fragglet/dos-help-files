CHANGING THE DEFAULT TAB SETTING
 
Overview: <Utilities Customize command>
 
Word's default tab setting is 0.5".
 
To change the default tab setting:
1. Choose Utilities Customize (Alt,U,U).
2. Type the default tab spacing you want in the
   Default Tab box.
3. Choose OK.
 
More Help: <Customizing Word>
 
Using Manual: Ch. 9, "Tabs and Tables"

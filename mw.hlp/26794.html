26794
THE STYLE SHEET IS IN USE BY ANOTHER USER OR
PROGRAM.
 
The style sheet you attempted to open or attach
is already in use by another network user.
 
■ Choose File Open again when the style sheet is
  no longer in use.
■ If more than one user needs access to a style
  sheet simultaneously, make the style sheet
  read-only to prevent unauthorized changes to
  the style sheet. For more information, see
  Ch. 4 in the Getting Started booklet.

SEARCHING FOR CHARACTER FORMATTING
 
Overview: <Edit Search command>
 
To search for a specific character formatting:
1. Choose Edit Search (Alt,E,S).
2. Select either Up or Down for the direction
   you want to search the document.
3. Choose Search For Formatting Only (Alt+F).
4. Choose Character (Alt+C).
5. Choose the combination of font, point size,
   color, format, and position you want to search
   for.
6. Choose OK.
 
To repeat the search, press Shift+F4.

DO YOU WANT TO REPLACE THE EXISTING INDEX?
 
You asked Word to create a new index.
 
■ To replace the existing index with a newly
  compiled index, choose Yes. Word overwrites any
  special formatting you have applied when it
  replaces the existing index. You can
  permanently format the index using the
  automatic index style.
■ To leave the existing index as it is, choose
  No.

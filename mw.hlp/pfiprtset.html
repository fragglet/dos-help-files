SELECTING A PRINTER
 
Overview: <File Printer Setup command>
 
To select a printer:
1. Choose File Printer Setup (Alt,F,R).
2. In the Printer File box, type or select a
   printer driver (.PRD) filename.
   If the printer driver is not in the current
   directory, you can use the Directories box
   to move to the directory where the printer
   driver is stored.
3. In the Printer Name box, select the name of
   your printer. Check the Printer Manual for
   correct printer driver (.PRD) filenames and
   model names.
4. If you have more than one printer attached to
   your computer, make sure the correct port
   is displayed in the Connect To box.
5. Choose OK.
 
Using Manual: Ch. 10, "Printing"

SPELLING OPTIONS DIALOG BOX       (Screen 1 of 2)
 
Options
 Always Suggest: If turned on, Word suggests
   alternate spellings each time a misspelled word
   is found. Otherwise, suggestions are only made
   if Suggest is chosen in the Spelling dialog
   box. Spell checking is faster if this option is
   turned off.
 
 Check Punctuation: Checks for text with misplaced
   punctuation characters. Spell checking is
   faster with this option turned off.
 
 Ignore All Caps: Ignores words with all capital
   letters when spell checking.
SPELLING OPTIONS DIALOG BOX       (Screen 2 of 2)
 
Lookup
  With Quick turned on, the first two letters of a
  misspelled word are assumed to be correct and
  suggestions are made accordingly. Complete
  provides a more thorough search for
  alternatives.
 
User Dictionary
  Select the dictionary you want to use or type
  the name of a dictionary you want to create.
 
Directories
  If the dictionary you want is not in the current
  directory, select the appropriate directory.

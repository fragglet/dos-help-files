DELETING A MACRO
 
Overview: <Macro Edit command>
 
To delete a macro:
1. Choose Macro Edit (Alt,M,E).
2. In the Macro Name box, type the macro name or
   select it from the Macros list.
3. Choose the Delete button.
4. Choose the Close button or press Esc.
 
More Help: <Working with macros>
 
Using Manual: Ch. 22, "Macros"

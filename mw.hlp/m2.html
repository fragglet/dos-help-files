FILE OPEN
 
Procedures: <Opening a document>
 
The File Open command opens an existing document
or <style sheet> in a new <window>. If you
have a document on your screen that you no longer
want to use, close the window, then use the File
Open command. Otherwise, Word will keep both
files open.
 
■ The Open dialog box lists documents in the
  current directory with the .DOC <extension>.
 
■ Use the File File Management command to help
  you find files.  You can search for documents
  in multiple directories or sort documents based
  on summary sheet criteria.
 
■ Create a new document with the File New
  command.

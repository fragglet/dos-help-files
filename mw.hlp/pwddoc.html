ACTIVATING A WINDOW
 
Overview: <Window open windows>
 
To activate a window:
 
Using the mouse:
■ Click in the window you want to activate.
 
If the window you want to activate is covered
by other windows and not visible:
1. Display the Window menu.
2. Select the window you want to activate from
   the open windows list.
 
Using the keyboard:
1. Display the Window menu (Alt,W).
2. Press the number in the open windows list that
   corresponds to the window you want to
   activate.

           <Window Close command>

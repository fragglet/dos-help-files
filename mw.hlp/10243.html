10243
THE FILENAME IS NOT VALID.
 
Word displays this message if:
 
■ The filename is too long. Type a shorter name,
  maximum 8 characters, followed by a 3-letter
  extension.
■ You typed a character that is not valid in
  filenames. Correct the character or select a
  filename from the list.
 
To open files:
■ To open documents and style sheets, choose File
  Open.
■ To open a glossary file, choose Edit Glossary
  Open Glossary.

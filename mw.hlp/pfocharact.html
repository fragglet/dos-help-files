FORMATTING CHARACTERS USING CHARACTER COMMAND
 
Overview: <Format Character command>
 
To format characters using the Format Character
command:
1. Select the text to format.
2. Choose Format Character (Alt,T,C).
3. In the Character dialog box, select any
   character formats you want to apply to the
   selected text.
4. Choose OK.
 
To repeat formatting:
1. Select the next text you want to format the
   same way.
2. Press F4.

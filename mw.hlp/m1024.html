EDIT CUT
 
Procedures: <Using the Edit Cut command>
 
Use the Edit Cut command to remove the selected
text or graphic from the document and store it in
the <scrap>.
 
■ Using the Edit Cut command is a quick way to
  move document contents from one place to
  another.
 
■ Cutting document contents to the scrap replaces
  the contents previously stored there.
 
■ If nothing is selected the Edit Cut command
  deletes the character above the cursor and
  places it in the scrap.
 
■ Use the <glossary> for long-term storage of
  frequently used text or graphics.
 
■ Press Shift+Del to cut selected text.
 
■ If you have chosen to use Word 5.0 function
  keys or to use Ins as the overtype key, see the
  Word 5.0 To 5.5 Help topics for the correct key
  functionality.

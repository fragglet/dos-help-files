DELETING FILES
 
Overview: <File File Management command>
 
To delete a file:
1. Choose File File Management (Alt,F,F).
2. If the document you want to delete is not
   listed in the Files box, use the Search button
   to search for and list the document.
3. Select or mark the document(s) you want to
   delete.
4. Choose the Delete button.
5. Choose OK to confirm the deletion.
6. Choose Close to return to your document.
 
More Help: <Managing files>
 
Using Manual: Ch. 14, "File Management"

33002
FILE OPTIONS DIALOG BOX           (Screen 1 of 1)
 
Default Path
  Enter the default path you want your system to
  use to locate and open files during the current
  Word session.
 
Directories
  Select the directory you want to include in
  the default path.
 
Always Use As Default
  Turn on this option for Word to continue
  following the specified path as the default for
  saving and opening files. It will be retained
  from one Word session to the next.

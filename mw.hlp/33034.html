SECTION DIALOG BOX                (Screen 1 of 4)
 
Columns
  Type the number of columns for the section.
  Type the spacing between columns.
 
Place Footnotes
  To print footnotes or annotations on the same
  page they refer to, select Same Page.
  To print footnotes or annotations at the end of
  a section, select End Of Section.
 
Line Numbers
  To print line numbers in the left column of
  your document, turn on Add Line Numbers.
SECTION DIALOG BOX                (Screen 2 of 4)
 
Line Numbers (continued)
  In the From Text box, type the spacing you want
  between the numbers and text. The number must
  be smaller than the left margin measurement.
  To have Word count in increments greater than
  1, type a new increment in the Count By box.
 
Restart At
  Select the option you prefer for how Word
  numbers the lines:
   New Page     Begins renumbering at new page
   New Section  Begins renumbering at new
                section
   Continuous   Numbers entire document
SECTION DIALOG BOX                (Screen 3 of 4)
 
Section Start
  Select the page-break printing option you want
  for the section:
   New Page    Breaks the page at the section
               mark
   Continuous  Begins the new section without a
               page break
   Column      Breaks a column at the section
               mark
   Even Page   Begins the section on the next
               even-numbered page
SECTION DIALOG BOX                (Screen 4 of 4)
 
Section Start (continued)
   Odd Page    Begins the new section on the next
               odd-numbered page
 
Margins
  Displays the Section Margins dialog box to set
  margins for a section.
 
Page Numbers
  Displays the Page Numbers dialog box to set
  position and format of page numbers in section.

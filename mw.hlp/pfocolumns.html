SETTING UP MULTIPLE COLUMNS OF TEXT
 
Overview: <Format Section command>
 
To set up multiple columns of text:
1. Choose Format Section (Alt,T,S).
2. In the Columns box, type the number of columns
   you want and the amount of space you want
   between columns.
3. Choose OK.
 
Note: To display the columns on the screen as
      they will appear when printed, choose View
      Layout (Alt,V,L).
 
More Help: <Formatting a section>
 
Using Manual: Ch. 7, "Page Formatting"

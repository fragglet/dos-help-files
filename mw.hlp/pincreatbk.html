INSERTING A BOOKMARK
 
Overview: <Insert Bookmark command>
 
To insert a bookmark:
1. Select the text you want to use as a bookmark.
2. Choose Insert Bookmark (Alt,I,M).
3. Type a bookmark name or select an existing
   name from the list of names.
   If you select an existing bookmark name,
   Word will ask if you want to overwrite it
   with the new text selection.
4. Choose OK.
 
More Help: <Working with bookmarks>
 
Using Manual: Ch. 13, "Bookmarks and
              Cross-Referencing"

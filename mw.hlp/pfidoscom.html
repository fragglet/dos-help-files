RUNNING DOS COMMANDS FROM WORD
 
Overview: <File DOS Commands command>
 
To run a single DOS command from Word:
1. Choose File DOS Commands (Alt,F,D).
2. Type the operating system command.
3. Choose OK.
4. After the command has run, press any key to
   return to Word.
 
To exit Word temporarily and run commands from
the DOS prompt:
1. Choose File DOS Commands (Alt,F,D).
2. In the Command box, type COMMAND, if it is not
   already displayed.
3. Choose OK.
4. Run as many commands as you wish from the DOS
   prompt.
5. When ready to return to Word, type Exit at the
   DOS prompt and then press any key.

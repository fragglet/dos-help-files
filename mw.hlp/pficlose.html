CLOSING A DOCUMENT
 
Overview: <File Close command>
 
To close a document:
1. Choose File Close (Alt,F,C).
2. If you have made changes to the document
   since you last saved it, Word will prompt
   you to save the document.
   Choose Yes to save the document with any
   changes you have made, or No to discard
   unsaved changes.

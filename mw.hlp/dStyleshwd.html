dStyleshwd
Style sheet window
 
The window that appears on the screen when you
choose Format Define Styles or File Open and open
a style sheet. You can view, create, and revise
styles in the style sheet window. This window is
similar to the document window in that you can
size, move, maximize, and split it. Many commands
appear on menus in both the document and style
sheet windows and perform the same functions on
the text. The Edit Rename Style, Insert New
Style, and Insert Style Sheet commands appear
only in the style sheet window.

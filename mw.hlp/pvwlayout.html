USING LAYOUT VIEW
 
Overview: <View Layout command>
 
To turn on layout view:
■ Choose View Layout (Alt,V,L).
 
To jump from one object or column to another:
 
Press:                    To:
Alt+5+Right Arrow         Jump to the next object
                          or column
 
Alt+5+Left Arrow          Jump to previous object
                          or column
 
More Help: <View menu>
 
Using Manual: Ch. 20, "Layout Features for
                       Graphics and Text"

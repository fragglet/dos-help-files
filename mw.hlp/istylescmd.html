STYLE SHEET COMMANDS
 
Use the following commands to work with style
sheets.
 
In the document window:
<Format Apply Style>
<Format Attach Style Sheet>
<Format Define Styles>
<Format Record Style>
 
In the style sheet window:
<Edit Rename Style>
<Insert New Style>
<Insert Style Sheet>

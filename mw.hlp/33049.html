REPLACE CHARACTER FORMATTING DIALOG BOX
                                  (Screen 1 of 3)
 
Select the character formatting you want to
replace. You can choose combinations of the
formatting options available in this dialog box.
 
Font
  Type or select from the list box the font you
  want to replace.
 
Point Size
  Type or select from the list box the point size
  you want to replace.
REPLACE CHARACTER FORMATTING DIALOG BOX
                                  (Screen 2 of 3)
Color:
  Type or select from the list box the color you
  want to replace.
 
Format Options List
  Turn on the character formatting that you want
  to replace.
 
Position
  Turn on the options that correspond to the
  character formatting you want to replace.
REPLACE CHARACTER FORMATTING DIALOG BOX
                                 (Screen 3 of 3)
 
For more information on character formatting and
the contents of this dialog box, see the Format
Character Help topics.

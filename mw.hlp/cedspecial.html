SEARCH AND REPLACE SPECIAL CHARACTERS
 
Overview: <Edit Replace command>
 
To search for or replace special characters, such
as paragraph marks or tab characters, type the
following character combinations in the "Text To
Search For:" or "Replace With:" box in the Search
or Replace dialog box.
 
To search or replace:        Type:
 
Caret character                ^^
Newline character              ^n
Nonbreaking space              ^s
Optional hyphen                ^-
Paragraph mark                 ^p
Question mark                  ^?
Section mark or hard           ^d
 page break
Tab character                  ^t
White space                    ^w
Unspecified character          ?
 For example, type gr?y
 to search for both
 "gray"and "grey"
 
You cannot use a question mark (?) for an
unspecified character in replacement text.

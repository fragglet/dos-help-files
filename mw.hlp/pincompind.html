COMPILING AN INDEX
 
Overview: <Insert Index command>
 
To compile an index:
1. Insert index codes in the Word document.
2. Choose Insert Index (Alt,I,I).
3. Choose OK.
   The index is compiled and inserted at the end
   of the document. A section mark separates the
   index from the document.
 
More Help: <Indexing>
 
Using Manual: Ch. 19, "Indexes"

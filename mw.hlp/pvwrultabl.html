CREATING A TABLE USING THE RULER
 
Overview: <View Ruler command>
 
To create a table using the ruler:
1. Display the ruler by clicking the ruler/ribbon
   icon or by choosing View Ruler (Alt,V,R).
2. Using the direction keys or the mouse, move the
   cursor to where you want the table to begin.
3. Set tab stops for each column in the table. For
   more information on this step, see:
   <Setting and aligning tabs>.
4. Type the text for the first column in the
   first row.
5. Press Tab to move to the next column and type
   the text for that column.
   Repeat this step for each column in the row.
6. Press Shift+Enter to move to the next row.
   Shift+Enter starts a new line without
   starting a new paragraph.
7. Press Enter to end the table.

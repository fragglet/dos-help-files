dPath
Path
 
The location of a file in a directory structure
on your disk. For example, if you created the
subdirectory LETTERS in your WORD directory,
and stored a file there named UNCLEAL.DOC, the
path to that file would be C:\WORD\LETTERS. To
load the file UNCLEAL.DOC, you would need to
indicate the correct path to the file.

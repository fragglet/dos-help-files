FILE SAVE AS
 
Procedures: <Naming and saving a document>
 
Use the File Save As command to save and name
your document with a specified <file format>.
 
■ Word saves documents in Word format, as
  unformatted text with or without line breaks,
  or in Rich Text Format (RTF).
 
■ Documents saved in Text Only (ASCII) format
  can be used with programs that cannot read
  Word's formatting.
 
■ Choose the File Save command to save a document
  with its existing name and format.
 
■ Press Alt+F2 to name and save a document.

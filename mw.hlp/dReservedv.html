dReservedv
Reserved variable
 
A specially named field Word has set aside for
use in macro instructions. For example, the
reserved variable SAVE tests the status bar to
check if Word has advised you to save your work;
if Word is advising you to save, the macro can
then choose File Save to protect your work. Macro
instructions can test for the values of reserved
variables but cannot change them except in
limited cases. See also: Status bar; Variable.

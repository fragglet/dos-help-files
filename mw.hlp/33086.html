FILE PRINTER SETUP                (Screen 1 of 2)
 
Use the File Printer Setup command to select a
printer.
 
To select a printer:
1. Choose File Printer Setup (Alt,F,R).
2. Type a printer drive (.PRD) filename in the
   Printer File box, or select one in the Files
   box.
   If the printer driver is not in the current
   directory, you can use the Directories box to
   move to the directory where the printer driver
   is stored.
3. In the Printer Name box, select the name of
   your printer.
FILE PRINTER SETUP                (Screen 2 of 2)
 
4. If you have more than one printer attached to
   your computer, make sure the correct port
   is displayed in the Connect To box.
5. Select the type of Paper Feed you will use.
6. Select 75, 150, or 300 dpi Graphics
   Resolution.
7. If you want to use queued printing, select
   Use Print Queue.
8. Choose OK.
 
For more information about the File Printer Setup
command, go to the document window, and choose
Help Index Commands.

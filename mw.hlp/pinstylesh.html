MERGING STYLE SHEETS
 
Overview: <Insert Style Sheet command>
 
To merge style sheets:
1. Choose Format Define Styles (Alt,T,D) to
   open the style sheet window.
2. If you have not opened one of the style sheets
   you want to merge, choose File Open (Alt,F,O)
   and open one of the style sheets.
3. Choose Insert Style Sheet (Alt,I,S).
4. In the Style Sheet Name box, type the name of
   the style sheet you want to merge with the
   current style sheet, or choose a style sheet
   from the Files list.
   If the style sheet you want to merge is not in
   the Files list, select the appropriate
   directory from the Directories list.
5. Choose OK.
 
Note: If you get a message indicating duplicate
      styles in the style sheets, press F1 for
      Help on the options.
 
More Help: <Working with styles and style sheets>
 
Using Manual: Ch. 28, "Style Sheets"

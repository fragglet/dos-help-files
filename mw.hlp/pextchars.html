INSERTING A SPECIAL ASCII CHARACTER
 
To insert a special <ASCII> character:
1. Turn on NumLock.
2. Hold down the Alt key and type the numeric
   code for the character you want. (You must type
   the number code on the numeric keypad.)
3. Release Alt.
 
Note: For more information, see Appendix A, "The
      Extended Character Set," in the Using
      Manual.

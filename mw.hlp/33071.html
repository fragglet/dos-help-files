ATTACH STYLE SHEET DIALOG BOX     (Screen 1 of 2)
 
Style Sheet Name
  Type the name of the style sheet you want to
  attach, or select the style sheet from the
  Files list.
 
Files
  Displays the current directory and lists the
  available style sheets in that directory.
  Select the style sheet you want to attach.
 
Directories
  If the style sheet you want is not in the list,
  select the appropriate directory.
ATTACH STYLE SHEET DIALOG BOX     (Screen 2 of 2)
 
Style Sheet (Attach or Detach)
  Indicate whether you want to attach or detach a
  style sheet. If you select Detach and choose
  OK, Word asks if you want to convert all
  formatting to direct formatting. If you choose
  No, you will lose all formatting.

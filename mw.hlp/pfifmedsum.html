EDITING SUMMARY SHEETS OF EXISTING DOCUMENTS
 
Overview: <File File Management command>
 
To edit <summary sheets> of existing documents:
1. Choose File File Management (Alt,F,F).
2. Choose the Summary button.
3. Revise the summary information as needed.
4. Choose OK.
5. Choose Close to return to your document.
 
More Help: <Managing files>
 
Using Manual: Ch. 14, "File Management"

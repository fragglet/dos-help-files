RIBBON                            (Screen 1 of 3)
 
Style
  Displays the current style I.D. You can select
  from the list of available styles by clicking
  the Down Arrow next to the Style box or by
  pressing Ctrl+S, then Alt+Down Arrow.
  If the selected text has more than one style
  applied, no style I.D. is shown on the ribbon.
  Double-clicking on "Style" displays the Apply
  Style dialog box.
RIBBON                            (Screen 2 of 3)
 
Font
  Displays the current font. You can select from
  the list of available fonts by clicking the down
  arrow next to the Font box or by pressing
  Ctrl+F, then Alt+Down Arrow. Double-clicking on
  "Font" displays the Character dialog box.
 
Pts
  Displays the current point size of the font. You
  can select from the list of available point
  sizes by clicking the arrow next to the Pts box
  or by pressing Ctrl+P, then Alt+Down Arrow.
  Double-clicking on "Pts" displays the Character
  dialog box.
RIBBON                            (Screen 3 of 3)
 
Bold/Ital/UL
  Displays the current text format: bold, italic,
  or underline. You can click a format to turn it
  on or off, or press Ctrl+B, Ctrl+I, or Ctrl+U
  for bold, italic, or underline formatting.
  Formats that are turned on are highlighted on
  the ribbon

ADDING A BORDER OR BOX TO A PARAGRAPH
 
Overview: <Format Borders command>
 
To add a paragraph border:
1. Move the cursor into the paragraph to which
   you want to add a border.
2. Choose Format Borders (Alt,T,B).
3. Under Border Type, select Box Each Paragraph
   or select Lines.
4. If you selected Lines, specify Left, Right,
   Top, or Bottom.
5. Select the Line Style you want. Any options
   not supported by your printer will be dimmed.
6. If you are printing on a color printer, select
   a color in the Color box.
7. Choose OK.
 
More Help: <Adding paragraph borders and shading>

CREATING A BULLETED LIST USING A MACRO
 
Overview: <Macro Run command>
 
To create a bulleted list using a macro:
1.  Position the cursor where you want to begin
    the bulleted list.
2.  Choose Macro Edit (Alt,M,E).
3.  Choose the Open Glossary button.
4.  Select the MACRO.GLY file from the Files list.
    (If the file is not listed, indicate the path
    to the file in the Directories box, or insert
    the Utilities disk that contains the file.)
5.  Choose OK.
6.  Choose Close.
7.  Choose Macro Run (Alt,M,R).
8.  Select the BULLETED_LIST macro from the list.
9.  Choose OK.
10. Follow the screen prompts.
 
Note: If the MACRO.GLY glossary is open, you can
      also run the BULLETED_LIST macro by
      pressing the <key code> for the macro:
      Ctrl+W,8
 
More Help: <Opening a glossary file>

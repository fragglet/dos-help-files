THIS DICTIONARY DOES NOT EXIST.
DO YOU WANT TO CREATE A NEW DICTIONARY?
 
The user dictionary that you instructed Word to
find either does not exist or is not in the
current user dictionary directory.
 
■ To create a dictionary with the name you typed,
  choose OK.
■ To search for the dictionary in another
  directory using the Directories box, choose
  Cancel.

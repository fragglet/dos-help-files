PAGE NUMBERS DIALOG BOX           (Screen 1 of 3)
 
Page Number Position
  Turn on the position you want for page numbers
  in your document.
   From Top      Sets the absolute position of
                 the page number from the top of
                 the page.
   From Bottom   Sets the absolute position of
                 the page number from the
                 bottom of the page.
   None          Turns page numbers off in this
                 section of the document.
PAGE NUMBERS DIALOG BOX           (Screen 2 of 3)
 
Format
  Select a page number format from the list box.
  Each choice illustrates its corresponding page
  number format.
 
Start At
  Type the page number you want on the first page
  of the section you are printing.
 
Align Page Number At
  Turn on the page number location you want.
   Left Margin      Locates the page number at
                    the left margin of the
                    document.
PAGE NUMBERS DIALOG BOX           (Screen 3 of 3)
 
Align Page Number At (continued)
   Center           Centers the page number on
                    the page.
   Right Margin     The page number prints at the
                    right margin of the document.
   From Left Edge   The page number prints from
                    the left edge of the paper.
                    Type the measurement you want.

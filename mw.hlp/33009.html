COPY DIALOG BOX                   (Screen 1 of 2)
 
Path Name
  If the current path is not the one you want to
  copy to, type a different pathname.
 
Directories
  If you want to copy to a different directory,
  select the appropriate directory.
 
Copy Style Sheets
  Turn on this option to copy styles attached to
  documents.
COPY DIALOG BOX                   (Screen 2 of 2)
 
Delete Files After Copy
  Turn on this option to delete files from the
  correct location after you copy them to a
  different location.

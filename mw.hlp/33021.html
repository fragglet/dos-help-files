REPLACE DIALOG BOX                (Screen 1 of 3)
 
Text To Search For
  Type the text you want to find. Text will be
  ignored if you choose Replace Formatting Only.
 
Replace With
  Type the replacement text.
 
Whole Word
  Finds occurrences that are words by themselves
  and not part of a larger word. Turn off to find
  all occurrences.
REPLACE DIALOG BOX                (Screen 2 of 3)
 
Confirm Changes
  If you do not want to be prompted to confirm
  each replacement, turn off Confirm Changes. Word
  will replace all occurrences in the document or
  selection if Confirm Changes is turned off.
 
Match Upper/Lowercase
  Finds only those occurrences that have the
  specified combination of uppercase and
  lowercase letters.
REPLACE DIALOG BOX                (Screen 3 of 3)
 
Replace Formatting Only
  Displays the Replace Formatting dialog box.
  Text typed in the Text To Search For or Replace
  With box will be ignored if you choose
  Replace Formatting Only.

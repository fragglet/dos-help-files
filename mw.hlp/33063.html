SORT DIALOG BOX                   (Screen 1 of 2)
 
Sort Order
  Ascending   Sorts items in sequence from A to Z,
              or numbers from lowest to highest.
  Descending  Sorts items in sequence from Z to A,
              or numbers from highest to lowest.
 
Key Type
  Alphanumeric  Sorts letters first, then numbers.
  Numeric       Sorts only numbers.
SORT DIALOG BOX                   (Screen 2 of 2)
 
Sort Column Only
  Turn on to sort only the selected column.
 
Case Sensitive
  Turn on to sort uppercase words before
  lowercase words; otherwise, Word ignores
  capitalization.

USING THE INSERT KEY FOR OVERTYPING
 
Overview: <Utilities Customize command>
 
To select the Word 5.0 Insert and Delete key
functionality:
1. Choose Utilities Customize (Alt,U,U).
2. Turn off the Use INS For Overtype Key check
   box.
3. Choose OK.
 
Note: Word 5.5 Help topics are written for Word
      5.5 key assignments. If you change the key
      assignments, see the Word 5.0 Using Manual
      for procedural information.

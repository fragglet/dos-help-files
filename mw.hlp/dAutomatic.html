dAutomatic
Automatic style
 
A style that Word applies to text unless you
apply a different style. Word provides automatic
styles that format characters, paragraphs, and
normal sections in your document. The formatting
instructions are predefined and determine the
format of your text automatically. You can change
the default formatting in Word by editing the
appropriate automatic style. For example, the
automatic paragraph style, Normal, determines the
default formatting for all paragraphs, including
the font type and size; if you change the font
type in this style, the normal paragraphs in your
documents default to this new font. See also:
Style; Style sheet.

FILE PRINT QUEUE
 
Procedures: <Queueing printing>
 
Use the File Print Queue command to continue
working while Word prints a document. It is
available only when the Print Queue option is
selected with the File Printer Setup command.
You can use the File Print Queue command to stop,
pause, continue, or restart printing from the
beginning of the document.
 
■ Use the File Printer Setup command to turn on
  the Use Print Queue option.
 
■ The number of documents in the queue is limited
  only by available disk space.

SUMMARY DIALOG BOX                (Screen 1 of 4)
 
File Name
  Displays the selected document's filename.
 
Size
  Indicates the total number of characters in the
  selected document.
 
Directory
  Shows where the document is located in your
  system.
 
Title
  Type the title of the document.
SUMMARY DIALOG BOX                (Screen 2 of 4)
 
Author
  Type the name of the person who wrote the
  document.
 
Operator
  Type the name of the person who typed the
  document.
 
Keywords
  Type one or more keywords you want to use in
  document searches.
SUMMARY DIALOG BOX                (Screen 3 of 4)
 
Comments
  Type comments you want to record about the
  document.
 
Version Number
  Type the document's version number, if it has
  one.
 
Date Created
  By default, a new document's Date Created is the
  current date. The Date Created always reflects
  the first date the document was saved. You can
  change this date.
SUMMARY DIALOG BOX                (Screen 4 of 4)
 
Date Saved
  The last date the document was saved displays.
  You can change this date.

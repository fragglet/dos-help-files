FILE SAVE ALL
 
Procedures: <Saving all active files>
 
Use the File Save All command to save all open
documents, any open style sheets, and the current
glossary file.
 
■ "Saving" displays in the message bar, followed
  by the filename and <path> of each document as
  it is saved. When the message disappears, the
  files are saved.
 
■ You must name each document with the Save As
  dialog box before it is saved.

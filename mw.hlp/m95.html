EDIT RENAME STYLE
 
Procedures: <Changing a style name>
 
Use the Edit Rename Style command to change the
<style I.D.>, <key code>, or <remark> for a
style. This command is available on the Edit menu
only when the <style sheet window> is <active>.
 
■ You change a style name to avoid duplication of
  styles or to give a style a more meaningful
  name.
 
More Help: <Style sheet commands>
 
Using Manual: Ch. 28, "Style Sheets"

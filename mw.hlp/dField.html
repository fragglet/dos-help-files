dField
Field
 
For form letters or documents, a piece of text
that is inserted into a main document from a data
document during printing.
 
A "field name" is used to mark the place in a main
document where text from a field will appear. You
type the field name, enclosed in chevrons
(« and »), in a main document at the location(s)
where you want to add information from a data
document. See also: Main document.

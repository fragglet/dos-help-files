WORD CANNOT COPY THIS FILE.
 
There are several possible reasons for this
message:
■ Your disk may be full and there is no space
  available to store a copy of this file. Choose
  File Exit Word and delete any files you no
  longer use.
■ The file you want to copy may be damaged. Use a
  backup copy of the file to make a new copy.
■ Word ran out of available memory while copying
  the file. Choose File Close All to save all
  open documents, style sheets, or glossary
  files. Then choose File File Management Copy
  again

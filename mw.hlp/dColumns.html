dColumns
Columns
 
Word documents can be created using a single
column, two or more newspaper-style columns, or a
combination of column formats. Another type of
column can created with tabs to form tables.
 
In the status bar, the column number shows the
current horizontal position of the cursor,
counting from the left margin.

PRINTER SETUP DIALOG BOX          (Screen 1 of 4)
 
Printer File
  Type the name of your printer file.
 
Files
  Select a printer file.
 
Directories
  Select the directory for the path you want to
  copy to.
 
Printer Name
  Select your printer's model name. The list
  displayed depends on the currently selected
  printer file.
PRINTER SETUP DIALOG BOX          (Screen 2 of 4)
 
Connect To
  Indicate your printer connection's location.
 
Paper Feed
  Select a paper source for your printer.
   Manual       Stops the printer at the end
                of every page and waits for you
                to insert another sheet of paper.
   Continuous   Prints continuously until the end
                of the document.
   Bin 1        Feeds paper from bin 1 for all
                pages.
   Bin 2        Feeds paper from bin 2 for all
                pages.
PRINTER SETUP DIALOG BOX          (Screen 3 of 4)
 
Paper Feed (continued)
   Bin 3        Feeds paper from bin 3 for all
                pages.
   Mixed        Feeds the first page from bin 1
                and all other pages from bin 2.
   Envelopes    Prints envelopes if the printer
                supports an envelope feeder.
 
Graphics Resolution
  Select a resolution level appropriate
  to your printer. The higher the resolution,
  the greater the time required to print.
 
 
PRINTER SETUP DIALOG BOX          (Screen 4 of 4)
 
Use Print Queue
  Turn on Use Print Queue to continue working in
  Word while your document is printing.
 
Skip Downloading Fonts
  If your .PRD file supports downloadable fonts,
  Word prompts you to download each time you
  print.  If you do not want to be prompted, turn
  on this option.

WINDOW ARRANGE ALL
 
Use the Window Arrange All command to display all
open windows on the Word screen. Windows are
arranged so that none overlap.
 
■ Use Window Arrange All (Alt,W,A) to view windows
  that may be hidden or to organize the windows on
  your screen.
 
■ You can arrange up to nine open document
  windows, plus the Help window, on the Word
  screen.
 
More Help: <Window menu>
 
Using Manual: Ch.12, "Windows"

REMOVING TABS USING THE RULER
 
Overview: <View Ruler command>
 
To remove tabs using the ruler:
 
Using the mouse:
1. Display the ruler by clicking the ruler/ribbon
   icon.
2. Select the paragraph(s) you want to remove
   tabs from.
3. <Drag> the tab you want to remove off
   the ruler.
 
Using the keyboard:
1. Select the paragraph(s) you want to remove
   tabs from.
2. Press Ctrl+Shift+F10 to activate and move the
   cursor to the ruler.
3. Move the cursor to the tab stop you want to
   remove using the Up Arrow and Down Arrow keys.
4. Press Delete to remove the tab stop.
5. Press Enter.

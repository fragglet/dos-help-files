WINDOW MAXIMIZE
 
Procedures: <Maximizing a window>
 
Use the Window Maximize command to expand the
active window to fill the Word screen.
 
■ When you maximize a window, other open windows
  remain open behind the maximized window.
 
■ Use the Window Maximize command to quickly
  enlarge your view of a document.
 
■ When the Window Maximize command is in effect,
  MX is displayed in the <status bar> and any
  windows opened are maximized.
 
■ Use the Window Restore command to reduce a
  maximized window to its original size.
 
■ If the window is already maximized, the
  Window Maximize command is dimmed.
 
■ To maximize the active window, press Ctrl+F10.

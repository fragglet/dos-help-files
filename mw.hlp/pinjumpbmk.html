GOING TO A BOOKMARK
 
Overview: <Insert Bookmark command>
 
To go to a bookmark:
1. Choose Edit Go To (Alt,E,G).
2. Select the bookmark name or type the name in
   the Go To box.
3. Choose OK.
 
More Help: <Working with bookmarks>
 
Using Manual: Ch. 13, "Bookmarks and
              Cross-Referencing"

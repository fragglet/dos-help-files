MACRO RECORD                      (Screen 1 of 2)
 
Use the Macro Record command to record a macro
while you are previewing a document.
 
To record a macro:
1. Choose Macro Record (Alt,M,C).
2. In the Macro Name box, type a macro name.
3. If you want to assign a key code, with the
   cursor in the Macro Keys box, press Ctrl and
   a one- or two-character key code. For example,
   Ctrl+M,C.
4. Choose OK.
   MR is displayed on the status bar.
MACRO RECORD                      (Screen 2 of 2)
 
5. Type the keystrokes and choose the commands you
   want to record. (Mouse actions are not recorded
   in macros.)
6. Choose Macro Stop Recorder (Alt,M,C).
   The new macro is automatically inserted in the
   current glossary.
 
For more information about the Macro Record
command, go to the document window, and choose
Help Index Commands.

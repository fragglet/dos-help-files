UTILITIES WORD COUNT
 
Use the Utilities Word Count command to count the
number of words in a document or selection.
 
■ This command counts the number of words in a
  selection or the number of words in a document
  if nothing is selected.
 
■ The number of words counted is entered in the
  scrap and displayed in the message bar.
 
■ To use this command, choose Utilities Word Count
  (Alt,U,W).
 
More Help: <Utilities menu commands>
 
Using Manual: Ch. 5, "Revising a Document"

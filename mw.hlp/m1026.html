EDIT REPLACE
 
Procedures: <Replacing text or formatting>
 
Use the Edit Replace command to search for and
replace specified text or formatting in the
active document. For example, you can replace
all occurrences of "don't" with "do not" or
replace all single underlining with double
underlining.
 
■ If any text is selected, Word replaces only
  within the selection.
 
■ Use the Edit Undo (Alt+Backspace) command to
  reverse accidentally replaced text or
  formatting.
 
■ Use a question mark (?) as a wildcard in your
  search text to match a single unspecified
  character.
  For more information on special characters, see
  the <Search and replace special characters>
  Help topic.

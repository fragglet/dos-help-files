FILE EXIT WORD
 
Procedures: <Ending a Word session>
 
Use the File Exit Word command to end your Word
session and return to the system prompt.
 
■ If you have already saved all documents before
  you exit, Word directly returns you to the
  system prompt.
 
■ If any open documents, style sheets, or
  glossary files have unsaved changes, Word
  displays a message asking you if you want to
  save changes.
 
■ Press Alt+F4 to exit Word.

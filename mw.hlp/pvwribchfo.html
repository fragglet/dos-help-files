APPLYING CHARACTER FORMATS USING THE RIBBON
 
Overview: <View Ribbon command>
 
To apply bold, italic, or underline character
formats with the ribbon:
1. Select the text you want to format, or
   position the cursor where you want to begin
   typing text with the new format.
2. Click the format you want to apply.
 
Note: If the selected text contains more than one
      character format, question marks will appear
      in the format names on the ribbon.
 
More Help: <Format Character command>
 
Using Manual: Ch. 6, "Text Formatting: Characters
                      and Paragraphs"

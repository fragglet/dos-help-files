VIEW RIBBON
 
Procedures: <Using the Ribbon>
 
Use the View Ribbon command to display or hide
the ribbon. The ribbon displays the font, size,
and format of the selected text and the style I.D.
of the current style.
 
■ Use the ribbon to quickly format text using the
  mouse or keyboard.
 
■ The ribbon settings refer only to the active
  window.
 
■ You can record style sheet changes using the
  ribbon. See the <Format Record Style> command
  for more information.

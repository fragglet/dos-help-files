REPAGINATE NOW DIALOG BOX         (Screen 1 of 1)
 
Confirm Page Breaks
  Turn on Confirm Page Breaks to confirm Word's
  placement of each page break. Word displays the
  confirmation prompt in the message bar. You can
  reposition the proposed page break using the Up
  Arrow or Down Arrow keys. Press Enter to confirm
  the page break position.
 
  If Confirm Page Breaks is turned off, Word
  automatically places page breaks.

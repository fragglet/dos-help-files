CHANGING A HEADER/FOOTER TO A NORMAL PARAGRAPH
 
Overview: <Format Header/Footer>
 
To change a header or footer to a normal
paragraph:
1. Position the cursor in the header or footer
   paragraph.
2. Choose Format Header/Footer (Alt,T,H).
3. Select the None option.
4. Choose OK.
 
More Help: <Formatting headers and footers>
 
Using Manual: Ch. 7, "Page Formatting"

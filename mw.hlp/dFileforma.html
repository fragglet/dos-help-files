dFileforma
File format
 
The format in which data is stored in a file.
Data can be stored in files in different file
formats. For example, Word normally stores a
document in "Word" file format, which includes
the text and all the formatting applied to the
document. A file can also be saved in "Text-Only"
file format, where the text is saved but without
any formatting; in "Text Only with Line Breaks"
format; or in "Rich Text Format" (RTF). Use the
File Save As command to save a document in a file
format other than Word file format. See also:
File; Import.

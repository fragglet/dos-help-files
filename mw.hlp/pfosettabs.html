SETTING TAB STOPS WITH PRECISE MEASUREMENTS
 
Overview: <Format Tabs command>
 
To set tab stops with precise measurements:
1. Position the cursor in the paragraph you want
   to set tabs for, or select the paragraphs you
   want to set tabs for.
2. Choose Format Tabs (Alt,T,T).
3. In the Tab Position box, type a measurement.
   If you want a measurement other than inches,
   include the unit. Example: 0.8 cm.
4. Choose Left, Right, Center, Decimal, or
   Vertical alignment.
5. Choose the Set button.
6. Repeat steps 3-5 for each tab stop you want to
   set.
7. Choose OK.
 
Note: Tab stops will not be displayed on the
      ruler until you choose OK.

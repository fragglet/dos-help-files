CHANGING THE DEFAULT CHARACTER FONT AND SIZE
 
Overview: <Format Define Styles command>
 
To change the <default> character font and
point size:
1. Choose Format Define Styles (Alt,T,D) to
   open the style sheet window.
2. Choose Insert New Style (Alt,I,N).
3. In the Style Type box, select Paragraph.
4. In the Style I.D. box, type or select Normal.
5. Choose OK.
6. Choose Format Character (Alt,T,C).
7. Make changes to the settings in the Font and
   Point Size boxes.
8. Choose OK.
 
More Help: <Working with styles and style sheets>
 
Using Manual: Ch. 28, "Style Sheets"

EDIT REPEAT
 
Use the Edit Repeat command to repeat the last
typing, editing, or formatting action carried out.
This command repeats only your last action.
 
■ You can use this command to quickly add a
  sentence or make revisions in several places
  in a long document.
 
■ Repeat is dimmed if the last action cannot be
  repeated.
 
■ If you accidentally repeat an action, choose
  Edit Undo (Alt+Backspace) to undo it.
 
■ A quick way to repeat an action is to use the
  repeat key, F4.

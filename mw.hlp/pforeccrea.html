CREATING A STYLE BY RECORDING FORMATTING
 
Overview: <Format Record Style command>
 
To create a style by recording formatting:
 
Using the Format Record Style command:
1. Select the text that has the format you want
   to record.
2. Choose Format Record Style (Alt,T,R).
3. In the Key Code box, type a one- or
   two-character code for the style.
4. In the Style Type box, select the type of
   style you are recording.
5. In the Remark box, type a note to help you
   remember the purpose of the style.
6. In the Style I.D. box, use the style I.D. Word
   proposes or select another one from the list.
7. Choose OK.
 
Using the Style box on the <ribbon>:
1. Select the text that has the format you want
   to record.
2. Choose View Ribbon (Alt,V,B) or press Ctrl+S
   to activate the ribbon.
3. In the Style box on the ribbon, select an
   appropriate style I.D from the list.
4. Press Enter or click the style I.D. with the
   mouse.
   Word displays the Record Style dialog box. You
   can assign a key code and remark to the style
   the same as in the procedure above.
5. Choose OK.

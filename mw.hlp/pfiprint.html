PRINTING A DOCUMENT
 
Overview: <File Print command>
 
To print a document:
1. Make sure the document you want to print is in
   the active window.
2. Choose File Print (Alt,F,P).
3. Select any options you want to specify.
4. Choose OK.
 
Using Manual: Ch. 10, "Printing"

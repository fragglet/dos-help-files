RENAMING A DOCUMENT
 
Overview: <File File Management command>
 
To rename a document:
1. Choose File File Management (Alt,F,F).
2. If the document you want to rename is not
   listed in the Files box, use the Search button
   to search for and list the document.
3. Select the document you want to rename.
4. Choose the Rename button.
5. Type the new filename.
6. Choose OK.
7. Choose Close to return to your document.
 
More Help: <Managing files>
 
Using Manual: Ch. 14, "File Management"

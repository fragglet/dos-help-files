14352
WORD CANNOT ADD TO THE DICTIONARY OF A DOCUMENT
THAT HAS NEVER BEEN SAVED.
 
You attempted to add a word to an unnamed
document dictionary. Word requires a document
name before it can create or add words to a
document dictionary.
 
To save a word to a document dictionary:
1. Choose OK.
2. Choose Close to exit Utilities Spelling.
3. Choose File Save As to name and save the
   document.
4. Choose Utilities Spelling, and when the word
   comes up again, add it to the document
   dictionary.

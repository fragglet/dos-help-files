NEW STYLE DIALOG BOX              (Screen 1 of 2)
 
Key Code
  Type a one- or two-character code for the
  style. For example, for a style for indented
  paragraphs, type IP.
 
Style Type
  Select the type of new style to insert:
  Paragraph   Determines paragraph formatting.
  Character   Determines character formatting.
  Section     Determines formatting for a
              document or a section of a
              document.
NEW STYLE DIALOG BOX              (Screen 2 of 2)
 
Remark
  Type a note to help you remember the purpose of
  the style.
 
Style I.D.
  Word suggests a style I.D. You can accept it
  or select another available style I.D. from the
  list.

USING COMMANDS
 
To:                            Press:
Activate or deactivate menu    Alt
Carry out a command            Enter
Cancel a command               Esc
Choose a menu                  Alt or F10, then
                                bold letter
Copy to scrap                  Ctrl+Ins
Delete selection to scrap      Shift+Del
Erase selection permanently    Del
Insert from scrap              Shift+Ins
Maximize the window            Ctrl+F10
Select command                 Alt+bold letter in
                                menu, then bold
                                letter in command
Undo last action               Alt+Backspace

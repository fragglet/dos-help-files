MACRO EDIT                        (Screen 1 of 2)
 
Use the Macro Edit command to open, save, or merge
glossaries, and to delete or clear macro entries
while you are previewing a document.
 
■ To open a glossary, choose Open Glossary in the
  Edit Macro dialog box.
■ To save a glossary, choose Save Glossary in the
  Edit Macro dialog box.
■ To merge glossaries, choose Merge in the Edit
  Macro dialog box.
■ To delete a macro entry, select or type the
  entry name and choose Delete in the Edit Macro
  dialog box.
MACRO EDIT                        (Screen 2 of 2)
 
■ To clear all macro entries, choose Clear All in
  the Edit Macro dialog box.
 
For more information about the Macro Edit
command, go to the document window, and choose
Help Index Commands.

REMOVING POSITIONING FROM A FRAME
 
Overview: <Format Position command>
 
To remove positioning from a frame:
1. Select the paragraph(s) that you have
   positioned.
2. Choose Format Position (Alt,T,O).
3. Choose the Reset button.
4. Choose OK.
 
More Help: <Positioning text or graphic frames>
 
Using Manual: Ch. 20, "Layout Features for
                       Graphics and Text"

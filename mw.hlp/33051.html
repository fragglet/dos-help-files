PARAGRAPH DIALOG BOX              (Screen 1 of 4)
 
Alignment
  Select the alignment you want for the
  paragraph(s). The Justified option aligns each
  line flush left and right with the margins.
 
Indents
  Type a measurement for the indentations:
   From Left    Indent from the left margin
   From Right   Indent from the right margin
   First Line   Indent only the first line of
                the paragraph(s)
PARAGRAPH DIALOG BOX              (Screen 2 of 4)
 
Indents (continued)
   Note: To create a hanging indent, type a
         negative number in the First Line box
         and a positive number in the From Left
         box.
 
Spacing
  Type the spacing (measured in lines) you want
  for the paragraph(s):
   Before   Spacing before the paragraph(s)
   After    Spacing after the paragraph(s)
   Line     Spacing between each line of the
            paragraph(s)
PARAGRAPH DIALOG BOX              (Screen 3 of 4)
 
Keep Paragraph
  Turn on options to control paragraph printing:
   Together   Prevents Word from starting a new
              page within a paragraph
   With Next  Prevents a page break between two
              specific paragraphs
 
Side By Side
  Turn on Side By Side to align the selected
  paragraphs next to each other. Note that the
  paragraph indents must be set so the paragraphs
  fit side by side. Choose View Layout to see the
  paragraphs as they will look when printed.
PARAGRAPH DIALOG BOX              (Screen 4 of 4)
 
Tabs
  Displays the Tabs dialog box to set tabs.
 
Borders
  Displays the Borders dialog box to format
  paragraph borders.
 
Position
  Displays the Position dialog box to position
  text or graphics.

MERGING AND PRINTING SPECIFIC RECORDS
 
Overview: <File Print Merge command>
 
To merge and print specific records:
1. Open the main document.
2. Choose File Print Merge (Alt,F,M).
3. In the Records box, type the numbers of the
   records you want to print.
4. Choose the Print button.
5. Choose OK to start printing.

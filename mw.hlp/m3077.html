FORMAT MARGINS
 
Procedures: <Formatting margins>
 
Use the Format Margins command to set page size
and page <margins>.
 
■ Settings you make with this command apply only
  to the active document, unless you turn on the
  Use As Default check box. Turn this on if you
  want the page and margin settings you set to be
  the <default> settings for all new documents.
  Note: These settings will be overridden by any
        changes made to the <automatic style>
        for section formatting.
 
■ Use the Gutter box to add a <gutter> margin
  to your document.
 
More Help: <Format menu commands>
 
Using Manual: Ch. 7, "Page Formatting"

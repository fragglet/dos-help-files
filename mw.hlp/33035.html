FOOTNOTE DIALOG BOX               (Screen 1 of 1)
 
Auto-Numbered Reference
  For automatically-numbered footnotes, select
  the Auto-Numbered Reference option.
 
Footnote Reference Mark
  For a different type of reference mark, type
  the character or characters you want to use in
  the Footnote Reference Mark box. For example,
  you might want to type an asterisk (*), or the
  word "note."

26643
THE DIRECTORY OR FILENAME IS NOT VALID.
 
Word searched for a particular file, but the path
or filename you specified does not exist.
 
1. Choose Search in the File Management dialog
   box again.
2. Retype the pathname correctly. Remember, the
   pathname begins with the drive name. For
   example, type C:\WORD\FILES to search the
   \FILES subdirectory of the \WORD directory
   on the C: drive.

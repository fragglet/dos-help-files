dPrinterdr
Printer driver (file)
 
A file for each of the printers supported by
Word. The file includes codes for controlling the
printer, a list of the fonts available on the
printer, and the dimensions of the characters in
each font. These files have the extension .PRD.
Select a printer driver with the File Printer
Setup command.

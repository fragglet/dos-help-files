FORMAT HEADER/FOOTER
 
Procedures: <Formatting headers and footers>
 
Use the Format Header/Footer command to create
and align <headers> and <footers>.
 
■ If the Format Header/Footer command is
  dimmed, you may be in <layout> view.
  Choose the View Layout command to leave
  layout view; then format headers and footers.
  If the command is still dimmed, make sure
  the cursor is in the document window.
 
■ Header or footer text can be formatted, edited,
  or deleted the same as other text in Word.
 
■ Headers are preset to print 0.5 inch from the
  top of the page, and footers 0.5 inch from the
  bottom. You can change these settings in the
  Header/Footer Options dialog box.
 
■ A caret (^) appears on the screen to the left
  of header or footer text.

RECORD STYLE DIALOG BOX           (Screen 1 of 3)
 
Key Code
  Type a one- or two-character code for the
  style. For example, for a style for indented
  paragraphs, type IP.
 
Style Type
  Select the type of style you want to record:
  Character   Records a character style such
              as font, superscript and subscript,
              bold, or italics.
  Paragraph   Records a paragraph style.
  Section     Records a style for a document or a
              section of a document.
RECORD STYLE DIALOG BOX           (Screen 2 of 3)
 
Remark
  Type a note to help you remember the purpose of
  the style.
 
Style I.D.
  Type the style I.D. for the style you want to
  record, or select it from the list. If you
  record a new style, Word displays an available
  style I.D.
 
Style to Record
  Lists the current style information for the
  style in the Style I.D. box.
RECORD STYLE DIALOG BOX           (Screen 3 of 3)
 
Define Styles
  Opens the style sheet window, where you can
  create and revise styles.

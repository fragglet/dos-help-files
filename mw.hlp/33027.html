OPEN GLOSSARY DIALOG BOX          (Screen 1 of 2)
 
File Name
  Type the name of the glossary file you want
  to open, or select a glossary file from the
  Files box.
 
Files
  Displays the available glossary files in the
  current directory. Select the glossary file
  you want to open.
 
Directories
  Select the appropriate directory if the
  glossary file you want to open is not in the
  current directory.
OPEN GLOSSARY DIALOG BOX          (Screen 2 of 2)
 
Read Only
  If you turn on the Read Only option, you cannot
  save changes made to the opened glossary file.

CHANGING POINT SIZE USING THE RIBBON
 
Overview: <View Ribbon command>
 
To change the point size with the ribbon:
 
Using the keyboard:
1. Select the text you want to change, or
   position the cursor where you want to begin
   typing text with the new point size.
2. Press Ctrl+P.
3. Press the Up Arrow and Down Arrow keys to
   scroll through the list of point sizes.
4. With the new point size in the Pts box, press
   Enter.
 
Using the mouse:
1. Select the text you want to change, or
   position the cursor where you want to begin
   typing text with the new point size.
2. Click the arrow to the right of the Pts box.
   A complete list of available point sizes
   is displayed. The sizes are given in points.
3. Click the point size you want to use.
 
More Help: <Format Character command>
 
Using Manual: Ch. 6, "Text Formatting: Characters
                      and Paragraphs"

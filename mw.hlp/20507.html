WORD WILL OVERWRITE THE RTF FILE FORMAT WITH
WORD'S FORMATTING.
 
You displayed an RTF document in Word format.
Now you have chosen File Save As to save the
document in Word format. Confirm that you want to
overwrite the RTF formatting with Word
formatting.
 
■ To overwrite the RTF formatted document, choose
  OK.
■ If you don't want to overwrite the RTF
  document, choose Cancel. Then save the file
  with a different filename.

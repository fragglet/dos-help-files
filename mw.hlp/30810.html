30810
THE FIELD NAME IS NOT VALID.
 
A macro variable name must begin with a letter
and cannot contain characters other than letters
and numbers. The name cannot exceed 64
characters.
 
1. Check the macro for a variable name that does
   not follow these rules.
2. If you find an incorrect macro variable name,
   rename it and try to run the macro again.

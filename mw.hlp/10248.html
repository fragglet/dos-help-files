10248
THE FILE IS READ-ONLY AND CANNOT BE WRITTEN TO.
 
The file has been set so that you can read
information from it but cannot change it. You
cannot save a read-only file with its original
name.
 
■ To save the file with a different name or to a
  different directory, choose File Save As.
■ To change the read-only attribute of the file
  and then save it, choose File DOS Commands or
  File OS/2 Commands.

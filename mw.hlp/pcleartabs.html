CLEARING ALL TAB STOPS
 
Overview: <Format Tabs command>
 
To clear all tab stops:
1. Position the cursor in the paragraph
   containing the tab stops you want to delete,
   or select a group of paragraphs.
   Note: If you select a group, only the tab
         stops common to all paragraphs will be
         shown on the ruler or in the dialog box.
2. Choose Format Tabs (Alt,T,T).
3. Choose the Clear All button.
4. Choose OK.
 
More Help: <Working with tabs>
 
Using Manual: Ch. 9, "Tabs and Tables"

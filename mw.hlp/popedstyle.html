OPENING AND EDITING A STYLE SHEET
 
Overview: <File Open command>
 
To open and edit a style sheet:
1. Choose File Open (Alt,F,O).
2. In the Show Files box, select the Style Sheets
   option.
3. In the File Name box, type the name of the
   style sheet or select a name from the Files
   list.
4. Choose OK.
5. Use the menu commands in the style sheet
   window to make changes to the style sheet.
6. Choose File Close to return to the current
   document.

FORMAT DEFINE STYLES
 
Procedures: <Creating a style>
 
Use the Format Define Styles command to view,
create, and revise <styles> and <style sheets>.
When you choose Format Define Styles, the
<style sheet window> opens. To close the
style sheet window, press Ctrl+F4 or click the
close icon in the upper-left corner of the window.
 
■ Use the menu commands in the style sheet window
  just as you would in any document window. For
  example, you can use the Edit Cut and Edit
  Paste commands to cut and paste styles within a
  style sheet or from one style sheet to another.
 
■ The Format Record Style command is a quicker
  way to create and revise individual styles.
  Format Define Styles is best for major
  revisions to style sheets; for example, cutting
  and pasting styles from one style sheet to
  another, changing the font for a number of
  similar styles, and merging style sheets.
 
■ To see the effect of changes made to a style on
  your document, choose the Window Arrange All
  command. This enables you to view the style
  sheet and the document on your screen at the
  same time.
 
■ The Edit Rename Style, Insert New Style, and
  Insert Style Sheet commands, which only appear
  in the style sheet window, perform special
  functions for working with style sheets. See
  <Style sheet commands> for more information.
 
■ To work with styles effectively, familiarize
  yourself with the terms <key code>, <remark>,
  <style type>, and <style I.D.>.

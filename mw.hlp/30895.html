STYLES CANNOT BE INSERTED INTO A DOCUMENT.
 
■ Apply styles to your document using the Format
  Apply Style command, the ribbon, or key codes.
■ To edit a style, choose Format Define Styles.
■ For more information, select Procedures from
  the Help Index and see the style sheets topics,
  or see Ch. 28, "Style Sheets," in the Using
  Manual.

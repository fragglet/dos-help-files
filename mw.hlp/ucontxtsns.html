HELP ABOUT COMMANDS, DIALOG BOXES, AND MESSAGES
 
You can get Help information about <commands>,
or any <dialog box> or <message> displayed
on the screen. For example, if you choose a
command and Word displays a dialog box, press F1
to view a Help screen describing the options in
the dialog box.
 
Command topics describe how to use each command.
Related tips are included in a bulleted list. You
can also <jump> from each command topic to
procedure topics, which provide step-by-step
instructions for specific tasks.
 
To get Help about a menu command:
1. Press Alt.
2. Use the direction keys to select a command
   from one of Word's menus.
3. Press F1.
 
To cancel the Help window:
■ Choose Exit at the top of the Help window or
  press Esc.
  -or-
■ Using the mouse, click the upper-left corner
  of the Help window.
 
To get Help about a dialog box:
■ Press F1 while the dialog box is displayed.
 
To get Help about a message:
■ Press F1 while the message is displayed.
 
To cancel Help for a dialog box or message:
■ Choose OK or press Esc.

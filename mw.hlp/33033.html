BREAK DIALOG BOX                  (Screen 1 of 2)
 
Insert Break
  Turn on Page, Column, or Section break.
 
Type of Section Break
  If you select Section break, indicate where the
  first page of the new section begins.
   Next Page   Begins a new section on the
               following page
   Continuous  Begins a new section immediately
               below the current section, on
               the same page
BREAK DIALOG BOX                  (Screen 2 of 2)
 
Type of Section Break (continued)
   Even Page   Begins a new section on the next
               even page
   Odd Page    Begins a new section on the next
               odd page

DELETING A STYLE FROM A STYLE SHEET
 
Overview: <Format Define Styles command>
 
To delete a style from a style sheet:
1. Choose Format Define Styles (Alt,T,D) to
   open the style sheet window.
2. Select the style you want to delete.
3. Choose Edit Cut (Alt,E,T).
 
Note: A style you delete is placed in the <scrap>.
      Use Edit Paste to insert it elsewhere.
 
More Help: <Working with styles and style sheets>
 
Using Manual: Ch. 28, "Style Sheets"

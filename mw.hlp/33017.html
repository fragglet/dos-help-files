PRINT MERGE DIALOG BOX            (Screen 1 of 2)
 
All
  Turn on All to merge all information from a
  specified file.
 
Records
  Turn on Records and enter specific records if
  you want to merge specific records from a file.
 
Print
  Choose Print to go to the Print dialog box and
  print the merged document.
PRINT MERGE DIALOG BOX            (Screen 2 of 2)
 
New Document
  Choose New Document to create one new document
  from the merged files.

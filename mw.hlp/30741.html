THE DATE IS NOT IN A VALID FORMAT.
 
Word requires a particular format for dates when
you use the File Management Search, Date Saved,
and Date Created features.
 
■ Choose Utilities Customize to see the Date
  format you selected.
■ Type the date in the format specified with
  Utilities Customize. Separate the elements of
  the date with a hyphen (-), slash (/), or
  period (.).

PRINTING MORE THAN ONE DOCUMENT
 
Overview: <File File Management command>
 
To print more than one document:
1. Choose File File Management (Alt,F,F).
2. Select or mark the documents you want to
   print.
3. Choose the Print button.
4. In the Print box, select Document to print the
   entire document.
   If you want to print summary information only,
   select Summary Info.
5. To print summary information along with the
   document, choose the Options button, turn on
   Summary Info in the Include box, and choose
   OK.
6. Choose OK to close the Print box and print
   your selection.
7. Choose Close to return to your document.

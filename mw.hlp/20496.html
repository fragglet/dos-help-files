DO YOU WANT TO DISPLAY IN WORD FORMAT?

The document you want to open is an RTF (Rich
Text Format) file. RTF files contain ASCII text
and formatting instructions used by some PC,
Macintosh, and XENIX/UNIX programs.
 
■ To display the file in Word format, choose Yes.
■ To display the file in RTF format with ASCII
  codes, choose No.
■ To cancel the command, choose Cancel.

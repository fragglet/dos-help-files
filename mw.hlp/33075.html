RUN MACRO DIALOG BOX              (Screen 1 of 1)
 
Macro Name
  Type the name of the macro you want to run, or
  select the macro from the list.
 
Step
  If you want the macro to run one step at a
  time, turn on the Step option. When you choose
  OK, the first step of the macro will run; then
  you can press any key (except Esc) to run the
  next step. Press Esc to exit the macro or turn
  off the Step option.
 
Macro Edit
  Displays the Edit Macro dialog box.

FORMAT BORDERS
 
Procedures: <Adding paragraph borders and shading>
 
Use the Format Borders command to add borders or
shading to selected paragraphs.
 
■ If you select multiple paragraphs, each
  one will display the borders you specify.
 
■ If you edit the text contained in bordered
  paragraphs, the border lines or shading expand
  or shrink to fit the edited text.
 
■ Shading does not show on the screen in text
  view. Choose the File Print Preview command to
  see how the shading will look when you print
  the document.

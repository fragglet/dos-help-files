WORD DOES NOT RECOGNIZE THE FORMAT OF THAT
FILE. DO YOU STILL WANT TO INSERT THE FILE?
 
You attempted to import or update a file that was
not in Word, Multiplan, Lotus, or Microsoft Excel
format. These are the file formats that Word
recognizes.
 
■ To insert the file in its existing format,
  choose OK.
■ To return to your document, choose Cancel.
  Then convert the file to a recognizable format
  before inserting it in the Word document.

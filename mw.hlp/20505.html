SAVING 'TEXT ONLY' WILL RESULT IN LOSS OF
FORMATTING.
 
Selecting Text Only or Text Only With Breaks in
the Save As dialog saves the document without
Word formatting.
 
■ To save the document without formatting, choose
  OK.
■ To keep the formatting as it is, choose Cancel.
  Then select a different format in the Format
  box.

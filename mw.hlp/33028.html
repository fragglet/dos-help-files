MERGE GLOSSARY DIALOG BOX         (Screen 1 of 1)
 
File Name
  Type or select from the Files box the name of
  the glossary file you want to merge with the
  active glossary file.
 
Files
  Shows the available glossary files in the
  current directory. Select the glossary file
  you want to merge.
 
Directories
  Select the appropriate directory if the
  glossary file you want to merge is not in the
  current directory.

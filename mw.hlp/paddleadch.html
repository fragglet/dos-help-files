ADDING LEADER CHARACTERS TO TAB STOPS
 
Overview: <Format Tabs command>
 
To add <leader characters>:
1. Position the cursor in the appropriate
   paragraph, or select a group of paragraphs.
2. Choose Format Tabs (Alt,T,T).
3. In the Tab Position box, select the tab
   position you want to set leader characters
   for, or type a new position to create a new
   tab stop.
4. Under Leader, select the character you want
   to add.
5. If it is a new tab stop, select the alignment
   you want.
6. Choose the Set button.
7. Repeat steps 3-6 for each tab stop.
8. Choose OK.
 
More Help: <Working with tabs>
 
Using Manual: Ch. 9, "Tabs and Tables"

FILE CLOSE ALL
 
Procedures: <Closing all files>
 
Use the File Close All command to close all
documents, style sheets, and glossary files.
 
■ File Close All cannot be reversed with the
  Edit Undo command.
 
■ Word prompts you to save changes in each
  document, style sheet, and glossary file before
  closing it.

CHANGING THE DEFAULT FORMAT FOR PARAGRAPHS
 
Overview: <Format Define Styles command>
 
To change the default format for paragraphs:
1. Choose Format Define Styles (Alt,T,D).
2. Select the Normal (paragraph) style.
3. Modify the style's format by choosing Format
   Paragraph or Format Character and selecting
   the options you want.
4. Choose File Save to save changes to the style
   sheet.
5. Choose File Close to close the style sheet
   window.
 
More Help: <Formatting paragraphs>
 
Using Manual: Ch. 6, "Text Formatting: Characters
                      and Paragraphs"

INSERT BOOKMARK
 
Procedures: <Working with bookmarks>
 
Use the Insert Bookmark command to mark and
assign a name to selected text. The first and
last characters of a selection serve as anchors
for the bookmark. Bookmarks are stored with a
document until you remove them.
 
■ Use bookmarks to move quickly to specific
  locations in a document.
 
■ If you remove a bookmark designation, the
  bookmark text is not deleted.
 
■ If you are in overtype mode, you can edit text
  between anchors without changing the bookmark.
 
■ If you copy bookmark text within the same
  document, the bookmark designation is not
  copied.
 
■ You can move or copy a bookmark (text and
  designation) to another document, as long as
  the destination document does not contain a
  bookmark with the same name.

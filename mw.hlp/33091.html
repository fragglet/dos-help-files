FILE DOS COMMANDS
 
Procedures: <Running DOS commands from Word>
 
Use File DOS Commands to run DOS commands without
ending your Word session. You can choose to run
one DOS command directly from the File DOS
Commands dialog box, or to exit Word temporarily
and run more than one command from the DOS system
prompt.
 
■ If you are using a system without a hard disk,
  Word displays a message prompting you to
  change disks if necessary.
 
■ When you change disks and press Y, the
  operating system starts.

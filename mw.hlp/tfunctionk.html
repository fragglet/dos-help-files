FUNCTION KEYS
 
You can reassign function keys to match Word 5.0
during Setup or you can use the Utilities
Customize command to reassign keys. The F1 key is
permanently assigned to Help, however. If you use
Word 5.0 function keys, press Alt+Shift+F1 to
move the cursor to the next window.

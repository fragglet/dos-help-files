FORMATTING CHARACTERS USING SPEED KEYS
 
Overview: <Format Character command>
 
To format characters using speed formatting keys:
1. Select the text to format.
2. Press Ctrl+the key for the format you want:
 
   For this format:            Press Ctrl+
   Bold                        B
   Italic                      I
   Underline                   U
   Double underline            D
   Small caps                  K
   Hidden                      H
   Superscript                 +
   Subscript                   =
   To remove character format  Z
    without changing current
    font, point size, or color
   Normal default character    Spacebar
 
Note: If you have assigned <key codes> to
      <macros>, press Ctrl+A and then
      press the format key. This prevents you
      from accidentally running a macro.

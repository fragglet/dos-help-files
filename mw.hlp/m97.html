INSERT STYLE SHEET
 
Procedures: <Merging style sheets>
 
Use the Insert Style Sheet command to merge
the <style sheet> that is open with another
style sheet. This command is available on the
Insert menu only when the <style sheet window>
is <active>.
 
■ If there are duplicate styles in the style
  sheets you merge, Word displays a message for
  each duplicate style; you can skip the style
  and continue merging, rename the style and
  merge it, or cancel the merge.

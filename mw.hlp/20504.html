THE FILE DOES NOT EXIST. DO YOU WANT TO
CREATE IT NOW?
 
Word cannot find the filename in this directory
and asks if you want to create a new file with
the name you typed.
 
■ To create a new file with the filename you
  typed, choose OK.
■ To return to the File Open dialog box, choose
  Cancel. Type the correct filename, or choose
  one from the Files list.

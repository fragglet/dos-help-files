OPENING A GLOSSARY FILE
 
Overview: <Edit Glossary command>
 
To open a glossary file:
1. Choose Edit Glossary (Alt,E,O).
2. Choose Open Glossary.
3. Type or select from the Files box the name
   of the glossary file you want to open.
4. Choose OK.
 
More Help: <Using glossaries>
 
Using Manual: Ch. 18, "Glossaries"

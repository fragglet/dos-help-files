SELECTING THE WORD 5.0 FUNCTION KEYS
 
Overview: <Utilities Customize command>
 
To select the Word 5.0 function keys:
1. Choose Utilities Customize (Alt,U,U).
2. Turn on the Use Word 5.0 Function Keys option
   in the Settings box.
3. Choose OK.
 
Note: You can get Help by pressing F1 regardless
      of whether you're using the Word 5.0 or 5.5
      function keys. However, the Help topics
      only refer to the Word 5.5 function keys.
      For more information about the 5.0 function
      keys, choose Word 5.0 To 5.5 on the Help
      menu.

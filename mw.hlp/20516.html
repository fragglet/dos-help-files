DO YOU WANT TO SAVE CHANGES TO (FILENAME)?
 
You have made changes to a document, style sheet,
or glossary file, or created a new one. If you do
not save before exiting Word, you will lose the
work you have done since you last saved.
 
■ To save changes, choose Yes.
■ To lose changes, choose No.

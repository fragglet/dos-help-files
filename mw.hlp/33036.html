FILE DIALOG BOX                   (Screen 1 of 2)
 
File Name
  Type the name of the file to be inserted, or
  select the file from the files list.
 
Range
  Type the range of cells to be inserted from the
  spreadsheet or select from the list, if you do
  not want to insert the entire file. The format
  for typing a range of cells is A1-F8, for
  example. "All" is the default selection.
 
Files
  Select a file to be inserted in a Word
  document.
FILE DIALOG BOX                   (Screen 2 of 2)
 
Directories
  If the file you want is not in the Files list,
  select the appropriate directory.
 
Update Link
  Choose Update Link to update imported text or
  spreadsheet data

CLOSING ALL FILES
 
Overview: <File Close All command>
 
To close all files, including glossary files and
style sheets:
1. Choose File Close All (Alt,F,L).
2. If you have made changes to a document
   since you last saved it, Word will prompt
   you to save the document.
   Choose Yes to save the document with any
   changes you have made, or No to discard
   unsaved changes.
 
Using Manual: Ch. 11, "Storing Documents"

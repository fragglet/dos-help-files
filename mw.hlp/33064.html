PARAGRAPH BORDERS DIALOG BOX       (Screen 1 of 2)
 
Border Type
  Select the option you want for your border(s):
   None                Turns off border(s)
   Box Each Paragraph  Encloses paragraph(s) in
                       a box
   Lines               Draws a border on the Top,
                       Bottom, Left, or Right, or
                       any combination
 
Line Style
  Select the line style you want for the border.
  If you are using a color printer, indicate the
  color in which you want the lines to print.
PARAGRAPH BORDER DIALOG BOX       (Screen 2 of 2)
 
Background Shading
  Set options for the background shading of the
  paragraph(s):
   Percentage   Controls the degree of shading
   Color        Determines the color of shading

ADDING PAGE NUMBERS TO HEADERS OR FOOTERS
 
Overview: <Format Header/Footer>
 
To add page numbers to headers or footers:
1. Position the cursor in the header or footer
   paragraph where you want the page number to
   print.
2. Type: page
   Make sure "page" has a blank space before it.
3. Press F3.
 
More Help: <Formatting headers and footers>
 
Using Manual: Ch. 7, "Page Formatting"

FILE PRINT
 
Procedures: <Printing a document>
 
The File Print command prints and controls
printing of documents.
 
■ To specify printer type, model, and which
  computer port is used to send information from
  the computer to the printer, use the
  File Printer Setup command. Check the Printer
  Manual to for your printer type and model.
 
■ To print several documents at once, use the
  File File Management command's Print feature.
 
More Help: <File menu commands>
 
Using Manual: Ch. 10, "Printing"

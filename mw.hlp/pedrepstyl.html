REPLACING STYLE FORMATTING
 
Overview: <Edit Replace command>
 
To replace a particular style:
1. Choose Edit Replace (Alt,E,E).
2. Turn off Confirm Changes if you do not want to
   confirm each replacement.
3. Choose Replace Formatting Only (Alt+F).
4. Choose Style (Alt+S).
5. Type the key code of the style you want to
   search for.
6. Type the key code of the style you want to
   replace with.
7. Choose OK.
 
To repeat the replacement for the next
occurrence, press F4.

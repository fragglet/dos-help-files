INDEX DIALOG BOX                  (Screen 1 of 1)
 
Separate Page Numbers From Entry By
  Index entries and page numbers are separated
  by one tab. You can specify a different
  measurement.
 
Indent Each Level
  The standard indent level is 0.2 inches. You can
  set the indent for each level in the index to a
  number you designate, or you can use a style
  sheet to determine index indenting.
 
Capitalize Main Entries
  Main index entries are automatically
  capitalized, unless you turn off this option.

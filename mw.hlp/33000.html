33000
NEW DIALOG BOX                   (Screen 1 of 1)
 
Type
  Turn on Document or Style Sheet to indicate the
  type of file you want to open.
 
Use Style Sheet
  Type or select the style sheet to be used with
  the document. NORMAL.STY is the default style
  sheet. Other available style sheets are listed.
 
Directories
  If the style sheet you want is not in the list,
  select the appropriate directory.

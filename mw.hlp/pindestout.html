DESIGNATING ENTRIES WITH AN OUTLINE
 
Overview: <Insert Table of Contents command>
 
To designate table of contents entries using an
outline:
1. Create an outline for the document as you
   normally would, assigning levels to the
   headings.
2. Adjust the outline so that only the headings
   you want included in the table of contents are
   visible.
 
More Help: <Creating a table of contents>
           <View Outline command>
 
Using Manual: Ch. 29, "Tables of Contents"

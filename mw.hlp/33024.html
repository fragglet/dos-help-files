GO TO DIALOG BOX                  (Screen 1 of 2)
 
Go To
  Type the letter and number combination that
  specifies the location you want to go to. Use
  P for page, S for section, A for annotation,
  or F for footnote. For example, type P4S2 to go
  to page four of section two.
 
Bookmark Name
  To go to a bookmark, select a bookmark name
  from the available bookmarks in the list.
GO TO DIALOG BOX                  (Screen 2 of 2)
 
Page
  To go to the next page, turn on Page. You don't
  need to type text in the Go To box to use the
  option buttons.
 
Bookmark
  Turn on Bookmark to go to the next bookmark.
 
Footnote
  Turn on Footnote to go to the next footnote.
 
Annotation
  Turn on Annotation to go to the next annotation.

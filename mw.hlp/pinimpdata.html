IMPORTING SPREADSHEET DATA
 
Overview: <Insert File command>
 
To import spreadsheet data:
1. Open the Word document that will receive the
   spreadsheet data.
2. In the Word document, move the cursor to the
   area where you want to insert data.
3. Choose Insert File (Alt,I,F).
4. In the File Name box, type the name of the
   spreadsheet file, or select the spreadsheet
   filename from the Directories and Files boxes.
5. Type a range of cells in the Range box or
   select from the list. Leave the Range box
   blank to import the entire spreadsheet. A
   range of cells might be listed as A1-E6. This
   would import all the spreadsheet data in rows
   A through E and columns 1 through 6.
6. To update the data in the future, turn on Link
   to insert hidden codes needed for updating.
7. Choose OK.

dMaindocum
Main document
 
A document containing standard text that is used
in creating a form document. In a typical form
document, such as a phone bill, Word merges the
standard text of the main document with the
varying text of a data document. The main
document contains special fields where Word
inserts varying text, such as each customer's
mailing address, from the data document. See
also: Data document; Field; Form document.

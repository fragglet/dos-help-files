RENAME STYLE DIALOG BOX           (Screen 1 of 2)
 
Key Code
  Type a one- or two-character code for the
  style. For example, for a style for indented
  paragraphs, type IP.
 
Remark
  Type a note to help you remember the purpose of
  the style.
 
Style I.D.
  In the Style I.D. box, do one of the following:
  ■ Accept the style I.D. that Word suggests.
  ■ Select an available style I.D. from the list.
RENAME STYLE DIALOG BOX           (Screen 2 of 2)
 
Style I.D. (continued)
    Any style I.D. in the list without parentheses
    to the right of it is available.
 
Style To Rename
  Lists the current formatting information for
  the style you are renaming.

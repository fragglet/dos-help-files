33001
OPEN DIALOG BOX                   (Screen 1 of 3)
 
File Name
  Type the name of the document you want to open.
 
Files
  Select one of the filenames listed in the
  current directory.
 
Directories
  If the file you want is not in the Files list,
  select the appropriate directory.
OPEN DIALOG BOX                   (Screen 2 of 3)
 
Show Files
  Select the type of files you want to list in
  the Files box. The default file type is
  Documents.
 
Read Only
  Turn on Read Only to open a file for viewing
  only. You cannot edit a file that is opened as
  Read Only.
 
File Management
  Choose the File Management button to display
  the File Management dialog box and search for
  other documents you want to open.
OPEN DIALOG BOX                   (Screen 3 of 3)
 
Options
  Choose Options to display the Options dialog
  box and set the default path that Word
  automatically uses to open and save files.

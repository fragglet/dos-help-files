10253
WORD CANNOT FIND THE STANDARD DICTIONARY. CHOOSE
UTILITIES CUSTOMIZE TO UPDATE THE SPELLER PATH.
 
Word cannot find the standard dictionary file on
your disk.
 
■ To see if SPELL-AM.LEX is in the Speller Name
  box, choose Utilities Customize.
■ Verify that the dictionary is in the specified
  directory. If it is not, either correct the
  pathname or copy SPELL-AM.LEX to the directory
  specified.

SEARCHING FOR FILES
 
Overview: <File File Management command>
 
To search for files:
1. Choose File File Management (Alt,F,F).
2. Choose the Search button.
3. Fill in the search options with information
   you want to use as search criteria.
4. Choose OK.
   The Search dialog box closes. Word looks for
   and displays a list of files that meet the
   search criteria.
5. Choose Close to return to your document.
 
More Help: <Managing files>

COMPILING A TABLE OF CONTENTS
 
Overview: <Insert Table of Contents command>
 
To compile a table of contents:
1. Choose Insert Table of Contents (Alt,I,C).
2. To create a table of contents from an outline,
   turn on Outline in the Create Table Of Contents
   From box.
   Turn on Codes to compile a table of contents
   from table codes.
3. Choose OK.
 
More Help: <Creating a table of contents>
 
Using Manual: Ch. 29, "Tables of Contents"

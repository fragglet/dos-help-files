DO YOU WANT TO OVERWRITE THE EXISTING FILE?
 
You have proposed a name already assigned to an
existing file. If you overwrite the existing
file, its previous contents will be replaced by
those of the file you are renaming.
 
■ To overwrite the existing file, choose OK.
■ If you do not want to overwrite the existing
  file, choose Cancel. Type a different name for
  the file you are renaming.

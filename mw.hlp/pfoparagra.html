FORMATTING A PARAGRAPH USING PARAGRAPH COMMAND
 
Overview: <Format Paragraph command>
 
To format a paragraph:
1. Select the paragraph you want to format.
2. Choose Format Paragraph (Alt,T,P).
3. In the Paragraph dialog box, select options
   and/or type measurements for indents and line
   spacing.
4. Choose OK.
 
To repeat formatting:
1. Select the next paragraph you want to format.
2. Press F4.
 
More Help: <Formatting paragraphs>
 
Using Manual: Ch. 6, "Text Formatting: Characters
                      and Paragraphs"

EDIT GO TO                        (Screen 1 of 2)
 
Use the Edit Go To command to move the cursor
to any page, bookmark, footnote, or annotation in
the document.
 
1. Choose Edit Go To (Alt,E,G).
2. To Go To:     Type:
   Annotation    A and the annotation number
   Bookmark      Bookmark name (or select from
                 the list)
   Footnote      F and the footnote number
   Page          The page number
   Section       S and a section number
3. Choose OK.
EDIT GO TO                        (Screen 2 of 2)
 
For more information about the Edit Go To
command, go to the document window, and choose
Help Index Commands.

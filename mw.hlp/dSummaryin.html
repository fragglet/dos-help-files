dSummaryin
Summary information
 
A collection of descriptive information about a
document such as author, title, subject, and
operator. You typically fill in summary
information the first time you save a document,
and the information is saved along with the
document for later reference. Recording summary
information is an option that you can turn on and
off with the Utilities Customize command. Use the
File Management command to view or edit summary
information.

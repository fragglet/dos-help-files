PRINT PREVIEW WINDOW              (Screen 1 of 2)
 
Use the File Print Preview command to display a
document as it will look when printed. You must
have a video adapter with graphics capabilities
installed.
 
■ You cannot edit a document in print preview.
 
■ Graphics are not displayed if Draft is selected
  in the Print Options dialog box.
 
■ Press F1 for Help.
PRINT PREVIEW WINDOW              (Screen 2 of 2)
 
To preview pages:
1. Move the cursor to the page you want to
   preview.
2. Choose File Print Preview (Alt,F,V), or press
   Ctrl+F9.
   The document is displayed beginning where the
   cursor was when you chose File Print Preview.
 
For more information about the File Print Preview
command, go to the document window, and choose
Help Index Commands.

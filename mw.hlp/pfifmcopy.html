COPYING OR ARCHIVING DOCUMENTS
 
Overview: <File File Management command>
 
To copy or archive documents:
1. Choose File File Management (Alt,F,F).
2. If the document you want to copy or archive
   is not listed in the Files box, use the Search
   button to search for and list the document.
3. Select or mark the document(s) you want to
   work with.
4. Choose the Copy button.
5. Choose the directory you want to copy files
   to, or type the pathname.
6. Turn on Copy Style Sheets to copy style sheets
   with a document.
7. Turn on Delete Files After Copy to delete
   files from their current location after they
   are copied.
8. Choose OK.
9. Choose Close to return to your document.
 
More Help: <Managing files>
 
Using Manual: Ch. 14, "File Management"

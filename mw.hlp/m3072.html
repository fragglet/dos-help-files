EDIT PASTE
 
Procedures: <Inserting text or graphics>
 
The Edit Paste command inserts a copy of the
<scrap> contents in front of the cursor in
the active document.
 
■ The Edit Paste command is unavailable if the
  scrap is empty.
 
■ To insert the contents of the scrap quickly,
  press Shift+Ins.

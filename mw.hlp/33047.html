CHARACTER DIALOG BOX              (Screen 1 of 2)
 
Font
  Type or select from the list the font you want.
  A font is a set of characters with a unique
  design.
 
Point Size
  Type or select from the list the point size you
  want. Point size is the size of text characters.
  There are 72 points per inch.
 
Color
  Type or select from the list the color you
  want for your text.
CHARACTER DIALOG BOX              (Screen 2 of 2)
 
Position
  Turn on the position you want:
  Normal       Aligns as regular text.
  Superscript  Aligns a half space above
               normal text, as in a footnote.
  Subscript    Aligns a half space below
               normal text, as in formulas.
 
Options
  In the bottom left of the dialog box, turn on
  any of the formatting options you want for your
  text. For example, you can make your text both
  bold and italic.

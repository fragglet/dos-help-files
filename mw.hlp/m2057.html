WINDOW RESTORE
 
Use the Window Restore command to reduce a
maximized window to its original size.
 
■ The Restore command is available only if the
  active window has been maximized.
 
■ Choose Window Restore (Alt,W,R), press
  Ctrl+F5, or click the Maximize icon to restore
  the active window.

FILE MANAGEMENT DIALOG BOX        (Screen 1 of 6)
 
Sorted By
  Indicates how the documents are listed in the
  Files box.
 
View
  The document list is displayed in short, long,
  or full view. Short view is the default. You can
  change to Full or Long view using the Options
  button. Full view includes document summary
  information. Long view displays filename,
  author, and title, unless you choose to sort by
  other information.
FILE MANAGEMENT DIALOG BOX        (Screen 2 of 6)
 
Open As Read Only
  Turn on Open As Read Only if you only want to
  view the document and do not want to edit it.
 
Files
  Documents are listed by directories, but you
  can sort by author, operator, date changed,
  date created, or character count. Once a list
  of documents is displayed, you can turn on a
  single document or a group of documents to work
  with.
FILE MANAGEMENT DIALOG BOX        (Screen 3 of 6)
 
Search
  Choose Search to display the Search dialog box,
  specify the criteria and directories for a
  document search, and display the filenames that
  match the search criteria.
 
Options
  Choose Options to display the Options dialog
  box and indicate how you want to list and view
  documents.
FILE MANAGEMENT DIALOG BOX        (Screen 4 of 6)
 
Delete
  Choose Delete to delete the marked or selected
  files. If there are no marked files, the
  selected files will be deleted. Word prompts
  you to confirm before carrying out the
  deletions.
 
Copy
  Choose Copy to display the Copy dialog box and
  copy marked documents to another drive or
  directory. You can delete copied files from the
  source directory and copy style sheets with
  documents.
FILE MANAGEMENT DIALOG BOX        (Screen 5 of 6)
 
Summary
  Choose Summary to display the Summary dialog
  box and view or edit document summary
  information. Include information you can later
  use for document searches.
 
Print
  Choose Print to display the Print dialog box
  and print summary information with or without
  the document(s) you have selected.
 
Rename
  Choose Rename to display the Rename dialog box
  and rename the selected document.
FILE MANAGEMENT DIALOG BOX        (Screen 6 of 6)
 
Open
  Choose Open to open the document you have
  selected.

SAVING ALL ACTIVE FILES
 
Overview: <File Save All command>
 
To save all active files:
■ Choose File Save All (Alt,F,E).
  Word saves all open documents, style sheets,
  and the current glossary file.
 
Using Manual: Ch. 11, "Storing Documents"

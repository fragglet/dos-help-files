OPTIONS DIALOG BOX                (Screen 1 of 2)
 
Sort Files By
  Turn on an option for how you want the
  filenames listed in the Files box of the File
  Management dialog box.
 
View File
  Turn on Short, Long, or Full, to determine what
  information you will view in the Files list of
  the File Management dialog box. Full view
  includes document summary information. Long
  view displays filename, author, and title,
  unless you choose to sort by other information.
OPTIONS DIALOG BOX                (Screen 2 of 2)
 
Update List After Copy Or Rename
  Turn on to update the Files list in the File
  Management dialog box after you copy or rename
  a file.

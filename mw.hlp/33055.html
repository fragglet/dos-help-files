TABLE OF CONTENTS DIALOG BOX      (Screen 1 of 2)
 
Create Table Of Contents From
  Turn on Outline to indicate that you want to
  create a table of contents from an outline. The
  default selection, Codes, indicates that you
  want to create a table of contents based on
  table codes.
 
Page Numbers
  Turn off the Show Page Numbers option to compile
  a table of contents without page numbers.
TABLE OF CONTENTS DIALOG BOX      (Screen 2 of 2)
 
Indent Each Level
  Choose whether you want Word to indent table of
  contents levels by the space that you specify,
  or to use a style sheet to determine table of
  contents indenting. The default indent is 0.4
  inches.

AUTOSAVE BACK UP FILES EXIST.
DO YOU WANT TO RECOVER FILES NOW?
 
■ To recover files with all changes made before
  the last autosave, choose Yes.
■ To disregard the message, choose No. Word keeps
  backup files with all changes made before the
  last autosave. If you choose No, the message
  will appear again the next time you start Word.

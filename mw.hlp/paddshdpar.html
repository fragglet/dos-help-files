ADDING SHADING TO A PARAGRAPH
 
Overview: <Format Borders command>
 
To add shading to a paragraph:
1. Place the cursor in the paragraph you want to
   shade.
2. Choose Format Borders (Alt,T,B).
3. Under Background Shading, select the shading
   you want from the Percentage list.
4. If you are using a color printer, select the
   shading color you want from the Color list.
5. Choose OK.
 
Note: Shading does not appear on the screen in
      text view; to preview shading, choose the
      File Print Preview command.

PRINT QUEUE DIALOG BOX            (Screen 1 of 2)
 
Message
  The message bar provides a status of the print
  queue, such as how many documents are waiting
  to print.
 
Pause
  Choose the Pause button to pause printing.
 
Continue
  Choose the Continue button to continue
  printing.
PRINT QUEUE DIALOG BOX            (Screen 2 of 2)
 
Restart File
  Choose the Restart File button to restart
  printing from the beginning of your document.
 
Stop Queue
  Choose the Stop Queue button to cancel a file
  that is printing.

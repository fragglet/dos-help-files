10245
THERE IS A PROBLEM WITH THE DISK.
 
There is a problem with a disk you are working
with. Before you continue:
 
■ Use the CHKDSK command to make sure the disk is
  formatted.
■ Make sure the disk is correctly inserted in the
  disk drive, and that the drive door is closed.
■ Remove the write-protect tab from the disk, if
  there is one.

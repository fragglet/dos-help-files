AN ENDREPEAT INSTRUCTION MUST FOLLOW A REPEAT
INSTRUCTION.
 
Word cannot run a macro that has (a) an ENDREPEAT
instruction that is not preceded by a REPEAT
instruction, or (b) a REPEAT instruction that is
not followed by an ENDREPEAT.
 
â–  Edit the macro so that it includes a REPEAT
  instruction for each ENDREPEAT instruction.

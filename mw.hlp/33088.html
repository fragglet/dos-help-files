MACRO RUN                         (Screen 1 of 2)
 
Use the Macro Run command to run a macro while
you are previewing a document.
 
To run a macro using the Macro Run command:
1. Move the cursor to where you want the macro
   to start running.
2. Choose Macro Run (Alt,M,R).
3. In the Macro Name box, type the macro name or
   select it from the list. (The list contains
   all the macros in the current glossary.)
4. Choose OK.
MACRO RUN                         (Screen 2 of 2)

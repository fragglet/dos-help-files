dStandardd
Standard dictionary
 
The dictionary provided on the Spell disk. This
dictionary contains over 130,000 words; it is
also referred to as the main dictionary. Word
checks the spelling of words in a selection or
document against this dictionary first, and then
checks the document or user dictionary if either
exists. See also: Document dictionary; User
dictionary.

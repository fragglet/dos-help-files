INSERT PAGE NUMBERS
 
Procedures: <Inserting page numbers>
 
Use the Insert Page Numbers command to add
page numbers to a document.
 
■ Word provides five page number formats:
  roman uppercase (II), roman lowercase (ii),
  arabic (2), and alphabetical uppercase and
  lowercase.
 
■ Page numbers start printing on the first page
  of the <section>.
 
■ Each page number prints in the same position on
  every page of a section, at either the top or
  the bottom.

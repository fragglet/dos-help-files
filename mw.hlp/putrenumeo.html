NUMBERING AN ENTIRE OUTLINE
 
Overview: <Utilities Renumber command>
 
To number an entire outline:
1. Select the first character or heading in the
   outline.
2. Choose Utilities Renumber (Alt,U,R).
3. Select the All option.
4. Turn on the Restart Sequence check box.
5. Choose OK.
 
More Help: <Numbering paragraphs and outlines>
 
Using Manual: Ch. 24, "Outlining"

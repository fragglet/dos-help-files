dGalleryco
Gallery command
 
In Word 5.0 and earlier versions of Word, the
command used to view and edit style sheets.
In Word 5.5, the Format Define Styles command has
the same function as the Gallery command. For
more information on working with styles and style
sheets, choose the "Styles and style sheets"
topic from the Procedures Index in Help.

CHANGING THE PAGE SIZE
 
Overview: <Format Margins command>
 
To change the page size:
1. Choose Format Margins (Alt,T,M).
2. In the Width and Height boxes, type the
   dimensions of the paper you are using.
3. Choose OK.
 
More Help:  <Formatting margins>
 
Using Manual: Ch. 7, "Page Formatting"

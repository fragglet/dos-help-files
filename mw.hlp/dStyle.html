Style
 
A set of formatting instructions that produces a
specific appearance for the text it is applied
to. Styles are applied to characters (text),
paragraphs, and sections (page divisions). You
can standardize your document by applying the
same style to all text you want to appear in the
same format. Styles can be created and stored in
style sheets, which are collections of styles
used for a particular type of document. The
commands for working with style sheets are found
on the Format menu. Styles can also be applied
using key codes or the Style box on the ribbon.
See also: Key code; Ribbon; Style sheet.

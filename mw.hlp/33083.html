STYLE SHEET DIALOG BOX            (Screen 1 of 1)
 
Style Sheet Name
  Type the name of the style sheet you want to
  insert, or select the style sheet from the
  Files list.
 
Files
  Displays the available style sheets in the
  current directory.
 
Directories
  If the style sheet you want to insert is not in
  the Files list, select the appropriate
  directory.

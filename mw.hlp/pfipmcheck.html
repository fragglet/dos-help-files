PRINT MERGE CHECKLIST
 
Overview: <File Print Merge command>
 
Use these questions to check your main and data
documents before you print merge:
 
Main Document
 
■ Does the main document start with a DATA
  instruction that specifies the name of the
  data file?
■ Are field names in the main document enclosed
  by chevrons (« »)?
  To create chevrons, press Ctrl+[ for the left
  chevron or Ctrl+] for the right chevron.
 
Data Document
 
■ Does the data document begin with a header
  record that contains field names from the main
  document?
■ Is each data record a single paragraph?

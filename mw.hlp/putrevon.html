TURNING REVISION MARKS ON AND OFF
 
Overview: <Utilities Revision Marks command>
 
To turn on Revision Marks:
1. Choose Utilities Revision Marks (Alt,U,M).
2. Turn on Mark Revisions.
3. Select an option from the Mark New Text With
   box. For example, select Bold if you want to
   display and print all revised text in bold
   letters. If you select Nothing, revised text
   is not marked.
4. Select where you want to place revision bars
   from the Revision Bars box.
5. Choose OK.
 
Note: Word displays RM in the status bar when
      Revision Marks is turned on.
 
To turn off Revision Marks:
1. Choose Utilities Revision Marks (Alt,U,M).
2. Turn off Mark Revisions.

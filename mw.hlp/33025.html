GLOSSARY DIALOG BOX               (Screen 1 of 4)
 
Glossary Name
  Type the name of the glossary entry you want
  or select a name from the Names box.
 
Optional Keys
  Press Ctrl+the key or keys you want to use to
  define the optional key code for a particular
  glossary entry.
GLOSSARY DIALOG BOX               (Screen 2 of 4)
 
Names
  The Names box lists all available glossary
  entry names. The selected glossary entry and
  its contents or the selected text are displayed
  below the Names box, as is the glossary file
  name.
 
Define
  Stores selected text as a glossary entry in
  the active glossary file.
 
Insert
  Inserts the selected glossary entry into the
  active document in front of the cursor.
GLOSSARY DIALOG BOX               (Screen 3 of 4)
 
Delete
  Removes a selected glossary entry from the
  active glossary file.
 
Clear All
  Removes all glossary entries, except the
  supplied entries, from the active glossary file.
  This function is not permanent until you save
  glossary changes.
 
Open Glossary
  Opens a glossary file other than the active
  glossary file. This command displays the Open
  Glossary dialog box.
GLOSSARY DIALOG BOX               (Screen 4 of 4)
 
Save Glossary
  Displays the Save Glossary dialog box.
  Choose Save Glossary to save glossary file
  changes.
 
Merge
  Displays the Merge Glossary dialog box.
  Type or select the glossary file you want to
  merge with the current glossary file.

EDITING OR DELETING WORDS IN A DICTIONARY
 
Overview: <Utilities Spelling command>
 
To edit or delete words in a dictionary:
1. Choose File Open (Alt,F,O).
2. Type the dictionary filename or select the
   dictionary file from the Files list that
   contains the words you want to delete or edit.
   Dictionary files all have a .CMP extension.
3. Delete or edit the words.
4. Choose File Save to save the changes you have
   made.
 
Note: You cannot edit or delete words provided in
      the main dictionary (SPELL-AM.LEX).
 
More Help: <Spelling procedures>
 
Using Manual: Ch.26, "Spelling and Hyphenation"

THE SPECIFIED RANGE IS 3-DIMENSIONAL.
ONLY THE RANGE ON THE FIRST SHEET WILL BE
IMPORTED.
 
Lotus 1-2-3 version 3.0 supports "3-dimensional
spreadsheets" with 3 axes. This is a group of
parallel spreadsheets that are linked together
and stored in a single file. Since Word can only
import a single spreadsheet at a time, only the
first sheet in the specified range is imported.

CREATING A DATA DOCUMENT
 
Overview: <File Print Merge command>
 
To create a data document:
1. Type a <header record> as the first paragraph
   of the data document. For example:
   city,state,zip,phone
2. Type a comma, or press tab once, between field
   names.
3. End the header record with a paragraph mark.
4. Type each data record, separating the field
   entries with commas or tab characters,
   whichever you used in the header record.
5. End each data record with a paragraph mark.

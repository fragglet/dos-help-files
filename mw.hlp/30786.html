30786
NO TEXT OR GRAPHIC IS FOUND IN THE WINDOWS
CLIPBOARD.
 
The text or graphic you want to insert is not in
the Windows Clipboard or is in a format that is
not valid for use in Word documents.
 
To try again:
1. Press Alt+Tab to return to Windows.
2. Verify that the text or graphic you want to
   insert in the Word document is located in the
   Windows Clipboard.
3. Return to Word and choose Edit Glossary again.

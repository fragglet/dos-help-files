THE SCRATCH FILE IS FULL.
 
As you work, Word records your revisions in a
temporary file (the "scratch file") with the
.TMP extension. Word has reached the size limit
for the .TMP file. You can make no more changes
until you clear the computer's memory.
 
1. Close and save all open documents, style
   sheets, and glossaries with File Close All.
2. Choose File Open to reopen the document you
   want to work on.

26762
THE OPERATING SYSTEM CANNOT FIND THE COMMAND OR
PROGRAM THAT YOU TYPED.
 
There are a number of possible solutions:
 
■ You might have typed an incorrect path to the
  program or command. Retype the program or
  command with the correct path.
■ If you have a floppy disk system, make sure
  that COMMAND.COM is on the disk in drive A.
■ Check your disks for the program or command you
  want to run. Insert the correct disk, if
  necessary.

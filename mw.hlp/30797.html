THE RANGE IS NOT VALID.
 
You typed a range of cells or an assigned range
name that Word does not recognize.
 
■ Correct any typing errors.
■ Check the syntax for specifying the range in
  the program used to create the spreadsheet and
  retype the range.
■ To import the entire spreadsheet, leave the
  Range box blank.
■ See Ch. 27, "Spreadsheet and Document
  Linking," in the Using Manual for more
  information on range values.

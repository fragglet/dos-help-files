dFootnoter
Footnote reference mark
 
A number or other character placed in the main
text of a document to indicate that additional
information about the topic being discussed
appears in a footnote.
 
"Automatic footnote reference mark" refers to an
option in the Insert Footnote command where Word
automatically assigns the next footnote number
and renumbers all footnotes accordingly as you
add or delete other footnotes.

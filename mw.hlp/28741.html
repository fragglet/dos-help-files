THERE IS INSUFFICIENT MEMORY FOR FILES RETRIEVED.
NARROW THE SEARCH.
 
There is not enough available memory for Word to
store all the filenames and associated data for
the documents you are searching for. You must
reduce the number of files that match the search
criteria.
 
■ Limit the number of Search Path directories in
  the Search dialog box so that Word searches
  fewer directories.
■ Reduce the number of search criteria in the
  other boxes in the Search dialog box.

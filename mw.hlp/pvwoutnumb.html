NUMBERING AN OUTLINE
 
Overview: <View Outline command>
 
To number the headings in an entire outline:
1. Choose View Outline (Alt,V,O). You must be in
   outline view to number outline headings.
2. Choose Utilities Renumber (Alt,U,R).
3. In the Renumber Paragraphs box, select the All
   option.
4. Turn on the Restart Sequence check box to
   number from the first number in the sequence,
   or turn off to number from the current
   position.
5. Choose OK.
 
To remove heading numbers:
1. Choose View Outline (Alt,V,O).
2. Place the cursor anywhere in the outline if
   you want to remove all heading numbers, or
   select the portion of the outline you want to
   remove heading numbers from.
3. Choose Utilities Renumber (Alt,U,R).
4. Select the Remove option.
5. Choose OK.

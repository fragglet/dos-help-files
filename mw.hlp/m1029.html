VIEW PREFERENCES
 
Procedures: <Choosing preferences>
 
Use the View Preferences command to:
 
■ Show or hide non-printing characters.
 
■ Show or hide line breaks, <style bar>, menu
  bar, window borders, and message bar.
 
■ Show or hide the vertical and horizontal
  <scroll bars>.
 
■ Change the cursor speed.
 
■ Show or hide line numbers and count blank
  spaces.
 
■ Select the monitor display mode.
 
■ Change screen colors with the Colors dialog box.
 
■ Customize other Word settings with the
  Customize dialog box.
 
More Help: <View menu>
           <Utilities Customize command>
 
Using Manual: Ch. 3, "The Word Screen"

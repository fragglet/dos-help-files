INTERRUPTING OR CANCELING A MACRO
 
Overview: <Macro Run command>
 
To interrupt or cancel a macro:
■ Press Esc.
  Word displays a message asking if you want to
  continue running the macro:
  * To stop running the macro and return to the
    active document, choose Cancel.
  * To continue running the macro one step at a
    time, turn on the Single Step option and
    choose OK.
  * To continue running the macro, choose OK.
 
Note: If you cancel a macro that has changed a
      document, be sure to correct the results
      of any unwanted actions.
 
More Help: <Working with macros>
 
Using Manual: Ch. 22, "Macros"

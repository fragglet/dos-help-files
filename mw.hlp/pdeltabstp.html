DELETING A TAB STOP
 
Overview: <Format Tabs command>
 
To delete a tab stop:
1. Position the cursor in the paragraph
   containing the tab stop you want to delete.
2. Choose Format Tabs (Alt,T,T).
3. In the Tab Position box, type the tab position
   you want to delete, or select the tab from the
   list.
4. Choose the Clear button.
5. Repeat step 3-4 for each tab stop you want to
   delete from the paragraph.
6. Choose OK.
 
More Help: <Working with tabs>
 
Using Manual: Ch. 9, "Tabs and Tables"

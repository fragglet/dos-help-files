SEARCH FOR CHARACTER DIALOG BOX   (Screen 1 of 2)
 
Select the character formatting you want to
search for. You can choose combinations of the
formatting options available in this dialog box.
 
Font
  Type or select from the list box the font you
  want to search for.
 
Point Size
  Type or select from the list box the point size
  you want to search for.
SEARCH FOR CHARACTER DIALOG BOX   (Screen 2 of 2)
 
Color
  Type or select from the list box the color you
  want to search for.
Format
  Turn on boxes that correspond to the character
  formatting you want to find.
Position
  Turn on the position option that corresponds
  to the character formatting you want to find.
 
For more information on character formatting and
the contents of this dialog box, see the Format
Character Help topics.

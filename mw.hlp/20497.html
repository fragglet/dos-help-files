WORD WAS UNABLE TO DOWNLOAD A FONT.
DO YOU WANT TO PRINT ANYWAY?
 
If you choose to print, your desired font, line
lengths, and other format choices might not print
as designated.
 
To print with the correct font:
1. Make sure the printer is connected properly
   and is online.
2. Make sure the correct font file is in the
   directory or drive specified in the File
   Printer Setup dialog box.
3. Choose File Printer Setup to select the port.
4. Turn the printer off and on to clear memory.
5. Reduce the number of fonts in the document.

Hidden text
 
A character format used to show or hide
designated text. Hidden text can be used for
comments or nonprinting characters, table
of contents (.c.) and index codes (.i.), and for
graphics imports (.g.) and spreadsheet linking
(.l.).
 
Use the Format Character command to apply or
remove hidden text. Use the View Preferences
command to set whether or not hidden text is
displayed on the screen. Use the File Print
command to set whether or not to print hidden
text.

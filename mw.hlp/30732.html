30732
WORD DOES NOT ACCEPT MACRO ARGUMENTS GREATER
THAN 32767 OR LESS THAN 0.
 
You have used an argument in a macro instruction
that is greater than 32767 or less than 0. For
example, «tab 40000».
 
■ Edit the argument so the values fall within the
  acceptable range. The preceding example should
  be changed to read «tab 40» or «repeat 2».

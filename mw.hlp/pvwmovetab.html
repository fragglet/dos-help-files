MOVING CUSTOM TAB STOPS USING THE RULER
 
Overview: <View Ruler command>
 
To move tab stops:
 
Using the mouse:
1. Display the ruler by clicking the ruler/ribbon
   icon.
2. Move the cursor to the paragraph or select a
   group of paragraphs with the tab stops you
   want to move.
   Word shows only the tab stops common to all the
   selected paragraphs.
3. <Drag> the tab stop you want to move
   to a new location on the ruler.
 
Using the keyboard:
1. Move the cursor to the paragraph or select a
   group of paragraphs with the tab stops you
   want to move.
   Word shows only the tab stops common to all the
   selected paragraphs.
2. Press Ctrl+Shift+F10 to activate and move the
   cursor to the ruler.
3. Move the cursor to the tab stop you want to
   move using the Up Arrow and Down Arrow keys.
4. Hold down the Ctrl key and move the tab stop to
   the position you want using the Right Arrow
   and Left Arrow keys.
5. Press Enter.

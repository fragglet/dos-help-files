UTILITIES REPAGINATE NOW
 
Procedures: <Repaginating a document>
 
Use the Utilities Repaginate Now command to auto-
matically set, review, or modify <page breaks>.
 
■ When you turn on the Confirm Page Breaks option,
  Word prompts you for confirmation from the
  message bar.
 
■ Word scrolls to the next page break after each
  confirmation.
 
■ Page breaks placed by the Repaginate Now command
  cannot be selected; you must use the Repaginate
  Now command to move or delete them.
 
■ You can move <soft page breaks> using the
  direction keys when Word prompts you for
  confirmation. You can only confirm or delete
  <hard page breaks>.

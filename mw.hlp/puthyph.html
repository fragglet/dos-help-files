HYPHENATING TEXT
 
Overview: <Utilities Hyphenate command>
 
To insert hyphens:
1. Do one of the following:
   ■ Move the cursor to the beginning of the
     document to hyphenate the entire document.
   ■ Select the contents you want to hyphenate.
2. Choose Utilities Hyphenate (Alt,U,H).
3. In the Hyphenate dialog box:
   ■ Turn on Confirm if you want to confirm each
     proposed hyphenation. You can use the
     direction keys to change Word's proposed
     division point.
   ■ Turn on Hyphenate Caps if you want to
     hyphenate words containing capital letters.
4. Choose OK.
 
If you have turned on Confirm, Word will prompt
you to confirm each hyphenation. Use the direction
keys to change Word's proposed division point.
 
Using Manual: Ch. 26, "Spelling and Hyphenation"

CUSTOMIZE DIALOG BOX              (Screen 1 of 5)
 
Autosave
  Specify in minutes how often you want Word to
  automatically save your work. Turn on Confirm
  if you want Word to prompt you at each autosave.
 
Settings
  Background Pagination
   Turn on for automatic repagination.
 
  Prompt For Summary Info
   Turn on if you want Word to prompt you for
   summary information the first time you save
   a document.
CUSTOMIZE DIALOG BOX              (Screen 2 of 5)
 
Settings (continued)
  Widow/Orphan Control
   Turn on to prevent Word from leaving a single
   line at the top or bottom of a page.
 
  Use Word 5.0 Function Keys
   Turn on to change the keyboard to Word 5.0
   functionality.
 
  Note: Online Help procedures use the Word 5.5
        key functionality.  See the Word 5.0 to
        5.5 Help topic for more information on
        the Word 5.0 function keys.
CUSTOMIZE DIALOG BOX              (Screen 3 of 5)
 
Settings (continued)
  Use INS For Overtype Key
   If turned on, the Insert key toggles overtype
   mode on and off. If you turn this option off,
   the Insert key inserts the scrap contents.
   See the Word 5.0 to 5.5 Help topic for more
   information on the Word 5.0 function keys.
 
  Show Keys On Menus
   Turn on to display short cut keys on menus.
CUSTOMIZE DIALOG BOX              (Screen 4 of 5)
 
Decimal  Select the type of decimal you want.
Date     Select MDY or DMY format.
Time     Select a 12 or 24 hour clock.
Measure  Select your preferred unit of measure.
Mute     Turn on or off Word's alert beep.
 
Line Draw
  Select the line draw character you want.
 
Default Tab
  Type the default tab spacing for the ruler.
CUSTOMIZE DIALOG BOX              (Screen 5 of 5)
 
Speller Name
  Select a speller name from the list. If the
  speller you want is not in the current
  directory, choose the appropriate directory
  from the Directories list.
 
Preferences
  Displays the Preferences dialog box.

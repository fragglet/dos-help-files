CHOOSING A DIFFERENT LINE DRAW CHARACTER
 
Overview: <Utilities Line Draw command>
 
To choose a different line draw character:
1. Choose Utilities Customize (Alt,U,U).
2. Select the character you want to use from the
   Line Draw Character list (use the Up Arrow and
   Down Arrow keys to scroll through the list), or
   type a character. For example, type "$" to draw
   lines consisting of dollar signs.
3. Choose OK to set the desired line draw
   character.
 
More Help: <Using line draw>
 
Using Manual: Ch. 21, "Line Drawing"

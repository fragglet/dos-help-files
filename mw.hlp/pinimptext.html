IMPORTING TEXT
 
Overview: <Insert File command>
 
To import text:
1.  Open the document that contains text you want
    to import.
2.  Select the text you want to import.
3.  Choose Insert Bookmark (Alt,I,M).
4.  In the Bookmark Name box, type a name for the
    selected text.
5.  Choose OK.
6.  Open the document that will receive the
    imported text.
7.  Move the cursor to where you want to insert
    text.
8.  Choose Insert File (Alt,I,F).
9.  In the File Name box, type the name of the
    document that contains the selected text or
    use the Directories and Files boxes to select
    a filename.
10. In the Range box, type the name of the
    bookmark you assigned to the text to be
    imported or select from the list. Leave the
    Range box blank to import the entire
    document.
11. To update the text in the future, turn on
    Link to insert hidden codes needed for
    updating.
12. Choose OK.

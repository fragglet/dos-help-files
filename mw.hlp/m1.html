FILE NEW
 
Procedures: <Creating a new document>
 
Use the File New command to create a new
document or <style sheet> in Word.
 
■ Open an existing document with the File Open
  command.
 
■ Word assumes you are creating a document in
  the current drive and directory, and proposes
  that you attach the default style sheet,
  NORMAL.STY.

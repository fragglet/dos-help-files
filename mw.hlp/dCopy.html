dCopy
Copy
 
To copy selected text or a graphic from a
document and place it in the scrap, in a
glossary, or in another location in the document,
without removing it from its original place in
the document. Use the Edit Copy command to copy,
and the Edit Paste command to insert the
copied text or graphic in another document or in
a different location in the same document. See
also: Cut and paste; Select.

30763
THE KEY CODE DOES NOT EXIST.
SEE THE STYLE SHEET FOR VALID KEY CODES.
 
The key code you typed is not listed on the style
sheet attached to your document.
 
■ Try pressing the keys again, making sure you
  type the key code correctly.
■ If the message appears again, check the
  attached style sheet by choosing Format Define
  Styles. If the style you are looking for is not
  listed, look for it on a different style sheet.
■ To open another style sheet, choose File Open.

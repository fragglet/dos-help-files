DO YOU WANT TO REPLACE THE EXISTING STYLE?
 
The key code or I.D. you specified is already
assigned to a style.
 
■ If you want to redefine the existing style,
  choose Yes.
■ If you do not want to replace the style that
  already exists, choose No. Then choose a
  different key code and I.D.

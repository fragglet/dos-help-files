WINDOW SIZE
 
Procedures: <Changing window size>
 
Use the Window Size command to change the size of
a window using the direction keys.
 
■ When you choose the Window Size command, the
  window border is highlighted and the word SIZE
  is displayed at the right end of the
  <status bar>.
 
■ You can resize windows to more conveniently
  view multiple windows.
 
■ Press Enter to set the window size and return to
  normal view.

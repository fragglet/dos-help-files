AN ENDCOMMENT INSTRUCTION MUST FOLLOW A COMMENT
INSTRUCTION.
 
Word cannot run a macro that has (a) an
ENDCOMMENT instruction that is not preceded by a
COMMENT instruction, or (b) a COMMENT instruction
that is not followed by an ENDCOMMENT.
 
â–  Edit the macro to include an ENDCOMMENT
  instruction for each COMMENT instruction.

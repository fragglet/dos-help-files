─────Specify Name of Bound Program (/O)─────────────────────────────────────
 
     Syntax:  /O[UTFILE] outfile
 
     Specifies the name of the bound program file in the <outfile>
     field. If you do not specify a path, BIND creates the file in the
     current directory.
 
     If this option is omitted, the bound program replaces the OS/2
     program.

─────BIND Overview──────────────────────────────────────────────────────────
 
     BIND converts protected-mode programs to run in real mode as well
     as protected mode. Once converted, the same .EXE file can be run
     under DOS and OS/2.
 
     Command-Line Syntax
 
     BIND infile [implibs] [linklibs] [options]

     Fatal LINK error L1063
 
     out of memory for CodeView information
 
     LINK was given too many object files with debug information, and
     it ran out of space to store them.
 
     Reduce the number of object files that have full debug information
     by compiling some files with either /Zd instead of /Zi or no
     CodeView option at all.

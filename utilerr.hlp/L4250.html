     ILINK warning L4250
 
     'number' undefined symbols
 
     A number of symbols were referred to in fixups but never publicly
     defined in the program. The given number indicates how many of
     these undefined symbols were found.
 
     This warning indicates an incremental violation, which causes
     ILINK to run the command specified with the /E option (or the
     default LINK command). Running a full link with /INCR should
     correct the condition.

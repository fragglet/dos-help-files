     Math error M6201
 
     'function' : DOMAIN error
 
     An argument to the given function was outside the domain of legal
     input values for that function.
 
     For example, the following expression causes this error:
 
          SQRT(-1.0)
 
     This error calls the matherr() function with the function name,
     its arguments, and the error type. You can rewrite matherr().

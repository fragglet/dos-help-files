     LIB warning U4156
 
     'library' : output-library specification ignored
 
     A new library was created because the filename specified in the
     <oldlibrary> field did not exist, but a filename was also
     specified in the <newlibrary> field. LIB ignored the <newlibrary>
     specification.
 
     For example, both of the following command lines cause this error
     if project.lib does not already exist:
 
          LIB project.lib +one.obj, new.lst, project.lib
          LIB project.lib +one.obj, new.lst, new.lib

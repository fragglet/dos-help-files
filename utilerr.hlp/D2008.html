     Command-line error D2008
 
     limit of 'option' exceeded at 'string'
 
     The given option was specified too many times. The given string is
     the argument to the option that caused the error.
 
     If the environment variable CL or FL is set, the compiler reads
     options specified in the environment variable before options
     specified on the command line.

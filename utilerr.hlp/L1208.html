     Fatal ILINK error L1208
 
     invalid .SYM file
 
     The .SYM file is invalid.
 
     Delete the file, and do a full link.
 
     If the problem persists, note the circumstances of the error and
     notify Microsoft Corporation by following the instructions in the
     Microsoft Product Assistance Request form at the back of one of
     your manuals.

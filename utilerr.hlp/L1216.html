     Fatal ILINK error L1216
 
     symbol 'symbol' multiply defined
 
     The given symbol was defined more than once.
 
     This is a breaking error and, like an incremental violation, it
     causes ILINK to run the command specified with the /E option (or
     the default LINK command). This error indicates a condition that
     prevents incremental linking and that cannot be corrected by
     running a full link.

     Fatal NMAKE error U1064
 
     MAKEFILE not found and no target specified
 
     No description file was found, and no target was specified.
 
     A description file can be specified either with the /F option or
     in a file named MAKEFILE. Note that NMAKE can create a target
     using an inference rule even if no description file is specified.

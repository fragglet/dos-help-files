     LINK warning L4011
 
     /PACKC value exceeding 64K-36 unreliable
 
     The packing limit specified with the /PACKC option was in the
     range 65,501─65,536 bytes. Code segments with a size in this range
     are unreliable on some versions of the 80286 processor.

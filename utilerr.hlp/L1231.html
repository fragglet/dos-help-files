     Fatal ILINK error L1231
 
     file 'filename' was not full linked
 
     An object file specified in the ILINK command line was not
     specified in the list of files in the most recent full link.
 
     This error indicates an incremental violation, which causes
     ILINK to run the command specified with the /E option (or the
     default LINK command). Running a full link with /INCR should
     correct the condition.

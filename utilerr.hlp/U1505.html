     Error U1505
 
     cannot read from file 'filename'
 
     PWBRMAKE could not read from the given file.
 
     One of the following may have occurred:
 
        ■ The file was corrupt.
 
        ■ The file was truncated. Truncation can occur if the compiler
          runs out of disk space or is interrupted when it is creating
          the .SBR file.

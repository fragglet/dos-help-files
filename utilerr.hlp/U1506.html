     Error U1506
 
     cannot open file 'filename'
 
     PWBRMAKE could not open the given file.
 
     One of the following may have occurred:
 
        ■ No more file handles were available. In DOS, increase the
          number of file handles by changing the FILES setting in
          CONFIG.SYS to allow a larger number of open files. FILES=20
          is the recommended setting.
 
        ■ The file was locked by another process.
 
        ■ The disk was full.
 
        ■ A hardware error occurred.
 
        ■ The specified output file had the same name as an existing
          subdirectory.

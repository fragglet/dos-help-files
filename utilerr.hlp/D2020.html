     Command-line error D2020
 
     'option' option requires extended keywords to be enabled (/Ze)
 
     The /Gc or /Gr option was specified on the same command line as
     the /Za option.
 
     The /Gc and /Gr options require the extended keyword _cdecl to be
     enabled. To enable _cdecl and make library functions accessible,
     specify the /Ze option.

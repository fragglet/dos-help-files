     NMK error U1451
 
     out of memory
 
     There was not enough available memory to complete the operation.
 
     One of the following may be a cause:
 
        ■ There are too many TSR programs installed. Remove some TSRs.
 
        ■ A previous command did not release memory when it terminated.
          This can happen if you attempt to run a TSR from within NMK.
 
        ■ There are too many active command shells. Close the current
          shell by entering EXIT at the operating-system prompt.

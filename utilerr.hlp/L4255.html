     ILINK warning L4255
 
     changed segment definition 'segment'
 
     The segment contribution changed for the given object file. It
     contributed to a segment to which it did not previously contribute,
     or a segment contribution was removed.
 
     This warning indicates an incremental violation, which causes
     ILINK to run the command specified with the /E option (or the
     default LINK command). Running a full link with /INCR should
     correct the condition.

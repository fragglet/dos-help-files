     Command-line warning D4001
 
     listing overrides assembly output
 
     An assembly listing was not generated because another listing
     option (/Fc or /Fl) was specified. The other option took effect.
 
     Check the CL or FL environment variable for option specifications.
     To create both listings, compile separately with each option.
 
     For example, the following command line causes this warning:
 
          cl /Fc /Fa program.c

     LINK warning L4036
 
     no automatic data segment
 
     The application did not define a group named DGROUP.
 
     DGROUP has special meaning to LINK, which uses it to identify the
     automatic (or default) data segment used by the operating system.
     Most OS/2 and Windows applications require DGROUP.
 
     This warning will not be issued if DATA NONE is declared or if the
     executable file is a dynamic-link library.

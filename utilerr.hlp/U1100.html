     EXEHDR error U1100
 
     invalid magic number 'number'
 
     EXEHDR discovered an unknown signature in the header for the file.
 
     The signature in the header for a file gives the operating system
     under which the executable file will run. EXEHDR recognizes
     signatures for DOS and OS/2 only.

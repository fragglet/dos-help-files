     Fatal NMAKE error U1056
 
     cannot find command processor
 
     The command processor was not found.
 
     NMAKE uses COMMAND.COM or CMD.EXE as a command processor to
     execute commands. It looks for the command processor first by the
     full pathname given by the COMSPEC environment variable. If
     COMSPEC does not exist, NMAKE searches the directories specified
     by the PATH environment variable.

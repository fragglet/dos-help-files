     NMAKE warning U4004
 
     too many rules for target 'targetname'
 
     Multiple blocks of build commands were specified for a target
     using single colons (:) as separators.
 
     To use multiple dependency blocks for the same target, specify a
     pair of colons (::) as the separator.

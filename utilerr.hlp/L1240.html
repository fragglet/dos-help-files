     Fatal ILINK error L1240
 
     far references in STRUCT fields not supported
 
     ILINK currently does not support STRUCT definitions like the
     following:
 
          extrn   func:FAR
          rek     STRUCT
          far_adr DD      func    ; Initialized far address
                                  ;  within a STRUCT
          rek     ENDS
 
     To use ILINK, change the code to get rid of the far address
     within the STRUCT.

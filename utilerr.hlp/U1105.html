     EXEHDR error U1105
 
     minimum allocation less than stack; correcting minimum
 
     If the minimum allocation is not enough to accommodate the stack
     (either the original stack request or the modified request), the
     minimum allocation value is adjusted.
 
     This error applies only to DOS programs.

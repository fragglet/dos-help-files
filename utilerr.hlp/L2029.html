     LINK error L2029
 
     'symbol' : unresolved external
 
     A symbol was declared to be external in one or more modules, but
     it was not publicly defined in any module or library.
 
     The name of the unresolved external symbol is given, then a list
     of object modules that contain references to this symbol. This
     message and the list are written to the map file, if one exists.
 
     One cause of this error is using the /NOI option for files that
     use case inconsistently.

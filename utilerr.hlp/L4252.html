     ILINK warning L4252
 
     file 'filename' does not exist
 
     ILINK could not find the given file.
 
     This warning indicates an incremental violation, which causes
     ILINK to run the command specified with the /E option (or the
     default LINK command). Running a full link with /INCR should
     correct the condition.

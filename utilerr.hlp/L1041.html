     Fatal LINK error L1041
 
     resident names table overflow
 
     The size of the resident names table exceeded 65,535 bytes.
 
     An entry in the resident names table is made for each exported
     routine designated RESIDENTNAME and consists of the name plus
     three bytes of information. The first entry is the module name.
 
     Reduce the number of exported routines or change some to
     nonresident status.

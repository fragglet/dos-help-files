     Fatal LIB error U1157
 
     comma or newline missing
 
     A comma or carriage return was expected in the command line but
     did not appear.
 
     This may indicate an incorrectly placed comma, as in the following
     command line:
 
          LIB math.lib, -mod1 +mod2;
 
     The line must be entered as follows:
 
          LIB math.lib -mod1 +mod2;

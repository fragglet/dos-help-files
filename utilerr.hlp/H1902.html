     HELPMAKE error H1902
 
     out of disk space for swap file
 
     The current drive or directory is full.
 
     HELPMAKE uses a temporary swap file, written to the current drive
     and directory. The temporary file can grow to 1.5 times the size
     of the input files (for large help files) and is not removed until
     the final help file is completed.

     BIND error U1269
 
     duplicate global name : 'name'
 
     The given global name was defined in more than one place in the
     specified libraries, making a unique fixup impossible.
 
     This error can be caused by specifying both OS2.LIB and
     DOSCALLS.LIB. To correct the error, specify only OS2.LIB.

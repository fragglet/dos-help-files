     Fatal NMAKE error U1099
 
     stack overflow
 
     The description file being processed was too complex for the
     current stack allocation in NMAKE.
 
     NMAKE has a default allocation of 0x3000 (12K).
 
     To increase NMAKE's stack allocation, run the EXEHDR utility with
     a larger stack option:
 
          EXEHDR /STACK:stacksize
 
     where <stacksize> is a number greater than the current stack
     allocation in NMAKE.

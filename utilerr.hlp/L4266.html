     ILINK warning L4266
 
     invalid .EXE file
 
     The executable file is invalid.
 
     Make sure the linker is current.
 
     This warning indicates an incremental violation, which causes
     ILINK to run the command specified with the /E option (or the
     default LINK command). Running a full link with /INCR should
     correct the condition.
 
     If the warning persists, note the circumstances of the warning and
     notify Microsoft Corporation by following the instructions in the
     Microsoft Product Assistance Request form at the back of one of
     your manuals.

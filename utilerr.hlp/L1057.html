     Fatal LINK error L1057
 
     data record too large
 
     An LEDATA record (in an object module) contained more than 1,024
     bytes of data. This is a translator error. (LEDATA is a DOS term
     explained in the "Microsoft MS-DOS Programmer's Reference" and in
     other DOS reference books.)
 
     Note which translator (compiler or assembler) produced the
     incorrect object module. Please report the circumstances of the
     error to Microsoft Corporation by following the instructions in
     the Microsoft Product Assistance Request form at the back of one
     of your manuals.

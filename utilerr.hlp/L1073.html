     Fatal LINK error L1073
 
     file-segment limit exceeded
 
     The number of physical or file segments exceeded the limit of 255
     imposed by OS/2 protected mode and by Windows for each application
     or dynamic-link library.
 
     A file segment is created for each group definition, nonpacked
     logical segment, and set of packed segments.
 
     Reduce the number of segments, or put more information into each
     segment. Use the /PACKC option or the /PACKD option or both.

     Fatal LINK error L1045
 
     too many TYPDEF records
 
     An object module contained more than 255 TYPDEF records. These
     records describe communal variables.
 
     This error can appear only with programs produced by the Microsoft
     FORTRAN Compiler or other compilers that support communal
     variables. (TYPDEF is a DOS term. It is explained in the
     "Microsoft MS-DOS Programmer's Reference" and in other reference
     books on DOS.)

─────$LIST and $NOLIST ─────────────────────────────────────────────────────
 
     Action
 
     $LIST prompts the user for a file name and sends subsequent listing
     information to the source-listing file. $NOLIST suppresses that
     information until a subsequent $LIST metacommand appears.
 
     Syntax  $[NO]LIST
 
     Remarks
 
     The default is $LIST.
 
     $LIST and $NOLIST can appear anywhere in a source file.

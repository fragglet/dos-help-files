EXTERN Attribute
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────EXTERN Attribute───────────────────────────────────────────────────────
 
     Place EXTERN in a variable declaration to indicate that the
     variable is allocated in source file written in a language other
     than FORTRAN. FORTRAN requires EXTERN when accessing variables
     declared in another language.
 
     Restrictions: Cannot apply to formal arguments

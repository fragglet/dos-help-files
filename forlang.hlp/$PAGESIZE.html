─────$PAGESIZE ─────────────────────────────────────────────────────────────
 
     Action
 
     $PAGESIZE sets the length of subsequent pages to a specified
     length.
 
     Syntax  $PAGESIZE:n
 
     Remarks
 
     The argument <n> must be at least 15. The default page size is 63
     lines.
 
     The PAGESIZE metacommand is equivalent to the /Sp compiler option.
 
     See Also: ◄Line Size and Page Size►
 
     Example
 
     $PAGESIZE:60

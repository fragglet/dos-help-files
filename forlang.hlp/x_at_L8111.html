getfontinfo
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The getfontinfo function gets the current font characteristics and
     stores them in the <fontinfo> structure (defined in FGRAPH.FD).
 
     Return Value
 
     The getfontinfo function returns a negative number if a font has
     not been registered or loaded; otherwise, it returns a nonnegative
     number.

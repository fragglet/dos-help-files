getwindowcoord
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The getwindowcoord subroutine translates the view coordinates (<x>,
     <y>) to window coordinates and returns them in a wxycoord
     structure (defined in FGRAPH.FD). The structure is returned in the
     <wpos> argument.
 
     See also: ◄Graphic Coordinate Systems►
 
     Return Value
 
     None.

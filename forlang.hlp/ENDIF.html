─────END IF─────────────────────────────────────────────────────────────────
 
     Action
 
     Terminates a block IF statement
 
     Syntax
 
     END IF
 
     Remarks
 
     There must be a matching END IF statement for every block IF
     statement in a program unit.
 
     Example
 
           IF (n .LT. 0) THEN
             x = -n
             y = -n
           END IF

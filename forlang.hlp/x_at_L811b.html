gettextwindow
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The gettextwindow function finds the boundaries of the current text
     window. The text window is the region of the screen to which output
     from the outtext function is limited. By default, this is the
     entire screen unless it has been redefined by the settextwindow
     function.
 
     Screen definitions by settextwindow have no effect on output from
     the outgtext function. Displaying text with outgtext is limited to
     the current viewport.
 
     Return Value
 
     None.

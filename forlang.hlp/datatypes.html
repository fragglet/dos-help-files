          ◄INTEGER►

          ◄COMPLEX►
          ◄LOGICAL►

          ◄STRUCTURE►
 
     Select one for more information.
 
     The data type of a variable, array, symbolic constant, or function
     can be declared in a specification statement. A type statement can
     also dimension an array variable and initialize variables and
     arrays.

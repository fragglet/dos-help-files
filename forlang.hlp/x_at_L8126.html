outtext
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The outtext subroutine sends the string <text> to the screen,
     including any trailing blanks. This subroutine works in any screen
     mode.
 
     No formatting is provided.
 
     Text output begins at the current text position.
 
     To display text using fonts, use the outgtext subroutine.
 
     Return Value
 
     None.

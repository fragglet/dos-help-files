     The setpixel and setpixel_w functions set a pixel at the specified
     location to the current color.
 
     The setpixel function sets the color of the pixel at the
     view-coordinate point (<x>, <y>).
 
     The setpixel_w function sets the color of the pixel at the
     window-coordinate point (<wx>, <wy>).
 
     Return Value
 
     The functions return the previous color index of the target pixel.
     If the function fails (for example, the point lies outside the
     clipping region), it returns -1.

clearscreen
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The clearscreen subroutine erases the target area, filling it with
     the current background color. The <area> parameter can be one of
     the following symbolic constants: $GCLEARSCREEN, $GVIEWPORT, or
     $GWINDOW.
 
     Constant           Meaning
 
     $GCLEARSCREEN      Clears the entire screen
 
     $GVIEWPORT         Clears only the current viewport
 
     $GWINDOW           Clears only the current window
 
     Return Value
 
     None.

Tabs/Column Position
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Tabs/Column Position───────────────────────────────────────────────────
 
     The tab character's interpretation depends on which column it
     is in:
 
     Column      Interpretation
 
     1-5         The character following the tab character in the
                 source line is interpreted as being in column 7.
 
     6 -72       The tab character is interpreted as a single blank,
                 unless it is in a character or Hollerith constant.
                 A tab character in a character or Hollerith constant
                 is interpreted as a tab character.
 
     NOTE: Unlike text editors, FORTRAN always interprets a tab in
     columns 6 through 72 as a single character. If your editor
     expands tabs to multiple spaces, it is possible for text
     appearing beyond column 72 in your editor to be processed by the
     FORTRAN compiler.

─────Remainders─────────────────────────────────────────────────────────────
 
     MOD, AMOD, and DMOD return division remainders as follows (AMOD and
     DMOD use exactly the same formula as MOD):
 
     MOD (genA, genB) = genA-(INT (genA / genB) * genB)
 
     If genB is 0, the result is undefined.

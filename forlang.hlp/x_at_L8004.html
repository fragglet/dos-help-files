─────Controlling Focus in Child Windows─────────────────────────────────────
 
     The IOFOCUS parameter determines whether a window receives the
     focus when a READ or WRITE statement is executed.
 
     See: ◄The IOFOCUS Parameter►
 
     The FOCUSQQ function gives the focus to a child window.
 
     See: ◄The FOCUSQQ Function►
 
     The INQFOCUSQQ function returns the number of the unit that
     currently has the focus.
 
     See: ◄The INQFOCUSQQ Function►

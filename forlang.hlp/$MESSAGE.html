─────$MESSAGE───────────────────────────────────────────────────────────────
 
     Action
 
     $MESSAGE sends a character string to the standard output device
     during the first compiler pass.
 
     Syntax  $MESSAGE:'string'
 
     Parameter          Description
 
     string             A character constant
 
     Example
 
     $MESSAGE:'Compiling program'

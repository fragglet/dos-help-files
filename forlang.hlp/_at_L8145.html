     The unregisterfonts subroutine frees memory previously allocated
     and used by the registerfonts function. The unregisterfonts
     function removes the header information for all fonts and unloads
     the currently selected font data from memory.
 
     Any attempt to use the setfont or outgtext routines after calling
     unregisterfonts results in an error.
 
     Return Value
 
     None.

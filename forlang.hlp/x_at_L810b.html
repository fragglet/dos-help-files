getactivepage
 ◄Summary► ◄Example►                         ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The getactivepage function returns the number of the current
     active page. All hardware combinations support at least one page
     (page number 0). In OS/2, only page 0 is valid.
 
     Return Value
 
     The function returns the page number of the current active page.

Using Cascaded Menus
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Microsoft Advisor──────────────────────────────────────────────────────
 
     A menu command followed by a right arrow opens a cascaded menu.
 
       File
     ┌──────────────┐
     │ New          │
     │ Open...      │
     ├──────────────┤
     │ Print...     │
     ├──────────────┤
     │ More Files  →┌───────────────┐
     │ Exit         │ 1 First File  │
     └──────────────│ 2 Second File │
                    │ 3 Second File │
                    └───────────────┘
 
     To open a cascaded menu, click the command or select the command
     and press ENTER or the RIGHT ARROW key.
 
     To close a cascaded menu, click the command or press the LEFT ARROW
     key.
 
     To close all open menus, press ESC.

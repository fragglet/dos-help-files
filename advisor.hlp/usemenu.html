     The name of each menu is listed at the top of the screen.
 
     To pull down a menu:
 
     Mouse: Click the name of the menu with the left mouse button.
 
     Keyboard: Press ALT and the first letter of the menu you want.
 
     To choose a command from a menu:
 
       File
     ┌──────────────┐
     │ New          │    Point to the command and click the left mouse
     │ Open...      │    button
     ├──────────────┤
     │ Print...     │    OR
     ├──────────────┤
     │ More Files  →│    Use the UP and DOWN keys to select the command,
     │ Exit         │    and press ENTER
     └──────────────┘
 
     Commands followed by an ellipsis (...) open a dialog box.
 
     Commands followed by a right arrow (→) open a cascaded menu.
     See: ◄Using Cascaded Menus►

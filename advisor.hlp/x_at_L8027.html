Tips for Using Dialog Boxes
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Microsoft Advisor──────────────────────────────────────────────────────
 
     Follow these tips when using a dialog box:
 
        ■ Move the cursor to an item in the dialog box by repeatedly
          pressing the TAB key or by holding down the ALT key and
          pressing the highlighted key for the item.
 
        ■ To move quickly to an entry in a list box:
 
            1. Move the cursor to the list box.
 
            2. Type a letter to select the first entry that starts with
               that letter.
 
            3. Type the letter again to move to the next entry starting
               with that letter. Repeat this process until the entry you
               want is highlighted, or use the ARROW keys.
 
        ■ Press ESC or choose Cancel to quickly exit a dialog box
          without performing any other action.
 
        ■ Press F1 or choose Help to get Help on using the dialog box.

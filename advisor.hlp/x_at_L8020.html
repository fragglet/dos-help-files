Tips for Using Menus
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Microsoft Advisor──────────────────────────────────────────────────────
 
     Follow these tips to open a menu or choose a command.
 
       ■  Pull down a menu by pressing both the ALT key and the first
          letter of the menu you want.
 
       ■  If a command has a shortcut key, it appears after the command
          name on the menu. Press this key to choose the command
          without pulling down the menu.
 
       ■  In CodeView, you can enter a command quickly by typing it in
          the command window.
 
          For more information on using the Command window in CodeView,
          see: ◄Description of Command-Window Commands►

     A dialog box provides more information on a menu command before it
     is executed and provides Help on error messages.
 
        ┌─ Dialog Box ─┐               ┌───────────────┐
        │      <Help>  │   ─────────►  │   Help Info   │
        └──────────────┘               └───────────────┘
 
     To get Help on a dialog box:
     ──────────────────────────────────────────────────────────────
     Using the Mouse              │  Using the Keyboard
                                  │
     Point to the Help button,    │  Press F1.
     and click the left mouse     │
     button.                      │

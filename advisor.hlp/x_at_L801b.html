Using the Index
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
─────Microsoft Advisor──────────────────────────────────────────────────────
 
     Each index has an alphabet bar across the top of the screen. Each
     letter brings up index entries that begin with that letter.
 
     If you don't find what you are looking for in one index, choose
     ◄Index► at the top of each screen to view a list of all indexes in
     the Help system. Then choose the name of another index.
 
     To choose a letter from the index alphabet bar:
     ─────────────────────────────────────────────────────────────────
     Using the Mouse              │  Using the Keyboard
                                  │
     Point to the letter, and     │  Type the letter you want, and
     double-click the left mouse  │  press ENTER.
     button.                      │
                                  │  TAB to the letter you want, and
                                  │  press ENTER.

─────Microsoft Advisor──────────────────────────────────────────────────────
 
     Help is available for every command on a menu.
 
          File
        ┌───────────────┐       ┌──┐      ┌───────────────┐
        ▐▌New▐██████████▌   ──► │F1│ ──►  │               │
        │ Open...       │       └──┘      │ Help on "New" │
        │ Save          │                 │               │
        └───────────────┘                 └───────────────┘
 
     To get Help on a menu command:
 
       1. Open the menu.
       2. Use the ARROW keys to highlight the command you want.
       3. Click the right mouse button or press F1.

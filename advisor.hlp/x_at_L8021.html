─────Microsoft Advisor──────────────────────────────────────────────────────
 
     Dialog boxes generally require you to:
 
       ■ ◄Enter text► to identify the object that a command uses or
         operates upon.
 
       ■ ◄Select items from a list► to identify the object that a
         command uses or operates upon.
 
       ■ ◄Select items from a drop-down list► to identify the object
         that a command uses or operates upon.
 
       ■ ◄Turn options on and off► to determine how the program should
         perform an operation
 
       ■ ◄Choose Commands► to execute when you close the dialog box.

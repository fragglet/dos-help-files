 FIX(numeric-expression)
 INT(numeric-expression)
 
 Usage Notes
   ■ FIX(x) is equivalent to SGN(x)*INT(ABS(x)).
 
   ■ For a negative numeric-expression, FIX returns the first negative
     integer greater than numeric-expression; INT returns the first negative
     integer less than numeric-expression.

 [form.][ctlname.]REFRESH
 
 Usage Note
   ■ Generally, a form or control is automatically refreshed after all events
     have been processed. However, there may be situations where you want the
     form or control refreshed immediately. For example, if you use a file
     list box, directory list box, or drive list box to show the current
     status of the file system, you can use REFRESH to update the status
     when a change is made. See: ◄Change Event►

 Screen Mode Summary
 
 ■ The Hercules Graphics Card, Graphics Card Plus, and InColor adapters are
   supported, but only with monochrome monitors.
 
 ■ Many screen modes support more than one combination of rows and columns on
   the screen.
 
 ■ The following table briefly summarizes each of the screen modes. The color
   adapters referred to are:
   • IBM Monochrome Display and Printer Adapter (MDPA)
   • IBM Color Graphics Adapter (CGA)
   • IBM Enhanced Graphics Adapter (EGA)
   • IBM Video Graphics Array (VGA)
   • IBM Multicolor Graphics Array (MCGA)
   • Olivetti Color Adapter

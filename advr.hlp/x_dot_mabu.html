 [form.]MaxButton
 
 Usage Notes
   ■ A Maximize button allows users to enlarge a form to full-screen size.
     When a form is maximized, the Maximize button automatically becomes a
     Restore button. Minimizing or restoring automatically changes the
     Restore button back to a Maximize button.
     See: ◄Maximize Button Definition►  ◄Restore Button Definition►
 
   ■ Maximizing a form at run time generates a Resize event.
     See: ◄Resize Event►
 
   ■ If BorderStyle is set to 0, the MaxButton value is ignored even if set
     to True (-1). See: ◄BorderStyle Property►
 
   ■ The WindowState property reflects the current state of the window. If
     you set the WindowState property to 2 (Maximized), the form is maximized
     independently of whatever settings are in effect for the MaxButton and
     BorderStyle properties. See: ◄WindowState Property►

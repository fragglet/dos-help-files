 PEN(n%)
 
 Usage Notes
   ■ The PEN function does not work when the mouse driver is enabled because
     the mouse driver uses the PEN function's BIOS calls.
 
   ■ Use mouse function 14 to disable the driver's lightpen emulation. Mouse
     function 13 turns emulation back on.

 [form.]textbox.MultiLine
 
 Usage Notes
   ■ A text box control with its MultiLine property set to True (-1) is
     referred to as a multiline text box. A multiline text box automatically
     wraps text extending beyond the display area. See: ◄Text Box Control►
 
   ■ Scroll bars may also be added to a text box using the ScrollBars
     property. See: ◄ScrollBars Property►
 
   ■ On a form with no default button, pressing Enter in a multiline text
     box moves the cursor to the next line. If a default button exists, you
     must press Ctrl+Enter to move to the next line. See: ◄Default Property►

 [form.]ControlBox
 
 Usage Notes
   ■ Double-clicking the control-menu box closes the active window. If this
     is the last form displayed, this action will end the program.
 
   ■ The Control menu is often used to provide a window control keyboard
     interface for non-mouse users. See: ◄Control Menu Definition►
 
   ■ A Control-menu box cannot be displayed on a form with BorderStyle set
     to 0 (none). See: ◄BorderStyle Property►
 
   ■ Either a modal or a modeless window can have a Control-menu box.
     See: ◄Modal Vs. Modeless►

 [form.][ctlname.]Tag[ = stringexpression$]
 
 Usage Notes
   ■ Use this property to assign a unique identification string to an object
     without affecting any of its other property settings or causing side
     effects.
 
   ■ Use the Tag property when you need to identify a specific control that
     is passed to a procedure. This is done because CtlName and FormName
     cannot be used as unique identifiers at run time.
     See: ◄CtlName Property►  ◄FormName Property►
 
   ■ The Tag property is for generic data storage. You can use the Tag
     property to attach information to a control that is not already handled
     by any of its other properties.

  Directive:    SQLACCESS
 
  Function:     Specifies the name of the access plan to be created by
                precompiler services and stored in the database.
 
  Syntax:
 
     ►►─┬───┬─┬─────SQLACCESS─┬"access-plan"┬┬─►◄
        └─/─┘ │               └""───────────┘│
              └──NO─SQLACCESS────────────────┘
 
  Parameters:   access-plan          The root of a file-name, or a null
                                     string.
 
  Default:      SQLACCESS ""
 
  Phase:        Syntax check
 
  $SET:         Initial

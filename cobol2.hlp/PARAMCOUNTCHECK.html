  Directive:    PARAMCOUNTCHECK
 
  Function:     Allows the program to be called with fewer parameters
                than are specified in the relevant entry point's USING
                clause.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──PARAMCOUNTCHECK───────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOPARAMCOUNTCHECK  (with OMF "OBJ")
                PARAMCOUNTCHECK    (with OMF "GNT")
 
  Phase:        Generate
 
  $SET:         Initial

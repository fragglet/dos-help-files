  Directive:    CONVERTRET
 
  Function:     Causes COMP items specified in CALL ... RETURNING and
                EXIT PROGRAM ... RETURNING phrases to be converted to
                COMP-5.
 
  Syntax:
 
     ►►─┬───┬─┬────┬─CONVERTRET─────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOCONVERTRET
 
  Phase:        Syntax check
 
  $SET:         Initial

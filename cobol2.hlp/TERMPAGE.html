  Directive:    TERMPAGE
 
  Function:     Determines whether the last page of a report file
                is to be padded with blank lines until it is a full page
                in length.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──TERMPAGE──────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      TERMPAGE
 
  Phase:        Syntax check
 
  $SET:         Initial

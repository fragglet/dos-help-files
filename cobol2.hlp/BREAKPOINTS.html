  Directive:    BREAKPOINTS
 
  Function:     Specifies the number of breakpoints that can be set in any
                one program.
 
  Syntax:
 
     ►►─┬───┬──BREAKPOINTS──"integer"───────────►◄
        └─/─┘
 
  Parameters:   integer              Integer between 4 and 100 defining
                                     the number of breakpoints allowed.
 
  Default:      BREAKPOINTS "100"

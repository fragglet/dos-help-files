  Directive:    SYMBSTART
 
  Function:     Sets the number from which the compiler counts
                positions in the collating sequence when compiling the
                SYMBOLIC CHARACTERS clause.
 
  Syntax:
 
     ►►─┬───┬──SYMBSTART──"integer"─────────────►◄
        └─/─┘
 
  Parameters:   integer              The number to be used.
 
  Default:      SYMBSTART "1"
 
  Phase:        Syntax check
 
  $SET:         Initial

  Directive:    ALIAS
 
  Function:     Prevents the Compiler from performing certain
                optimizations that are unsafe when a program uses
                aliasing.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──ALIAS─────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOALIAS
 
  Phase:        Generate
 
  $SET:         Initial

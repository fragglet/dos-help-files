     See ◄Introduction to switches►
 
     Specifying +C2 disables the creation of a COBOL text window for
     Presentation Manager applications. However, if an RTS error occurs
     with this switch set, a text window displaying the error message text
     is created. This provides compatibility with previous releases of this
     COBOL system.

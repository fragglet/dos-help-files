  Directive:    SQLBIND
 
  Function:     Specifies the name of the bind file to be created by
                precompiler services.
 
  Syntax:
 
     ►►─┬───┬─┬─────SQLBIND──"bindfile-name"┬┬──►◄
        └─/─┘ │             └""─────────────┘│
              └──NO─SQLBIND──────────────────┘
 
 
  Parameters:   bindfile-name        A full file specification or a
                                     null string.
 
  Default:      SQLBIND ""
 
  Phase:        Syntax check

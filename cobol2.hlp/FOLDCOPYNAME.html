  Directive:    FOLDCOPYNAME
 
  Function:     Determines whether COPY file names should be converted
                to upper case or lower case.
 
  Syntax:
 
     ►►─┬───┬─┬─────FOLDCOPYNAME─"case"─┬───────►◄
        └─/─┘ └─NO──FOLDCOPYNAME────────┘
 
  Parameters:   case                 "upper" or "lower".
 
  Default:      NOFOLDCOPYNAME
 
  Phase:        Syntax check
 
  $SET:         Any

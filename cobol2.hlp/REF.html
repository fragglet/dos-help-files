  Directive:    REF
 
  Function:     Makes the compiler include in the source listing the
                intermediate code address of each data item or
                Procedure Division statement; and in the object code
                listing, the address of each Procedure Division
                statement.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──REF───────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOREF
 
  Phase:        Both
 
  $SET:         Any

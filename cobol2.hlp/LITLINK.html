LITLINK (↑ Object Code)
 ◄Key► ◄Details►                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
  Directive:    LITLINK
 
  Function:     Makes the compiler declare the literals in CALL
                literal statements as public symbols, so they are
                resolved at link time rather than run time.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──LITLINK───────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOLITLINK
 
  Phase:        Generate
 
  $SET:         Initial

────────────────────────────────────────────────────────────────────────────
 
  Component:    LINK
 
  Function:     Produces executable files from intermediate and
                native code object files created by the compiler.
 
  Syntax:
 
  ►►─LINK─┬─────────────────────────────────────────────────────────────┬►◄
          └obj-names┬─────────────────────────────────────────┬┬────┬┬─┬┘
                    └,─┬────────┬─┬───────────────────────────┤└opts┘└;┘
                       └run-file┘ └,─┬────────┬┬──────────────┤
                                     └lst-file┘└,─┬────┬┬─────┤
                                                  └libs┘└,─def┘
 
  Parameters:   obj-names        Object files.
                run-file         Executable output file.
                lst-file         File for map.
                libs             Library files.
                def              Definition file.
                opts             Linker options.

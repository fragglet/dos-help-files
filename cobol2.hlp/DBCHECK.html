  Directive:    DBCHECK
 
  Function:     Makes the compiler check that any Double-Byte Character
                Set (DBCS) literals only contain valid 16-bit DBCS
                characters.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──DBCHECK───────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NODBCHECK
 
  Phase:        Syntax check
 
  $SET:         Initial

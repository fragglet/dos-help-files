────────────────────────────────────────────────────────────────────────────
 
  Directive:    MASM
 
  Function:     Makes the compiler produce a listing file that can be
                submitted to Microsoft MASM to produce an .OBJ file.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──MASM──────────────────────►◄
        └─/─┘ └─NO─┘
 
 
  Parameters:   None
 
  Default:      NOMASM
 
  Phase:        Generate
 
  $SET:         Initial

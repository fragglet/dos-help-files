  Directive:    AUTOLOCK
 
  Function:     Makes the default locking AUTOMATIC rather than
                EXCLUSIVE for files opened I-O or EXTEND in a
                multi-user environment.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──AUTOLOCK──────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOAUTOLOCK
 
  Phase:        Syntax check
 
  $SET:         Initial

────────────────────────────────────────────────────────────────────────────
 
  Directive:    KEYCOMPRESS
 
  Function:     Specifies the type of key compression to be done on
                indexed files.
 
  Syntax:
 
     ►►─┬───┬─┬─────KEYCOMPRESS──"integer"─┬────►◄
        └─/─┘ └─NO──KEYCOMPRESS────────────┘
 
  Parameters:   integer              Must be between 1 and 7.
 
  Default:      NOKEYCOMPRESS
 
  Phase:        Syntax check
 
  $SET:         Any

  Directive:    OPTIONAL-FILE
 
  Function:     Makes the compiler treat all files opened for I-O or
                EXTEND as optional.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──OPTIONAL-FILE─────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      OPTIONAL-FILE
 
  Phase:        Syntax check
 
  $SET:         Initial

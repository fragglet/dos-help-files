     This function is achieved by the run-time system forcing all ACCEPT
     FROM/DISPLAY UPON CONSOLE statements to be executed such that I/O to
     and from the screen and keyboard can be redirected  by using the ">"
     and "<" operators on the DOS or OS/2 command line.
 
     This switch has no effect on the static linked run-time system
     or on OS/2. It must be set using the COBSW environment variable;
     setting it on the command line has no effect.

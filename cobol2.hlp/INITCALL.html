  Directive:    INITCALL
 
  Function:     Specifies modules to be executed.
 
  Syntax:
 
     ►►─┬───┬┬─────INITCALL──"module"─┬────────────┬┬►◄
        └─/─┘│                        └─"priority"─┘│
             └─NO──INITCALL─────────────────────────┘
 
  Parameters:   module              The module to be called.
 
                priority            The priority to assign to the execution
                                    of the module.
 
  Default:      NOINITCALL
 
  Phase:        Syntax check
 
  $SET:         Initial

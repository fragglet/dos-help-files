  Directive:    PREPROCESS
 
  Function:     Makes the compiler take the source program from a
                preprocessor instead of a source file.
 
  Syntax:
 
     ►►─┬───┬─┬────┬─PREPROCESS──┬─"name"─┬────────────┬┬──────┬┬─►◄
        └─/─┘ │    └─P───────────┘        └─directives─┘└─ENDP─┘│
              └─NO─┬─PREPROCESS──┬──────────────────────────────┘
                   └─P───────────┘
 
  Parameters:   name                 The preprocessor to use.
 
                directives           Directives to be passed directly to
                                     the preprocessor.
 
  Default:      NOPREPROCESS
 
  Phase:        Syntax check
 
  $SET:         No

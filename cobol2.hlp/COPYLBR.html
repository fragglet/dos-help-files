────────────────────────────────────────────────────────────────────────────
 
  Directive:    COPYLBR
 
  Function:     Makes the compiler treat the library specified in a
                COPY statement as an .LBR file.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──COPYLBR───────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOCOPYLBR
 
  Phase:        Syntax check
 
  $SET:         No

  Directive:    STRUCT
 
  Function:     Makes the compiler produce extra information so that
                you can use the structure feature of Animator.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──STRUCT────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOSTRUCT
 
  Phase:        Syntax check
 
  $SET:         No

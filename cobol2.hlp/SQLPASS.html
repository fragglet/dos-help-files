  Directive:    SQLPASS
 
  Function:     Specifies the password of the database.
 
  Syntax:
 
     ►►─┬───┬─┬─────SQLPASS──"password"─┬───────►◄
        └─/─┘ └─NO──SQLPASS─────────────┘
 
  Parameters:   password             An alphanumeric string obeying the
                                     rules for a database password
 
  Default:      NOSQLPASS
 
  Phase:        Syntax check
 
  $SET:         Initial

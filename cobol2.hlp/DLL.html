  Directive:    DLL
 
  Function:     Makes the compiler include a LIBRARY INITINSTANCE
                line in the .DEF file that is produced when you specify
                the DEFFILE directive.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──DLL───────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      DLL
 
  Phase:        Generate
 
  $SET:         Initial

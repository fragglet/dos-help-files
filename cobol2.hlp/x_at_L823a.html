     See ◄Introduction to switches►
 
     Using this switch increases memory by 21 Kbyte on DOS 3.3 and by
     28 Kbyte on DOS 4.x. If you are using DOS 5.0, you should set this
     switch if you have DOS=HIGH set in your CONFIG.SYS file.
 
     This switch can be used only with the shared Run-Time System; it has
     no effect on OS/2. This switch must be set using the COBSW
     environment variable; setting it on the command line has no effect.

  Directive:    BOUND
 
  Function:     Specifies that on each table access the subscript or
                index value is to be checked to ensure it is within
                the limits defined by the OCCURS clause.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──BOUND─────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      BOUND
 
  Phase:        Both
 
  $SET:         Initial

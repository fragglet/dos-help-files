     See ◄Introduction to switches►
 
     When attributes have been written to an area on the screen with a
     DISPLAY of a Screen Section item, then a subsequent DISPLAY in that
     area with no attribute specified will inherit those attributes,
     regardless of the setting of the +S6 switch.  When attributes have
     been written to a screen area with a DISPLAY statement using a data
     item that is not in a Screen Section, then a DISPLAY statement with
     no attribute specification will by default use the attributes already
     existing in that area.  If the +S6 switch is specified at run time,
     then normal attributes will be used instead.

  Directive:    SQLINIT
 
  Function:     Makes the program initialize SQL and log on to
                the database.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──SQLINIT──"mode"───────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   mode                 Specifies whether the database is to
                                     be used in shared or exclusive mode.
 
  Default:      SQLINIT "S"
 
  Phase:        Syntax check
 
  $SET:         Initial

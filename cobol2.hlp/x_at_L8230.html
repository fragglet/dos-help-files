     See ◄Introduction to switches►
 
     Setting the K1 switch to on enables the COBOL system to be used on
     a new IBM PC keyboard in conjunction with a keyboard resident program
     which knows only about the old IBM PC keyboard BIOS calls.
 
     It must be set using the COBSW environment variable; setting it on
     the command line has no effect.

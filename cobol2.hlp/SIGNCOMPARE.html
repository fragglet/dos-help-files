  Directive:    SIGNCOMPARE
 
  Function:     Affects the way some numeric comparisons are performed
                in a program using the EBCDIC character set.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──SIGNCOMPARE───────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOSIGNCOMPARE
 
  Phase:        Generate
 
  $SET:         Any

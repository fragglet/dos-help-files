     See your Language Reference for a description of the Debug facilities.
     This switch must be set on the command line.
 
     For example, the command:
 
       @CL = B:MYPROG (+D)
 
     loads the program MYPROG from drive B with the standard ANSI COBOL
     Debug module invoked.

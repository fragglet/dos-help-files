     These directives make the compiler create files containing extra
     information about the program, for input to software used for
     debugging.

  │◄EDITOR►     Error file for Editor  │ │◄CSI►*     For CSI               │
  │◄FLAGSINEDIT►Flags in error file    │ │◄GANIM►*   For Xilerator         │
  │◄RNIM►       Animate, don't compile │ │◄STRUCT►*  For Structure Animator│
  │◄SOURCE-EXT► Source ext for CodeView│ └─────────────────────────────────┘
  │◄TRACE►      Turn on READY TRACE    │
  │◄XNIM►       Compile and animate    │

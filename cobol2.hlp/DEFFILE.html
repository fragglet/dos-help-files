  Directive:    DEFFILE
 
  Function:     Makes the compiler produce a .DEF file, which can be
                used if a .DLL file is to be created from an .OBJ file.
 
  Syntax:
 
     ►►─┬───┬─┬───────DEFFILE──"file-name"┬─────►◄
        └─/─┘ └┬────┬─DEFFILE─────────────┘
               └─NO─┘
 
  Parameters:   file-name            A full file specification.
 
  Default:      NODEFFILE
 
  Phase:        Generate
 
  $SET:         Any

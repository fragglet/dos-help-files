  Component:    Animator
 
  Function:     You use it to test, debug and investigate COBOL
                programs interactively.
 
  Syntax:
 
  ►►──ANIMATE──┬──────────┬──prog-name─┬────────────────────────────────┬─►◄
               └─switches─┘            └─dirs───────┬───────────────────┤
                                                    └─END──command-line─┘
 
  Parameters:   switches         RTS switches.
                prog-name        Program to be animated.
                dirs             ANIMATOR directives.
                command-line     Command line for the program.
 
                See also ◄RTS switches► ◄Animator directives►

  Directive:    COMP
 
  Function:     Makes the compiler produce very compact and efficient
                code for some statements involving COMP data items, by
                treating COMP items as COMP-X.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──COMP──────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOCOMP
 
  Phase:        Syntax check
 
  $SET:         Initial

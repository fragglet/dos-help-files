  Directive:    OPT
 
  Function:     Specifies the level of optimization of the code
                produced in the .OBJ file.
 
  Syntax:
 
     ►►─┬───┬──OPT──"integer"───────────────────►◄
        └─/─┘
 
  Parameters:   integer              0, 1, or 2.
 
  Default:      OPT "2"
 
  Phase:        Generate
 
  $SET:         Any

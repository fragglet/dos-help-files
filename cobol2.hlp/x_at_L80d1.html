     The possible values of severity are:
 
      S     Severe error
      E     Error
      W     Warning
      I     Informational
 
 
    If FLAGCD is specified with no parameters, the messages are assigned
    the severity given by the FLAGAS directive, if specified; if FLAGAS
    is not specified, they are produced as flagging messages.

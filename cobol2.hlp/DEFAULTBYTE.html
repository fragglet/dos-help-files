  Directive:    DEFAULTBYTE
 
  Function:     Initializes each otherwise undefined byte of the Data
                Division to the character given.
 
  Syntax:
 
     ►►─┬───┬──DEFAULTBYTE──"integer"───────────►◄
        └─/─┘
 
  Parameters:   integer              ASCII code of the character,
                                     in decimal.
 
  Default:      DEFAULTBYTE "32"
                (That is, space)
 
  Phase:        Syntax check
 
  $SET:         Initial

     The address is 4 digits long and appears on the right-hand side.
 
     This directive needs a line width setting of at least 90;
     specifiying REF sets LISTWIDTH "90".
 
     Using both source and object code listings together, you can
     identify the code generated for each line of source code. This
     directive can also be useful in determining the locations
     reported in run-time error messages.

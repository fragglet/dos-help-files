────────────────────────────────────────────────────────────────────────────
 
  Directive:    SOURCE-EXT
 
  Function:     Specifies the file-name extension of the source file
                that a run-time debugger is to look for. This information is
                included in the .OBJ file.
 
  Syntax:
 
     ►►─┬───┬──SOURCE-EXT──"ext"────────────────►◄
        └─/─┘
 
  Parameters:   ext                  A file-name extension.
 
  Default:      SOURCE-EXT "CBL"
 
  Phase:        Generate
 
  $SET:         Initial

     Animator and the cooperative program ANIMUSER communicate using
     a network name, netname. The default name is COOPANIM. If more
     than one cooperative animation is likely to take place on the
     same network at the same time, each must use a different network
     name. In this case you must specify a unique name with this
     directive when starting Animator and ANIMUSER.

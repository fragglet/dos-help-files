     See ◄Introduction to switches►
 
     The default number of heaps made available for each application is
     384. Note that system software will use some of the heaps made
     available to an application, so the number of heaps actually usable
     will be slightly less than the number specified.
 
     You may need to increase the number of heaps when animating large
     applications.
 
     This switch must be set using the COBSW environment variable; setting
     it on the command line has no effect.

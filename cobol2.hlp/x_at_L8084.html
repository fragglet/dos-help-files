     The effect is as if constant-name had been defined in the program
     in a 78-level. Its value is that given in the parentheses or
     quotation marks. If parentheses are used, the constant will be of
     category numeric; if quotation marks are used it will be of category
     alphanumeric. To declare several figurative constants using this
     feature, use the CONSTANT directive repeatedly.

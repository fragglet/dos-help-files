────────────────────────────────────────────────────────────────────────────
 
  Directive:    SQLDB
 
  Function:     Specifies the name of the database that the program
                accesses.
 
  Syntax:
 
     ►►─┬───┬─┬─────SQLDB──"database-name"─┬────►◄
        └─/─┘ └─NO──SQLDB──────────────────┘
 
  Parameters:   database-name        An alphanumeric string obeying the
                                     rules for a Database Manager database
                                     name.
 
  Default:      NOSQLDB
 
  Phase:        Syntax check
 
  $SET:         Initial

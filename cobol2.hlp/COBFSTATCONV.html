  Directive:    COBFSTATCONV
 
  Function:     Causes EXTFH to use the user-supplied module specified
                in the COBFSTATCONV environment variable to convert the
                file status codes if an I-O error is encountered on a file.
 
 
  Syntax:
 
     ►►─┬───┬─┬────┬──COBFSTATCONV──────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOCOBFSTATCONV
 
  Phase:        Syntax check
 
  $SET:         Initial

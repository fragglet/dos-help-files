────────────────────────────────────────────────────────────────────────────
 
  Directive:    TRICKLECHECK
 
  Function:     Returnes a run-time error message if trickling occurs
                in a program that you want to generate with NOTRICKLE.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──TRICKLECHECK──────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOTRICKLECHECK
 
  Phase:        Generate
 
  $SET:         Initial

     See ◄Introduction to switches►
 
     This switch is particularly useful when accessing a printer which is
     attached to a local area network or when accessing slow local
     printers which keep "timing out" because their buffers are full.
 
     This switch is available for use only with the shared Run-Time System
     on DOS. You should be aware that with the P switch set, the system
     cannot detect whether the printer is available when it is opened; the
     availability of the printer will only be detected when an attempt is
     made to write to it.

  Directive:    PRINT-EXT
 
  Function:     Specifies the extension to be added to the file-name
                associated with the ASSIGN TO PRINTER clause.
 
  Syntax:
 
     ►►─┬───┬─────────PRINT-EXT─"extension"─────►◄
        └─/─┘
 
  Parameters:   extension            The extension to be added.
 
  Default:      No extension is added
 
  Phase:        Syntax check
 
  $SET:         Any

     With NOSQLDB2, any data item that is defined outside a
     BEGIN DECLARE-SECTION and is used in an SQL statement is assumed to
     be an SQLDA. This is as in Database Manager on OS/2.
 
     With SQLDB2 set, data-items are inspected to see if they are host
     variables or an SQLDA. This provides compatibility with DB2 on IBM
     mainframes.

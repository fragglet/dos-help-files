────────────────────────────────────────────────────────────────────────────
 
  Directive:    FCDREG
 
  Function:     Makes the compiler define special-registers giving
                access to files' File Control Descriptions (FCD) and
                Key Definition Blocks.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──FCDREG────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOFCDREG
 
  Phase:        Syntax check
 
  $SET:         Initial

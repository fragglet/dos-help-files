     You can use Animator directives with XNIM.
 
     With XNIM, no object file is produced by default.
 
     When it finishes, Animator will not save the files required for a
     subsequent animation.
 
     If the source file contains more than one non-nested program, it
     is the last program in the source file that will be animated.

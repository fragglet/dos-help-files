     This directive does not appear in the SETTING list if its state
     is the same as WRITELOCK. In this case, the state of the two
     directives is indicated by the FILESHARE directive.
 
     This directive is included for compatibility with earlier
     file-sharing products. When writing new programs you should
     use the locking syntax rather than this directive.

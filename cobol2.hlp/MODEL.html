  Directive:    MODEL
 
  Function:     Specifies the memory model.
 
  Syntax:
 
     ►►─┬───┬──MODEL──"type"────────────────────►◄
        └─/─┘
 
  Parameters:   type                 SMALL, COMPACT, MEDIUM, LARGE,
                                     or HUGE.
 
  Default:      MODEL "HUGE"       (for programs with more than 64K of data)
                MODEL "LARGE"      (for programs with less than 64K of data)
 
  Phase:        Generate
 
  $SET:         Initial

────────────────────────────────────────────────────────────────────────────
 
  Directive:    FLASH
 
  Function:     Sets the default condition under which Animator
                will automatically display the user screen.
 
  Syntax:
 
     ►►─┬───┬──FLASH──"mode"────────────────────►◄
        └─/─┘
 
  Parameters:   mode                 The mode in which Animator will
                                     display the user screen.
 
  Default:      FLASH "ZOOM"

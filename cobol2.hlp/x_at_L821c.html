     See ◄Introduction to switches►
 
     This switch is available for use only with the shared Run-Time
     System, which by default treats all screens as 25 lines long. The
     static linked system always handles screens of any length.
 
     The screen width cannot be altered; only screens 80 columns
     wide are supported.
 
     It must be set using the COBSW environment variable; setting it on
     the command line has no effect.

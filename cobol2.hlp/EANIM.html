  Directive:    EANIM
 
  Function:     Makes the compiler put information on line numbers and
                symbols into the .OBJ file for use by a run-time
                debugger such as Microsoft CodeView.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──EANIM─────────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOEANIM
 
  Phase:        Both
 
  $SET:         Any

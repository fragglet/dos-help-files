  Directive:    LITVAL-SIZE
 
  Function:     Specifies the number of bytes to pass if the SIZE
                clause is omitted when passing numeric literals BY
                VALUE.
 
  Syntax:
 
     ►►─┬───┬──LITVAL-SIZE──"integer"───────────►◄
        └─/─┘
 
  Parameters:   integer              A number up to 4. The number of
                                     bytes to pass.
 
  Default:      LITVAL-SIZE "4"
 
  Phase:        Syntax check
 
  $SET:         Any

────────────────────────────────────────────────────────────────────────────
 
  Directive:    INFORETURN
 
  Function:     Specifies the return-code value returned by the compiler
                when it produces only informational messages.
 
  Syntax:
 
     ►►─┬───┬──INFORETURN──"integer"────────────►◄
        └─/─┘
 
  Parameters:   integer              Must be between 0 and 4.
 
  Default:      INFORETURN "0"
 
  Phase:        Syntax check
 
  $SET:         Any

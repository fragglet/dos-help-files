  Directive:    REALOVL
 
  Function:     Causes overlays to be regarded as real rather than
                pseudo so that overlaying is done from disk.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──REALOVL───────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      REALOVL
 
  Phase:        Generate
 
  $SET:         Initial

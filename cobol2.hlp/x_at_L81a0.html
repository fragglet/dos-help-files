     The possible values of dbman are:
 
      IBMDBM    IBM OS/2 Extended Edition Database Manager
      IBMDBR    IBM OS/2 Extended Edition DOS Database Requester
      IBMES1DBM IBM Extended Services/2 V1.0 Database Manager
      IBMES1DBR IBM Extended Services/2 V1.0 Database Requester
      MSSQL     Microsoft SQL Server
      MSDR      Microsoft Server client running under DOS
 
      If no parameter is supplied, IBMDBM is assumed.

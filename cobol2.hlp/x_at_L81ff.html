     The possible values of mode are:
 
       step    Display user screen before executing each statement.
 
       zoom    Display user screen before zooming.
 
     This directive is reserved for use with add-on products supplied
     by Micro Focus. Do not change its setting unless you have an
     appropriate add-on product.
 
     The setting can be altered from within Animator.
 
     Using this you can make Animator display the user screen before it
     calls a subprogram written in another language. This will prevent
     the subprogram's output overwriting the Animator screen.

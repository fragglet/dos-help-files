  Directive:    COBIDY
 
  Function:     Specifies the path where the Animator information file
                (.IDY file) is to be written.
 
  Syntax:
 
     ►►─┬───┬─┬───────COBIDY──"path-name"──┬────►◄
        └─/─┘ └┬────┬─COBIDY───────────────┘
               └─NO─┘
 
  Parameters:   path-name            Path specification.
 
  Default:      NOCOBIDY
 
  Phase:        Syntax check
 
  $SET:         No

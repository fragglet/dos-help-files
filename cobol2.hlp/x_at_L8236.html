     Instead of the interactive pop-up window, a general OS/2 error
     message is displayed and control is returned to the operating
     system prompt without any user intervention.
 
     This switch has no effect on DOS. On OS/2 It must be set using
     the COBSW environment variable; setting it on the command line
     has no effect.

  Directive:    LIST
 
  Function:     Specifies the destination of the source listing file.
 
  Syntax:
 
     ►►─┬───┬─┬───────LIST─┬─"destination"─┬┬───►◄
        └─/─┘ │            └─()────────────┘│
              └┬────┬─LIST──────────────────┘
               └─NO─┘
 
  Parameters:   destination          A full file specification or
                                     a device-name.
 
  Default:      NOLIST
 
  Phase:        Both
 
  $SET:         Any                  for LIST and NOLIST
                None                 for LIST"file-name" and LIST""

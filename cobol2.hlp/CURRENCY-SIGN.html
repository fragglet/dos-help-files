  Directive:    CURRENCY-SIGN
 
  Function:     Specifies the currency sign to be recognized in
                the PICTURE clause.
 
  Syntax:
 
     ►►─┬───┬──CURRENCY-SIGN──"integer"─────────►◄
        └─/─┘
 
  Parameters:   integer              ASCII code of the character,
                                     in decimal.
 
  Default:      CURRENCY-SIGN "36"
                (that is, "$")
 
  Phase:        Syntax check
 
  $SET:         Initial

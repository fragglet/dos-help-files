HOLDVIEW Details (↑ Animator Directives)

────────────────────────────────────────────────────────────────────────────
 
     HOLDVIEW with no parameter specifies an infinite time. In this
     case, a keystroke will return control to the Animator. If
     specified, the parameter indicates the number of seconds
     before control is automatically returned to Animator.
 
     This is necessary when the user program is running in a different
     session, as Animator no longer has control and using a keystroke
     to terminate it will affect the keyboard buffer of the user session.

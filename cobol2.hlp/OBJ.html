────────────────────────────────────────────────────────────────────────────
 
  Directive:    OBJ
 
  Function:     Specifies the name of the object code file.
 
  Syntax:
 
     ►►─┬───┬─┬─────OBJ──"file-name"┬───────────►◄
        └─/─┘ └─NO──OBJ─────────────┘
 
  Parameters:   file-name            A full file specification.
 
  Default:      OBJ "source-name.OBJ"   (with OMF "OBJ")
                OBJ "source-name.GNT"   (with OMF "GNT")
 
  Phase:        Generate
 
  $SET:         Any

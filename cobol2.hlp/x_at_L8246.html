     See ◄Introduction to switches►
 
     The system will first look for a security key in the port specified
     by this switch, the default being the standard IBM port address. If a
     security key is not found at this address, adresses 632 and 888 will
     be checked.
 
     You may need this machine if you are using a clone machine that has
     non-standard parallel port addresses.

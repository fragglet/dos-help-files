  Directive:    SEGSIZE
 
  Function:     Specifies the maximum code segment size, beyond which
                the compiler automatically segments the code.
 
  Syntax:
 
     ►►─┬───┬──SEGSIZE──"integer"───────────────►◄
        └─/─┘
 
  Parameters:   integer              The maximum size.
 
  Default:      SEGSIZE "0"
 
  Phase:        Generate
 
  $SET:         Initial

DATACOMPRESS (↑ Format of Your Data Files)

────────────────────────────────────────────────────────────────────────────
 
  Directive:    DATACOMPRESS
 
  Function:     Specifies the type of data compression to be done on
                sequential and indexed files.
 
  Syntax:
 
     ►►─┬───┬─┬─────DATACOMPRESS──"integer"─┬───►◄
        └─/─┘ └─NO──DATACOMPRESS────────────┘
 
  Parameters:   integer              Must be 1. Specifies run-length
                                     encoding.
 
  Default:      NODATACOMPRESS
 
  Phase:        Syntax check
 
  $SET:         Any

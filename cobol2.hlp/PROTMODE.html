  Directive:    PROTMODE
 
  Function:     Causes the compiler to optimize certain statemets in
                such a way that the resulting object code can only be run
                in protect mode environments.
 
  Syntax:
 
     ►►─┬───┬─┬────┬──PROTMODE──────────────────►◄
        └─/─┘ └─NO─┘
 
  Parameters:   None
 
  Default:      NOPROTMODE
 
  Phase:        Generate
 
  $SET:         Initial

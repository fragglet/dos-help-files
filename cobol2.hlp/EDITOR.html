────────────────────────────────────────────────────────────────────────────
 
  Directive:    EDITOR
 
  Function:     Makes the compiler send error messages to a file in a
                format compatible with a specified Editor.
 
  Syntax:
 
     ►►─┬───┬─┬─────EDITOR──"editor-id"─┬───────►◄
        └─/─┘ └─NO──EDITOR──────────────┘
 
  Parameters:   editor-id            Must be MF or MS.
 
  Default:      NOEDITOR
 
  Phase:        Both
 
  $SET:         No

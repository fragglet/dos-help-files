     Specifying PARAS causes the generated code addresses of all paragraphs
     and sections to be produced. This list is placed in the listing file
     specified by the ASMLIST directive. The assembler listing that is
     normally in this file is suppressed if PARAS is specified.
 
     The ANIM directive must also be specified for the PARAS directive to
     work.

  Directive:    CALLSORT
 
  Function:     Defines the program to be called to handle all SORT
                and MERGE operations.
 
  Syntax:
 
     ►►─┬───┬───CALLSORT──"sort-name"───────────►◄
        └─/─┘
 
  Parameters:   sort-name            Root-name of a program to be
                                     called to process sorts and merges.
 
  Default:      CALLSORT "_SORT"
 
  Phase:        Syntax check
 
  $SET:         Initial

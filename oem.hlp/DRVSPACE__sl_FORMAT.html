                              DRVSPACE /FORMAT
 
Formats the selected compressed drive.
 
CAUTION:  Formatting a compressed drive deletes all the files it contains.
          You cannot unformat a drive that has been formatted by using
          DRVSPACE /FORMAT.
 
Syntax
 
    DRVSPACE /FORMAT drive:
 
Parameter
 
drive:
    Specifies the drive you want to format. (DriveSpace will not allow you
    to format drive C.)
 
Switch
 
/FORMAT
    Directs DriveSpace to format the specified compressed drive. This switch
    can be abbreviated as /F.

                               DEL──Examples
 
To delete the CAT.TMP file from the TEST directory on drive C, you can use
either of the following commands:
 
    del c:\test\cat.tmp
 
    erase c:\test\cat.tmp
 
To delete all the files in a directory named TEST on drive C, you can use
either of the following commands:
 
    del c:\test
 
    del c:\test\*.*

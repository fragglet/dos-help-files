──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  PushCS
 
  /* Push contents of CS code segment register onto stack. */
 
  Description
 
  Constructs far pointers to locations in the current code segement.
 
  Pseudocode equivalent:
 
  PushW(CS);

──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  Int3
 
  /* Software interrupt for the p-code machine */
 
  Description
 
  Acts as the software interrupt for the p-code machine, and is reserved
  for use by the debugger. It is always assigned the opcode of 0xCC.

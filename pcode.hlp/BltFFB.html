  Syntax
 
  BltFFB
 
  /* Block copy using far pointers. */
 
  Description
 
  Performs a byte block copy using far pointers and a count found on the
  top of the evaluation stack.
 
  Pseudocode equivalent:
 
  cb = PopW();
  lpbDest = PopL();
  lpbSrc = PopL();
  copy cb bytes from lpbSrc to lpbDest

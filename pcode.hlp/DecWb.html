──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  DecW<n>
 
  /* Decrement top word on stack */
 
  Possible Instructions
 
  DecW1, DecWb
 
  Description
 
  This instruction decrements the top of stack element by <n> and pushes
  the new value onto the stack.
 
  Pseudocode equivalent:
 
  w1 = Pop();
  Push(w1 - <n>);

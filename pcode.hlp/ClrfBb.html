──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  ClrfB<n>
 
  /* Clear Frame Byte */
 
  Possible Instructions
 
  ClrfBb
 
  Description
 
  Clears (sets to zero) a byte location on the frame.
 
  Pseudocode equivalent:
 
  [BP+(0xff00 & <n>)] = 0;
 
  The following statement generates this instruction:
 
  bFlag = 0;  /* generates ClrfB<n> */

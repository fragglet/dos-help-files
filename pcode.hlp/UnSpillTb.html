──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  UnSpillTb
 
  /* UnSpill floating point stack. */
 
  Description
 
  Reverses the operation of the SpillTb instruction by rereading the
  previous contents of the floating-point stack from memory.

  Syntax
 
  PopR
 
  /* Move floating point value from evaluation stack to floating point */
  /* stack                                                             */
 
  Description
 
  Moves a floating point value onto the floating point stack.
 
  Pseudocode equivalent:
 
  float f;
  f = PopR();
  FpPushT((temp-real) f);

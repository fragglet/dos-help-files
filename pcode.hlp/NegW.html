──P-Code Instructions───────────────────────────────────────────────────────
 
  Syntax
 
  NegW
 
  /* Negate word (2's complement) */
 
  Description
 
  Negates the top stack element and pushes it back onto the stack.
 
  w1 = popW();
  pushW(-w1);
 
  A simple expression such as
 
  if (-wt1 < 0)
 
  generates the NegW instruction.

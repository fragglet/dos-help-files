V86MMGR_Free_Page_Map_Region
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include v86mmgr.inc
 
VxDcall V86MMGR_Free_Page_Map_Region
 
The V86MMGR_Free_Page_Map_Region service unmap pages that were mapped by the
V86MMGR_Map_Pages service.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
ESI        Specifies the map handle to free.
 
Return Value
 
This service has no return value.
 
Comments
 
After a call to this service, the old map buffer address contains null
memory is invalid and the ESI register is undefined.
 
Uses
 
ESI, Flags

Debug_Test_Valid_Handle
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vmm.inc
 
mov     ebx, VM       ; VM handle to validate
VMMcall Debug_Test_Valid_Handle
 
The Debug_Test_Valid_Handle service determines whether the specified virtual
machine handle is valid.
 
This service is only available in the debugging version of Windows.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
VM         Specifies the virtual machine handle to check.
 
Return Value
 
This service has no return value.
 
Comments
 
This service does not return is the handle is not valid.
 
Virtual devices usually call this service by using the Assert_VM_Handle
macro.

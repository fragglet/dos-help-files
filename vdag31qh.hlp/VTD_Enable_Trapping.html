VTD_Enable_Trapping
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vtd.inc
 
VxDcall VTD_Enable_Trapping
 
The VTD_Enable_Trapping service must be called to re-enable timer I/O port
trapping after calling the VTD_Disable_Trapping service. This service must
be made once for every call to VTD_Disable_Trapping. Only when every disable
call has been matched by a call to this service will port trapping be
re-enabled.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
EBX        Specifies a virtual machine handle.
 
Return Value
 
This service has no return value.

include shell.inc
 
VxDcall SHELL_Resolve_Contention
 
The SHELL_Resolve_Contention service resolves contention for the MS-DOS
shell.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
EAX        Specifies the virtual machine handle of the current device
           owner.
 
EBX        Specifies the handle of the contending virtual machine. This
           parameter must identify the current virtual machine.
 
ESI        Points to an 8-byte string identifying the name of the device in
           contention. The name must be in uppercase letters and be padded
           with spaces if necessary.
 
Return Value
 
The EBX register contains the virtual machine handle of contention winner.
If the carry flag is set, contention could not be resolved.
 
Uses
 
EBX, Flags

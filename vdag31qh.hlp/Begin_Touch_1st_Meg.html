Begin_Touch_1st_Meg
 
The Begin_Touch_1st_Meg macro enables the first 1 megabyte of memory for the
current virtual machine. Virtual devices use this macro to examine and
modify addresses in the first megabyte. This macro is intended to be used
for debugging.
 
This macro has no parameters.
 
Return Value
 
This macro has no return value.
 
Comments
 
This macro does not generate code unless the DEBUG symbol is defined. If the
macro generates code, the code calls the Enable_Touch_1st_Meg service.
 
See Also
 
Enable_Touch_1st_Meg, End_Touch_1st_Meg

Clear_Mono_Screen
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vmm.inc
 
VMMcall Clear_Mono_Screen
 
The Clear_Mono_Screen service clears the secondary display screen by filling
it with spaces, and setting character attributes to normal.
 
This service has no parameters.
 
Return Value
 
This service has no return value.
 
Comments
 
This service has no effect in the retail version of Windows. It is intended
to be used with the debugging version.

VDD_PIF_State
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vdd.inc
 
VxDcall VDD_PIF_State
 
The VDD_PIF_State service informs the VDD about PIF bits for virtual machine
just created.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
EBX        Specifies the virtual machine handle.
 
AX         Specifies the PIF bits.
 
Return Value
 
This service has no return value.

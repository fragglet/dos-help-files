include vtd.inc
 
VxDcall VTD_Get_Interrupt_Period
 
The VTD_Get_Interrupt_Period service returns the current timer interrupt
frequency.
 
This service has no parameters.
 
Return Value
 
The EAX register specifies the length of time between ticks in
milliseconds.

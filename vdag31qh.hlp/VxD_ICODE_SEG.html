VxD_ICODE_SEG
 
The VxD_ICODE_SEG macro defines the start of an initialization code segment.
Virtual devices typically use this segment for code that initializes the
corresponding device. The system discards the segment after the
initialization is complete (after the Init_Complete message has been
processed by all virtual devices).
 
Comments
 
This macro creates a 32-bit segment named _ITEXT. The segment is assembled
for flat model memory so segment registers CS, DS, ES, and SS are assumed to
be equal.

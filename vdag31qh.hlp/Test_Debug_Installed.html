Test_Debug_Installed
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vmm.inc
 
VMMcall Test_Debug_Installed
 
je      not_installed       ; zero flag set if not installed
 
The Test_Debug_Installed service tests whether whether the debugging version
of the Windows VMM is running.
 
This service has no parameters.
 
Return Value
 
The zero flag is set if the debugging version is not installed.

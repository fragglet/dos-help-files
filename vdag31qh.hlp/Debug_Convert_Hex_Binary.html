Debug_Convert_Hex_Binary
                                                     ◄Up► ◄Next► ◄Previous►

include vmm.inc
 
mov     al, Number              ; number to convert
VMMcall Debug_Convert_Hex_Binary
 
mov     [BinNum], eax           ; binary number
 
The Debug_Convert_Hex_Binary service converts an 8-bit value to a 32-bit
value representing a binary number. The new value actually consists of eight
4-bit values with each value being either 0 or 1.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
Number     Specifies the number to convert.
 
Return Value
 
The EAX register contains the converted number.
 
Comments
 
This service is typically used in conjunction with the Trace_Out macro to
display values in binary notation.
 
Uses
 
EAX
 
See Also
 
Debug_Convert_Hex_Decimal, Trace_Out

_GetDemandPageInfo
                                                     ◄Up► ◄Next► ◄Previous►
────────────────────────────────────────────────────────────────────────────
 
include vmm.inc
 
VMMcall _GetDemandPageInfo, <<OFFSET32 DemandInfo>, flags>
 
The _GetDemandPageInfo service retrieves information used for demand paging,
copying the information to the specified structure. This service is for
exclusive use by the virtual paging device.
 
Parameter   Description
────────────────────────────────────────────────────────────────────────────
 
DemandInfo  Points to a DemandInfoStruc structure containing information for
            demand paging.
 
flags       Specifies the operation flags. This parameter must be set to 0.
 
Return Value
 
This service has no return value.

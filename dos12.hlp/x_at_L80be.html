MS OS/2 version 1.2 changes
 
DosEnterCritSec now returns zero if the function is successful. Otherwise,
it returns an error value. It did not return a value in earlier versions.
 
For MS OS/2 version 1.2, a count is maintained of the number of times
DosEnterCritSec is called. Normal thread execution is not restored until an
equal number of calls are made to DosExitCritSec.

#define INCL_DOSDEVICES
 
USHORT DosR2StackRealloc(usSize)
USHORT usSize;    /* new size for stack */
 
The DosR2StackRealloc function changes the size of a thread's ring-2 stack.
The function reallocates the stack as requested.
 
This function cannot be used from ring 2.
 
Parameter  Description
────────────────────────────────────────────────────────────────────────────
 
usSize     Specifies the size (in bytes) of the ring-2 stack. If the
           specified size is smaller then the current stack size, it will be
           ignored.
 
Return Value
 
The return value is zero if the function is successful. Otherwise, it is an
error value.

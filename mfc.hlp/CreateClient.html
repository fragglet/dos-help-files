  virtual BOOL CreateClient( LPCREATESTRUCT lpCreateStruct,
                             CMenu* pWindowMenu );
 
  Parameter        Description
 
  <lpCreateStruct> Pointer to a CREATESTRUCT structure.
 
  <pWindowMenu>    Pointer to the window pop-up menu.
 
  Remarks
 
  Creates the MDI client window that manages the CMDIChildWnds and fills
  out a CLIENTCREATESTRUCT.
 
  CreateClient should be called if you override the OnCreate member
  function.
 
  Return Value
 
  TRUE if successful; otherwise FALSE.

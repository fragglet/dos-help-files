  BOOL PlayMetaFile( HANDLE hMF );
 
  Parameter   Description
 
  <hMF>       Identifies the metafile.
 
  Remarks
 
  Plays the contents of the specified metafile on the given device. The
  metafile can be played any number of times.
 
  Return Value
 
  TRUE if the function is successful; otherwise FALSE.

  Windows 3.1 only
 
  HWND GetOpenClipboardWindow();
 
  Remarks
 
  Retrieves the handle of the window that currently has the Clipboard
  open.
 
  Return Value
 
  The handle of the window that currently has the Clipboard open if the
  function is successful; otherwise NULL.

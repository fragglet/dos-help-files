  virtual void Flush()
  throw( CFileException );
 
  Remarks
 
  Forces any data remaining in the file buffer to be written to the file.
 
  The use of Flush does not guarantee flushing of CArchive buffers. If you
  are using an archive, call CArchive::Flush first.

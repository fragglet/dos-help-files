  DWORD GetNearestColor( DWORD crColor ) const;
 
  Parameter   Description
 
  <crColor>   Specifies the color to be matched.
 
  Remarks
 
  Returns the closest logical color to a specified logical color that the
  given device can represent.
 
  Return Value
 
  An RGB color value that names the solid color closest to the <crColor>
  value that the device can represent.

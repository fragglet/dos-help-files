  afx_msg void OnDestroyClipboard();
 
  Remarks
 
  Called for the Clipboard owner when the Clipboard is emptied through a
  call to the ◄EmptyClipboard► Windows function.
 
  This message-handler member function calls the ◄Default► member
  function. Override this member function in your derived class to handle
  the ◄WM_DESTROYCLIPBOARD► message.
 
  See Also
 
  ◄::EmptyClipboard►, ◄CWnd::Default►, ◄WM_DESTROYCLIPBOARD►

  BOOL SelectStockObject( int nIndex );
 
  Parameter   Description
 
  <nIndex>    Specifies the type of stock object desired. See
              ◄CDC::SelectObject► for a list of the possible stock
              objects.
 
  Remarks
 
  Selects one of the predefined stock pens, brushes, or fonts into the
  CMetaFileDC.
 
  Return Value
 
  TRUE if the function is successful; otherwise FALSE.

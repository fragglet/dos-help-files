  void SetDlgItemText( int nID, const char FAR* lpString );
 
  Parameter   Description
 
  <nID>       Specifies the integer ID of the control whose text is to be
              set.
 
  <lpString>  Points to a CString or null-terminated string that is to be
              copied to the control.
 
  Remarks
 
  Sets the caption or text of a control owned by CWnd.
 
  See Also
 
  ◄::SetDlgItemText►, ◄WM_SETTEXT►, ◄CWnd::GetDlgItemText►

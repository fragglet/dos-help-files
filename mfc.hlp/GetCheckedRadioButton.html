  int GetCheckedRadioButton( int nIDFirstButton, int nIDLastButton );
 
  Parameter        Description
 
  <nIDFirstButton> Specifies the integer identifier of the first radio
                   button in the group.
 
  <nIDLastButton>  Specifies the integer identifier of the last radio
                   button in the group.
 
  Remarks
 
  This function retrieves the ID of the currently checked radio button in
  the specified group.
 
  Return Value
 
  ID of the checked radio button.

CListBox::AddString

  int AddString( const char FAR* lpItem );
 
  Parameter   Description
 
  <lpItem>    Points to the null-terminated string that is to be added.
 
  Remarks
 
  Adds a string to a list box. If the list box was not created with the
  ◄LBS_SORT► style, the string is added to the end of the list. Otherwise,
  the string is inserted into the list, and the list is sorted.
 
  Use ◄InsertString► to insert a string into a specific location within
  the list box.
 
  Return Value
 
  The zero-based index to the string in the list box. The return value is
  LB_ERR if an error occurs; the return value is LB_ERRSPACE if
  insufficient space is available to store the new string.

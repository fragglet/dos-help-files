  Protected: LONG Default();
 
  Remarks
 
  Calls the default window procedure. The default window procedure
  provides default processing for any window message that an application
  does not process. This member function is used to ensure that every
  message is processed. All CWnd On<Message> member functions call this
  member function.
 
  Return Value
 
  Depends on the message that was passed to this function.

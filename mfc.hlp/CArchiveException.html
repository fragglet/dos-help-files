  Description
 
  A CArchiveException object represents a serialization exception
  condition. The CArchiveException class includes a public data member
  that indicates the cause of the exception.
 
  #include <afx.h>
 
  See Also
 
  ◄CArchive►, ◄AfxThrowArchiveException►, Chapter 5,
  ◄Exception Processing►.
 
  Comments
 
  CArchiveException objects are constructed and thrown inside CArchive
  member functions. You can access these objects within the scope of a
  CATCH expression. The cause code is independent of the operating
  system.

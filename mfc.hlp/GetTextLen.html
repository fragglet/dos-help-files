  int GetTextLen( int nIndex ) const;
 
  Parameter   Description
 
  <nIndex>    Specifies the zero-based index of the string.
 
  Remarks
 
  Gets the length of a string in a list box item.
 
  Return Value
 
  The length of the string in bytes, excluding the terminating null
  character. If <nIndex> does not specify a valid index, the return value
  is LB_ERR.

  int GetSel( int nIndex ) const;
 
  Parameter   Description
 
  <nIndex>    Specifies the zero-based index of the item.
 
  Remarks
 
  Retrieves the selection state of an item. This member function works
  with both single and multiple-selection list boxes.
 
  Return Value
 
  A positive number if the specified item is selected; otherwise, it is 0.
  The return value is LB_ERR if an error occurs.

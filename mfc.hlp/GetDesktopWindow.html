  static CWnd* GetDesktopWindow();
 
  Remarks
 
  Returns the Windows desktop window. The desktop window covers the entire
  screen and is the area on top of which all icons and other windows are
  painted.
 
  Return Value
 
  Identifies the Windows desktop window. This pointer may be temporary,
  and should not be stored for later use.

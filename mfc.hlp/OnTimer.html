  afx_msg void OnTimer( UINT  nIDEvent );
 
  Parameter   Description
 
  <nIDEvent>  Specifies the identifier of the timer.
 
  Remarks
 
  Called after each interval specified in the ◄SetTimer► member function
  used to install a timer.
 
  This message-handler member function calls the ◄Default► member
  function. Override this member function in your derived class to handle
  the ◄WM_TIMER► message.
 
  See Also
 
  ◄CWnd::SetTimer►, ◄WM_TIMER►, ◄CWnd::Default►

  BOOL PASCAL AfxEnableMemoryTracking( BOOL bTrack );
 
  Parameter   Description
 
  <bTrack>    TRUE turns on memory tracking; FALSE turns it off.
 
  Remarks
 
  Diagnostic memory tracking is normally enabled in the Debug version of
  the Microsoft Foundation classes. Use this function to disable tracking
  on sections of your code that you know are allocating blocks correctly.
 
  Return Value
 
  The previous setting of the tracking-enable flag.

  void GetScrollRange( LPINT lpMinPos, LPINT lpMaxPos ) const;
 
  Parameter   Description
 
  <lpMinPos>  Points to the integer variable that is to receive the
              minimum position.
 
  <lpMaxPos>  Points to the integer variable that is to receive the
              maximum position.
 
  Remarks
 
  Copies the current minimum and maximum scroll-bar positions for the
  given scroll bar to the locations specified by <lpMinPos> and
  <lpMaxPos>.
 
  The default range for a scroll-bar control is empty (both values are
  0).

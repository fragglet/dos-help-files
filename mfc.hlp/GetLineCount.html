  int GetLineCount() const;
 
  Remarks
 
  Retrieves the number of lines in a multiple-line edit control.
 
  GetLineCount is only processed by multiple-line edit controls.
 
  Return Value
 
  An integer containing the number of lines in the multiple-line edit
  control. If no text has been entered into the edit control, the return
  value is 1.

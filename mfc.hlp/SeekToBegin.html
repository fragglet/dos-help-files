  void SeekToBegin()
  throw( CFileException );
 
  Remarks
 
  Sets the value of the file pointer to the beginning of the file.
  SeekToBegin() is equivalent to Seek(0L, CFile::begin).
 
  Example
 
  extern CFile cfile;
  cfile.SeekToBegin();

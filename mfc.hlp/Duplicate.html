  virtual CFile* Duplicate() const
  throw( CFileException );
 
  Remarks
 
  Constructs a duplicate CFile object for a given file. This is equivalent
  to the C run-time function dup.
 
  Example
 
  extern CFile* cfile1;
  CFile* cfile2 = cfile1->Duplicate();

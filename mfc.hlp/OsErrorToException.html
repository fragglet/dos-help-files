  static int OsErrorToException( LONG lOsError );
 
  Parameter   Description
 
  <lOsError>  An operating-system-specific error code.
 
  Remarks
 
  This static function returns an enumerator that corresponds to a given
  <lOsError> value. If the error code is unknown, then the function
  returns CFileException::generic.
 
  Return Value
 
  Enumerated value that corresponds to a given operating-system error
  value.
 
  Example
 
  ASSERT( CFileException::OsErrorToException( 5 ) ==
                    CFileException::accessDenied );

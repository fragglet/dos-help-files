  Windows 3.1 Only
 
  int GetFirstVisibleLine() const;
 
  Remarks
 
  An application calls GetFirstVisibleLine to determine the topmost
  visible line in an edit control.
 
  Return Value
 
  The zero-based index of the topmost visible line. For single-line edit
  controls, the return value is 0.

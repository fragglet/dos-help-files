  CWnd* GetParent() const;
 
  Remarks
 
  Retrieves the parent window (if any).
 
  Return Value
 
  Identifies the parent window. The value is NULL if the CWnd has no
  parent window.
 
  The returned pointer may be temporary, and should not be stored for
  later use.

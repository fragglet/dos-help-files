  HMENU Detach();
 
  Remarks
 
  Detaches a Windows menu from a CMenu object and returns the handle. The
  m_hMenu data member is set to NULL.
 
  Return Value
 
  The handle, of type HMENU, to a Windows menu, if successful; otherwise
  NULL.

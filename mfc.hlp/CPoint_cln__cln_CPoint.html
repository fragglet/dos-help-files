  CPoint();
  CPoint( int initX, int initY );
  CPoint( POINT initPt );
  CPoint( SIZE initSize );
 
  Parameter   Description
 
  <initX>     Sets the x member for the CPoint.
 
  <initY>     Sets the y member for the CPoint.
 
  <initPt>    Windows POINT structure used to initialize CPoint.
 
  <initSize>  Sets the x and y member equal to the corresponding values in
              <cx> and <cy> values in <initSize>.
 
  Remarks
 
  Constructs a CPoint object. If no arguments are given, x and y members
  are not initialized.

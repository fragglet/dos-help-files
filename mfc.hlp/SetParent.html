  CWnd* SetParent( CWnd* pWndNewParent );
 
  Parameter       Description
 
  <pWndNewParent> Identifies the new parent window.
 
  Remarks
 
  Changes the parent window of a child window.
 
  If the CWnd is visible, Windows performs the appropriate redrawing and
  repainting.
 
  Return Value
 
  A pointer to the previous parent window object. The returned pointer may
  be temporary.

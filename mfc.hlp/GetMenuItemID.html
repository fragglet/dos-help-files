  UINT GetMenuItemID( int nPos ) const;
 
  Parameter   Description
 
  <nPos>      Specifies the position (zero-based) of the menu item whose
              ID is being retrieved.
 
  Remarks
 
  Obtains the menu-item identifier for a menu item located at the position
  defined by <nPos>.
 
  Return Value
 
  The item ID for the specified item in a pop-up menu if the function is
  successful. If the specified item is a pop-up menu (as opposed to an
  item within the pop-up menu), the return value is -1. If <nPos>
  corresponds to a SEPARATOR menu item, the return value is 0.

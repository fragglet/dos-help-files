  CSize();
  CSize( int initCX, int initCY );
  CSize( SIZE initSize );
  CSize( POINT initPt );
 
  Parameter   Description
 
  <initCX>    Sets the cx member for the CSize.
 
  <initCY>    Sets the cy member for the CSize.
 
  <initSize>  Windows SIZE structure used to initialize CSize.
 
  <initPt>    Windows POINT structure used to initialize CSize.
 
  Remarks
 
  Constructs a CSize object. If no arguments are given, cx and cy members
  are not initialized.

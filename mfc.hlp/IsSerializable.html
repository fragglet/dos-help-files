  BOOL IsSerializable() const;
 
  Remarks
 
  IsSerializable tests whether this object is eligible for serialization.
  For a class to be serializable, its declaration must contain the
  DECLARE_SERIAL macro, and the implementation must contain the
  IMPLEMENT_SERIAL macro.
 
  NOTE: Do not override this function.
 
  Return Value
 
  TRUE if this object can be serialized; otherwise FALSE.
 
  Example
 
  CAge a(21);
  ASSERT( a.IsSerializable() );

  Windows 3.1 only
 
  BOOL QueryAbort() const;
 
  Remarks
 
  Calls the abort function installed by the ◄SetAbortProc► member function
  for a printing application and queries whether the printing should be
  terminated.
 
  Return Value
 
  The return value is TRUE if printing should continue or if there is no
  abort procedure. It is FALSE if the print job should be terminated. The
  return value is supplied by the abort function.

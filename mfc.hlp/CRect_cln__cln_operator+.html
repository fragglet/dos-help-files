  CRect operator +( POINT point ) const;
 
  Parameter   Description
 
  <point>     Contains a POINT or CPoint.
 
  Remarks
 
  Returns a new CRect that is equal to CRect displaced by <point>. Moves
  the CRect <point.x> units along the x-axis and <point.y> units along the
  y-axis. The <x> and <y> parameters are added to CRect's position.
 
  Return Value
 
  The CRect resulting from the offset by <point>.

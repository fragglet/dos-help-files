  BOOL PaintRgn( CRgn* pRgn );
 
  Parameter   Description
 
  <pRgn>      Identifies the region to be filled. The coordinates for the
              given region are specified in device units.
 
  Remarks
 
  Fills the region specified by <pRgn> with the selected brush.
 
  Return Value
 
  TRUE if the function is successful; otherwise FALSE.

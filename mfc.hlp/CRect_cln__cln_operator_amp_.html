  CRect operator &( const RECT& rect2 ) const;
 
  Parameter   Description
 
  <rect2>     Contains a RECT or CRect.
 
  Return Value
 
  A CRect that is the intersection of CRect and <rect2>. The intersection
  is the largest rectangle contained in both rectangles.

  BOOL RectInRegion( LPRECT lpRect ) const;
 
  Parameter   Description
 
  <lpRect>    Points to a RECT structure or CRect object.
 
  Remarks
 
  Determines whether any part of the rectangle specified by <lpRect> is
  within the boundaries of the region stored in the CRgn object.
 
  Return Value
 
  TRUE if any part of the specified rectangle lies within the boundaries
  of the region; otherwise FALSE.

  HANDLE Detach();
 
  Remarks
 
  Detaches a Windows GDI object from a CGdiObject object and returns a
  handle to the Windows GDI object.
 
  Return Value
 
  A HANDLE to the Windows GDI object detached, or NULL if no GDI object is
  attached.

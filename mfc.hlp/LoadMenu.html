  BOOL LoadMenu( const char FAR* lpMenuName );
  BOOL LoadMenu( UINT nIDMenu );
 
  Parameter    Description
 
  <lpMenuName> Points to a null-terminated string that contains the name
               of the menu resource to load.
 
  <nIDMenu>    Specifies the menu ID of the menu resource to load.
 
  Remarks
 
  Loads a menu resource from the application's executable file and
  attaches it to the CMenu object.
 
  Return Value
 
  TRUE if the menu resource was loaded successfully; otherwise FALSE.

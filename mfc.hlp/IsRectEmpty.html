  BOOL IsRectEmpty() const;
 
  Remarks
 
  Determines if CRect is empty. A rectangle is empty if the width and/or
  height are 0 or negative. Differs from IsRectNull, which determines if
  the rectangle is NULL.
 
  Return Value
 
  TRUE if CRect is empty. FALSE if CRect is not empty.

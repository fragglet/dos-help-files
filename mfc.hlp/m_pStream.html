──Microsoft Foundation Classes──────────────────────────────────────────────
 
  Syntax
 
  FILE* m_pStream;
 
  Remarks
 
  The m_pStream data member is the pointer to an open file as returned by
  the C run-time function fopen. It is NULL if the file has never been
  opened or has been closed.

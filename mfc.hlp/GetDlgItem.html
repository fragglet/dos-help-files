  CWnd* GetDlgItem( int nID ) const;
 
  Parameter   Description
 
  <nID>       Specifies the integer ID of the item to be retrieved.
 
  Remarks
 
  Retrieves a pointer to the specified control in a dialog box.
 
  Can be used with any parent-child pair, not just a dialog box, as long
  as the child window has a unique ID (as specified by the <nID> parameter
  in the ◄Create► member function that created the child window).
 
  The pointer returned is usually cast to the type of control identified
  by <nID>.
 
  Return Value
 
  A pointer to the given control. If no control with the integer ID given
  by the <nID> parameter exists, the value is NULL.
 
  The returned pointer may be temporary, and should not be stored.

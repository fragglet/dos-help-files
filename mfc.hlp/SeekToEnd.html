  DWORD SeekToEnd()
  throw( CFileException );
 
  Remarks
 
  Sets the value of the file pointer to the logical end of the file.
  SeekToEnd() is equivalent to  CFile::Seek(0L, CFile::end).
 
  Return Value
 
  The length of the file in bytes.
 
  Example
 
  extern CFile cfile;
  DWORD dwActual = cfile.SeekToEnd();

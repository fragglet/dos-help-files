  static void ThrowOsError( LONG lOsError );
 
  Parameter   Description
 
  <lOsError>  An operating-system-specific error code.
 
  Remarks
 
  This static function throws a CFileException corresponding to a given
  <lOsError> value. If the error code is unknown, then the function throws
  an exception coded as CFileException::generic.
 
  Example
 
  CFileException::ThrowOsError( 5 );  // "access denied"

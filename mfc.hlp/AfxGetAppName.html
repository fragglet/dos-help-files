  const char* AfxGetAppName( );
 
  Remarks
 
  Returns a null-terminated string containing the Windows application's
  name. This string is useful for diagnostic messages or as a root for
  temporary string names.
 
  Return Value
 
  A null-terminated string containing the application's name.

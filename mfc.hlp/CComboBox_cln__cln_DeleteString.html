  int DeleteString( UINT nIndex );
 
  Parameter   Description
 
  <nIndex>    Specifies the index to the string that is to be deleted.
 
  Remarks
 
  Deletes a string in the list box of a combo box.
 
  Return Value
 
  If the return value is greater than or equal to 0, then it is a count of
  the strings remaining in the list. The return value is CB_ERR if
  <nIndex> specifies an index greater then the number of items in the
  list.

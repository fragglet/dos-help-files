  int GetTopIndex() const;
 
  Remarks
 
  Retrieves the zero-based index of the first visible item in a list box.
  Initially, item 0 is at the top of the list box, but if the list box is
  scrolled, another item may be at the top.
 
  Return Value
 
  The zero-based index of the first visible item in a list box.

  static CTime GetCurrentTime();
 
  Remarks
 
  Returns a CTime object that represents the current time.
 
  Example
 
  CTime t = CTime::GetCurrentTime();
  ASSERT( ( t.GetYear() >= 1999 ) && ( t.GetYear() <= 2000 ) );

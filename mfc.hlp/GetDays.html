  LONG GetDays() const;
 
  Remarks
 
  Returns the number of complete days. This value may be negative if the
  time span is negative.
 
  Example
 
  CTimeSpan ts( 3, 1, 5, 12 ); // 3 days, 1 hour, 5 min, and 12 sec
  ASSERT( ts.GetDays() == 3 );

  void OemToAnsi();
 
  Remarks
 
  Converts all the characters in this CString object from the OEM
  character set to the ANSI character set. See the IBM PC Extended
  Character Set table and the ANSI table in the <Microsoft Windows
  Programmer's Reference>.
 
  This function is available only in the Windows compiled library of the
  Microsoft Foundation classes and is declared in AFX.H only if _WINDOWS
  is defined.
 
  Example
 
  CString s( '\253' );  // Octal oem code for '1/2'
  s.OemToAnsi();
  ASSERT( s == "\265" ); // Octal ANSI code for '1/2'

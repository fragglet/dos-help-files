──Microsoft Foundation Classes──────────────────────────────────────────────
 
  The Microsoft Foundation Class Library provides handler functions that
  you can override in your derived window classes. These message handlers
  can be divided into the following categories:
 
  ◄WM_COMMAND Message Handler►      Handles WM_COMMAND messages generated
                                    by user menu selections or menu access
                                    keys.
 
  ◄Child Window Message Handlers►   Handle notification messages from
                                    child windows.
 
  ◄WM_ Message Handlers►            Handle WM_ messages, such as
                                    WM_PAINT.
 
  ◄User-Defined Message Handlers►   Handle user-defined messages.
 
  For more information about this message map reference, see
  ◄How to Use the Cross-Reference►.

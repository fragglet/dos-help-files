  BOOL OpenClipboard();
 
  Remarks
 
  Opens the Clipboard. Other applications will not be able to modify the
  Clipboard until the ◄CloseClipboard► Windows function is called.
 
  The current CWnd object will not become the owner of the Clipboard until
  the ◄EmptyClipboard► Windows function is called.
 
  Return Value
 
  TRUE if the Clipboard is opened via CWnd, or FALSE if another
  application or window has the Clipboard opened.
 
  See Also
 
  ◄::CloseClipboard►, ◄::EmptyClipboard►, ◄::OpenClipboard►

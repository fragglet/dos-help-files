  static void DeleteTempMap();
 
  Remarks
 
  Called automatically by the CWinApp idle time handler, DeleteTempMap
  deletes any temporary CGdiObject objects created by FromHandle.
  DeleteTempMap detaches the Windows GDI object attached to a temporary
  CGdiObject object before deleting the CGdiObject object.

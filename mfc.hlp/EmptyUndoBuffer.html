  void EmptyUndoBuffer();
 
  Remarks
 
  Resets (clears) the undo flag of an edit control. The edit control will
  now be unable to undo the last operation. The undo flag is set whenever
  an operation within the edit control can be undone.
 
  The undo flag is automatically cleared whenever the ◄SetWindowText► or
  SetHandle member function is called.

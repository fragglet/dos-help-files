  BOOL operator ==( const RECT& rect ) const;
 
  Parameter   Description
 
  <rect>      Refers to a source rectangle.
 
  Remarks
 
  Determines if <rect> is equal to CRect by comparing the coordinates of
  their upper-left and lower-right corners.
 
  Return Value
 
  If the values of these coordinates are equal, returns TRUE; otherwise
  FALSE.

  void ClientToScreen( LPPOINT lpPoint ) const;
  void ClientToScreen( LPRECT lpRect ) const;
 
  Parameter   Description
 
  <lpPoint>   Points to a ◄POINT► structure or CPoint that contains the
              client coordinates to be converted.
 
  <lpRect>    Points to a ◄RECT► structure or CRect that contains the
              client coordinates to be converted.
 
  Remarks
 
  Converts the client coordinates of a given point or rectangle on the
  display to screen coordinates. The ClientToScreen member function uses
  the client coordinates in the ◄POINT► or ◄RECT► structure, or CPoint or
  CRect pointed to by <lpPoint> or <lpRect>, to compute new screen
  coordinates; it then replaces the coordinates in the structure with the
  new coordinates. The new screen coordinates are relative to the
  upper-left corner of the system display.
 
  The ClientToScreen member function assumes that the given point or
  rectangle is in client coordinates.

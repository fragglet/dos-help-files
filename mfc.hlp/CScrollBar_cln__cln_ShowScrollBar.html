  Windows 3.1 only
 
  void ShowScrollBar( BOOL bShow = TRUE);
 
  Parameter   Description
 
  <bShow>     Specifies whether the scroll bar is shown or hidden. If this
              parameter is TRUE, the scroll bar is shown; otherwise, it is
              hidden.
 
  Remarks
 
  Shows or hides a scroll bar.
 
  An application should not call this function to hide a scroll bar while
  processing a scroll-bar notification message.

  int GetWindowTextLength() const;
 
  Remarks
 
  Returns the length of the CWnd object caption title. If CWnd is a
  control, the GetWindowTextLength member function returns the length of
  the text within the control instead of the caption.
 
  Return Value
 
  Specifies the text length, not including any null-termination character.
  The value is 0 if no such text exists.

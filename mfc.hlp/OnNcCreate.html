  afx_msg BOOL OnNcCreate( LPCREATESTRUCT  lpCreateStruct  );
 
  Parameter        Description
 
  <lpCreateStruct> Points to the CREATESTRUCT data structure for CWnd.
 
  Remarks
 
  Called prior to the ◄WM_CREATE► message when the CWnd is first created.
 
  By default, scroll bars are initialized (the scroll-bar position and
  range are set) and the CWnd text is set. Memory used internally to
  create and maintain the window is allocated.
 
  This message-handler member function calls the ◄Default► member
  function. Override this member function in your derived class to handle
  the ◄WM_NCCREATE► message.
 
  Return Value
 
  TRUE if the nonclient area is created. It is FALSE if an error occurs;
  the Create function will return NULL in this case.

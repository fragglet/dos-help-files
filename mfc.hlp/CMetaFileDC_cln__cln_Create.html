  BOOL Create(const char FAR* lpFilename = NULL );
 
  Parameter    Description
 
  <lpFilename> Points to a null-terminated character string. Names an
               existing metafile on disk to load. If <lpFilename> is NULL,
               a new in-memory metafile is created.
 
  Remarks
 
  You construct a CMetaFileDC object in two steps. First, call the
  constructor CMetaFileDC, then call Create, which creates the Windows
  metafile device context and attaches it to the CMetaFileDC object.
 
  Return Value
 
  TRUE if the function is successful; otherwise FALSE.

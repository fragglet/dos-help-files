  BOOL EqualRgn( CRgn* pRgn ) const;
 
  Parameter   Description
 
  <pRgn>      Identifies a region.
 
  Remarks
 
  Checks the given region to determine whether it is equivalent to the
  region stored in the CRgn object.
 
  Return Value
 
  TRUE if the two regions are equivalent; otherwise FALSE.

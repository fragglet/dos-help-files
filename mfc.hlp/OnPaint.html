  afx_msg void OnPaint();
 
  Remarks
 
  Called when Windows or an application makes a request to repaint a
  portion of the CWnd.
 
  The WM_PAINT message is sent to this member function when the
  ◄UpdateWindow► member function is called, or when the update region is
  not empty and there are no pending messages.
 
  This message-handler member function calls the ◄Default► member
  function. Override this member function in your derived class to handle
  the ◄WM_PAINT► message.

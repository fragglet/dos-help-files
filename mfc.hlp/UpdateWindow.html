  void UpdateWindow();
 
  Remarks
 
  Updates the client area by sending a WM_PAINT message if the update
  region is not empty. The UpdateWindow member function sends a
  ◄WM_PAINT► message directly, bypassing the application queue. If the
  update region is empty, WM_PAINT is not sent.

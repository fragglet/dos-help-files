  Windows 3.1 only
 
  BOOL GetDroppedState() const;
 
  Remarks
 
  Call the GetDroppedState member function to determine whether the
  list-box of a drop-down combo box is visible (dropped down).
 
  Return Value
 
  Nonzero if the listbox is visible; otherwise, 0.

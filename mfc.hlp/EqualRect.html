  BOOL EqualRect( LPRECT lpRect ) const;
 
  Parameter   Description
 
  <lpRect>    Points to a RECT or CRect that contains the upper-left and
              lower-right corner coordinates of a rectangle.
 
  Return Value
 
  TRUE if the two rectangles have the same top, left, bottom, and right
  values; otherwise FALSE.

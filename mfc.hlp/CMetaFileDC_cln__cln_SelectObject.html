  BOOL SelectObject( CGdiObject* pObject );
 
  Parameter   Description
 
  <pObject>   Identifies the object to be selected into the CMetaFileDC.
              The selected object can be one of the following:
 
              CBitmap
              CBrush
              CFont
              CPen
              CRgn
              CPalette
 
  Remarks
 
  Selects an object into the CMetaFileDC.
 
  CMetaFileDC performs its own object cleanup, so an application does not
  have to reselect default objects when recording a metafile.
 
  Return Value
 
  TRUE if the function is successful; otherwise FALSE.

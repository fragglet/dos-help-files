  DWORD GetTextColor() const;
 
  Remarks
 
  Retrieves the current text color. The text color is the foreground color
  of characters drawn by using the GDI text-output functions ◄TextOut►,
  ◄ExtTextOut►, and ◄TabbedTextOut►.
 
  Return Value
 
  The current text color as an RGB color value.

  CPoint OffsetWindowOrg( int nWidth, int nHeight );
 
  Parameter   Description
 
  <nWidth>    Specifies the number of logical units to add to the current
              origin's x-coordinate.
 
  <nHeight>   Specifies the number of logical units to add to the current
              origin's y-coordinate.
 
  Remarks
 
  Modifies the window origin relative to the coordinates of the current
  window origin.
 
  Return Value
 
  The previous window origin (in logical coordinates) as a ◄CPoint►
  object.

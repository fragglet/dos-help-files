  CRect operator |( const RECT& rect2 ) const;
 
  Parameter   Description
 
  <rect2>     Contains a RECT or CRect.
 
  Return Value
 
  A CRect that is the union of CRect and <rect2>. A union is the smallest
  rectangle that contains both source rectangles.
 
  Windows ignores the dimensions of an empty rectangle, that is, a
  rectangle that has no height or has no width.

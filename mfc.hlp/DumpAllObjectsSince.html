  void DumpAllObjectsSince() const;
 
  Remarks
 
  Calls the Dump function for all objects of derived CObject classes that
  were allocated (and are still allocated) since the last ◄Checkpoint►
  call for this CMemoryState object.
 
  Use DumpAllObjectsSince in conjunction with ◄AfxCheckMemory► to match
  reported corrupted memory with the contents of the objects contained
  there.
 
  Calling DumpAllObjectsSince with an uninitialized CMemoryState object
  will dump out all objects currently in memory.
 
  Example
 
  See the example for the ◄CMemoryState► constructor.

  BOOL IsRectNull() const;
 
  Remarks
 
  Determines if the top, left, bottom, and right values of the CRect are
  all equal to 0. Differs from IsRectEmpty, which determines if the
  rectangle is empty.
 
  Return Value
 
  TRUE if CRect's top, left, bottom, and right values are all equal to 0;
  otherwise FALSE.

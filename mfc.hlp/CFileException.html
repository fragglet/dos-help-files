  Description
 
  A CFileException object represents a file-related exception condition.
  The CFileException class includes public data members that hold the
  portable cause code and the operating-system-specific error number. The
  class also provides static member functions for throwing file exceptions
  and for returning cause codes for both operating-system errors and C
  run-time errors.
 
  #include <afx.h>
 
  See Also
 
  ◄CFile►, Chapter 5, "◄Exception Processing►"
 
  Comments
 
  CFileException objects are constructed and thrown in CFile member
  functions and in member functions of derived classes. You can access
  these objects within the scope of a CATCH expression. For portability,
  use only the cause code to get the reason for an exception.

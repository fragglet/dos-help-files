  const CTimeSpan& operator =( const CTimeSpan& timeSpanSrc );
 
  Remarks
 
  The overloaded assignment operator copies the source CTimeSpan
  <timeSpanSrc> object into this CTimeSpan object.
 
  Example
 
  CTimeSpan ts1;
  CTimeSpan ts2( 3, 1, 5, 12 ); // 3 days, 1 hour, 5 min, and 12 sec
  ts1 = ts2;
  ASSERT( ts1 == ts2 );

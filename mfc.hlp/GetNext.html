  CObject*& GetNext( POSITION& rPosition );
  CObject* GetNext( POSITION& rPosition ) const;
 
  Parameter   Description
 
  <rPosition> A reference to a POSITION value returned by a previous
              GetNext, GetHeadPosition, or other member function call.
 
  Remarks
 
  GetNext gets the list element identified by <rPosition>, then sets
  <rPosition> to the POSITION value of the next entry in the list. You can
  use GetNext in a forward iteration loop if you establish the initial
  position with a call to GetHeadPosition or Find.
 
  You must ensure that your POSITION value represents a valid position in
  the list. If it is invalid, then the Debug version of the library
  asserts.
 
  If the retrieved element is the last in the list, then the new value of
  <rPosition> is set to NULL.
 
  It is possible to remove an element during an iteration. See the example
  for ◄RemoveAt►.
 
  Return Value
 
  See the return value description for ◄GetHead►.
 
  Example
 
     CObList list;
     POSITION pos;
 
     list.AddHead( new CAge( 21 ) );
     list.AddHead( new CAge( 40 ) ); // List now contains (40, 21)
     // Iterate through the list in head-to-tail order
     for( pos = list.GetHeadPosition(); pos != NULL; )
     {
  #ifdef _DEBUG
         afxDump << list.GetNext( pos ) << "\n";
  #endif
     }
 
  The results from this program are as follows:
 
  a CAge at $479C 40
  a CAge at $46C0 21

     Debug Information Has Changed Without Setting Debug Options.
 
     This message is displayed when you choose a different debugger in
     the COBOL Compiler Options dialog box without setting any debug
     options.
 
     When you choose OK, the Debug Options dialog box is displayed so
     you can verify the options.

COBOL Show Compiler Options
                                             ◄Up► ◄Contents► ◄Index► ◄Back►
────────────────────────────────────────────────────────────────────────────
 
     The Show Options dialog box lists the command-line options that
     are used for the options you selected in the main COBOL Compiler
     Options dialog box.
 
     The displayed options include the current options selected in the Set
     Debug Options or Set Release Options dialog boxes.

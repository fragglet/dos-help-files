  Message:      RT218
 
  Text:         Malformed MULTIPLE REEL/UNIT file
 
  Severity:     Fatal
 
  Explanation:  Either your file header is not correctly formatted, or
                you are not using a MULTIPLE REEL/UNIT file.
 
  Resolution:   You will have to attempt to run your program again
                using a backup copy of the relevant file.

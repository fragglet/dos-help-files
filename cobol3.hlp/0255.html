  Message:      0255
 
  Text:         User name <data-name> same as special register
 
  Explanation:  You have declared a data item with the same name as
                one of the special registers, which are data items
                that are automatically declared by your COBOL system.
 
  Resolution:   Check your Language Reference for a list of the
                names of these special registers. Alter the name of the
                data item in error.

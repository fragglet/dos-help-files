  Message:      RT189
 
  Text:         Intermediate code load error
 
  Severity:     Fatal
 
  Explanation:  You are unable to load your intermediate code.  You
                may have attempted to load intermediate code that either
                has not been successfully produced, or has been corrupted
                in some way.
 
  Resolution:   Try to obtain good intermediate code, for example, by
                resubmitting (or submitting) your source code to your
                COBOL system. You should then be able to load your
                code and run the program successfully.

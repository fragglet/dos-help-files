  Message:      0363
 
  Text:         <data-name> not in same record as first item in CODE-SET
                clause
 
  Explanation:  You have specified data-names in the CODE-SET clause
                that belong to more than one record.  All data-names
                specified in this clause must belong to a single
                record in the file.
 
  Resolution:   Edit your source code to delete one or more of the
                CODE-SET data-names to ensure that all data-names
                belong to only one record.

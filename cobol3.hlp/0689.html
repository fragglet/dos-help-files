  Message:      0689
 
  Text:         Condition name cannot be set to FALSE (no FALSE value
                specified)
 
  Explanation:  You have tried to set a level-88 item to FALSE without
                first having specified the SET TO FALSE statement.
 
  Resolution:   Specify the SET TO FALSE statement in your program.

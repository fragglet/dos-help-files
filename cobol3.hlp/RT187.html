  Message:      RT187
 
  Text:         Run-Time System not found on $COBDIR path
 
  Severity:     Fatal
 
  Explanation:  The run-time system cannot be found on the path you
                have set up in the COBDIR environment variable.
                Alternatively, you may not have installed your COBOL
                system correctly.
 
  Resolution:   Ensure that the run-time system is on the path you
                have set up in the COBDIR environment variable.
                Alternatively, ensure that your COBOL system has been
                installed correctly.  If it has not, you must
                reinstall your COBOL system.

  Message:      RT225
 
  Text:         Dynamic load error - program component missing
 
  Severity:     Fatal
 
  Explanation:  The run-time system cannot locate the root or overlay of
                a program that is currently loaded in memory.
 
  Resolution:   Either the library that contained the program has been
                canceled, or the program is no longer available on the
                program search path. Ensure that the program is available
                either on disk or on an open library.

  Message:      RT235
 
  Text:         Error in animation communications channel
 
  Severity:     Fatal
 
  Explanation:  Animator has encountered system limits or conflicts
                resulting in communications errors. Perhaps two logins on
                the same UNIX system are attempting to cross session
                animate the same program file.
 
  Resolution:   Copy the animated program file to a different directory
                and attempt animation.

  Message:      RT183
 
  Text:         Attempt to open line sequential file for I-O
 
  Severity:     Fatal
 
  Explanation:  You have tried to open a line-sequential file in the
                input-output open mode, but this mode is not supported
                for files with this organization.
 
  Resolution:   When your program has terminated you will have to
                recode your program to ensure that the file with
                organization line sequential is opened for input,
                output, or extend.  You will then be able to rerun
                your code.

  Message:      RT045
 
  Text:         Attempt to open an NLS file using an incompatible
                program
 
  Severity:     Fatal
 
  Explanation:  The NLS control information for a file within your
                program does not match the same NLS control information
                in the header of your index file. Alternatively, your index
                file has become corrupted.
 
  Resolution:   Rebuild your index file, or rerun your program using
                the backup copy of that file. If you have added a great
                deal of information since you last took a backup, you may
                wish to rebuild the file using a utility that is able to
                read the data, if it is not corrupt, and build a new
                index for it.

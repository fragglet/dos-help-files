  Message:      0064
 
  Text:         If file is EXTERNAL, then PADDING CHARACTER must be
                also
 
  Explanation:  If a file is defined as EXTERNAL then the data item
                specified in the associated PADDING CHARACTER clause
                must also be defined as EXTERNAL.
 
  Resolution:   Change your program.

  Message:      RT210
 
  Text:         File is closed with lock
 
  Severity:     Fatal
 
  Explanation:  You have tried to open a file which you have
                previously closed with lock.
 
  Resolution:   Recode your program to avoid opening a file which has
                previously been closed with lock.

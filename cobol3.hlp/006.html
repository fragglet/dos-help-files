  Message:      006
 
  Text:         Read failure on input file, not intermediate code or
                wrong version
 
  Severity:     Recoverable
 
  Explanation:  1.  the input file you have specified is not an
                    intermediate code file.
 
                2.  the intermediate code file was produced by a
                    version of your COBOL system that is not compatible
                    with your current version.
 
                3.  your input file is corrupt.
 
  Resolution:   1.  ensure that you specify an intermediate code file.

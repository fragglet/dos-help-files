  Message:      0252
 
  Text:         PASSWORD field <data-name> missing or illegal
 
  Explanation:  You have either not declared the data item specified in
                the PASSWORD clause in a file description entry, or you
                have declared the item incorrectly.
 
 
  Resolution:   Check your Language Reference for the correct
                form of the password data item. Add or correct the
                declaration of the data item in the PASSWORD clause.

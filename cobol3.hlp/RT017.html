  Message:      RT017
 
  Text:         Record error:  probably zero length
 
  Severity:     Recoverable
 
  Explanation:  You have probably tried to access a record that has
                had no value moved into it.
 
  Resolution:   Although this error is recoverable in the sense that
                it can be trapped, once it has been reported you must
                execute a STOP RUN statement immediately and then
                recode your program to ensure that the COBOL record
                length is not zero.

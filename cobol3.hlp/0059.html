  Message:      0059
 
  Text:         Cannot GENERATE this report name
 
  Explanation:  The report name that you have specified should
                contain all of the following:
 
                 o  a CONTROL clause
 
                 o  only one DETAIL report group
 
                 o  at least one body group.
 
  Resolution:   Check your Language Reference for details of the
                contents you require for a report name when it is used in a
                GENERATE statement.

  Message:      0054
 
  Text:         Class name required
 
  Explanation:  You have failed to define the class name in the
                SPECIAL-NAMES paragraph, or you have misspelled a
                COBOL class name.
 
  Resolution:   Either define the condition name in the SPECIAL-NAMES
                paragraph, or use one of the COBOL class names as
                specified in your Language Reference.

  Message:      RT070
 
  Text:         Too many indexed files open
 
  Severity:     Recoverable
 
  Explanation:  You are attempting to open an indexed file but you
                have already exhausted the system limit which specifies
                how many of these files can be opened at any one time.
 
  Resolution:   Close some of the open indexed files which you are not
                currently accessing.  You should then be able to open
                the indexed file which you require and to continue the
                program run.

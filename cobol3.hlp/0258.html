  Message:      0258
 
  Text:         ASSIGN data-name <data-name> illegal
 
  Explanation:  You have specified an ASSIGN data-name in a SELECT ...
                ASSIGN clause that is not unitary, that is, the data
                definition contains an OCCURS clause.
 
  Resolution:   Edit your source code to ensure that the ASSIGN
                data item does not include an OCCURS clause.

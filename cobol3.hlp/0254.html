  Message:      0254
 
  Text:         'VALUE OF' field <data-name> missing or illegal
 
  Explanation:  You have either not declared one or more of the data
                items named in the VALUE OF clause of an FD entry in the
                file section, or you have declared such an item incorrectly.
 
  Resolution:   Check your Language Reference for the correct
                format of such data items. Add or correct the declaration
                of the data item(s) in the VALUE OF clause.

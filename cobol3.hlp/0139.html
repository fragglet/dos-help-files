  Message:      0139
 
  Text:         Illegal LOCK MODE/file type combination
 
  Explanation:  The format of the LOCK MODE clause depends on the file
                type. You have specified a LOCK MODE clause which is
                incompatible with the type of the specified file.
                Non-shareable files must have lock mode EXCLUSIVE
                whilst shareable files may have lock mode AUTOMATIC or
                MANUAL.
 
  Resolution:   Refer to your Language Reference for further
                details on the use of this clause.

  Message:      RT066
 
  Text:         Attempt to add duplicate record key to indexed file
 
  Severity:     Fatal
 
  Explanation:  You have tried to add a duplicate key for a key which
                you have not defined as being able to have duplicates.
 
 
  Resolution:   As this error implies that there is a fault in the
                logic of your program you will probably need to
                recode.

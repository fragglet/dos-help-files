  Message:      0243
 
  Text:         VALUE of group item must be non-numeric or figurative
                constant
 
  Explanation:  If you specify a VALUE clause for a group item, the
                value in the clause must be either a non-numeric
                literal or a figurative constant.
 
  Resolution:   Revise the value in the VALUE clause.

  Message:      RT142
 
  Text:         File not open - cannot be closed
 
  Severity:     Recoverable
 
  Explanation:  You have tried to close a file which is not open which
                is impossible to achieve.
 
  Resolution:   You can abandon your attempt to close the relevant
                file and continue to run your program. However, as this
                error implies that your program contains a mistake in its
                logic, you may wish to close any open files, execute a
                STOP RUN statement and then recode.

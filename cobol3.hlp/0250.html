  Message:      0250
 
  Text:         STATUS field <data-name> missing or illegal
 
  Explanation:  You have either not declared the data item specified
                in the FILE STATUS clause of a file control entry, or you
                have declared it incorrectly.
 
  Resolution:   Check your Language Reference for details of the
                correct form of the FILE STATUS data item. Add or revise
                the definition of the FILE STATUS data item.

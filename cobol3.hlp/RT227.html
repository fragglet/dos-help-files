  Message:      RT227
 
  Text:         EXTERNAL data definition inconsistent
 
  Explanation:  Two or more programs are defining the same external data
                item, but the first loaded program has defined the size
                smaller than has the second or subsequent loaded program.
 
  Resolution:   Make sure that both or all of your programs define the
                size of the external data item as being the same.

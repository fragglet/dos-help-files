  Message:      0311
 
  Text:         Illegal syntax used with I-O verb
 
  Explanation:  Code following an I/O verb (for example READ or WRITE)
                violates the rules of COBOL syntax.
 
  Resolution:   See your Language Reference for details on
                the usage of the particular verb in your code which has
                caused this error message. Revise your code accordingly.

  Message:      RT215
 
  Text:         Cannot ANIMATE a program running COMMUNICATIONS
 
  Severity:     Fatal
 
  Explanation:  You have tried to animate a program which makes use of
                the communications module.  This cannot be done as
                both Animator and the communications module need full
                use of the CRT.
 
  Resolution:   You will have to run your program without the aid of
                Animator.

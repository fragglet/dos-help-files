  Message:      0021
 
  Text:         Too many qualifiers
 
  Explanation:  You have used too many qualifiers when referring to a
                qualified data name, procedure name, or text name. See
                your Language Reference for details of how many
                qualifiers you can specify.
 
  Resolution:   Change the reference in error so that it uses no more
                than the permitted number of qualifiers.

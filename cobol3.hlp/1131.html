  Message:      1131
 
  Text:         Statement generated no executable code
 
  Explanation:  This error may be produced when, for example, an
                INITIALIZE statement does not cause any code to be
                created.  It is an indication that the statement may
                be removed from the program.  If the error is ignored
                the effect is as if the statement were removed.

  Message:      RT037
 
  Text:         File access denied
 
  Severity:     Fatal
 
  Explanation:  Your attempt to access a file has been denied by the
                operating system.  You may have tried to write to a
                write-protected file or you could have attempted to
                read from an output device.
 
  Resolution:   Alter the access permission on the relevant file.
                Access can be read-only, if you just want to read the
                contents of the file without making any changes, or it
                can be read and write in which case you will be able
                to alter its contents.

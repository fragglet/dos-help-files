  Message:      RT141
 
  Text:         File already open - cannot be opened
 
  Severity:     Recoverable
 
  Explanation:  You have tried to open a file which is already open
                and so cannot be opened again.
 
  Resolution:   Cancel your second attempt to open the file and
                continue to run your program if the fact that the file
                is already open is acceptable to you.  However as this
                error implies that your program contains a mistake in its
                logic, you may wish to close any open files, execute a STOP
                RUN statement and then recode.

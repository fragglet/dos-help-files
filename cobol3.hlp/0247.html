  Message:      0247
 
  Text:         Index-name has been declared explicitly
 
  Explanation:  You have declared an index data item explicitly.  The
                item is declared implicitly by its appearance in an
                INDEX phrase, so you have in effect declared the same
                item twice.
 
  Resolution:   Delete the explicit declaration of the index data item.

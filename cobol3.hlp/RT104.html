  Message:      RT104
 
  Text:         Null file name used in a file operation
 
  Severity:     Fatal
 
  Explanation:  You specified a data-name for a file-name instead of a
                literal, and the data-item contained only spaces when you
                tried to open the file.
 
  Resolution:   Recode your program specifying the correct file-name.

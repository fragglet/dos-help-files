  Message:      RT118
 
  Text:         Symbol not found
 
  Severity:     Fatal
 
  Explanation:  You are unable to load your object file.  You have
                attempted to call a program that has not been specified in
                the COBPATH environment variable.
 
  Resolution:   Check that your COBPATH has been set up correctly. If
                not, revise your COBPATH to include the program being
                called.

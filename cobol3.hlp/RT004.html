  Message:      RT004
 
  Text:         Illegal file name
 
  Severity:     Recoverable
 
  Explanation:  A file-name contains an illegal character. This could
                be any character that is not part of the permitted
                character set or it could be the system-dependent
                delimiter, which on most systems is the space.
 
  Resolution:   Try the file operation again using the correct
                file-name.

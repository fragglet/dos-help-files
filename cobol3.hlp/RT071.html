  Message:      RT071
 
  Text:         Bad indexed file format
 
  Severity:     Fatal
 
  Explanation:  You are either using a file which has been corrupted,
                or there is an internal system error.
 
  Resolution:   If the disk you are using is corrupt, rerun your
                program using your backup copy of the disk.  If this
                is not the cause of the error then you should contact
                Technical Support who will try to discover the cause
                of your error and how it can be rectified.

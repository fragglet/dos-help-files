  Message:      005
 
  Text:         Open failure on temp file
 
  Severity:     Recoverable
 
  Explanation:  The file cannot be opened. The disk or directory
                may be full, or there may be a read-only file with
                the same name.
 
  Resolution:   If your disk or directory is full, delete any files
                you no longer need. If there is a read-only file with
                the same name, delete it.

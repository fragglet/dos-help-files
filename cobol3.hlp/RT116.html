  Message:      RT116
 
  Text:         Cannot allocate memory
 
  Severity:     Fatal
 
  Explanation:  For some reason a part of your run-time system is
                unable to allocate you sufficient memory to enable you
                to execute your code.
 
  Resolution:   You should try to reduce memory usage by cancelling
                programs that are not in use, then attempt the
                operation that caused this message again.

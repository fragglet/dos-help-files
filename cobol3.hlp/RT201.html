  Message:      RT201
 
  Text:         I-O error in paging system
 
  Severity:     Fatal
 
  Explanation:  There is no room available in your current directory
                or on the floppy disk which you are using, for the
                paging file.
 
  Resolution:   When your program has terminated, delete any files
                which you no longer need from your directory to make
                room for the paging file, or insert a new floppy disk.

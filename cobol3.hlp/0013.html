  Message:      0013
 
  Text:         User-name required
 
  Explanation:  You have not supplied a user-defined name at the
                specified place in your program.
 
  Resolution:   Insert a name ensuring that it conforms to the rules
                of COBOL syntax and that it is the correct type of
                name (for example a system name or a condition name).

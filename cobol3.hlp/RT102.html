  Message:      RT102
 
  Text:         Sequential file with non-integral number of records
 
  Severity:     Fatal
 
  Explanation:  You have either specified an incorrect record length for a
                sequential file, or the sequential file you are attempting
                to access is corrupt in some way, or the file which you
                have specified is not a sequential file.
 
  Resolution:   Recode your program so that it specifies the correct
                type of file, or if the error is the result of a
                corrupt file, attempt to run the program again using a
                backup copy of that file.

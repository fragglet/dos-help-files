  Message:      RT120
 
  Text:         Symbol string table of zero size
 
  Severity:     Fatal
 
  Explanation:  You probably have a malformed object file.
 
  Resolution:   Once the program has terminated you will need to
                correct your object file.  If this does not work,
                contact Technical Support who will try to help you to
                discover the specific cause of the error.

  Message:      0211
 
  Text:         PICTURE clause not compatible with qualifiers
 
  Explanation:  Your code contains a PICTURE clause that is qualified
                by the wrong type of qualifier.  For example, a data item
                may not be defined as PIC XX USAGE COMP because it is an
                alphanumeric PICTURE clause but the qualifier is numeric.
 
  Resolution:   Revise your code so that the PICTURE clause and its
                qualifiers are of the same type.

  Message:      0253
 
  Text:         LABEL RECORD or DATA RECORD <record-name> missing or
                illegal
 
  Explanation:  You have either not declared the record item specified
                in the LABEL RECORD or DATA RECORD clauses in a file
                description entry, or you have declared the item
                incorrectly.
 
  Resolution:   Check your Language Reference for the correct
                format of the label record and data record item. Add or
                correct the declaration of the record item in the LABEL
                RECORD or DATA RECORD clause.

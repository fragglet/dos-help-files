  Message:      0045
 
  Text:         Operand <data-name> has wrong size
 
  Explanation:  The operand in this statement is the wrong length (for
                example, you have specified a prompt character more
                than one character long).
 
  Resolution:   Check your Language Reference for the correct
                length of the operand.

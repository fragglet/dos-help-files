  Message:      RT105
 
  Text:         Memory allocation error
 
  Severity:     Fatal
 
  Explanation:  The run-time system is unable to allocate sufficient
                memory space to successfully carry out the attempted
                operation, probably because there is no memory
                space left on your system.
 
  Resolution:   You will have to obtain more memory in which to run
                your program.  Refer to your operating system
                documentation for details of how you can obtain more
                memory, if this is possible.

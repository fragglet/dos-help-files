  Message:      RT203
 
  Text:         CALL parameter not supplied
 
  Severity:     Fatal
 
  Explanation:  The item you are trying to access in the Linkage Section
                of the currently executing program has not been initialized.
 
  Resolution:   Recode your program to ensure that it contains all of
                the necessary parameters, or check that it is a valid
                caller.

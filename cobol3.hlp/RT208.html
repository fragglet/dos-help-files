  Message:      RT208
 
  Text:         Error in multi-user system
 
  Severity:     Fatal
 
  Explanation:  This is normally caused by an unexpected error
                occurring within the network or file-sharing
                facilities.  A corrupted network message will also
                return this error.
 
  Resolution:   Retry the unsuccessful operation. If the error
                persists, contact Technical Support who will try to help
                you to discover the specific cause of this error.

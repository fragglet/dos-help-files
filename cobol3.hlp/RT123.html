  Message:      RT123
 
  Text:         Unknown relocation type
 
  Severity:     Fatal
 
  Explanation:  You are using incompatible versions of the object file
                and the COBOL run-time library.
 
  Resolution:   Once the program has terminated, resubmit your object
                file to your COBOL system with the current version of your
                COBOL run-time library. If this does not work, contact
                Technical Support who will try to help you to discover the
                specific cause of the error.

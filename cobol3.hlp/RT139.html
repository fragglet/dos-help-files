  Message:      RT139
 
  Text:         Record length or key data inconsistency
 
  Severity:     Recoverable
 
  Explanation:  There is a discrepancy between the length of a record,
                or the keys which you have specified, in your current
                program and its definition in the program in which it
                was first opened.
 
  Resolution:   There is a fault in your program so you will probably
                need to edit your code, then resubmit it to your COBOL
                system before running it again.

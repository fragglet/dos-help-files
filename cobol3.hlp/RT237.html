  Message:      RT237
 
  Text:         Unable to initialize animated process
 
  Severity:     Fatal
 
  Explanation:  Animator has encountered system limits in starting your
                program. The animated program cannot start because of
                either insufficient memory or too many processess running
                on the UNIX system.
 
  Resolution:   Animate your program when the load on the system has
                reduced. Run the program without animation to determine if
                it runs successfully; if so, animate the program without
                cross session or intra session animation. Ask your system
                administrator to expand the number of process slots and
                virtual memory page maps.

  Message:      0121
 
  Text:         Cannot specify DYNAMIC or EXTERNAL with literal
                file-name
 
  Explanation:  You can specify DYNAMIC or EXTERNAL in the ASSIGN
                clause of a file description entry only if the file name is
                contained in a data item rather than given as a
                literal file name.
 
  Resolution:   Redesign your program so that the file name is
                contained in a data item.

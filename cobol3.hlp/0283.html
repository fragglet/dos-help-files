  Message:      0283
 
  Text:         An EXTERNAL file cannot be subject of SAME RECORD AREA
                clause
 
  Explanation:  You have specified a file in a SAME RECORD AREA clause
                in your program that is specified as EXTERNAL in an FD
                entry.
 
  Resolution:   Delete the EXTERNAL file in error from the SAME
                RECORD AREA clause in your program.

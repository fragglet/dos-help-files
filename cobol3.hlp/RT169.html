  Message:      RT169
 
  Text:         Illegal configuration information
 
  Severity:     Fatal
 
  Explanation:  You have attempted an operation for which your machine
                is not configured;  the most likely cause of this is
                that ADIS is not configured correctly.
 
  Resolution:   Check that ADIS is configured correctly.  See your
                COBOL System Reference for details of how you can
                reconfigure ADIS.

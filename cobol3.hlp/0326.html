  Message:      0326
 
  Text:         Literal cannot be receiving field
 
  Explanation:  You have specified a literal value as the receiving
                field in an operation involving an implicit or
                explicit move.  A receiving field must be a data item.
 
 
  Resolution:   Change the literal value to a reference to a data item.

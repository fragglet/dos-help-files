  Message:      0008
 
  Text:         Unknown COPY file <file-name> specified
 
  Explanation:  A file with the name file-name, specified in
                conjunction with a COPY statement, cannot be found.
 
  Resolution:   Change the file-name, or make the file available to
                your COBOL system. See your COBOL System Reference for
                details of the .CBL and .CPY extensions.

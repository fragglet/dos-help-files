  Message:      0263
 
  Text:         Not allowed when PAGE not specified in RD
 
  Explanation:  You have either specified a TYPE PH or TYPE PF clause
                without specifying a PAGE clause in the associated RD entry,
                or you have specified an absolute line number in the LINE
                NUMBER clause without specifying a PAGE clause in the
                associated RD entry.
 
  Resolution:   Add a PAGE clause to the RD entry.

LEFT$ Function Programming Example
 
This example prints the leftmost five characters of A$.
 
CLS                ' Clear screen
A$="BASIC LANGUAGE"
B$=LEFT$(A$, 5)
PRINT B$
 
Sample Output
 
BASIC

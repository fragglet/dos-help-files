ENVIRON$ Function Programming Example
 
This example uses the ENVIRON$ function to print the current settings
in the environment-string table:
 
CLS      ' Clear screen
I = 1
DO WHILE ENVIRON$(I) <> ""
    PRINT ENVIRON$(I)
    I = I + 1
LOOP

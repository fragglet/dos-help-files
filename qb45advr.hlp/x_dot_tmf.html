TIME$ - a function that returns the current time from the operating system
 
Syntax
  TIME$
    ■ an eight-character string is returned, in the form
        hh:mm:ss
      hh is the hour (00-23),
      mm is the minute (00-59), and
      ss is the second (00-59)

WIDTH Statement Programming Example
 
In the following example, the record width for file #1 (the printer) is set
to different widths:
 
'*** Programming example that uses WIDTH ***
OPEN "LPT1:" FOR OUTPUT AS #1
Test$ = "1234567890"
WIDTH #1, 3
PRINT #1, Test$
WIDTH #1, 4
PRINT #1, Test$
CLOSE
 
Sample Output (On a printer)
 
123
456
789
0
1234
5678
90

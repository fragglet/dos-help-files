DATE$ - a function that returns a string containing the current date
 
Syntax
  DATE$
    ■ a ten-character string is returned, in the form
        mm-dd-yyyy
      mm is the month (01-12),
      dd is the day (01-31), and
      yyyy is the year (1980-2099)

FIX Function Programming Example
 
The following statements illustrate the differences between INT and FIX:
 
CLS    ' Clear screen
PRINT INT(-99.8)
PRINT FIX(-99.8)
PRINT INT(-99.2)
PRINT FIX(-99.2)
 
Output
 
-100
-99
-100
-99

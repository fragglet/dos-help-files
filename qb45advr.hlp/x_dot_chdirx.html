CHDIR Statement Programming Example
 
Example 1
 
' Makes HOMESALES the current directory on the default
' drive.
CHDIR "HOMESALES"
 
Example 2
 
' Changes the current directory to USERS on drive B; it does
' not, however, change the default drive to B.
CHDIR "B:USERS"

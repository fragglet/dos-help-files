ON UEVENT Statement Programming Example
 
The following example illustrates the use of ON UEVENT GOSUB:
 
ON UEVENT GOSUB Event1
UEVENT ON
INPUT "Enter a number"; a
IF a = 5 THEN CALL Setuevent
END
'
Event1:
PRINT "Got to the event handler"
RETURN

HEX$ Function Programming Example
 
This example displays the hexadecimal value of a decimal number
you supply.
 
CLS          ' Clear screen
INPUT X
A$ = HEX$(X)
PRINT X; "decimal is "; A$; " hexadecimal"
 
Sample Output
 
? 32
 32 decimal is 20 hexadecimal

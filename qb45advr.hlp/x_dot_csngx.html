CSNG Function Programming Example
 
The following example shows how CSNG rounds before converting the
value:
 
A#=975.3421115#
B#=975.3421555#
PRINT A#; CSNG(A#); B#; CSNG(B#)
 
Sample Output
 
975.3421115  975.3421  975.3421555  975.3422

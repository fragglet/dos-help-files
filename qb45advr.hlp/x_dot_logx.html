LOG Function Programming Example
 
This example prints the value of e and then prints the natural logarithms
of e taken to the first, second, and third powers.
 
CLS                    ' Clear screen
PRINT EXP(1),
FOR I = 1 TO 3
    PRINT LOG(EXP(1) ^ I),
NEXT
 
Sample Output
 
2.718282       1       2       3
